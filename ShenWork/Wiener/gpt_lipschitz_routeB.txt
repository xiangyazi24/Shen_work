# Constants

Let

```lean
def Cdec : ℝ :=
  8 * (1 + 1 / Real.pi) ^ 2
```

This is exactly the committed decisive constant:

```lean
‖NormedSpace.exp (((-s:ℝ):ℂ) • f)‖
  ≤ Cdec * (1 + s * ‖GWA.gDeriv f‖)^2 * Real.exp (-δ * s)
```

under `UniformFloor f δ`. fileciteturn17file0L87-L93 fileciteturn18file0L3-L5

For a derivative bound parameter `Md ≥ 0`, define:

```lean
def negNormConst (s δ Md : ℝ) : ℝ :=
  (1 / Real.Gamma s) * Cdec *
    ((1 / δ) ^ s * Real.Gamma s
      + 2 * Md * (1 / δ) ^ (s + 1) * Real.Gamma (s + 1)
      + Md ^ 2 * (1 / δ) ^ (s + 2) * Real.Gamma (s + 2))

def negLipConst (s δ Md : ℝ) : ℝ :=
  (1 / Real.Gamma s) * Cdec *
    ((1 / δ) ^ (s + 1) * Real.Gamma (s + 1)
      + 2 * Md * (1 / δ) ^ (s + 2) * Real.Gamma (s + 2)
      + Md ^ 2 * (1 / δ) ^ (s + 3) * Real.Gamma (s + 3))
```

The first bounds `‖FnegEWA f s‖`. The second bounds the Lipschitz constant of `FnegEWA · s`.

The only reason `negLipConst` has powers `s+1`, `s+2`, `s+3` is the extra factor `t` from differentiating `exp(-t f)`.

# Core estimate for route B

Let

```lean
seg f g θ := ((1 - θ : ℝ) : ℂ) • g + ((θ : ℝ) : ℂ) • f
```

Then for `0 ≤ θ ≤ 1`,

```lean
seg f g θ = g + ((θ : ℝ) : ℂ) • (f - g)
```

and

```lean
seg f g 0 = g
seg f g 1 = f
```

If both endpoints have the same floor,

```lean
UniformFloor f δ
UniformFloor g δ
```

then every segment point has the same floor:

```lean
UniformFloor (seg f g θ) δ
```

because evaluation is linear and

```lean
δ ≤ Re(eval f)
δ ≤ Re(eval g)
0 ≤ θ
θ ≤ 1
```

imply

```lean
δ ≤ (1 - θ) * Re(eval g) + θ * Re(eval f).
```

No extra “interaction” condition is needed. A single common floor `δ` is enough.

Now fix `t ≥ 0`. Define

```lean
φ θ := NormedSpace.exp (((-t : ℝ) : ℂ) • seg f g θ)
```

Then

```lean
φ 1 - φ 0 = exp(-t f) - exp(-t g)
```

and Mathlib’s Banach-algebra exponential derivative gives

```lean
φ' θ =
  (((-t : ℝ) : ℂ) • (f - g))
    * NormedSpace.exp (((-t : ℝ) : ℂ) • seg f g θ)
```

up to commuted multiplication. Since the algebra is commutative, the order does not matter.

Assume

```lean
‖GWA.gDeriv f‖ ≤ Md
‖GWA.gDeriv g‖ ≤ Md
```

Then for `θ ∈ [0,1]`,

```lean
‖GWA.gDeriv (seg f g θ)‖ ≤ Md.
```

Therefore the committed decisive estimate gives

```lean
‖NormedSpace.exp (((-t : ℝ) : ℂ) • seg f g θ)‖
  ≤ Cdec * (1 + t * Md)^2 * Real.exp (-δ * t).
```

So the derivative of `φ` satisfies

```lean
‖φ' θ‖
  ≤ t * ‖f - g‖
      * Cdec * (1 + t * Md)^2 * Real.exp (-δ * t).
```

The mean-value inequality on `[0,1]` then gives the key lemma:


# Negative-power Lipschitz

For `s > 0`, `δ > 0`, and common floor/derivative bounds,

```lean
‖FnegEWA f s - FnegEWA g s‖
  ≤ negLipConst s δ Md * ‖f - g‖.
```

Proof:

```lean
FnegEWA f s - FnegEWA g s
=
(1 / Γ s) • ∫ t in Ioi 0,
  ((t ^ (s - 1) : ℝ) : ℂ) •
    (exp(-t f) - exp(-t g)).
```

Then

```lean
‖FnegEWA f s - FnegEWA g s‖
≤
(1 / Γ s) *
  ∫₀∞ t^(s-1) *
    ‖exp(-t f) - exp(-t g)‖ dt
```

and by the exponential-difference lemma,

```lean
≤
(1 / Γ s) *
  ∫₀∞ t^(s-1) *
    (t * ‖f-g‖ * Cdec * (1+t Md)^2 * exp(-δt)) dt
```

so

```lean
≤
negLipConst s δ Md * ‖f - g‖.
```


You already have the integrability template in `integrable_gammaIntegrandEWA`, which uses the decisive bound as the majorant. fileciteturn13file0L106-L122 For this Lipschitz version, add the same proof with the shifted majorant

```lean
def PsiBetaEWA (β : ℝ) (f : EWA T 1) : EWA T 1 :=
  FnegEWA (1 + f) β
```

Then:

```lean
theorem PsiBetaEWA_lipschitz
    {f g : EWA T 1} {β δ Md : ℝ}
    (hβ : 0 < β)
    (hδ : 0 < δ)
    (hMd : 0 ≤ Md)
    (hf_floor : UniformFloor (1 + f) δ)
    (hg_floor : UniformFloor (1 + g) δ)
    (hfD : ‖GWA.gDeriv f‖ ≤ Md)
    (hgD : ‖GWA.gDeriv g‖ ≤ Md) :
    ‖PsiBetaEWA β f - PsiBetaEWA β g‖
      ≤ negLipConst β δ Md * ‖f - g‖ := by
  -- apply `FnegEWA_lipschitz` to `1+f`, `1+g`
  -- use:
  --   (1+f) - (1+g) = f - g
  --   gDeriv (1+f) = gDeriv f
  --   gDeriv (1+g) = gDeriv g

      (by simpa using hfD)
      (by simpa using hgD)
```

# Hidden obstructions

The main obstruction is not analytic. It is **definition management**.

Your current `realPow_eval_EWA` proves an existential witness:

```lean
∃ F : EWA T 1, eval F = ...
```

fileciteturn14file0L37-L47

For Lipschitz, you need a named map. Add:

```lean
realPowLaplaceEWA γ m f := f^m * FnegEWA f ((m:ℝ)-γ)
```

and prove eval for that map. Do not formulate Lipschitz for an existentially chosen `F`.

The second obstruction is weight level. The committed WL file is specifically `EWA T 1`; its `evalSTCLM` and `FnegEWA` live at `EWA T 1`. fileciteturn13file0L33-L45 If your contraction is in `E_T^r` for `r > 1`, this route must be rerun at that `r`, or you need a high-weight theorem reducing `E_T^r` Lipschitz to finitely many derivative-level `E_T^1` estimates. For the theorem as stated in `E_T^1`, route B closes.

The third obstruction is the floor. Route B requires a **common floor along the segment**, but endpoint floors with the same `δ` are enough. For `Ψβ`, the needed assumption is not `UniformFloor f δ`; it is:

```lean
UniformFloor (1 + f) δ
UniformFloor (1 + g) δ
```

Then the segment of `1+f` and `1+g` keeps the same floor.

No hidden blow-up occurs in the Laplace integral. The extra factor from the difference estimate is exactly one factor of `t`, so the majorant is

```lean
t^s * (1 + t Md)^2 * exp(-δt),
```

which is integrable for `s > 0`, `δ > 0`.

# Final answer

Use **route B**.

The contraction-ready estimates should be:

```lean
‖FnegEWA f s - FnegEWA g s‖
  ≤ negLipConst s δ Md * ‖f - g‖
```

and, for `γ < m`, `1 ≤ m`,

```lean
‖realPowLaplaceEWA γ m f - realPowLaplaceEWA γ m g‖
  ≤
((m : ℝ) * R ^ (m - 1) * negNormConst ((m : ℝ) - γ) δ Md
  + R ^ m * negLipConst ((m : ℝ) - γ) δ Md)

```

For the denominator:

```lean
‖PsiBetaEWA β f - PsiBetaEWA β g‖
  ≤ negLipConst β δ Md * ‖f - g‖.
```

The constants do **not** directly depend on `T`; all time dependence is already absorbed into the `E_T` norm and the uniform floor. The small factor for contraction still comes from the divergence-Duhamel `C√T`, not from the nonlinear Wiener–Lévy Lipschitz constants.
