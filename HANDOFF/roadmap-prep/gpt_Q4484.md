
Thus p = 13 is finite engineering plus one serious reusable descent theorem, not a second modularity theorem. If true even-degree descent exactness is already available, the fixed arithmetic is small. If N18 is built only through Route C and never formalizes true genus-two descent, then p = 13 remains a second substantial block.
