Verdict: use the Mathlib lemma MeasureTheory.Measure.eqOn_Icc_of_ae_eq. It is exactly the closed-interval version of “continuous on the interval + a.e. equality on the restricted measure implies pointwise equality on the whole closed interval.” It upgrades through the endpoints; no extra endpoint argument is needed.

The relevant chain in Mathlib is:

```javascript
MeasureTheory.Measure.eqOn_Icc_of_ae_eq
  (hfg : f =ᵐ[μ.restrict (Icc a b)] g)
  (hf : ContinuousOn f (Icc a b))
  (hg : ContinuousOn g (Icc a b)) :
  EqOn f g (Icc a b)
```

Internally this is built from:

```javascript
MeasureTheory.Measure.eqOn_of_ae_eq
MeasureTheory.Measure.eqOn_open_of_ae_eq
```

and, for the endpoint upgrade, eqOn_Icc_of_ae_eq uses closure_interior_Icc hne. So the closed endpoints 0 and 1 are handled by continuity from the interior.

Paste-ready theorem:

```javascript
import Mathlib.MeasureTheory.Measure.OpenPos
import Mathlib.MeasureTheory.Measure.Haar.OfBasis

open Set MeasureTheory

noncomputable section

/-- Continuous on `[0,1]` and zero a.e. on `[0,1]` implies zero everywhere on `[0,1]`. -/
theorem cont_ae_zero_eq_zero {f : ℝ → ℝ}
    (hf : ContinuousOn f (Set.Icc (0 : ℝ) 1))
    (hae : f =ᵐ[volume.restrict (Set.Icc (0 : ℝ) 1)] 0) :
    ∀ x ∈ Set.Icc (0 : ℝ) 1, f x = 0 := by
  intro x hx
  have hae' :
      f =ᵐ[volume.restrict (Set.Icc (0 : ℝ) 1)]
        (fun _ : ℝ => (0 : ℝ)) := by
    simpa using hae
  exact
    (MeasureTheory.Measure.eqOn_Icc_of_ae_eq
      (μ := volume) (a := (0 : ℝ)) (b := 1)
      (by norm_num : (0 : ℝ) ≠ 1)
      hae'
      hf
      (continuous_const.continuousOn)) x hx
```

If volume’s open-positive instance is already imported in the local file, the Haar.OfBasis import can be omitted. The necessary typeclass is:

```javascript
MeasureTheory.Measure.IsOpenPosMeasure (volume : Measure ℝ)
```

This is why eqOn_Icc_of_ae_eq can rule out a nonzero continuous spike: any relatively open interval where f ≠ 0 has positive Lebesgue measure.

Do not use Continuous.ae_eq_iff_eq directly here unless you first convert the problem to a globally continuous function on a subtype or to a full-space a.e. equality. Your hypothesis is ContinuousOn plus volume.restrict (Icc 0 1), so the interval lemma is the clean target.
