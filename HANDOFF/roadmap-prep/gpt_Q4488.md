# Verdict

Yes: for the Paper 3 global-stability branch, a direct sequential LaSalle argument is cleaner to formalize than a general abstract invariance principle. But the correct argument is slightly stronger than “choose one convergent subsequence.” It must treat an arbitrary sequence of times tending to infinity, extract a convergent time-translated orbit, prove that the limiting entire orbit has constant Lyapunov energy, and then use the Lyapunov derivative identity to force zero dissipation. Uniqueness of the admissible zero-dissipation equilibrium then implies that every omega-limit point is the same coexistence state, and a short contradiction proves convergence of the full orbit.

The current repository interface

```javascript
TimeTranslateCompactnessConclusion D p K u v
```

is exactly the right compactness strength: it returns locally convergent translates together with an

```javascript
EntireClassicalSolution D p uInf vInf
```

limit. Mere precompactness of the individual slices u t is not enough, because one needs an entire limiting trajectory on which the PDE Lyapunov identity can be reapplied.

There are two genuine compatibility leaves still needed around the current abstract

```javascript
K.locallyConverges
```

relation:

1. local convergence must imply convergence of the Lyapunov energy on each fixed time slice;

1. local convergence must imply convergence in the norm used in the final stability statement, typically the interval sup norm.

Since CompactnessData.locallyConverges is currently an uninterpreted proposition-valued field, neither implication is definitional at the abstract level. They should be explicit bridge lemmas for the concrete interval compactness package.

# 1. The minimal direct LaSalle argument

Let

```plain text
e(t) = E(u(t), v(t)),
D(t) = dissipation(u(t), v(t)).
```

Assume on the global positive orbit:

```plain text
e is nonincreasing,
e is bounded below,
e'(t) = -D(t)       or, more generally, e'(t) ≤ -κ D(t) with κ>0,
D(t) ≥ 0.
```

The monotone convergence step gives a finite number EInf such that

```javascript
Tendsto (fun t => E (u t) (v t)) atTop (𝓝 EInf).
```

This is only a small order-theoretic producer: define EInf as the infimum of the tail range and use monotonicity plus the lower bound. It is useful to expose this as a standalone lemma instead of rebuilding the infimum argument inside the LaSalle proof.

## 1.1 Start with an arbitrary time sequence

Take any

```javascript
(times : ℕ → ℝ)
(htimes : Tendsto times atTop atTop).
```

Apply TimeTranslateCompactnessConclusion to obtain

```javascript
∃ subseq : ℕ → ℕ, StrictMono subseq ∧
∃ uInf vInf : ℝ → intervalDomain.Point → ℝ,
  K.locallyConverges
    (fun n s x => u (s + times (subseq n)) x) uInf ∧
  K.locallyConverges
    (fun n s x => v (s + times (subseq n)) x) vInf ∧
  EntireClassicalSolution intervalDomain p uInf vInf.
```

The order of s + times (...) versus times (...) + s is immaterial; use whichever matches the committed definition.

## 1.2 The energy of the entire limit is constant

Fix an arbitrary translated time s : ℝ. Since

```plain text
times (subseq n) + s → +∞,
```

the original energy limit gives

```javascript
E (u (times (subseq n) + s)) (v (times (subseq n) + s)) → EInf.
```

The energy-continuity bridge for K.locallyConverges gives the same sequence the limit

```javascript
E (uInf s) (vInf s).
```

Uniqueness of limits in ℝ therefore yields

```javascript
have hEeq : E (uInf s) (vInf s) = EInf := ...
```

for every s. Hence

```javascript
have hEconst :
    (fun s => E (uInf s) (vInf s)) = fun _ => EInf :=
  funext hEeq
```

The important point is that this proves constancy on the entire translated limiting orbit, not merely equality of the energy at one cluster point.

## 1.3 Constant energy forces zero dissipation

Because the translate limit is an EntireClassicalSolution, apply the same PDE Lyapunov derivative theorem to it. There is no need to pass u_t or the dissipation through the compactness limit:

```javascript
have hderiv :
    HasDerivAt
      (fun s => E (uInf s) (vInf s))
      (-(Diss (uInf 0) (vInf 0))) 0 :=
  lyapunov_hasDerivAt_of_entireSolution hentire 0
```

Rewrite the function with hEconst and compare with the derivative of a constant:

```javascript
have hderiv' :
    HasDerivAt (fun _ : ℝ => EInf)
      (-(Diss (uInf 0) (vInf 0))) 0 := by
  simpa [hEconst] using hderiv

have hconst : HasDerivAt (fun _ : ℝ => EInf) 0 0 :=
  hasDerivAt_const 0 EInf

have hnegD : -(Diss (uInf 0) (vInf 0)) = 0 :=
  hderiv'.unique hconst

have hDzero : Diss (uInf 0) (vInf 0) = 0 := by
  linarith
```

This derivative-at-one-time argument is shorter than proving that a nonnegative continuous function with zero interval integral is identically zero.

If the available PDE theorem is only the integrated identity

```plain text
E(Y b) + ∫_a^b Diss(Y s) ds = E(Y a),
```

then constancy of E(Y s) gives zero integral on every interval. Nonnegativity plus continuity of the dissipation gives Diss(Y 0)=0. That is an equally valid fallback, but the HasDerivAt route is the smallest Lean closure once the PDE differentiation lemma exists.

If the energy law has the inequality

```plain text
E'(t) ≤ -κ * Diss(t),    κ > 0,
```

then the limiting energy has derivative 0, so

```plain text
0 ≤ -κ * Diss(uInf 0, vInf 0).
```

Together with Diss ≥ 0, this again yields Diss = 0.

## 1.4 Zero dissipation and uniqueness

The branch-specific rigidity lemma should have one of the following two shapes.

Combined minimal form:

```javascript
zero_dissipation_unique :
  ∀ uInf vInf,
    EntireClassicalSolution intervalDomain p uInf vInf →
    Diss (uInf 0) (vInf 0) = 0 →
    uInf 0 = fun _ => uStar ∧
    vInf 0 = fun _ => vStar
```

Factored form:

```javascript
zero_dissipation_is_equilibrium :
  Diss f g = 0 → AdmissibleLimit p f g → IsEquilibrium p f g

equilibrium_unique :
  AdmissibleLimit p f g → IsEquilibrium p f g →
  f = fun _ => uStar ∧ g = fun _ => vStar
```

The factored form is mathematically more informative, while the combined form gives the smallest closing API.

A warning: “uniqueness of the positive algebraic constant equilibrium” is not by itself sufficient if the zero-dissipation set could contain nonconstant stationary states. The load-bearing branch lemma is really

```plain text
Diss = 0  ⇒  the slice is the target constant equilibrium.
```

Typically the gradient part of the dissipation first forces spatial constancy, and the reaction/moment part then forces the constant to equal the coexistence value. In the mass-constrained minimal branch, uniqueness must additionally use the conserved mass; there is otherwise a family of constant equilibria.

# 2. Why one subsequence is not enough

The argument

```plain text
pick t_n → ∞,
extract u(t_n) → φ,
show φ is an equilibrium
```

only proves that some equilibrium lies in the omega-limit set. It does not prove that all cluster points are equilibria, nor that the full orbit converges.

The correct quantifier is:

```plain text
for every times_n → ∞,
there is a subsequence whose translated limit has slice 0 equal to the target equilibrium.
```

Because times was arbitrary, every omega-limit point equals the same target. No explicit set-valued definition of ω(u) is required.

# 3. From unique cluster point to full convergence

Define the concrete pair sup-distance

```javascript
def intervalPairSupDistance
    (f g : intervalDomain.Point → ℝ)
    (uStar vStar : ℝ) : ℝ :=
  intervalDomain.supNorm (fun x => f x - uStar) +
    intervalDomain.supNorm (fun x => g x - vStar)
```

The final topological step is a reusable scalar lemma.

```javascript
theorem tendsto_zero_of_every_atTop_sequence_has_zero_subsequence
    {d : ℝ → ℝ}
    (hd : ∀ t, 0 ≤ d t)
    (hsub :
      ∀ times : ℕ → ℝ,
        Tendsto times atTop atTop →
          ∃ subseq : ℕ → ℕ, StrictMono subseq ∧
            Tendsto (fun n => d (times (subseq n))) atTop (𝓝 0)) :
    Tendsto d atTop (𝓝 0) := by
  -- Contrapositive.
  -- If d does not tend to 0, choose ε>0 and times n ≥ n with
  -- ε ≤ d (times n).  Then times → +∞.  `hsub` gives a subsequence
  -- along which d → 0, contradicting the inherited lower bound ε.
  ...
```

This lemma is elementary. Its proof uses the metric characterization of Tendsto ... atTop (𝓝 0), classical choice of a bad sequence, and the fact that a strictly monotone subsequence of naturals tends to atTop.

Applying it to

```javascript
d t := intervalPairSupDistance (u t) (v t) uStar vStar
```

gives full uniform convergence. The slice_supNorm_tendsto bridge turns the local convergence of the translated subsequence at time 0 into convergence of d (times (subseq n)) to zero.

The contradiction proof can also be written directly:

```plain text
suppose uniform convergence fails;
choose ε>0 and t_n→∞ with distance(u(t_n),uStar)≥ε;
time-translate compactness gives a subsequence converging to an entire limit;
LaSalle rigidity says its time-zero slice is the target equilibrium;
slice convergence says the distances tend to zero;
contradiction with the persistent ≥ε bound.
```

# 4. Paste-ready minimal data package

The following package matches the current repository abstractions and isolates only the two concrete-convergence bridges that CompactnessData itself cannot provide.

```javascript
namespace ShenWork.Paper3

open Filter Topology
open ShenWork.IntervalDomain

noncomputable section

def intervalPairSupDistance
    (f g : intervalDomain.Point → ℝ)
    (uStar vStar : ℝ) : ℝ :=
  intervalDomain.supNorm (fun x => f x - uStar) +
    intervalDomain.supNorm (fun x => g x - vStar)

structure IntervalDomainSequentialLaSalleData
    (p : CM2Params)
    (K : CompactnessData intervalDomain)
    (u v : ℝ → intervalDomain.Point → ℝ)
    (uStar vStar : ℝ) where

  energy :
    (intervalDomain.Point → ℝ) →
      (intervalDomain.Point → ℝ) → ℝ

  dissipation :
    (intervalDomain.Point → ℝ) →
      (intervalDomain.Point → ℝ) → ℝ

  /-- Produced from Lyapunov monotonicity and a lower bound. -/
  energy_tendsto :
    ∃ EInf : ℝ,
      Tendsto (fun t => energy (u t) (v t)) atTop (𝓝 EInf)

  /-- Continuity of the energy under the concrete time-translate topology. -/
  energy_of_locallyConverges :
    ∀ (U V : ℕ → ℝ → intervalDomain.Point → ℝ)
      (UInf VInf : ℝ → intervalDomain.Point → ℝ)
      (s : ℝ),
      K.locallyConverges U UInf →
      K.locallyConverges V VInf →
      Tendsto (fun n => energy (U n s) (V n s)) atTop
        (𝓝 (energy (UInf s) (VInf s)))

  /-- Reapply the PDE Lyapunov identity to every entire translate limit. -/
  energy_hasDerivAt_entire :
    ∀ UInf VInf,
      EntireClassicalSolution intervalDomain p UInf VInf →
      ∀ s : ℝ,
        HasDerivAt
          (fun r => energy (UInf r) (VInf r))
          (-(dissipation (UInf s) (VInf s))) s

  dissipation_nonneg :
    ∀ f g, 0 ≤ dissipation f g

  /-- This is zero-dissipation rigidity plus uniqueness in the admissible branch. -/
  zero_dissipation_unique :
    ∀ UInf VInf,
      EntireClassicalSolution intervalDomain p UInf VInf →
      dissipation (UInf 0) (VInf 0) = 0 →
      UInf 0 = (fun _ => uStar) ∧
      VInf 0 = (fun _ => vStar)

  /-- The concrete local convergence relation controls the final sup topology. -/
  slice_supNorm_tendsto :
    ∀ (F : ℕ → ℝ → intervalDomain.Point → ℝ)
      (FInf : ℝ → intervalDomain.Point → ℝ)
      (s : ℝ),
      K.locallyConverges F FInf →
      Tendsto
        (fun n =>
          intervalDomain.supNorm
            (fun x => F n s x - FInf s x))
        atTop (𝓝 0)

end

end ShenWork.Paper3
```

For an entropy with a logarithm, energy_of_locallyConverges normally uses the eventual uniform positive floor supplied by persistence. On a compact range [δ,M], the entropy integrand is uniformly continuous, so uniform slice convergence implies convergence of the integral. Without a positive floor, one needs a lower-semicontinuity argument instead; that is not the minimal route for the positive coexistence branch.

# 6. Dependency order for Step 7

The clean build order is:

1. Concrete translate topology. Define the interval K.locallyConverges relation strongly enough to yield fixed-slice uniform convergence.

1. Time-translate compactness. Produce TimeTranslateCompactnessConclusion from the positive-time uniform regularity bounds.

1. PDE Lyapunov identity. Prove the HasDerivAt or integrated energy-dissipation identity for every positive classical solution, hence also for every EntireClassicalSolution obtained as a translate limit.

1. Energy tail limit. Monotonicity plus a lower bound gives energy_tendsto.

1. Topology bridges. Prove energy_of_locallyConverges and slice_supNorm_tendsto for the concrete K.

1. Zero-dissipation rigidity. Show Diss=0 forces the target coexistence state; include mass conservation in the minimal branch.

1. Sequential LaSalle closure. Prove the two lemmas above. This step is then mostly uniqueness-of-limits, HasDerivAt.unique, and a bad-sequence contradiction.

1. Local exponential restart. Once uniform convergence places the orbit inside the Theorem 2.2 stable neighborhood, restart the solution and invoke the sectorial/spectral local exponential estimate to obtain the final C¹ exponential convergence claim.

The spectral gap/principal-mode engine is therefore not needed in the LaSalle closure itself. It re-enters only in the final local exponential upgrade.

# 7. Exact gaps to avoid hiding

The direct route closes only after the following are explicit:

- TimeTranslateCompactnessConclusion, not merely compactness of the set of slices;

- continuity of the Lyapunov energy under the chosen locallyConverges relation;

- a derivative or integrated energy identity applicable to every entire translate limit;

- zero-dissipation rigidity in the correct invariant class;

- uniqueness of the equilibrium in that class;

- fixed-slice convergence in the norm of the desired conclusion.

No full abstract omega-limit set, semiflow invariance theorem, or general LaSalle library is required. After these six facts, the global-stability closing argument is a finite, direct Lean proof.

Finally, this LaSalle package applies to the entropy-driven branches of Theorems 2.3–2.5. Branches whose paper proof uses a rectangle ODE or a different dissipation mechanism still need their own producer for zero_dissipation_unique or for eventual entry into the same equilibrium basin; the topological closing lemma can nevertheless be shared unchanged.
