# 1. Exact current classical comparison API

The public theorem at the end of IntervalBFormLinearDriftComparisonRegularDischarge.lean is literally:

```javascript
theorem neumann_interval_comparison_with_drift
    {T : ℝ} {B C : ℝ → ℝ → ℝ} {u₀ : ℝ → ℝ}
    {u : ℝ → ℝ → ℝ} :
    NeumannLinearDriftComparisonRegular T B C u₀ u
```

It is an alias for the proved maximum-principle implementation. Expanding the definition gives the actual hypotheses and conclusion:

```javascript
def NeumannLinearDriftComparisonRegular
    (T : ℝ) (B C : ℝ → ℝ → ℝ) (u₀ : ℝ → ℝ)
    (u : ℝ → ℝ → ℝ) : Prop :=
  ∀ w : ℝ → ℝ → ℝ,
    0 < T →
    NeumannLinearDriftCoefficientsRegular T B C →
    IsClassicalNeumannLinearDriftSuperSolution T B C u →
    (∀ x ∈ Set.Icc (0 : ℝ) 1, u 0 x = u₀ x) →
    IsClassicalNeumannLinearDriftSubSolution T B C w →
    (∀ x ∈ Set.Icc (0 : ℝ) 1, w 0 x ≤ u₀ x) →
    ∀ t x, 0 < t → t < T → x ∈ Set.Icc (0 : ℝ) 1 →
      w t x ≤ u t x
```

Thus:

- B and C may depend on both time and space;

- the output is for 0 < t < T, not at t = T;

- both endpoints x=0,1 are included in the conclusion;

- the theorem is normalized to unit diffusion.

The residual is

```javascript
neumannLinearDriftResidual B C q t x
  = dt q t x - dxx q t x - B t x * dx q t x - C t x * q t x
```

so the modeled equation is

```plain text
q_t = q_xx + B(t,x) q_x + C(t,x) q.
```

For a PDE with diffusion ν, one must either rescale time and coefficients or build/use a ν-parameterized comparison theorem. The current public theorem does not silently contain ν.

## Coefficient hypotheses

```javascript
def BoundedOnIntervalStrip (T : ℝ) (q : ℝ → ℝ → ℝ) : Prop :=
  ∃ M : ℝ, 0 ≤ M ∧
    ∀ t ∈ Set.Icc (0 : ℝ) T,
      ∀ x ∈ Set.Icc (0 : ℝ) 1,
        |q t x| ≤ M

structure NeumannLinearDriftCoefficientsRegular
    (T : ℝ) (B C : ℝ → ℝ → ℝ) : Prop where
  drift_bounded : BoundedOnIntervalStrip T B
  reaction_bounded : BoundedOnIntervalStrip T C
  reaction_lipschitz :
    ∃ L : ℝ, 0 ≤ L ∧
      ∀ ⦃t x a b : ℝ⦄,
        t ∈ Set.Icc (0 : ℝ) T →
        x ∈ Set.Icc (0 : ℝ) 1 →
        |C t x * a - C t x * b| ≤ L * |a - b|
```

There is no coefficient time derivative and no explicit coefficient continuity field. For the linear map r ↦ C(t,x)r, reaction_lipschitz follows immediately from a uniform bound on |C|, but it is nevertheless an explicit structure field.

## Classical supersolution hypotheses

```javascript
structure IsClassicalNeumannLinearDriftSuperSolution
    (T : ℝ) (B C : ℝ → ℝ → ℝ) (u : ℝ → ℝ → ℝ) : Prop where
  continuousOn_rect :
    ContinuousOn (fun p : ℝ × ℝ => u p.1 p.2)
      (Set.Icc (0 : ℝ) T ×ˢ Set.Icc (0 : ℝ) 1)
  time_hasDerivAt :
    ∀ ⦃t x : ℝ⦄,
      0 < t → t < T → x ∈ Set.Icc (0 : ℝ) 1 →
      HasDerivAt (fun τ : ℝ => u τ x) (dt u t x) t
  space_hasDerivAt :
    ∀ ⦃t x : ℝ⦄,
      0 < t → t < T → x ∈ Set.Icc (0 : ℝ) 1 →
      HasDerivAt (fun y : ℝ => u t y) (dx u t x) x
  space_second_hasDerivAt :
    ∀ ⦃t x : ℝ⦄,
      0 < t → t < T → x ∈ Set.Ioo (0 : ℝ) 1 →
      HasDerivAt (fun y : ℝ => dx u t y) (dxx u t x) x
  pde_ge :
    ∀ ⦃t x : ℝ⦄,
      0 < t → t < T → x ∈ Set.Ioo (0 : ℝ) 1 →
      0 ≤ neumannLinearDriftResidual B C u t x
  neumann :
    ∀ t, 0 < t → t < T →
      dx u t 0 = 0 ∧ dx u t 1 = 0
  bounded : BoundedOnIntervalStrip T u
```

The subsolution has the same continuity, first/second derivative, Neumann, and boundedness fields, with

```javascript
pde_le : neumannLinearDriftResidual B C w t x ≤ 0.
```

This is substantially stronger than u ∈ C_t L²_x ∩ L²_t H¹_x plus a weak PDE identity.

# 2. Why the weak (w-u)₊ proof cannot be passed to this theorem

The discharge proof is a pointwise parabolic maximum argument. It constructs an exponentially weighted and spatially perturbed difference, takes a maximum on the compact closed rectangle, excludes the two spatial endpoints using the Neumann derivatives, and at an interior maximizing point uses:

- a pointwise nonnegative time derivative;

- a pointwise zero first spatial derivative;

- a pointwise nonpositive second spatial derivative;

- the pointwise sub/supersolution residual inequalities.

A weak solution does not supply any of those HasDerivAt/dxx facts at a maximizing point. An integrated Dirichlet-form identity is not coercible into IsClassicalNeumannLinearDriftSuperSolution by definitional unfolding or a small adapter.

The restarted API confirms the same issue. SquareHeatRestartStripData contains

```javascript
super :
  IsClassicalNeumannLinearDriftSuperSolution L
    (restartTimeShift τ B) (restartTimeShift τ C)
    (restartTimeShift τ u)
```

and square_heat_hbarrier_via_t0_restart invokes neumann_interval_comparison_with_drift using that field. The time shift fixes the S_N(0)=0 trace vacuity, but it does not weaken the required regularity of the solution.

IsClassicalNeumannLinearDriftSuperSolution is not intrinsically uninhabited—smooth classical functions can inhabit it. The accurate repository verdict is narrower:

- the square-heat barrier side can be made classical at positive time;

- the intended truncated mild limit does not currently have an acyclic producer for the complete classical-super package at the point where Jensen positivity is needed;

- spatial C² slice regularity alone is not sufficient: the structure also asks for time HasDerivAt, joint closed-rectangle continuity, the pointwise PDE inequality, Neumann traces, and boundedness;

- deriving those through a bootstrap that already uses strict positivity would be circular.

Hence the classical route is valid only after an independent, positivity-free C^{2,1} plus pointwise-PDE producer has been proved. It is not the right target for the present weak-energy build.

# 3. Correct weak matched-divergence route

Work on a restarted strip s ∈ [0,L], with original time τ+s. Let

```javascript
Uτ s x := u (τ + s) x
Wτ s x := restartedSquareHeatBarrier τ M f s x
z  s x := Wτ s x - Uτ s x
```

Keep both equations in the same divergence form:

```plain text
q_t = ν q_xx - χ₀ ∂ₓ(q g) + c q.
```

The weak supersolution form for Uτ is

```plain text
⟨∂ₜUτ, φ⟩
  + ν ∫ Uτ_x φ_x
  - χ₀ ∫ Uτ g φ_x
  - ∫ c Uτ φ ≥ 0
```

for nonnegative tests. The weak subsolution form for Wτ has ≤ 0. Subtracting gives, for z=Wτ-Uτ,

```plain text
⟨∂ₜz, φ⟩
  + ν ∫ z_x φ_x
  - χ₀ ∫ z g φ_x
  - ∫ c z φ ≤ 0.
```

Test with φ=z₊. Stampacchia gives

```plain text
∂ₓz₊ = 1_{z>0} ∂ₓz,
qquad z z₊ = z₊².
```

Therefore, with

```plain text
E(s) = ∫₀¹ z₊(s,x)² dx,
```

one obtains

```plain text
(1/2) E'(s) + ν ∫₀¹ |∂ₓz₊|²
  ≤ χ₀ ∫₀¹ z₊ g ∂ₓz₊ + ∫₀¹ c z₊².
```

If

```plain text
|g(s,x)| ≤ G,
 c(s,x) ≤ C₀,
 ν > 0,
```

then

```plain text
|χ₀ ∫ z₊ g ∂ₓz₊|
  ≤ (ν/2) ∫ |∂ₓz₊|²
    + (χ₀² G² / (2ν)) E(s),
```

and hence

```plain text
(1/2) E'(s)
  ≤ (χ₀² G² / (2ν) + C₀) E(s).
```

Equivalently,

```plain text
E'(s) ≤ K E(s),
K = χ₀² G² / ν + 2 C₀.
```

If one uses |c| ≤ C instead of the sharper upper bound, take C₀=C.

The restart initial comparison is exactly

```javascript
∀ x ∈ Set.Icc (0 : ℝ) 1,
  restartedSquareHeatBarrier τ M f 0 x ≤ restartTimeShift τ u 0 x
```

or, after simplification,

```javascript
∀ x ∈ Set.Icc (0 : ℝ) 1,
  squareHeatBarrier M f τ x ≤ u τ x.
```

Thus E 0 = 0. Gronwall gives E s = 0, so Wτ s ≤ Uτ s almost everywhere. Closed-interval spatial continuity of both slices then upgrades the inequality to every x∈[0,1], including the endpoints.

## Boundary bookkeeping

For the chemotaxis field

```plain text
g = R_x / (1+R)^β,
```

Neumann resolver data gives g(0)=g(1)=0. This is what removes the boundary term when deriving the weak divergence form from a classical identity. Once the weak identity is packaged directly, the comparison core should consume the volume pairing ∫ q g φ_x; it should not re-expand the drift.

The barrier calculus may still use the already available g_x bound to prove that Wτ is a subsolution, but the comparison/Gronwall step itself should not carry an uncancelled g_x Uτ z₊ term.

# 4. Paste-ready architecture for Codex

The clean implementation is two layers.

## Layer A: scalar zero-energy closure

Use the gap with the sign that reuses the existing negative-part vocabulary:

```javascript
def restartComparisonGap
    (τ : ℝ) (u w : ℝ → ℝ → ℝ) : ℝ → ℝ → ℝ :=
  fun s x => u (τ + s) x - w s x

def restartComparisonEnergy
    (τ : ℝ) (u w : ℝ → ℝ → ℝ) : ℝ → ℝ :=
  fun s => ∫ x,
    (negativePart (restartComparisonGap τ u w s x)) ^ 2
      ∂ intervalMeasure 1
```

Since

```plain text
negativePart (Uτ-Wτ) = (Wτ-Uτ)₊,
```

this is exactly the desired comparison energy.

A minimal scalar package can be:

