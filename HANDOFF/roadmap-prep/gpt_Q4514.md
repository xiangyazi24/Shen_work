# 1. Exact public semigroup theorem

The theorem in ShenWork/PDE/IntervalSemigroupConeAtoms.lean is:

```javascript
set_option maxHeartbeats 800000 in
/-- Strict positivity of the propagator: for `t > 0` and a continuous
nonnegative input that is positive somewhere on `[0,1]`, the propagated
value is strictly positive at every point. -/
theorem intervalFullSemigroupOperator_pos
    {t : ℝ} (ht : 0 < t) {f : ℝ → ℝ}
    (hf_cont : ContinuousOn f (Set.Icc (0 : ℝ) 1))
    (hf_nonneg : ∀ y ∈ Set.Icc (0 : ℝ) 1, 0 ≤ f y)
    {y₀ : ℝ} (hy₀ : y₀ ∈ Set.Icc (0 : ℝ) 1)
    (hf_pos : 0 < f y₀)
    (x : ℝ) :
    0 < intervalFullSemigroupOperator t f x
```

The load-bearing fact is the final argument:

```javascript
(x : ℝ)
```

There is no hypothesis

```javascript
x ∈ Set.Ioo 0 1
```

and not even a hypothesis

```javascript
x ∈ Set.Icc 0 1.
```

The conclusion is proved for every real x. Therefore the following are direct specializations:

```javascript
have hS0 : 0 < intervalFullSemigroupOperator t f 0 :=
  ShenWork.IntervalSemigroupConeAtoms.intervalFullSemigroupOperator_pos
    ht hf_cont hf_nonneg hy₀ hf_pos 0

have hS1 : 0 < intervalFullSemigroupOperator t f 1 :=
  ShenWork.IntervalSemigroupConeAtoms.intervalFullSemigroupOperator_pos
    ht hf_cont hf_nonneg hy₀ hf_pos 1
```

No endpoint membership proof is needed.

# 3. Exact square-barrier theorem

The barrier is defined by

```javascript

  Real.exp (-M * t) * (intervalFullSemigroupOperator t f x) ^ 2
```

and its positivity theorem is:

```javascript
theorem squareHeatBarrier_pos
    {M t : ℝ} (ht : 0 < t) {f : ℝ → ℝ}
    (hf_cont : ContinuousOn f (Set.Icc (0 : ℝ) 1))
    (hf_nonneg : ∀ y ∈ Set.Icc (0 : ℝ) 1, 0 ≤ f y)
    (hf_pos_somewhere :
      ∃ y₀ ∈ Set.Icc (0 : ℝ) 1, 0 < f y₀)
```

# 4. Exact seed hypotheses

The source-side package is:

```javascript
structure SquareHeatSeed (u₀ f : ℝ → ℝ) : Prop where
  continuousOn : ContinuousOn f (Set.Icc (0 : ℝ) 1)
  nonneg : ∀ y ∈ Set.Icc (0 : ℝ) 1, 0 ≤ f y
  pos_somewhere : ∃ y₀ ∈ Set.Icc (0 : ℝ) 1, 0 < f y₀
  square_le_initial : ∀ y ∈ Set.Icc (0 : ℝ) 1, f y ^ 2 ≤ u₀ y
```

Thus the positivity theorem does not ask for an explicit condition

```javascript
0 < ∫ y in (0 : ℝ)..1, f y
```

or an explicit mass lower bound. It asks for:

```javascript
continuousOn + nonneg + positive somewhere.
```

For a continuous nonnegative function, this is enough because continuity turns one positive value into positivity on a nondegenerate interval of positive measure.

The following formulation is therefore also sufficient:

```javascript
∃ y₀ ∈ Set.Icc (0 : ℝ) 1, f y₀ ≠ 0
```

provided hf_nonneg is available. One converts it by:

```javascript
rcases hf_nonzero with ⟨y₀, hy₀, hy₀_ne⟩
have hy₀_pos : 0 < f y₀ :=
  lt_of_le_of_ne (hf_nonneg y₀ hy₀) (Ne.symm hy₀_ne)
```

A warning is appropriate: for a merely measurable function, “positive at one point” is not enough, because that point can have measure zero. In that lower-regularity setting one should assume positivity on a positive-measure set or 0 < ∫ f. The repository theorem deliberately avoids this issue by assuming ContinuousOn.

# 6. Why the result includes the endpoints

The repository does not prove positivity by asserting that each term of the cosine series is positive; that would be false because

```plain text
cos(kπx) cos(kπy)
```

changes sign.

Instead it unfolds the reflected lattice-Gaussian representation

```plain text
p_N(t,x,y)
  = Σ_{k∈ℤ}
      [G_t(x-y+2k) + G_t(x+y+2k)].
```

Every summand is nonnegative, and the k = 0 contribution is strictly positive for every real x,y when t > 0. Hence

```plain text
0 < p_N(t,x,y)
```

for all real x,y, including

```plain text
x = 0, x = 1, y = 0, y = 1.
```

The semigroup theorem then chooses a small closed interval around a point y₀ where f y₀ > 0, obtains a positive minimum of the kernel on that interval, and bounds the integral below by a strictly positive constant.

This proof is stronger and cleaner for endpoint use than an interior maximum-principle proof followed by a boundary argument.

# Final conclusion

The endpoint is fully closed:

```plain text
τ > 0
+ f continuous on [0,1]
+ f ≥ 0 on [0,1]
+ f > 0 somewhere
⇒ S_N(τ)f(x) > 0 for every real x
⇒ in particular at x = 0 and x = 1
⇒ exp(-Mτ)(S_N(τ)f(x))² > 0 on the closed interval.
```

Therefore the Jensen comparison lower bound gives

```plain text
u(t,x) ≥ w(t,x) > 0
```

for every x : intervalDomainPoint, including both Neumann boundary points. There is no remaining endpoint-positivity gap.
