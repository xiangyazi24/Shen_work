## Verdict

The comparison must be formulated in the same divergence form for W and u:

```javascript
J_g[z] := ν * ∂ₓ z - χ₀ * g * z
L_div[g,c] z := ∂ₓ (J_g[z]) + c * z
```

The supersolution u should satisfy, weakly,

```plain text
u_t ≥ ∂ₓ(ν u_x - χ₀ g u) + c_* u,
```

and the barrier W should satisfy, weakly,

```plain text
W_t ≤ ∂ₓ(ν W_x - χ₀ g W) + c_* W.
```

Then for z := W - u, subtraction gives

```plain text
z_t ≤ ∂ₓ(ν z_x - χ₀ g z) + c_* z,
```

so testing with z₊ gives only the matched drift term

```plain text
χ₀ ∫ g z₊ ∂ₓz₊,
```

which is Young-absorbable. There is no uncancelled u-flux term.

This is the key distinction from the false driftless route. If W is only a subsolution of

```plain text
W_t ≤ ν W_xx - M W
```

while u solves

```plain text
u_t = ∂ₓ(ν u_x - χ₀ g u) + c_u u,
```

then subtracting leaves the bad term

```plain text
+ χ₀ ∫ u g ∂ₓ z₊,
```

which is not controlled by z₊ and does not close by Gronwall.

## 1. Exact matched divergence operator

Use a fixed coefficient pair (g, c_*) on the comparison window. Here

```plain text
g(t,x) = ∂ₓR[u](t,x) / (1 + R[u](t,x))^β,
```

with g(t,0)=g(t,1)=0 from the Neumann resolver. The zero-order coefficient can be either the exact reaction coefficient

```plain text
c_u(t,x) = a - b * u(t,x)^α,
```

or any lower coefficient c_* satisfying

```plain text
c_*(t,x) ≤ c_u(t,x)
```

on the window. The lower-coefficient form is often cleaner because u ≥ 0 then makes u a supersolution:

```plain text
u_t - ∂ₓ(νu_x - χ₀ g u) - c_* u
  = (c_u - c_*) u ≥ 0.
```

A safe choice is a constant lower bound

```plain text
c_* := a - b * M^α,
```

provided 0 ≤ u ≤ M. If the exact c_u is already available with enough regularity, using c_* = c_u is also fine.

The integrated weak form for a fixed nonnegative test φ is:

```plain text
⟨u(t)-u(s), φ⟩
  + ∫_s^t ∫ (ν u_x - χ₀ g u) φ_x
  - ∫_s^t ∫ c_* u φ
  ≥ 0
```

for the supersolution u, and

```plain text
⟨W(t)-W(s), φ⟩
  + ∫_s^t ∫ (ν W_x - χ₀ g W) φ_x
  - ∫_s^t ∫ c_* W φ
  ≤ 0
```

for the subsolution W.

Subtracting with z = W-u and then using the Stampacchia test φ=z₊ gives

```plain text
(1/2) d/dt ∫ z₊^2
  ≤ -ν ∫ |∂ₓz₊|^2
     + χ₀ ∫ g z₊ ∂ₓz₊
     + ∫ c_* z₊^2.
```

With |g| ≤ A_g and c_* ≤ C_+, Young gives

```plain text
χ₀ ∫ g z₊ ∂ₓz₊
  ≤ |χ₀| A_g ∫ |z₊| |∂ₓz₊|
  ≤ (ν/2) ∫ |∂ₓz₊|^2
     + (χ₀^2 A_g^2 / (2ν)) ∫ z₊^2.
```

Therefore

```plain text
(1/2) d/dt ∫ z₊^2
  ≤ (χ₀^2 A_g^2 / (2ν) + C_+) ∫ z₊^2.
```

Since z₊(s)=0 from the seed comparison, Gronwall gives z₊=0, hence W≤u. If this is first obtained a.e. in space, use the continuous-a.e.-zero upgrade to obtain pointwise comparison on the closed interval.

Boundary requirements:

```plain text
u_x = W_x = 0 at x=0,1,
g(t,0)=g(t,1)=0.
```

Then the boundary flux

```plain text
ν z_x - χ₀ g z
```

vanishes at both endpoints.

## 2. What squareHeatResidualCore_nonpos_of_bounds gives

The existing square-heat residual core is a nondivergence residual calculator:

```javascript
neumannLinearDriftResidual B C w t x
  = w_t - w_xx - B * w_x - C * w
```

and the core estimate proves nonpositivity for

```plain text
w_t ≤ w_xx + B w_x + C w.
```

So it does not by itself produce the matched divergence weak subsolution. It can still be used, but only after the following identification.

For the matched divergence operator

```plain text
∂ₓ(ν W_x - χ₀ g W) + c_* W
```

expanded in nondivergence form:

```plain text
ν W_xx - χ₀ g W_x + (c_* - χ₀ g_x) W.
```

Thus in the repo’s current ν = 1 residual convention, instantiate

```javascript
B t x := -χ₀ * g t x
C t x := cStar t x - χ₀ * gx t x
```

where gx is the spatial derivative of g. Then

```plain text
w_t - w_xx - B w_x - C w ≤ 0
```

is exactly

```plain text
w_t ≤ ∂ₓ(w_x - χ₀ g w) + c_* w.
```

For a general diffusion coefficient ν, use the generalized core

```plain text
-M h^2 - 2ν hx^2 - 2b h hx - c h^2 ≤ 0
```

with the condition

```plain text
M ≥ A^2 / (2ν) + D.
```

The existing lemma is the ν=1 instance.

So the correct answer is:

```plain text
squareHeatResidualCore_nonpos_of_bounds is usable,
but only as the nondivergence algebraic residual that verifies
W_t ≤ ∂ₓ(νW_x - χ₀gW) + c_*W after setting
B=-χ₀g and C=c_*-χ₀g_x.
```

It is not the final weak matched-divergence comparison theorem, and using the older NeumannLinearDriftComparison directly would re-enter the classical/nondivergence route rather than the weak (w-u)₊ route.

## 4. What Codex should not do

Do not prove only:

```plain text
W_t ≤ νW_xx - M W.
```

That is a driftless subsolution and is unmatched.

Do not subtract a driftless W equation from the chemotaxis u equation. That leaves:

```plain text
χ₀ ∫ u g ∂ₓz₊,
```

which is not controlled by z₊.

Do not expand u into nondivergence form as the primary comparison identity. The final comparison should consume the weak divergence identity for u. Expanding the barrier W to verify its subsolution residual is acceptable, because that expansion is only a proof of the same divergence inequality for W.

## Bottom line

The correct target is:

```plain text
W is a weak subsolution of the same divergence operator
  ∂ₓ(ν∂ₓ· − χ₀ g ·) + c_*·
for which u is a weak supersolution.
```

squareHeatResidualCore_nonpos_of_bounds is useful, but it must be wrapped by a conversion lemma with

```plain text
B = -χ₀ g,
C = c_* - χ₀ g_x.
```

Once that conversion exists, the (W-u)₊ energy comparison closes by Young + Gronwall and produces the Jensen strict-positivity barrier without the divergence-form mismatch.
