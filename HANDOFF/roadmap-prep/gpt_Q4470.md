# Executive verdict

Yes, X_1(13) is the genus-two prime-torsion seam, but it is not a literal arithmetic clone of N18.

The precise structure is:

```plain text
X_1(13) has genus 2 and is hyperelliptic;
J_1(13) is Q-simple of GL2 type;
J_1(13) is attached to the newform orbit 13.2.e.a;
the coefficient field is Q(sqrt(-3));
the nebentypus has order 6;
rank J_1(13)(Q)=0;
J_1(13)(Q) ~= Z/19Z;
X_1(13)(Q) consists of the six Q-rational cusps.
```

Over

```plain text
K=Q(zeta_13)^+,
[K:Q]=6,
```

the curve becomes bielliptic and

```plain text
J_1(13)_K ~ E_+ x E_-
```

for two explicit elliptic quotients over K. This geometric split / push-pull layer ports directly from N18.

However, the N18 arithmetic miracle does not repeat. For N18, the cubic elliptic factor had a rational 3-isogeny and a tiny 3-Selmer computation. For X_1(13), the elliptic quotients have K-rational 19-torsion and no comparably small known 2- or 3-isogeny descent. The published proof that the factors have rank zero uses modularity plus results of Yuan/Gross, not an elementary finite local descent.

Therefore the shortest axiom-free Lean route is:

```plain text
fixed genus-2 weak 2-descent on J_1(13)(Q)
  + the K-splitting only as a separatedness target
  + the already-built elliptic 2-adic formal-kernel machinery
  + the N18 Mumford table / reduced-divisor uniqueness pattern.
```

That route reuses most of Route C, but it has one genuinely new arithmetic block: a fixed 2-descent certificate for the X_1(13) Jacobian.

# 1. Exact modular and Jacobian data

## 1.1 Curve models

A convenient hyperelliptic model is

```plain text
C13 : y^2 = f13(x),

f13(x)=x^6+4*x^5+6*x^4+2*x^3+x^2+2*x+1.
```

An optimized generalized model, particularly convenient for the Tate-normal-form bridge, is

```plain text
C13opt :
y^2+(x^3+x^2+1)*y=x^2+x.
```

Completing the square and making an elementary coordinate change identifies these two models.

The six rational points are

```plain text
infinityPlus,
infinityMinus,
(0,1),
(0,-1),
(-1,1),
(-1,-1)
```

on the sextic model. On C13opt the four affine points are

```plain text
(0,0),
(0,-1),
(-1,0),
(-1,-1),
```

plus the two points at infinity. All six are cusps.

## 1.2 Newform structure

The relevant modular form is exactly

```plain text
13.2.e.a.
```

Its LMFDB data are:

```plain text
level             13,
weight             2,
dimension          2,
character orbit    13.e,
character order    6,
coefficient field  Q(sqrt(-3)),
analytic rank      0,
inner twists       2.
```

Thus J_1(13) is a modular abelian surface of GL2 type with multiplication by the quadratic field Q(sqrt(-3)). It is not a product of elliptic curves over Q, and it is not a CM product. The LMFDB Sato-Tate label is of the non-CM SU(2)[C6] type.

The associated genus-two isogeny class is 169.a.

## 1.3 Algebraic rank and rational torsion

The algebraic statement is

```plain text
rank J_1(13)(Q)=0,
J_1(13)(Q) ~= Z/19Z.
```

The rational cuspidal divisor classes generate the order-19 subgroup. The rank-zero statement follows unconditionally from the split over K together with the rank-zero theorem for the two elliptic factors. It is not merely an inference from the LMFDB analytic-rank field.

# 2. The splitting field and elliptic factors

## 2.1 Field of definition

Use

```plain text
K=Q(a),

a^6-a^5-5*a^4+4*a^3+6*a^2-3*a-1=0.
```

This is the maximal totally real subfield Q(zeta_13)^+. The six elliptic involutions form one Galois orbit, so an individual elliptic quotient is not defined over Q; the natural splitting field for the quotient construction is this degree-six field.

## 2.2 Explicit bielliptic model

Define

```plain text
b0=-a^5+5*a^3-6*a,
c0mob=b0-13.
```

One elliptic involution acts on the original x-coordinate by

```plain text
x |-> (x+b0)/(c0mob*x-1).
```

For the even-sextic model, put

```plain text
d1=-a^5+2*a^4+3*a^3-6*a^2+1,
d2= a^5-2*a^4-5*a^3+8*a^2+6*a-5,

X=(x+d1)/(x+d2),
Y=y/(x+d2)^3.
```

Then

```plain text
Y^2=c6*X^6+c4*X^4+c2*X^2+c0
```

with

```plain text
c0=( -52*a^5+48*a^4+240*a^3-193*a^2-218*a+175)/208,

c2=( -18*a^5+99*a^3+38*a^2-77*a-25)/208,

c4=( 40*a^5-238*a^3-9*a^2+296*a+69)/208,

c6=( 30*a^5-48*a^4-101*a^3+164*a^2-a-11)/208.
```

## 2.3 Quotient maps

The two degree-two quotient maps are now completely transparent.

For the first involution:

```plain text
q_+(X,Y)=(U,V)=(X^2,Y),

E_+ : V^2=c6*U^3+c4*U^2+c2*U+c0.
```

For the product of this involution with the hyperelliptic involution:

```plain text
q_-(X,Y)=(U,V)=(1/X^2,Y/X^3),

E_- : V^2=c0*U^3+c2*U^2+c4*U+c6.
```

Equivalent monic models are

```plain text
E_+^m :
y^2=x^3+c4*x^2+(c2*c6)*x+c0*c6^2,

E_-^m :
y^2=x^3+c2*x^2+(c4*c0)*x+c6*c0^2.
```

One standard monic twist of E_+ used in the literature is

```plain text
y^2=x^3+B*x^2+C*x+D
```

where

```plain text
B=(40*a^5-238*a^3-9*a^2+296*a+69)/208,

C=(-a^5+11*a^4+38*a^3-19*a^2-73*a-16)/3328,

D=(180*a^5-4*a^4-939*a^3-30*a^2+1092*a+252)/692224.
```

The two quotient curves have

```plain text
rank E_+(K)=rank E_-(K)=0,
E_+(K) ~= Z/19Z,
E_-(K) ~= Z/19Z.
```

Consequently

```plain text
J_1(13)(K) ~= (Z/19Z)^2.
```

# 3. What does and does not port from N18 Route C

## 3.1 Geometry: direct port

The following N18 infrastructure ports almost verbatim:

```plain text
- explicit involution on a genus-two curve;
- invariant and anti-invariant quotient maps;
- divisor pushforward and pullback;
- q^* q_* = 1+rho_*;
- complementary involutions;
- Phi : Pic0(C) -> E_+ x E_-;
- Psi : E_+ x E_- -> Pic0(C);
- Psi o Phi = [2].
```

The same abstract lemma should be reused:

```javascript
theorem complementary_double_cover_pushPull
    (sigma tau : CurveAut C)
    (hhyp : tau = hyperelliptic.comp sigma)
    ... :
    Psi.comp Phi = nsmulHom 2
```

Only the fixed rational-function identities and the degree-six coefficient field are new.

## 3.2 Arithmetic: not a direct port

For N18, the factor E0/L had a rational 3-torsion point and a rational 3-isogeny. The rank-zero certificate was a four-dimensional cube-class computation.

For X_1(13):

```plain text
- the natural visible torsion is order 19;
- the natural isogeny is degree 19;
- the base field has degree 6 and unit rank 5;
- the dual 19-isogeny local conditions are much larger;
- no published small 2- or 3-isogeny descent replaces this calculation.
```

The published rank proof uses modularity and theorems of Yuan/Gross. Formalizing that route in Lean would be much larger than the rest of Mazur's torsion project.

So the slogan is:

```plain text
Route-C geometry: yes.
Route-C N18 elliptic 3-descent: no.
```

# 4. The shortest axiom-free Lean route

The recommended proof is a hybrid.

## Block A: fixed genus-two weak 2-descent over Q

Let

```plain text
G=Pic0(C13)(Q),
H=<delta13>,
order(delta13)=19.
```

Prove the fixed certificate

```javascript
theorem J13_weak_two_descent :
    forall D : J13Q,
      exists h : H, exists Q : J13Q,
        D = (h : J13Q) + 2 • Q
```

using the same generalized Mumford and square-class infrastructure developed for N18. The descent algebra is the fixed sextic étale algebra

```plain text
A13=Q[T]/(f13(T)).
```

This is the single genuinely new arithmetic certificate. It must list the global square classes and their local images; it should not be replaced by a generic number-field algorithm.

The proof only needs weak descent, not the full statement ker(delta)=2J as an abstract library theorem.

## Block B: push infinite 2-divisibility to the elliptic factors

Because 19 kills H, the group-theory lemma from Q4293 gives:

```javascript
lemma nineteen_smul_infinitelyTwoDivisible
    (D : J13Q) :
    InfinitelyDivisible 2 (19 • D)
```

after iterating J13_weak_two_descent.

Map this point through

```plain text
Phi : J13Q ->+ E_+(K) x E_-(K).
```

Both components are infinitely 2-divisible.

## Block C: elliptic formal-kernel separatedness only

The quotient elliptic curves have good reduction above 2; in fact the modular surface acquires everywhere good reduction over K.

For each factor choose

```plain text
red_i : E_i(K) ->+ Etilde_i(F_64)
```

and let r_i be an explicitly computed exponent of the finite reduction group. Put

```plain text
R=lcm(r_+,r_-).
```

Then

```plain text
R • Phi(19 • D)
```

lies in the two formal kernels and is still infinitely 2-divisible. Reuse the N15/N21/N18 local theorem:

```javascript
theorem ellipticFormalKernel_no_infinitelyTwoDivisible :
    forall P : red.ker,
      InfinitelyDivisibleIn 2 red.ker P -> P=0
```

and conclude

```plain text
R • Phi(19 • D)=0.
```

Applying the complementary push-pull relation gives

```plain text
2*R*19 • D=0.
```

Thus J13Q has an explicit finite exponent, without Mordell-Weil finite generation and without proving the elliptic factor ranks.

The clean reusable abstract theorem is:

```javascript
theorem boundedExponent_of_weakDescent_split
    {G A1 A2 : Type*}
    [AddCommGroup G]
    [AddCommGroup A1] [AddCommGroup A2]
    (H : AddSubgroup G)
    (Phi : G ->+ A1 × A2)
    (Psi : A1 × A2 ->+ G)
    (hPsiPhi : forall x, Psi (Phi x)=2 • x)
    (hweak : forall x,
      exists h : H, exists y,
        x=(h:G)+2 • y)
    (hH19 : forall h:H, 19 • (h:G)=0)
    (R : Nat)
    (hTargetSep : forall a,
      InfinitelyDivisible 2 a -> R • a=0) :
    forall x:G, (38*R) • x=0 := by
  intro x
  have hxdiv : InfinitelyDivisible 2 (19 • x) :=
    infinitelyDivisible_of_weakDescent hweak hH19 x
  have hphi : R • Phi (19 • x)=0 :=
    hTargetSep _ (hxdiv.map Phi)
  have := congrArg Psi hphi
  simpa [map_nsmul,hPsiPhi,mul_nsmul] using this
```

## Block D: pin the exact rational Jacobian

The local Euler factors give especially clean finite-field groups:

```plain text
#J13(F_2)=19,
#J13(F_3)=19.
```

For reference, the rational local polynomials are

```plain text
P_2(T)=1+3T+5T^2+6T^3+4T^4,
P_3(T)=1+2T+T^2+6T^3+9T^4.
```

Evaluating at T=1 gives 19 in both cases.

Once every rational Jacobian point is torsion by the explicit annihilator:

- reduction at 2 injects every odd-primary part into a group of order 19;

- reduction at 3 eliminates the possible 2-primary part;

- the explicit rational cuspidal class has exact order 19.

Therefore

```plain text
J_1(13)(Q)=<delta13> ~= Z/19Z.
```

## Block E: enumerate the six rational points

Use the N18 reduced-Mumford pattern:

```javascript
theorem C13_rational_points
    (P : C13PointQ) :
    P=infinityPlus ∨
    P=infinityMinus ∨
    P=(0,1) ∨ P=(0,-1) ∨
    P=(-1,1) ∨ P=(-1,-1)
```

The proof is:

```plain text
Abel-Jacobi injects C13(Q) into J13(Q);
every image is k*delta13 for k in Fin 19;
compare the canonical reduced Mumford triple of a point
with the fixed 19-entry torsion table;
only six table entries have degree <=1;
finish by fin_cases k.
```

This is constructive and requires no Chabauty or Faltings.

# 5. Direct Tate-normal-form endgame

The order-13 modular bridge is smaller than the N18 one.

For the Tate normal form

```plain text
E(b,c): y^2+(1-c)xy-by=x^3-bx^2,
P=(0,0),
```

introduce the standard raw variables r,s. The raw equation for exact order 13 is

```plain text
F13(r,s)=
 r^3-r^2*s^4+5*r^2*s^3-9*r^2*s^2+4*r^2*s-2*r^2
 -r*s^3+6*r*s^2-3*r*s+r-s^3=0.
```

Sutherland's optimized model is C13opt, and the optimized-to-raw map is

```plain text
r=1-x*y,
s=1-x*y/(y+1).
```

On the nonexceptional chart this has the elementary inverse

```plain text
x=(1-r)*(1-s)/(s-r),
y=(s-r)/(1-s).
```

Thus the final bridge can be proved entirely by field_simp and ring_nf:

```javascript
theorem C13opt_point_of_exactOrder13
    (E : WeierstrassCurve Q) [E.IsElliptic]
    (P : (E/Q).Point)
    (hP : addOrderOf P=13) :
    exists x y : Q,
      C13optEq x y ∧ OptNonCusp13 x y
```

Together with the six-point theorem, direct evaluation shows every rational point lies outside OptNonCusp13, proving

```javascript
theorem no_rational_point_of_order_13
    (E : WeierstrassCurve Q) [E.IsElliptic] :
    ¬ HasRationalPointOfOrder E 13
```

without formalizing the abstract moduli scheme X_1(13).

# 6. Alternative routes and why they are not shorter

## Published split-rank route

Kamienny-Newman prove that J_1(13) splits over K and that both elliptic factors have rank zero. Their rank proof uses modularity and results of Yuan/Gross. This is mathematically elegant but not a small Lean dependency.

## Mazur-Tate 1973

The original exclusion of rational order 13 used descent in flat cohomology. It avoids explicit equations, but formalizing finite flat group schemes and the required cohomology would be much larger than the fixed computational route.

## A rational elliptic quotient

There is none. J_1(13) is Q-simple, and the elliptic involutions are only defined over K. So there is no N15/N21-style rank-zero elliptic quotient over Q.

## Full elliptic descent on the K-factors

A 19-isogeny descent over a degree-six field is possible in principle, but it is not a small analogue of the N18 3-isogeny computation. A full 2-descent on the elliptic factors uses a cubic algebra over K, hence an absolute degree-18 algebra. Both are less attractive than the hybrid route above.

# 7. Does Layer 2 collapse to p=11 plus p=13?

Conditionally, yes.

If the existing prime layer already excludes all primes p>=17, and the composite seams 15,16,18,21,25,35,49,... are handled separately, then the only exceptional small prime curves left are:

```plain text
p=11 : X_1(11), genus 1, easy rank-zero elliptic curve;
p=13 : X_1(13), genus 2, the fixed route described here.
```

But the statement is not true from the inequality |T|<=16 alone: prime torsion of order 17,19,... must be excluded somewhere. So the collapse is valid only relative to the already-existing general p>=17 layer.

The practical verdict is:

```plain text
p=11: small.
p=13: medium, not another mountain.
```

Assuming the N18 generalized-Mumford, push-pull, elliptic formal-kernel, finite-reduction, and Tate-normal-form layers are reusable, the genuinely new work is concentrated in one fixed X_1(13) weak 2-descent certificate and a 19-entry Mumford table.

A realistic incremental size is roughly

```plain text
K-splitting fixed identities:         500-900 lines
J13 weak 2-descent certificate:      1200-2500 lines
finite reductions and annihilator:    300-600 lines
19-entry table and six-point proof:    400-800 lines
Tate order-13 bridge:                 250-500 lines
-----------------------------------------------
total new code:                      2650-5300 lines
```

This is materially smaller than rebuilding N18 from scratch, but it is not merely replacing constants in the N18 3-isogeny file.
