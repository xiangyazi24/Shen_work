# Independent verification of `C(ℚ) = cusps` for `X₁(18)`

Let

```text
C : y² = f(x),
f(x) = x⁶ + 4x⁵ + 10x⁴ + 10x³ + 5x² + 2x + 1.
```

The claimed rational points are

```text
(-1,  1), (-1, -1),
( 0,  1), ( 0, -1),
∞₊, ∞₋.
```

Thus there are **four affine points plus two points at infinity**, hence six rational points in total. The wording “six affine plus infinity” would be incorrect.

The conclusion is independently verified by a one-prime Mordell–Weil sieve at `p = 5`. In fact, `p = 5` is perfect:

```text
#C(F₅) = 6,
```

and the six points of `C(F₅)` are exactly the reductions of the six rational cusps. Since `[42]J(ℚ)=0` and `5 ∤ 42`, good-reduction injectivity on prime-to-`5` torsion makes

```text
C(ℚ) → C(F₅)
```

injective after composing with the Abel–Jacobi embedding. Therefore the six known points are all of `C(ℚ)`.

This is independent of the earlier `42`-fiber enumeration: it uses only the already-proved annihilator, one finite-field point count, and the standard prime-to-residue-characteristic injectivity of reduction on Jacobian torsion.

# 1. Finite-field point counts

For an odd prime `p` of good reduction, because `f` is monic of even degree `6`, the smooth projective model has two `F_p`-rational points at infinity. Hence

```text
#C(F_p) = p + 2 + Σ_{x∈F_p} χ_p(f(x)),
```

where `χ_p(0)=0`, `χ_p(square)=1`, and `χ_p(nonsquare)=-1`.

A direct `gcd(f mod p, f' mod p)=1` calculation verifies good reduction at `p=5,7,11,13`.

## 1.1 Exact table

| `p` | Values `f(0),…,f(p-1)` modulo `p` | `Σχ(f(x))` | `#C(F_p)` |
|---:|---|---:|---:|
| 5 | `1,3,2,3,1` | `-1` | **6** |
| 7 | `1,5,2,5,1,5,1` | `1` | **10** |
| 11 | `1,0,6,6,8,0,2,2,7,0,1` | `-4` | **9** |
| 13 | `1,7,2,12,2,11,6,1,12,4,12,7,1` | `1` | **16** |

The quadratic-residue sets used are

```text
F₅ squares:  {0,1,4}
F₇ squares:  {0,1,2,4}
F₁₁ squares: {0,1,3,4,5,9}
F₁₃ squares: {0,1,3,4,9,10,12}.
```

## 1.2 The decisive `p=5` computation

As functions on `F₅`, Fermat gives `x⁵=x`, so

```text
f(x)
 = x⁶ + 4x⁵ + 2x + 1
 = x² - x + 2x + 1
 = x² + x + 1.
```

Thus

```text
x = 0,1,2,3,4
f = 1,3,2,3,1.
```

Only `x=0` and `x=4=-1` give squares, and each gives the two values `y=±1`. Together with the two points at infinity,

```text
C(F₅) = {
  (0,1), (0,-1),
  (4,1), (4,-1),
  ∞₊, ∞₋
}.
```

These are exactly the reductions of

```text
(0,±1), (-1,±1), ∞₊, ∞₋ ∈ C(ℚ).
```

Therefore the known rational points already surject onto `C(F₅)`.

# 2. Why reduction is injective on `C(ℚ)`

Raw reduction of rational points need not be injective in general. The missing input is supplied by the Jacobian annihilator.

Fix one rational point at infinity, say `∞₋`, and use the Abel–Jacobi map

```text
ι : C(ℚ) → J(ℚ),
ι(P) = [P - ∞₋].
```

For a genus-two curve, `ι` is injective. Indeed, if

```text
[P - Q] = 0,
```

then `P-Q` is principal. A nonconstant function with divisor `P-Q` would define a degree-one map `C→P¹`, forcing `C` to have genus zero. Hence `P=Q`. In the project this is the existing concrete theorem `abelJacobi_injective`, proved through reduced Mumford representatives.

Let

```text
red_C : C(ℚ) → C(F₅),
red_J : J(ℚ) → J(F₅).
```

These maps are compatible:

```text
red_J([P-Q]) = [red_C(P)-red_C(Q)].
```

Now suppose

```text
red_C(P) = red_C(Q).
```

Set

```text
D := [P-Q] ∈ J(ℚ).
```

Then

```text
red_J(D)=0.
```

By the Block-5/6 arithmetic conclusion,

```text
42 • D = 0.
```

For an abelian variety with good reduction at residue characteristic `p`, the kernel of reduction has no torsion of order prime to `p`; equivalently, every finite-order element in the kernel has `p`-power order. Applied at `p=5`, the order of `D` is both

```text
a divisor of 42
```

and

```text
a power of 5.
```

Since `gcd(42,5)=1`, the order is `1`, so `D=0`. Abel–Jacobi injectivity then gives `P=Q`.

Therefore

```text
red_C : C(ℚ) → C(F₅)
```

is injective.

Since the six known rational cusps already map bijectively onto all six elements of `C(F₅)`, one obtains

```text
C(ℚ) = {(-1,±1), (0,±1), ∞₊, ∞₋}.
```

## Important correction concerning `#J(F_p)`

No computation of `#J(F₅)` is needed.

The useful criterion is

```text
p ∤ 42,
```

combined with prime-to-`p` torsion injectivity in the reduction kernel. It is **not** necessary to prove that `gcd(#J(F_p),42)=1`.

Indeed, if rational Jacobian torsion is nontrivial and reduction is injective, its image must occur inside `J(F_p)`, so one should not expect `#J(F_p)` to be coprime to all of its orders.

This makes the one-prime proof substantially smaller than a conventional Mordell–Weil sieve that explicitly computes `J(F_p)` and subgroup intersections.

# 3. What the other primes say

The other counts are useful consistency checks but are not as sharp.

## `p=7`

```text
#C(F₇)=10.
```

There are four extra residue points beyond the six cusp reductions. Moreover `7 | 42`, so the simple prime-to-residue-characteristic injectivity argument does not automatically control the possible `7`-primary part of the reduction kernel.

Thus `p=7` is not the preferred sieve prime.

## `p=11`

```text
#C(F₁₁)=9.
```

Here `11 ∤ 42`, so the same argument proves that reduction is injective on `C(ℚ)`. Consequently

```text
#C(ℚ) ≤ 9.
```

But the three extra points in `C(F₁₁)`—the points with `y=0` at `x=1,5,9`—must still be eliminated by a subgroup-intersection sieve or another prime.

## `p=13`

```text
#C(F₁₃)=16.
```

Again `13 ∤ 42`, so reduction is injective, but ten extra residue points remain. This gives only

```text
#C(ℚ) ≤ 16.
```

Therefore the four-prime data are fully consistent with six rational points, but the exact proof is already obtained from `p=5` alone.

# 4. Cusps of `X₁(18)`

Direct substitution gives

```text
f(0)=1,
f(-1)=1.
```

Hence the four finite rational points are

```text
(0,±1), (-1,±1).
```

Because the polynomial is monic of degree `6`, the smooth projective normalization has two rational points at infinity, distinguished by

```text
y/x³ → +1
```

and

```text
y/x³ → -1.
```

Thus the six rational points are

```text
(-1,±1), (0,±1), ∞₊, ∞₋.
```

Under the explicit Tate-normal-form/birational bridge for `X₁(18)`, all six are cusps: at each point one of the forbidden Tate denominators vanishes or the discriminant of the Tate curve vanishes.

There are **six `ℚ`-rational cusps** on this model. The full geometric cusp set over `ℚ̄` is larger: the standard cusp-count formula for `Γ₁(18)` gives

```text
(1/2) Σ_{d|18} φ(d)φ(18/d) = 16
```

geometric cusps. Only the six displayed cusps are rational over `ℚ`.

# 5. Independent Chabauty–Coleman certificate

The elementary Coleman bound alone is not exact. Since `g=2`, `p=5>2g`, the coarse bound gives

```text
#C(ℚ) ≤ #C(F₅) + 2g - 2 = 6 + 2 = 8.
```

So merely quoting the basic bound does **not** prove that there are exactly six points.

A refined disk-by-disk Chabauty argument does give the exact answer.

Because `rank J(ℚ)=0`, every regular differential annihilates the closure of `J(ℚ)` under the `5`-adic logarithm. Choose

```text
ω = (1 + 2x) dx / y.
```

This is a regular differential on the genus-two curve.

## 5.1 Nonvanishing at all points of `C(F₅)`

At the four finite points, `x` is either `0` or `4=-1`:

```text
1 + 2·0 = 1 ≠ 0 mod 5,
1 + 2·4 = 9 = 4 ≠ 0 mod 5.
```

Also `y=±1`, so `dx/y` is a nonzero local differential there.

At infinity, use `t=1/x`. Since

```text
y = ±t⁻³(1+O(t)),
dx = -t⁻² dt,
```

one has

```text
dx/y = ∓t(1+O(t)) dt.
```

Therefore

```text
(1+2x)dx/y
 = (1+2/t) · ∓t(1+O(t))dt
 = ∓(2+t+O(t²))dt,
```

whose value at `t=0` is `∓2`, nonzero modulo `5`.

Thus the reduction of `ω` vanishes at none of the six points of `C(F₅)`.

## 5.2 One zero per residue disk

For a fixed rational base point `P₀`, the Coleman function

```text
F(P) = ∫_{P₀}^P ω
```

vanishes on every rational point because the corresponding Jacobian point is torsion and its `5`-adic logarithm is zero.

On each residue disk, the derivative of `F` is `ω`. Since the reduction of `ω` is nonzero, Strassmann/Coleman gives at most one zero of `F` in that disk.

There are exactly six residue disks because `#C(F₅)=6`, and each disk already contains one of the six rational cusps. Hence each disk contains exactly that one rational point and no other:

```text
#C(ℚ)=6.
```

This supplies a second independent mathematical certificate, separate both from the `42`-fiber enumeration and from the simple injective-reduction proof.

For Lean, however, the one-prime reduction proof is much smaller because Mathlib currently has no Coleman-integration infrastructure.

## 6.2 Generic injectivity lemma

```lean
/-- If J(Q) is killed by N and p is coprime to N, good reduction makes
    curve reduction injective through the Abel–Jacobi embedding. -/
theorem curveReduction_injective_of_jacobian_annihilator
    {Cq Cp Jq Jp : Type*}
    [AddCommGroup Jq] [AddCommGroup Jp]
    (N p : ℕ)
    (abelJacobiQ : Cq → Jq)
    (abelJacobiP : Cp → Jp)
    (curveRed : Cq → Cp)
    (jacRed : Jq →+ Jp)
    (hAJinj : Function.Injective abelJacobiQ)
    (hcompat : ∀ P, jacRed (abelJacobiQ P) = abelJacobiP (curveRed P))
    (hann : ∀ D : Jq, N • D = 0)
    (hcoprime : Nat.Coprime N p)
    (hkernel : ∀ D : Jq,
      jacRed D = 0 → IsOfFinAddOrder D →
      ∃ k : ℕ, addOrderOf D = p ^ k) :
    Function.Injective curveRed := by
  intro P Q hPQ
  apply hAJinj
  let D := abelJacobiQ P - abelJacobiQ Q
  have hredD : jacRed D = 0 := by
    simp [D, hcompat, hPQ]
  have hfin : IsOfFinAddOrder D :=
    isOfFinAddOrder_iff_nsmul_eq_zero.mpr
      ⟨N, by
        -- `N` is positive in the concrete application (`N=42`).
        omega, hann D⟩
  obtain ⟨k, hk⟩ := hkernel D hredD hfin
  have hord_dvd_N : addOrderOf D ∣ N :=
    addOrderOf_dvd_of_nsmul_eq_zero (hann D)
  have hpPow_dvd_N : p ^ k ∣ N := hk ▸ hord_dvd_N
  have hk0 : k = 0 := by
    -- A positive power of `p` cannot divide `N` when `p` and `N` are coprime.
    exact Nat.Coprime.pow_dvd_right_iff hcoprime.symm |>.mp hpPow_dvd_N
  have hord : addOrderOf D = 1 := by simpa [hk, hk0]
  have hD : D = 0 := addOrderOf_eq_one_iff.mp hord
  simpa [D, sub_eq_zero] using hD
```

The exact final two arithmetic lemmas may have slightly different Mathlib names; the mathematical interface is the important part. A more stable project implementation should package the local fact directly as

```lean
jacRed_injective_on_torsionBy_of_coprime
```

and avoid manipulating `addOrderOf` in the capstone.

# 7. Final assessment

The independent cross-check succeeds.

```text
#C(F₅)  = 6
#C(F₇)  = 10
#C(F₁₁) = 9
#C(F₁₃) = 16
```

The six known rational points have distinct reductions at all four primes. More importantly:

```text
#C(F₅)=6
+ every F₅-point is a cusp reduction
+ [42]J(ℚ)=0
+ 5∤42
+ prime-to-5 torsion injectivity under good reduction
+ Abel–Jacobi injectivity
```

implies directly that

```text
C(ℚ) = {(-1,±1), (0,±1), ∞₊, ∞₋}.
```

A refined Chabauty–Coleman check with

```text
ω=(1+2x)dx/y
```

gives the same result because `ω` is nonzero in every one of the six `F₅` residue disks.

Therefore the Block-7 rational-point conclusion is independently validated, and the one-prime reduction proof is substantially smaller than the proposed `42`-fiber enumeration.
