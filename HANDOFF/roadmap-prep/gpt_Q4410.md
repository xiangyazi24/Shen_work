## Executive corrections

There are two convention issues to settle before writing any formulas.

1. If the identity triple is to be

   ```text
   (1, 0, 0),
   ```

   then the infinity counter must be the **centered signed counter** used in this answer:

   \[
   D(u,v,k)
   =D_{\rm aff}(u,v)+k\,\infty_+-(r+k)\,\infty_-,
   \qquad r=\deg u.
   \tag{B}
   \]

   The balanced range is

   \[
   -1\le k\le 1-r.
   \]

   This is the shift `k = nInf - 1` from the nonnegative convention

   \[
   D_{\rm aff}+nInf\,\infty_+ +(2-r-nInf)\infty_-
      -(\infty_++\infty_-),
   \]

   whose identity is `(1,0,1)`. In that older convention the hyperelliptic counter is

   \[
   nInf' = 2-r-nInf.
   \]

2. The deck involution `sigma` does **not** preserve the two infinities. It exchanges them with two affine points over `x=A0`. Consequently there is no formula of the form

   ```text
   sigma(u,v,k) = (some transformed u, some transformed v, a scalar function of k)
   ```

   valid in every chart. The correct explicit formula is a transformed affine pair plus at most three fixed, tiny Mumford classes. This is still completely computable and uses only a bounded number of Cantor additions.

The formulas below are denominator-cleared wherever possible. The final equalities of canonical triples should be proved by applying `classOf_injective` from the balanced-Mumford module after checking the displayed polynomial identities. Do **not** unfold the complete normalization algorithm on both sides of the trace identity.

# 2. Hyperelliptic involution on a balanced triple

```text
D = (u,v,k),
r = natDegree u,
```

represent the divisor in `(B)`. Since this curve is `y^2=f(x)`, the hyperelliptic involution is simply `y -> -y`. Its exact centered-triple formula is

\[
\boxed{
h(u,v,k)=
\bigl(u,\,(-v)\bmod u,\,-r-k\bigr).
}
\tag{H-MUMFORD}
\]

Because `v` is already reduced modulo `u`, the second coordinate is normally just `-v`; keeping `% u` in the definition makes the canonical-remainder invariant explicit.

For a general model

\[
y^2+H(x)y=F(x),
\]

the second coordinate is

\[
(-v-H)\bmod u.
\]

The elliptic notation `a1*x+a3` is not intrinsic here; it would be the particular polynomial `H` in such a generalized hyperelliptic model.

## 2.1 Direct Cantor verification of `D + h(D) = 0`

\[
k_0=k_1+k_2+\deg d,
\tag{CENTERED-CANTOR}
\]

where

\[
d=\gcd(u_1,u_2,v_1+v_2).
\]

For `D` and `h(D)`:

\[
u_1=u_2=u,\qquad v_1+v_2=0,\qquad d=u.

Therefore

\[
u_0=\frac{u^2}{d^2}=1,\qquad v_0=0,

and

\[
k_0=k+(-r-k)+r=0.

# 3. Explicit `sigma` on a Mumford pair

## 3.1 Homogeneous polynomial transform

For `p : L[X]` and an integer `d >= natDegree p`, define

```lean
def homogSub (d : Nat) (p P Q : L[X]) : L[X] :=
  ∑ i in Finset.range (d + 1),
    C (p.coeff i) * P^i * Q^(d-i)
```

and set

```lean
PX := C A * X - C a
QX := X - C A
```

so that, away from `X=A`, `PX/QX = m(X)`.

Suppose first that `u(A) != 0`, and write `r=deg u`. Define

\[
U_{\sigma}^{\rm raw}(X)
 = (X-A)^r u(m(X))
 =\operatorname{homogSub}_r(u,AX-a,X-A),
\]

\[
\boxed{
 u_\sigma(X)=u(A)^{-1}U_{\sigma}^{\rm raw}(X).
}
\tag{SIGMA-U}
\]

The leading coefficient of `U_sigma_raw` is exactly `u(A)`, so `u_sigma` is monic.

For the `v` coordinate use the degree-three homogenization, regardless of `deg v`:

\[
V_{\sigma}^{\rm raw}(X)
=q^{-3}(X-A)^3v(m(X))
=q^{-3}\operatorname{homogSub}_3(v,AX-a,X-A),
\]

\[
\boxed{
 v_\sigma=V_{\sigma}^{\rm raw}\bmod u_\sigma.
}
\tag{SIGMA-V}
\]

For `deg u <= 2`, this is especially concrete. If

\[
u=X^2+u_1X+u_0,\qquad v=v_1X+v_0,
\]

then

\[
U_{\sigma}^{\rm raw}
=(AX-a)^2+u_1(AX-a)(X-A)+u_0(X-A)^2,
\]

\[
V_{\sigma}^{\rm raw}
=q^{-3}\bigl(
 v_1(AX-a)(X-A)^2+v_0(X-A)^3
\bigr).
\]

The certifying pointwise identities are: if `u(alpha)=0` and `alpha != A`, then

\[
u_\sigma(m(\alpha))=0,
\]

and

\[
v_\sigma(m(\alpha))
=\frac{q^3v(\alpha)}{(\alpha-A)^3}.
\]

The divisibility

\[
u_\sigma\mid f-v_\sigma^2
\]

follows directly from `(C-SIGMA)` after reducing modulo `u_sigma`.

## 3.2 The finite exceptional branch `u(A)=0`

The pole of `m` is `x=A`, so this branch must be handled rather than hidden behind field division.

Let

\[
e_A=\operatorname{ord}_{X-A}(u)\in\{0,1,2\},
\qquad
u_A=u/(X-A)^{e_A}.
\]

Use exact division by the monic polynomial `X-A`. The curve relation gives

\[
v(A)^2=f(A)=q^6,
\]

so, in characteristic zero,

\[
v(A)=q^3\quad\text{or}\quad v(A)=-q^3.
\]

A reduced Mumford pair cannot contain both hyperelliptic conjugates at `x=A`; hence all `e_A` copies have one sign. Put

\[
e_+=
\begin{cases}
e_A,&v(A)=q^3,\\
0,&v(A)=-q^3.
\end{cases}
\]

Apply `(SIGMA-U)` and `(SIGMA-V)` to `u_A` and `v % u_A`. The removed `R_+` factors become `infinityPlus`; the removed `R_-` factors become `infinityMinus`.

## 3.3 Three fixed tiny triples

In the centered convention define

```text
eta      = (1,       0,       1)   -- infinityPlus - infinityMinus
Rplus    = (X - A,   q^3,     0)   -- R+ - infinityMinus
Rminus   = (X - A,  -q^3,     0)   -- R- - infinityMinus
```

Let `sigmaFiniteBase(D)` be the normalized semi-triple

```text
(u_sigma, v_sigma, 0)
```

for the part of the affine support away from `A`. It represents

\[
\sigma(D_{\rm aff,away})-r_0\infty_-,
\qquad r_0=\deg u_A.
\]

Then the **complete** triple-level formula is

\[
\boxed{
\sigma_*(u,v,k)
=\operatorname{normalize}\Bigl(
  \operatorname{sigmaFiniteBase}(u_A,v)
  +e_+\,\eta
  +k\,Rplus
  -(r+k)\,Rminus
\Bigr).
}
\tag{SIGMA-TRIPLE}
\]

Every scalar in this expression is tiny:

```text
ePlus ∈ {0,1,2},
k ∈ {-1,0,1},
r+k ∈ {-1,0,1,2}.
```

Thus `zsmul` may be implemented by a finite branch instead of a general loop.

### Why `(SIGMA-TRIPLE)` is exact

The centered divisor is

\[
D=D_{\rm aff}+k\infty_+-(r+k)\infty_-.
\]

Applying `sigma` gives

\[
\sigma D
=\sigma D_{\rm aff}+kR_+-(r+k)R_-.
\]

The first two terms inside the normalizer in `(SIGMA-TRIPLE)` represent

\[
\sigma D_{\rm aff}-r\infty_-.
\]

Adding `k(R_+-infinityMinus)` and subtracting
`(r+k)(R_--infinityMinus)` cancels the artificial `-r infinityMinus`, leaving exactly the displayed divisor `sigma D`.

Finally define

```lean
def tauTriple (D : N18Mumford L) : N18Mumford L :=
  hyperelliptic (sigmaTriple D)
```

and prove

```lean
@[simp] theorem tauTriple_eq_neg_sigma (D) :
    tauTriple D = - sigmaTriple D := by
  simp [tauTriple, hyperelliptic_eq_neg]
```

# 5. Explicit pushforward `qPlus_*`

Let

\[
D=(u,v,k),\qquad r=\deg u.
\]

Over a splitting field, with multiplicities, write the affine divisor as

\[
D_{\rm aff}=\sum_i P_i,\qquad P_i=(\alpha_i,v(\alpha_i)).
\]

The exact group formula is

\[
\boxed{
q_{+*}D
=\sum_i q_+(P_i)
  +k(1,c)-(r+k)(1,-c).
}
\]

Since `(1,-c)=-(1,c)`, this simplifies to

\[
\boxed{
q_{+*}D
=\operatorname{pushAffinePlus}(u,v)
 +(r+2k)\,(1,c).
}
\tag{PUSH-PLUS}
\]

This formula is valid even when some affine point maps to the elliptic origin; such a point contributes zero to `pushAffinePlus`.

The coefficients `r+2k` in the balanced range are only

```text
r=0: -2, 0, 2
r=1: -1, 1
r=2:  0.
```

## 5.1 Computing `pushAffinePlus` without factoring `u`

Set

\[
A_+(X)=(X-r_+)^2,\qquad B_+(X)=(X-r_-)^2.
\]

First remove every exact factor `X-rMinus` from `u`; those source points map to the elliptic origin. Call the remaining monic polynomial `u0` and put `d=deg u0 <= 2`.

### Degree zero

Return `O`.

### Degree one

If

\[
u_0=X-\alpha,
\]

return the point

\[
\left(
\frac{A_+(\alpha)}{B_+(\alpha)},
\frac{c\,v(\alpha)}{(\alpha-r_-)^3}
\right).
\]

### Degree two: image `t`-polynomial

Define the resultant in the `X` variable

\[
R_+(T)
=\operatorname{Res}_X
\bigl(u_0(X),A_+(X)-T B_+(X)\bigr).
\]

Because `u0(rMinus) != 0`, its leading coefficient as a polynomial in `T` is nonzero. Normalize it to a monic polynomial

\[
U_+(T)=R_+(T)/\operatorname{lc}(R_+).
\]

Its roots, with multiplicity, are the two `t`-coordinates of the image divisor.

### Distinct image coordinates

Write

\[
U_+(T)=T^2+u_1T+u_0.
\]

Find `w1,w0 : L` such that

\[
\boxed{
(X-r_-)
\bigl(w_1A_+(X)+w_0B_+(X)\bigr)
\equiv c\,v(X)\pmod{u_0(X)}.
}
\tag{INTERP-PLUS}
\]

Then

\[
W(T)=w_1T+w_0
\]

passes through both image points.

This is a literal `2 x 2` solve. If

```text
RA = ((X-rMinus)*Aplus) % u0 = a1*X+a0
RB = ((X-rMinus)*Bplus) % u0 = b1*X+b0
RC = (c*v) % u0                 = c1*X+c0,
```

put

\[
\Delta=a_1b_0-a_0b_1.
\]

For `Delta != 0`:

\[
w_1=\frac{c_1b_0-c_0b_1}{\Delta},
\qquad
w_0=\frac{a_1c_0-a_0c_1}{\Delta}.
\]

On

\[
s^2=C_3t^3+C_2t^2+C_0,
\]

the sum of the two points is

\[
\boxed{
 t_{\Sigma}
 =u_1+\frac{w_1^2-C_2}{C_3},
\qquad
 s_{\Sigma}=-(w_1t_{\Sigma}+w_0).
}
\tag{CHORD-SUM}
\]

This follows by dividing

\[
C_3T^3+C_2T^2+C_0-W(T)^2
\]

by `UPlus(T)` and taking the negative of the third intersection.

### Repeated image coordinate

If `Delta=0`, then in characteristic zero the two source points have the same image point `Q=(t0,s0)` under the degree-two quotient. The pushforward is `2Q`.

The repeated root is

\[
t_0=-u_1/2.
\]

Obtain `s0` from the denominator-free congruence

\[
s_0(X-r_-)^3\equiv c\,v(X)\pmod {u_0(X)}.
\]

If `s0=0`, return `O`. Otherwise put

\[
\lambda=\frac{3C_3t_0^2+2C_2t_0}{2s_0},

\[
\boxed{

\qquad
 s_{2Q}=\lambda(t_0-t_{2Q})-s_0.

\tag{DOUBLE-PLUS}

This repeated-coordinate branch is necessary. A constant interpolation polynomial through two distinct source points mapping to the same elliptic point does not encode the tangent multiplicity on the quotient cubic.

# 6. Explicit pullback `qPlus^*`

Interpret an elliptic point `Q` as the degree-zero divisor class `[Q-O]`. The pullback is

\[
q_+^*[Q-O]=[q_+^{-1}(Q)-q_+^{-1}(O)].
\]

The fiber over `O` consists of the two hyperelliptic conjugate points above `x=rMinus`, and

\[
q_+^{-1}(O)\sim\infty_++\infty_-
\]

by the divisor of `x-rMinus`.

## 6.1 Generic finite point

Let

\[
Q=(t_0,s_0),\qquad t_0\ne1.
\]

Define

\[
U_Q(X)=(X-r_+)^2-t_0(X-r_-)^2.
\]

Its leading coefficient is `1-t0`, so define the monic polynomial

\[
\boxed{
 u_Q=(1-t_0)^{-1}U_Q.
}
\]

Define

\[
\boxed{
 v_Q=
 \left(
 \frac{s_0}{c}(X-r_-)^3
 \right)\bmod u_Q.
}
\]

Then

\[
\boxed{
q_+^*(t_0,s_0)
=(u_Q,v_Q,-1).
}
\tag{PULL-PLUS-GENERIC}
\]

The counter is `-1` because a degree-two centered triple with `k=-1` represents

\[
q_+^{-1}(Q)-\infty_+-\infty_-
\sim q_+^{-1}(Q)-q_+^{-1}(O).
\]

## 6.2 The chart `t=1`

Here `U_Q` loses its quadratic term and the fiber contains an infinity point. The two exact branches are

\[
\boxed{
q_+^*(1,c)=(X-A,q^3,0)=Rplus,
}
\]

because the fiber is `infinityPlus + RPlus`, and

\[
\boxed{
q_+^*(1,-c)=(X-A,-q^3,-1),
}
\]

because the fiber is `infinityMinus + RMinus`.

Finally

\[
q_+^*(O)=(1,0,0).
\]

Thus a complete Lean definition has only three point branches:

```lean
def pullPlusQ : QPlus.Point → N18Mumford L
  | 0 => 0
  | .some t s hcurve =>
      if ht : t = 1 then
        if hs : s = c then Rplus
        else RminusMinusInfinityPlus
      else
        genericFiberTriple t s ht
```

The curve equation and `FPlus(1)=c^2` prove that the `t=1` point not equal to `(1,c)` is `(1,-c)`.

## 6.3 The minus quotient

For a finite point `(T0,S0)` on `QMinus`, `T0 != 1`, use

\[
U_Q^-(X)=(X-r_-)^2-T_0(X-r_+)^2,
\]

\[
u_Q^-=(1-T_0)^{-1}U_Q^-,
\]

\[
v_Q^-=\left(\frac{S_0}{c}(X-r_+)^3\right)\bmod u_Q^-,
\]

and the centered counter `-1`.

The exceptional branches are

\[
\boxed{
q_-^*(1,c)=(X-A,-q^3,0)=Rminus,
}
\]

because the fiber is `infinityPlus + RMinus`, and

\[
\boxed{
q_-^*(1,-c)=(X-A,q^3,-1),
}
\]

because the fiber is `infinityMinus + RPlus`.

Also `qMinus^*(O)=0`.

# 7. The ring certificates for `qPlus^* qPlus_* = 1 + sigma_*`

The entire trace identity reduces to the following fixed formulas plus additivity.

## 7.1 Fiber `x`-polynomial

For source coordinate `x` and fiber variable `X`, the exact denominator-free identity is

\[
\boxed{
\begin{aligned}
&(x-r_-)^2(X-r_+)^2
 -(x-r_+)^2(X-r_-)^2\\
&\qquad=4q\,(X-x)
 \bigl((x-A)X-(Ax-a)\bigr).
\end{aligned}
}
\tag{FIBER-PLUS}
\]

If `x != A`, the second linear factor is

\[
(x-A)(X-m(x)).
\]

Thus the finite fiber is exactly the pair of points with `x`-coordinates `x` and `m(x)`. At `x=A` the degree drops, recording the missing infinity point. At `x=rMinus` the formula becomes the doubled `x=rMinus` fiber over the elliptic origin.

For the minus quotient the corresponding identity is simply the negative:

\[
\boxed{
\begin{aligned}
&(x-r_+)^2(X-r_-)^2
 -(x-r_-)^2(X-r_+)^2\\
&\qquad=-4q\,(X-x)
 \bigl((x-A)X-(Ax-a)\bigr).
\end{aligned}
}
\tag{FIBER-MINUS}
\]

These are pure `ring` identities using only the definitions of `rPlus`, `rMinus`, `A`, and `q`.

## 7.2 Fiber `y`-coordinate

For `qPlus`, away from `x=rMinus`, the pullback interpolation polynomial before reduction is

\[
V_{x,y}^+(X)
=\frac{y(X-r_-)^3}{(x-r_-)^3}.
\]

It satisfies

\[
V_{x,y}^+(x)=y,
\]

and, by `(M2)`,

\[
V_{x,y}^+(m(x))
=\frac{q^3y}{(x-A)^3},
\]

which is exactly the `y`-coordinate of `sigma(x,y)`.

For `qMinus`, use

\[
V_{x,y}^-(X)
=\frac{y(X-r_+)^3}{(x-r_+)^3}.
\]

Then

\[
V_{x,y}^-(m(x))
=-\frac{q^3y}{(x-A)^3},
\]

which is the `y`-coordinate of `tau(x,y)`.

## 7.3 Exceptional fibers

The remaining identities are the finite table

```text
qPlus^{-1}(1, c)  = infinityPlus  + Rplus
qPlus^{-1}(1,-c)  = infinityMinus + Rminus
qPlus^{-1}(O)     = points over x=rMinus

qMinus^{-1}(1, c) = infinityPlus  + Rminus
qMinus^{-1}(1,-c) = infinityMinus + Rplus
qMinus^{-1}(O)    = points over x=rPlus.
```

For example,

\[
(X-r_+)^2-(X-r_-)^2=-4q(X-A),
\]

and reducing

\[
\frac{c(X-r_-)^3}{c}
\]

modulo `X-A` gives `q^3`; the negative sign gives `-q^3`.

## 7.4 Raw divisor equality

For every point `P`, including ramification points with multiplicity,

\[
q_+^*q_*(P)=P+\sigma(P),
\]

and

\[
q_-^*q_*(P)=P+\tau(P).
\]

Therefore, for the centered divisor

\[
D=D_{\rm aff}+k\infty_+-(r+k)\infty_-,
\]

one gets the literal divisor identities

\[
\boxed{
q_+^*q_{+*}D=D+\sigma D,
}
\tag{TRACE-PLUS-DIV}
\]

and

\[
\boxed{
q_-^*q_{-*}D=D+\tau D.
}
\tag{TRACE-MINUS-DIV}
\]

No correction divisor remains: the infinity terms match because

\[
q_+^*(1,\pm c)=\infty_\pm+R_\pm
\]

and similarly for `qMinus` with the crossed signs above.

# 8. Canonical triple equalities

Once Block 1 has

```lean
classOf_injective : Function.Injective classOf
classOf_add        : classOf (D1 + D2) = classOf D1 + classOf D2
```

and the explicit push/pull formulas have been proved to represent the displayed divisors, the canonical-triple statements are:

```lean
theorem pullPlus_pushPlus (D : N18Mumford L) :
    pullPlusQ (pushPlusQ D) = D + sigmaTriple D := by
  apply classOf_injective
  -- Rewrite the explicit point/fiber formulas.
  exact tracePlusClass D

 theorem pullMinus_pushMinus (D : N18Mumford L) :
    pullMinusQ (pushMinusQ D) = D + tauTriple D := by
  apply classOf_injective
  exact traceMinusClass D
```

Here `tracePlusClass` and `traceMinusClass` should be proved from `(FIBER-PLUS)`, `(FIBER-MINUS)`, the two `V` evaluations, and the exceptional-fiber table. This is the right place for `ring`, `field_simp`, polynomial remainder normalization, and the already proved `normalize_sound` theorem.

Do not prove these by reducing both sides through every Cantor branch. Equality in the oriented ideal/divisor-class target plus `classOf_injective` gives equality of the unique balanced triples.

# 9. Explicit `Psi o Phi = [2]`

On the intermediate quotient models define

```lean
def Phi (D : N18Mumford L) : QPlus.Point × QMinus.Point :=
  (pushPlusQ D, pushMinusQ D)

 def Psi (P : QPlus.Point × QMinus.Point) : N18Mumford L :=
  pullPlusQ P.1 + pullMinusQ P.2
```

Then

```lean
theorem psi_phi (D : N18Mumford L) :
    Psi (Phi D) = 2 • D := by
  rw [Psi, Phi, pullPlus_pushPlus, pullMinus_pushMinus]
  rw [tauTriple_eq_neg_sigma]
  abel
```

Mathematically the calculation is

\[
\begin{aligned}
\Psi\Phi(D)
&=(D+\sigma D)+(D+\tau D)\\
&=(D+\sigma D)+(D-\sigma D)\\
&=2D.
\end{aligned}
\]

After transport by `plusIso` and `minusIso`, the same theorem is the desired

\[
\boxed{
q_+^*q_{+*}+q_-^*q_{-*}=[2]
}
\]

for the maps landing in `E0(L)` and `Ehat0(L)`.

  generic fiber triples
