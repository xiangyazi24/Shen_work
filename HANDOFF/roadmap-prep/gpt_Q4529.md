# 2. Exact Jensen mirror

Set, on a restart interval beginning at `r₀`,

```lean
V r x := U r x - W r x
z r x := negativePart (V r x).
```

Then

```text
z = V₋ = (W-U)₊,
negativePartTest V = -z.
```

The desired result `W≤U` is exactly `V≥0`.

The correspondence is:

| Negative-part nonnegativity proof | Jensen matched comparison |
|---|---|
| trajectory `u` | difference trajectory `V := U-W` |
| bad part `u₋` | bad part `V₋=(W-U)₊` |
| test `-u₋` | test `-V₋=-(W-U)₊` |
| target `u≥0` | target `V≥0`, hence `W≤U` |
| `negativePartEnergy u` | `negativePartEnergy V` |
| `negativePartDissipation u` | `negativePartDissipation V` |
| initial negative energy tends to zero | restart seed gives `V(r₀)≥0`, hence energy zero at restart |
| Gronwall | exactly the same Gronwall |
| zero energy → pointwise nonnegativity | exactly the same continuous-representative upgrade |

So the Jensen producer should prove a `NegativePartEnergyCoreDataFor` package for the shifted difference trajectory, then call the generic nonnegativity theorem.

# 3. The only PDE-specific difference

The two routes are not identical **before** the scalar energy inequality.

For the original truncated solution, the chemotaxis term cancels:

```lean
truncatedChemFluxLifted_mul_negDeriv_integral_eq_zero.
```

For the Jensen comparison it generally does **not** cancel. Instead U and W must first be put in the same matched divergence operator

```text
P[q] = q_t - q_xx + χ₀ ∂x(g q) - c q.
```

After subtracting, `V=U-W` satisfies the appropriate weak inequality for `P[V]`. Testing with `-V₋` produces

```text
(1/2) E'(r) + D(r)
  ≤ |χ₀| ∫ |g| V₋ |∂xV₋| + C₀ E(r),
```

where favorable barrier-residual terms have already been discarded and `C₀` bounds the zeroth-order coefficient.

If

```text
|g(r,x)| ≤ G,
```

then pointwise Young gives

```text
|χ₀| G V₋ |∂xV₋|
  ≤ (1/2)|∂xV₋|² + (χ₀² G²/2)V₋².
```

After integration,

```text
(1/2)E' + (1/2)D
  ≤ (C₀ + χ₀²G²/2) E.
```

Dropping the remaining nonnegative dissipation yields

```text
E' ≤ (2C₀ + χ₀²G²) E.
```

Thus the one genuinely new comparison lemma should be something of the form

```lean
theorem matchedComparison_half_energy_deriv_le
    ... :
    (1 / 2 : ℝ) * E' r ≤ C * negativePartEnergy V r
```

or, preferably retaining dissipation,

```lean
theorem matchedComparison_energy_dissipation_le
    ... :
    (1 / 2 : ℝ) * E' r
      + (1 / 2 : ℝ) * negativePartDissipation V r
      ≤ C * negativePartEnergy V r.
```

Everything after this theorem is shared infrastructure.

These declarations are specialized to the original truncated PDE and should not be forced onto the comparison:


More precisely:

- the first theorem uses the `u₊`/`u₋` disjoint-support cancellation, which is absent for `V=U-W`;
- `negativePart_half_energy_deriv_le_regular` hardcodes that cancellation and the truncated logistic source;
- the mild-to-weak data are specialized to the single physical truncated trajectory, whereas comparison needs the subtracted matched-divergence identity for U and W.

The reusable **ideas** are the same, but the Jensen route needs one comparison-specific weak identity and one comparison-specific right-hand-side absorption theorem.

# 6. Paste-ready final closure

After the matched-divergence calculation has supplied the energy frontier, the final Lean proof should be short. Schematically:

```lean
let V : ℝ → intervalDomainPoint → ℝ :=
  fun r x => U r x - W r x
let E : ℝ → ℝ := negativePartEnergy V

have hE_nonneg :
    ∀ r, r₀ < r → r ≤ r₁ → 0 ≤ E r := by
  intro r hr hr1
  exact MeasureTheory.integral_nonneg_of_ae
    (Eventually.of_forall fun x => negativePartEnergyDensity_nonneg V r x)

have hderiv_le :
    ∀ r, r₀ < r → r < r₁ → E' r ≤ K * E r := by
  intro r hr0 hr1
  have hhalf := matchedComparison_half_energy_deriv_le
    Hmatched Hchains Hbounds hr0 hr1.le
  nlinarith

have hgron :
    ∀ s t, r₀ < s → s ≤ t → t ≤ r₁ →
      E t ≤ E s * Real.exp (K * (t-s)) := by
  intro s t hs hst ht
  exact intervalDomainL2_gronwall_exp_of_diffIneq
    (E := E) (E' := E') (K := K)
    hst
    (hEcont.mono ...)
    (fun r hr => hEderiv r ...)
    (fun r hr => hderiv_le r ...)

have hE_zero :
    ∀ t, r₀ < t → t ≤ r₁ → E t = 0 := by
  -- either direct from E r₀ = 0, if the derivative package includes r₀,
  -- or use the positive-time shifted form below.
  let Es : ℝ → ℝ := fun τ => E (r₀ + τ)
  exact energy_eq_zero_of_positive_time_gronwall
    hEs_nonneg hEs_gron hEs_initial_vanishes

have hV_nonneg :
    ∀ t, r₀ < t → t ≤ r₁ → ∀ x, 0 ≤ V t x := by
  intro t ht ht1 x
  exact zero_energy_to_pointwise_nonneg t ht ht1 (hE_zero t ht ht1) x

have hW_le_U :
    ∀ t, r₀ < t → t ≤ r₁ → ∀ x, W t x ≤ U t x := by
  intro t ht ht1 x
  have := hV_nonneg t ht ht1 x
  dsimp [V] at this
  linarith
```

The 3000-line Jensen build should be cut at the following seam:

```text
NEW comparison-specific work:
  matched U/W weak identity
  + time/diffusion negative-part chain for V=U-W
  + matched drift Young absorption
  + barrier residual sign
  → one scalar differential inequality.

REUSE unchanged:
```

So the comparison is a true sign-flipped mirror **at the energy architecture level**, but not at the chemotaxis cancellation line. The matched-divergence Young estimate replaces the original `u₊·∂x u₋=0` cancellation; after that replacement, the remainder should be almost entirely shared code.
