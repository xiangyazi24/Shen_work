# hhom and hzero bookkeeping

hhom and hzero are easy but must be supplied at each place they are consumed:

1. Heat pass WSE0 -> WCE2: homogeneous term must decay like order 2; zero mode must be bounded for the source order 0.

1. Heat pass WSE1 -> WCE3: homogeneous term must decay like order 3; zero mode must be bounded for the source order 1.

1. Final pass-2 summability: homogeneous restart tail is super-polynomial from positive restart time, and the zero mode is harmless because λ_0 = 0, but if the theorem carries an explicit hzero, still pass the zero-mode lemma.

The reusable facts are:

```javascript
have hhom2 := homogeneous_heat_decay_of_restartBase_bound (order := 2) ...
have hhom3 := homogeneous_heat_decay_of_restartBase_bound (order := 3) ...
have hhomPass2 := homogeneous_heat_pass2_of_restartBase_bound ...
have hzero2 := zero_mode_bound_of_source_zero_or_bounded ...
have hzero3 := zero_mode_bound_of_source_zero_or_bounded ...
have hzeroPass2 := zero_mode_bound_of_source_zero_or_bounded ...
```

For the divergence piece, hzero is usually discharged by the endpoint flux identity ∫ ∂ₓQ = Q(1)-Q(0)=0. For the full B-form source, the logistic zero mode is only bounded, not zero; so do not accidentally reuse the divergence-zero lemma for the full source unless the hzero field is only for the chem-div component.

# Bottom line

The chain terminates in two source passes after the base WSE0. The only real wiring risks are:

- starting from WCE0 instead of the required WSE0;

- using the chem-div-only ladder_pass2_from_divergence_A0 as if it were the full B-form source envelope;

- forgetting to simpa the Nat reductions (2 - 1) = 1 and (3 - 1) = 2;

- forgetting that hhom/hzero must be passed separately for each heat/final theorem call.

With those handled, WSE2 -> restartCoeff_eigenvalue_weighted_summable_of_pass2(_envelope) closes the R leaf needed upstream by the HSpectral bootstrap.
