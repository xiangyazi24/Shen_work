# N18 Block 4 adversarial QA: the 3-isogeny constants

## Executive verdict

There is a coordinate error in the informal statement, but the underlying isogeny constants are sound once the coordinates are named correctly.

The most dangerous integration mistake is therefore not a wrong coefficient in C,A,B; it is mixing one of the three coordinate systems. The build should use distinct Lean names for the original rational model, the translated order-3 model, and the L-minimal model.

# 3. Recomputed Vélu map

On

[

The target equation is

[

boxed{

## 3.1 Exact polynomial certificate

The signs and coefficients are certified by

[

begin{aligned}

&bigl(Aw+Bbigr)^2

&qquad-C^3-30u^4C-26u^6\

&=A^2bigl(w^2-3uw+2w-u^3bigr).

end{aligned}

]

This is a pure polynomial identity. In Lean it should close with ring:

```javascript
def C (u : ℚ) : ℚ := u^3 - 6*u + 4
def A (u : ℚ) : ℚ := u^3 + 6*u - 8
def B (u : ℚ) : ℚ := -18*u^2 + 24*u - 8

lemma velu_identity (u w : ℚ) :
    (A u*w + B u)^2
      - 3*u*(C u)*(A u*w + B u)
      + 2*u^3*(A u*w + B u)
      - (C u)^3 - 30*u^4*(C u) - 26*u^6
    = (A u)^2 * (w^2 - 3*u*w + 2*w - u^3) := by
  simp [A, B, C]
  ring
```

Consequently there is no sign error in C,A,B.

These are the entire kernel. Do not try to discharge these cases by field_simp; they are branches of the total morphism.

## 7.4 From Selmer survivors to weak descent

# 8. Why the annihilator is exactly 21

Let

```plain text
G = E0(L),
H = <T>, with exponent 3,
B = Etilde(F3), with exponent 7.
```

Choose the common exponent

[

boxed{e=operatorname{lcm}(3,7)=21.}

]

It kills both H and the reduction quotient.

From weak descent,

[

P=h_0+3P_1.

]

Multiplying by e kills h₀:

[

eP=3eP_1.

]

Iterating gives, for every n,

[

boxed{eP=3^n eP_n.}

]

Because e kills the finite reduction group, every eP_n and eP lies in the formal kernel.

At the prime π, the formal multiplication series has the form

[

3=3z+z^2R(z),

qquad R(z)inmathcal O_{L_pi}[[z]].

]

For r=v_π(z)≥1,

[

v_π(3z)=r+3,

qquad

v_π(z^2R(z))ge2r,

]

hence

[

v_π([3]z)

]

A fixed nonzero formal point cannot therefore be divisible by every power of 3. It follows that eP=0 for every P, so

[

boxed{[21]E_0(L)=0.}

]

No extra factor is needed:

- ramification contributes vπ(3)=3, which strengthens the valuation gain;

- the reduction is good, so there is no component-group or Tamagawa factor;

- supersingularity does not hurt because only a lower bound is used;

- full local 3-divisibility is not separated, but multiplication by 21 first kills the order-7 reduction quotient and moves the argument into the formal kernel.

The exact computed group E0(L)≅Z/21 shows that 21 is sharp. Neither 3 nor 7 alone can annihilate it.
