# 1. The concrete linearization

The one-dimensional model is

```plain text
u_t = u_xx
      - χ₀ ∂x( u^m v_x / (1+v)^β )
      + u(a-bu^α),

0   = v_xx - μv + νu^γ,

u_x = v_x = 0 at x=0,1.
```

Let (uStar,vStar) be a positive constant equilibrium and write

```plain text
u = uStar + ε U,
v = vStar + ε V.
```

Because vStar is constant, vStar_x = 0. Consequently, in the first variation of the flux

```plain text
u^m v_x (1+v)^(-β),
```

all variations of u^m and (1+v)^(-β) are multiplied by vStar_x and vanish. The only linear term is

```plain text
uStar^m (1+vStar)^(-β) V_x.
```

The reaction derivative is

```plain text
rhoStar := d/du [u(a-bu^α)] at u=uStar
         = a - b(α+1)uStar^α.
```

At the positive logistic equilibrium, b*uStar^α=a, so

```plain text
rhoStar = -a*α.
```

For the minimal case a=b=0, rhoStar=0, which is also -a*α.

Thus the linearized system used by the current Paper 3 sigma definition is

```plain text
U_t = U_xx
      - χ₀ * uStar^m/(1+vStar)^β * V_xx
      - a*α*U,

0   = V_xx - μV + ν*γ*uStar^(γ-1)*U,

U_x = V_x = 0 at x=0,1.
```

This is valid at either equilibrium family encoded in the repository. For a completely arbitrary constant state that is not one of those equilibria, replace -a*α by the general reaction Jacobian rhoStar.

## Cosine-mode block

Let

```plain text
eₖ(x) = cos(kπx),
λₖ = (kπ)².
```

Write

```plain text
U(t,x)=Uₖ(t)eₖ(x),
V(t,x)=Vₖ(t)eₖ(x).
```

Since eₖ''=-λₖeₖ, the mode equations are

```plain text
Uₖ' = -(λₖ+aα) Uₖ
      + χ₀ * [uStar^m λₖ/(1+vStar)^β] * Vₖ,

0   = νγuStar^(γ-1) Uₖ - (μ+λₖ)Vₖ.
```

Equivalently, this is the descriptor block

```plain text
[1 0] [Uₖ']   [-(λₖ+aα)   χ₀ uStar^m λₖ/(1+vStar)^β] [Uₖ]
[0 0] [Vₖ'] = [ νγuStar^(γ-1)          -(μ+λₖ)       ] [Vₖ].
```

It is not an ordinary 2 × 2 dynamical system, because there is no Vₖ' equation. In Lean, avoid ordinary matrix eigenvalues and Hurwitz criteria. Solve the algebraic row:

```plain text
Vₖ = νγuStar^(γ-1)/(μ+λₖ) * Uₖ,
```

then substitute into the first row.

# 2. The exact scalar mode eigenvalue

After eliminating Vₖ,

```plain text
Uₖ' = σₖ Uₖ,
```

with

```plain text
σₖ = -λₖ - aα
     + χ₀ νγ uStar^(m+γ-1) λₖ
       / ((1+vStar)^β (μ+λₖ)).
```

This is definitionally the repository function

```javascript
def sigma
    (p : CM2Params) (uStar vStar lambdaN : ℝ) : ℝ :=
  -lambdaN +
    p.χ₀ * p.ν * p.γ *
      (uStar ^ (p.m + p.γ - 1) * lambdaN) /
        ((1 + vStar) ^ p.β * (p.μ + lambdaN)) -
    p.a * p.α
```

from ShenWork/Paper3/Statements.lean.

For every nonzero mode, define the positive chemotaxis coefficient

```plain text
κₖ := νγ uStar^(m+γ-1) λₖ
      / ((1+vStar)^β (μ+λₖ)) > 0.
```

Then

```plain text
σₖ = -(λₖ+aα) + χ₀ κₖ
    = (χ₀-χₖ*) κₖ,
```

where

```plain text
χₖ* = (λₖ+aα)/κₖ
     = (1+vStar)^β /(νγuStar^(m+γ-1))
       * (λₖ+aα)(μ+λₖ)/λₖ.
```

```plain text

χ₀ = χₖ*  ↔  σₖ = 0,
χₖ* < χ₀  ↔  0 < σₖ,
```

provided k ≠ 0, uStar>0, and vStar≥0.

# 3. Which mode determines the threshold?

The exact nonzero-mode stability criterion is

```plain text
∀ k≠0, σₖ<0
  ↔ χ₀ < inf_{k≠0} χₖ*.
```

This is already encoded as

```javascript
def LinearlyStable ... :=
  ∀ n : ℕ, n ≠ 0 → sigma p uStar vStar (S.eigenvalue n) < 0

def LinearlyUnstable ... :=
  ∃ n : ℕ, n ≠ 0 ∧ 0 < sigma p uStar vStar (S.eigenvalue n)
```

and

```javascript
def paperCriticalSensitivity ... :=
  sInf (paperCriticalSensitivitySet S p uStar vStar)
```

## Important correction: mode 1 is not always the minimizer

The λ-dependent part of χₖ* is

```plain text
h(λ) = (λ+aα)(μ+λ)/λ
     = λ + aαμ/λ + μ+aα.
```

It is U-shaped, with continuous minimum at

```plain text
λ = sqrt(aαμ).
```

Hence one may reduce the entire nonzero spectrum to k=1 only under the genuine first-mode-dominance condition

```plain text
a α μ ≤ λ₁².
```

On [0,1], λ₁=π², so this condition is

```plain text
a α μ ≤ π⁴.
```

Under this hypothesis,

```plain text
χ₁* ≤ χₖ* for every k≠0,
```

and therefore

```plain text
LinearlyStable ↔ χ₀ < χ₁*,
LinearlyUnstable ↔ χ₁* < χ₀,
χ₀=χ₁* gives the critical mode σ₁=0.
```

Without this regime, a higher mode can be the first unstable mode. A theorem claiming unconditional k=1 dominance would be false.

## The zero mode

The chemotaxis contribution contains λ₀, so it vanishes at k=0:

```plain text
σ₀ = -aα.
```

There are two cases.

1. Positive logistic equilibrium: a>0, so σ₀=-aα<0. Full-space linear stability is equivalent to stability of all nonzero modes.

1. Minimal model a=b=0: σ₀=0. The constant mode is mass-neutral, so asymptotic stability is only on the fixed-mass/zero-mean subspace. This is why the repository definition LinearlyStable quantifies only over n ≠ 0.

Thus the phrase “all modes are negative iff the k=1 condition holds” is correct for the positive equilibrium under first-mode dominance. For the minimal equilibrium, it must be read as “all nonzero modes are negative,” with the zero mode removed.
