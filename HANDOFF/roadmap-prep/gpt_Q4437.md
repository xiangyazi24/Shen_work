# 1. Does U_t ∈ L¹(0,T;H⁻¹) hold at u ≥ 0?





Semantically yes; as a standalone committed Lean object, not yet.

# 2. The finite-increment identity is the correct Leaf 1

For 0 ≤ s < t ≤ DT.T and one fixed bounded absolutely-continuous spatial test φ, the target should be

```plain text

  + ∫ₛᵗ ∫₀¹ U_x(r) φ_x dr
=
  χ₀ ∫ₛᵗ ∫₀¹ Q(r) φ_x dr
```

Equivalently,


This is the exact matched-divergence weak identity needed for the comparison argument.
