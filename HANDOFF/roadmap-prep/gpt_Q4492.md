The Paper 3 dynamical block should be built as two partly parallel trunks that meet at global stability:

1. Compactness / persistence trunk: eventual bounds → rolling-window smoothing → tail compactness → uniform persistence.

1. Spectral / sectorial trunk: cosine-mode linearization → critical threshold → sectorial decay / nonlinear Duhamel stability.

Then global stability is the merge point: it needs compactness plus persistence plus the Lyapunov/LaSalle engine, and it uses the spectral-sectorial side for equilibrium selection, local attractivity, and rate upgrades.

1. Build the spectral engine. This is relatively algebraic and reuses existing SpectralData, sigma, paperCriticalSensitivity, and unit interval cosine spectrum. It unlocks all threshold language and prevents later theorem statements from drifting.

1. Assemble P3.3 stability dichotomy. Combine step 2 + step 7.

1. Assemble P3.6 global stability. Combine step 4 + step 5 + step 6, and use step 7 if the target requires exponential or C¹ rate upgrades.

## Critical path

The longest dependency chain is not the 2×2 modal algebra. The critical path is:

```plain text
absorbing set
  → rolling-window smoothing
    → time-translate compactness
      → uniform persistence
        → Lyapunov dissipation identity
          → LaSalle / omega-limit singleton
            → global stability
```

The spectral-sectorial branch is also substantial, but it can proceed in parallel:

```plain text
2×2/scalar spectral engine
  → critical threshold / spectral gap
    → sectorial decay
      → local nonlinear stability
        → stability dichotomy and rate upgrades
```

## One-line answer

Prove spectral engine early, but do compactness and persistence before global stability. The dynamical merge point is Lyapunov/LaSalle, not the linear mode calculation. The global-stability proof will fail without compact omega-limit sets and uniform persistence, even if the spectral threshold algebra is already complete.
