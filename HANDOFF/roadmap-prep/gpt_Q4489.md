# Definitive verdict

The relevant convergence target in the strict-logistic branch 0 < a, 0 < b is the spatially constant positive coexistence equilibrium

This pair is uniquely determined among positive constant equilibrium pairs. That uniqueness is pure algebra and should be packaged as a small reusable theorem.

It is not sound to state, without additional parameter hypotheses and PDE analysis, that this is the unique positive steady state in the full function space. Chemotaxis systems can admit nonconstant steady states; Paper 3's mode-instability threshold is precisely a warning that the constant state need not control the full positive steady-state set. Moreover, in the minimal branch a=b=0, the repository already defines a whole family

```javascript
minimalEquilibrium p uStar
```

parameterized by every uStar > 0, so even uniqueness among positive constants is false there.

Therefore the correct Roadmap package is:

```plain text
strict-logistic branch:
  unique positive constant equilibrium pair       -- pure algebra
  constant pair is a global classical steady state -- already in repo

LaSalle/global stability:
  zero dissipation forces u = uStar
  elliptic uniqueness then forces v = vStar        -- rigidity, not global steady-state uniqueness
```

# 2. Pure algebraic existence and uniqueness

Define a predicate only for constant scalar pairs:

```javascript
namespace ShenWork.Paper3

noncomputable section

/-- Positive scalar pair satisfying the two constant-equilibrium equations. -/
def IsPositiveConstantCoexistenceEquilibrium
    (p : CM2Params) (q : ℝ × ℝ) : Prop :=
  0 < q.1 ∧
  0 < q.2 ∧
  q.1 * (p.a - p.b * q.1 ^ p.α) = 0 ∧
  p.μ * q.2 = p.ν * q.1 ^ p.γ
```

Then the paste-ready theorem is:

```javascript
theorem positiveConstantCoexistenceEquilibrium_existsUnique
    (p : CM2Params) (ha : 0 < p.a) (hb : 0 < p.b) :
    ∃! q : ℝ × ℝ, IsPositiveConstantCoexistenceEquilibrium p q := by
  let hab : 0 < p.a ∧ 0 < p.b := ⟨ha, hb⟩
  let qStar : ℝ × ℝ := positiveEquilibrium p hab
  refine ⟨qStar, ?_, ?_⟩
  · exact
      ⟨positiveEquilibrium_fst_pos p hab,
       positiveEquilibrium_snd_pos p hab,
       positiveEquilibrium_reaction_zero p hab,
       positiveEquilibrium_elliptic_relation p hab⟩
  · intro q hq
    rcases hq with ⟨hu, _hv, hreaction, helliptic⟩

    have hlog : p.a - p.b * q.1 ^ p.α = 0 :=
      (mul_eq_zero.mp hreaction).resolve_left hu.ne'

    have hqPow : q.1 ^ p.α = p.a / p.b := by
      apply (eq_div_iff (ne_of_gt hb)).2
      nlinarith

    have hStarPos : 0 < qStar.1 := by
      simpa [qStar] using positiveEquilibrium_fst_pos p hab

    have hStarPow : qStar.1 ^ p.α = p.a / p.b := by
      simpa [qStar] using positiveEquilibrium_fst_rpow_alpha p hab

    have hPowEq : q.1 ^ p.α = qStar.1 ^ p.α :=
      hqPow.trans hStarPow.symm

    have huEq : q.1 = qStar.1 := by
      apply le_antisymm
      · exact
          (Real.rpow_le_rpow_iff hu.le hStarPos.le p.hα).mp hPowEq.le
      · exact
          (Real.rpow_le_rpow_iff hStarPos.le hu.le p.hα).mp hPowEq.ge

    have hStarElliptic :
        p.μ * qStar.2 = p.ν * qStar.1 ^ p.γ := by
      simpa [qStar] using positiveEquilibrium_elliptic_relation p hab

    have hmul : p.μ * q.2 = p.μ * qStar.2 := by
      calc
        p.μ * q.2 = p.ν * q.1 ^ p.γ := helliptic
        _ = p.ν * qStar.1 ^ p.γ := by rw [huEq]
        _ = p.μ * qStar.2 := hStarElliptic.symm

    have hvEq : q.2 = qStar.2 :=
      mul_left_cancel₀ (ne_of_gt p.hμ) hmul

    exact Prod.ext huEq hvEq
```

The key Mathlib fact is

```javascript
Real.rpow_le_rpow_iff
    (hx : 0 ≤ x) (hy : 0 ≤ y) (hz : 0 < z) :
    x ^ z ≤ y ^ z ↔ x ≤ y
```

from Mathlib 4.29.1. It expresses strict monotonicity/injectivity of x ↦ x^α on [0,∞) when α>0. No PDE theorem enters the uniqueness proof.

An equivalent shorter proof can use

```javascript
Real.strictMonoOn_rpow_Ici_of_exponent_pos p.hα
```

and its injOn consequence.

# 4. Why full positive-steady-state uniqueness is the wrong theorem

The following stronger target should not be introduced unconditionally:

```javascript
∃! (U, V : intervalDomain.Point → ℝ),
  PositiveSteadyState p U V
```

There are two concrete objections.

## 4.1 The minimal branch has a continuum of constants

The repository defines

```javascript
def minimalEquilibrium (p : CM2Params) (uStar : ℝ) : ℝ × ℝ :=
  (uStar, p.ν / p.μ * uStar ^ p.γ)
```

and proves, when a=b=0, both the reaction balance and elliptic relation for every uStar. Hence there is no unique positive equilibrium in that branch. The conserved mass selects which member of the family is the convergence target.

## 4.2 Nonconstant steady states are not ruled out algebraically

For a nonconstant steady state the equations are

with Neumann boundary conditions. These do not reduce pointwise to the two scalar algebraic equations because the diffusion and chemotaxis terms need not vanish. Excluding nonconstant solutions is a genuine elliptic PDE rigidity theorem and generally requires a small-sensitivity/energy condition. Above a modal threshold the constant state is linearly unstable, so an unconditional full-cone uniqueness theorem would be especially suspect.

# 5. What LaSalle/global stability actually needs

The clean route is not “all positive steady states are unique.” It is:

```plain text
omega-limit point
+ Lyapunov derivative vanishes
→ dissipation equals zero
→ u is the constant uStar
→ elliptic equation forces v to be the constant vStar.
```

For the strict-logistic branch, the repository's theta dissipation is

```javascript
chemotaxisThetaDissipation intervalDomain uStar p.α U
```

with integrand

Since α>0, the integrand vanishes exactly when U=uStar. Thus the needed rigidity target is closer to:

```javascript
theorem thetaDissipation_eq_zero_forces_constant
    {p : CM2Params} {uStar : ℝ}
    {U : intervalDomain.Point → ℝ}
    (huStar : 0 < uStar)
    (hU_nonneg : ∀ x, 0 ≤ U x)
    (hU_cont : Continuous U)
    (hzero :
      chemotaxisThetaDissipation intervalDomain uStar p.α U = 0) :
    ∀ x, U x = uStar
```

The exact continuity type may need to be stated through intervalDomainLift/ContinuousOn to match the interval API. The proof is: nonnegative continuous integrand + zero integral implies pointwise zero; then strict monotonicity of real powers gives U=uStar.

# Bottom line

- Existence: already proved in-repo, including the full classical Neumann PDE witness.

- Uniqueness: pure algebra and easy, but only among positive constant scalar pairs, under 0<a and 0<b.

- Full positive cone: no unconditional uniqueness theorem; that statement is not algebraic and may be false.

- Minimal model: explicitly has a one-parameter equilibrium family, so uniqueness fails.

- LaSalle target: prove zero-dissipation rigidity to the canonical positiveEquilibrium, not global uniqueness of all steady states.
