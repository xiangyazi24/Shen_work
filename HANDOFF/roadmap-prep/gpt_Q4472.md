# Verdict

The execution-ready Paper 3 spectral engine should be built around the repo's existing scalar reduced mode growth sigma, not around a new independent dynamic 2×2 parabolic-parabolic matrix. The underlying linearization does have a natural 2×2 mode block, but because the signal equation in the current Paper 2/Paper 3 statement layer is elliptic,

```javascript
0 = Δv - μ v + ν u^γ,
```

the v-mode is algebraic. Eliminating it gives exactly the existing scalar mode growth

```javascript
def sigma (p : CM2Params) (uStar vStar lambdaN : ℝ) : ℝ :=
  -lambdaN +
    p.χ₀ * p.ν * p.γ *
      (uStar ^ (p.m + p.γ - 1) * lambdaN) /
        ((1 + vStar) ^ p.β * (p.μ + lambdaN)) -
    p.a * p.α
```

from Paper3/Statements.lean. So the best Lean target is: define a small explicit mode-matrix/Schur-complement wrapper, prove its determinant threshold reduces to the existing sigmaCriticalChiPaperFormula, and then reuse the already-landed LinearlyStable, LinearlyUnstable, paperCriticalSensitivity, and exact first-mode-threshold lemmas.

# 1. Exact mode block

Let λ = λ_k = (kπ)^2, perturb

```plain text
u = ū + ξ_k(t) cos(kπx),
v = v̄ + η_k(t) cos(kπx).
```

For the chemotaxis flux

```plain text
-χ₀ ∂ₓ( u^m (1+v)^(-β) ∂ₓv )
```

at a constant equilibrium, the derivative of the mobility factor does not contribute at first order, because ∂ₓ v̄ = 0. Define

```plain text
A      := ū^m / (1+v̄)^β
f_u    := ∂_u [u(a - b u^α)] at ū
        = a - b(1+α) ū^α
g_u    := ∂_u [ν u^γ] at ū
        = ν γ ū^(γ-1)
g_v    := ∂_v[-μv] = -μ.
```

For the positive logistic equilibrium, ū^α = a/b, so

```plain text
f_u = -aα.
```

The elliptic linearized mode equations are

```plain text
ξ'_k = (-λ + f_u) ξ_k + χ₀ A λ η_k
0    = g_u ξ_k - (μ + λ) η_k.
```

Thus the natural algebraic block is

```plain text
B_k = [[ -λ + f_u,        χ₀ A λ ],
       [  g_u,           -(μ + λ) ]].
```

If you temporarily view it as a parabolic-parabolic dynamic matrix, the same entries give the trace/determinant test; for the current elliptic problem, the real object is the Schur complement obtained from the second row.

Eliminating η_k gives

```plain text
η_k = g_u/(μ+λ) · ξ_k
```

and therefore

```plain text
ξ'_k = [ -λ + f_u + χ₀ A λ g_u/(μ+λ) ] ξ_k.
```

At the positive equilibrium, this is exactly

```plain text
σ_k = -λ - aα
      + χ₀ ν γ ū^(m+γ-1) λ / ((1+v̄)^β(μ+λ)).
```

This matches the repo sigma p ū v̄ λ.

# 2. Trace/determinant condition

For the 2×2 dynamic-form block

```plain text
tr B_k  = (-λ + f_u) - (μ + λ)
        = f_u - μ - 2λ.

det B_k = (-λ + f_u)(-(μ+λ)) - χ₀ A λ g_u
        = (λ - f_u)(μ+λ) - χ₀ A g_u λ.
```

At the positive equilibrium f_u = -aα, so

```plain text
tr B_k = -(μ + aα + 2λ) < 0
```

for all modes, and

```plain text
det B_k = (λ + aα)(μ+λ)
          - χ₀ · νγ ū^(m+γ-1) · λ / (1+v̄)^β.
```

For λ > 0, det B_k > 0 is equivalent to

```plain text
χ₀ < ((1+v̄)^β / (νγ ū^(m+γ-1))) · ((λ+aα)(μ+λ)/λ).
```

That right-hand side is exactly

```javascript
sigmaCriticalChiPaperFormula p uStar vStar lambdaN
```

already defined in the repo.

For the elliptic system, the equivalent scalar condition is simply

```javascript
sigma p uStar vStar λ < 0
```

and, for nonzero modes with 0 < λ, this is equivalent to

```javascript
p.χ₀ < sigmaCriticalChi p uStar vStar λ
```

or, after the existing formula lemma,

```javascript
p.χ₀ < sigmaCriticalChiPaperFormula p uStar vStar λ.
```

# 3. Dichotomy and critical mode

Do not hard-code k = 1 as the critical mode globally. The repo correctly defines

```javascript
def paperCriticalSensitivity
    (S : SpectralData) (p : CM2Params) (uStar vStar : ℝ) : ℝ :=
  sInf (paperCriticalSensitivitySet S p uStar vStar)
```

where the set ranges over all nonzero modes. This gives the correct general dichotomy:

```javascript
LinearlyStable S p uStar vStar
  ↔ ∀ n, n ≠ 0 → p.χ₀ < sigmaCriticalChiPaperFormula p uStar vStar (S.eigenvalue n)
```

and instability when some nonzero mode satisfies the reverse strict inequality.

The repo already proves the useful first-mode specialization in the first-mode-dominant regime

```plain text
a α μ ≤ firstNonzero².
```

Then the threshold is attained at mode 1, and the dichotomy can be stated with S.eigenvalue 1. Outside that regime, the correct critical mode is the discrete minimizer/infimum of the threshold set, not necessarily k = 1.
