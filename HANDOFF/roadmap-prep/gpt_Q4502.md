# N17 fixed-case draft: `no_rational_point_of_order_17`

## Executive verdict

The case `p = 17` is genuinely **fixed-curve tractable**.  It does not require the
uniform Eisenstein-quotient/formal-immersion package.

The correct diagram is

```text
(E/Q, P of exact order 17)
        |
        | forget the generator
        v
noncuspidal point of X_0(17)(Q)
        |
        | fixed model isomorphism
        v
E17(Q),  E17 : y^2 + x*y + y = x^3 - x^2 - x - 14.
```

Here

```text
g(X_1(17)) = 5,
g(X_0(17)) = 1,
X_0(17) = 17a1 = LMFDB 17.a3,
E17(Q) ~= Z/4Z.
```

There is one essential correction to the tempting argument:

```text
X_0(17)(Q) does NOT consist only of cusps.
```

With the normalization used below, its four rational points are

```text
O,                       cusp infinity,
T  = (7,13),              cusp 0,
2T = (11/4,-15/8),        noncuspidal 17-isogeny point,
-T = (7,-21),             noncuspidal 17-isogeny point.
```

Thus the rank-zero calculation reduces order `17` to **two fixed fibers** of
`X_1(17) -> X_0(17)`.  Those fibers have no rational generator.  A particularly
small certificate uses their two source `j`-invariants

```text
jA = -(17^2 * 101^3)/2,
jB = -(17 * 373^3)/2^17.
```

Both are `3`-adically integral.  An elliptic curve over `Q` with a rational
point of order `17` cannot have integral `j` at `3`:

* with good reduction, the order-17 point injects into `E(F_3)`, while the
  Hasse bound gives `#E(F_3) <= 7`;
* with additive potentially-good reduction, the prime-to-3 torsion injects
  into a special-fibre group whose order has no factor `17`;
* multiplicative reduction would force `v_3(j) < 0`.

So `p = 17` requires

```text
one elliptic 2-isogeny descent
+ the already shared 2-adic formal-kernel separatedness theorem
+ two fixed modular-fiber certificates.
```

It does **not** require a genus-five Jacobian, and it does **not** require the
uniform `PrimeTailPackage`.

# 1. The fixed rank-zero elliptic curve

The minimal model is

```text
E17 : y^2 + x*y + y = x^3 - x^2 - x - 14.
```

Its discriminant is `-17^4`.  The following transformation exposes its
rational point of order two:

```text
X = 4*x - 11,
V = 8*y + 4*x + 4.
```

It gives

```text
ES : V^2 = X^3 + 30*X^2 + 289*X
           = X*(X^2 + 30*X + 289).
```

The point `(11/4,-15/8)` becomes `(0,0)`.  The standard degree-two quotient is

```text
ED : W^2 = U^3 - 60*U^2 - 256*U
           = U*(U-64)*(U+4),
```

and, away from `X = 0`, the isogeny is

```text
U = V^2/X^2,
W = V*(289-X^2)/X^2.
```

The point `T=(7,13)` becomes `(17,136)` and maps to `(64,0)` on `ED`.

## 1.1 Tiny Kummer calculation

For

```text
y^2 = x^3 + A*x^2 + B*x,
```

a squareclass `d | B` in the two-isogeny Kummer image has a primitive integral
solution of

```text
N^2 = d*M^4 + A*M^2*e^2 + (B/d)*e^4.
```

For `ES`, the only squareclasses are `1, -1, 17, -17`.  The negative classes
are excluded by the identities

```text
d = -1:
  RHS = -((M^2 - 15*e^2)^2 + 64*e^4),

d = -17:
  RHS = -(17*(M^2-e^2)^2 + 4*M^2*e^2).
```

The classes `1` and `17` occur; for `17` use `(M,e,N)=(1,1,8)`.  Hence

```text
alpha(ES(Q)) = {1,17}.
```

For `ED`, the candidate classes are `1,-1,2,-2`.  The classes `1,-1` occur.
The classes `2,-2` are excluded by a short 2-adic split:

```text
M odd:       contradiction modulo 8;
v2(M)=1:     divide by 16, contradiction modulo 8;
v2(M)=2:     divide by 64, contradiction modulo 8;
v2(M)>=3:    the right side has odd valuation 7.
```

Thus

```text
alpha'(ED(Q)) = {1,-1}.
```

The standard two-isogeny exact sequence, including its rational kernel term,
then gives the actual weak-descent statement

```text
E17(Q) = <T> + 2*E17(Q),       order(T)=4.
```

This is the only global arithmetic input needed by the separatedness engine.

# 4. Final scope assessment

```text
p = 17 is a fixed case.
```

It is smaller than N18:

```text
N18: genus-two Picard group + split over a cubic field + 3-isogeny descent.
N17: one rank-zero elliptic curve + two fixed fibres.
```

The fixed proof is not merely “rank `X_0(17)` is zero,” because two
noncuspidal rational isogeny points remain.  Once those two fibres are checked,
`no_rational_point_of_order_17` is independent of the uniform prime-tail
formal-immersion theorem.
