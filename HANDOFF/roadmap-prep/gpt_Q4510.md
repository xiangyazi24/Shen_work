# 3. Existence: exact elementary proof

Existence has three stages:

1. extract a semi-Mumford pair from an arbitrary oriented invertible fractional
   ideal;
2. reduce the affine degree to at most two;
3. balance the infinity integer.

No Riemann‚ÄďRoch theorem is needed.

## 3.1 From an arbitrary integral ideal to `‚ü®u,y-v‚ü©`

Write

```text

```

```text
C(J) = { q ‚ąą R | ‚ąÉ p, p+qy ‚ąą J },
B(J) = J ‚ą© R.
```

Both are nonzero.

* For `B(J)`, take a nonzero `z=p+qy‚ąąJ`.  Its conjugate is
  `zŐĄ=p-qy`; since `A` is a domain,
  
  ```text
  0 ‚Č† z zŐĄ = p¬≤-q¬≤f ‚ąą J‚ą©R.
  ```

* For `C(J)`, if an element of `J` already has nonzero `y` coefficient there
  is nothing to prove.  Otherwise choose `0‚Č†a‚ąąJ‚ą©R`; then `ay‚ąąJ`, so
  `a‚ąąC(J)`.

Choose principal generators

```text

```

and choose `b+cy‚ąąJ`.  Then

```text

```

Indeed, if `p+qy‚ąąJ`, write `q=sc`; subtracting `s(b+cy)` leaves
`p-sb‚ąąJ‚ą©R=(a)`.

Stability under multiplication by `y` gives

```text

```

The first follows from `ay‚ąąJ`, whose `y` coefficient is `a`; the second follows
from


whose `y` coefficient is `b`.

Write

```text

```

After scaling the fractional ideal by the principal scalar `c‚ĀĽ¬Ļ`,


Because `(y-v)(y+v)=f-v¬≤` belongs to the intersection of this ideal with `R`,


a semi-Mumford pair.

For a fractional ideal, first use

to clear its denominator.  The principal scalar used in this step updates the

## 3.4 Balancing the infinity integer

Now assume `r=deg u‚Č§2`.  The target interval is

```text

```

There are two reduced-to-reduced baby steps.

### High step: decrease `n`

If `n>2-r`, use

```text

vŐÉ = A‚ąě-s.
```

The complement still has degree at most two.

If `s‚Č†0`, let `d=deg s<r`.  Then

```text

```

At `‚ąě‚āä`,

```text

```

The first summand has a zero of order `2`, while `s` has a pole of order `d`.
Thus

```text
ord‚āä(y-vŐÉ)=-d,
ord‚āä(őĪ)=3-r.
```

If `s=0`, then `u|8x`, `deg u'=1-r`, and

```text
ord‚āä(y-A‚ąě)=2,
```

so the same formula remains true:

```text
ord‚āä(őĪ)=3-r.
```

Therefore

```text

```

Because `n>2-r`, one has `n‚Č•3-r`, hence `n'‚Č•0`: the high step never
overshoots below the lower boundary.

If `s‚Č†0`, the new upper bound is

```text

```

The excess above the upper bound drops by

```text
[n-(2-r)] - [n'-(2-r')] = r-d ‚Č• 1.
```

In the exceptional `s=0` branch it drops by `r+2‚Č•2`.

### Low step: increase `n`

If `n<0`, use

```text
t = (-A‚ąě-v) mod u,
vŐÉ = -A‚ąě-t.
```

Again `vŐÉ‚Č°v mod u`.

If `t‚Č†0`, let `d=deg t<r`.  Then

```text
deg u'=d+3-r.
```

At `‚ąě‚āä`, `y+A‚ąě` has a pole of order `3`, so

```text
ord‚āä(y-vŐÉ)=ord‚āä(y+A‚ąě+t)=-3,
ord‚āä(őĪ)=d-r.
```

Hence

```text

```

If `t=0`, then `u|8x`, `deg u'=1-r`, and

```text
ord‚āä(őĪ)=-2-r,
n'=n+r+2.
```

Since `n‚Č§-1`, in the nonexceptional branch

```text
n'‚Č§r-d-1=2-r',
```

and in the exceptional branch

```text
n'‚Č§r+1=2-r'.
```

Thus the low step never overshoots above the upper boundary.
If `n'` is still negative, its deficiency decreases by `r-d‚Č•1`, or by
`r+2` in the exceptional branch.

### Well-founded measure

For `r‚Č§2`, define

```text

  = max(0,-n) + max(0,n-(2-r)).
```

Only one summand can be nonzero.  `HIGH` or `LOW` strictly decreases this
natural-number measure.  Therefore balancing terminates.

Every step preserves the oriented class by `(‚ėÖ)` and `(ORIENT)`.  Its output
has

```text

0‚Č§n‚Č§2-deg u.
```

This proves `classOf_surjective` by an explicit terminating algorithm.

# 4. Uniqueness without Riemann‚ÄďRoch

This is the shortest Lean route for `reduced_unique_of_principal`.

Let



## 4.1 The `‚ąě‚āč` valuation

Conjugate `(I)` by the hyperelliptic involution and multiply it by `(I)`.
Using

```text

```

```text

```

The quotient `u‚āĀ(őĪőĪŐĄ)/u‚āā` lies in `K(x)` and is a unit of the affine ring.
A rational function in `K(x)` with no finite zero or pole is constant, so its
infinity degree is zero.  Therefore

```text
ord‚āä(őĪ)+ord‚āč(őĪ)=r‚āĀ-r‚āā.
```

Together with `(P)`,

```text
ord‚āč(őĪ)=r‚āĀ+n‚āĀ-r‚āā-n‚āā.                             (M)
```

The balance inequalities imply the pole bounds

```text

ord‚āč(őĪ) ‚Č• -(2-r‚āĀ-n‚āĀ).                            (B-)
```

Indeed `n‚āā‚Č•0` gives `(B+)`, and `r‚āā+n‚āā‚Č§2` gives `(B-)`.

## 4.2 Normal form for an element of the inverse ideal





```text

  ‚Č• min(ord‚āä őĪ, ord‚āä őĪŐĄ)
```


hence `m‚Č§-1`, contradicting `m‚Č•0`.  Therefore


So `őĪ=a(x)` is a polynomial in `x`.

## 4.4 The only possible nonconstant polynomial is the canonical exception




a contradiction.

Therefore `őĪ` is a nonzero constant.

# 5. The exact Riemann‚ÄďRoch proof, for comparison

If a curve-divisor and Riemann‚ÄďRoch API is eventually available, existence and
uniqueness become much shorter.

Let

```text
B=‚ąě‚āä+‚ąě‚āč.
```

The differential `dx/y` has a simple zero at each infinity, so

```text
K_C ‚ąľ B.
```

For a degree-zero divisor `D`, Riemann‚ÄďRoch applied to `D+B` says

```text
‚Ąď(D+B)-‚Ąď(K_C-D-B)
  = deg(D+B)+1-g
  = 1.
```

Thus `‚Ąď(D+B)‚Č•1`; choose a nonzero function `a` with

```text
E=div(a)+D+B‚Č•0.
```

Then `deg E=2` and

```text
D ‚ąľ E-B.
```

If `E` is a hyperelliptic fiber, then `E‚ąľB` and the class is zero; choose the
canonical representative `B-B`.  Otherwise `E` contains no conjugate pair and
has a unique balanced Mumford encoding.

For uniqueness, if two degree-two effective divisors `E‚āĀ,E‚āā` are linearly
equivalent, Riemann‚ÄďRoch gives

```text
‚Ąď(E‚āĀ)=1+‚Ąď(K_C-E‚āĀ).
```

Since `deg(K_C-E‚āĀ)=0`,

```text
‚Ąď(K_C-E‚āĀ)>0  iff  E‚āĀ‚ąľK_C.
```

Therefore:

```text
E‚āĀ‚ą¶K_C  ‚áí ‚Ąď(E‚āĀ)=1 ‚áí E‚āā=E‚āĀ,

E‚āĀ‚ąľK_C  ‚áí E‚āĀ is a hyperelliptic fiber.
```

The semireduced graph condition excludes an affine conjugate fiber, and the
balanced convention chooses `‚ąě‚āä+‚ąě‚āč` as the sole canonical representative.
Hence `E‚āĀ=E‚āā` in the exceptional case as well.

This is the exact Riemann‚ÄďRoch statement needed.  The incorrect statement is
that every effective divisor of degree two has `‚Ąď=1`.
