# 5. Cusp certificates

Use the optimized model

```plain text
Copt : V^2+(U^3-2U^2+3U+1)V+2U=0.
```

The conversion from the sextic is

```plain text
U=-x,
V=(y-(U^3-2U^2+3U+1))/2.
```

Define

```plain text
Nr=U^2-U*V-3U+1,
Dr=(U-1)^2*(U*V+1),

Ns=U^2-2U-V,
Ds=U^2-U*V-3U-V^2-2V.
```

When Dr*Ds != 0, put

```plain text
r=Nr/Dr,
s=Ns/Ds,
b=r*s*(r-1),
c=s*(r-1)
```

and evaluate the Tate discriminant

```plain text
Delta=b^3*(16b^2-8bc^2-20bc+b+c(c-1)^3).
```

A fiber point is marked as a cusp by one of the finite witnesses

```plain text
Dr=0,
Ds=0,
Delta=0,
or the point is one of the two infinities.
```
