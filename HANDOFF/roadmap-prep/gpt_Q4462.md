# Reference Lean 4 draft for the Z/2 × Z/14 structural bridge

## Executive verdict

The bridge is an explicit parametrization theorem. It does not need the rational-point theorem for the genus-four modular curve. The shortest implementation is:

```plain text
injective ZMod 2 × ZMod 14 → E(Q)
  → R14 of exact order 14 and independent T0 of exact order 2
  → reuse the existing order-14 Tate normalization of R14
  → Q7 = 2R14 has Tate coordinates (b,bc)
  → explicitly renormalize Q7 to the universal order-7 curve E_d
  → transport T0, 7R14, and T0+7R14 and take their three x-coordinates ξ0,ξ1,ξ2
  → Bruin–Najman rational functions t,η+,η−,u,w
  → two denominator-cleared identities
  → the symmetric N14 equation and a noncusp certificate.
```

There is no group-theoretic gap in extracting the marked torsion data: current Mathlib already has the exact lemma

```javascript
addOrderOf_injective
```

for injective additive homomorphisms. The only substantial project-local producers are:

1. an adapter to the existing order-14 Tate/Kubert bridge;

1. the explicit pointed change of variables taking Q7=(b,bc) to E_d, while transporting the three rational 2-torsion points;

1. the generated polynomial certificates for the Bruin–Najman quotient coordinates and their denominator-open conditions.

The draft below uses sorry exactly at those producer boundaries. Every downstream composition, including the target curve equation and nondegeneracy, is explicit.

A model-convention warning is unavoidable: the formulas below land first on the standard symmetric model

If the repository definitions E_N14_AffineEquation and E_N14_DegenerateParameter are this model, the final adapter is simpa. If the repository uses the Derickx–Sutherland optimized model, retain one fixed birational adapter theorem; do not alter the torsion/Tate part.

/-! ======================================================================
    3. Explicitly renormalize Q7=(b,bc) to the universal E₁(7) curve

/-- At `Q7=(b,bc)` on the order-14 Tate curve, the tangent denominator is
`2y+a₁x+a₃ = bc`. -/
def q7A (b c : ℚ) : ℚ := b * c

/-- Tangent slope at `Q7=(b,bc)`. -/
def q7Lambda (b c : ℚ) : ℚ :=
  (b - c + c ^ 2) / c

/-- Quadratic coefficient after translating Q7 to the origin and removing
its tangent. -/
def q7B (b c : ℚ) : ℚ :=
  2 * b - q7Lambda b c ^ 2 - (1 - c) * q7Lambda b c

/-- New `a₁` before the final scaling. -/
def q7C (b c : ℚ) : ℚ :=
  1 - c + 2 * q7Lambda b c

/-- Tate parameters after normalizing Q7. -/
def q7b (b c : ℚ) : ℚ :=
  -(q7B b c) ^ 3 / (q7A b c) ^ 2

/-- Tate parameters after normalizing Q7. -/
def q7c (b c : ℚ) : ℚ :=
  1 - q7C b c * q7B b c / q7A b c

/-- Universal order-seven parameter. -/
def q7d (b c : ℚ) : ℚ :=
  q7b b c / q7c b c

/-- Universal order-seven curve

`y² + (1+d-d²)xy + (d²-d³)y = x³ + (d²-d³)x²`. -/
def tate7Curve (d : ℚ) : WeierstrassCurve ℚ :=
  ⟨1 + d - d ^ 2, d ^ 2 - d ^ 3, d ^ 2 - d ^ 3, 0, 0⟩

  + d ^ 4 * (d - 1) ^ 2

/-- The standard symmetric N14 polynomial. -/
def n14g (z : ℚ) : ℚ :=
  z ^ 3 + z ^ 2 - 2 * z - 1

/-- Standard affine model of `X₁(2,14)`. -/
def N14EquationStd (u w : ℚ) : Prop :=
  n14g u * w * (w + 1) + n14g w * u * (u + 1) = 0

/-- A strong affine cusp predicate. Proving its negation also handles the
weaker repository convention `u=0 ∨ u=-1`. -/
def N14DegenerateStrong (u : ℚ) : Prop :=
  u = 0 ∨ u = -1 ∨ n14g u = 0
