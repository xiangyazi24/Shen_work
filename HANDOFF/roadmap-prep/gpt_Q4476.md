# Verdict

For Paper 3 uniform persistence on [0,1], the clean execution target is not a pure scalar logistic comparison from the mass equation alone. The robust route is:

1. build an eventual absorbing regularity/boundedness window;

1. prove an eventual positive mass floor using logistic growth plus compactness/regularity, not merely the initial mass Gronwall;

1. convert the mass floor to a pointwise eventual floor by a fixed-window parabolic Harnack / positivity-propagation lemma, or by an equivalent fixed-window weak barrier comparison with uniform constants.

This is mostly independent of the step-3 spectral mode engine. It is a dynamical lower-barrier/compactness input, not a cosine-mode stability calculation. It does share infrastructure with P3.5 compactness/regularization: eventual absorbing estimates and rolling positive-time regularity.

# Why mass Gronwall alone is not enough

The identity

```plain text
m(t) = ∫₀¹ u(t,x) dx,
m'(t) = ∫₀¹ u(t,x) (a - b u(t,x)^α) dx
```

is useful because the chemotaxis flux integrates out under no-flux/Neumann boundary conditions. But the simple lower estimate

```plain text
m'(t) ≥ -C m(t)
```

only gives

```plain text
m(t) ≥ m(0) e^{-Ct},
```

which decays to zero and does not imply uniform persistence as t → ∞.

A pointwise scalar comparison u_t ≥ νu_xx + u(a - b u^α - ...) is also not the clean route, because the chemotaxis term is in divergence form and includes a first-order drift. A scalar logistic ODE lower solution is not automatically a subsolution of the drifted parabolic operator unless the drift is included/matched.

The usable mass argument is instead a uniform non-extinction argument:

```plain text
If eventual regularity is bounded and the total mass is very small,
then the whole profile is uniformly small.
For uniformly small u, the reaction satisfies
  u(a - b u^α) ≥ (a/2) u,
so the mass derivative is positive.
Therefore the mass cannot have arbitrarily small late-time dips.
```

This uses an eventual compactness/regularity estimate to turn small mass into small sup norm, or at least into an estimate that makes ∫u^{α+1} small compared with ∫u.

## 1. Eventual absorbing regularity

This belongs with P3.5, but P3.2 consumes it. A useful abstraction is:

```javascript
structure IntervalDomainEventualAbsorbingRegularity
    (p : CM2Params) : Prop where
  bound :
    ∀ u v : ℝ → intervalDomain.Point → ℝ,
      PositiveGlobalBoundedSolution intervalDomain p u v →
        ∃ T₀ M G : ℝ, 0 < M ∧ 0 ≤ G ∧
          ∀ t, T₀ ≤ t →
            (∀ x : intervalDomain.Point, 0 ≤ u t x ∧ u t x ≤ M) ∧
            -- choose the concrete regularity already available/needed downstream:
            -- C^α, Lipschitz, or C¹; Lipschitz is enough for a mass→sup floor.
            (∀ x y : intervalDomain.Point,
              |u t x - u t y| ≤ G * intervalDomain.dist x y)
```

If intervalDomain.dist is not the actual available metric API, replace that last line with the existing interval lift formulation:

```javascript
∀ x ∈ Set.Icc (0 : ℝ) 1, ∀ y ∈ Set.Icc (0 : ℝ) 1,
  |intervalDomainLift (u t) x - intervalDomainLift (u t) y| ≤ G * |x - y|
```

The exact regularity norm can be weakened/strengthened as convenient. The purpose is only to prevent narrow arbitrarily tall spikes in the mass-to-pointwise conversion.

  -- 3. use small-mass-small-sup: if ∫u is below m*, then sup u ≤ η

  -- 5. HmassId gives m'(t) ≥ (a/2)m(t) whenever m(t)<m*

This is the real “average persistence” lemma. It is not just m(t) ≥ m(0)e^{-Ct}.

## 3. Mass floor to pointwise floor

This is the key P3.2 lower-barrier/Harnack lemma:

```javascript
def IntervalDomainMassToPointwisePersistence
    (p : CM2Params) : Prop :=
  ∀ u v : ℝ → intervalDomain.Point → ℝ,
    PositiveGlobalBoundedSolution intervalDomain p u v →
      (∃ m₀ > 0, ∀ᶠ t in atTop,
        m₀ ≤ intervalDomain.integral (u t)) →
      ∃ δu > 0,
        ∀ᶠ t in atTop, ∀ x : intervalDomain.Point, δu ≤ u t x
```

There are two mathematically equivalent ways to prove this:

### Option A: parabolic Harnack / heat-kernel lower bound

A fixed-window Harnack statement gives, for all late t,

```plain text
inf_x u(t,x) ≥ c_H ∫ u(t-τ,x) dx,
```

with τ > 0 fixed and c_H > 0 depending on the eventual coefficient bounds. Combining with the eventual mass lower bound gives δu = c_H m₀.

This is the cleanest theorem shape but the heaviest PDE infrastructure.

### Option B: matched-divergence weak barrier with uniform constants

This is closer to the Jensen machinery already being built. On each fixed late-time window [s,s+τ], build a nontrivial square-heat seed from the mass lower bound, then compare a drift-matched barrier w to u using the same (w-u)₊ energy argument. Because all coefficients are eventually uniformly bounded, the barrier constants can be chosen uniformly for all late restarts.

This avoids naming a full Harnack theorem, but analytically it is a Harnack/positivity-propagation lemma in disguise. In Lean, this may be cheaper if the Jensen weak comparison infrastructure is already stable.

# Relation to step-3 spectral engine

Uniform persistence should not be implemented as a consequence of the 2×2 cosine-mode stability engine. It only needs:

- positive global bounded solution input;

- eventual absorbing regularity/compactness;

- mass/non-extinction argument;

- parabolic lower-propagation/Harnack or matched-barrier floor;

- elliptic lower bound for v once u is eventually bounded below.

The spectral engine is for threshold/local/global stability. Persistence is a lower-barrier dynamical theorem. It will later feed stability/global-stability by keeping trajectories away from the boundary of the positive cone, but it is not derived from linearized modal stability.

# Single best implementation path

1. IntervalDomainEventualAbsorbingRegularity from P3.5 rolling smoothing / absorbing set.

1. IntervalDomainEventualMassLower using mass derivative + small-mass-small-sup contradiction/barrier.
