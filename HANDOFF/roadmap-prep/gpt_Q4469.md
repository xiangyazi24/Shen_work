# 7. Hardest lemma and honest status

The hardest theorem is

```javascript
noncomputable def raw49Bridge : Raw49Bridge
```

or, in coordinate form,

```javascript
exactOrder49Tate → NoncuspidalPoint X049.
```

It is not a rank computation. It is the explicit identification of the forgetful modular quotient with the chosen Weierstrass model of X₀(49).

There are two valid formalization choices:

- Moduli/Vélu choice: define a rational cyclic 49-isogeny object and prove a classification theorem sending it to 49a1. This is conceptually clean and does not require displaying huge rational functions.

- Coordinate-certificate choice: compute diamond-invariant functions on the raw (b,c) curve, provide explicit numerators and denominators for (x₄₉(b,c),y₄₉(b,c)), and prove the curve equation plus cusp-denominator clauses using field_simp and ring_nf.

The present answer supplies the complete target arithmetic and the compile-checked descent skeleton, but does not fabricate those missing invariant rational functions. Once that one bridge is installed, the final contradiction is less than ten lines.
