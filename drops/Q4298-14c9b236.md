# Definitive verdict

The estimate is mathematically true for every integer `r ≥ 0`:

```text
δ ≤ u ≤ M,   δ > 0,
u ∈ H^r(0,1)
⇒
‖u^γ‖_{H^r} ≤ C(r,γ,M,δ) ‖u‖_{H^r}.
```

There are, however, two important formalization corrections.

1. **The ordinary Sobolev-algebra estimate is not enough.**  Merely using
   `H^r · H^r ⊂ H^r` gives a polynomial estimate such as

   ```text
   ‖u^γ‖_{H^r} ≤ P(‖u‖_{H^r}),
   ```

   not the requested tame estimate, linear in the top Sobolev norm with a constant depending only on `M,δ,γ,r`.  The linear estimate requires a genuine Moser/Gagliardo–Nirenberg product bound.

2. **For the current chemotaxis ladder, do not formalize arbitrary `r` first.**  The presently open `SourceFromSolutionEnvelopePass` is generically indexed by `m`, but the finite ladder described in `IntervalMildPositiveTimeRegularityV6.lean` only needs the power-source outputs through spatial order two.  Thus the paper-core implementation should first prove the explicit `r=1` and `r=2` estimates.  A generic Faà di Bruno theorem is a useful later abstraction, but it is substantially more infrastructure than the current closure needs.

The cleanest current Lean route is therefore:

```text
positive global cosine representative of the slice
→ explicit rpow first/second derivative formulas
→ a Neumann-specific tame L⁴ estimate for uₓ
→ componentwise L² bounds
→ H¹/H² Nemytskii estimate.
```

This avoids abstract weak-Sobolev composition theory entirely.

# 1. Use the correct representative and norm

On the physical interval, the solution slice is represented by its globally smooth cosine synthesis.  Use that global representative `g : ℝ → ℝ`, with

```text
g = intervalDomainLift u   on [0,1].
```

Do **not** differentiate the zero extension `intervalDomainLift u` at `0` or `1`: when the physical endpoint value is positive, that extension is discontinuous there.  The cosine synthesis is the correct globally differentiable object; transfer identities back to the interval by `EqOn` or one-sided derivative congruence.

For formalization, the most convenient Sobolev norm is the equivalent sum norm

```text
N_r(g) := ∑_{j=0}^r ‖D^j g‖_{L²(0,1)}.
```

It avoids repeated square-root algebra.  Since there are only finitely many components,

```text
N_r(g)
```

is equivalent to the usual Hilbert norm

```text
(∑_{j=0}^r ‖D^j g‖₂²)^{1/2}.
```

A componentwise estimate with one common constant proves either convention.

In Lean, set

```lean
abbrev intervalMeasure01 : Measure ℝ :=
  volume.restrict (Set.Ioc (0 : ℝ) 1)

noncomputable def intervalDerivL2 (j : ℕ) (g : ℝ → ℝ) : ℝ :=
  MeasureTheory.lpNorm ((deriv^[j]) g) (2 : ℝ≥0∞) intervalMeasure01

noncomputable def intervalHNormSum (r : ℕ) (g : ℝ → ℝ) : ℝ :=
  ∑ j ∈ Finset.range (r + 1), intervalDerivL2 j g
```

or use explicit fields `g`, `g₁`, `g₂` for the first two orders.  The explicit-field version is less brittle.

# 2. Scalar derivatives and uniform power factors

Let

```text
F(z) = z^γ,   z > 0.
```

For every integer `k ≥ 0`,

```text
F^{(k)}(z) = (γ)ₖ z^{γ-k},
```

where

```text
(γ)ₖ = γ(γ-1)⋯(γ-k+1)
```

is the falling factorial.

For `δ ≤ z ≤ M`, define

```text
K_k(δ,M,γ) := max(δ^{γ-k}, M^{γ-k}).
```

Then


# 3. Exact `r=1` estimate

Assume

```text
0 ≤ u ≤ M,
γ ≥ 1.
```

The value estimate is

```text
|u^γ| = u · u^{γ-1} ≤ M^{γ-1}|u|,
```

so

```text
‖u^γ‖₂ ≤ M^{γ-1} ‖u‖₂.
```

The chain rule gives

```text
D(u^γ) = γu^{γ-1}Du,
```

therefore

```text
‖D(u^γ)‖₂ ≤ γM^{γ-1} ‖Du‖₂.
```

Consequently, for either the sum norm or the square-sum norm,

```text
‖u^γ‖_{H¹}
≤ C₁ ‖u‖_{H¹},

C₁ := M^{γ-1} · max(1,γ).
```

This proof does not use `δ`.

The load-bearing Mathlib lemma is

```lean
HasDerivAt.rpow_const
```

with derivative shape

```lean
f' * γ * (f x)^(γ-1).
```

Because `1 ≤ γ`, its side condition

```lean
f x ≠ 0 ∨ 1 ≤ γ
```

is discharged by the right disjunct even at zeros.

A Lean skeleton is:

```lean
have hpow : HasDerivAt (fun y => u y ^ γ)
    (du x * γ * u x ^ (γ - 1)) x :=
  (hu_deriv x hx).rpow_const (Or.inr hγ)
```

followed by a pointwise absolute-value estimate, `MemLp` multiplication by a bounded coefficient, and `lpNorm` monotonicity.

# 4. `r=2`: the crucial tame estimate

The second derivative is

```text
D²(u^γ)
 = γ(γ-1)u^{γ-2}(Du)²
   + γu^{γ-1}D²u.
```

Set

```text
K₂ := max(δ^{γ-2}, M^{γ-2}).
```

Then

```text
‖D²(u^γ)‖₂
≤ γ|γ-1|K₂ ‖(Du)²‖₂
  + γM^{γ-1} ‖D²u‖₂.
```

The issue is therefore

```text
‖(Du)²‖₂ = ‖Du‖₄².
```

## Why the proposed `H¹ ↪ L∞` argument is not enough

Using only

```text
‖Du‖∞ ≤ C(‖Du‖₂ + ‖D²u‖₂)
```

gives

```text
‖Du‖₄² ≤ ‖Du‖∞ ‖Du‖₂,
```

which is quadratic in the high Sobolev norm.  It proves membership in `H²`, but not the requested tame linear estimate with a constant depending only on `M,δ,γ`.

For the Neumann problem there is a much cleaner identity.

## Neumann-tame `L⁴` lemma

Assume

```text
Du(0)=Du(1)=0,
|u|≤M.
```

Differentiate

```text
u(Du)³:

D[u(Du)³] = (Du)⁴ + 3u(Du)²D²u.
```

The boundary term vanishes, so

```text
∫₀¹ (Du)⁴
 = -3∫₀¹ u(Du)²D²u.
```

Let

```text
A := ∫₀¹ (Du)⁴,
B := ∫₀¹ (D²u)².
```

By `|u|≤M` and Cauchy–Schwarz,

```text
A
≤ 3M ∫₀¹ |(Du)²D²u|,

(∫₀¹ |(Du)²D²u|)² ≤ A B.
```

Thus

```text
A² ≤ 9M²AB.
```

If `A=0` the result is trivial; otherwise cancellation gives

```text
A ≤ 9M²B.
```

Taking square roots yields the exact tame estimate

```text
‖Du‖₄² ≤ 3M ‖D²u‖₂.                 (★)
```

This is the cleanest paper-core lemma.  It is stronger and more useful here than the generic interval `H¹→L∞` estimate.

## Resulting explicit `H²` constant

From `(★)`,

```text
‖D²(u^γ)‖₂
≤ A₂ ‖D²u‖₂,
```

where

```text
A₂ := γM^{γ-1} + 3γ|γ-1|K₂M.
```

Together with the order-zero and order-one estimates,

```text
A₀ := M^{γ-1},
A₁ := γM^{γ-1},
```

we obtain

```text
‖u^γ‖_{H²} ≤ C₂ ‖u‖_{H²},
```

with the explicit safe constant

```text
C₂ := max(A₀, max(A₁,A₂)).
```

For the sum norm one may instead take `A₀+A₁+A₂`; the displayed maximum also works for the usual square-sum norm because it bounds every derivative component separately.

The floor enters only through

```text
K₂ = max(δ^{γ-2},M^{γ-2}).
```

# 5. General integer `r`: exact Faà di Bruno bookkeeping

For `ℓ≥1`, define

```text
P(ℓ,k)
 := { (m₁,…,m_ℓ) : ℕ^ℓ |
      ∑_{j=1}^ℓ m_j = k,
      ∑_{j=1}^ℓ j m_j = ℓ }.
```

Faà di Bruno gives

```text
D^ℓ(u^γ)
 = ∑_{k=1}^ℓ (γ)ₖ u^{γ-k}
     ∑_{m∈P(ℓ,k)}
       [ℓ! / ∏_{j=1}^ℓ (m_j! (j!)^{m_j})]
       ∏_{j=1}^ℓ (D^j u)^{m_j}.          (FDB)
```

The first three orders are:

```text
D(u^γ)
 = γu^{γ-1}u',

D²(u^γ)
 = γ(γ-1)u^{γ-2}(u')² + γu^{γ-1}u'',

D³(u^γ)
 = γ(γ-1)(γ-2)u^{γ-3}(u')³
   + 3γ(γ-1)u^{γ-2}u'u''
   + γu^{γ-1}u'''.
```

## The actual tame product lemma needed

For every partition term in `(FDB)`, with

```text
k = ∑m_j,
∑j m_j = ℓ,
```

one needs

```text
‖∏_{j=1}^ℓ (D^j u)^{m_j}‖₂
≤ T_{ℓ,m}
   (M^{k-1} ‖D^ℓu‖₂ + M^k).             (TP)
```

A standard proof chooses, for each occurrence of `D^j u`,

```text
p_j = 2ℓ/j.
```

Then

```text
∑ occurrences 1/p_j
 = (1/(2ℓ))∑j m_j
 = 1/2,
```

so Hölder closes in `L²`.  The one-dimensional Gagliardo–Nirenberg estimate is

```text
‖D^j u‖_{L^{2ℓ/j}}
≤ G_{ℓ,j}
   (M^{1-j/ℓ} ‖D^ℓu‖₂^{j/ℓ} + M),
        1≤j<ℓ.                            (GN)
```

Multiplying the leading terms in `(GN)` gives exactly

```text
M^{k-1} ‖D^ℓu‖₂
```

because the derivative orders sum to `ℓ`.  Mixed choices are reduced to the same linear top-order term plus `M^k` by weighted Young inequalities.  This proves `(TP)`.

Since the interval has length one and `u≥δ`,

```text
‖u‖₂ ≥ δ.
```

Therefore the lower-order remainder is absorbed linearly:

```text
M^k ≤ (M^k/δ) ‖u‖₂ ≤ (M^k/δ) ‖u‖_{H^ℓ}.
```

This is exactly where the positive floor can also absorb bounded-domain lower-order terms in a generic Gagliardo–Nirenberg theorem.

## An explicit general constant

Put

```text
K_k := max(δ^{γ-k},M^{γ-k}),

b_{ℓ,m} := ℓ! / ∏_{j=1}^ℓ (m_j!(j!)^{m_j}).
```

A valid component constant is

```text
C_0 := M^{γ-1},

C_ℓ := |(γ)₁|K₁
       + ∑_{k=2}^ℓ |(γ)ₖ|K_k
           ∑_{m∈P(ℓ,k)} b_{ℓ,m}T_{ℓ,m}
             (M^{k-1}+M^k/δ).
```

Then

```text
‖D^ℓ(u^γ)‖₂ ≤ C_ℓ ‖u‖_{H^ℓ}.
```

Finally one may take

```text
C(r,γ,M,δ) := max_{0≤ℓ≤r} C_ℓ
```

for the componentwise/square-sum norm, or `∑C_ℓ` for the sum norm.

This proves the requested general estimate.
