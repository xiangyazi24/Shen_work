## 1. Exact scalar mathematics

Put

```text
C := χ₀ μ Θ_{β-1},
B := b + C,
δ := min (m - 1) α,
s := min 1 (a / B),
θ := s^(1/δ).
```

From `1 < m` and `0 < α`, one has `0 < δ`.  From `0 < b`, `0 < χ₀`, `0 < μ`, and `1 ≤ β`, the repository lemma

```lean
Theta_beta_pos_of_nonneg
```

gives `0 < Θ_{β-1}`, hence `0 < C` and `0 < B`.

For `0 < y ≤ 1`,

```text
δ ≤ α,
δ ≤ m - 1,
1 + δ ≤ 1 + α,
1 + δ ≤ m.
```

On the unit interval, real powers decrease as the exponent increases, so

```text
y^(1+α) ≤ y^(1+δ),
y^m     ≤ y^(1+δ).
```

Consequently,

```text
b y^(1+α) + C y^m ≤ (b+C) y^(1+δ),
```

and therefore

```text
a y - B y^(1+δ) ≤ generalMLogisticRhs p y.
```

The exact Lean lemma is:

```lean
theorem generalM_logistic_dominates_single_power_on_unit
    {p : CM2Params} (hm : 1 < p.m) (hχ0 : 0 < p.χ₀)
    (hβ : 1 ≤ p.β) {y : ℝ} (hy : 0 < y) (hy1 : y ≤ 1) :
    p.a * y - generalMScalarLossCoeff p *
        y ^ (1 + generalMScalarDelta p) ≤
      generalMLogisticRhs p y := by
  have hδα : generalMScalarDelta p ≤ p.α := min_le_right _ _
  have hδm : generalMScalarDelta p ≤ p.m - 1 := min_le_left _ _
  have hpowα : y ^ (1 + p.α) ≤ y ^ (1 + generalMScalarDelta p) :=
    Real.rpow_le_rpow_of_exponent_ge hy hy1 (by linarith)
  have hpowm : y ^ p.m ≤ y ^ (1 + generalMScalarDelta p) :=
    Real.rpow_le_rpow_of_exponent_ge hy hy1 (by linarith)
  have hTheta : 0 < Theta_beta (p.β - 1) :=
    Theta_beta_pos_of_nonneg (by linarith)
  have hC : 0 ≤ generalMChemLoss p :=
    (mul_pos (mul_pos hχ0 p.hμ) hTheta).le
  have hbpow := mul_le_mul_of_nonneg_left hpowα p.hb
  have hCpow := mul_le_mul_of_nonneg_left hpowm hC
  simp only [generalMScalarLossCoeff, generalMLogisticRhs] at ⊢
  nlinarith
```

### Why the global logistic theorem cannot be called

For `y > 1`, the power monotonicity reverses:

```text
y^(1+α) ≥ y^(1+δ),
y^m     ≥ y^(1+δ).
```

Thus above one the faithful field is generally **below**, not above, the single-power logistic field.  Therefore

```lean
RightLowerDiniGE z (generalMLogisticRhs p) (Set.Ioi 0)
```

does not imply the global hypothesis expected by

```lean
strict_logistic_subsolution_le_of_RightLowerDiniGE
```

for the auxiliary field `a*y - B*y^(1+δ)`.  Making that inference would strengthen the hypothesis falsely.

A piecewise transformed field is not shorter here: the existing Bernoulli/logistic comparison theorem requires the pure logistic field globally, while a sound transformed field would have to change above one and would lose that API.  The shortest sound repair is to localize the first-contact proof and guarantee that the **comparison orbit**, hence every possible contact value, stays below one.

### Where `min 1` comes from

The unrestricted single-power logistic equilibrium would be

```text
(a / B)^(1/δ).
```

The field domination is available only up to level one.  Therefore cap the equilibrium base at

```text
s := min 1 (a / B)
```

and use

```text
θ := s^(1/δ).
```

Since `0 < s ≤ 1` and `0 < 1/δ`, one has `0 < θ ≤ 1`, and

```text
θ^δ = s.
```

This is exactly the source of the displayed `min 1`.

For each strict subthreshold `0 < ℓ < θ`, define a strict logistic carrier by

```text
aℓ := B * ℓ^δ,
η  := a - aℓ.
```

Because `ℓ^δ < θ^δ = s ≤ a/B`, one has `η > 0`.  Its carrying capacity is exactly

```text
(aℓ / B)^(1/δ) = ℓ.
```

Start it at

```text
y0 := min (z T0) ℓ.
```

Then `0 < y0 ≤ ℓ`, and the existing lemma

```lean
bernoulliLogisticSolution_le_max_of_nonneg_time
```

keeps the carrier below `ℓ ≤ 1` for all forward times.  At a first contact, `z x = Y x`, so the local domination is invoked only at `Y x ∈ (0,1]`.  The strict loss `η * Y x > 0` supplies the strict boundary inequality needed by the existing first-crossing engine.

This proves every `ℓ < θ` is below the liminf.  A final `ℓ ↑ θ` argument gives the exact threshold.

## 3. Reciprocal exponent identity and exact threshold: complete code

The real-power normalization is obtained by the positive-order identity

```text
1 / min x y = max (1/x) (1/y),    x>0, y>0.
```

The exact Lean proof is the two-case order split requested:

```lean
theorem generalMScalar_inv_delta {p : CM2Params} (hm : 1 < p.m) :
    1 / generalMScalarDelta p =
      max (1 / (p.m - 1)) (1 / p.α) := by
  by_cases hle : p.m - 1 ≤ p.α
  · have hinv : 1 / p.α ≤ 1 / (p.m - 1) :=
      one_div_le_one_div_of_le (sub_pos.mpr hm) hle
    rw [generalMScalarDelta, min_eq_left hle, max_eq_left hinv]
  · have hle' : p.α ≤ p.m - 1 := le_of_not_ge hle
    have hinv : 1 / (p.m - 1) ≤ 1 / p.α :=
      one_div_le_one_div_of_le p.hα hle'
    rw [generalMScalarDelta, min_eq_right hle', max_eq_right hinv]
```

## 4. Eventual comparison and liminf assumptions

The exact eventual/liminf chain is:

1. For every `t ≥ T0`, apply
   `strict_logistic_subsolution_le_of_local_RightLowerDiniGE` on `[T0,t]`.
2. Package the result with `eventually_ge_atTop T0` to obtain
   `∀ᶠ t in atTop, Y t ≤ z t`.

4. Compose with `bernoulliLogisticSolution_tendsto_atTop`.
5. Apply `le_liminf_of_eventually_tendsto_lower`.
