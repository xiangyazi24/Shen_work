# Verdict

The apparent requirement

```text
p₀ > γ
```

is **not** the Paper 2 critical-branch requirement. It is an artifact of the repository's current coarse cross-diffusion estimate

```text
rho = 2 * γ,
```

which comes from squaring an `L²` resolver-source bound and produces a `p + 2γ` moment. Under only

```lean
hχ : p.χ₀ < chiBeta p
```

the authors cannot, in general, choose a directly bounded exponent `p₀ > γ`. For `γ * N ≥ 2`, the critical hypothesis guarantees only

```text
max 1 (γ*N/2) < (2*β - 1)/χ₀,
```

not

```text
γ*N < (2*β - 1)/χ₀.
```

So retaining `rho = 2γ` would require a strictly stronger sensitivity assumption than Theorem 1.2.

The actual Chen–Ruau–Shen mechanism uses **both** of the following:

1. a direct finite-`p` energy estimate which proves `LpPowerBoundedBefore` for every

   ```text
   1 < p < (2*β - 1)/χ₀;
   ```

2. a sharper weighted elliptic estimate giving

   ```text
   rho = γ,
   ```

   rather than `rho = 2γ`.

Then the critical inequality

```text
χ₀ < 2 * (2*β - 1) / max 2 (γ*N)
```

is exactly what makes the interval

```text
max 1 (γ*N/2) < p₀ < (2*β - 1)/χ₀
```

nonempty. On the unit interval, where `N = 1`, choose

```text
max 1 (γ/2) < p₀ < (2*β - 1)/χ₀.
```

Thus for large `γ` the correct seed may lie between `γ/2` and `γ`; no `γ < 2` assumption and no `p₀ > γ` seed are needed.

This is the argument in Chen–Ruau–Shen, arXiv:2512.14858, §4.3, especially equations (4.6)–(4.13).

# 1. Why the current `L²` seed does not close the critical branch

The current concrete estimate in

```text
ShenWork/Paper2/IntervalDomainCrossDiffusionBootstrap.lean
```

ends in the theorem

```lean
intervalDomain_crossDiffusionEnergyTerm_bootstrap_bound
```

with the high moment

```lean
intervalDomain.integral
  (fun x => (u t x) ^ (pExp + 2 * p.γ)).
```

At the statement layer this is

```lean
CrossDiffusionBootstrapEstimate intervalDomain p T (2 * p.γ) u v.
```

The abstract bootstrap threshold is

```lean
p₀ > max 1 (rho * (p.N : ℝ) / 2).
```

Substituting `rho = 2γ` gives

```text
p₀ > max 1 (γ*N).
```

For the interval this becomes `p₀ > γ`. The existing `p₀ = 2` seed therefore works only when `γ < 2`.

This is a real mismatch, not a missing inequality between constants. The proof of `intervalDomain_crossDiffusionEnergyTerm_bootstrap_bound` structurally passes through

```lean
intervalDomain_sourceL2_mul_power_le
```

and the product

```text
(∫ u^(2γ)) * (∫ u^p) ≤ ∫ u^(p+2γ).
```

No rearrangement of that proof changes `2γ` to `γ`.

Moreover, the critical condition does not generally provide a direct exponent above `γ*N`. Put

```text
B := 2*β - 1,
M := max 2 (γ*N).
```

The hypothesis is

```text
χ₀ < 2*B/M.
```

When `γ*N ≥ 2`, this gives

```text
B/χ₀ > γ*N/2,
```

but it need not give

```text
B/χ₀ > γ*N.
```

The latter would require the stronger condition

```text
χ₀ < B/(γ*N).
```

Therefore option (a), if interpreted as “prove a direct seed with `p₀ > γ*N` while keeping the current `rho = 2γ`,” cannot follow from Theorem 1.2's stated hypothesis.

# 2. What the paper actually does

Assume the attractive subcase

```text
0 < χ₀,
m = 1,
β ≥ 1.
```

The nonpositive case is already handled by the negative/nonpositive-sensitivity theorem and should be split off rather than forced through this critical bootstrap.

## Step 1: direct boundedness at every subcritical finite exponent

For every exponent `q` satisfying

```text
1 < q
q * χ₀ < 2*β - 1,
```

the authors test the `u`-equation by `u^(q-1)`. They deliberately do not rely on logistic damping: the theorem is valid for every `b ≥ 0`, so the proof may discard the nonpositive term `-b*u^(q+α)`.

Using the weighted elliptic identity for `v`, with the signal exponent replaced by

```text
2*β - 1,
```

together with the paper's Lemma 2.5 and the strict inequality

```text
q * χ₀ < 2*β - 1,
```

the chemotaxis contribution is absorbed into the diffusion term. Lemma 4.1 then converts the remaining gradient control into coercivity of the `q`-moment. The resulting estimate has the form

```text
(1/q) * d/dt ∫ u^q
  ≤ - ∫ u^q + C_q * (∫ u)^q.                    (★)
```

On a fixed finite horizon `[0,T)`, the mass is bounded. Indeed, after integrating the PDE in space, the diffusion and divergence terms vanish and

```text
M'(t) = a*M(t) - b*∫u^(1+α) ≤ a*M(t).
```

Hence

```text
M(t) ≤ exp(a*T) * M(0)
```

for `0 < t < T`. Applying scalar Gronwall to (★) gives

```lean
LpPowerBoundedBefore intervalDomain q T u.
```

The important point is that this is available for **every**

```text
1 < q < (2*β - 1)/χ₀.
```

It is not merely an `L²` estimate.

## Step 2: the cross-diffusion increment is `γ`, not `2γ`

For an arbitrary Moser exponent `p > 1`, after the first spatial Young inequality the difficult factor is

```text
∫ u^p * |∇v|² / (1+v)^(2β).
```

Set

```text
q := (p + γ)/γ.
```

Then `q > 1`, `γ*q = p + γ`, and a second Young inequality gives

```text
u^p * A
  ≤ ε*u^(p+γ) + C_ε*A^q,

A := |∇v|² / (1+v)^(2β).
```

Now

```text
A^q = |∇v|^(2q) / (1+v)^(2β*q).
```

Apply the weighted gradient estimate with its beta parameter set to

```text
β' := 2*β - 1.
```

Because

```text
1 + β' = 2*β,
```

its denominator is exactly `(1+v)^(2β*q)`, and its right side is

```text
∫ u^(γ*q) = ∫ u^(p+γ).
```

Consequently

```text
crossDiffusionEnergyTerm(p)
  ≤ ε * weightedGradient(p)
      + C_ε * ∫u^(p+γ).
```

This is precisely

```lean
CrossDiffusionBootstrapEstimate intervalDomain p T p.γ u v.
```

Thus the paper uses

```text
rho = γ.
```

There is no need for a still smaller `rho`; `γ` already matches the critical constant exactly.

## Step 3: choose the seed exponent

Let

```text
L := max 1 (γ*N/2),
U := (2*β - 1)/χ₀.
```

For `χ₀ > 0`, the critical hypothesis

```text
χ₀ < 2*(2*β - 1)/max 2 (γ*N)
```

is equivalent to

```text
L < U.
```

Choose, for example,

```text
p₀ := (L + U)/2.
```

Then

```text
max 1 (γ*N/2) < p₀
p₀ * χ₀ < 2*β - 1.
```

Step 1 supplies

```lean
LpPowerBoundedBefore intervalDomain p₀ T u,
```

and Step 2 supplies

```lean
CrossDiffusionBootstrapEstimate intervalDomain p T p.γ u v.
```

This is exactly the bootstrap package expected by `IntervalDomainTheorem12`.

# 3. Evaluation of the proposed options

## Option (a): a direct higher-`p` absorbing inequality

**Yes, but with the correct range and role.**

The paper proves the direct finite-`p` estimate for all

```text
1 < p < (2*β - 1)/χ₀.
```

The decisive estimate is stronger than a generic

```text
Y' ≤ C*Y + C'.
```

It has a dissipative form like

```text
(1/p)Y' ≤ -Y + C_p*M(t)^p.
```

But this does not, under the critical hypothesis alone, necessarily reach `p > γ*N`. It reaches an exponent above `γ*N/2`, which is sufficient only after the cross estimate is sharpened to `rho = γ`.

So (a) is a necessary half of the paper's solution, but it does not rescue the coarse `rho = 2γ` route.

## Option (b): use a lower `rho`

**Yes. The correct paper value is `rho = γ`.**

The current `rho = 2γ` estimate is not sharp enough. The replacement should use the existing statement-layer

```lean
WeightedGradientEstimate
```

with exponent `q = (p+γ)/γ` and beta parameter `2β-1`.

There is no need to search for `rho < γ`; the theorem's constant was designed around `rho = γ`.

## Option (c): the actual Theorem 1.2 mechanism

**The actual mechanism is (a) plus (b).**

It is not:

```text
L² seed → repeatedly bootstrap until above γ.
```

It is:

```text
subcritical direct L^p estimate
  + weighted elliptic cross estimate with rho=γ
  + one exponent selected in the overlap interval
  → Corollary 2.1 finite-p bootstrap
  → Proposition 2.5 L∞.
```

# 5. Recommended Lean decomposition

The clean implementation should add three analytic producers and one short scalar/wiring lemma.

## A. Paper-faithful direct seed family

First define the exact finite-`p` output:

```lean
def CriticalM1SubcriticalLpSeed
    (p : CM2Params) (T : ℝ)
    (u : ℝ → intervalDomain.Point → ℝ) : Prop :=
  ∀ pExp : ℝ,
    1 < pExp →
    pExp * p.χ₀ < 2 * p.β - 1 →
      LpPowerBoundedBefore intervalDomain pExp T u
```

The main base-seed theorem should have the shape:

```lean
theorem intervalDomain_criticalM1_subcriticalLpSeed
    (p : CM2Params)
    (hβ : 1 ≤ p.β)
    (hm : p.m = 1)
    (hχpos : 0 < p.χ₀)
    {u₀ : intervalDomain.Point → ℝ}
    (hu₀ : PositiveInitialDatum intervalDomain u₀)
    {T : ℝ} (hT : 0 < T)
    {u v : ℝ → intervalDomain.Point → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (htrace : InitialTrace intervalDomain u₀ u) :
    CriticalM1SubcriticalLpSeed p T u
```

`htrace` is useful for controlling the initial moment and mass, though the exact low-level producer may instead take those bounds explicitly.

### The genuinely new core inside this theorem

Package the paper's equations (4.6)–(4.10) as a differential estimate:

```lean
theorem intervalDomain_criticalM1_subcriticalLp_differential
    (p : CM2Params)
    (hβ : 1 ≤ p.β)
    (hm : p.m = 1)
    (hχpos : 0 < p.χ₀)
    {T pExp : ℝ}
    {u v : ℝ → intervalDomain.Point → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hp : 1 < pExp)
    (hsub : pExp * p.χ₀ < 2 * p.β - 1) :
    ∃ C : ℝ, 0 ≤ C ∧
      ∀ t, 0 < t → t < T →
        (1 / pExp) *
            deriv
              (fun τ =>
                intervalDomain.integral
                  (fun x => (u τ x) ^ pExp)) t
          ≤ -intervalDomain.integral
                (fun x => (u t x) ^ pExp)
            + C * (intervalDomain.integral (u t)) ^ pExp
```

This theorem should consume:

1. the existing classical `L^p` time-chain/PDE/IBP infrastructure from `IntervalDomainEnergyStep`;
2. a new interval version of the paper's weighted elliptic identity (1.27), specialized to `m=1` and signal exponent `2β-1`;
3. the algebraic Lemma 2.5 estimate that turns `pExp * χ₀ < 2β-1` into a strictly positive absorption margin;
4. a concrete interval implementation of Lemma 4.1, naturally expressed through `LpMassGradientInterpolationEstimate`;
5. the existing mass derivative identity and scalar Gronwall.

The wrapper from this differential estimate to `CriticalM1SubcriticalLpSeed` is routine.

## B. Sharp `rho = γ` cross estimate

Isolate the weighted-gradient input actually required from Proposition 2.2:

```lean
def CriticalM1WeightedGradientFamily
    (p : CM2Params) (T : ℝ)
    (u v : ℝ → intervalDomain.Point → ℝ) : Prop :=
  ∀ q : ℝ, 1 < q →
    ∃ Mstar > 0,
      WeightedGradientEstimate
        intervalDomain q (2 * p.β - 1) p.γ Mstar T u v
```

Then prove:

```lean
theorem intervalDomain_crossDiffusionBootstrap_rho_gamma
    (p : CM2Params)
    (hβ : 1 ≤ p.β)
    (hm : p.m = 1)
    {T : ℝ}
    {u v : ℝ → intervalDomain.Point → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hWG : CriticalM1WeightedGradientFamily p T u v) :
    CrossDiffusionBootstrapEstimate
      intervalDomain p T p.γ u v
```

The proof is the two-Young argument above with

```lean
q := (pExp + p.γ) / p.γ.
```

This theorem should be **new**, rather than modifying

```lean
intervalDomain_crossDiffusionEnergyTerm_bootstrap_bound.
```

The old theorem remains useful elsewhere as a coarse `rho=2γ` route.

## C. Scalar exponent-selection lemma

This part is elementary and should be proved independently of the PDE:

```lean
theorem exists_criticalM1_seed_exponent
    (p : CM2Params)
    (hβ : 1 ≤ p.β)
    (hχpos : 0 < p.χ₀)
    (hcrit : p.χ₀ < chiBeta p) :
    ∃ p0 : ℝ,
      max 1 (p.γ * (p.N : ℝ) / 2) < p0 ∧
      p0 * p.χ₀ < 2 * p.β - 1
```

A stable proof sets

```lean
L := max 1 (p.γ * (p.N : ℝ) / 2)
U := (2 * p.β - 1) / p.χ₀
p0 := (L + U) / 2.
```

The only nontrivial scalar rewrite is

```text
max 1 (a/2) = max 2 a / 2
```

for `a ≥ 0`. Then `hcrit` gives `L < U`.

If the interval theorem explicitly carries `p.N = 1`, add the specialization:

```lean
theorem exists_criticalM1_seed_exponent_interval
    ...
    (hN : p.N = 1) :
    ∃ p0,
      max 1 (p.γ / 2) < p0 ∧
      p0 * p.χ₀ < 2 * p.β - 1
```

## D. Final critical bootstrap packet

The final producer matching `IntervalDomainTheorem12.hcriticalBootstrap` is then short:

```lean
theorem intervalDomain_criticalM1_bootstrap
    (p : CM2Params)
    (hβ : 1 ≤ p.β)
    (hm : p.m = 1)
    (hχpos : 0 < p.χ₀)
    (hcrit : p.χ₀ < chiBeta p)
    {u₀ : intervalDomain.Point → ℝ}
    (hu₀ : PositiveInitialDatum intervalDomain u₀)
    {T : ℝ} (hT : 0 < T)
    {u v : ℝ → intervalDomain.Point → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (htrace : InitialTrace intervalDomain u₀ u)
    (hSeed : CriticalM1SubcriticalLpSeed p T u)
    (hCross :
      CrossDiffusionBootstrapEstimate intervalDomain p T p.γ u v) :
    ∃ rho > 0,
      CrossDiffusionBootstrapEstimate intervalDomain p T rho u v ∧
        ∃ p0 > max 1 (rho * (p.N : ℝ) / 2),
          LpPowerBoundedBefore intervalDomain p0 T u := by
  obtain ⟨p0, hp0, hsub⟩ :=
    exists_criticalM1_seed_exponent p hβ hχpos hcrit
  refine ⟨p.γ, p.hγ, hCross, p0, ?_, ?_⟩
  · simpa using hp0
  · have hp0_one : 1 < p0 :=
      lt_of_le_of_lt (le_max_left _ _) hp0
    exact hSeed p0 hp0_one hsub
```

Once the atomic producers are available, omit `hSeed` and `hCross` as arguments and construct them internally.

# 6. Recommended build order

The lowest-risk execution order is:

```text
1. Prove the scalar exponent-selection lemma.

2. Prove the sharp cross estimate
     WeightedGradientEstimate at beta' = 2β-1
       → CrossDiffusionBootstrapEstimate with rho = γ.

3. Prove the subcritical finite-p differential inequality
     q*χ₀ < 2β-1
       → (1/q)Yq' ≤ -Yq + C*M^q.

4. Use the existing mass identity + fixed-horizon Gronwall
       → CriticalM1SubcriticalLpSeed.

5. Select p₀ in the overlap interval and assemble the exact
   `hcriticalBootstrap` packet.
```

Step 2 and Step 3 share weighted elliptic machinery but are logically distinct:

- Step 2 is the Proposition 2.2 route producing the `p+γ` cross moment.
- Step 3 is the special `m=1` subcritical energy argument producing the seed range `pχ₀<2β-1`.

Do not try to derive Step 3 from the current `rho=2γ` `CrossDiffusionBootstrapEstimate`; that would reintroduce the bad threshold.

# 7. Exact answer to the large-`γ` question

For `γ ≥ 2`, the base seed is **not** obtained by upgrading the existing `L²` estimate until it crosses `γ`.

Instead:

```text
choose p₀ with
  max 1 (γ/2) < p₀ < (2β-1)/χ₀;
```

prove `LpPowerBoundedBefore intervalDomain p₀ T u` directly by the special `m=1` subcritical energy inequality; and feed it to Corollary 2.1 together with the sharper

```lean
CrossDiffusionBootstrapEstimate intervalDomain p T p.γ u v.
```

Therefore the exact new base-seed leaf is:

```lean
CriticalM1SubcriticalLpSeed p T u
```

or equivalently its concrete producer

```lean
intervalDomain_criticalM1_subcriticalLpSeed.
```

The genuinely new analytic core is not another `L²` theorem. It is the paper-faithful weighted elliptic absorption estimate yielding finite-`p` bounds throughout

```text
1 < p < (2β-1)/χ₀,
```

paired with the Proposition 2.2 weighted-gradient estimate that lowers the cross-bootstrap increment from `2γ` to `γ`.

# Bottom line

```text
Current repo path:
  L² seed + rho=2γ
  → needs p₀>γN
```
