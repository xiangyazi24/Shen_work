# `--help`

`--help` is the conventional command-line flag for displaying a program's usage information without running its main operation.

```bash
shen2 --help
```

For help on a subcommand, use:

```bash
shen2 <subcommand> --help
```

A help screen commonly shows the command syntax, positional arguments, available options, subcommands, defaults, and examples. The exact output depends on the installed `shen2` program and version.
