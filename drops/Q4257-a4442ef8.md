# Q4257 — minimal frozen-drift positivity interface

## Verdict

The frozen-coefficient gauge route works, but with one important correction:

```lean
-- Correct drift-removing gauge if w = u * φ:
φ x = Real.exp ((1 / (2 * ν)) * ∫ ξ in 0..x, b₀ ξ)
```

Equivalently, if you insist on

```lean
φ x = Real.exp (-(1 / (2 * ν)) * ∫ ξ in 0..x, b₀ ξ)
```

then the drift-removing variable is

```lean
w = u / φ
```

not `w = u * φ`.

With the sign as written in the question, `w = u * exp(-∫b/(2ν))` does **not** remove the first-order drift. It doubles it. The standard Liouville reduction giving

```text
c̃₀ = c₀ - b₀²/(4ν) - b₀'/2
```

uses `w = u * exp(+∫b₀/(2ν))`.

## Gauge computation

Let

```text
B(x) = ∫₀ˣ b₀(ξ) dξ
φ(x) = exp(B(x)/(2ν))
w(t,x) = φ(x) u(t,x).
```

Then

```text
φ'/φ = b₀/(2ν).
```

Write

```text
q(x) = φ'(x)/φ(x) = b₀(x)/(2ν).
```

Then

```text
w_t = φ u_t
w_x = φ (u_x + q u)
w_xx = φ (u_xx + 2q u_x + (q' + q²)u).
```

Since

```text
u_t = ν u_xx + b₀ u_x + c₀ u,
```

we get

```text
w_t - ν w_xx
  = φ[(b₀ - 2νq)u_x + (c₀ - ν(q' + q²))u].
```

By the choice `q=b₀/(2ν)`, the drift term vanishes:

```text
b₀ - 2νq = 0.
```

Also

```text
ν(q' + q²)
  = ν(b₀'/(2ν) + b₀²/(4ν²))
  = b₀'/2 + b₀²/(4ν).
```

Therefore

```text
w_t = ν w_xx + c̃₀ w
```

with

```lean
cTilde x = c₀ x - deriv b₀ x / 2 - (b₀ x)^2 / (4 * ν)
```

or, if you have a named derivative `db₀`:

```lean
def cTilde (x : ℝ) : ℝ :=
  c₀ x - db₀ x / 2 - (b₀ x)^2 / (4 * ν)
```

No `b_t` appears because `b₀` is frozen/autonomous. If you try the same gauge with a genuinely time-dependent `b(t,x)`, then

```text
φ_t/φ = (1/(2ν)) ∫₀ˣ ∂ₜb(t,ξ)dξ
```

appears in the zero-order coefficient. That requires control of `b_t`, which in chemotaxis depends on `u_t` through the resolver. This is exactly why the frozen-coefficient route is preferable as a minimal interface.

## Boundary conditions

Assume Neumann for `u`:

```text
u_x(t,0)=0,
u_x(t,1)=0.
```

Then

```text
w_x = φ(u_x + b₀ u/(2ν)).
```

So if

```text
b₀(0)=0,
b₀(1)=0,
```

then

```text
w_x(t,0)=φ(0)u_x(t,0)=0,
w_x(t,1)=φ(1)u_x(t,1)=0.
```

Thus the gauge preserves homogeneous Neumann boundary conditions **only under the endpoint drift condition**

```lean
hb₀_left  : b₀ 0 = 0
hb₀_right : b₀ 1 = 0
```

For the chemotaxis drift, this means you need the physical Neumann condition for the signal:

```text
R_x(t,0)=R_x(t,1)=0
```

because

```text
b = -χ₀ R_x/(1+R)^β.
```

Then `b₀(0)=b₀(1)=0` follows from `R_x=0` at the endpoints and denominator positivity.

## Positivity step for the driftless potential equation

After the gauge, the problem is:

```text
w_t = ν w_xx + c̃₀(x) w,
∂ₓw(t,0)=∂ₓw(t,1)=0,
w(t₀,·) ≥ 0,
w(t₀,·) ≢ 0.
```

Assume a lower bound

```lean
hctilde_lb : ∀ x ∈ Set.Icc (0:ℝ) 1, -C ≤ cTilde x
```

with `0 ≤ C`. Define

```text
v(t,x) = exp(C*(t-t₀)) w(t,x).
```

Then

```text
v_t = ν v_xx + (c̃₀ + C)v,
```

and

```text
c̃₀ + C ≥ 0.
```

Since `w≥0` iff `v≥0`, and `u≥0` is already known, the source term `(c̃₀+C)v` is nonnegative.

The minimal comparison lemma you need is the mild/variation-of-constants form:

```lean
theorem driftless_nonnegative_potential_lower_by_heat
    {v : ℝ → intervalDomainPoint → ℝ}
    {t₀ t C : ℝ}
    (ht : t₀ < t)
    (hv_nonneg : ∀ τ ∈ Set.Icc t₀ t, ∀ x, 0 ≤ v τ x)
    (hd_nonneg : ∀ x ∈ Set.Icc (0:ℝ) 1, 0 ≤ d x)
    (hv_mild :
      ∀ τ ∈ Set.Icc t₀ t, ∀ x,
        v τ x = intervalFullSemigroupOperator (ν * (τ - t₀))
          (intervalDomainLift (v t₀)) x
        + ∫ s in t₀..τ,
            intervalFullSemigroupOperator (ν * (τ - s))
              (fun y => d y * intervalDomainLift (v s) y) x)
    (hheat_pos_preserve_or_nonneg_kernel : ... ) :
    ∀ x ∈ Set.Icc (0:ℝ) 1,
      intervalFullSemigroupOperator (ν * (t - t₀))
        (intervalDomainLift (v t₀)) x ≤ v t x := by
  -- integral term is nonnegative by kernel nonnegativity, d≥0, v≥0
```

Then combine it with the already-formalized heat-kernel strict positivity:

```lean
theorem heat_kernel_strict_pos
    {f : intervalDomainPoint → ℝ} {σ : ℝ}
    (hσ : 0 < σ)
    (hf_nonneg : ∀ x, 0 ≤ f x)
    (hf_nontriv : ∃ x, 0 < f x) :
    ∀ x : intervalDomainPoint,
      0 < intervalFullSemigroupOperator σ (intervalDomainLift f) x
```

Thus

```text
v(t,x) ≥ Sν(t-t₀)v(t₀)(x) > 0.
```

Hence

```text
v(t,x)>0,
w(t,x)=exp(-C(t-t₀))v(t,x)>0,
u(t,x)=w(t,x)/φ(x)>0.
```

The gauge factor is strictly positive everywhere:

```lean
hφ_pos : ∀ x, 0 < φ x := fun x => Real.exp_pos _
```

so strict positivity is preserved both ways.

## Minimal formalization interface

I would split the implementation into two lemmas.

### 1. Gauge calculation lemma

```lean
noncomputable def frozenGauge
    (ν : ℝ) (b₀ : ℝ → ℝ) (x : ℝ) : ℝ :=
  Real.exp ((1 / (2 * ν)) * ∫ ξ in (0:ℝ)..x, b₀ ξ)

noncomputable def frozenCTilde
    (ν : ℝ) (b₀ db₀ c₀ : ℝ → ℝ) (x : ℝ) : ℝ :=
  c₀ x - db₀ x / 2 - (b₀ x)^2 / (4 * ν)

theorem frozen_gauge_equation
    {ν : ℝ} (hν : 0 < ν)
    {b₀ db₀ c₀ : ℝ → ℝ}
    (hb_deriv : ∀ x ∈ Set.Ioo (0:ℝ) 1, HasDerivAt b₀ (db₀ x) x)
    -- regularity of integral/gauge; can also assume directly:
    (hB_deriv : ∀ x ∈ Set.Ioo (0:ℝ) 1,
      HasDerivAt (fun z => ∫ ξ in (0:ℝ)..z, b₀ ξ) (b₀ x) x)
    {u : ℝ → ℝ → ℝ} {t₀ t : ℝ}
    (hu_pde : ∀ τ ∈ Set.Ioo t₀ t, ∀ x ∈ Set.Ioo (0:ℝ) 1,
      HasDerivAt (fun r => u r x)
        (ν * deriv (fun z => deriv (fun y => u τ y) z) x
          + b₀ x * deriv (fun y => u τ y) x
          + c₀ x * u τ x) τ)
    -- plus spatial C² enough to justify w_xx calculation
    :
    let φ := frozenGauge ν b₀
    let w := fun τ x => φ x * u τ x
    ∀ τ ∈ Set.Ioo t₀ t, ∀ x ∈ Set.Ioo (0:ℝ) 1,
      -- pointwise PDE for w:
      HasDerivAt (fun r => w r x)
        (ν * deriv (fun z => deriv (fun y => w τ y) z) x
          + frozenCTilde ν b₀ db₀ c₀ x * w τ x) τ := by
  -- pure calculus/ring after proving φ'/φ=b/(2ν)
```

For Lean, it may be simpler to assume the gauge PDE as a field in a structure rather than prove this calculus immediately. The calculus lemma can be proved later.

### 2. Positivity theorem from the gauged equation

This is the useful local interface:

```lean
theorem neumann_frozen_drift_positivity
    {ν t₀ t C : ℝ} (hν : 0 < ν) (htt : t₀ < t) (hC : 0 ≤ C)
    {b₀ db₀ c₀ : ℝ → ℝ}
    (hb_left : b₀ 0 = 0) (hb_right : b₀ 1 = 0)
    (hctilde_lb : ∀ x ∈ Set.Icc (0:ℝ) 1,
      -C ≤ frozenCTilde ν b₀ db₀ c₀ x)
    {u : ℝ → intervalDomainPoint → ℝ}
    (hu_nonneg : ∀ τ ∈ Set.Icc t₀ t, ∀ x, 0 ≤ u τ x)
    (hu0_nonneg : ∀ x, 0 ≤ u t₀ x)
    (hu0_nontriv : ∃ x, 0 < u t₀ x)
    -- gauge equation/mild representation for w after the transform:
    (hgauged_mild :
      let φ := fun x : intervalDomainPoint => frozenGauge ν b₀ x.1
      let w := fun τ x => φ x * u τ x
      let v := fun τ x => Real.exp (C * (τ - t₀)) * w τ x
      ∀ τ ∈ Set.Icc t₀ t, ∀ x,
        v τ x =
          intervalFullSemigroupOperator (ν * (τ - t₀))
            (intervalDomainLift (v t₀)) x.1
          + ∫ s in t₀..τ,
              intervalFullSemigroupOperator (ν * (τ - s))
                (fun y =>
                  (frozenCTilde ν b₀ db₀ c₀ y + C) *
                    intervalDomainLift (v s) y) x.1)
    (hheat_strict_pos :
      ∀ {σ : ℝ} (hσ : 0 < σ) {f : intervalDomainPoint → ℝ},
        (∀ x, 0 ≤ f x) → (∃ x, 0 < f x) →
          ∀ x, 0 < intervalFullSemigroupOperator (ν * σ)
            (intervalDomainLift f) x.1)
    (hheat_nonneg :
      -- semigroup positivity preserving, needed for nonnegative Duhamel source
      ... ) :
    ∀ x : intervalDomainPoint, 0 < u t x := by
  classical
  let φ : intervalDomainPoint → ℝ := fun x => frozenGauge ν b₀ x.1
  let w : ℝ → intervalDomainPoint → ℝ := fun τ x => φ x * u τ x
  let v : ℝ → intervalDomainPoint → ℝ :=
    fun τ x => Real.exp (C * (τ - t₀)) * w τ x

  have hφ_pos : ∀ x, 0 < φ x := by
    intro x
    exact Real.exp_pos _

  have hv0_nonneg : ∀ x, 0 ≤ v t₀ x := by
    intro x
    dsimp [v, w, φ]
    have : Real.exp (C * (t₀ - t₀)) = 1 := by ring_nf; simp
    positivity

  have hv0_nontriv : ∃ x, 0 < v t₀ x := by
    rcases hu0_nontriv with ⟨x, hx⟩
    refine ⟨x, ?_⟩
    dsimp [v, w, φ]
    have hφx := hφ_pos x
    positivity

  have hheat_pos : ∀ x,
      0 < intervalFullSemigroupOperator (ν * (t - t₀))
        (intervalDomainLift (v t₀)) x.1 := by
    intro x
    have hσ : 0 < t - t₀ := sub_pos.mpr htt
    exact hheat_strict_pos hσ hv0_nonneg hv0_nontriv x

  have hsource_nonneg : ∀ s ∈ Set.Icc t₀ t, ∀ y,
      0 ≤ (frozenCTilde ν b₀ db₀ c₀ y + C) * intervalDomainLift (v s) y := by
    intro s hs y
    -- Use hctilde_lb on y∈[0,1]; outside intervalDomainLift(v s) y = 0.
    by_cases hy : y ∈ Set.Icc (0:ℝ) 1
    · have hd : 0 ≤ frozenCTilde ν b₀ db₀ c₀ y + C := by
        have h := hctilde_lb y hy
        linarith
      have hv : 0 ≤ intervalDomainLift (v s) y := by
        -- from hu_nonneg and φ>0, exp>0
        simp [intervalDomainLift, hy, v, w, φ]
        positivity
      exact mul_nonneg hd hv
    · simp [intervalDomainLift, hy]

  have hv_lower :
      ∀ x, intervalFullSemigroupOperator (ν * (t - t₀))
          (intervalDomainLift (v t₀)) x.1 ≤ v t x := by
    intro x
    -- expand hgauged_mild at τ=t; Duhamel term is nonnegative by hheat_nonneg and hsource_nonneg
    sorry

  intro x
  have hv_pos : 0 < v t x := lt_of_lt_of_le (hheat_pos x) (hv_lower x)
  dsimp [v, w, φ] at hv_pos
  have hpos_factor : 0 < Real.exp (C * (t - t₀)) * φ x := by
    positivity
  exact (mul_pos_iff_of_pos_left hpos_factor).mp hv_pos
```

The `sorry` above is the comparison-to-heat part. It should be factored as its own lemma, because it is reusable and much easier than a full drifted maximum principle.

## The comparison lemma to isolate

```lean
theorem neumann_heat_lower_of_nonnegative_duhamel_source
    {ν t₀ t : ℝ} (hν : 0 < ν) (htt : t₀ ≤ t)
    {v : ℝ → intervalDomainPoint → ℝ}
    {q : ℝ → ℝ → ℝ}
    (hv_mild : ∀ x,
      v t x = intervalFullSemigroupOperator (ν * (t - t₀))
          (intervalDomainLift (v t₀)) x.1
        + ∫ s in t₀..t,
            intervalFullSemigroupOperator (ν * (t - s)) (q s) x.1)
    (hq_nonneg : ∀ s ∈ Set.Icc t₀ t, ∀ y, 0 ≤ q s y)
    (hheat_nonneg : ∀ {σ : ℝ}, 0 ≤ σ → ∀ f,
      (∀ y, 0 ≤ f y) → ∀ x,
        0 ≤ intervalFullSemigroupOperator (ν * σ) f x) :
    ∀ x,
      intervalFullSemigroupOperator (ν * (t - t₀))
          (intervalDomainLift (v t₀)) x.1 ≤ v t x := by
  intro x
  rw [hv_mild x]
  have hduh_nonneg :
      0 ≤ ∫ s in t₀..t,
            intervalFullSemigroupOperator (ν * (t - s)) (q s) x.1 := by
    apply intervalIntegral.integral_nonneg htt
    intro s hs
    exact hheat_nonneg (by nlinarith [hs.2]) (q s) (hq_nonneg s hs) x.1
  linarith
```

This is the main lemma needed after the gauge. It uses only positivity preservation of the driftless Neumann heat semigroup, not strict positivity.

Strict positivity enters only through the initial heat leg.

## Conditions needed for chemotaxis

For the chemotaxis drift

```text
b(t,x) = -χ₀ R_x(t,x)/(1+R(t,x))^β,
```

the frozen route requires, at the chosen restart time `t₀`:

1. `b₀(x)=b(t₀,x)` is bounded and has bounded spatial derivative.
2. `b₀(0)=b₀(1)=0`. This follows from Neumann `R_x(t₀,0)=R_x(t₀,1)=0`.
3. `c₀` is bounded below after adding the Liouville correction:

```text
c̃₀ = c₀ - b₀'/2 - b₀²/(4ν).
```

4. The solution on the frozen local problem really satisfies the autonomous frozen-coefficient PDE or at least the corresponding mild identity for the gauged variable.

Point 4 is important. The original chemotaxis solution has time-dependent drift `b(t,x)`, not frozen `b₀(x)`. The frozen-coefficient theorem applies directly only to a comparison/evolution built with frozen coefficients, or to a short-window approximation if you can control the residual

```text
(b(t,x)-b₀(x)) u_x + (c(t,x)-c₀(x)) u.
```

For a strict-positivity producer, the cleanest use is:

- prove the gauge positivity theorem for the autonomous frozen operator;
- then use it as a local comparison kernel if you have a mild identity with nonnegative residual, or a residual that can be absorbed into the zero-order lower bound.

If the residual has no sign, freezing alone is not a complete proof for the original time-dependent drift. In that case, either control the time-dependent gauge directly, paying a `b_t` term, or introduce a comparison theorem that handles the residual.

## Final recommendation

Formalize the following minimal stack:

1. `frozen_gauge_equation`: pure calculus, with the corrected positive-sign gauge.
2. `neumann_heat_lower_of_nonnegative_duhamel_source`: driftless heat comparison for nonnegative Duhamel source.
3. `neumann_driftless_potential_strict_pos`: positivity for `w_t=νw_xx+c̃w`, `c̃≥-C`.
4. `neumann_frozen_drift_positivity`: combine gauge + driftless potential positivity.

The exact Liouville coefficient is:

```text
c̃₀ = c₀ - b₀'/2 - b₀²/(4ν).
```

The correct gauge is:

```text
w = u * exp(+(1/(2ν))∫₀ˣ b₀).
```

The Neumann boundary condition is preserved if and only if the frozen drift vanishes at the endpoints:

```text
b₀(0)=b₀(1)=0.
```

For the actual chemotaxis drift, the frozen route is viable only after handling the mismatch between `b(t,x)` and `b(t₀,x)`. The autonomous frozen theorem itself is clean and formalization-friendly; using it for the non-autonomous chemotaxis equation still needs either a residual comparison or a time-dependent gauge with enough `b_t` control.
