# Problem 2.7: the canonical non-circular Hermite–Padé kernel, and the missing level-11 certificate

Let

\[
L=\zeta(2)+\zeta(3),
\qquad
\mathcal W[R]
=\int_0^1\!\int_0^1
R(xy)\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy.
\]

## Executive answer

There is a completely explicit, non-circular Hermite–Padé polynomial system naturally attached to the functional \(\mathcal W\). Its type-II polynomials are determined by the two positive measures

\[
\boxed{
 d\mu_2(t)=(-\log t)\,dt,
 \qquad
 d\mu_3(t)=\frac12\log^2t\,dt,
 \qquad 0<t<1.
}
\tag{1}
\]

For a multi-index \(\mathbf n=(n_2,n_3)\), the monic polynomial \(Q_{\mathbf n}\) of degree \(N=n_2+n_3\) is characterized by

\[
\int_0^1 t^kQ_{\mathbf n}(t)\,d\mu_2(t)=0
\quad(0\le k<n_2),
\tag{2}
\]

\[
\int_0^1 t^kQ_{\mathbf n}(t)\,d\mu_3(t)=0
\quad(0\le k<n_3).
\tag{3}
\]

It has an explicit determinant formula with rational entries, given below. On the step line

\[
(n_2,n_3)=(0,0),(1,0),(1,1),(2,1),(2,2),\ldots,
\tag{4}
\]

these polynomials satisfy a four-term recurrence. Moreover,

\[
\boxed{
Q_{\mathbf n}(1)L-P_{2,\mathbf n}(1)-P_{3,\mathbf n}(1)
=\mathcal W[Q_{\mathbf n}],
}
\tag{5}
\]

where the two numerator polynomials are defined independently by Hermite–Padé approximation. Thus

\[
\boxed{R_n(x,y)=Q_{\mathbf n(n)}(xy)}
\tag{6}
\]

is a genuine non-circular kernel for a simultaneous approximation system to \(\zeta(2)\) and \(\zeta(3)\).

However, and this is the load-bearing point, **no exact identification of this canonical system with the printed Problem 2.7 recurrence is presently known**. The level-11 spectral identity fixes the asymptotic cubic, but it does not fix the finite recurrence coefficients, the extension class, or the initial framing. The directly centered-binomial-transformed Cooper sequence has already been checked and does not satisfy the challenge recurrence.

Accordingly:

- the determinant below is the strongest explicit non-circular candidate available from the mixed moment functional itself;
- it is not yet legitimate to assert that \(Q_{\mathbf n}(1)=q_n\) for the challenge;
- there is no published level-11 Rodrigues formula or Almkvist–Zeilberger certificate for the displayed degree-12 operator;
- the known Dauguet–Zudilin simultaneous construction is an important model, but it produces a different hypergeometric system.

A complete proof now reduces to one finite operator comparison: derive the step-line recurrence of the explicit polynomials below and test whether it is related to the challenge operator by a shift, scalar hypergeometric gauge, rational pullback, or a finite Ore intertwiner.

# 1. The two Markov functions are exactly the dilogarithm and trilogarithm

For \(|z|>1\), put

\[
F_r(z)=\int_0^1\frac{(-\log t)^{r-1}}{(r-1)!(z-t)}\,dt
\qquad(r=2,3).
\tag{7}
\]

Expanding \((z-t)^{-1}\) gives

\[
F_r(z)
=\sum_{m=0}^{\infty}\frac{z^{-m-1}}{(m+1)^r}
=\boxed{\operatorname{Li}_r(1/z)}.
\tag{8}
\]

Hence, by continuation from \(z>1\),

\[
F_2(1)=\zeta(2),
\qquad
F_3(1)=\zeta(3).
\tag{9}
\]

The moments are rational:

\[
\boxed{
\mu_{2,m}:=\int_0^1t^m\,d\mu_2(t)=\frac1{(m+1)^2},
\qquad
\mu_{3,m}:=\int_0^1t^m\,d\mu_3(t)=\frac1{(m+1)^3}.
}
\tag{10}
\]

This is the precise two-measure system asked for. The denominator polynomial in a simultaneous Padé approximation is type II. The dual type-I system exists as well, but the common denominator \(q_n\) belongs naturally to type II.

The system is nondegenerate. Indeed

\[
\frac{d\mu_3}{d\mu_2}(t)=-\frac12\log t,
\]

and the functions

\[
1,t,\ldots,t^{n_2-1},
\quad
(-\log t),t(-\log t),\ldots,t^{n_3-1}(-\log t)
\]

form a confluent Müntz/Chebyshev system on \((0,1)\). Thus all step-line multi-indices are normal and the monic type-II polynomial is unique.

# 2. Exact determinant formula for the non-circular polynomial

Let \(N=n_2+n_3\). Define the \(N\times N\) moment matrix

\[
\mathcal M_{\mathbf n}
=
\begin{pmatrix}
\left(\dfrac1{(r+c+1)^2}\right)_{\substack{0\le r<n_2\\0\le c<N}}\\[2mm]
\left(\dfrac1{(r+c+1)^3}\right)_{\substack{0\le r<n_3\\0\le c<N}}
\end{pmatrix},
\qquad
\Delta_{\mathbf n}=\det\mathcal M_{\mathbf n}.
\tag{11}
\]

Then

\[
\boxed{
Q_{\mathbf n}(t)=\frac1{\Delta_{\mathbf n}}
\det\begin{pmatrix}
\left(\dfrac1{(r+c+1)^2}\right)_{\substack{0\le r<n_2\\0\le c\le N}}\\[2mm]
\left(\dfrac1{(r+c+1)^3}\right)_{\substack{0\le r<n_3\\0\le c\le N}}\\[2mm]
1&t&t^2&\cdots&t^N
\end{pmatrix}.
}
\tag{12}
\]

The cofactor of \(t^N\) is \(\Delta_{\mathbf n}\), so the polynomial is monic. Equations (2)–(3) follow by replacing the final row with the corresponding moment row.

The first step-line polynomials are

\[
Q_0(t)=1,
\tag{13}
\]

\[
\boxed{Q_1(t)=t-\frac14,}
\tag{14}
\]

\[
\boxed{Q_2(t)=t^2-\frac{16}{27}t+\frac1{27},}
\tag{15}
\]

and

\[
\boxed{
Q_3(t)=t^3-\frac{4779}{4600}t^2
+\frac{1107}{4600}t-rac{133}{18400}.
}
\tag{16}
\]

These formulas are independent of the challenge recurrence and therefore genuinely non-circular.

# 3. Exact Hermite–Padé numerator and error identity

For any polynomial \(Q\), define

\[
P_{r,Q}(z)
=\int_0^1\frac{Q(z)-Q(t)}{z-t}\,d\mu_r(t),
\qquad r=2,3.
\tag{17}
\]

The quotient is polynomial in \(z\), so \(P_{r,Q}\in\mathbb Q[z]\) whenever \(Q\in\mathbb Q[z]\). The exact remainder identity is

\[
\boxed{
Q(z)F_r(z)-P_{r,Q}(z)
=\int_0^1\frac{Q(t)}{z-t}\,d\mu_r(t).
}
\tag{18}
\]

For \(Q=Q_{\mathbf n}\), the orthogonality makes the right side vanish to the Hermite–Padé order prescribed by \(\mathbf n\) at infinity.

At \(z=1\), summing the two identities gives

\[
\begin{aligned}
&Q_{\mathbf n}(1)(\zeta(2)+\zeta(3))
-P_{2,Q_{\mathbf n}}(1)-P_{3,Q_{\mathbf n}}(1)\\
&\quad=
\int_0^1\frac{Q_{\mathbf n}(t)}{1-t}
\left(-\log t+\frac12\log^2t\right)dt.
\end{aligned}
\tag{19}
\]

Using

\[
\int_0^1\!\int_0^1 f(xy)\,dx\,dy
=\int_0^1(-\log t)f(t)\,dt,
\tag{20}
\]

this becomes

\[
\boxed{
\begin{aligned}
&Q_{\mathbf n}(1)L
-P_{2,Q_{\mathbf n}}(1)-P_{3,Q_{\mathbf n}}(1)\\
&\qquad=
\int_0^1\!\int_0^1
Q_{\mathbf n}(xy)
\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy.
\end{aligned}}
\tag{21}
\]

If

\[
Q(t)=\sum_{j=0}^{N}c_jt^j,
\]

then the rational numerator is especially simple:

\[
\boxed{
P_{2,Q}(1)+P_{3,Q}(1)
=\sum_{j=0}^{N}c_j\bigl(H_j^{(2)}+H_j^{(3)}\bigr),
}
\tag{22}
\]

with \(H_0^{(r)}=0\). Formula (22) is an exact rational-arithmetic test for every proposed kernel.

# 4. Why a four-term recurrence appears automatically

For a perfect system of two measures, the monic step-line polynomials satisfy

\[
\boxed{
tQ_n(t)=Q_{n+1}(t)+\beta_nQ_n(t)
+\gamma_nQ_{n-1}(t)+\delta_nQ_{n-2}(t).
}
\tag{23}
\]

The two second-kind remainders

\[
\mathcal R_{r,n}(z)
=Q_n(z)F_r(z)-P_{r,Q_n}(z)
\]

satisfy the same recurrence. At \(z=1\), therefore,

\[
\boxed{
E_{n+1}=(1-\beta_n)E_n-\gamma_nE_{n-1}-\delta_nE_{n-2},
}
\tag{24}
\]

where

\[
E_n=Q_n(1)L-P_{2,Q_n}(1)-P_{3,Q_n}(1).
\]

The denominator values \(Q_n(1)\) and each numerator value satisfy (24) as well. This gives the structural explanation for an order-three scalar recurrence: two measures produce a four-term step-line recurrence.

To identify the challenge, one must prove that, for some shift and gauge \(h_n\),

\[
q_n^{\rm challenge}=h_nQ_{n+n_0}(1)
\tag{25}
\]

and

\[
p_n^{\rm challenge}=h_n
\bigl(P_{2,Q_{n+n_0}}(1)+P_{3,Q_{n+n_0}}(1)\bigr).
\tag{26}
\]

This is an exact, finite test. It has not yet been passed.

If the monic MOP recurrence is written

\[
v_{n+3}=a_nv_{n+2}+b_nv_{n+1}+c_nv_n
\tag{27}
\]

and \(u_n=h_nv_n\), with \(r_n=h_{n+1}/h_n\), then

\[
\begin{aligned}
u_{n+3}={}&a_nr_{n+2}u_{n+2}
+b_nr_{n+1}r_{n+2}u_{n+1}\\
&+c_nr_nr_{n+1}r_{n+2}u_n.
\end{aligned}
\tag{28}
\]

Thus the first coefficient determines \(r_{n+2}\); the other two give exact compatibility tests. This is the quickest way to rule in or rule out a scalar hypergeometric gauge.

# 5. What can honestly be called a Rodrigues formula here

There is no published classical scalar Rodrigues formula for the **challenge-specific** level-11 kernel. In particular, the Cooper level-11 sequence

\[
1,4,28,268,3004,36784,\ldots
\]

is modular and satisfies a four-term recurrence, but no binomial-sum or Beukers polynomial formula is known for it. Cooper explicitly emphasizes that passing from a recurrence to a binomial representation has no general algorithm.

For the direct logarithmic two-measure system, formula (12) is the rigorous explicit replacement. It is a confluent Cauchy/Jacobi moment determinant. Indeed,

\[
\frac1{(m+1)^2}
=-\left.\frac{\partial}{\partial\alpha}
\frac1{m+1+\alpha}\right|_{\alpha=0},
\tag{29}
\]

\[
\frac1{(m+1)^3}
=\left.\frac12\frac{\partial^2}{\partial\alpha^2}
\frac1{m+1+\alpha}\right|_{\alpha=0}.
\tag{30}
\]

Consequently (12) is a confluent parameter-derivative of a Cauchy/Jacobi moment determinant.

Equivalently, introduce

\[
w_{2,\varepsilon}(t)=\frac{1-t^\varepsilon}{\varepsilon},
\qquad
w_{3,\varepsilon}(t)=
\frac{1-2t^\varepsilon+t^{2\varepsilon}}{2\varepsilon^2}.
\tag{31}
\]

Then

\[
w_{2,\varepsilon}(t)\to-\log t,
\qquad
w_{3,\varepsilon}(t)\to\frac12\log^2t.
\tag{32}
\]

The determinant (12) is the confluent \(\varepsilon\to0\) limit of the corresponding Jacobi-power moment problem. This is a genuine Rodrigues-type **confluent construction**, but it must not be confused with an identified level-11 Rodrigues formula.

Any more explicit nested-derivative formula for the challenge would be a new result.

# 6. Where level 11 would have to enter

Cooper's level-11 homogeneous period is parameterized by

\[
Z(\tau)=
\left(\sum_{j,k\in\mathbb Z}q^{j^2+jk+3k^2}\right)^2,
\qquad
X(\tau)=\frac{\eta(\tau)^2\eta(11\tau)^2}{Z(\tau)}.
\tag{33}
\]

Its singular polynomial is

\[
C_{11}(t)=t^3-20t^2+56t-44.
\tag{34}
\]

The challenge cubic

\[
P_{27}(\mu)=4\mu^3-220\mu^2+8\mu-1
\]

satisfies

\[
\boxed{
16P_{27}\!\left(\frac{(t-2)^2}{4}\right)
=-C_{11}(t)C_{11}(4-t).
}
\tag{35}
\]

At the generating-function level, the centered-binomial/even-section map uses

\[
s=\frac{\sqrt z}{16},
\qquad
X_+=\frac{s}{1+2s},
\qquad
X_-=-\frac{s}{1-2s},
\tag{36}
\]

or, inversely,

\[
\boxed{z=\frac{256X^2}{(1-2X)^2}.}
\tag{37}
\]

This produces an explicit transformed **homogeneous** level-11 period. It does not produce the normal-function extension carrying \(\zeta(3)\), nor does it determine a real polynomial measure on \([0,1]\).

Therefore a true level-11 Hermite–Padé solution would require two new ingredients:

1. a pair of level-11 Markov/Eichler functions \(F_2^{(11)},F_3^{(11)}\) whose connection values are \(\zeta(2),\zeta(3)\) or their required linear combination;
2. type-II polynomials for their measures or cycles whose step-line recurrence is Ore-equivalent to the challenge operator.

Because the two recessive modes form a nonreal conjugate pair, the relevant representation may use complex cycles or a modular normal function rather than two positive measures on one real interval. A fixed positive Beukers phase cannot by itself encode the full level-11 Stokes/connection data.

# 7. Known hypergeometric precedent: Dauguet–Zudilin

Dauguet and Zudilin constructed simultaneous approximations

\[
r_n=q_n\zeta(2)-p_n,
\qquad
\widehat r_n=q_n\zeta(3)-\widehat p_n
\]

with the **same** denominator \(q_n\). Their construction uses explicit Mellin–Barnes/hypergeometric rational functions, a Whipple transformation to identify the denominators, and Gosper–Zeilberger creative telescoping to obtain an order-three recurrence.

A specialized weight-two rational term has the form

\[
R_n^{(2)}(t)=
\frac{(t+1)_{8n}(t+n+1)_{6n}(t+2n+1)_{8n}}
{(8n)!^2(t+9n+1)_{6n+1}},
\tag{38}
\]

while a weight-three companion is

\[
\widehat R_n^{(3)}(t)=
\frac{(2t+11n+2)_{5n}}{(5n)!}
\frac{(t+1)_{8n}}{(8n)!}
\frac{(7n)!}{(t+9n+1)_{7n+1}}
\frac{(6n)!}{(t+10n+1)_{6n+1}}.
\tag{39}
\]

This proves that the requested kind of simultaneous certificate exists in principle. It does **not** identify the Problem 2.7 recurrence. Searches using the distinctive coefficients \(946\), \(104060\), and the endpoint quadratics find no published occurrence, and no Ore equality with the Dauguet–Zudilin recurrence has been established.

Thus the answer to ask (4) is:

\[
\boxed{
\text{There is a known AZ certificate for a different simultaneous }
\zeta(2),\zeta(3)\text{ system, but none known for this recurrence.}
}
\]

References:

- S. Dauguet and W. Zudilin, *On simultaneous Diophantine approximations to \(\zeta(2)\) and \(\zeta(3)\)*, J. Number Theory 145 (2014), 362–387, arXiv:1401.5322.
- S. Cooper, *Apéry-like sequences defined by four-term recurrence relations*, arXiv:2302.00757.

# 8. Exact Sage computation for the canonical candidate

The following code constructs the non-circular determinant polynomials, their numerators, and their four-term recurrence. It then tests the printed challenge recurrence directly.

```python
from sage.all import *

R.<t> = PolynomialRing(QQ)


def moment(weight, m):
    """weight=2 or 3."""
    return QQ(1) / (m + 1)**weight


def step_index(N):
    """(0,0),(1,0),(1,1),(2,1),(2,2),..."""
    return ((N + 1)//2, N//2)


def Q_ab(n2, n3):
    """Monic type-II polynomial for (-log t)dt and log(t)^2/2 dt."""
    N = n2 + n3
    if N == 0:
        return R(1)

    rows = []
    rhs = []

    # Unknowns are c_0,...,c_{N-1}; coefficient of t^N is 1.
    for k in range(n2):
        rows.append([moment(2, k+j) for j in range(N)])
        rhs.append(-moment(2, k+N))

    for k in range(n3):
        rows.append([moment(3, k+j) for j in range(N)])
        rhs.append(-moment(3, k+N))

    M = matrix(QQ, rows)
    c = M.solve_right(vector(QQ, rhs))
    Q = t**N + sum(c[j]*t**j for j in range(N))

    # Exact orthogonality checks.
    for k in range(n2):
        assert sum(Q[j]*moment(2, k+j) for j in range(N+1)) == 0
    for k in range(n3):
        assert sum(Q[j]*moment(3, k+j) for j in range(N+1)) == 0
    return Q


def harmonic(m, r):
    return sum(QQ(1)/k**r for k in range(1, m+1))


def numerator_at_one(Q):
    return sum(Q[j]*(harmonic(j, 2) + harmonic(j, 3))
               for j in range(Q.degree()+1))


def coeff_vector(P, length):
    return vector(QQ, [P[j] for j in range(length)])


def step_recurrence(Qs, n):
    """
    Solve t Q_n = Q_{n+1} + beta Q_n + gamma Q_{n-1} + delta Q_{n-2}.
    Valid for n>=2.
    """
    target = t*Qs[n] - Qs[n+1]
    length = n + 2
    cols = [
        coeff_vector(Qs[n],   length),
        coeff_vector(Qs[n-1], length),
        coeff_vector(Qs[n-2], length),
    ]
    M = matrix(QQ, cols).transpose()
    beta, gamma, delta = M.solve_right(coeff_vector(target, length))
    assert target == beta*Qs[n] + gamma*Qs[n-1] + delta*Qs[n-2]
    return beta, gamma, delta


MAXN = 14
Qs = [Q_ab(*step_index(N)) for N in range(MAXN+1)]

assert Qs[0] == 1
assert Qs[1] == t - QQ(1)/4
assert Qs[2] == t**2 - QQ(16)/27*t + QQ(1)/27
assert Qs[3] == (t**3 - QQ(4779)/4600*t**2
                 + QQ(1107)/4600*t - QQ(133)/18400)

q_mop = [Q(1) for Q in Qs]
p_mop = [numerator_at_one(Q) for Q in Qs]

for N in range(4):
    print(N, step_index(N), Qs[N], q_mop[N], p_mop[N])

for n in range(2, MAXN):
    print('rec', n, step_recurrence(Qs, n))


# Printed Problem 2.7 coefficients, with the alternating-sign convention
# A_n u_{n+3} - B_n u_{n+2} + C_n u_{n+1} - D_n u_n = 0.
def A(n):
    return (1024*(2*n+5)**4*(2*n+7)**3*(2*n+9)**3
            *(946*n**2+6407*n+10860))


def B(n):
    return (128*(2*n+7)**3*(2*n+9)**3
            *(104060*n**6+1745370*n**5+12145238*n**4
              +44886481*n**3+92943995*n**2
              +102256019*n+46709052))


def C(n):
    return (16*(n+3)**4*(2*n+9)**3
            *(3784*n**5+57792*n**4+351019*n**3
              +1059230*n**2+1587211*n+944620))


def D(n):
    return ((n+3)**4*(n+4)**6
            *(946*n**2+4515*n+5399))


def residual(seq, n):
    return (A(n)*seq[n+3] - B(n)*seq[n+2]
            + C(n)*seq[n+1] - D(n)*seq[n])

print('direct MOP residuals:')
for n in range(min(8, MAXN-2)):
    print(n, residual(q_mop, n))
```

A nonzero residual rules out the unpulled system immediately. A positive result would still require checking the numerator values and the initial normalization. If the direct test fails, the next search should deform the Markov functions by the level-11 modular pullback rather than increasing the degree of an unrelated polynomial ansatz.

# 9. Final status

The requested non-circular construction is now precise:

\[
\boxed{
Q_n(t)=Q_{\lceil n/2\rceil,\lfloor n/2\rfloor}(t)
}
\]

with \(Q_{a,b}\) given by the rational determinant (12). It is the canonical type-II Hermite–Padé denominator for

\[
\operatorname{Li}_2(1/z),\qquad \operatorname{Li}_3(1/z),
\]

and it gives the exact mixed error integral (21).

What remains unproved—and should not be blurred with the construction itself—is

\[
\boxed{
Q_n(1)\stackrel{?}{=}h_n^{-1}q_n^{\rm challenge}
}
\]

for some explicit gauge, together with the analogous numerator identity. No published level-11 Rodrigues formula or AZ certificate currently closes this comparison. The spectral norm identity identifies the correct modular field and asymptotic covering, but the missing object is still the level-11 normal-function/Ore certificate.
