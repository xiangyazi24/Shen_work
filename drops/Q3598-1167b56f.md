# Q3598 audit: general-χ boundary min-point residual

## 1. Can `BoundaryMinPersistenceWindowBound p` be proved now for all `p.χ₀ ≤ 0`?

I do **not** see a direct axiom-clean proof from the currently landed endpoint lemmas.

What exists now:

* `BFormPositiveDatumLocal.boundaryMinPersistenceWindowBound_chiZero` in `IntervalBFormPositiveDatumQuantWiring.lean` proves the full windowed residual when `p.χ₀ = 0` by dispatching to:
  * `MinPersistenceAtoms.hbdry_left_chi0` in `IntervalDomainBoundaryHbound.lean`,
  * `MinPersistenceAtoms.hbdry_right_chi0` in `IntervalDomainBoundaryHboundRight.lean`.
* `MinPersistenceAtoms.min_point_estimate_interior` in `IntervalDomainMinPointInterior.lean` proves the general-χ min-point lower slope **at interior argmins**, using:
  * `MinPersistenceAtoms.chemDiv_at_critical` from `IntervalDomainChemDivCritical.lean`, and
  * `MinPersistenceAtoms.flux_coeff_bound` from `IntervalDomainFluxCoeffBound.lean`.

The gap is the boundary analogue of the interior chemotaxis-divergence input.  The existing chi-zero boundary proofs deliberately avoid it: after `p.χ₀ = 0`, the PDE interior limit is just `uₓₓ = u_t - reaction`, so the one-sided second-derivative boundary test plus reaction algebra is enough.  For `p.χ₀ ≤ 0`, the same proof would need a one-sided endpoint control of the chemotaxis contribution in the interior PDE limit:

```text
u_t = u_xx - χ₀ * chemDiv + reaction.
```

The interior proof already has the correct sign algebra: if at the min point

```text
chemDiv = m * G,   |G| ≤ K₁,   m ≥ 0,   χ₀ ≤ 0,
```

then

```text
-χ₀ * chemDiv = |χ₀| * m * G ≥ - |χ₀| * K₁ * m.
```

That sign step is exactly the `hcd_lb` step in `MinPersistenceAtoms.min_point_estimate`.  What is not present is an endpoint/one-sided theorem proving the corresponding `chemDiv = m * G` and `|G| ≤ fluxCoeffConst ...` in the boundary limit.

So the honest answer is: **do not try to prove `BoundaryMinPersistenceWindowBound p` for general `χ₀ ≤ 0` directly yet**.  Introduce a minimal endpoint residual split and let the hQuant routes consume the combined provider.

## 2. Minimal honest residual to introduce

The cleanest small residual is a left/right endpoint split of the already-corrected windowed boundary statement.  It should live in:

```text
ShenWork/Paper2/IntervalBFormPositiveDatumQuantWiring.lean
```

inside:

```lean
namespace ShenWork.Paper2.BFormPositiveDatumLocal
```

right after `BoundaryMinPersistenceWindowBound`, or right after `boundaryMinPersistenceWindowBound_chiZero` if you want to keep compatibility declarations together.

This residual is honest because it names exactly the missing boundary analytic estimates without pretending to have endpoint chemDiv continuity.  It is also useful because future analytic work can prove left and right independently, just like `hbdry_left_chi0` and `hbdry_right_chi0`.

```lean
/-- Left endpoint windowed boundary min-point derivative residual.  This is the
left half of `BoundaryMinPersistenceWindowBound`; future analytic work should
prove it by taking the one-sided interior PDE limit at `0` and bounding the
chemotaxis-divergence coefficient. -/
def BoundaryMinPersistenceWindowLeftBound (p : CM2Params) : Prop :=
  ∀ {u₀ : intervalDomainPoint → ℝ},
    PositiveInitialDatum intervalDomain u₀ →
    ∀ {M : ℝ}, 0 < M → (∀ x, |u₀ x| ≤ M) →
    ∀ {t₁ T : ℝ} {u v : ℝ → intervalDomainPoint → ℝ}, 0 < t₁ →
      IsPaper2ClassicalSolution intervalDomain p T u v →
      InitialTrace intervalDomain u₀ u →
      ∀ s ∈ Set.Ico (t₁ / 2) T,
        intervalDomainLift (u s) 0 =
          sInf (intervalDomainLift (u s) '' Set.Icc (0 : ℝ) 1) →
        -(|p.χ₀| * ShenWork.MinPersistenceAtoms.fluxCoeffConst p.β
              (p.ν * (SupNormBridge.regimeBound p M) ^ p.γ)
            + p.b * (SupNormBridge.regimeBound p M) ^ p.α)
            * sInf (intervalDomainLift (u s) '' Set.Icc (0 : ℝ) 1) ≤
          deriv (fun r => intervalDomainLift (u r) 0) s

/-- Right endpoint windowed boundary min-point derivative residual. -/

/-- Combined endpoint provider for the corrected windowed boundary residual. -/
structure BoundaryMinPersistenceWindowEndpointBounds (p : CM2Params) : Prop where
  left : BoundaryMinPersistenceWindowLeftBound p
  right : BoundaryMinPersistenceWindowRightBound p

/-- Left/right endpoint residuals imply the combined windowed boundary residual. -/
theorem boundaryMinPersistenceWindowBound_of_endpointBounds
    {p : CM2Params}
    (h : BoundaryMinPersistenceWindowEndpointBounds p) :
    BoundaryMinPersistenceWindowBound p := by
  intro u₀ hu₀ M hM hbnd t₁ T u v ht₁ hsol htr s hs ys hys hys01 harg
  rcases hys01 with h0 | h1
  · subst h0
    exact h.left hu₀ hM hbnd ht₁ hsol htr s hs harg
  · subst h1
    exact h.right hu₀ hM hbnd ht₁ hsol htr s hs harg

/-- Old combined windowed residual packaged as endpoint bounds.  Optional
compatibility helper. -/
theorem endpointBounds_of_boundaryMinPersistenceWindowBound
    {p : CM2Params}
    (h : BoundaryMinPersistenceWindowBound p) :
    BoundaryMinPersistenceWindowEndpointBounds p where
  left := by
    intro u₀ hu₀ M hM hbnd t₁ T u v ht₁ hsol htr s hs harg
    exact h hu₀ hM hbnd ht₁ hsol htr s hs 0
      (by constructor <;> norm_num) (Or.inl rfl) harg
  right := by
    intro u₀ hu₀ M hM hbnd t₁ T u v ht₁ hsol htr s hs harg

      (by constructor <;> norm_num) (Or.inr rfl) harg
```

This is the smallest axiom-clean patch.  It does not prove new analysis, but it isolates the missing analysis in the form future endpoint proofs can target.

## 4. Signs and the future analytic endpoint theorem

The target is:

```lean
-(|p.χ₀| * fluxCoeffConst p.β (p.ν * M' ^ p.γ) + p.b * M' ^ p.α) * m
  ≤ deriv (fun r => intervalDomainLift (u r) ys) s
```

where:

```lean
m = sInf (intervalDomainLift (u s) '' Set.Icc 0 1)
M' = SupNormBridge.regimeBound p M
```

The sign bookkeeping should match the existing interior lemma:

```lean
MinPersistenceAtoms.min_point_estimate
```

The PDE is:

```text
u_t = u_xx - χ₀ * chemDiv + u * (a - b*u^α)
```

At a min point, if the endpoint one-sided analysis gives:

```text
u_xx_limit ≥ 0,
chemDiv_limit = m * G,
|G| ≤ fluxCoeffConst β (ν*M'^γ),
0 ≤ m ≤ M',
```

then because `χ₀ ≤ 0`:

```text
-χ₀ * chemDiv_limit = |χ₀| * m * G ≥ - |χ₀| * fluxCoeffConst ... * m.
```

The reaction term gives:

```text
m*(a - b*m^α) ≥ - b*M'^α*m.
```

Adding these with `u_xx_limit ≥ 0` yields exactly the target lower bound.

The future analytic theorem should therefore be a one-sided boundary analogue of:

```lean
MinPersistenceAtoms.min_point_estimate_interior
```

not a new global persistence theorem.  A possible future theorem shape is:

```lean
theorem hbdry_left_chi_nonpos
    {p : CM2Params} {T s M' : ℝ} {u v : ℝ → intervalDomainPoint → ℝ}
    (hχ : p.χ₀ ≤ 0)
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hs0 : 0 < s) (hsT : s < T) (hM' : 0 ≤ M')
    (hu_le : ∀ x : intervalDomainPoint, u s x ≤ M')
    (hargmin : intervalDomainLift (u s) 0 =
        sInf (intervalDomainLift (u s) '' Set.Icc (0:ℝ) 1)) :
    -(|p.χ₀| * fluxCoeffConst p.β (p.ν * M' ^ p.γ) + p.b * M' ^ p.α)
        * sInf (intervalDomainLift (u s) '' Set.Icc (0:ℝ) 1)
      ≤ deriv (fun r => intervalDomainLift (u r) 0) s
```

and similarly:

```lean
theorem hbdry_right_chi_nonpos ...
```

Those are the direct general-χ analogues of `hbdry_left_chi0` and `hbdry_right_chi0`.  Proving them will require endpoint versions of the chemDiv critical factorization/bound, or an equivalent one-sided limit theorem.  The current codebase has the interior pieces (`chemDiv_at_critical`, `flux_coeff_bound`, `min_point_estimate_interior`) but not this boundary endpoint passage.
