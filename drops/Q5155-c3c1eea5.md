# Bottom line

The shortest sound route **does not require a fourth derivative of the heat kernel**, a sectorial package, or a new global Schauder theorem.

For the singular chemotaxis term, split the third-kernel history at a cutoff `c < t` chosen close to `t`:

```text
∫₀ᵗ Third(t-s,F(s)) ds
  = ∫₀ᶜ Third(t-s,F(s)) ds
    + ∫ᶜᵗ Third(t-s,F(s)) ds.
```

* On `[0,c]`, the lag is uniformly positive.  Use the already-proved spatial integration-by-parts identity
  `ThirdOp = HessOp (deriv F(s))`.  Each integrand is then spatially continuous because `HessOp` is spatially differentiable.  The raw third-kernel estimate supplies one constant integrable majorant, so `intervalIntegral.continuousAt_of_dominated_interval` gives spatial continuity of the old history.
* On `[c,t]`, do **not** try to dominate the raw `τ⁻³ᐟ²` singularity.  Use the existing positive-window Hölder modulus of `deriv F(s)` and the existing theorem
  `wholeLineCauchyFluxThird_recent_abs_le_positive`.  It gives a uniform tail bound of order `(t-c)^(rho/2)`.  Hence

  ```text
  |Tail(y) - Tail(x)| ≤ |Tail(y)| + |Tail(x)| → 0
  ```

  uniformly in the spatial points as `c ↑ t`.

This proves continuity, which is all `ContDiff ℝ 2` needs.  A new spatial Hölder estimate for the entire third-kernel history would be stronger than necessary.  If a quantitative `C^sigma` tail theorem is desired, the existing `wholeLineCauchyHeatHessOp_Ctheta_to_Ceta` already provides it after `ThirdOp = HessOp (deriv F)`; I give that optional signature below.

For strict interior times `0 < t < T`, all analytic leaves already exist.  The only missing theorem is the spatial-continuity assembly.  To cover the closed local endpoint `t = T` as well, two existing wrappers should be copied with `t ≤ T` instead of `t < T`; their proofs use only the weak inequality.

# 2. Why the tempting shortcuts are wrong

## 2.1 Time continuity is not spatial continuity

The warning in the question is correct.  Evaluating at one fixed `x` and varying terminal time does not show that the second derivative is continuous as a function of `x`.

## 2.2 The positive-time PDE does not shorten this proof

`wholeLineCauchyBUCMildFixedPoint_time_hasDerivAt_positive` and
`wholeLineCauchyBUCMildFixedPoint_physical_pde_hasDerivAt` express the time derivative using the same spatial second derivative:

```text
u_t = u_xx - u - χ F_x + R.
```

The generator-continuity theorem used there is again continuity in time at a fixed spatial point.  Solving this identity for `u_xx` would require spatial continuity of `u_t`, which is not available independently.  Using that route would merely move the same gap and risks circularity.

## 2.3 A raw third-kernel dominated-convergence proof fails at the terminal edge

The raw estimate is of order

```text
(t-s)^(-3/2),
```

which is not integrable near `s = t`.  Therefore one cannot apply dominated convergence to the whole interval with the bounded flux source alone.

## 2.4 A fourth heat-kernel derivative is unnecessary

For the old interval, pointwise continuity of `ThirdOp(t-s,F(s))` can be obtained by

```text
ThirdOp(t-s,F(s)) = HessOp(t-s, deriv F(s))
```

and `wholeLineCauchyHeatHessOp_hasDerivAt`.  No theorem about a fourth Gaussian derivative is needed.

### One uniform majorant


and the exponent `-3/2` is nonpositive.  Therefore:

```lean

    have hlagLower : t - c ≤ t - s := by linarith
```

and similarly at `y`.  This gives

```text
|Tail(x)|, |Tail(y)|
  ≤ W * ((t-c)^(rho/2)/(rho/2)) * HFd.
```

Choose `c` so the right side is less than `eps/4`.  Then

```lean
calc
  |Tail y - Tail x| ≤ |Tail y| + |Tail x| := abs_sub _ _
  _ < eps / 4 + eps / 4 := add_lt_add htail_y htail_x
  _ = eps / 2 := by ring
```

## 4.3 Split and close continuity

Use

```lean
wholeLineCauchyFluxThirdHistory_intervalIntegrable_positive
```

for the complete histories at `x` and `y`, then obtain old and recent integrability with `IntervalIntegrable.mono_set`.  Split with:

```lean
intervalIntegral.integral_add_adjacent_intervals
```

The final estimate is:

```text
|Full(y) - Full(x)|
  ≤ |Old(y) - Old(x)| + |Tail(y)| + |Tail(x)|
  < eps/2 + eps/4 + eps/4
  = eps.
```

This proves `ContinuousAt` at every `x`, hence `Continuous`.

* The old flux dominated-convergence proof works on `Ioc 0 c`, so the potentially nonsmooth stored source at `s = 0` is excluded almost everywhere.

# 8. Optional stronger tail Hölder theorem

If downstream work benefits from a quantitative spatial modulus rather than mere continuity, the mathematically natural missing generic lemma is:

```lean
/-- A recent third-kernel history is C^sigma when the source derivative is
uniformly C^rho. -/
theorem wholeLineCauchyHeatThirdHistory_Crho_to_Csigma
    {F : ℝ → WholeLineBUC}
    {c t C D H rho sigma : ℝ}
    (hct : c < t)
    (hC : 0 ≤ C) (hD : 0 ≤ D) (hH : 0 ≤ H)
    (hrho0 : 0 < rho) (hrho1 : rho < 1)
    (hsigma0 : 0 < sigma) (hsigma1 : sigma < 1)
    (hrel : sigma * (1 + rho) < rho)
    (hFbound : ∀ s ∈ Set.Ioc c t, ∀ y, |(F s).1 y| ≤ C)
    (hFderiv : ∀ s ∈ Set.Ioc c t, ∀ y,
      HasDerivAt (F s).1 (deriv (F s).1 y) y)
    (hFdcont : ∀ s ∈ Set.Ioc c t,
      Continuous (deriv (F s).1))
    (hFdbound : ∀ s ∈ Set.Ioc c t, ∀ y,
      |deriv (F s).1 y| ≤ D)
    (hFdholder : ∀ s ∈ Set.Icc c t, ∀ x y,
      |deriv (F s).1 x - deriv (F s).1 y| ≤
        H * |x - y| ^ rho)
    (hint : ∀ x, IntervalIntegrable
      (fun s => wholeLineCauchyHeatThirdOp (t - s) (F s).1 x)
      volume c t) :
    ∃ K : ℝ, 0 ≤ K ∧ ∀ x y,
      |(∫ s in c..t,
          wholeLineCauchyHeatThirdOp (t - s) (F s).1 x) -
        (∫ s in c..t,
          wholeLineCauchyHeatThirdOp (t - s) (F s).1 y)| ≤
        K * |x - y| ^ sigma
```

Proof for almost every `s < t`:

```lean
rw [wholeLineCauchyHeatThirdOp_eq_hessOp_deriv
  (sub_pos.mpr hst)
  (hFbound s hs) (hFdbound s hs)
  (hFderiv s hs) (hFdcont s hs)]

exact wholeLineCauchyHeatHessOp_Ctheta_to_Ceta
  (sub_pos.mpr hst) hD hH
  hrho0 hrho1 hsigma0 hsigma1
  (hFdcont s hs).aestronglyMeasurable
  (hFdbound s hs) (hFdholder s ...) x y
```

The resulting majorant is

```text
const * (t-s)^(-1 + (rho - sigma*(1+rho))/2),
```

which is integrable by

```lean
intervalIntegrable_sub_rpow_hess_Ctheta_to_Ceta hrel.
```

A convenient internal choice is

```lean
sigma := rho / (2 * (1 + rho)).
```

Then

```text
sigma * (1 + rho) = rho/2 < rho.
```

This theorem is sound and reusable, but it is **not required** for the shortest `Continuous` proof because uniform smallness of the recent tail is enough.

# 10. Circularity audit

The proposed route is acyclic.

1. `WholeLineCauchyFluxHolderBootstrap` and `WholeLineCauchyC1HolderBootstrap` produce the positive-time first derivative and its Hölder modulus.
2. `WholeLineCauchyFluxC1Bootstrap` produces a `C¹` physical flux and a Hölder derivative.
3. `WholeLineCauchyC2Bootstrap` uses those facts to establish pointwise existence of the second derivative.
4. The new `WholeLineCauchySpatialC2` file imports the already-finished pointwise rung and proves only continuity of the explicit second-derivative formula.
5. `ContDiff ℝ 2` is assembled after both existence and continuity are independently proved.

No step consumes `ContDiff ℝ 2` for the canonical slice.

`wholeLineCauchyFluxSourceTrajectory_restartC1Data_positive` is safe despite living in a later file: its proof uses only the C¹/Hölder flux results.  If import layering is a concern, inline the exact four facts listed in §1.2.
