# Definitive audit

**The answer is not simply “all fields are routine wiring.”** There are two distinct findings.

1. **On an open active-time interval `0 < t < T`, there is no HSpectral-style nonlinear regularity crux.** Once positive-time `H¹`/Lipschitz admissibility of the negative-part test is available, the source/flux bounds and continuity are available, and the concrete Neumann semigroup facts are supplied, all fields reduce to one shared textbook analytic theorem: the Neumann semigroup weak-Duhamel differentiation identity, with an `H⁻¹` version after the already-proved restricted `B_N` duality.

2. **The present bundle is quantified over `0 < t ≤ T`, and for the truncated trajectory this closed-endpoint formulation is generally false before nonnegativity is known.** The truncated Picard limit is extended by zero for times `s > T`, while its slice at `s = T` need not be zero. Consequently the right-sided endpoint averages in `source_endpoint_l2_lebesgue` and `chem_endpoint_l2_lebesgue` generally converge to zero at `t = T`, not to the nonzero source/flux pairing at `T`. The corresponding two-sided `deriv`-based Duhamel fields and `tested_mild_decomposition` can fail at `T` for the same reason.

This is why the committed `IntervalTruncatedEnergyProducerV6.lean` now says that the mathematically correct order is to prove open-time variational nonnegativity first and only then fill the legacy closed-endpoint bundle trivially, because the negative-part test is zero. A coding agent trying to construct the current closed bundle *before* Gronwall can be stuck on false endpoint goals, not merely grinding slowly.

## `source_endpoint_l2_lebesgue`

For

```text
D_F(t) = ∫₀ᵗ S_N(t-s) F(s) ds,
```

the new-endpoint portion of the difference quotient is

```text
(1/h) ∫ₜ^{t+h} ⟨S_N(t+h-s)F(s), φ⟩ ds.
```

At an interior time `t < T`, this tends to `⟨F(t),φ⟩` from:

- right `L²` continuity of `F` at `t`;
- strong continuity `S_N(r)g → g` in `L²` as `r ↓ 0`;
- a local `L²` bound or semigroup contraction;
- the elementary average/Lebesgue-point argument.

**Classification: analytic but light, at interior times.** It is not definitional, but it is a standard compact argument.

**Terminal obstruction:** at `t = T`, the truncated source is zero for almost every `s ∈ (T,T+h]`. Therefore the average is zero, whereas the target is

```text
⟨F(T), φ⟩,
```

which need not vanish before nonnegativity is proved. Thus the current `t ≤ T` field is not generally dischargeable.

## `chem_endpoint_l2_lebesgue`

After the lagwise restricted duality,

```text
⟨B_N(r) Q(s), φ⟩ = -⟨Q(s), ∂ₓS_N(r)φ⟩.
```

The endpoint limit becomes

```text
-(1/h) ∫ₜ^{t+h} ⟨Q(s), ∂ₓS_N(t+h-s)φ⟩ ds
    → -⟨Q(t), φₓ⟩.
```

This requires:

- `Q : C_t L²_x` locally on the active interval;
- `φ ∈ H¹`;
- **form-domain strong continuity**

```text
∂ₓS_N(r) φ → φₓ  in L²,  r ↓ 0;
```

- a uniform local bound for `∂ₓS_N(r)φ`.

Ordinary `L²` strong continuity alone does not syntactically give the gradient convergence; one also uses spectral commutation or strong continuity of the Neumann semigroup on its form domain. This is the technically sharpest endpoint lemma, but it is still standard linear semigroup analysis, not a new chemotaxis regularity theorem.

**Classification: analytic, medium; the hardest endpoint sublemma.**

It has the same terminal obstruction: the right average uses `Q(s)=0` for `s>T`, while the target `-⟨Q(T),φₓ⟩` need not be zero.

## `semigroup_form_identity`

This field says, for `w(t)=S_N(t)u₀`,

```text
∫ w_t(t) φ + ∫ wₓ(t) φₓ = 0.
```

It is important not to justify this as “the heat leg is nonnegative, so it contributes zero.” The test is the negative part of the **full solution**, not of the homogeneous heat leg. Heat positivity does not make either integral separately zero.

The identity follows because the homogeneous heat leg solves the weak heat equation:

```text
w_t = Δ_N w,
⟨-Δ_N w, φ⟩ = ∫ wₓ φₓ.
```

It needs semigroup differentiability/generator identification and the Neumann form identity for an `H¹` test.

**Classification: genuinely analytic but standard; not algebraic.** Once a reusable theorem

```lean
heatSemigroup_tested_form_identity
```

is available, this particular field is short wiring.

## `source_duhamel_differentiation`

This is

```text
∫ (D_F)_t(t) φ + ∫ (D_F)ₓ(t) φₓ = ∫ F(t) φ.
```

It is **not definitional**. Its proof combines:

- the endpoint-Lebesgue term;
- cancellation of the old-history difference quotient with the gradient form term;
- dominated convergence using the `(t-s)^(-1/2)` majorant;
- semigroup strong continuity and the generator/form identity.

**Classification: shared analytic semigroup lemma, medium effort.** Once abstracted, the logistic instance is routine.

## `hminusone_duhamel_differentiation_after_restricted_duality`

This is the `H⁻¹` counterpart:

```text
∫ (D_Q)_t(t) φ + ∫ (D_Q)ₓ(t) φₓ = -∫ Q(t) φₓ.
```

The lagwise premise rewrites the B-form operator pairing into the Neumann heat-semigroup gradient pairing. Then the proof uses the chem endpoint lemma and chem DCT majorant exactly as above.

**Classification: the hardest bundle field if written monolithically, but not because the `B_N` duality is open.** After endpoint, DCT, and restricted duality are supplied, it is a standard semigroup weak-differentiation assembly.

At `t=T` it is again generally false for the zero-extended trajectory before the test is known to vanish.

## `tested_mild_decomposition`

At an interior time, after all three legs are known differentiable and the mild identity holds on a neighborhood of `t`, this is mostly:

**Classification: algebraic/wiring after the analytic differentiability lemmas.** It is not purely definitional from an equality at one time: one needs a local-in-time functional equality to transfer derivatives. The terminal two-sided-derivative issue again prevents a naive closed-time proof.

# 2. The restricted `B_N` duality is not the hard leaf

The committed theorem is

```lean
BNDualityForFluxTestAt.duality
```

in `IntervalBFormCron2RegularFluxDuality.lean`. Its data are only bounded/measurable flux and bounded/measurable test functions. The proof supplies product-measure integrability and invokes the concrete regular kernel identity.

Conceptually it proves

```text
∫ (B_N(τ)g)(x) φ(x) dx
  = -∫ g(y) ∂ᵧ(S_N(τ)φ)(y) dy,
      τ>0.
```

The proof is:

1. expand the concrete Neumann kernel derivative;
2. justify Fubini from bounded/integrable `g` and `φ`;
3. use the derivative-swap identity for the symmetric Neumann kernel;
4. identify the inner integral with `∂ₓS_N(τ)φ`.

There is **no physical integration by parts in `Q`** and no requirement that `Q(0)=Q(1)=0` in this restricted kernel-duality theorem. Neumann boundary behavior is already encoded in the reflected Neumann kernel. Flux endpoint vanishing is relevant to a separate physical proof starting from `∂ₓQ` and integrating by parts directly, but it is not the missing hypothesis here.

The repository itself already describes the remaining frontier as

```lean
TruncatedMildSemigroupWeakAfterBNDualityOn
```

and explicitly says that the stall is a Neumann semigroup weak-Duhamel differentiation theorem, **not** a chemotaxis-duality gap.

# 4. Required interface correction

Do not ask the pre-Gronwall producer for the current closed-time bundle verbatim. Replace the active-time quantifier

```lean
∀ t, 0 < t → t ≤ T → ...
```

by

```lean
∀ t, 0 < t → t < T → ...
```

for the endpoint and differentiation fields, or define an explicit open-time bundle such as

```lean
NegativePartStandardHeatSemigroupDuhamelFactsOnOpenTime p T u₀ u
```

Then prove the energy inequality on every compact window

```text
[ε, T']  with  0 < ε < T' < T.
```

Use the already-proved initial negative-energy vanishing as `ε ↓ 0`, apply Gronwall on each `T' < T`, and finally pass `T' ↑ T` using continuity of the negative-part energy. No derivative or right-Lebesgue assertion at the terminal time is needed.

The alternative committed route—prove variational nonnegativity first, then fill the old closed bundle by the zero test—is also logically sound, but that bundle cannot then be the input used to prove the same nonnegativity.
