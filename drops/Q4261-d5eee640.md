# 2. Comparison does not intrinsically require a classical supersolution

Parabolic comparison is valid for weak energy solutions. The classical maximum principle is only one proof of it.

Consider on a restarted strip `(0,L)√ó(0,1)` the operator

```text
Pz := z_t - ŒΩ z_xx - B z_x - C z,
```

with `ŒΩ>0`, bounded measurable `B,C`, and homogeneous Neumann diffusion boundary conditions. Suppose

```text

Pw ‚â§ 0       weakly,
w(0,¬∑) ‚â§ u(0,¬∑).
```

Set

```text

```

Then `Pz‚â§0`. Test the weak inequality with

```text
z‚Çä := max z 0.
```

The rigorous implementation uses Steklov time averages or the standard Hilbert-triple chain rule. Formally the calculation is

```text
(1/2) d/dt ‚Äñz‚Çä‚Äñ¬≤‚ÇÇ + ŒΩ ‚Äñ‚àÇ‚Çìz‚Çä‚Äñ¬≤‚ÇÇ
  ‚â§ ‚à´ B (‚àÇ‚Çìz‚Çä) z‚Çä + ‚à´ C z‚Çä¬≤.
```

On `{z>0}` one has

```text

‚àÇ‚Çìz=‚àÇ‚Çìz‚Çä
```

a.e. If `|B|‚â§A` and `C‚â§Cbar`, then

```text
‚à´ B (‚àÇ‚Çìz‚Çä) z‚Çä
  ‚â§ A ‚Äñ‚àÇ‚Çìz‚Çä‚Äñ‚ÇÇ ‚Äñz‚Çä‚Äñ‚ÇÇ
  ‚â§ (ŒΩ/2) ‚Äñ‚àÇ‚Çìz‚Çä‚Äñ¬≤‚ÇÇ + (A¬≤/(2ŒΩ)) ‚Äñz‚Çä‚Äñ¬≤‚ÇÇ,
```

and


Therefore


```text

```

a.e. Joint or slice continuity then upgrades the conclusion to every `x‚àà[0,1]`.

No pointwise `u_xx` occurs anywhere.

# 3. Can this be proved ‚Äúdirectly at the mild level‚Äù?

There are two meanings of that phrase.

## 3.1 Yes: mild identity ‚Üí weak inequality ‚Üí energy comparison

This is the clean route. Start from the B-form mild identity, test it against suitable spatial test functions, obtain the distributional/variational equation, and apply the weak comparison calculation above.

The repository already follows this philosophy for nonnegativity. In particular:

```text
ShenWork/Paper2/IntervalBFormCron2SemigroupWeakDuhamel.lean
ShenWork/Paper2/IntervalTruncatedEnergyProducerV6.lean
```

construct the negative-part tested weak identity from the mild decomposition, including the `H^{-1}` chemotaxis Duhamel leg, and then feed an energy/Gronwall argument. That infrastructure proves that a mild solution can support a sign argument without first proving pointwise `u_xx`.

For barrier comparison, repeat the same pattern for

```text
v := u-w.
```

Instead of proving `u_- = 0`, prove

```text
v_- = 0.
```

This is especially compatible with the existing code: the comparison proof becomes another negative-part energy proof, now for the difference between the mild solution and the smooth barrier.

## 3.2 No: not by a naive order estimate on the ordinary heat Duhamel formula

One cannot simply write

```text
z(t)=S_ŒΩ(t)z(0)+‚à´‚ÇÄ·µó S_ŒΩ(t-r)[B(r) z_x(r)+C(r)z(r)]dr
```

and invoke positivity of `S_ŒΩ`. The term

```text
B z_x
```

has no sign, so the integrand is not order preserving. Rewriting it with a conjugate/gradient kernel does not create a sign either.

An abstract positive evolution family for the full time-dependent operator

```text
ŒΩŒî_N+B(t,x)‚àÇ‚Çì+C(t,x)
```

would give a mild comparison theorem, but proving positivity of that evolution family is essentially another formulation of the same weak/classical comparison principle. It is far more machinery than the energy proof.

So the correct statement is:

> A mild solution can be compared without classical `u_xx`, but the proof should pass through a weak tested identity, not through termwise positivity of the base heat-semigroup Duhamel formula.

# 4. Minimal regularity for the weak comparison

A standard sufficient package on every restarted compact strip `[0,L]` is:

```text

u_t,w_t ‚àà L¬≤(0,L;H^{-1}(0,1));
```

together with the variational sub/supersolution inequalities and the initial ordering in `L¬≤` or a.e.

The `C([0,L];L¬≤)` field may be derived from the first two Bochner-space fields by the standard Hilbert-triple theorem, but carrying it explicitly is easier in Lean.

The Neumann condition should be encoded naturally in the weak diffusion term

```text

```

rather than as a pointwise endpoint derivative. No boundary integral appears.

For the final pointwise comparison one additionally uses continuity of `u` and `w` on the closed positive-time strip. The barrier is smooth; the truncated Picard limit now has joint-continuity infrastructure on compact positive-time windows.

# 7. The H√∂lder/Schauder fallback also breaks the circle

Suppose one elects to keep the existing classical comparison theorem rather than add a weak one.

Strict positivity is still unnecessary.

## 7.1 Powers at zero

For every `Œ≥‚â•1`, the map

```text
r‚Ü¶r^Œ≥
```

is Lipschitz on each bounded interval `[0,M]`. Hence

```text
u‚ààC^Œ∏,
0‚â§u‚â§M
‚áí
u^Œ≥‚ààC^Œ∏.
```

For noninteger `1<Œ≥<2`, the second derivative of `r^Œ≥` is singular at zero, but that is irrelevant here. Elliptic Schauder needs a `C^Œ∏` right-hand side to produce `R‚ààC^{2+Œ∏}`; it does not need `u^Œ≥‚ààC¬≤`.

## 7.2 Two-stage bootstrap

The accurate bootstrap is:

```text
bounded weak/mild u
‚Üí positive-time H√∂lder regularity of u
‚Üí u^Œ≥ ‚àà C^Œ∏
‚Üí R ‚àà C^{2+Œ∏}
‚Üí q ‚àà C^{1+Œ∏}, q_x ‚àà C^Œ∏
‚Üí B,C_exact ‚àà C^Œ∏
‚Üí linear parabolic Schauder
‚Üí u ‚àà C^{1+Œ∏/2,2+Œ∏}
```

on strips bounded away from the initial time.

This yields exactly the derivatives required by `IsClassicalNeumannLinearDriftSuperSolution`.

Two cautions:

- Mere abstract continuity of `u` does not imply H√∂lder continuity. The H√∂lder exponent must come from the parabolic mild equation, De Giorgi/Nash theory, or semigroup smoothing.
- The divergence chemotaxis term is the real regularity issue. One should first establish enough Sobolev regularity to use `‚àÇ‚Çì(uq)=q u_x+q_xu` weakly, then apply linear Schauder. Saying only ‚Äú`u^Œ≥` is continuous, therefore `u` is classical‚Äù skips this step.

For regularity up to `x=0,1`, use a Neumann boundary Schauder theorem or an even-reflection argument. Restarting at `œÑ>0` removes initial-time compatibility issues.

This route is mathematically sound, but in Lean it is much larger than the weak energy comparison because Mathlib does not presently supply a ready-made full parabolic Schauder package matching these project conventions.

# Final answer

- **The presently landed comparison theorem is classical**, so it cannot directly consume the current mild/weak solution.
- **There is no mathematical circularity.** Weak parabolic comparison applies at the energy level and requires no pointwise `u_xx`.
- The clean proof is **mild ‚Üí comparison-specific weak identity ‚Üí negative-part energy for `u-w` ‚Üí Gronwall ‚Üí pointwise comparison by continuity**.
- The minimal solution regularity is essentially `L¬≤_tH¬π_x`, time derivative in `L¬≤_tH^{-1}_x`, an `L¬≤` time trace, and bounded drift/reaction coefficients; the present development has the analytic ingredients on positive-time compact windows, though not yet one consolidated comparison structure.

- Independently of circularity, the exact linear reaction coefficient must include `-œá‚ÇÄq_x` unless a separate sign/residual lemma justifies dropping it.

So the squared-heat-barrier strategy is **sound and non-circular**, but the shortest honest repository closure is a new weak Neumann drift-comparison layer plus the exact `q_x` coefficient adapter‚Äînot an assumption of strict positivity and not a naive order argument on the heat Duhamel formula.
