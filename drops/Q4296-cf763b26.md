# Q4296 — flux regularity and `WindowSourceEnvelope` for the chemotaxis source

## Executive verdict

The mathematical chain is standard, but for the Lean port the cleanest route is **not** to build a general Sobolev/Moser/elliptic-regularity library first. For the immediate `WindowSourceEnvelope` leaf, use the one-dimensional coefficient route directly:

```text
uniform spatial derivative bounds on the window
  ⇒ repeated integration by parts for cosine/sine coefficients
  ⇒ explicit summable envelope E_k.
```

This is much cheaper than proving full abstract statements such as

```text
R ∈ H^{r+2},  (1+R)^(-β) ∈ H^{r+1},  Q = u R_x/(1+R)^β ∈ H^r.
```

Those Sobolev statements are mathematically correct under the usual hypotheses, but Mathlib does not provide a turnkey 1D Neumann elliptic-regularity + Moser-composition + Sobolev-algebra stack at the level needed here. In this project, the formalization should consume the already-available Schauder / derivative bounds and prove coefficient decay by elementary interval integration by parts.

The important correction is this:

```text
W^{1,∞} alone gives only O(1/k) cosine decay.
```

That is not summable. To get a summable `E_k`, the logistic coefficient needs at least a uniform `W^{2,1}` bound with the first derivative vanishing at the endpoints, and the chemotaxis-divergence coefficient needs one more derivative of the flux, for example a uniform `W^{3,1}` or bounded-variation bound on `Q''`.

So the safe formalization target is:

```text
L(t,x) = u(t,x) (a - b u(t,x)^α)        uniformly W^{2,1}
Q(t,x) = u(t,x) R_x(t,x)/(1+R(t,x))^β  uniformly W^{3,1}, Q(0)=Q(1)=0
```

on the compact positive-time window `[c,T']`. Then define

```text
E_0      = a separate finite bound for the k=0 coefficient,
E_k      = C / (kπ)^2  for k ≥ 1.
```

This is summable and uniform in time.

## 1. Elliptic gain: mathematically true, but do not make it the first Lean target

The classical statement is:

```text
(μ - ν ∂xx) R = ν u^γ,
R_x(0)=R_x(1)=0,
μ,ν>0.
```

For integer or real Sobolev order `r ≥ 0`, the Neumann resolvent satisfies

```text
‖R‖_{H^{r+2}} ≤ C_{μ,ν,r} ‖u^γ‖_{H^r}.
```

In cosine coefficients, this is transparent. If

```text
u^γ(x) = Σ f_k cos(kπx),
R(x)   = Σ R_k cos(kπx),
```

then

```text
R_k = ν f_k / (μ + ν (kπ)^2).
```

Thus every coefficient gets two powers of decay at high frequency. A Lean-friendly spectral version is likely much easier than abstract elliptic regularity:

```lean
lemma neumann_resolvent_cosine_gain_two
    {r : ℕ} {f : ℕ → ℝ}
    (hf : Summable fun k => weight r k * f k^2) :
    Summable fun k => weight (r+2) k *
      (ν * f k / (μ + ν * ((k:ℝ)*Real.pi)^2))^2 := ...
```

For the `WindowSourceEnvelope` leaf, however, you do not actually need to state `R ∈ H^{r+2}`. It is enough to use concrete uniform derivative bounds:

```text
|R|, |R_x|, |R_xx|, |R_xxx|, ... ≤ constants depending on M and the window.
```

If your Schauder / positive-time bootstrap already provides those constants, use them directly.

## 2. Composition with `(1+R)^(-β)`

Mathematically:

```text
R ≥ 0  ⇒  1+R ≥ 1.
```

Therefore the map

```text
z ↦ (1+z)^(-β)
```

is smooth on the entire range of `R`. If `R ∈ H^{s}` and `s > 1/2` in one dimension, the usual Moser composition theorem gives

```text
(1+R)^(-β) ∈ H^s,
‖(1+R)^(-β)‖_{H^s} ≤ C(β, s, ‖R‖_{L∞}, lower-bound) · ‖R‖_{H^s}.
```

For the Lean target, avoid proving this general Moser theorem if possible. Since the denominator is bounded away from zero, just differentiate the quotient explicitly to the finite order needed by the coefficient IBP.

Let

```text
D := (1+R)^(-β).
```

Then the first few derivatives are combinations of `R_x`, `R_xx`, `R_xxx`, etc. multiplied by powers of `(1+R)^(-β-m)`. Because `1+R ≥ 1`, all denominator powers are bounded by `1` when the exponent is negative:

```text
|(1+R)^(-β-m)| ≤ 1,     β,m ≥ 0.
```

So the derivative bounds reduce to polynomial bounds in the resolver derivatives. For example, schematically:

```text
|D_x|   ≤ β |R_x|,
|D_xx|  ≤ β |R_xx| + β(β+1)|R_x|^2,
|D_xxx| ≤ β |R_xxx| + Cβ |R_x||R_xx| + Cβ |R_x|^3.
```

This is much more formalizable than a general Sobolev composition theorem.

## 3. Product estimate for the flux

The flux is

```text
Q = u · R_x · (1+R)^(-β).
```

The Sobolev product statement is:

```text
H^r · H^{r+1} ⊂ H^r       in one dimension,
```

and, for `r > 1/2`, `H^r` is an algebra. The usual tame product bound is

```text
‖fg‖_{H^r} ≤ C_r (‖f‖_{H^r} ‖g‖_{L∞} + ‖f‖_{L∞} ‖g‖_{H^r}).
```

Again, for the immediate coefficient envelope, do not need the abstract product theorem. Use finite-order derivative bounds:

```text
Q'   = u' R_x D + u R_xx D + u R_x D',
Q''  = finite sum of products of derivatives of u, R, D,
Q''' = finite sum of products of derivatives of u, R, D.
```

If the window supplies uniform bounds for these factors, then it supplies uniform bounds for

```text
‖Q'‖_{L1}, ‖Q''‖_{L1}, ‖Q'''‖_{L1}, |Q''(0)|, |Q''(1)|.
```

Those are the quantities the coefficient proof really needs.

## 4. Coefficient decay: what is actually needed

Let

```text
L(t,x) := u(t,x) (a - b u(t,x)^α)
```

and

```text
Q(t,x) := u(t,x) R_x(t,x)/(1+R(t,x))^β.
```

Assume the B-form source coefficient has the shape

```text
s_k(t) = s_k^log(t) + s_k^chem(t),
```

where

```text
s_k^log(t)  = ∫_0^1 L(t,x) cos(kπx) dx,
s_k^chem(t) = -χ₀ · kπ · ∫_0^1 Q(t,x) sin(kπx) dx
```

up to harmless normalization constants.

### Logistic coefficient

For `k ≥ 1`, if `L_x(t,0)=L_x(t,1)=0` and `L_xx(t,·) ∈ L¹`, then twice integrating by parts gives

```text
|s_k^log(t)| ≤ ‖L_xx(t,·)‖_{L¹} / (kπ)^2.
```

The endpoint condition is natural if `u_x=0` at the Neumann boundary, because

```text
L_x = (a - b u^α - bα u^α) u_x
```

formally, and hence `L_x=0` at the boundary when `u_x=0`.

A uniform window bound

```text
∀ t ∈ [c,T'], ‖L_xx(t,·)‖_{L¹} ≤ C_log
```

therefore gives

```text
|s_k^log(t)| ≤ C_log/(kπ)^2.
```

Important: `W^{1,∞}` alone only gives

```text
|s_k^log(t)| ≤ ‖L_x(t,·)‖_{L¹}/(kπ),
```

which is not summable. So the `k^{-2}` claim needs the second-derivative input or bounded variation of `L_x`.

### Chemotaxis-divergence coefficient

Because `R` has Neumann boundary condition, `R_x(t,0)=R_x(t,1)=0`. Hence

```text
Q(t,0)=Q(t,1)=0.
```

For `k ≥ 1`, set

```text
I_k(t) := kπ ∫_0^1 Q(t,x) sin(kπx) dx.
```

The first integration by parts uses `Q(0)=Q(1)=0`:

```text
I_k(t) = ∫_0^1 Q_x(t,x) cos(kπx) dx.
```

That only gives `O(1)`, not summability.

A second integration by parts gives

```text
I_k(t) = -1/(kπ) ∫_0^1 Q_xx(t,x) sin(kπx) dx.
```

That gives `O(1/k)`, still not summable.

To get a summable `O(1/k²)` envelope, integrate one more time, requiring `Q_xx` to have bounded variation or `Q_xxx ∈ L¹`:

```text
∫_0^1 Q_xx sin(kπx)
  = [-Q_xx cos(kπx)/(kπ)]_0^1
    + 1/(kπ) ∫_0^1 Q_xxx cos(kπx) dx.
```

Therefore

```text
|I_k(t)| ≤
  ( |Q_xx(t,0)| + |Q_xx(t,1)| + ‖Q_xxx(t,·)‖_{L¹} ) / (kπ)^2.
```

Thus, with a uniform window bound

```text
∀ t ∈ [c,T'],
  |Q_xx(t,0)| + |Q_xx(t,1)| + ‖Q_xxx(t,·)‖_{L¹} ≤ C_Q,
```

we get

```text
|s_k^chem(t)| ≤ |χ₀| C_Q/(kπ)^2.
```

If your Schauder input gives only `C^{2,α}` for `Q`, then there is an alternative Hölder-Fourier estimate giving roughly

```text
|I_k(t)| ≤ C / k^{1+α},
```

which is also summable for `α>0`. But that Hölder coefficient-decay proof is likely more painful in Lean than proving or assuming a finite `Q_xxx` / bounded-variation envelope.

## 5. Define the envelope

For `k = 0`, handle the coefficient separately. The chemotaxis divergence contributes zero or is bounded by a finite constant depending on how the coefficient is normalized. The logistic part is bounded by

```text
|s_0^log(t)| ≤ ‖L(t,·)‖_{L¹} ≤ C_0.
```

For `k ≥ 1`, define

```text
C₂ := C_log + |χ₀| C_Q.
```

Then a convenient envelope is

```lean
noncomputable def sourceEnvelope (C0 C2 : ℝ) : ℕ → ℝ
  | 0     => C0
  | k + 1 => C2 / (((k + 1 : ℕ) : ℝ) * Real.pi)^2
```

with hypotheses

```lean
0 ≤ C0,
0 ≤ C2.
```

Then prove:

```lean
lemma sourceEnvelope_summable
    (hC0 : 0 ≤ C0) (hC2 : 0 ≤ C2) :
    Summable (sourceEnvelope C0 C2) := by
  -- one finite head plus comparison with C2 / (π^2) * 1/(k+1)^2
  -- use the local reciprocal-square summability lemma, or prove by comparison.
  ...
```

And the uniform coefficient bound:

```lean
lemma sourceCoeff_abs_le_sourceEnvelope
    {r : ℝ} (hr : r ∈ Set.Icc c T') (k : ℕ) :
    |truncatedBFormSourceCoeff p u r k| ≤ sourceEnvelope C0 C2 k := by
  cases k with
  | zero =>
      -- use the k=0 finite bound
      exact hzero hr
  | succ j =>
      have hkpos : 0 < (j.succ : ℕ) := Nat.succ_pos j
      have hlog : |sLog r (j.succ)| ≤ C_log / (((j.succ:ℕ):ℝ) * Real.pi)^2 := ...
      have hchem : |sChem r (j.succ)| ≤ |p.χ₀| * C_Q / (((j.succ:ℕ):ℝ) * Real.pi)^2 := ...
      calc
        |sLog r j.succ + sChem r j.succ| ≤ |sLog r j.succ| + |sChem r j.succ| := abs_add _ _
        _ ≤ C_log / (((j.succ:ℕ):ℝ) * Real.pi)^2
              + |p.χ₀| * C_Q / (((j.succ:ℕ):ℝ) * Real.pi)^2 := add_le_add hlog hchem
        _ = (C_log + |p.χ₀| * C_Q) /
              (((j.succ:ℕ):ℝ) * Real.pi)^2 := by ring
        _ = sourceEnvelope C0 C2 j.succ := by simp [sourceEnvelope, C2]
```

Then package it as the project-level `WindowSourceEnvelope`:

```lean
lemma windowSourceEnvelope_of_uniform_derivative_bounds
    ... :
    WindowSourceEnvelope sourceCoeff c T' (sourceEnvelope C0 C2) := by
  refine ⟨sourceEnvelope_summable hC0 hC2, ?_⟩
  intro r hr k
  exact sourceCoeff_abs_le_sourceEnvelope hr k
```

The exact field names may differ, but this is the right shape: one summable sequence plus a uniform coefficient bound for every time in the compact window.

## 6. Uniformity over `[c,T']`

Uniformity is the main point. The constants must depend only on the compact window data, not on the time `r`.

The needed uniform constants are:

```text
C_log ≥ sup_{r∈[c,T']} ‖∂xx L(r,·)‖_{L¹},
C_Q   ≥ sup_{r∈[c,T']} (|Q_xx(r,0)| + |Q_xx(r,1)| + ‖Q_xxx(r,·)‖_{L¹}),
C0    ≥ sup_{r∈[c,T']} |s_0(r)|.
```

If the Schauder bootstrap provides uniform bounds such as

```text
‖u(r)‖_{C^m}, ‖R(r)‖_{C^m} ≤ C_window
```

for all `r∈[c,T']`, then these constants are immediate. On the unit interval, `L¹` norms are bounded by `L∞` norms, so Lean can avoid measure-theoretic complexity:

```text
‖F‖_{L¹(0,1)} ≤ ‖F‖_{L∞(0,1)} · 1.
```

This is often the easiest formal route.

## 7. Recommended Lean architecture

Do not start with a general theorem named something like:

```lean
sobolev_flux_regular_Hr
```

That will force you to build or port a lot of Sobolev infrastructure: elliptic regularity, Moser composition, Sobolev algebra, trace/boundary facts, and embeddings.

Instead add a purpose-built coefficient envelope theorem:

```lean
/-- Uniform `ℓ¹` source envelope from finite-order 1D derivative bounds. -/
theorem windowSourceEnvelope_of_W2_logistic_W3_flux
    {c T' : ℝ}
    {sourceCoeff : ℝ → ℕ → ℝ}
    {L Q : ℝ → ℝ → ℝ}
    {C0 Clog CQ : ℝ}
    (hcT : c ≤ T')
    (hC0_nonneg : 0 ≤ C0)
    (hClog_nonneg : 0 ≤ Clog)
    (hCQ_nonneg : 0 ≤ CQ)
    -- coefficient decomposition
    (hsource_zero : ∀ r ∈ Set.Icc c T', |sourceCoeff r 0| ≤ C0)
    (hsource_succ : ∀ r ∈ Set.Icc c T', ∀ k : ℕ,
       sourceCoeff r (k+1) =
         logCoeff L r (k+1) + chemCoeff Q r (k+1))
    -- logistic IBP hypotheses
    (hlog_bound : ∀ r ∈ Set.Icc c T', ∀ k : ℕ,
       |logCoeff L r (k+1)| ≤
         Clog / ((((k+1:ℕ):ℝ) * Real.pi)^2))
    -- chem-div IBP hypotheses
    (hchem_bound : ∀ r ∈ Set.Icc c T', ∀ k : ℕ,
       |chemCoeff Q r (k+1)| ≤
         CQ / ((((k+1:ℕ):ℝ) * Real.pi)^2)) :
    WindowSourceEnvelope sourceCoeff c T'
      (sourceEnvelope C0 (Clog + CQ)) := by
  ...
```

Then prove separate small lemmas generating `hlog_bound` and `hchem_bound` from derivative hypotheses:

```lean
lemma cosineCoeff_decay_two_of_deriv_boundary_and_second_L1 : ...
lemma chemDivCoeff_decay_two_of_flux_boundary_and_third_L1 : ...
```

These are elementary one-dimensional integration-by-parts lemmas and are much more tractable than an abstract Sobolev theory.

## Bottom line

The `WindowSourceEnvelope` is available from the positive-time spatial regularity, but the correct minimal derivative requirements are stronger than plain `W^{1,∞}`.

A safe summable envelope is:

```text
E_0 = C0,
E_k = (C_log + |χ₀| C_Q)/(kπ)^2,  k≥1,
```

where

```text
C_log controls ‖∂xx logistic(u)‖_{L¹},
C_Q controls |Q_xx(0)| + |Q_xx(1)| + ‖Q_xxx‖_{L¹}.
```

The formalization should target direct coefficient estimates by repeated integration by parts. Full Sobolev `H^r` flux regularity is mathematically valid but is not the cheapest Lean path for this leaf.
