# Definitive verdict

Yes.  In the weak matched-divergence Jensen comparison, the logistic reaction is frozen along the already constructed solution `u` as the scalar zero-order coefficient

```text
c_u(t,x) := a - b * u(t,x)^α.
```

The identity is algebraic:

```text
u * (a - b*u^α) = c_u * u.
```

It does **not** divide by `u`.  Consequently it remains valid at `u = 0` and needs no strict-positive floor.

For the barrier, use the same coefficient:

```text
u_t = ν*u_xx - χ₀*∂x(g*u) + c_u*u,
w_t ≤ ν*w_xx - χ₀*∂x(g*w) + c_u*w.
```

Then `z := w-u` satisfies the homogeneous linear inequality

```text
z_t ≤ ν*z_xx - χ₀*∂x(g*z) + c_u*z.
```

Testing by `η := z₊` produces the reaction contribution

```text
∫ c_u * η^2,
```

not an inhomogeneous term involving `u`.  Since `b ≥ 0`, `u ≥ 0`, and `α > 0`,

```text
c_u = a - b*u^α ≤ a,
```

so

```text
∫ c_u * η^2 ≤ a * ∫ η^2.
```

Thus the reaction is absorbed directly into the Gronwall constant.  Strict positivity is not used.

There are two other bounds, needed for the square-barrier residual and for the repository's generic coefficient-regularity record:

```text
-c_u ≤ b*M^α,
|c_u| ≤ a + b*M^α,
```

whenever `0 ≤ u ≤ M`.

The three bounds have different roles:

```text
c_u ≤ a                 comparison-energy / Gronwall upper bound
-c_u ≤ b*M^α            square-heat subsolution damping loss
|c_u| ≤ a + b*M^α       BoundedOnIntervalStrip / linear Lipschitz field
```

# 1. Exact matched-divergence form

Put

```text
κ := -χ₀ = |χ₀| > 0,
g := R_x / (1+R)^β.
```

The chemotaxis equation is

```text
u_t = ν*u_xx + κ*∂x(g*u) + u*(a-b*u^α).
```

Define

```text
c_u(t,x) := a - b*u(t,x)^α.
```

Then the exact frozen linear operator is

```text
L_u φ := ν*φ_xx + κ*∂x(g*φ) + c_u*φ.
```

Equivalently, in the original sign convention,

```text
L_u φ := ν*φ_xx - χ₀*∂x(g*φ) + c_u*φ.
```

The target solution satisfies `u_t = L_u u` exactly.  A barrier only has to satisfy `w_t ≤ L_u w`.

This is the clean choice.  Do **not** define `c_u` as `f(u)/u`; the displayed formula `a-b*u^α` is valid without any division and therefore at zeros.

## Truncated equation

If the pre-energy equation is written with the positive part,

```text
u * (a - b*(u₊)^α),
```

then the frozen coefficient can already be defined as

```text
c_trunc := a - b*(u₊)^α.
```

It is bounded without any sign information because `u₊ ≥ 0`.  After the negative-part energy theorem gives `u ≥ 0`, use

```lean
positivePart_eq_self_of_nonneg
```

to rewrite `c_trunc = c_u`.  Strict positivity still does not enter.

# 2. Term-by-term Stampacchia bookkeeping

Let

```text
z := w-u,
η := z₊,
E(t) := ∫_0^1 η(t,x)^2 dx,
D(t) := ∫_0^1 |η_x(t,x)|^2 dx.
```

Subtract the weak subsolution and solution identities.  Because both use the **same divergence flux** `g times the dependent variable`, the drift difference is exactly `g*z`.  The Stampacchia chain rules give

```text
z*η = η^2,
z_x*η_x = |η_x|^2.
```

Assuming the Neumann/no-flux endpoint term vanishes, one obtains

```text
(1/2) E'(t) + ν D(t)
  ≤ χ₀ ∫_0^1 g*η*η_x dx + ∫_0^1 c_u*η^2 dx.
```

If `|g| ≤ G`, then Young's inequality gives

```text
|χ₀|*G*∫|η|*|η_x|
  ≤ (ν/2) D + (χ₀^2*G^2/(2ν)) E.
```

For the reaction term,

```text
c_u*η^2 = (a-b*u^α)*η^2 ≤ a*η^2,
```

because `b*u^α ≥ 0`.  Therefore

```text
(1/2) E'(t) + (ν/2)D(t)
  ≤ (a + χ₀^2*G^2/(2ν))*E(t),
```

and hence

```text
E'(t) ≤ (2*a + χ₀^2*G^2/ν)*E(t).
```

With `E(s)=0`, Gronwall gives `E=0`, hence `w≤u`.

For the repository's unit-diffusion normalization `ν=1`, this is

```text
E'(t) ≤ (2*a + χ₀^2*G^2)*E(t).
```

A coarser generic estimate may replace `a` by `a+b*M^α`, but this throws away the favorable logistic sign.  The sharp weak-comparison contribution is just `a`.

# 3. Why no strict positivity or delta floor is needed

The only scalar facts used are

```text
0 ≤ u ≤ M,
0 < α,
0 ≤ a,
0 ≤ b.
```

Mathlib supplies

```lean
Real.rpow_nonneg
Real.rpow_le_rpow
```

so

```text
0 ≤ u^α ≤ M^α.
```

There is no occurrence of

```text
u⁻¹,
u^(α-1),
u^(α-2),
log u,
or a derivative of u^α.
```

At `u=0`, `u^α` is harmless for `α>0`.  The positive floor is needed later for certain higher-order real-power Nemytskii derivatives, not for this zero-order comparison coefficient.

Indeed, even the repository's parameter hypothesis

```lean
p.hα : 0 < p.α
```

is sufficient here; the stronger theorem-level assumption `1 ≤ α` is unnecessary for this particular leaf.

# 4. Exact bounds and paste-ready scalar Lean lemma

A reusable scalar lemma is:

```lean
namespace ShenWork.Paper2.JensenReaction

/-- Frozen logistic zero-order coefficient. -/
def coeff (p : CM2Params) (u : ℝ → ℝ → ℝ) : ℝ → ℝ → ℝ :=
  fun t x => p.a - p.b * (u t x) ^ p.α

/-- All three bounds used by comparison, the square barrier, and coefficient
regularity.  No positive lower floor on `U` is assumed. -/
theorem coeff_pointwise_bounds
    (p : CM2Params) {U M : ℝ}
    (hU0 : 0 ≤ U) (hUM : U ≤ M) :
    p.a - p.b * U ^ p.α ≤ p.a ∧
      -(p.a - p.b * U ^ p.α) ≤ p.b * M ^ p.α ∧
      |p.a - p.b * U ^ p.α| ≤ p.a + p.b * M ^ p.α := by
  have hM0 : 0 ≤ M := le_trans hU0 hUM
  have hU_pow0 : 0 ≤ U ^ p.α := Real.rpow_nonneg hU0 _
  have hM_pow0 : 0 ≤ M ^ p.α := Real.rpow_nonneg hM0 _
  have hpow_le : U ^ p.α ≤ M ^ p.α :=
    Real.rpow_le_rpow hU0 hUM p.hα.le
  have hbu0 : 0 ≤ p.b * U ^ p.α :=
    mul_nonneg p.hb hU_pow0
  have hbM0 : 0 ≤ p.b * M ^ p.α :=
    mul_nonneg p.hb hM_pow0
  have hbu_le : p.b * U ^ p.α ≤ p.b * M ^ p.α :=
    mul_le_mul_of_nonneg_left hpow_le p.hb
  constructor
  · linarith
  constructor
  · linarith [p.ha, hbu_le]
  · apply abs_le.mpr
    constructor
    · linarith [p.ha, hbu_le, hbM0]
    · linarith [hbu0, hbM0]

end ShenWork.Paper2.JensenReaction
```

The first component is the Gronwall bound, the second is the barrier-loss bound, and the third is the generic strip bound.

```lean
namespace ShenWork.Paper2.JensenReaction

open Set
open ShenWork.Paper2.BFormPositiveDatumNegPart

/-- The frozen logistic coefficient is uniformly bounded on the strip. -/
theorem coeff_boundedOnIntervalStrip
    (p : CM2Params) {T M : ℝ} {u : ℝ → ℝ → ℝ}
    (hM : 0 ≤ M)
    (hu : ∀ t ∈ Set.Icc (0 : ℝ) T,
      ∀ x ∈ Set.Icc (0 : ℝ) 1, 0 ≤ u t x ∧ u t x ≤ M) :
    BoundedOnIntervalStrip T (coeff p u) := by
  let Clog : ℝ := p.a + p.b * M ^ p.α
  have hClog : 0 ≤ Clog := by
    dsimp [Clog]
    exact add_nonneg p.ha
      (mul_nonneg p.hb (Real.rpow_nonneg hM _))
  refine ⟨Clog, hClog, ?_⟩
  intro t ht x hx
  have hb := coeff_pointwise_bounds p (hu t ht x hx).1 (hu t ht x hx).2
  simpa [coeff, Clog] using hb.2.2

/-- Multiplication by the frozen logistic coefficient has the required uniform
linear Lipschitz constant. -/
theorem coeff_reaction_lipschitz
    (p : CM2Params) {T M : ℝ} {u : ℝ → ℝ → ℝ}
    (hM : 0 ≤ M)
    (hu : ∀ t ∈ Set.Icc (0 : ℝ) T,
      ∀ x ∈ Set.Icc (0 : ℝ) 1, 0 ≤ u t x ∧ u t x ≤ M) :
    ∃ L : ℝ, 0 ≤ L ∧
      ∀ ⦃t x r s : ℝ⦄,
        t ∈ Set.Icc (0 : ℝ) T →
        x ∈ Set.Icc (0 : ℝ) 1 →
        |coeff p u t x * r - coeff p u t x * s| ≤ L * |r-s| := by
  let L : ℝ := p.a + p.b * M ^ p.α
  have hL : 0 ≤ L := by
    dsimp [L]
    exact add_nonneg p.ha
      (mul_nonneg p.hb (Real.rpow_nonneg hM _))
  refine ⟨L, hL, ?_⟩
  intro t x r s ht hx
  have hc := (coeff_pointwise_bounds p
    (hu t ht x hx).1 (hu t ht x hx).2).2.2
  calc
    |coeff p u t x * r - coeff p u t x * s|
        = |coeff p u t x| * |r-s| := by
            rw [← mul_sub, abs_mul]
    _ ≤ L * |r-s| :=
      mul_le_mul_of_nonneg_right (by simpa [L] using hc) (abs_nonneg _)

end ShenWork.Paper2.JensenReaction
```

# 6. Load-bearing divergence versus nondivergence distinction

The matched weak operator is

```text
ν*φ_xx - χ₀*∂x(g*φ) + c_u*φ.
```

Here the reaction coefficient is exactly

```text
c_u = a-b*u^α.
```

However, the repository's older residual

```lean
neumannLinearDriftResidual B C φ
```

represents the nondivergence equation

```text
φ_t = φ_xx + B*φ_x + C*φ.
```

Expanding the chemotaxis divergence gives

```text
-χ₀*∂x(g*φ)
  = (-χ₀*g)*φ_x + (-χ₀*g_x)*φ.
```

Therefore the exact coefficients for that API are

```text
B(t,x)      := -χ₀*g(t,x),
C_full(t,x) := a - b*u(t,x)^α - χ₀*g_x(t,x).
```

The term `-χ₀*g_x` is not part of the logistic reaction; it is the divergence-to-nondivergence correction.  But it must be present in `C_full` if the nondivergence API is used.

Thus:

```text
matched-divergence weak comparison: C = c_u;
existing nondivergence residual:     C = c_u - χ₀*g_x.
```

Passing `B=-χ₀*g` and `C=c_u` to the nondivergence residual while claiming to represent the original divergence PDE leaves the uncancelled term

```text
-χ₀*g_x*u.
```

That is a coefficient mismatch, not a logistic-reaction problem.

If `|g_x| ≤ Lg`, then the full coefficient satisfies

```text
C_full ≤ a + |χ₀|*Lg,
-C_full ≤ b*M^α + |χ₀|*Lg,
|C_full| ≤ a + b*M^α + |χ₀|*Lg.
```

Hence the logistic contribution remains exactly `a` in the comparison upper bound and `b*M^α` in the barrier-loss bound; the `g_x` contribution is separate.

# 7. Square-heat barrier constant

The repository theorem

```lean
squareHeatResidualCore_nonpos_of_bounds
```

uses hypotheses of the form

```text
|B| ≤ A,
-C ≤ D,
A^2/2 + D ≤ Mbar
```

in the unit-diffusion normalization.

For the logistic coefficient alone,

```text
-c_u = -a + b*u^α ≤ b*M^α,
```

so the exact reaction loss is

```text
D_react := b*M^α.
```

For the full expanded coefficient,

```text
D_full := b*M^α + |χ₀|*Lg
```

is valid.  The square-barrier damping may therefore be chosen as

```text
Mbar ≥ A^2/2 + b*M^α + |χ₀|*Lg
```

for the repository's normalized diffusion.  With physical diffusion `ν`, the square-completion term becomes `A^2/(2ν)`.

Notice again that `a` does not increase `D_react`: the positive linear growth `+a*u` helps the lower barrier, while the negative logistic term produces the possible zero-order loss.

# 9. Minimal Jensen reaction-side target

For a weak matched-divergence producer, the clean interface is simply:

```lean
def JensenLogisticCoefficientData
    (p : CM2Params) (T M : ℝ) (u : ℝ → ℝ → ℝ) : Prop :=
  (∀ t ∈ Set.Icc (0 : ℝ) T,
    ∀ x ∈ Set.Icc (0 : ℝ) 1, 0 ≤ u t x ∧ u t x ≤ M) ∧
  (∀ t ∈ Set.Icc (0 : ℝ) T,
    ∀ x ∈ Set.Icc (0 : ℝ) 1,
      coeff p u t x ≤ p.a) ∧
  (∀ t ∈ Set.Icc (0 : ℝ) T,
    ∀ x ∈ Set.Icc (0 : ℝ) 1,
      -coeff p u t x ≤ p.b * M ^ p.α)
```

The producer is a direct application of `coeff_pointwise_bounds`.  The weak upper-gap theorem then consumes only the upper field `coeff ≤ a`; the square-heat barrier consumes only the lower field `-coeff ≤ b*M^α`.

# Final conclusion

The reaction side of Jensen is fully acyclic:

```text
u ≥ 0 and |u| ≤ M
  -> 0 ≤ u ≤ M
  -> 0 ≤ u^α ≤ M^α
  -> c_u := a-b*u^α is bounded
  -> c_u*z₊² ≤ a*z₊²
  -> homogeneous Gronwall closes.
```

No strict positivity, no compact-slab floor, and no higher Nemytskii regularity are required.  The only caution is to keep the operator representations straight:

```text
c_u                          = logistic zero-order coefficient;
c_u - χ₀*g_x                = full nondivergence zero-order coefficient.
```

With that distinction enforced, the logistic reaction is a routine bounded coefficient in the Jensen comparison.
