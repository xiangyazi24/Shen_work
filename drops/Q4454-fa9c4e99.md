# Definitive verdict

On the unit interval, the **linear** P3.4/H3.1 semigroup block should be built by explicit Neumann-cosine multipliers.  A general Henry/Kato sectorial-operator development is avoidable if the formalization is intentionally interval-specific.

However, the decay is **not generally**

```text
exp(-lambda_1 * t),  lambda_1 = pi^2.
```

That is the decay of the pure zero-mean Neumann heat semigroup.  The actual linearized chemotaxis-growth operator has mode multiplier

```text
sigma_k = sigma p uBar vBar lambda_k,
lambda_k = (k*pi)^2,
```

so its linear decay rate is controlled by the uniform negative spectral bound

```text
gap = - sup { sigma_k | k is a relevant mode }.
```

The existing repository theorem using `unitIntervalNeumannHeatSemigroupP0Compl` and rate `pi^2` closes only a **conditional pure-heat comparison**.  It does not yet identify the semigroup of the actual chemotaxis linearization.  Roadmap step 3's physical-PDE-to-mode bridge must be connected to `diagonalSemigroupCoeff (growth := fun k => sigma ... lambda_k)` before this can be called the P3.4 linear semigroup.

The full nonlinear local exponential conclusion is also more than the linear gap.  It still needs weighted-mode smoothing, a genuine fractional norm and its `C^1` embedding, a quadratic/local-Lipschitz nonlinear remainder estimate, and a Duhamel bootstrap.  These can all be done diagonally on `[0,1]`; abstract sectoriality is not logically necessary, but its analytic content is not optional.

# 1. Exact linearized semigroup

For a constant equilibrium `(uBar,vBar)`, step 3 gives the scalar mode equation

```text
U_k'(t) = sigma_k * U_k(t),
```

where

```text
sigma_k = -lambda_k - a*alpha
          + chi0 * nu*gamma*uBar^(m+gamma-1)*lambda_k
              / ((1+vBar)^beta * (mu+lambda_k)).
```

The elliptic perturbation is recovered algebraically:

```text
V_k(t) = nu*gamma*uBar^(gamma-1)/(mu+lambda_k) * U_k(t).
```

Therefore the linearized semigroup is

```lean
ShenWork.PDE.SectorialOperator.diagonalSemigroupCoeff
  (fun k =>
    ShenWork.Paper3.sigma p uBar vBar
      (ShenWork.Paper3.unitIntervalNeumannSpectrum.eigenvalue k))
  t a
```

and not the pure heat multiplier `exp(-lambda_k*t)`.

The exact coefficient solution is

```text
U_k(t) = exp(t*sigma_k) U_k(0).
```

## Relevant modes

There are two distinct cases.

### Positive logistic equilibrium

Here

```text
sigma_0 = -a*alpha < 0.
```

All modes are relevant.  A full spectral gap is any `delta > 0` satisfying

```text
forall k, sigma_k <= -delta.
```

The sharp value is

```text
delta_sharp = - sup_k sigma_k.
```

### Minimal equilibrium, `a=b=0`

Here

```text
sigma_0 = 0.
```

The constant mode is mass-neutral.  Decay is stated on the fixed-mass/zero-mean subspace:

```text
a 0 = 0,
forall k != 0, sigma_k <= -delta.
```

The sharp nonzero-mode gap is

```text
delta_sharp_nz = - sup_{k != 0} sigma_k.
```

This matches the repository's `LinearlyStable` convention, which quantifies only over nonzero modes.

# 2. A usable explicit uniform gap

Modewise negativity alone is not enough for a semigroup estimate unless it is upgraded to a **uniform** negative bound.  In this model the upgrade is elementary from the explicit threshold formula.

Write

```text
kappa(lambda) = nu*gamma*uBar^(m+gamma-1)*lambda
                / ((1+vBar)^beta*(mu+lambda)).
```

For `lambda > 0`,

```text
sigma(lambda) = (chi0 - chiCritical(lambda)) * kappa(lambda).
```

Suppose

```text
chi0 < chiStar,
chiStar = paperCriticalSensitivity S p uBar vBar.
```

Set

```text
eta = chiStar - chi0 > 0.
```

Every nonzero mode satisfies `chiStar <= chiCritical(lambda_k)`, hence

```text
chi0 - chiCritical(lambda_k) <= -eta.
```

Also `lambda -> lambda/(mu+lambda)` is increasing and `lambda_k >= lambda_1` for `k != 0`.  Therefore

```text
kappa(lambda_k) >= kappa(lambda_1) > 0
```

and

```text
sigma_k <= - eta * kappa(lambda_1).
```

Thus a concrete nonzero-mode gap is

```text
delta_nz =
  (paperCriticalSensitivity S p uBar vBar - chi0) *
  (nu*gamma*uBar^(m+gamma-1)*lambda_1 /
    ((1+vBar)^beta*(mu+lambda_1))).
```

For the positive equilibrium, take

```text
delta = min (a*alpha) delta_nz.
```

For the minimal equilibrium, take `delta = delta_nz` on `a 0 = 0`.

This is usually not the sharp gap, but it is explicit, positive, and perfectly adequate for Lean.

The first-mode value `pi^2` enters through the lower bound `lambda_1 <= lambda_k`; it is **not itself** the decay rate after chemotaxis is added.

Under an additional theorem identifying the critical threshold with the mode-one threshold, the same formula can be written with

```text
eta = chiCritical(lambda_1) - chi0.
```

But first-mode threshold dominance is not needed for the general `chi0 < sInf` gap argument above.

# 4. Exact decay target

There are two levels which must not be conflated.

## Linear target

For a weighted cosine coefficient norm

```text
||a||_{X^sigma}^2 = sum_k (1+lambda_k)^(2*sigma) |a_k|^2,
```

the linear diagonal semigroup should satisfy

```text
||T_lin(t)a||_{X^sigma}
  <= exp(-delta*t) ||a||_{X^sigma}.
```

The same multiplier acts on every weighted summand, so this is no harder than the existing unweighted `L^2` result.

For smoothing, one additionally needs, for `r <= s`,

```text
||T_lin(t)a||_{X^s}
  <= C * t^(-(s-r)) * exp(-delta0*t) * ||a||_{X^r},
```

with any `0 < delta0 < delta`.

The polynomial `t` factor is obtained modewise from the high-frequency estimate

```text
sigma_k <= C0 - c*lambda_k.
```

That high-frequency coercivity is a separate required output of step 3.  A mere uniform gap does not by itself prove analytic smoothing.

## Paper-level nonlinear target

The repository's intended H3.1/Theorem 2.2 conclusion is local exponential convergence in the stability package's `C^1` distance:

```text
N.c1Distance (u t) (fun _ => uBar)
+ N.c1Distance (v t) (fun _ => vBar)
  <= A * exp(-rate*t).
```

The initial perturbation is small in an `X_p^sigma`-type norm.  The current abstract range is

```text
1/2 < sigma < 1,
1 < pNorm.
```

For the concrete Hilbert cosine route (`pNorm = 2` and weight `(1+lambda)^(2*sigma)`), the direct coefficient proof of a `C^1` embedding requires

```text
3/4 < sigma.
```

This is exactly the threshold appearing in

```lean
derivativeReciprocalFractionalPowerWeight_summable_of_sigma_gt_three_quarters.
```

For a genuine `L^p` fractional-domain route, the usual one-dimensional embedding condition is

```text
2*sigma - 1/pNorm > 1.
```

Thus `1/2 < sigma` and `1 < pNorm` alone are not enough for the final `C^1` upgrade.  The concrete theorem must either use `sigma > 3/4` in the Hilbert route or add the correct `pNorm`-dependent embedding condition.

For the nonlinear theorem, choose

```text
0 < rate < delta.
```

Using the exact linear gap as the nonlinear rate is unnecessarily brittle.  The Duhamel convolution closes cleanly with a strictly smaller rate.

## C. Weighted smoothing

Prove the scalar multiplier estimate

```lean
theorem linearizedGrowth_weighted_exp_le
    {r s delta0 t : Real}
    (hrs : r <= s) (ht : 0 < t) (hdelta0 : 0 < delta0)
```

and sum it in weighted `ell^2`.

This is the interval replacement for abstract analytic-semigroup smoothing.

## D. Honest `C^1` reconstruction

Complete the derivative cosine series:

```text
sum_k -(k*pi/L) a_k sin(k*pi*x),
```

prove uniform convergence from the derivative reciprocal trace, identify it as the derivative of `cosineSeries`, and export

```lean
||cosineSeries a||_C1 <= Csigma * ||a||_Xsigma
```

for `sigma > 3/4`.

## E. Resolver transfer

Use the mode multiplier

```text
nu*gamma*uBar^(gamma-1)/(mu+lambda_k)
```

to control the linearized `v` perturbation.  For the nonlinear resolver, prove local Lipschitz control of

```text
u * (u^gamma - uBar^gamma)
```

near the positive equilibrium and use the existing elliptic gain.

## F. Nonlinear remainder

After subtracting the linearization, prove on a sufficiently small positive neighborhood

```text
||N(w)-N(z)||_{X^r}
  <= C * (||w||_{X^sigma}+||z||_{X^sigma}) * ||w-z||_{X^sigma}.
```

This includes the flux Nemytskii/product terms and the logistic remainder.

## G. Duhamel bootstrap

For

```text
w(t) = T_lin(t) w0 + integral_0^t T_lin(t-s) N(w(s)) ds,
```

use the weighted smoothing estimate and the norm

```text
sup_{t >= 0} exp(rate*t) ||w(t)||_{X^sigma}
```

with `0 < rate < delta` to obtain the small-data contraction or continuation estimate.  Then apply the `C^1` embedding and resolver transfer.

# 7. Final answer to the three questions

1. **Can abstract Henry sectorial theory be avoided?**  Yes, on `[0,1]`, by using the actual `sigma` cosine multiplier, weighted coefficient spaces, explicit smoothing, and a direct Duhamel contraction.  This avoids proving a general theorem about closed sectorial operators.  It does not avoid proving the corresponding decay/smoothing/remainder estimates.

2. **Is the rate `lambda_1 = pi^2`?**  Only for the pure zero-mean heat semigroup.  For the chemotaxis linearization the sharp rate is the negative spectral bound of `sigma_k`.  A convenient explicit rate is the threshold margin times the mode-one chemotactic coefficient, with `min (a*alpha)` for the positive equilibrium.  The nonlinear conclusion should use any strictly smaller positive rate.

3. **What does P3.4 consume?**

   ```text
   Paper 2 cosine basis/Hilbert reconstruction
   + step-3 physical linearization -> sigma_k coefficient ODE
   + uniform sigma gap
   + high-mode sigma_k <= C-c*lambda_k
   + weighted diagonal decay/smoothing
   + X^sigma -> C^1 embedding
   + elliptic resolver transfer
   + quadratic nonlinear remainder
   + small-data Duhamel existence/bootstrap.
   ```

The linear spectral-decay portion is close to a wiring task because `SectorialOperator.lean` is already generic in `growth`.  The genuinely new P3.4 work is the weighted smoothing, honest `C^1` norm bridge, and nonlinear Duhamel estimate around the actual `sigma` semigroup.
