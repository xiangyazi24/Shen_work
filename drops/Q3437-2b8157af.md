# Q3437: zero-right `lapL2sq` continuity producer audit

## Verdict

No existing committed theorem at `f77a3d35e53e85d20110c689566ce0e0696ca5bc` produces

```lean
ContinuousWithinAt (fun τ => lapL2sq u τ) (Set.Ici (0 : ℝ)) 0
```

and no existing theorem gives a zero-start slab representative/equality strong enough to derive it automatically.

The currently committed infrastructure gives:

* strict positive-time `lapL2sq` continuity from either literal `liftDeriv2` joint continuity or a continuous representative plus interior-spatial equality;
* bridge wrappers that consume the endpoint lap seam;
* H¹ endpoint energy continuity for `H1energy`, not H²/laplacian continuity;
* strict positive-time physical/chemotaxis representatives from classical data.

The narrowest honest next producer is therefore a **zero-slab representative theorem** or an explicitly assumed **zero-right lap trace theorem**. Among the options listed, the zero-slab representative route is the cheapest in Lean because it directly reuses the already-landed interval-integral continuity theorem.

`H1InitialEndpointData` controls only

```lean

```

```lean

There is no theorem in the current APIs upgrading H¹-energy trace to H²/laplacian trace. Such an upgrade would be analytically false without additional regularity/trace input.

theorem lapL2sq_continuousWithinAt_zero_of_zeroSlab_interior_eq_continuous
    {u : ℝ → intervalDomainPoint → ℝ} {F : ℝ → ℝ → ℝ} {b : ℝ}
    (hb0 : 0 < b)

        (Function.uncurry (fun t x => liftDeriv2 u t x))

```

Proof route:

1. Use the existing theorem with `a = 0`:
   ```lean
   have hIcc : ContinuousOn (fun τ => lapL2sq u τ) (Set.Icc (0 : ℝ) b) :=
     lapL2sq_continuousOn_Icc_of_strictSlab_interior_eq_continuous hFcont hEqInterior
   ```
2. Extract continuity within `Set.Icc 0 b` at `0`.
3. Convert from `𝓝[Set.Icc 0 b] 0` to `𝓝[Set.Ici 0] 0`, using local equality near `0` since `0 < b`:
   ```lean
   have hsets : Set.Icc (0 : ℝ) b =ᶠ[𝓝 (0 : ℝ)] Set.Ici (0 : ℝ) := by
     filter_upwards [Set.Iio_mem_nhds hb0] with y hy
     constructor
     · intro hyIcc; exact hyIcc.1
     · intro hyIci; exact ⟨hyIci, le_of_lt hy⟩
   ```
   Then use `hsets.nhdsWithin_eq` / `nhdsWithin_congr` depending on the local Mathlib API.

This proof is local and should be much cheaper than a dominated-convergence theorem.

## Alternative routes and why they are less attractive

### Dominated convergence for `(liftDeriv2 u τ)^2`

A dominated-convergence route would need a theorem like:

```lean
∀ᵐ x, Tendsto (fun τ => liftDeriv2 u τ x) (𝓝[Set.Ici 0] 0) (𝓝 (liftDeriv2 u 0 x))
```

plus an integrable domination for the squared fields near `0`, or some L² convergence statement. No current API provides those atoms. This would require more measure theory and more analytic input than the zero-slab representative route.

### Gradient-energy / laplacian trace package

A PDE regularity theorem could directly assert H²/lap trace at zero. That would be mathematically honest, but it is strictly more analytic than the generic zero-slab representative reducer. It is the right eventual source if the construction provides an H² initial trace, but it is not the cheapest next Lean edit.

### H¹ endpoint energy route

Do not use this. `H1InitialEndpointData` and `H1energy_continuousWithinAt_zero_of_initialEndpointData` concern `H1energy`, not `lapL2sq`. There is no valid theorem chain from H¹ energy convergence to squared-laplacian convergence in the current repository.
