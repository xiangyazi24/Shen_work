# 3. The exact shifted identity for `A(epsilon) Z(t)`

Let

```text
h = t - a > 0
```

and let the full mild candidate be

```text
Z(t) = S(h) Z0 + integral_a^t S(t-q) F(q) dq.
```

For `0 < epsilon <= h`, generator compatibility and Bochner commutation give

```text
A(epsilon) Z(t)
  = A(epsilon + h) Z0
    + integral_a^t A(epsilon + t - q) F(q) dq.
```

Changing variables

```text
r = t + epsilon - q
```

gives

```text
A(epsilon) Z(t)
  = A(epsilon + h) Z0
    + integral_epsilon^(h+epsilon) A(r) F(t+epsilon-r) dr.
```

The committed shift decomposition splits this into four pieces:

```text
Hom_epsilon
  = A(epsilon+h) Z0,

Base_epsilon
  = integral_epsilon^h A(r) F(t-r) dr,

Err_epsilon
  = integral_epsilon^h
      A(r) [F(t+epsilon-r) - F(t-r)] dr,

Tail_epsilon
  = integral_h^(h+epsilon) A(r) F(t+epsilon-r) dr.
```

The exact repository identity is assembled from

```lean
weightedMovingHeatL2Generator_apply_fullGeneratorCandidate_lag
intervalIntegral_weightedMovingHeatL2Generator_add_lag_eq_shift
intervalIntegral_weightedMovingHeatL2Generator_shift_decomposition
```

in `WholeLineWeightedRegularityGeneratorShift.lean` and used verbatim in
`WholeLineWeightedRegularityGeneratorShiftConvergence.lean`.

No spatial derivative of `F` is introduced. In particular, this is not a
`FluxDerivative` argument.

## 4.1 Homogeneous term

```text

```

by

```lean

```

because `h > 0`. There is no singular endpoint in this term.

## 4.2 Base term

The older concrete cancellation theorem gives

```text

  integral_0^h A(r) [F(t-r)-F(t)] dr
    + (S(h)-I)F(t).
```

The Hölder modulus gives

```text
norm (F(t-r)-F(t)) <= H r^theta,
```

so the cancellative integrand has the integrable majorant

```text
C H r^(-1+theta).
```

The relevant abstract producers in `WholeLineWeightedRegularityH2.lean` are


and require exactly `0 < theta`.

## 4.3 Translation error

The time shift introduced by applying `A(epsilon)` is not optional. Hölder
continuity gives, on `r ∈ [epsilon,h]`,

```text
norm (F(t+epsilon-r)-F(t-r)) <= H epsilon^theta.
```

Together with `norm A(r) <= C/r`, this yields

```text
norm Err_epsilon
  <= C H epsilon^theta log(h/epsilon).
```

The committed theorem is

```lean
intervalIntegral_weightedMovingHeatL2Generator_translation_error_norm_le
```

and the elementary logarithmic fence

```lean

```

gives

```text

  <= C H (2/theta) h^(theta/2) epsilon^(theta/2),
```

which tends to zero.

This is one of the two places where a mere uniform bound on `F` is
insufficient.

## 4.4 Upper tail

On `[h,h+epsilon]`, the generator is away from zero, so

```text
norm A(r) <= C h^(-1).
```

The uniform forcing bound `norm F <= K` gives

```text
norm Tail_epsilon <= C h^(-1) K epsilon -> 0.
```

The exact producer is

```lean

```

in the shift infrastructure.

Uniform boundedness is sufficient for this tail, but not for the Base or Err
terms.

# 10. Circularity and hidden-premise audit

## No `hWxx2`

The endpoint-generator convergence uses only the mild state, the analytic
`r^-1` generator bound, bounded forcing, Hölder forcing, and measurable
histories. `Wxx` appears only after the abstract generator has been obtained
and identified by the classical pointwise PDE.

## No `hWx2` circularity in the generator-domain step

The semigroup endpoint theorem itself does not assume `hWx2`. In the full
Paper 1 DAG, the already-produced `hWx2` is used upstream to obtain exact-weight
forcing control and downstream in the integration-by-parts pairing. It is not
proved from the generator-domain theorem and then fed back into it.

## No assumed `Wx` continuity

The first-gradient trajectory is the committed heat-gradient candidate. Its
square-root time modulus follows from the `t^-1/2` heat-gradient kernel and a
uniform exact-weight forcing bound. It does not use `Wxx`, the material time
derivative, or a `FluxDerivative` package.

## No uniform-forcing-only endpoint argument

Uniform forcing controls histories away from the endpoint and the short upper
tail. It does not control

```text
A(r)(F(t-r)-F(t))
```

or the translation error. The proof correctly requires `theta > 0`.

## No heat-Hessian `tau^-1` history

The singular order-one generator is never integrated against the raw source
without cancellation at zero. The terminal value is subtracted first, and
the constant-source part is evaluated by the semigroup orbit FTC.

## No stronger weight

All objects are in `WholeLineRealL2` at the exact headline exponent `eta`.
The restart lift, forcing trajectory, semigroup estimates, generator
cancellation, and pairing identity all stay at that weight.

## No `FluxDerivative`

The forcing Hölder route uses the already committed matched nonlinear source
and coefficient estimates. The generator-domain proof treats `F` as an
`L²` trajectory and never differentiates the chemotactic flux in space.

## No assumed package or axiom

The audited files end with `#print axioms` checks for their principal
theorems. The route consists of concrete heat-kernel CLMs, Bochner interval
integrals, exact semigroup identities, and Mathlib derivative/integral
lemmas.
