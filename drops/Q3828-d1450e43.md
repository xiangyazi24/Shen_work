# Q3828 — weak positive data for 1D Neumann chi-nonpositive chemotaxis

## Answer

**Yes**, the theorem should remain mathematically true for weak positive data

```text
u0 >= 0,  u0 not identically 0,
```

including data that vanish at the boundary, such as `u0(x)=x(1-x)`, provided “classical solution” means classical for every positive time with initial trace `u(t) -> u0` as `t -> 0+`. If one instead requires full classical regularity and Neumann compatibility at `t=0`, then boundary-vanishing examples such as `x(1-x)` generally fail the compatibility condition, so the statement must be interpreted in the standard positive-time/classical-with-initial-trace sense.

## (1) Local classical existence

**Yes**, local existence should hold.

For nonnegative continuous data, standard parabolic theory gives a mild solution locally in time. Since the principal part is the linear heat operator and the nonlinear chemotaxis term is lower-order once the elliptic equation for `v` is solved, the solution regularizes instantly: for every `t>0`, `u` is spatially smooth enough to be classical, and `v` is elliptically smooth from

```text
mu v - v_xx = nu u^gamma.
```

The lack of a strict positive floor at `t=0` is not a local-existence obstruction. It only prevents using a theorem whose hypotheses require `inf_[0,1] u0 > 0` at time zero.

## (2) Strict positivity for positive time

**Yes**, the solution should be strictly positive for `t>0`.

The nonnegative cone is invariant: at `u=0`, the reaction term is

```text
a*u - b*u^alpha = 0,
```

and the chemotaxis flux has the leading factor `u^m`, so it does not force the solution negative at zero. Once nonnegativity is known, the strong maximum principle / positivity of the Neumann heat kernel gives strict positivity for every positive time, because the initial datum is not identically zero.

Boundary vanishing at `t=0` is not a problem. The Neumann heat kernel is positive on the closed interval for `t>0`, so positivity reaches the boundary as well.

## (3) Global continuation for `chi0 <= 0`

**Yes**, the global continuation argument still works.

The maximum-principle estimate used for `chi0 <= 0` does not require a positive floor at time zero. It only requires a classical nonnegative solution on positive times. At a spatial maximum of `u`, one has `u_x=0` and `u_xx <= 0`. The elliptic equation and maximum principle for `v` give the sign needed for the chemotaxis term when `chi0 <= 0`, so the chemotaxis contribution is nonpositive at the maximum. Thus the usual logistic upper inequality remains:

```text
M'(t) <= M(t) * (a - b*M(t)^alpha),
```

up to the standard Dini/maximum-principle interpretation. This yields the same `L^infty` continuation bound as in the strict-floor case.

## Final conclusion

The strict positive floor is a convenience for a particular classical/local theory or spectral bootstrap route, not a mathematical necessity for the global existence theorem. For weak nonnegative nontrivial data, the correct formulation is:

```text
u is a mild solution from t=0,
u is classical for every t>0,
u(t) -> u0 as t -> 0+,
u(t,x) > 0 for all t>0 and x in [0,1],
and the chi0 <= 0 logistic maximum bound prevents blow-up.
```

So the mathematical answer is **YES**, with the compatibility-at-time-zero caveat above.
