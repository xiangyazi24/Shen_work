# Q3948: breaking the Level 2 circularity for the truncated chemotaxis ladder

## Verdict

Yes: the circularity you identified is real if `truncatedBFormSourceCoeff_bound_positive_time` is proved from only the fields currently exposed by the truncated mild existence data: boundedness and continuity of the positive-time slice. Continuity of `u(t)` plus elliptic regularity for `v(t)` gives a continuous flux

```text
F(t,x) = positivePart(u(t,x)) * v_x(t,x) / (1 + Rv(t,x))^β,
```

but it does **not** give `F ∈ W^{1,1}`. The missing ingredient is exactly some spatial first-derivative/BV/Lipschitz information on `u(t)`. `positivePart` preserves Lipschitz/BV/W¹,¹ regularity, but it does not create it from mere continuity.

Therefore the proof cannot be:

```text
continuous slice + ball bound
  ⇒ flux W¹,¹
  ⇒ chemDiv coefficients O(1)
  ⇒ H¹
  ⇒ gradient bound.
```

That is circular at the first arrow.

The non-circular route is an **iterate-level positive-window Lipschitz bootstrap**, not the existing `χ₀ = 0` source tower. The key is that the chemotaxis leg must be estimated after rewriting the conjugate kernel term by integration by parts **at the operator level** on positive windows where the previous iterate has a controlled derivative. This avoids the nonintegrable `1/(t-s)` singularity.

## What is wrong with the naive Level 2 proof?

For a fixed positive time `s`, the Level 2 theorem wants

```lean
∃ C, 0 ≤ C ∧ ∀ k,
  |truncatedBFormSourceCoeff p u s k| ≤ C.
```

The logistic part is fine: bounded slice plus continuity gives bounded cosine coefficients.

The chemotaxis part is

```text
truncatedChemDivSourceCoeff p u s k
  = kπ * sineInner(F(s), k),
```

where `F = truncatedChemFluxLifted p (u s)`. From only `F` continuous, the crude estimate is

```text
|kπ * sineInner(F,k)| ≤ C k,
```

because `|sineInner(F,k)| ≤ 2 ||F||∞`. To improve this to `O(1)`, one integrates by parts in `x`:

```text
kπ * sineInner(F,k) = boundary + cosCoeff(F').
```

Since the Neumann resolver gives `F(0)=F(1)=0`, the boundary vanishes and the coefficient is controlled by `∫ |F'|`. But this requires `F ∈ W^{1,1}` or at least bounded variation/absolute continuity.

Continuity of `u` does not imply that. A continuous function can fail to be BV or W¹,¹; multiplying it by a nonzero C¹ function still need not be BV. `positivePart` is only Lipschitz as a scalar map: it preserves W¹,¹/Lipschitz/BV when the input already has that regularity, but it does not upgrade a merely continuous function.

So the proposed proof of `truncatedBFormSourceCoeff_bound_positive_time` is circular if the required flux regularity is obtained from the later gradient bound.

## Why the crude spectral route does not start the ladder

Your `O(k)` calculation is also correct. If the only bound on the source is

```text
|src_k(τ)| ≤ A + B k,
```

then the standard eigenvalue gain gives

```text
|Duh_k(t)| ≤ (A + B k) / λ_k = O(1/k),
```

and therefore

```text
λ_k |Duh_k(t)|² = O(1).
```

This is not summable. Splitting the Duhamel integral helps for the early interval `[0,t/2]`, because the heat factor gives exponential damping there, but it does not help on the near interval `[t/2,t]` unless one already has a better source bound on that positive window. Thus some positive-window `O(1)` source estimate is genuinely needed before the H¹ step.

## The correct non-circular route

The right route is not:

```text
limit continuity ⇒ limit flux W¹,¹.
```

It is:

```text
iterate n has positive-window Lipschitz bound
  ⇒ iterate n flux is W¹,∞ on that window
  ⇒ the conjugate kernel chem leg can be rewritten as an ordinary heat source
     with bounded source F'_n
  ⇒ iterate n+1 has a positive-window Lipschitz bound
  ⇒ uniform iterate Lipschitz bound on positive windows
  ⇒ limit is Lipschitz on positive windows
  ⇒ limit flux is W¹,∞
  ⇒ limit source coefficients are O(1) on positive windows.
```

The important technical point is how to handle the apparent `1/(t-s)` singularity.

### Do not differentiate the weak chem leg with only bounded flux

The truncated mild map contains the chemotaxis leg in B-form/conjugate-kernel form:

```text
B_N(r)F(x) = - ∫₀¹ ∂_y K_N(r,x,y) F(y) dy.
```

If one differentiates this in `x` while only using `||F||∞`, one gets

```text
∂_x B_N(r)F(x) = - ∫₀¹ ∂_x∂_y K_N(r,x,y) F(y) dy,
```

and the kernel estimate is of size `r^{-1}`. The near-time integral

```text
∫₀ᵗ (t-s)^{-1} ds
```

diverges. This is the singularity you found.

This estimate is too rough and is the wrong way to prove the truncated G1 line.

### Use operator-level integration by parts first

For the Picard iterate, on a positive window, the previous iterate has a Lipschitz/W¹,∞ bound. Therefore its flux `F_n(s)` is W¹,∞ and the endpoint flux vanishes:

```text
F_n(s,0) = F_n(s,1) = 0.
```

Then integrate by parts in the **operator** before differentiating in `x`:

```text
B_N(r)F_n
  = - ∫₀¹ ∂_y K_N(r,x,y) F_n(y) dy
  =   ∫₀¹ K_N(r,x,y) F'_n(y) dy
  = S_N(r)(F'_n)(x).
```

Hence the chemotaxis contribution in the mild map becomes an ordinary heat-semigroup Duhamel term with scalar source `F'_n`:

```text
(-χ₀) * B_N(t-s)F_n(s)
  = -χ₀ * S_N(t-s)(F'_n(s)).
```

The full iterate update can be rewritten on the positive window as

```text
u_{n+1}(t)
  = S_N(t-a) u_{n+1}(a)
    + ∫_a^t S_N(t-s) [ L_n(s) - χ₀ F'_n(s) ] ds,
```

where `L_n = truncatedLogisticLifted p (u_n s)` and `a > 0` is a restart time, usually `a = lo/2` for a read window `[lo,hi]`.

Now differentiating in `x` uses only the first kernel derivative:

```text
∂_x S_N(t-s)[L_n(s) - χ₀F'_n(s)],
```

whose L∞→L∞ bound has the integrable singularity `(t-s)^(-1/2)`, exactly the one handled by `gradDuhamel_sup_bound`.

This is the bound that handles the singularity. The `1/(t-s)` term disappears because one derivative is kept on the flux, not placed on the heat kernel.

## The positive-window Volterra estimate

Fix a positive window

```text
0 < lo ≤ hi ≤ T.
```

Use a restart time `a = lo/2`. Define an iterate gradient seminorm on the enlarged window:

```text
G_n(lo,hi) = sup { |∂_x u_n(s,x)| : s ∈ [lo/2, hi], x ∈ [0,1] }.
```

The base iterate is homogeneous heat:

```text
u_0(t) = S_N(t)u₀,
```

so the existing heat gradient atom gives

```text
G_0(lo,hi) ≤ Cg * M / sqrt(lo/2).
```

Assume `G_n(lo,hi) ≤ G`. Then the flux derivative has a bound of the form

```text
||F'_n(s)||∞ ≤ A_F(p,M) + B_F(p,M) * G,
```

uniformly for `s ∈ [lo/2,hi]`. Schematically:

```text
F_n = positivePart(u_n) * resolverGrad(u_n) / (1 + resolver(u_n))^β,
```

and its derivative is bounded by terms involving:

```text
|u_n'| * |resolverGrad(u_n)|,
|positivePart(u_n)| * |resolverSecond(u_n)|,
```

plus the denominator derivative if the denominator is included. The resolver terms are controlled from the ball bound and elliptic estimates; the only iterative term is linear in `|u_n'|`. If positivity has already been proved on the window, `positivePart(u_n)=u_n`; otherwise use the Sobolev/Lipschitz chain rule

```text
|(positivePart ∘ u_n)'| ≤ |u_n'|   a.e.
```

The truncated logistic source is bounded by a constant

```text
||L_n(s)||∞ ≤ A_L(p,M).
```

Thus the total scalar source in the restarted Duhamel representation satisfies

```text
||L_n(s) - χ₀ F'_n(s)||∞
  ≤ A_L(p,M) + |χ₀| * (A_F(p,M) + B_F(p,M) * G_n(lo,hi)).
```

Applying the heat-gradient Duhamel atom on `[a,t]` gives a recurrence of the form

```text
G_{n+1}(lo,hi)
  ≤ Cg * M / sqrt(lo/2)
    + 2 * Cg * sqrt(hi - lo/2)
      * (A_L + |χ₀| * A_F + |χ₀| * B_F * G_n(lo,hi)).
```

Equivalently,

```text
G_{n+1} ≤ A_win + B_win * G_n.
```

There are two Lean-friendly ways to close this.

1. **Small-window contraction.** Prove the estimate on windows whose length makes

```text
B_win = 2 * Cg * sqrt(hi - lo/2) * |χ₀| * B_F < 1.
```

Then

```text
sup_n G_n ≤ A_win / (1 - B_win).
```

Cover a larger positive interval by finitely many such windows, restarting at the left endpoint of each one. This is usually the easiest formal route.

2. **Fractional/Volterra Gronwall.** Work with the unsupremized inequality

```text
g_{n+1}(t)
  ≤ Cg*M/sqrt(t)
    + C ∫₀ᵗ (t-s)^(-1/2) (1 + g_n(s)) ds.
```

The convolution of `(t-s)^(-1/2)` with `s^(-1/2)` is finite:

```text
∫₀ᵗ (t-s)^(-1/2) s^(-1/2) ds = π.
```

This gives a finite positive-time profile, often expressible using a Mittag-Leffler/fractional Gronwall envelope. This is mathematically clean but likely heavier in Lean than the small-window contraction.

For the repo, I would implement the small-window/restart version first.

## Passing the iterate bound to the limit

You do not need convergence of derivatives. Uniform convergence of `u_n(t)` to the Picard limit plus a uniform Lipschitz bound is enough.

If for all `n`,

```text
|u_n(t,x) - u_n(t,y)| ≤ G |x-y|
```

and `u_n(t,·) → u(t,·)` uniformly, then

```text
|u(t,x) - u(t,y)| ≤ G |x-y|.
```

So the limit slice is Lipschitz on `[0,1]`. Then:

```text
positivePart(u(t)) ∈ W¹,∞,
resolverGrad(u(t)) is W¹,∞/C¹ with bounds from elliptic estimates,
F(t) = truncatedChemFluxLifted p (u(t)) ∈ W¹,∞.
```

This gives the Level 2 chemDiv coefficient bound for the limit by the sine-coefficient IBP.

For the formal theorem, prefer a window version:

```lean
theorem truncatedBFormSourceCoeff_bound_on_positive_window
    {p : CM2Params} {u₀ : intervalDomainPoint → ℝ}
    (DT : TruncatedConjugateMildExistenceData p u₀)
    {lo hi : ℝ} (hlo : 0 < lo) (hlohi : lo ≤ hi) (hhiT : hi ≤ DT.T) :
    ∃ C : ℝ, 0 ≤ C ∧ ∀ s ∈ Set.Icc lo hi, ∀ k : ℕ,
      |truncatedBFormSourceCoeff p
        (truncatedConjugatePicardLimit p u₀ DT.T) s k| ≤ C
```

Then derive the pointwise version by taking `lo = s`, `hi = s`, or more usefully use `[t/2,t]` for the coefficient ladder.

The single-time theorem is not enough for Duhamel estimates; the ladder needs a uniform source envelope on a positive time window.

## Does the existing `g1_kernel_bound` apply to truncated iterates?

Not as currently packaged.

The theorem `g1_kernel_bound` itself is an abstract two-atom estimate: if the derivative of a value splits as

```text
∂ₓS(t)u₀ + ∫₀ᵗ ∂ₓS(t-s)L(s) ds
```

and `|L| ≤ CL`, it returns the profile

```text
G1profile p M t = Cg/sqrt(t)*M + Cg*2*sqrt(t)*CL.
```

So the **kernel atom** can be reused once the truncated chemotaxis leg has been rewritten as an ordinary bounded scalar source. But the existing `G1profile`, `CL`, and tower wiring are logistic-only and live in the `χ₀ = 0` regime.

Concretely:

* `IntervalPicardIterateUniform.lean` says in its header that the G1 line is for the `χ₀ = 0` map and that `CL := M*(p.a + p.b*M^p.α)` is the logistic source bound.
* `IntervalPicardSourceTower.lean` carries `TowerInputs.hχ0 : p.χ₀ = 0`.
* Its `SourceWin` agreement is with `cosineCoeffs (logisticLifted p ...)`, not with the truncated B-form source.

Therefore the existing `IntervalPicardSourceTower` cannot be used as the truncated Level 2 proof. Reuse the lower kernel estimates (`gradDuhamel_sup_bound`, the homogeneous heat gradient atom), but build a new truncated window-gradient carrier and a new truncated window-source-envelope producer.

* `gradDuhamel_sup_bound` for `∂ₓS(t-s)` applied to a bounded scalar source. In the truncated case the scalar source is `L_n - χ₀ F'_n` after operator-level IBP.

## How this fixes the Q3945 ladder

Once `truncatedBFormSourceCoeff_bound_on_positive_window` is available, the Q3945 Sobolev ladder is sound, but it should be phrased window-locally/restarted:

```text
positive-window source O(1)
  ⇒ restart at τ=t/2
  ⇒ H¹ at time t
  ⇒ source ℓ² on a later positive window
  ⇒ H² / gradient ℓ¹
  ⇒ source ℓ¹
  ⇒ Σ λ_k |c_k(t)| < ∞.
```

The Level 2 theorem must not be proved from the final gradient bound. It must be produced by the iterate/window Lipschitz bootstrap described above. After that, the higher ladder steps may use the coefficient machinery normally.

One small correction to the earlier mental model: the bootstrap should not rely on a source bound at a single time. Eigenvalue gain for Duhamel tails needs a uniform envelope on a time interval such as `[t/2,t]`. Therefore the central product should be a `WindowSourceEnvelope 0` or equivalent window theorem, not merely

```text
∀ fixed s, ∃ C_s, ∀ k, |src_k(s)| ≤ C_s.
```

## Bottom line

1. **Yes, the circularity is real.** Continuity plus elliptic regularity does not imply the flux is W¹,¹.
2. **The correct non-circular route is an iterate-level positive-window Lipschitz/Volterra bootstrap.** First prove uniform Lipschitz bounds for Picard iterates on windows bounded away from zero; pass those bounds to the limit; then obtain flux W¹,∞ and the bounded source coefficients.
3. **The iterate approach works only if the chemotaxis leg is handled by operator-level IBP.** Do not estimate `∂ₓB_N(t-s)F` by `||∂ₓ∂ᵧK||₁ ||F||∞`; that gives the divergent `1/(t-s)`. Instead, use zero boundary flux and `F_n ∈ W¹,∞` to rewrite `B_N(t-s)F_n = S_N(t-s)F'_n`, then apply the existing `∂ₓS_N` gradient Duhamel bound with its integrable `(t-s)^(-1/2)` singularity.
