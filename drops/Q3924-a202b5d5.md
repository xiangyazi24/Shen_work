# Q3924 — proof strategy for `truncatedPositiveTimeSpectralData_of_existenceData`

## Executive answer

The efficient route is **not** to prove all 11 fields independently from raw semigroup estimates.  Split the proof into three bottom packages and then fill the structure by projections:

1. **Coefficient ladder package** for fields **1–6**:
   use the truncated coefficient mild identity plus the `WindowCoefficientEnvelope` / pass-4 ladder.  The pass-4 envelope should provide enough decay for the time derivative, gradient, Laplacian-test, and source-test summability fields.

2. **Positive-part Sobolev/classical calculus package** for fields **7–8 and 10–11**:
   do **not** use analyticity as the first route.  The cheaper route is a `C¹` zero-crossing lemma: for a `C¹` real function, `positivePart f` is differentiable away from the countable set `{x | f x = 0 ∧ f' x ≠ 0}`.  This set is discrete, hence countable.  Derivative bounds come from the chain rule and the slice `C¹`/resolver bounds.

3. **Continuity package** for field **9**:
   this is direct: continuity of `positivePart`, continuity of the resolver value/gradient, and positivity of the denominator.

Important answer to (A):

* The fact that the chemotaxis source is only `ℓ²` is still enough for **field 3** (`grad_l1`) because Duhamel gives `1/λ_k ~ 1/k²`, so after multiplying by `k`, one gets `q_k/k`, and `ℓ² · (1/k) ∈ ℓ¹` by Cauchy-Schwarz.
* But `ℓ²` source alone is **not enough** for **field 1** (`deriv_l1`) if `truncatedPicardCoeffTimeDeriv = src - λ a`.  That derivative contains the endpoint source.  To prove field 1 as an `ℓ¹` statement, use the pass-4 / `WindowCoefficientEnvelope` output, not just the one-step heat gain.

Important answer to (B):

* The real-analytic zero-set route is stronger than needed and likely too expensive/circular.  The inhomogeneous heat solution at time `t` need not be analytic unless the time-`t` source has analytic spatial regularity.  A mild solution with merely continuous/truncated source gives smoothing from the past but an endpoint source obstruction for time derivatives.
* Use `C¹` zero-crossing: the bad set for differentiability of `positivePart ∘ f` is `{f = 0, f' ≠ 0}`, which is countable for `C¹ f` on an interval.

Important answer to (C):

* `chem_cont` is straightforward once the resolver field has continuity.  The only special detail is the denominator: prove `0 < 1 + R` on `[0,1]`, typically from `0 ≤ R`.

### Field 3: is the chemotaxis `ℓ²` source enough for gradient `ℓ¹`?

Yes.  The bottom lemma is:

```lean
/-- Duhamel `1/λ` smoothing upgrades an `ℓ²` source to gradient-weighted `ℓ¹`. -/
theorem grad_l1_of_l2_source_duhamel_gain
    {q : ℕ → ℝ} {b : ℕ → ℝ}
    (hq : Summable (fun k : ℕ => q k ^ 2))
    (hb : ∀ k, |b k| ≤ |q k| / (λN k + 1)) :
    Summable (fun k : ℕ => |b k| * ((k : ℝ) * Real.pi)) := by
  -- Compare `k / (λ_k + 1)` to `C / (k+1)`.
  -- Then use Cauchy-Schwarz: `∑ |q_k|/(k+1)` is summable from `q ∈ ℓ²`
  -- and `∑ 1/(k+1)^2`.
  sorry
```

For the final proof, you can use the stronger pass-4 envelope instead:

```lean
have hgrad : Summable (fun k =>
    |truncatedPicardCoeff p u₀ U t k| * ((k : ℝ) * Real.pi)) := by
  -- from `lap_weighted_l1`: since `k*pi ≤ C*(λ_k+1)` and the `k=0` term is zero.
  exact grad_l1_of_lap_weighted_l1 H.lap_weighted_l1
```

Add the wrapper:


### Field 1 warning

Do not try to prove

```lean
Summable (fun k => |truncatedPicardCoeffTimeDeriv p u₀ U t k|)
```

from only `source ∈ ℓ²`.  Since


it contains the endpoint source.  If the source is merely `ℓ²`, this is not an `ℓ¹` theorem.  Use pass-4 / `WindowCoefficientEnvelope` to obtain either:

# Package II — positive-part differentiability off a countable set

## Do not use analyticity first

The analytic-zero-set argument is not the best route.  It may even be false at the needed level if the source is only finitely smooth at the current endpoint.  The cheap theorem is a `C¹` zero-crossing lemma.

## Bottom lemma: bad zero set is countable

```lean
/-- For a `C¹` function on an interval, zeros where the derivative is nonzero are
isolated, hence countable. -/
theorem countable_zero_set_of_hasDerivAt_ne_zero
    {f f' : ℝ → ℝ} {I : Set ℝ}
    (hI : I = Set.Icc (0 : ℝ) 1)
    (hf : ∀ x ∈ I, HasDerivAt f (f' x) x)
    (hfcont : ContinuousOn f' I) :
    Set.Countable {x | x ∈ I ∧ f x = 0 ∧ f' x ≠ 0} := by
  -- At `x` with `f x = 0` and `f' x ≠ 0`, derivative nonzero gives local strict
  -- monotonicity/sign change, so no other zero in a small ball.
  -- Isolated subsets of second-countable spaces are countable.
  sorry
```

If proving local strict monotonicity is inconvenient, prove the simpler `separated by rational intervals` version: for each `m`, restrict to `|f' x| ≥ 1/(m+1)` and use continuity of `f'` plus derivative definition to isolate zeros; then countable union over `m`.

## Bottom lemma: positivePart differentiable off that set

```lean
/-- `positivePart ∘ f` is differentiable outside the countable crossing-zero set. -/
theorem positivePart_comp_differentiable_off_countable
    {f f' : ℝ → ℝ}
    (hf : ∀ x ∈ Set.Icc (0 : ℝ) 1, HasDerivAt f (f' x) x)
    (hfcont : ContinuousOn f' (Set.Icc (0 : ℝ) 1)) :
    ∃ E : Set ℝ, E.Countable ∧
      ∀ x ∈ Set.Icc (0 : ℝ) 1, x ∉ E →
        DifferentiableAt ℝ (fun y => positivePart (f y)) x := by
  refine ⟨{x | x ∈ Set.Icc (0:ℝ) 1 ∧ f x = 0 ∧ f' x ≠ 0}, ?_, ?_⟩
  · exact countable_zero_set_of_hasDerivAt_ne_zero rfl hf hfcont
  · intro x hx hxE
    by_cases hpos : 0 < f x
    · -- locally `positivePart(f)=f`; use continuity of f.
      sorry
    · by_cases hneg : f x < 0
      · -- locally zero.
        sorry
      · -- f x = 0; since x not in bad set, f' x = 0; positivePart(f)=o(|x|), derivative 0.
        sorry
```

## Bottom lemma: derivative bound for positivePart

```lean
/-- Derivative of `positivePart ∘ f` is bounded by the derivative bound for `f`. -/
theorem deriv_positivePart_comp_bound
    {f : ℝ → ℝ} {B : ℝ}
    (hB : 0 ≤ B)
    (hfderiv : ∀ x ∈ Set.Icc (0 : ℝ) 1,
      |deriv f x| ≤ B) :
    ∀ x ∈ Set.Icc (0 : ℝ) 1,
      |deriv (fun y => positivePart (f y)) x| ≤ B := by
  -- If `f x > 0`, derivative equals `deriv f x`.
  -- If `f x < 0`, derivative is 0.
  -- If `f x = 0`, derivative exists only when needed; as a bound on `deriv`, use
  -- `deriv_eq_zero_of_not_differentiableAt` or prove by cases with `f' = 0`.
  sorry
```

In Mathlib, `deriv` at non-differentiable points is `0`, so the bound is still safe.

## Direct answers to (A), (B), (C)

### (A) Is `ℓ²` chem source + heat `1/λ` gain enough for field 3?

Yes.  If `q ∈ ℓ²` and `a_k ≲ q_k / λ_k`, then

```text
k |a_k| ≲ |q_k| / k,
```

and `∑ |q_k|/k < ∞` by Cauchy-Schwarz.  So field 3 is safe.

But this is **not enough for field 1**.  The time derivative contains the endpoint source.  For `deriv_l1`, use pass-4 / `WindowCoefficientEnvelope`, or prove a direct `ℓ¹` source envelope.

### (B) Should we use analyticity for positivePart differentiability off countable?

No, not as the primary route.  Use the cheaper `C¹` zero-crossing route.  For `C¹ f`, the nondifferentiability of `positivePart f` occurs only at zeros with `f' ≠ 0`; that set is isolated/countable.  This avoids proving real analyticity of the Picard limit slices.

If the repo already has analytic slice results, they can be used, but they are stronger than needed and risk becoming circular.

### (C) Does `chem_cont` need special attention?

Only the denominator.  Prove:

```lean
∀ x ∈ Set.Icc (0 : ℝ) 1, 0 < 1 + intervalNeumannResolverR p w x
```

from resolver nonnegativity.  Then continuity follows from:

```text
positivePart continuous
intervalDomainLift w continuous
resolverGradReal continuous
intervalNeumannResolverR continuous
rpow/div continuity on positive denominator
```

No countability or spectral summability is involved in field 9.
