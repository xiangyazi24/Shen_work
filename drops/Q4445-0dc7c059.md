# Executive verdict

The long pole really splits into **two independent analytic engines** plus two short adapters:

1. **Theorem 2.2, stable side:** the semigroup of the **actual linearized chemotaxis operator** in a genuine fractional phase space, followed by a nonlinear Duhamel bootstrap. This is the reusable local exponential-stability engine.
2. **Theorems 2.3–2.5, global side:** positive-time compactness plus a branch-specific global mechanism—entropy dissipation for Theorem 2.4(i)/(ii) and Theorem 2.5(i), but not for every branch.
3. **Moment-to-uniform convergence:** once moment decay and slice compactness are available, this is a relatively short sequential contradiction argument, not a new PDE mountain.
4. **Uniform-to-`C¹` exponential convergence:** once the orbit enters the local stable neighborhood, this is a restart application of the Theorem 2.2 engine. It is mostly wiring, although the quantifier order of the exponential prefactor must be corrected.

There are four audit corrections to make before coding.

## Audit correction A: the current interval PDE is only the `m = 1` flux

The repository has already recorded this precisely:

```lean
intervalDomainChemotaxisDiv_actual_flux_u_power_one
intervalDomainChemotaxisDiv_actual_independent_of_m
```

The committed operator is

```text
∂ₓ (u · vₓ / (1+v)^β),
```

whereas Paper 3 uses

```text
∂ₓ (u^m · vₓ / (1+v)^β).
```

Therefore the general-`m` entropy computation below **cannot** be derived from the current

```lean
IsPaper2ClassicalSolution intervalDomain p T u v
```

unless `p.m = 1`. Before proving the general Paper 3 Lyapunov identity, add a Paper-3-specific interval flux and solution predicate, or generalize the concrete interval `chemotaxisDiv` itself. Otherwise the formal computation will silently prove the wrong PDE.

## Audit correction B: differentiate the entropy, not the theta moment

The present

```lean
IntervalDomainThetaDissipationDerivativeDecayData
```

asks for

```text
D'(t) ≤ -rate · D(t),
D(t) = ∫ (u-u*) (u^θ-u*^θ).
```

That is substantially stronger than the paper’s route and is not the right first target. The paper differentiates an entropy `F`, proves

```text
F'(t) ≤ -κ D(t),
```

integrates this to obtain `D ∈ L¹(t₀,∞)`, and then obtains `D(t) → 0` from uniform continuity or from a direct compactness/LaSalle argument. The replacement producer should therefore be an **entropy-controls-moment** package, not a derivative-of-moment package.

## Audit correction C: the bare heat gap `π²` is not the full linearized gap

`IntervalDomainSectorial.lean` currently compares the nonlinear orbit to

```lean
unitIntervalNeumannHeatSemigroupP0Compl
```

and then inserts the bare heat decay `exp (-π² t)`. For the actual stability threshold, the relevant diagonal rates are

```text
σₙ = sigma p u* v* λₙ,
```

not merely `-λₙ`. The stable decay constant is any number below the **spectral bound gap**

```text
ω* = - sup { σₙ : n ≠ 0 }
```

with the zero mode treated separately. It is not automatically the first nonzero Neumann eigenvalue, and it is not generally enough to prove decay of the bare heat semigroup.

For the nonminimal equilibrium the constant mode is damped by the logistic linearization. For the minimal model it is neutral and must be removed by the mass constraint.

## Audit correction D: the global exponential prefactor should normally depend on the orbit

A local stability theorem gives uniform local constants `A₀, rate, δ`. Qualitative global convergence gives an orbit-dependent entrance time `T₀`. Restarting yields

```text
dist(t) ≤ A₀ exp (-rate · (t-T₀)) dist(T₀),
```

so the rewritten bound `C exp (-rate t)` has an orbit-dependent `C`. Several current conclusion-shaped frontier fields place `∃ A, ∃ rate` **before** `∀ u v`; that requires a uniform entrance-time theorem and is stronger than the usual global-stability conclusion. The faithful order is at least

```lean
∃ rate > 0, ∀ u v, hypotheses → ∃ C > 0, ...
```

or simply the already-defined orbitwise predicate

```lean
ExponentialC1Convergence intervalDomain N u v uStar vStar.
```

## 1.5 The dissipation inequality used downstream

Complete the square in the gradient terms:

```text
-Aₘ X² + χ₀AₘXY
 = -Aₘ (X-χ₀Y/2)² + χ₀²AₘY²/4
≤ χ₀²AₘY²/4,
```

where

```text
X = |Uₓ|/U^m,
Y = |Vₓ|/(1+V)^β.
```

Thus

```text
F'(t)
≤ (χ₀²Aₘ/4) ∫ Vₓ²/(1+V)^(2β)
  - b Dα(t),
```

with

```text
Dα(t) = ∫ (U-uStar)(U^α-uStar^α)
      = chemotaxisThetaDissipation intervalDomain uStar p.α (u t).
```

A branch-specific elliptic estimate of the form

```text
∫ Vₓ²/(1+V)^(2β) ≤ C_ell · Dα(t)
```

then gives

```text
F'(t) ≤ -κ Dα(t),
κ = b - χ₀² Aₘ C_ell / 4.
```

Under the paper’s first strong-threshold formula this is rewritten as

```text
κ = b · (1 - (χ₀/χ**)²) > 0.
```

The second strong-threshold branch uses the eventual lower bound on `v`, chooses a positive-time start `t₀`, and obtains a corresponding `κ_ε > 0` on `[t₀,∞)`.

The minimal `m=1`, `a=b=0` branch uses

```text
h₁(s) = s-uStar-uStar·log(s/uStar),
```

and the exact identity

```text
F'
 = -uStar ∫ Uₓ²/U²
   +χ₀uStar ∫ UₓVₓ/[U(1+V)^β].
```

Its coercivity step is different and should remain a separate producer.

## 3.2 Direct sequential LaSalle argument

Let

```text
E(t) = entropy,
D(t) = theta dissipation.
```

From `E ≥ 0` and `E' ≤ -κD`, `E` is decreasing and has a finite limit `E∞`. For any sequence `tₙ → ∞`, compactness gives a subsequence such that

```text
u(tₙ+·) → u∞(·)
```

locally uniformly. For each fixed `L > 0`,

```text
κ ∫₀ᴸ D(tₙ+s) ds
≤ E(tₙ)-E(tₙ+L)
→ E∞-E∞ = 0.
```

Local uniform convergence and boundedness pass the integral to the limit:

```text
∫₀ᴸ D∞(s) ds = 0.
```

The integrand is continuous and nonnegative, hence

```text
D∞(s)=0
```

for every `s`. Since `r ↦ r^theta` is strictly increasing for `theta>0`,

```text
(r-uStar)(r^theta-uStar^theta)=0
```

if and only if `r=uStar`. Therefore

```text
u∞(s,x)=uStar
```

for all `s,x`. The elliptic uniqueness theorem then gives `v∞=vStar`.

Finally, if the original orbit did not converge uniformly, choose `tₙ` with

```text
supNorm (u(tₙ)-uStar) ≥ eps.
```

The locally uniform subsequence converges at time `0` to the constant `uStar`, a contradiction.

This proves uniform convergence directly and avoids:

```text
construction of an omega-limit set,
invariance of that set,
a general LaSalle theorem,
abstract compact-semiflow infrastructure.
```

# 5. The shared Theorem 2.2 engine

## 5.1 Uniform modal spectral gap

The load-bearing spectral theorem is not merely `LinearlyStable`. It is a **uniform** gap.

```lean
noncomputable def intervalLinearizedSpectralGap
    (p : CM2Params) (uStar vStar : ℝ) : ℝ :=
  -sSup {r : ℝ | ∃ n : ℕ, n ≠ 0 ∧
    r = sigma p uStar vStar
      (unitIntervalNeumannSpectrum.eigenvalue n)}

/-- Strict modal stability gives a positive uniform gap. -/
theorem intervalLinearizedSpectralGap_pos_of_linearlyStable
    {p : CM2Params} {uStar vStar : ℝ}
    (hstable : LinearlyStable unitIntervalNeumannSpectrum p uStar vStar) :
    0 < intervalLinearizedSpectralGap p uStar vStar
```

The proof must use the explicit high-frequency behavior

```text
sigma_n → -∞,
```

not the invalid principle “an infinite family of negative numbers has a negative supremum.” Concretely:

```text
choose K so sigma_n ≤ -1 for n≥K;
take the maximum of the finitely many sigma_n for 1≤n<K;
strict negativity of that finite maximum gives the gap.
```

## 5.3 Nonlinear remainder and weighted Duhamel estimate


```lean
structure IntervalLinearizedNonlinearRemainderData
    (p : CM2Params) (uStar vStar sigma0 : ℝ) where
  remainder : FractionalPowerSpace 1 sigma0 →
    FractionalPowerSpace 1 0
  zero : remainder 0 = 0
  quadratic :
    ∃ C > 0, ∃ radius > 0,
      ∀ z, ‖z‖ ≤ radius →
        ‖remainder z‖ ≤ C * ‖z‖ ^ 2

    ∃ C > 0, ∃ radius > 0,

```
