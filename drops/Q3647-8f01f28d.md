## Why current per-time source decay does not yield a uniform record

`IntervalMildSourceDecay.lean` exposes:

```lean
def sourceCoeffQuadraticDecay_of_mildSolution_of_closedC2_neumann (p : CM2Params)
    {u₀ : intervalDomainPoint → ℝ}
    (D : GradientMildSolutionData p u₀)
    {t : ℝ} (ht : 0 < t) (htT : t ≤ D.T)
    (hC2 : ContDiffOn ℝ 2 (intervalDomainLift (D.u t)) (Set.Icc (0 : ℝ) 1))
    (hN0 : Filter.Tendsto (deriv (intervalDomainLift (D.u t)))
      (nhdsWithin (0 : ℝ) (Set.Ioi 0)) (nhds 0))
    (hN1 : Filter.Tendsto (deriv (intervalDomainLift (D.u t)))
      (nhdsWithin (1 : ℝ) (Set.Iio 1)) (nhds 0)) :
    SourceCoeffQuadraticDecay p (D.u t)
```

The constructed source-decay constant is slice-dependent. The proof path uses a per-slice closed-`C²`/Neumann certificate for `D.u t` and wraps the resulting coefficient decay into a `SourceCoeffQuadraticDecay`. Even though the code shape writes a constant of the form `C₀ / t + B_R`, the API exposes no uniform bound for the chosen constants as `t ↓ 0`. In particular, the route through restart-cosine representations gives:

```lean
def sourceCoeffQuadraticDecay_of_mildSolution (p : CM2Params)
    {u₀ : intervalDomainPoint → ℝ}
    (D : GradientMildSolutionData p u₀)
    (H : HasRestartCosineRepresentations D.T D.u)
    {t : ℝ} (ht : 0 < t) (htT : t < D.T) :
    SourceCoeffQuadraticDecay p (D.u t)
```

This is still per-time. It returns an existential constant hidden in the `SourceCoeffQuadraticDecay` record for each `t`; it does not return a single `Csrc` valid over `0 < t ≤ D.T`.

There is also an endpoint mismatch: the restart theorem is stated with `t < D.T`, while `ChemFluxCthetaSourceOn` and Task176's `UniformSourceCoeffQuadraticDecayOn` are stated on `0 < s → s ≤ T`. That can be repaired by working below the horizon or by using a direct closed-`C²` route at the endpoint, but it does not solve the `s ↓ 0` uniformity problem.

## Why `powerSource_window_uniform_decay` is not enough for the full window

`IntervalResolverPowerDecay.lean` has the strong producer:

```lean
theorem powerSource_window_uniform_decay
    {ν γ M m : ℝ} (hν : 0 ≤ ν) (hγ : 0 < γ) (hm : 0 < m)
    {w : ℝ → intervalDomainPoint → ℝ} {c' d' : ℝ} (hcd' : c' ≤ d')
    (bc : ℝ → ℕ → ℝ)
    (hbsum : ∀ σ ∈ Set.Icc c' d',
      Summable (fun n => unitIntervalCosineEigenvalue n * |bc σ n|))
    (hagree : ∀ σ ∈ Set.Icc c' d', Set.EqOn (intervalDomainLift (w σ))
        (fun x => ∑' n, bc σ n * cosineMode n x) (Set.Icc (0 : ℝ) 1))
    (hlb : ∀ σ ∈ Set.Icc c' d', ∀ x ∈ Set.Icc (0 : ℝ) 1,
      m ≤ intervalDomainLift (w σ) x)
    (hub : ∀ σ ∈ Set.Icc c' d', ∀ x ∈ Set.Icc (0 : ℝ) 1,
      intervalDomainLift (w σ) x ≤ M)
    {G1 G2 : ℝ}
    (hG1 : ∀ σ ∈ Set.Icc c' d', ∀ x ∈ Set.Icc (0 : ℝ) 1,
      |deriv (intervalDomainLift (w σ)) x| ≤ G1)
    (hG2 : ∀ σ ∈ Set.Icc c' d', ∀ x ∈ Set.Icc (0 : ℝ) 1,
      |deriv (deriv (intervalDomainLift (w σ))) x| ≤ G2) :
    ∃ C : ℝ, 0 ≤ C ∧
      (∀ σ ∈ Set.Icc c' d', ∀ k : ℕ, 1 ≤ k →
        |cosineCoeffs (fun x => ν * intervalDomainLift (w σ) x ^ γ) k|
          ≤ C / ((k : ℝ) * Real.pi) ^ 2) ∧
      (∀ σ ∈ Set.Icc c' d',
        |cosineCoeffs (fun x => ν * intervalDomainLift (w σ) x ^ γ) 0| ≤ C)
```

This is a genuine useful theorem. But it requires:

- a compact time window `Set.Icc c' d'`;
- a positive lower bound `m > 0` on that whole window;
- uniform first and second spatial derivative bounds on that window;
- per-slice cosine representations on that window.

This is exactly the right shape for an away-from-zero window, e.g. `[ε, D.T]` with `0 < ε`. It is not a theorem on `(0, D.T]`. The set `(0,D.T]` is not compact, and the available parabolic smoothing constants and positive lower bounds can degenerate as `s ↓ 0`. Initial Holder plus heat-kernel contractive coupling gives a spatial Holder modulus for `u`; it does not give the initial source `ν·u₀^γ` a quadratic cosine-coefficient decay or a uniform second derivative bound.

## Answer to question 1

No. `UniformSourceCoeffQuadraticDecayOn p D.u D.T Csrc` is not derivable from the current interfaces without adding an initial/source regularity assumption or using a different weak route.

A single coefficient-decay constant on all `0 < t ≤ T` would require information at the initial face, for example a quadratic decay bound for the initial power-source coefficients, or a weak route that avoids second-derivative/source-decay control entirely.

Then isolate the real parabolic coefficient estimate instead of pretending it is already produced:

```lean
/-- Duhamel-form coefficient estimate for the power source.  This is the real
analytic producer needed to propagate initial coefficient decay uniformly in time. -/
structure PowerSourceCoeffDuhamelBoundOn
    (p : CM2Params) (u : ℝ → intervalDomainPoint → ℝ)
    (u₀ : intervalDomainPoint → ℝ) (T BR : ℝ) : Prop where
  BR_nonneg : 0 ≤ BR
  bound : ∀ t, 0 < t → t ≤ T → ∀ k : ℕ, 1 ≤ k →
    |(intervalNeumannResolverSourceCoeff p (u t) k).re| ≤
      |(intervalNeumannResolverSourceCoeff p u₀ k).re| *
        Real.exp (-((k : ℝ) * Real.pi) ^ 2 * t) +
      BR / ((k : ℝ) * Real.pi) ^ 2
```

Given these two records, the full uniform record is easy and honest:

```lean
/-- Initial coefficient decay plus a Duhamel coefficient estimate gives a single
uniform source-coefficient decay constant on `0 < t ≤ T`. -/
theorem UniformSourceCoeffQuadraticDecayOn_of_initialDecay_and_duhamelBound
    {p : CM2Params} {u : ℝ → intervalDomainPoint → ℝ}
    {u₀ : intervalDomainPoint → ℝ} {T C₀ BR : ℝ}
    (Hinit : InitialPowerSourceCoeffQuadraticDecay p u₀ C₀)
    (HD : PowerSourceCoeffDuhamelBoundOn p u u₀ T BR) :
    UniformSourceCoeffQuadraticDecayOn p u T (C₀ + BR)
```

Proof sketch:

For `t > 0`, `k ≥ 1`, set `λ = ((k : ℝ) * Real.pi)^2`. From `HD.bound`,

```lean
|a_k(t)| ≤ |a_k(0)| * exp (-λt) + BR/λ.
```

Use `Real.exp_nonneg` and `Real.exp_le_one_iff` with `-λt ≤ 0` to get

```lean
|a_k(0)| * exp (-λt) ≤ |a_k(0)| ≤ C₀/λ.
```

Then add `BR/λ` and rewrite:

```lean
C₀ / λ + BR / λ = (C₀ + BR) / λ.
```

This theorem does not smuggle the hard part: `PowerSourceCoeffDuhamelBoundOn` is the remaining real analytic source-frontier. It cleanly records what must be proved from the derived parabolic equation for `u^γ` if arbitrary `θ` is required.
