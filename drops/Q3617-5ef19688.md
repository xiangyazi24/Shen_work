## 1. Pointwise lower bound for reflected images

The inequality

```lean
η ≤ |x + y + 2 * (k : ℝ)|
```

is true for all `k : ℤ`, provided:

```lean
0 < η
x ∈ Set.Icc η (1 - η)
y ∈ Set.Icc (0 : ℝ) 1
```

No separate `η < 1/2` hypothesis is needed for the inequality itself; if `Set.Icc η (1 - η)` is empty, the theorem is vacuous.  The `η < 1/2` hypothesis is only needed in the eventual interior-strip API.

Clean case split:

* If `0 ≤ k`, then `(k : ℝ) ≥ 0`, so

  ```lean
  x + y + 2 * (k : ℝ) ≥ x ≥ η > 0,
  ```

  hence the absolute value is the expression itself.

* If `¬ 0 ≤ k`, then `k ≤ -1`, so `(k : ℝ) ≤ -1`.  Using `x ≤ 1 - η` and `y ≤ 1`,

  ```lean
  x + y + 2 * (k : ℝ) ≤ (1 - η) + 1 - 2 = -η,
  ```

  hence the absolute value is the negative of the expression and is at least `η`.

Recommended lemma:

```lean
-- File: ShenWork/PDE/IntervalConjugateKernelMassDefect.lean
namespace ShenWork.IntervalNeumannFullKernel

/-- On an interior x-strip, every reflected image is at least `η` away from the
whole-line origin. -/
theorem reflected_shift_abs_ge_eta
    {η x y : ℝ} (hη : 0 < η)
    (hx : x ∈ Set.Icc η (1 - η))
    (hy : y ∈ Set.Icc (0 : ℝ) 1) (k : ℤ) :
    η ≤ |x + y + 2 * (k : ℝ)| := by
  by_cases hk : (0 : ℤ) ≤ k
  · have hkr : (0 : ℝ) ≤ (k : ℝ) := by exact_mod_cast hk
    have harg_ge : η ≤ x + y + 2 * (k : ℝ) := by nlinarith [hx.1, hy.1, hkr]
    have harg_nonneg : 0 ≤ x + y + 2 * (k : ℝ) :=
      le_trans hη.le harg_ge
    rw [abs_of_nonneg harg_nonneg]
    exact harg_ge
  · have hk_le : k ≤ (-1 : ℤ) := by omega
    have hkr : (k : ℝ) ≤ -1 := by exact_mod_cast hk_le
    have harg_le : x + y + 2 * (k : ℝ) ≤ -η := by nlinarith [hx.2, hy.2, hkr]
    have harg_nonpos : x + y + 2 * (k : ℝ) ≤ 0 := by linarith [hη]
    rw [abs_of_nonpos harg_nonpos]
    linarith

end ShenWork.IntervalNeumannFullKernel
```

Likely Lean pitfall: the integer negative branch should use `omega` to prove `k ≤ -1`; `linarith` alone will not derive this from `¬ 0 ≤ k` on `ℤ`.

## 3. Interchange and tiling implementation notes

### Public first-moment integrability

The current `IntervalSemigroupUniform.lean` has a private helper for integrability of `fun z => |z| * heatKernel t z`.  Since `ShenWork.tsum_cell_integral_eq_integral` requires `Integrable g`, make this public in the mass-defect file.  The proof can be copied narrowly from the private helper:

```lean
theorem heatKernel_abs_moment_integrable {t : ℝ} (ht : 0 < t) :
    Integrable (fun z : ℝ => |z| * heatKernel t z) := by
  have hb : (0 : ℝ) < 1 / (4 * t) := by positivity
  have hfun : (fun z : ℝ => |z| * heatKernel t z) =
      fun z => 1 / Real.sqrt (4 * Real.pi * t) *
        (|z| * Real.exp (-(1 / (4 * t)) * z ^ 2)) := by
    ext z
    unfold heatKernel
    rw [show -z ^ 2 / (4 * t) = -(1 / (4 * t)) * z ^ 2 from by ring]
    ring
  have habs : (fun z : ℝ =>
      1 / Real.sqrt (4 * Real.pi * t) *
        (|z| * Real.exp (-(1 / (4 * t)) * z ^ 2))) =
      fun z => 1 / Real.sqrt (4 * Real.pi * t) *
        ‖z * Real.exp (-(1 / (4 * t)) * z ^ 2)‖ := by
    ext z
    congr 1
    rw [Real.norm_eq_abs, abs_mul, abs_of_nonneg (Real.exp_nonneg _)]
  rw [hfun, habs]
  exact (integrable_mul_exp_neg_mul_sq hb).norm.const_mul _
```

```lean
    (hη : 0 < η) (hx : x ∈ Set.Icc η (1 - η)) :
    (∫ y in (0 : ℝ)..1, intervalNeumannReflectedKernelPart t x y)
      ≤ (4 * t / Real.sqrt (4 * Real.pi * t)) / η := by
  calc
    (∫ y in (0 : ℝ)..1, intervalNeumannReflectedKernelPart t x y)
        ≤ (1 / η) * (∫ z : ℝ, |z| * heatKernel t z) :=
      reflectedKernelPart_integral_le_first_abs_moment_div ht hη hx
    _ = (4 * t / Real.sqrt (4 * Real.pi * t)) / η := by
      rw [ShenWork.IntervalSemigroupUniform.heatKernel_first_abs_moment ht]
      ring

/-- Same square-root estimate used elsewhere in the approximate-identity proof. -/
theorem reflectedKernelPart_integral_le_sqrt_div
    {t : ℝ} (ht : 0 < t) {η x : ℝ}
    (hη : 0 < η) (hx : x ∈ Set.Icc η (1 - η)) :
    (∫ y in (0 : ℝ)..1, intervalNeumannReflectedKernelPart t x y)
      ≤ 2 * Real.sqrt t / η := by
  have h4pit_pos : 0 < 4 * Real.pi * t := by positivity
  have hpi_ge : 4 * t ≤ 4 * Real.pi * t := by nlinarith [Real.pi_gt_three]
  have hsqrt4t : Real.sqrt (4 * t) = 2 * Real.sqrt t := by
    have h4t_eq : (4 : ℝ) * t = (2 * Real.sqrt t) * (2 * Real.sqrt t) := by
      have := Real.mul_self_sqrt ht.le
      nlinarith
    rw [show (4 : ℝ) * t = (2 * Real.sqrt t) * (2 * Real.sqrt t) from h4t_eq,
      Real.sqrt_mul_self (by positivity : (0 : ℝ) ≤ 2 * Real.sqrt t)]
  have hmoment_le : 4 * t / Real.sqrt (4 * Real.pi * t) ≤ 2 * Real.sqrt t := by
    rw [div_le_iff₀ (Real.sqrt_pos_of_pos h4pit_pos)]
    calc
      4 * t = 2 * Real.sqrt t * Real.sqrt (4 * t) := by
        rw [hsqrt4t]
        nlinarith [Real.mul_self_sqrt ht.le]
      _ ≤ 2 * Real.sqrt t * Real.sqrt (4 * Real.pi * t) :=
        mul_le_mul_of_nonneg_left (Real.sqrt_le_sqrt hpi_ge) (by positivity)
  calc
    (∫ y in (0 : ℝ)..1, intervalNeumannReflectedKernelPart t x y)
        ≤ (4 * t / Real.sqrt (4 * Real.pi * t)) / η :=
      reflectedKernelPart_integral_le_first_abs_moment_formula ht hη hx
    _ ≤ (2 * Real.sqrt t) / η := by
      exact div_le_div_of_nonneg_right hmoment_le hη.le
    _ = 2 * Real.sqrt t / η := by ring
```

## Lean pain points checklist

### Summability

* For one-sided reflected sums, do **not** try to invoke `summable_cell_heat_interval_integral` directly; derive one-sided summability by nonnegative domination by the paired summand.
* For weighted moment sums, you need the public first-moment paired summability lemma; otherwise `integral_tsum_of_summable_integral_norm` will not have the needed summability hypothesis.

### Integral/tsum interchange

* Use `volume.restrict (Set.Ioc 0 1)` and convert with:

  ```lean
  intervalIntegral.integral_of_le (by norm_num : (0 : ℝ) ≤ 1)
  ```

  This matches `IntervalFullKernelMass.lean` and avoids measure mismatch.

* Norm integrals simplify by `Real.norm_of_nonneg (heatKernel_nonneg ht _)`.

### Multiplying inequalities by `1/η`

* Keep the factor as `(1 / η) * ...` while proving inequalities.
* Use:

  ```lean
  rw [show (1 / η) * |arg| = |arg| / η by ring]
  rw [le_div_iff₀ hη]
  ```

  to turn `1 ≤ |arg| / η` into `η ≤ |arg|`.

### Square-root horizon

* The standard estimate

  ```lean
  4 * t / Real.sqrt (4 * Real.pi * t) ≤ 2 * Real.sqrt t
  ```

  is already replicated in `IntervalSemigroupC1ApproxIdentity.lean`.  Reuse that proof pattern.

* For a uniform reflected mass theorem, choose

  ```lean
  δ := (ε * η / 4) ^ 2
  ```

  or smaller.  Then `2 * sqrt(t) / η < ε` is easy.

* For the interior derivative approximation mass-defect term with coefficient `M` and factor `2`, choose a more conservative

  ```lean
  δm := (ε * η / (16 * M)) ^ 2
  ```

  to avoid tight arithmetic.

### Nonnegativity of reflected mass

You will need:

```lean
have hR_nonneg : 0 ≤ ∫ y in (0 : ℝ)..1,
    intervalNeumannReflectedKernelPart t x y :=
  intervalIntegral.integral_nonneg_of_forall (by norm_num)
    (fun y => reflectedKernelPart_nonneg ht x y)
```

This lets you rewrite `|-2 * Rmass|` as `2 * Rmass` via `abs_mul`, `abs_neg`, `abs_of_nonneg`.
