## 1. From the ellÂą ladder to source `TimeC1On`

The ladder must output two different things on every positive window `[lo, hi]`:

1. a summable envelope for the source coefficients `a s k`; and
2. enough time regularity to define and bound `adot s k`.

These are not the same proof.

### Logistic part

For the logistic source

```text
L(u) = a*u - b*u^alpha,
```

positive-time strict positivity gives a lower floor for `u`, so real powers are smooth. Once the mode ODE has produced `u_t`, the derivative is

```text
d_t L(u) = (a - b*alpha*u^(alpha-1)) * u_t.
```

Thus the logistic `DuhamelSourceTimeC1On` proof needs:

* source coefficient envelope from the ladder;
* per-mode derivative by differentiating the cosine integral;
* continuity and a uniform bound for the derivative coefficients from window bounds on `u` and `u_t`.

### Chemotaxis-divergence part

Let

```text
v(t) = R_p(u(t)^gamma),
F(t,x) = u(t,x)^m * (1 + v(t,x))^(-beta) * v_x(t,x),
H(t,x) = d_x F(t,x).
```

`H` is the chemotaxis-divergence source. The clean 1D route is through the flux `F`, not through a premature divergence expansion during the fixed-point stage.

The zeroth mode is special: Neumann gives `v_x(t,0)=v_x(t,1)=0`, hence `F(t,0)=F(t,1)=0`, so the mean of `H=d_x F` is zero.

For `k > 0`, integrate by parts:

```text
cosCoeff(H(t), k) = k*pi * sineCoeff(F(t), k)
```

up to the repoâ€™s normalization. For time regularity, first obtain

```text
v_t = R_p(nu * gamma * u^(gamma-1) * u_t),
(v_x)_t = d_x R_p(nu * gamma * u^(gamma-1) * u_t).
```

Then

```text
F_t = m*u^(m-1)*u_t*(1+v)^(-beta)*v_x
      - beta*u^m*(1+v)^(-beta-1)*v_t*v_x
      + u^m*(1+v)^(-beta)*(v_x)_t.
```

For the `DuhamelSourceTimeC1On.hderivBound` field, the best definition is

```text
adot_chem(t,k) = cosCoeff(d_x F_t(t), k),
```

because then

```text
|adot_chem(t,k)| <= ||d_x F_t(t)||_L1
```

uniformly in `k`. The integration-by-parts formula with `k*pi*sineCoeff(F_t)` identifies the derivative, but by itself it carries a factor `k`; it does not give the uniform scalar derivative bound unless extra spatial decay is supplied.

Therefore the chemotaxis path is:

1. source coefficient ellÂą envelope for `H=d_x F` from the ladder;
2. source coefficient continuity from positive-window spatial continuity;
3. mode ODE gives `u_t`;
4. elliptic resolver gives `v_t` and `(v_x)_t`;
5. flux formula gives `F_t` and `d_x F_t` in `L1` or bounded form;
6. define `adot_chem` by the coefficient of `d_x F_t`;
7. prove derivative, continuity of `adot_chem`, and uniform derivative bound.

## 2. Is the mode ODE circular?

Not if staged correctly.

The per-mode ODE is

```text
u_hat_k'(t) = -lambda_k * u_hat_k(t) + g_hat_k(t),
```

where `g_hat_k` is the B-form source coefficient. To use this, we need `g_hat_k` continuous first. That first continuity should come from the positive-window spatial ladder and continuity of the mild trajectory, not from `u_t`.

The non-circular order is:

```text
positive-window spatial ladder
  -> source coefficients g_hat_k continuous in time
  -> per-mode ODE gives u_hat_k C1
  -> reconstruct u_t from summable coefficient bounds
  -> chain rule gives source coefficients g_hat_k C1
  -> DuhamelSourceTimeC1On
```

The circular order to avoid is:

```text
need DuhamelSourceTimeC1On to get u_t,
need u_t to get DuhamelSourceTimeC1On.
```

Only source coefficient continuity is needed before the ODE. Source coefficient `C1` is proved after the ODE has produced `u_t`.

## Main cautions

1. `u in C^{1,theta}` alone is not the same as `DuhamelSourceTimeC1On`. The latter needs a summable source coefficient envelope and a uniform derivative coefficient bound.

2. For chemDiv, separate the source envelope for `d_x F` from the time derivative bound for `d_t d_x F`. The former comes from the ellÂą ladder; the latter comes after `u_t` is available.

3. Use the zeroth-mode boundary cancellation. Since `v_x=0` at the Neumann endpoints, the flux vanishes at the endpoints, and the chemDiv mean is zero.

4. Prove source coefficient continuity before invoking the mode ODE. Prove source coefficient `C1` only after the mode ODE has produced `u_t`.

5. Keep everything positive-window-local. Avoid global `lo=0` time regularity unless a theorem really proves it. The existing `DuhamelSourceTimeC1On.shift_zero` and `restrict_hi` tools can convert positive-window data into the shapes needed by restart consumers; see `IntervalDuhamelSourceTimeC1On.lean:46-103`.

## Recommendation

Package the ladder output as a window record that explicitly contains both ellÂą envelopes and coefficient continuity for the logistic and chemDiv sources. Then prove the ODE/time-bootstrap lemmas before attempting `DuhamelSourceTimeC1On`. This avoids circularity and gives a single per-`t0` source witness that can feed `hTimeNhd`, `hPdeAgreement`, and `hResolverData` in parallel.
