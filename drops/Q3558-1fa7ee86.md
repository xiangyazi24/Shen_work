## Sqrt inequality skeletons

### Taxis part

From `|taxisPart| ≤ V₁ * |u_x|` on `[0,1]` and `0 ≤ V₁`, prove the square inequality, integrate it, then take square roots.  The target uses `H1grad_sq_integral_eq_H1gradL2Norm_sq` to identify the gradient integral with the canonical norm.

private theorem taxis_sqrt_bound_of_factor
    {p : CM2Params} {u v : ℝ → intervalDomainPoint → ℝ}
    {T V₁ τ : ℝ}
    (hV1 : 0 ≤ V₁)
    (hgrad_sq : IntervalIntegrable
      (fun x => (deriv (intervalDomainLift (u τ)) x) ^ 2) volume (0 : ℝ) 1)
    (htaxis_sq : IntervalIntegrable
      (fun x => (H1PhysicalChemTaxisPart p u v τ x) ^ 2) volume (0 : ℝ) 1)
    (hfactor : ∀ x, x ∈ Set.Icc (0 : ℝ) 1 →
      |H1PhysicalChemTaxisPart p u v τ x| ≤
        V₁ * |deriv (intervalDomainLift (u τ)) x|) :
    Real.sqrt
        (∫ x in (0 : ℝ)..1, (H1PhysicalChemTaxisPart p u v τ x) ^ 2)
      ≤ V₁ * H1gradL2Norm u τ := by
  let part : ℝ → ℝ := fun x => H1PhysicalChemTaxisPart p u v τ x
  let ux : ℝ → ℝ := fun x => deriv (intervalDomainLift (u τ)) x
  have hright_int : IntervalIntegrable
      (fun x => V₁ ^ 2 * (ux x) ^ 2) volume (0 : ℝ) 1 := by
    simpa [ux] using hgrad_sq.const_mul (V₁ ^ 2)
  have hsq_point : ∀ x, x ∈ Set.Icc (0 : ℝ) 1 →
      (part x) ^ 2 ≤ V₁ ^ 2 * (ux x) ^ 2 := by
    intro x hx
    have h := hfactor x hx
    have h_rhs_nonneg : 0 ≤ V₁ * |ux x| :=
      mul_nonneg hV1 (abs_nonneg (ux x))
    have h_abs_sq : |part x| ^ 2 ≤ (V₁ * |ux x|) ^ 2 := by
      nlinarith [h, abs_nonneg (part x), h_rhs_nonneg]
    calc
      (part x) ^ 2 = |part x| ^ 2 := by rw [sq_abs]
      _ ≤ (V₁ * |ux x|) ^ 2 := h_abs_sq
      _ = V₁ ^ 2 * (ux x) ^ 2 := by
        rw [mul_pow, sq_abs]
  have hInt_le :
      (∫ x in (0 : ℝ)..1, (part x) ^ 2) ≤
        V₁ ^ 2 * ∫ x in (0 : ℝ)..1, (ux x) ^ 2 := by
    calc
      (∫ x in (0 : ℝ)..1, (part x) ^ 2)
          ≤ ∫ x in (0 : ℝ)..1, V₁ ^ 2 * (ux x) ^ 2 := by
            exact intervalIntegral.integral_mono_on
              (μ := volume) (a := (0 : ℝ)) (b := 1)
              (f := fun x => (part x) ^ 2)
              (g := fun x => V₁ ^ 2 * (ux x) ^ 2)
              (by norm_num) (by simpa [part] using htaxis_sq) hright_int hsq_point
      _ = V₁ ^ 2 * ∫ x in (0 : ℝ)..1, (ux x) ^ 2 := by
            rw [intervalIntegral.integral_const_mul]
  have hgrad_eq :
      (∫ x in (0 : ℝ)..1, (ux x) ^ 2) = (H1gradL2Norm u τ) ^ 2 := by
    simpa [ux] using H1grad_sq_integral_eq_H1gradL2Norm_sq u τ
  have hsqrt_rhs :
      Real.sqrt (V₁ ^ 2 * ∫ x in (0 : ℝ)..1, (ux x) ^ 2) =
        V₁ * H1gradL2Norm u τ := by
    have hnonneg : 0 ≤ V₁ * H1gradL2Norm u τ :=
      mul_nonneg hV1 (H1gradL2Norm_nonneg u τ)
    rw [hgrad_eq]
    have hsq : V₁ ^ 2 * (H1gradL2Norm u τ) ^ 2 =
        (V₁ * H1gradL2Norm u τ) ^ 2 := by ring
    rw [hsq, Real.sqrt_sq_eq_abs, abs_of_nonneg hnonneg]
  calc
    Real.sqrt (∫ x in (0 : ℝ)..1, (part x) ^ 2)
        ≤ Real.sqrt (V₁ ^ 2 * ∫ x in (0 : ℝ)..1, (ux x) ^ 2) :=
          Real.sqrt_le_sqrt hInt_le
    _ = V₁ * H1gradL2Norm u τ := hsqrt_rhs

## Why this is non-circular

This proof uses only:

* classical strict-positive-time regularity and representative continuity from `hsol`,
* the new pointwise factor bounds from `hchem`,
* elementary interval integral monotonicity and Cauchy-Schwarz/product integrability helpers,
* the already-landed classical reaction producer for the separate reaction term.

It does **not** use `IsPaper2BoundedBefore`, any bounded-before resolver package, or a uniform-in-time H¹ bound.  The constants `V₁`, `V₂`, and `M` still enter only through the caller-supplied `H1PhysicalChemFactorBoundsBefore` or the already public resolver-sup reducer.  That is the right frontier: producing those constants uniformly remains a separate source residual, but converting them to fixed-time L² sqrt data is formal/regularity work.
