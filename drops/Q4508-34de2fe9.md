# 1. Paper 3 P3.4: nonlinear local exponential C¹ stability

## Target

A suitable interval-specific capstone is the previously prepared target

```lean
theorem intervalDomain_localExponentialC1_of_sigmaDiagonal
    (hgap : UniformNegativeSigmaGap p uStar vStar activeModes δ)
    (hlin : LinearizedCosineCoefficientEvolution p uStar vStar)
    (hrem : LocalQuadraticRemainderBound p uStar vStar σ ρ)
    ... :
    ∃ ε C rate : ℝ,
      0 < ε ∧ 0 < C ∧ 0 < rate ∧
      ∀ u₀, perturbationNorm σ u₀ < ε →
        ∀ t, 0 ≤ t →
          intervalDomainC1Norm (u t - fun _ => uStar) ≤
            C * Real.exp (-rate * t) * perturbationNorm σ u₀
```

The exact repository-facing theorem can use the coefficient Hilbert space rather than introducing a general abstract sectorial operator. For direct `C¹` control from cosine coefficients, the audited threshold is naturally `σ > 3/4`.

## Why this is the number-one implementation long pole

The linear part is cheap now:

```plain text
cosine mode k
→ elliptic algebraic multiplier for V_k
→ U_k' = sigma_k U_k
→ exp(sigma_k t).
```

The nonlinear theorem still needs all of the following to coexist in one Lean layer:

1. **Weighted modal phase spaces.** The proof needs norms strong enough to control products and embed into `C¹`, but weak enough that the semigroup gains regularity with an integrable singularity.
2. **Smoothing with a singular time kernel.** One needs estimates of the form
   
   \[
   \|S(t)a\|_{X^\sigma}
      \le C t^{-(\sigma-\rho)}e^{-\delta t}\|a\|_{X^\rho}.
   \]
3. **Quadratic nonlinear remainder bounds.** After subtracting the linearization, the chemotaxis term contains products of the perturbation, resolver outputs, and derivatives. These must be proved locally quadratic in compatible coefficient norms.
4. **Resolver coupling.** The elliptic variable is not independent; every nonlinear estimate must pass through the Neumann resolver and its difference estimates.
5. **The zero mode.** The positive logistic equilibrium has a strictly stable zero mode, while the minimal model has a neutral mass mode and must be restricted to the fixed-mass subspace.
6. **A singular Volterra bootstrap.** The final Duhamel inequality is not ordinary scalar Gronwall; it has a kernel such as `(t-s)^(-θ) exp(-δ(t-s))`.

A monolithic Codex dispatch is likely to thrash on norm coercions and choose incompatible spaces.

## Recommended attack

Build it as five independently compiling layers:

```plain text
A. weighted coefficient norm and C¹ embedding;
B. diagonal semigroup decay and smoothing;
C. linearized physical-PDE → coefficient-ODE correctness bridge;
D. local quadratic remainder in X^ρ;
E. singular-Volterra exponential bootstrap.
```

Reuse:

```lean
cosineMode
cosineEigenvalue
unitIntervalCosineHilbertBasis
diagonalSemigroupCoeff
diagonalSemigroupCoeff_l2_norm_le_of_growth_le_on_nonzero
sigma
intervalNeumannResolverCoeff_elliptic
```

Do **not** begin by formalizing general Henry/Kato sectorial theory. The modal route is shorter and more auditable. Give Codex one layer at a time, with the norm and exponent choices fixed in the specification.

# 2. Paper 3 P3.5–P3.6: time-translate compactness and ω-limit entry

## Target

The hard compactness producer should look like:

```lean
theorem intervalDomain_timeTranslateCompactness
    (hsol : PositiveGlobalBoundedSolution intervalDomain p u v)
    (htailReg : UniformPositiveTimeTailRegularity p u v) :

        IsEntireClassicalSolution intervalDomain p U V
```

The downstream dynamical capstone is then conceptually:

```lean
theorem intervalDomain_omegaLimit_enters_localStableBasin
    (hcompact : IntervalDomainTimeTranslateCompactnessData p u v)
    (hlyap : IntervalDomainLyapunovDissipationData p u v uStar)
    (hrigid : ZeroDissipationRigidity p uStar vStar)
    (hlocal : LocalExponentialC1Stability p uStar vStar) :
    EventuallyAtTop fun t =>
      intervalDomainC1Norm (u t - fun _ => uStar) < hlocal.radius
```

## Why this is a multi-day grind

Paper 2 positive-time regularity is only a substrate. Paper 3 needs estimates uniform on **translated tail slabs**, not constants depending on one fixed finite interval. The proof must supply:

1. uniform `C²_x`/`C¹_x` or Hölder bounds for all sufficiently late translated solutions;
2. enough time equicontinuity to apply space-time Arzelà–Ascoli;
3. subsequence extraction simultaneously for `u`, `v`, derivatives, and source terms;
4. continuity of the elliptic resolver under the chosen convergence;
5. passage to the nonlinear chemotaxis divergence and reaction in the limit;
6. preservation of positivity, bounds, Neumann data, and the entire-solution property;
7. Lyapunov constancy on an ω-limit trajectory;
8. zero-dissipation rigidity, including the measure-theoretic step from zero integral of a nonnegative continuous integrand to pointwise equality;
9. eventual entry into the local `C¹` stable basin.

The local entropy differentiation identity prepared in Q4474 is only one component. The hardest part is making the compactness topology strong enough for PDE passage while deriving it from estimates that are genuinely uniform as the translation time tends to infinity.

## Recommended attack

Separate the proof into three files/packages:

```plain text
1. UniformPositiveTimeTailRegularity
   global/eventual bounds → uniform translated-slab C²_x and time modulus;

2. IntervalTimeTranslateCompactness
   Arzelà–Ascoli extraction + resolver convergence + PDE passage;

3. IntervalOmegaLimitLyapunovRigidity
   Lyapunov constancy → zero dissipation → constant equilibrium → local basin.
```

The theorem contracts must be designed manually before dispatch. Codex is well suited to the compact-interval estimates and convergence wiring once the exact topologies and representatives are fixed; it is poorly suited to choosing those topologies autonomously.

This producer has the highest leverage inside Paper 3: it feeds persistence contradiction arguments, global stability, and the transition from Lyapunov decay to local exponential convergence.

# 3. Paper 1: `PaperRotheTailUniform`

## Exact current frontier

The repository property is essentially:

```lean
def PaperRotheTailUniform
    (p : CMParams) (c lam M κ Λ : ℝ)
    (hprodAll : ∀ v, PaperRotheStepProducer p c lam M κ Λ v)
    (hκ : 0 ≤ κ) (hM : 0 ≤ M) : Prop :=
  ∀ R > 0, ∀ ε > 0,
    ∃ K : ℕ, ∀ v : ℝ → ℝ,
      InMonotoneWaveTrapSet κ M v →
      ∀ k, K ≤ k → ∀ x ∈ Set.Icc (-R) R,
        |rotheSeqOfPaper p c lam M κ Λ v (hprodAll v) hκ hM k x -
          rotheLimit
            (rotheSeqOfPaper p c lam M κ Λ v (hprodAll v) hκ hM) x| < ε
```

The A+ weighted-Hölder source-box work and Route-A `greenCore` solve the **one-step existence/regularity** problem. They do not prove this uniform-in-`v`, uniform-in-tail-index convergence.

## Why this is especially risky

For each fixed `v` and `x`, monotone bounded convergence is easy. Even locally uniform convergence for one fixed orbit follows from equi-Lipschitzness plus pointwise convergence. The stated property is much stronger: one `K` must work for every frozen profile in the whole trap.

There is currently no automatic theorem turning

```plain text
pointwise monotone convergence
+ per-orbit equi-Lipschitz
```

into uniform convergence over an uncountable parameter class.

The obvious parameterized-Dini route has a possible circularity:

```plain text
finite-step dependence + continuity of the limit map
→ Dini on a compact parameter family
→ uniform tail.
```

But the current `paperRotheContinuousDependence` obtains continuity of the limit map from `PaperRotheSeqStepDependence + PaperRotheTailUniform`. Thus one cannot invoke continuity of the limit to prove `PaperRotheTailUniform` unless it is obtained independently.

There is also a satisfiability risk: `InMonotoneWaveTrapSet` may be too large for a uniform rate unless one has one of:

- a quantitative contraction/decrement independent of `v`;
- compactness of the parameter class in a topology for which the stationary limit has an independently proved closed graph plus uniqueness;
- a stronger orbit invariant that gives a uniform modulus.

This is the remaining theorem most likely to expose that an interface must be weakened or strengthened rather than merely ground through.

## Recommended attack

Before a long Codex run, perform a two-way theorem-shape test.

### Route A: quantitative decay

Try to prove a uniform one-step estimate such as

```plain text
D_{k+1}(v) ≤ q D_k(v),  q<1 independent of v,
```

or a uniform energy decrement that yields an explicit tail modulus. If available, this is the cleanest route.

### Route B: compactness plus independent uniqueness

1. Prove `PaperRotheSeqStepDependence` first.
2. Show the family of admissible frozen profiles is sequentially compact in the exact topology used by the Green operator, including the left/right tail controls.
3. Prove a closed-graph theorem for stationary Rothe limits.
4. Prove uniqueness of the stationary limit in the relevant lower-pinned class, independently of `PaperRotheTailUniform`.
5. Deduce continuity of the limit map and apply a parameterized Dini theorem on each compact window.

If neither quantitative decay nor stationary uniqueness is true, the current global `PaperRotheTailUniform` statement is probably too strong. In that case the right fix is an interface change to compact parameter subsets or a weaker closed-graph convergence theorem—not an axiom and not pressure on Codex to prove a false statement.

Dispatch this only after the mechanism is chosen. The first Codex job should be a **satisfiability audit/prototype lemma**, not the full property.

# Maximum-leverage cross-paper infrastructure

## Build once: 1D Neumann elliptic resolver Lᵖ smoothing and difference bounds

The single best cross-paper investment is a concrete interval resolver package of the form

```lean
structure IntervalNeumannResolverLpSmoothingData
    (μ : ℝ) where
  W2p_bound :
    ∀ p f, 1 < p →
      ‖intervalNeumannResolver μ f‖_{W2p} ≤ C μ p * ‖f‖_{Lp}
  deriv_Lq_bound :
    ∀ p q f, admissible1D p q →
      ‖∂x (intervalNeumannResolver μ f)‖_{Lq} ≤
        Cgrad μ p q * ‖f‖_{Lp}
  deriv_difference :
    ∀ p q f g, admissible1D p q →
      ‖∂x (intervalNeumannResolver μ f -
            intervalNeumannResolver μ g)‖_{Lq} ≤
        Cgrad μ p q * ‖f-g‖_{Lp}
```

At minimum, expose the concrete estimates actually consumed by the papers, for example

\[
\|\partial_xR_\mu[f]\|_{L^q}
 \le C_{\mu,p,q}\|f\|_{L^p},
\qquad
\|\partial_x(R_\mu[f]-R_\mu[g])\|_{L^q}
 \le C_{\mu,p,q}\|f-g\|_{L^p}.
\]

This one package unblocks:

- **Paper 2 Lᵖ:** the chemotaxis cross-term absorption and finite exponent descent;
- **Paper 3 P3.2:** persistence estimates involving the signal gradient;
- **Paper 3 P3.4:** local quadratic remainder bounds for the nonlinear Duhamel argument;
- **Paper 3 P3.5:** resolver convergence under time-translate compactness;
- **Paper 3 P3.6:** Lyapunov cross-term control and `C¹` upgrade.

Paper 1 uses a whole-line frozen elliptic convolution rather than the interval resolver, but the proof can still share a lower-level kernel/Young-convolution API if it is factored cleanly.

This is a better shared investment than abstract Krein–Rutman or general sectorial-operator theory.
