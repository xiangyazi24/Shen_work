# Recommended Lean route: a linear relaxing barrier to arbitrary M > MChi

The paper uses the nonlinear scalar ODE

```text
B' = B(1 − (1−χ) B^α).
```

Formalizing its exact closed form is possible, but it introduces avoidable negative real powers. A simpler Lean proof gives exactly the limsup statement needed by Section 5.

Fix any `M > MChi p`. Define

```text
κ(M) = (1−χ) M^α − 1  > 0,
B_{M,C}(t) = M + (C−M) exp(−κ(M)t),
```

with `C ≥ M` and `u₀(x) ≤ C`.

For the actual Paper-1 system,

```text
u_t = u_xx − χ (u^m v_x)_x + u(1−u^α),
0   = v_xx − v + u^γ,
```

expansion gives, in the critical case `m+γ−1=α`,

```text
u_t
 = u_xx − χm u^(m−1)u_x v_x
   − χu^m v
   + u − (1−χ)u^(α+1).
```

Since `χ>0` and `v≥0`, the term `−χu^m v` is favorable. Let

```text
F(s) = s − (1−χ)s^(α+1)
     = s(1 − (1−χ)s^α).
```

For every `s ≥ M`,

```text
F(s)
 ≤ s(1 − (1−χ)M^α)
 = −κ(M)s
 ≤ −κ(M)(s−M).
```

But

```text
B'_{M,C}(t) = −κ(M)(B_{M,C}(t)−M).
```

Hence `B' ≥ F(B)`, so `B_{M,C}` is a supersolution. It decreases to `M`. Comparison yields

```text
u(t,x) ≤ B_{M,C}(t) → M.
```

Because this works for every `M > MChi`, it implies

```text
UniformLimsupLe (wholeLineCauchyGlobalU p u₀) (MChi p).
```

This route is preferable to the exact nonlinear ODE because:

- all time dependence uses the already-developed exponential-barrier algebra;
- the restart identity is immediate;
- no `rpow` of a time-dependent inverse is needed;
- the target needed by energy dissipation is precisely eventual `u≤M` for an arbitrary selected `M>MChi`, not an exact finite-time `u≤MChi` statement.
