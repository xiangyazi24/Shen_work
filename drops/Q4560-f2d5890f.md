# Definitive verdict

There are two different issues in the proposed “one remaining lemma,” and they should be separated.

1. **The principal ideal is a square ideal.** This part is valid, and in fact stronger than stated: assuming only `y ≠ 0`, one can prove

   ```lean
   ∃ I : Ideal 𝓞_K,
     Ideal.span ({descentInteger x z} : Set 𝓞_K) = I ^ 2.
   ```

   The branch `y ^ 2 ∣ Δ` is not needed for this. Every prime-ideal multiplicity is even, including the primes over `2`, `3`, and `11`.

2. **The element is not a square in `K`.** This does **not** follow from primitivity, the norm identity, or the ideal-square statement. The equality

   ```text
   descentElement(x,z) ∈ K×²
   ```

   is the triviality of the usual cubic Kummer class; equivalently, it says that the corresponding rational point is divisible by `2` in the elliptic-curve group. To deduce that such a point is exceptional requires a separate global theorem—an explicit halving/infinite-descent lemma, a minimal-counterexample argument, or an already-proved theorem of the form

   ```lean
   IsSquare (descentElement x z) → Exceptional x z y.
   ```

   Lutz–Nagell cannot be applied merely from `IsSquare δ`: it applies after torsion has been established.

So the correct audit conclusion is:

```text
primitive solution + y ≠ 0
  ⟹ span{δ} is a square ideal                       -- local ideal arithmetic

¬Exceptional + (square δ ⟹ Exceptional)
  ⟹ δ is not a square                              -- separate global descent
```

The first implication is the clean finite ideal-factorization lemma requested below. The second implication is not contained in the listed hypotheses.

# 1. Arithmetic setup and the two discriminants

Let

```text
m(T) = T³ - 4T² + 4T - 2,
K = ℚ(α),
m(α) = 0,
O = 𝓞_K,
```

and put

```text
A = x - 24z²,
B = 18z²,
d = descentInteger(x,z) ∈ O,
δ = A + Bα ∈ K,
J = span{d}.
```

The proved norm identity is

```text
N_{K/ℚ}(δ) = x³ - 432xz⁴ + 8208z⁶ = y².
```

There are two unrelated discriminants in the discussion.

## 1.1 The cubic-field discriminant

A direct calculation gives

```text
disc(m) = -44 = -2²·11.
```

The local ideal argument needs the already-proved monogenic/integral-basis fact

```text
O = ℤ[α]
```

or an equivalent integral basis `1, α, α²`. This is load-bearing: it lets one identify a prime ideal from the residue of `α`. If the class-number proof has not already exposed this fact as an API lemma, add it as a prerequisite.

The ramified rational primes of `K` are exactly `2` and `11`.

## 1.2 The Mordell-model discriminant

For

```text
E_M : Y² = X³ - 432X + 8208,
```

the short-Weierstrass discriminant is

```text
Δ_M = -16(4(-432)³ + 27·8208²)
    = -2¹²·3¹²·11
    = -11·6¹².
```

The condition `y² ∣ Δ_M` is the later Lutz–Nagell-style exceptional condition. It is not what makes `J` a square ideal.

# 2. Prime decomposition at the finite exceptional set

The useful factorizations follow from the integral basis and the reductions of `m`.

## 2.1 The prime over `2`

The polynomial `m` is Eisenstein at `2`, and modulo `2`

```text
m(T) ≡ T³.
```

Thus there is one prime ideal

```text
P₂ = (2, α),
(2) = P₂³,
N(P₂) = 2.
```

Its ramification index is `3`, but its residue degree is `1`. For ideal norms it is the residue degree—not the ramification index—that multiplies the exponent.

## 2.2 The prime over `3`

Modulo `3`,

```text
m(T) ≡ T³ + 2T² + T + 1.
```

It has no root in `F₃`, hence is irreducible. Therefore `3` is inert:

```text
(3) = P₃,
N(P₃) = 3³.
```

This prime is not in the field discriminant, but it must be treated separately because the coefficient `B = 18z²` is divisible by `3`.

## 2.3 The primes over `11`

Modulo `11`,

```text
m(T) ≡ (T - 8)²(T - 10).
```

Hence

```text
P₈  = (11, α - 8),
P₁₀ = (11, α - 10),
(11) = P₈² P₁₀,
N(P₈) = N(P₁₀) = 11.
```

Only `P₈` is ramified, but both residue degrees are `1`.

Thus the field-different primes are over `{2,11}`, while the coefficient `18` makes `{2,3,11}` the finite list worth handling explicitly in the descent proof.

# 3. First coprimality lemma: no prime divisor of `d` lies over a divisor of `z`

Let `P` be a nonzero prime ideal of `O`, lying over a rational prime `p`, and suppose

```text
J ≤ P,
```

equivalently `d ∈ P`.

Then

```text
p ∤ z.
```

Indeed, if `p ∣ z`, then the image of `z` in `O/P` is zero. Therefore all terms containing `z²` vanish modulo `P`, and

```text
d = (x - 24z²) + 18z²α ≡ x mod P.
```

Since `d ∈ P`, one gets `x ∈ P`. Contracting to `ℤ` gives `p ∣ x`. This contradicts `gcd(x,z)=1`.

This is the basic primitive-pair input. It should be formalized before any case split on `p`.

A suitable project lemma shape is:

```lean
theorem prime_below_descentInteger_not_dvd_z
    {x z : ℤ}
    (hcop : IsCoprime x z)
    {P : Ideal O} (hPprime : P.IsPrime) (hP0 : P ≠ ⊥)
    {p : ℕ} (hp : Nat.Prime p)
    (hlies : P.comap (algebraMap ℤ O) = Ideal.span ({(p : ℤ)} : Set ℤ))
    (hdP : d x z ∈ P) :
    ¬ (p : ℤ) ∣ z.
```

Use whatever existing `LiesOver` predicate the pinned Mathlib version provides in place of the displayed contraction equality.

# 4. Away from `2` and `3`: a prime divisor of `d` is unique above its rational prime

Assume now

```text
p ∤ 18z.
```

By the preceding lemma, `p ∤ z`, so for `p ≠ 2,3` this condition is automatic.

In the residue field `O/P`, `B = 18z²` is invertible. From `d ∈ P`,

```text
A + Bα ≡ 0 mod P,
```

so

```text
α ≡ r := -(A/B)
  = (24z² - x)/(18z²) mod p.
```

The right side lies in the prime field `F_p`.

Because `O = ℤ[α]`, the residue field `O/P` is generated over `F_p` by the residue of `α`. But that residue is already the scalar `r`. Hence

```text
O/P = F_p,
```

so the residue degree is `1` and

```text
N(P) = p.
```

Moreover, `P` is forced to be

```text
P = (p, α - r).
```

Consequently, if another prime ideal `Q` over the same `p` also contains `d`, then the identical calculation gives the same scalar `r`, and

```text
Q = (p, α - r) = P.
```

Therefore:

> For every `p ≠ 2,3`, at most one prime ideal over `p` occurs in the factorization of `J`, and its residue degree is `1`.

This argument includes `p=11`; ramification does not invalidate it. For Lean, it may nevertheless be easier to use the explicit `P₈/P₁₀` calculation below at `11` and reserve this generic lemma for `p ∤ 2·3·11`.

Recommended lemma shape:

```lean
theorem unique_prime_above_dvd_descentInteger_of_not_dvd_18
    {P Q : Ideal O} {p : ℕ}
    (hp : Nat.Prime p)
    (hp2 : p ≠ 2) (hp3 : p ≠ 3)
    (hPprime : P.IsPrime) (hQprime : Q.IsPrime)
    (hPlies : LiesOverIntPrime P p)
    (hQlies : LiesOverIntPrime Q p)
    (hPd : d x z ∈ P) (hQd : d x z ∈ Q)
    (hcop : IsCoprime x z) :
    P = Q ∧ Ideal.absNorm P = p.
```

The proof should go through the quotient ring and the integral-basis/generation theorem, rather than through abstract conjugates.

# 5. Explicit bookkeeping at `11`

The explicit congruences are useful as a small finite certificate.

At `P₈`, substitute `α ≡ 8`:

```text
d ≡ x - 24z² + 18·8z²
  = x + 120z²
  ≡ x - z² mod 11.
```

At `P₁₀`, substitute `α ≡ 10`:

```text
d ≡ x - 24z² + 18·10z²
  = x + 156z²
  ≡ x + 2z² mod 11.
```

If both `P₈` and `P₁₀` divided `J`, then

```text
x - z² ≡ 0,
x + 2z² ≡ 0 mod 11.
```

Subtracting gives

```text
3z² ≡ 0 mod 11,
```

hence `11 ∣ z`, and then `11 ∣ x`, contradicting primitivity.

Thus at most one of `P₈`, `P₁₀` divides `J`. Since both have norm `11`, the norm parity immediately forces its exponent to be even.

Possible Lean lemmas:

```lean
theorem descentInteger_mem_P8_iff :
    d x z ∈ P8 ↔ (11 : ℤ) ∣ (x - z ^ 2)

theorem descentInteger_mem_P10_iff :
    d x z ∈ P10 ↔ (11 : ℤ) ∣ (x + 2 * z ^ 2)

theorem not_mem_both_P8_P10
    (hcop : IsCoprime x z) :
    ¬ (d x z ∈ P8 ∧ d x z ∈ P10)
```

These are preferable to hiding the ramified-prime check inside a large generic theorem.

A notable consequence is that the proposed Lutz–Nagell exception cannot be used to evade this check: if `11 ∣ y`, then `v₁₁(y²) ≥ 2`, whereas `v₁₁(Δ_M)=1`, so `y² ∤ Δ_M`. The `11`-adic ideal parity must be proved; fortunately the calculation above proves it directly.

# 6. The special primes `2` and `3`

## 6.1 At `2`

There is only one prime ideal `P₂` over `2`, and `N(P₂)=2`. If

```text
v_{P₂}(J)=e₂,
```

then

```text
v₂(N(J)) = e₂.
```

Since `N(J)=|N(δ)|=y²`, the left side is even. Therefore `e₂` is even.

The ramification index `3` does not enter this equation. The norm of `P₂^e` is `2^e`, not `2^{3e}`.

## 6.2 At `3`

There is only one prime ideal `P₃` over `3`, with `N(P₃)=3³`. If

```text
v_{P₃}(J)=e₃,
```

then

```text
v₃(N(J)) = 3e₃.
```

Again `N(J)=y²`, so `3e₃` is even. Since `3` is odd, `e₃` is even.

This explains exactly why inertness at `3` is enough even though the residue degree is not `1`: the residue degree is still odd.

# 7. Global parity of every prime-ideal exponent

Assume `y ≠ 0`. Then `δ ≠ 0` by the norm identity, hence

```text
J ≠ ⊥.
```

Let `P` be any nonzero prime ideal and let `p` be the rational prime below it. Write

```text
a_P = v_P(J).
```

If `a_P=0`, it is even. Suppose `a_P>0`.

The preceding local analysis gives:

```text
p = 2:       P=P₂,  f(P/2)=1, and P is the only divisor over 2;
p = 3:       P=P₃,  f(P/3)=3, and P is the only divisor over 3;
p ≠ 2,3:    P is the unique prime over p dividing J, and f(P/p)=1.
```

In particular, in every case:

```text
f(P/p) is odd,
```

and no other prime over `p` has positive multiplicity in `J`.

The ideal-norm valuation formula therefore reduces to

```text
v_p(N(J)) = f(P/p) · a_P.
```

On the other hand,

```text
N(J) = |N_{K/ℚ}(δ)| = |y²| = |y|²,
```

so

```text
v_p(N(J)) = 2v_p(|y|)
```

is even. An odd integer times `a_P` is even only when `a_P` is even. Hence

```text
∀ P, Even (v_P(J)).
```

This is the entire ideal-square argument.

## Mathlib-facing norm formulation

A convenient formal statement avoids introducing a separate valuation object and uses normalized factor counts:

```lean
theorem descentInteger_normalizedFactor_count_even
    (hcop : IsCoprime x z)
    (hy0 : y ≠ 0)
    (hnorm : Algebra.norm ℚ (descentElement x z) = y ^ 2) :
    ∀ P : Ideal O,
      Even ((normalizedFactors J).count P).
```

The one reusable number-field bridge to isolate is:

```lean
theorem natFactorization_absNorm_eq_residueDegree_mul_count
    {J P : Ideal O} (hJ0 : J ≠ ⊥)
    {p f : ℕ}
    (hp : Nat.Prime p)
    (hPprime : P.IsPrime)
    (hPnorm : Ideal.absNorm P = p ^ f)
    (hunique : ∀ Q,
      Q.IsPrime → Ideal.absNorm Q % p = 0 →
      Q ∈ normalizedFactors J → Q = P) :
    (Nat.factorization (Ideal.absNorm J)) p =
      f * (normalizedFactors J).count P.
```

The exact spelling of the ideal-norm/principal-ideal bridge varies slightly by Mathlib revision. Put that API friction in a small lemma such as:

```lean
theorem absNorm_span_descentInteger :
    Ideal.absNorm J = y.natAbs ^ 2.
```

Its proof uses the proved field norm identity and the existing theorem that the absolute norm of a principal ideal is the natural absolute value of the algebraic norm.

Then `Nat.factorization_pow` gives the evenness of the left side.

# 8. “All multiplicities even” implies “the ideal is a square”

This part does not require that `O` be a PID. It is valid in every Dedekind domain because the monoid of nonzero integral ideals has unique factorization.

Mathlib supplies the `UniqueFactorizationMonoid` structure for ideals of a Dedekind domain. The stable route is through

```lean
normalizedFactors J
prod_normalizedFactors_eq_self hJ0
```

rather than trying to locate a specialized theorem named “ideal is square iff valuations even.”

## 8.1 A small generic multiset lemma

Prove once:

```lean
theorem exists_multiset_half_of_count_even
    {ι : Type*} [DecidableEq ι]
    (s : Multiset ι)
    (h : ∀ a, Even (s.count a)) :
    ∃ t : Multiset ι, s = t + t.
```

This is finite combinatorics. It can be proved by induction on `s`, removing two copies of its head at each step.

## 8.2 The ideal theorem

Then prove:

```lean
theorem ideal_eq_sq_of_normalizedFactors_count_even
    {A : Type*} [CommRing A] [IsDomain A] [IsDedekindDomain A]
    {J : Ideal A}
    (hJ0 : J ≠ ⊥)
    (heven : ∀ P : Ideal A,
      Even ((normalizedFactors J).count P)) :
    ∃ I : Ideal A, J = I ^ 2 := by
  classical
  obtain ⟨t, ht⟩ :=
    exists_multiset_half_of_count_even (normalizedFactors J) heven
  let I : Ideal A := t.prod
  refine ⟨I, ?_⟩
  rw [← prod_normalizedFactors_eq_self hJ0, ht]
  simp [I, pow_two]
```

The final `simp` may need `Multiset.prod_add`; mathematically it is simply

```text
prod(t+t) = prod(t)·prod(t).
```

This lemma is cleaner than constructing a finite product with exponents divided by `2`, and it is completely reusable.

Applying it gives the desired theorem:

```lean
theorem descentInteger_span_is_square_ideal
    (hcop : IsCoprime x z)
    (hz : 0 < z)
    (hy0 : y ≠ 0)
    (hcurve : y ^ 2 = x ^ 3 - 432 * x * z ^ 4 + 8208 * z ^ 6) :
    ∃ I : Ideal O,
      Ideal.span ({descentInteger x z} : Set O) = I ^ 2.
```

`hz` is not used in the local prime parity itself; it belongs to the primitive-projective model and may be retained in the public theorem signature.

The PID/class-number-one result is used only **afterward**: it principalizes `I`, turning

```text
(d)=I²
```

into

```text
d = unit · w²,
```

and the unit squareclass computation reduces the unit to `±1` or `±ε`.

# 9. Relation to the exceptional branch

The ideal-square theorem gives a stronger and cleaner disjunction than the one proposed:

```lean
theorem exceptional_or_squareIdeal
    ... :
    y = 0 ∨ y ^ 2 ∣ ΔM ∨
      ∃ I : Ideal O,
        Ideal.span ({descentInteger x z} : Set O) = I ^ 2 := by
  by_cases hy0 : y = 0
  · exact Or.inl hy0
  by_cases hdisc : y ^ 2 ∣ ΔM
  · exact Or.inr (Or.inl hdisc)
  · exact Or.inr (Or.inr
      (descentInteger_span_is_square_ideal ... hy0 ...))
```

But note that `hdisc` is irrelevant to the proof in the third branch. The mathematically natural theorem is simply `hy0 → squareIdeal`.

The exceptional condition belongs later, when classifying torsion/infinite-descent endpoints. It does not absorb odd ideal valuations.

# 10. The non-square assertion is a separate global theorem

Let

```text
X = x/z²,
Y = y/z³,
θ = 24 - 18α.
```

Then `θ` is a root of

```text
F(T)=T³-432T+8208,
```

and

```text
δ = (x-24z²)+18z²α
  = x-z²θ
  = z²(X-θ).
```

Since `z²` is a square in `K×`, `δ` and `X-θ` define the same class in

```text
K×/K×².
```

For the cubic model `Y²=F(X)`, the standard full `2`-descent/Kummer map is

```text
κ : E(ℚ)/2E(ℚ) → ker(N : K×/K×² → ℚ×/ℚ×²),
κ(P) = [X(P)-θ].
```

Its kernel theorem says

```text
κ(P)=1  ↔  P ∈ 2E(ℚ).
```

Therefore

```text
IsSquare δ  ↔  the associated point P is twice a rational point.
```

There is no contradiction here. A double need not be torsion, and Lutz–Nagell does not say that an integral double is exceptional.

## 10.1 Why Lutz–Nagell does not close this by itself

Lutz–Nagell gives, schematically,

```text
P torsion and P integral
  ⟹ y(P)=0 or y(P)² divides Δ.
```

It does not give

```text
P ∈ 2E(ℚ)
  ⟹ P torsion.
```

Using it in that direction would assume the rank-zero conclusion that the descent is meant to prove.

## 10.2 What extra lemma would make the requested conclusion valid

Any one of the following is sufficient.

### Route A: an already-proved square-branch Diophantine theorem

If the downstream coefficient analysis proves

```lean
theorem descentElement_square_imp_exceptional
    (hcop : IsCoprime x z) (hz : 0 < z)
    (hcurve : ...)
    (hsq : IsSquare (descentElement x z)) :
    y = 0 ∨ y ^ 2 ∣ ΔM,
```

then under `¬Exceptional` one obtains immediately

```lean
have hnotSquare : ¬ IsSquare (descentElement x z) := by
  intro hsq
  exact hnotExceptional
    (descentElement_square_imp_exceptional ... hsq)
```

This is the correct way to fill the requested `NOT a square` field.

### Route B: explicit infinite descent/minimality

Prove that a square Kummer class gives a rational half `Q`, and that either:

```text
Q is exceptional,
```

or its primitive Mordell coordinates have a strictly smaller well-founded measure than those of `P`. Choose a minimal nonexceptional counterexample. Then `δ` cannot be a square.

The public lemma must include the minimality hypothesis or be applied inside a minimal-counterexample proof. Without it, one step of halving does not yield a contradiction.

### Route C: a completed Kummer/rank theorem

A theorem that the kernel quotient is trivial or that `E(ℚ)` is the visible `5`-torsion group also settles the issue, but that is the global rank-zero theorem itself and should not be used as an input to its own proof.

## 10.3 Integrality of a hypothetical square root

If

```text
δ = w² in K,
```

then `w` is an algebraic integer. Indeed, `w` satisfies the monic equation

```text
T²-δ=0
```

over `O`, and `O` is integrally closed. Thus one may write

```text
w = a+bα+cα²,
```

with integral coordinates in the chosen integral basis and invoke an already-proved coefficient/parity theorem. This is a useful adapter, but it does not by itself prove exceptionality.

A Lean-facing lemma is:

```lean
theorem square_root_mem_ringOfIntegers
    {w : K}
    (hw : w ^ 2 = algebraMap O K (descentInteger x z)) :
    ∃ wO : O, algebraMap O K wO = w.
```

The proof uses integrality of `descentInteger`, transitivity of integrality, and the fact that `O` is the integral closure of `ℤ` in `K`.

# 11. Recommended Lean decomposition

The following five lemmas isolate the ideal-square argument cleanly.

## L1. Primitive support exclusion

```lean
theorem descent_prime_not_over_z
    (hcop : IsCoprime x z)
    {P : Ideal O} {p : ℕ}
    (hP : P.IsPrime) (hP0 : P ≠ ⊥)
    (hp : Nat.Prime p)
    (hlies : LiesOverIntPrime P p)
    (hd : descentInteger x z ∈ P) :
    ¬ (p : ℤ) ∣ z
```

## L2. Unique local divisor with odd residue degree

Package all local cases into one statement:

```lean
theorem descent_prime_unique_odd_degree
    (hcop : IsCoprime x z)
    {P : Ideal O} (hPfac : P ∈ normalizedFactors J) :
    ∃ p f : ℕ,
      Nat.Prime p ∧ Odd f ∧
      Ideal.absNorm P = p ^ f ∧
      ∀ Q : Ideal O,
        Q ∈ normalizedFactors J →
        p ∣ Ideal.absNorm Q → Q = P
```

Proof split:

```text
p=2: P=P₂, f=1;
p=3: P=P₃, f=3;
p=11: use P₈/P₁₀ and not_mem_both;
otherwise: quotient-root uniqueness, f=1.
```

## L3. Even factor multiplicities

```lean
theorem descent_normalizedFactor_count_even
    (hcop : IsCoprime x z)
    (hy0 : y ≠ 0)
    (hnorm : Algebra.norm ℚ (descentElement x z) = y ^ 2) :
    ∀ P : Ideal O,
      Even ((normalizedFactors J).count P)
```

This is the norm-valuation calculation.

## L4. Generic Dedekind square-ideal constructor

```lean
theorem ideal_eq_sq_of_normalizedFactors_count_even
    {A : Type*} [CommRing A] [IsDomain A] [IsDedekindDomain A]
    {J : Ideal A}
    (hJ0 : J ≠ ⊥)
    (heven : ∀ P : Ideal A,
      Even ((normalizedFactors J).count P)) :
    ∃ I : Ideal A, J = I ^ 2
```

This is purely algebraic and reusable.

## L5. The concrete ideal-square theorem

```lean
theorem descentInteger_span_eq_sq
    (hcop : IsCoprime x z)
    (hz : 0 < z)
    (hy0 : y ≠ 0)
    (hcurve : y ^ 2 =
      x ^ 3 - 432 * x * z ^ 4 + 8208 * z ^ 6) :
    ∃ I : Ideal O,
      Ideal.span ({descentInteger x z} : Set O) = I ^ 2
```

Then add the non-square result only through a separately named hypothesis/theorem:

```lean
theorem descentIdealSquare_and_notSquare_of_notExceptional
    ...
    (hsq_exc : IsSquare (descentElement x z) → Exceptional x z y)
    (hnotexc : ¬ Exceptional x z y) :
    (∃ I, Ideal.span ({descentInteger x z} : Set O) = I ^ 2) ∧
      ¬ IsSquare (descentElement x z) := by
  constructor
  · exact descentInteger_span_eq_sq ...
  · exact fun hsq => hnotexc (hsq_exc hsq)
```

This signature makes the dependency honest.

# 12. Final corrected theorem statement

The strongest unconditional theorem justified by the listed inputs is:

```lean
theorem descentInteger_ideal_square_unless_zero
    (hcop : IsCoprime x z)
    (hz : 0 < z)
    (hcurve : y ^ 2 =
      x ^ 3 - 432 * x * z ^ 4 + 8208 * z ^ 6) :
    y = 0 ∨
      ∃ I : Ideal O,
        Ideal.span ({descentInteger x z} : Set O) = I ^ 2
```

The exact user-requested theorem

```lean
Exceptional x z y ∨
  ((∃ I, span{d}=I²) ∧ ¬ IsSquare δ)
```

is valid only after supplying one additional global bridge:

```lean
IsSquare δ → Exceptional x z y.
```

If the already-completed coefficient/parity work proves that bridge, then the final proof is short. If it does not, that bridge—not the prime-ideal parity—is the genuine remaining global descent gap.

# Bottom line

- The square-ideal proof is rigorous and finite.
- The primes requiring explicit bookkeeping are `2`, `3`, and `11`.
- At every prime, the unique occurring prime ideal has odd residue degree, so its exponent is even because the norm is a square.
- The Lutz–Nagell exception is not needed to construct the square ideal.
- “`δ` is not a square” is not a local consequence. It must come from a separate square-branch descent theorem, a minimality/infinite-descent argument, or a completed rank/Kummer theorem.
