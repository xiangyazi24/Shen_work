## Exact theorem signature: generic `hD_cont` producer

**Type:** real producer for the `hD_cont` assumption.

Suggested theorem:

```lean

package is continuous.  This discharges the `hD_cont` input of
`holderCosineCoeff_summable_diffOn`; it does not assert endpoint no-flux. -/
theorem differentiatedMildSliceDiffOn_derivWithin_clamp01_continuous
    {χ₀ t₀ θ η CQ HQ M : ℝ} {Q : ℝ → ℝ → ℝ}
    {w initLeg reactLeg : ℝ → ℝ} {Ainit Achem Areact : ℝ}

```

Proof sketch:

1. Reproduce the same assembled Hölder estimate as in `chemMild_C1eta_slice_diffOn`:
   ```lean
   hHolder : ∀ x ∈ Set.Icc (0:ℝ) 1, ∀ y ∈ Set.Icc (0:ℝ) 1,
     |derivWithin w (Set.Icc (0:ℝ) 1) x -
       derivWithin w (Set.Icc (0:ℝ) 1) y| ≤
       (Ainit + |χ₀| * Achem + Areact) * |x-y|^η
   ```
   using:
   - `differentiatedMildSliceDiffOn_derivWithin D hx`
   - `D.init_holder`
   - `D.chem_holder`
   - `D.react_holder`
   - triangle inequality and `ring`, exactly like the existing proof.
2. Prove the assembled constant is nonnegative:
   ```lean
   have hK_nn : 0 ≤ Ainit + |χ₀| * Achem + Areact := ...
   ```
   from `D.Ainit_nn`, `D.Achem_nn`, `D.Areact_nn`.
3. Use:
   ```lean
   holderBound_continuousOn_Icc hη0 hK_nn hHolder
   ```
   to get `ContinuousOn` of `derivWithin w (Icc 0 1)` on `[0,1]`.
4. Compose with clamp:
   ```lean
   have hmaps : Set.MapsTo clamp01 Set.univ (Set.Icc (0:ℝ) 1) :=
     fun x _ => clamp01_mem x
   have hcomp : ContinuousOn
       (fun x => derivWithin w (Set.Icc (0:ℝ) 1) (clamp01 x)) Set.univ :=
     hD_on.comp clamp01_continuous.continuousOn hmaps
   exact continuousOn_univ.mp hcomp
   ```

This is very likely to compile because it reuses only already-compiled proof fragments from the same file.
