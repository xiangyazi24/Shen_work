# Problem 2.7: exact centered-binomial transform, recurrence audit, and what remains unproved

Let \(T_n\) be Cooper's level-11 sequence

\[
(n+1)^3T_{n+1}
=2(2n+1)(5n^2+5n+2)T_n
-8n(7n^2+1)T_{n-1}
+22n(2n-1)(n-1)T_{n-2},
\]

with \(T_0=1,T_1=4,T_2=28\), and define

\[
W_n:=(KT)_n
=\frac1{256^n}\sum_{k=0}^{2n}
\binom{2n}{k}(-2)^{2n-k}T_k.
\tag{0.1}
\]

The conclusions are:

1. The values \(W_0,\ldots,W_{10}\) are computed exactly below.
2. **\(W_n\) does not satisfy the printed Problem 2.7 recurrence.** The failure already occurs at \(n=2\), and an exact cross-multiplied residual is nonzero modulo \(101\).
3. Therefore the challenge denominator \(q_n\) is not \(W_n\), not a constant multiple of it, and—using the established asymptotic \(q_n/W_n\asymp n^{3/2}\)—not a constant multiple of any fixed shift \(W_{n+r}\).
4. The centered-binomial transform does have an exact holonomic description: it is a rational pullback, followed by even projection and rescaling. This proves that \(W_n\) is P-recursive and gives a finite algorithm for its recurrence, but the expanded minimal recurrence has not yet been certified in the repository.
5. The initial conditions for the challenge numerator \(p_n\) and denominator \(q_n\) are given exactly below.
6. The Poincaré roots give a very rapid **analytic mode ratio**, but no irrationality measure for 
   \(
   \zeta(2)+\zeta(3)
   \)
   follows. In fact, the irrationality of this sum is not presently established by the recurrence.
7. The spectral identity is verified exactly at \(t=0,1,2,3\).

# 2. The exact holonomic transform

Let

\[
Z(X)=\sum_{n\ge0}T_nX^n.
\]

First form the ordinary shifted binomial transform

\[
b_m=\sum_{k=0}^m\binom{m}{k}(-2)^{m-k}T_k.
\]

Its generating function is exactly

\[
B(r):=\sum_{m\ge0}b_mr^m
=\frac1{1+2r}
Z\!\left(\frac{r}{1+2r}\right).
\tag{2.1}
\]

Since \(W_n=256^{-n}b_{2n}\), its generating function is the even part

\[
\boxed{
\begin{aligned}
\mathcal W(z):=\sum_{n\ge0}W_nz^n
=\frac12\Bigg[&
\frac1{1+\sqrt z/8}
Z\!\left(\frac{\sqrt z}{16+2\sqrt z}\right)\\
&+\frac1{1-\sqrt z/8}
Z\!\left(-\frac{\sqrt z}{16-2\sqrt z}\right)
\Bigg].
\end{aligned}}
\tag{2.2}
\]

Cooper's generating function is annihilated by

\[
\begin{aligned}
\mathcal L_{11}={}&\theta_X^3
-2X(2\theta_X+1)(5\theta_X^2+5\theta_X+2)\\
&+8X^2(\theta_X+1)\bigl(7(\theta_X+1)^2+1\bigr)\\
&-22X^3(\theta_X+1)(\theta_X+2)(2\theta_X+3),
\end{aligned}
\tag{2.3}
\]

where 
\(
\theta_X=X\,d/dX
\).

Equation (2.2) gives an exact annihilator construction for 
\(
\mathcal W
\):

1. pull back 
   \(
   \mathcal L_{11}
   \)
   along
   \(
   X=\sqrt z/(16+2\sqrt z)
   \)
   and
   \(
   X=-\sqrt z/(16-2\sqrt z)
   \);
2. apply the two scalar gauges 
   \(
   (1+\sqrt z/8)^{-1}
   \)
   and
   \(
   (1-\sqrt z/8)^{-1}
   \);
3. take the least common left multiple over 
   \(
   \mathbf Q(\sqrt z)\langle\theta_z\rangle
   \);
4. descend the invariant operator through 
   \(
   \sqrt z\mapsto-\sqrt z
   \)
   to 
   \(
   \mathbf Q(z)\langle\theta_z\rangle
   \).

This proves rigorously that \(W_n\) is P-recursive; the resulting order is at most six before minimization. Equivalently, a Chyzak-style creative-telescoping certificate has the form

\[
L_K^{(n)}\!igl(K(n,k)T_k\bigr)
=(S_k-1)\left[
K(n,k)\sum_{j=0}^2R_j(n,k)T_{k+j}
\right],
\tag{2.4}
\]

where

\[
K(n,k)=256^{-n}\binom{2n}{k}(-2)^{2n-k}.
\]

The expanded minimal operator \(L_K\) is the object that must be computed before one can test an exact scalar gauge or Ore intertwiner with the challenge operator. Equality of Poincaré roots does not determine \(L_K\).

# 3. Exact proof that \(W_n\) is not a Problem 2.7 solution

Use the recurrence convention implemented in the challenge repository:

\[
u_{n+1}
=\frac{B_n}{A_n}u_n
-\frac{C_{n-1}}{A_{n-1}}u_{n-1}
+\frac{D_{n-2}}{A_{n-2}}u_{n-2},
\qquad n\ge2.
\tag{3.1}
\]

At \(n=2\), define

\[
R_2:=W_3-rac{B_2}{A_2}W_2
+rac{C_1}{A_1}W_1
-rac{D_0}{A_0}W_0.
\tag{3.2}
\]

The needed integer coefficients are

\[
\begin{aligned}
A_0&=1737926668800000,\\
A_1&=43448940241394688,\\
A_2&=539444795559081984,\\
B_2&=463727541823493376,\\
C_1&=21827035693056,\\
D_0&=1791258624.
\end{aligned}
\tag{3.3}
\]

Since

\[
W_0=1,\quad W_1=\frac1{16},\quad
W_2=\frac{355}{2^{14}},\quad
W_3=\frac{44611}{2^{22}},
\]

multiplication by \(2^{22}A_0A_1A_2\) gives the integer numerator

\[
\begin{aligned}
N={}&44611A_0A_1A_2
-355\,2^8B_2A_0A_1\\
&+2^{18}C_1A_0A_2
-2^{22}D_0A_1A_2.
\end{aligned}
\tag{3.4}
\]

Reducing the exact integer expression modulo \(101\) gives

\[
\boxed{N\equiv88\pmod{101}.}
\tag{3.5}
\]

Therefore \(N\ne0\), hence \(R_2\ne0\). This is a short exact certificate that

\[
\boxed{L_{27}W\ne0.}
\]

So the centered-binomial/even transform has the correct limiting spectral map but is **not** itself a solution of the printed finite-\(n\) recurrence.

A second instructive failure is obtained by forcing \(S_n=256^nW_n\) into Cooper's five-parameter four-term shape. Its spectral polynomial would force

\[
(a,b,c,d,e)=(110,16,-32,64,8),
\]

and hence predict

\[
(n+1)^3S_{n+1}
=(2n+1)(110n^2+110n+16)S_n
+n(-32n^2+64)S_{n-1}
+8n(2n-1)(n-1)S_{n-2}.
\]

At \(n=2\), the left side is

\[
27S_3=4817988,
\]

whereas the right side is

\[
3380S_2-128S_1+48S_0=4797600.
\]

Thus even the simplest cubic Cooper-form recurrence with the correct spectrum fails exactly.

# 4. Comparison with the challenge denominator and numerator

The challenge denominator is initialized by

\[
\boxed{
\begin{aligned}
q_0&=-215040420000,\\
q_1&=-\frac{167282265043404}{905},\\
q_2&=-\frac{964185327658080}{6071}.
\end{aligned}}
\tag{4.1}
\]

The numerator is initialized by

\[
\boxed{
\begin{aligned}
p_0&=-612218384750,\\
p_1&=-\frac{9525021973931919}{18100},\\
p_2&=-\frac{29561828382772029}{65380}.
\end{aligned}}
\tag{4.2}
\]

Both sequences are then defined by (3.1).

They do not differ from \(W_n\) by a constant rescaling. If \(q_n=cW_n\), then \(c=q_0\), and one would have

\[
q_1=\frac{q_0}{16}=-13440026250.
\]

But the actual value is

\[
q_1=-\frac{167282265043404}{905},
\]

and these are plainly unequal. Moreover, the established asymptotic

\[
\frac{q_n}{W_n}\asymp n^{3/2}
\]

rules out a constant multiple of any fixed shift \(W_{n+r}\), because \(W_{n+r}/W_n\) tends to a nonzero exponential constant while the extra \(n^{3/2}\) remains.

The viable possibilities are therefore more general:

\[
q_n=h_n\bigl(u_0(n)W_n+u_1(n)W_{n+1}+u_2(n)W_{n+2}\bigr),
\tag{4.3}
\]

with a rank-one hypergeometric twist \(h_n\) and rational functions \(u_j(n)\). Proving (4.3) requires the exact transformed operator \(L_K\) and an Ore-intertwiner identity. It cannot be inferred from the Poincaré polynomial alone.

# 5. Status of the claimed limit

The numerical recurrence strongly supports

\[
\frac{p_n}{q_n}\longrightarrow
L:=\zeta(2)+\zeta(3).
\tag{5.1}
\]

The target period has the exact elementary representation

\[
\boxed{
L=\int_0^1\!\int_0^1
\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy.}
\tag{5.2}
\]

However, the centered-binomial transform does **not** prove (5.1), because it transports only homogeneous Cooper solutions and it fails the printed finite-\(n\) recurrence. In particular, it does not manufacture the new weight-three contribution 
\(
\zeta(3)
\).

A complete proof still needs one of the following finite certificates:

1. an explicit double integral \(I_n\) such that
   \[
   I_n=p_n-Lq_n;
   \]
2. a creative-telescoping identity proving \(L_{27}I=0\);
3. the three initial equalities \(I_j=p_j-Lq_j\), \(j=0,1,2\);
4. a decay estimate showing \(I_n/q_n\to0\).

Equivalently, one may construct a modular/Eisenstein normal function and an exact connection formula. Until such a certificate is supplied, (5.1) remains a numerically verified challenge identity rather than a consequence of the spectral transform.

# 6. Exact Poincaré roots and analytic convergence rate

The limiting characteristic polynomial in the original recurrence scale is

\[
\chi(\lambda)
=\lambda^3-\frac{55}{64}\lambda^2
+\frac1{2048}\lambda-\frac1{2^{20}}.
\tag{6.1}
\]

Writing 
\(
\mu=64\lambda
\)
 gives

\[
F(\mu)=4\mu^3-220\mu^2+8\mu-1.
\tag{6.2}
\]

Let

\[
U=\sqrt[3]{1327067+1419\sqrt{33}},
\qquad
V=\sqrt[3]{1327067-1419\sqrt{33}},
\]

where the real cube roots are chosen. The three exact roots of \(F\) are

\[
\boxed{
\mu_0=\frac{55}{3}+\frac{U+V}{6},}
\tag{6.3}
\]

and

\[
\boxed{
\mu_\pm
=\frac{55}{3}-\frac{U+V}{12}
\pm\frac{i\sqrt3}{12}(U-V).}
\tag{6.4}
\]

The actual Poincaré roots are

\[
\lambda_j=\frac{\mu_j}{64}.
\]

Numerically,

\[
\mu_0=54.96369509838273\ldots,
\qquad
\lambda_0=0.8588077359122301\ldots.
\]

Since

\[
\mu_0\mu_+\mu_-=\frac14,
\]

one has the exact modulus formulas

\[
|\mu_\pm|=\frac1{2\sqrt{\mu_0}},
\qquad
|\lambda_\pm|=\frac1{128\sqrt{\mu_0}}.
\]

Therefore cancellation of the dominant mode would give the analytic error ratio

\[
\boxed{
\rho_{\rm an}
=\left|\frac{\lambda_\pm}{\lambda_0}\right|
=\frac1{2\mu_0^{3/2}}
=0.0012270326576006968\ldots.}
\tag{6.5}
\]

This is a Poincaré–Perron convergence rate. It is **not** an irrationality measure.

Indeed, an irrationality measure requires:

- a proof that \(L\) is irrational;
- integer or controlled-denominator linear forms;
- an exponential upper bound for their heights;
- a matching lower/nonvanishing estimate.

None of these arithmetic denominator estimates follows from (6.1). The irrationality of

\[
\zeta(2)+\zeta(3)
\]

itself is not established by the present formula. Consequently:

\[
\boxed{
\text{no rigorous irrationality measure follows from Problem 2.7 at present.}}
\]

If integer-cleared linear forms were later shown to have coefficient growth
\(
\exp(\sigma n+o(n))
\)
 and size
\(
\exp(-\tau n+o(n))
\), a standard linear-form lemma would give a bound of the form
\(
1+\sigma/\tau
\). The spectral ratio alone does not determine 
\(
\sigma
\)
 because the denominator-clearing growth is unknown.

# 7. Exact verification of the spectral identity

Put

\[
P_{27}(x)=4x^3-220x^2+8x-1,
\qquad
C_{11}(t)=t^3-20t^2+56t-44.
\]

The identity is

\[
16P_{27}\!\left(\frac{(t-2)^2}{4}\right)
=-C_{11}(t)C_{11}(4-t).
\tag{7.1}
\]

At the requested integer values:

| \(t\) | \((t-2)^2/4\) | \(16P_{27}((t-2)^2/4)\) | \(C_{11}(t)\) | \(C_{11}(4-t)\) | \(-C_{11}(t)C_{11}(4-t)\) |
|---:|---:|---:|---:|---:|---:|
| 0 | \(1\) | \(-3344\) | \(-44\) | \(-76\) | \(-3344\) |
| 1 | \(1/4\) | \(-203\) | \(-7\) | \(-29\) | \(-203\) |
| 2 | \(0\) | \(-16\) | \(-4\) | \(-4\) | \(-16\) |
| 3 | \(1/4\) | \(-203\) | \(-29\) | \(-7\) | \(-203\) |

These are exact integer/rational evaluations, not floating-point checks.

# 8. Exact reproducibility code

The following script uses integer arithmetic and `fractions.Fraction` throughout. It computes the table, verifies the recurrence mismatch, generates the challenge numerator and denominator, and checks the spectral identity.

```python
from __future__ import annotations

from fractions import Fraction as Q
from math import comb


def cooper_terms(N: int) -> list[int]:
    """Return T_0,...,T_N exactly."""
    if N < 0:
        return []
    T = [1, 4, 28]
    if N <= 2:
        return T[: N + 1]

    for n in range(2, N):
        numerator = (
            2 * (2*n + 1) * (5*n*n + 5*n + 2) * T[n]
            - 8 * n * (7*n*n + 1) * T[n - 1]
            + 22 * n * (2*n - 1) * (n - 1) * T[n - 2]
        )
        denominator = (n + 1) ** 3
        assert numerator % denominator == 0
        T.append(numerator // denominator)
    return T


def transformed_terms(N: int) -> tuple[list[int], list[Q]]:
    """Return S_n and W_n=S_n/256^n for 0<=n<=N."""
    T = cooper_terms(2 * N)
    S: list[int] = []
    W: list[Q] = []
    for n in range(N + 1):
        s = sum(
            comb(2*n, k) * (-2) ** (2*n - k) * T[k]
            for k in range(2*n + 1)
        )
        S.append(s)
        W.append(Q(s, 256**n))
    return S, W


EXPECTED_S = [
    1,
    16,
    1420,
    178444,
    26111740,
    4168190656,
    703927525756,
    123665831109760,
    22369129340965180,
    4138141710936410560,
    779297227311695157520,
]

S, W = transformed_terms(10)
assert S == EXPECTED_S

print("Exact centered-binomial/even transform")
for n, (s, w) in enumerate(zip(S, W)):
    print(f"n={n:2d}  S_n={s}  W_n={w}")


# Challenge coefficients.
def A(n: int) -> int:
    return (
        1024 * (2*n + 5)**4 * (2*n + 7)**3 * (2*n + 9)**3
        * (946*n*n + 6407*n + 10860)
    )


def B(n: int) -> int:
    return 128 * (2*n + 7)**3 * (2*n + 9)**3 * (
        104060*n**6 + 1745370*n**5 + 12145238*n**4
        + 44886481*n**3 + 92943995*n**2
        + 102256019*n + 46709052
    )


def C(n: int) -> int:
    return 16 * (n + 3)**4 * (2*n + 9)**3 * (
        3784*n**5 + 57792*n**4 + 351019*n**3
        + 1059230*n**2 + 1587211*n + 944620
    )


def D(n: int) -> int:
    return (n + 3)**4 * (n + 4)**6 * (
        946*n*n + 4515*n + 5399
    )


# Exact finite-n failure at n=2.
R2 = (
    W[3]
    - Q(B(2), A(2)) * W[2]
    + Q(C(1), A(1)) * W[1]
    - Q(D(0), A(0)) * W[0]
)
assert R2 != 0
print("\nChallenge residual at n=2:", R2)

N = (
    44611 * A(0) * A(1) * A(2)
    - 355 * 2**8 * B(2) * A(0) * A(1)
    + 2**18 * C(1) * A(0) * A(2)
    - 2**22 * D(0) * A(1) * A(2)
)
assert N % 101 == 88
print("Cross-multiplied residual numerator mod 101:", N % 101)


# Challenge numerator and denominator initial data.
p = [
    Q(-612218384750),
    Q(-9525021973931919, 18100),
    Q(-29561828382772029, 65380),
]
q = [
    Q(-215040420000),
    Q(-167282265043404, 905),
    Q(-964185327658080, 6071),
]


def extend_solution(u: list[Q], N: int) -> list[Q]:
    out = list(u)
    for n in range(2, N):
        out.append(
            Q(B(n), A(n)) * out[-1]
            - Q(C(n - 1), A(n - 1)) * out[-2]
            + Q(D(n - 2), A(n - 2)) * out[-3]
        )
    return out

p = extend_solution(p, 30)
q = extend_solution(q, 30)
assert q[0] * W[1] != q[1] * W[0]


# Exact spectral identity at t=0,1,2,3.
def P27(x: Q) -> Q:
    return 4*x**3 - 220*x**2 + 8*x - 1


def C11(t: int) -> int:
    return t**3 - 20*t**2 + 56*t - 44


for t in range(4):
    x = Q((t - 2)**2, 4)
    lhs = 16 * P27(x)
    rhs = -C11(t) * C11(4 - t)
    assert lhs == rhs
    print(f"t={t}: x={x}, lhs={lhs}, rhs={rhs}")
```
