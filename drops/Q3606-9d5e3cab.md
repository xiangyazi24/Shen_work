# Q3606 audit: general `χ₀ ≤ 0` endpoint boundary min residual

## Short answer

The proposed route is mathematically the right one, but the full endpoint proof is **not** currently a short consequence of the landed lemmas.  The missing piece is not just algebra; it is a one-sided endpoint chemotaxis-divergence limit/bound.

The key hidden issue is that the existing interior lemma

```lean
ShenWork.MinPersistenceAtoms.chemDiv_at_critical
```

only applies at a point where

```lean
HasDerivAt (intervalDomainLift u) 0 x.1
```

is known.  In the endpoint proof, the nearby interior points `x → 0+` or `x → 1-` are not spatial critical points.  At the endpoint min one only has the Neumann one-sided limit

```lean
Tendsto (deriv (intervalDomainLift (u s))) (nhdsWithin 0 (Set.Ioi 0)) (nhds 0)
```

or the right-end analogue.  Thus the chemotaxis divergence has to be controlled as a **one-sided interior limit**, not by pointwise `chemDiv_at_critical` at each nearby point.

So the smallest honest patch is to introduce a narrow residual for the one-sided endpoint chemotaxis-divergence limit, then prove `hbdry_left_chi_nonpos` / `hbdry_right_chi_nonpos` from that residual.  Do not claim `BoundaryMinPersistenceWindowBound p` directly for all `p.χ₀ ≤ 0` yet.

## Why the full endpoint proof is not just `chemDiv_at_critical`

For interior `y`, the chemotaxis divergence expands as the derivative of

```lean
fun y => U y * deriv V y / (1 + V y) ^ p.β
```

where `U = intervalDomainLift (u s)` and `V = intervalDomainLift (v s)`.  It contains two pieces:

```text
U_x(y) * V_x(y) / (1+V(y))^β
+ U(y) * ∂x(V_x/(1+V)^β)(y)
```

At an interior spatial critical point of `u`, the first term vanishes, which is exactly what `chemDiv_at_critical` uses.  Near a boundary, the nearby interior `y` is not critical.  One instead needs:

```text
U_x(y) → 0                 (Neumann endpoint limit for u),
V_x(y) bounded / → 0       (v-slice bounds and/or Neumann endpoint limit),
U(y) → m                  (closed-domain continuity),
∂x(V_x/(1+V)^β)(y) → C     (or at least has a bounded one-sided limit).
```

The current chi-zero endpoint proof avoids all of this because `χ₀ = 0` deletes the chemotaxis term from the PDE before taking the one-sided limit.

## The key proof change from `hbdry_left_chi0`

The chi-zero proof has, on `x ∈ Ioo 0 1`:

```lean
deriv (deriv (intervalDomainLift (u s))) x = Gtime x - R x
```

where:

```lean
Gtime x = deriv (fun r => intervalDomainLift (u r) x) s
R x = intervalDomainLift (u s) x *
  (p.a - p.b * (intervalDomainLift (u s) x) ^ p.α)
```

For `χ₀ ≤ 0`, use instead:

```lean
deriv (deriv (intervalDomainLift (u s))) x =
  Gtime x + p.χ₀ * boundaryChemDivReal p (u s) (v s) x - R x
```

Reason: PDE says

```text
Gtime = uxx - χ₀ * chemDiv + R
```

so

```text
uxx = Gtime + χ₀ * chemDiv - R.
```

Then, from the chem-div residual, choose `Gchem` with:

```lean
hGchem_bd : |Gchem| ≤ fluxCoeffConst p.β (p.ν * M' ^ p.γ)
hChem_lim : Tendsto (boundaryChemDivReal p (u s) (v s))
  (nhdsWithin 0 (Set.Ioo 0 1))
  (nhds (intervalDomainLift (u s) 0 * Gchem))
```

After converting the filter using

```lean
have hfilter : nhdsWithin (0:ℝ) (Set.Ioo 0 1) = nhdsWithin 0 (Set.Ioi 0) :=
  nhdsWithin_Ioo_eq_nhdsGT h01
```

get:

```lean
Tendsto (fun x => deriv (deriv (intervalDomainLift (u s))) x)
  (nhdsWithin 0 (Set.Ioi 0))
  (nhds (Gtime 0 + p.χ₀ * (m * Gchem) - R 0))
```

where:

```lean
m = intervalDomainLift (u s) 0
```

The boundary second-derivative test then gives:

```lean
0 ≤ Gtime 0 + p.χ₀ * (m * Gchem) - R 0
```

so:

```lean
R 0 - p.χ₀ * (m * Gchem) ≤ Gtime 0
```

Since `p.χ₀ ≤ 0`, `-p.χ₀ = |p.χ₀|`, and `|Gchem| ≤ K`,

```text
-p.χ₀ * m * Gchem = |χ₀| * m * Gchem ≥ - |χ₀| * K * m.
```

Together with:

```text
R0 = m * (a - b*m^α) ≥ - b*M'^α*m,
```

this gives:

```text
-(|χ₀|*K + b*M'^α)*m ≤ Gtime0.
```

Finally rewrite `m = sInf ...` via `hargmin` and `Gtime0 = deriv (fun r => intervalDomainLift (u r) 0) s` by definition.

```lean
have hK_nonneg : 0 ≤ K :=
  ShenWork.MinPersistenceAtoms.fluxCoeffConst_nonneg p.hβ
    (mul_nonneg p.hν.le (Real.rpow_nonneg hM' _))

have hchem_lb : -(|p.χ₀| * K) * m ≤ -p.χ₀ * (m * Gchem) := by
  have hχabs : |p.χ₀| = -p.χ₀ := abs_of_nonpos hχ
  have hcoef_nonneg : 0 ≤ -p.χ₀ * m := mul_nonneg (by linarith) hm_nonneg
  have hG_lb : -K ≤ Gchem := (abs_le.mp hGchem_bd).1
  rw [hχabs]
  calc
    -(-p.χ₀ * K) * m = -p.χ₀ * m * (-K) := by ring
    _ ≤ -p.χ₀ * m * Gchem := mul_le_mul_of_nonneg_left hG_lb hcoef_nonneg
    _ = -p.χ₀ * (m * Gchem) := by ring

have hreact_lb : -(p.b * M' ^ p.α) * m ≤ R 0 := by
  rw [hR0]
  have hpow_le : m ^ p.α ≤ M' ^ p.α := Real.rpow_le_rpow hm_nonneg hm_le p.hα.le
  have hmα_nonneg : 0 ≤ m ^ p.α := Real.rpow_nonneg hm_nonneg _
  have h1 : p.b * m ^ p.α ≤ p.b * M' ^ p.α :=
    mul_le_mul_of_nonneg_left hpow_le p.hb
  nlinarith [mul_nonneg p.ha hm_nonneg, mul_nonneg p.hb hmα_nonneg]

have htarget_to_Rchem :
    -(|p.χ₀| * K + p.b * M' ^ p.α) * m ≤ R 0 - p.χ₀ * (m * Gchem) := by
  have hexpand : -(|p.χ₀| * K + p.b * M' ^ p.α) * m =
      -(|p.χ₀| * K) * m + -(p.b * M' ^ p.α) * m := by ring
  rw [hexpand]
  linarith [hchem_lb, hreact_lb]

exact le_trans htarget_to_Rchem hGtime_ge
```

## Can the chem-div endpoint limit be proved from existing infrastructure?

Probably yes mathematically, but not as a short current patch.

A future proof of `BoundaryChemDivLeftLimitBound` should show, as `y → 0+`:

```text
∂x( U * V_x / (1+V)^β )(y)
= U_x(y) * V_x(y)/(1+V(y))^β
  + U(y) * ( -β*(1+V(y))^(-β-1)*V_x(y)^2
             + (1+V(y))^(-β)*V_xx(y) )
```

with:

* `U_x(y) → 0` from the Neumann one-sided limit for `u`;
* `V_x(y)` bounded by `v_slice_coeff_bounds` and in fact `V_x(y) → 0` from Neumann for `v`;
* `U(y) → U(0)` from closed continuity of the `u` slice;
* `V(y) → V(0)` from closed continuity of the `v` slice;
* `V_xx(y)` controlled by the elliptic identity

  ```lean
  deriv (deriv (intervalDomainLift (v s))) y
    = p.μ * intervalDomainLift (v s) y
      - p.ν * (intervalDomainLift (u s) y) ^ p.γ
  ```

  and closed continuity of `u`, `v`, rather than needing an independent endpoint value of `v_xx`.

The current APIs contain most ingredients (`v_slice_coeff_bounds`, `pde_v`, Neumann endpoint limits, closed-domain continuity), but no single theorem packages the one-sided product-rule/limit passage for the chemDiv derivative.  Also note that `intervalDomainChemotaxisDiv` itself is an ordinary derivative of the zero-extension at the subtype coordinate, so for endpoints it is safer to work with `boundaryChemDivReal` under `nhdsWithin _ (Ioo 0 1)` rather than asking for ordinary endpoint derivatives.

## Bottom line

The route is valid, but the missing analytic theorem is a one-sided endpoint chemotaxis-divergence factor/bound.  The current interior infrastructure is not enough by direct application because endpoint-adjacent interior points are not critical points of `u`, and `intervalDomainChemotaxisDiv` is not the right object to evaluate at the endpoint as an ordinary zero-extension derivative.  The next honest patch should expose and consume:
