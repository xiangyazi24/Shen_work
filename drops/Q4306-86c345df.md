# Definitive verdict

Yes. The squared-heat-barrier comparison closes at the weak energy level with no prior strict positivity and no pointwise `u_xx`.

Let

```text
z := w - u,
z₊ := max z 0,
E(t) := ∫₀¹ z₊(t,x)² dx,
D(t) := ∫₀¹ |∂ₓz₊(t,x)|² dx.
```

If `u` is a weak solution and `w` is a weak subsolution of the **same divergence-form operator**

```text
v_t = ν v_xx - χ₀ ∂ₓ(v g) + c v,
```

then, for almost every time,

```text
(1/2) E'(t) + (ν/2) D(t) ≤ C E(t),

C := χ₀² G² / (2ν) + Kc,
G  := ‖g‖∞,
Kc := ‖c‖∞.
```

Consequently

```text
E(t) ≤ E(s) exp(2 C (t-s)).
```

If `w(s)≤u(s)`, then `E(s)=0`; hence `E(t)=0`, so `w≤u` almost everywhere. Spatial continuity upgrades this to pointwise comparison. Since the squared heat barrier is strictly positive for `t>s`, one obtains `u(t,x)>0`.

The coefficient `Kc=‖c‖∞` is safe. The sharper choice is

```text
Kc := essSup c₊,
```

because only an upper bound for `c` enters the comparison estimate.

# 1. Exact weak inequality and sign convention

Work on a restarted strip

```text
Q_{s,T} := (s,T) × (0,1),
0 ≤ s < T.
```

Assume

```text
u,w ∈ L²(s,T;H¹(0,1)) ∩ C([s,T];L²(0,1)),
u_t,w_t ∈ L²(s,T;H¹(0,1)*)
```

and

```text
g,c ∈ L∞(Q_{s,T}),
ν>0.
```

The solution `u` and subsolution `w` are required to use the same `g` and `c`:

```text
u_t - νu_xx + χ₀∂ₓ(ug) - cu = 0,
w_t - νw_xx + χ₀∂ₓ(wg) - cw ≤ 0.
```

Subtracting gives

```text
z_t - νz_xx + χ₀∂ₓ(zg) - cz ≤ 0,
z=w-u.
```

The corresponding variational inequality is: for every nonnegative spatial test `φ∈H¹(0,1)`, for almost every time `t`,

```text
⟨z_t,φ⟩
  + ν ∫₀¹ z_x φ_x
  - χ₀ ∫₀¹ z g φ_x
  - ∫₀¹ c z φ
≤ 0.                                      (W)
```

This sign is load-bearing. It follows from

```text
∫ χ₀ ∂ₓ(zg) φ = -χ₀ ∫ zg φ_x
```

once the boundary term is discharged.

## Boundary handling

For a classical derivation,

```text
ν∫ z_xx φ
  = ν[z_x φ]₀¹ - ν∫z_x φ_x,

-χ₀∫∂ₓ(zg) φ
  = -χ₀[zg φ]₀¹ + χ₀∫zg φ_x.
```

The diffusion boundary term vanishes because both `u` and `w` have homogeneous Neumann derivative. The drift boundary term vanishes because `g(t,0)=g(t,1)=0`.

For the Lean proof, the cleaner design is to **start with `(W)`**. The natural Neumann boundary conditions are already encoded in the weak bilinear form, so no endpoint integral has to be represented. In this formulation only `g∈L∞` is needed for the energy estimate; a trace of `g` is needed only in the producer that converts a classical/divergence statement to `(W)`.

# 2. Stampacchia truncation identities

Define

```text
T(r) := r₊ := max r 0.
```

For `z(t,·)∈H¹(0,1)`, the Lipschitz composition theorem gives

```text
z₊(t,·) ∈ H¹(0,1)
```

and the Stampacchia chain rule gives

```text
∂ₓz₊ = 1_{z>0} ∂ₓz                         a.e.    (S1)
```

Therefore

```text
z z₊ = z₊²,                                  (S2)

z g ∂ₓz₊ = z₊ g ∂ₓz₊,                       (S3)

z_x ∂ₓz₊ = |∂ₓz₊|²                          (S4)
```

almost everywhere.

Explanation of `(S3)`: on `{z>0}`, `z=z₊`; on `{z≤0}`, `∂ₓz₊=0` almost everywhere. The value on the zero set causes no problem.

The time-chain rule is the convex Hilbert-triple chain rule for

```text
Φ(r) := (1/2)(r₊)²,
Φ'(r)=r₊.
```

It yields

```text
(d/dt) (1/2 ∫₀¹ z₊²) = ⟨z_t,z₊⟩             a.e.    (T)
```

and also the integrated identity

```text
1/2 E(t) - 1/2 E(s)
  = ∫_s^t ⟨z_τ,z₊(τ)⟩ dτ.
```

If the repository does not yet have a generic `H¹-H^{-1}` truncation chain rule, do not build a full Sobolev-composition library first. Mirror the existing negative-part architecture and expose `(T)`, `(S1)` and `(S4)` as specialized fields for the comparison gap. This is exactly how `NegativePartEnergyEstimateRegularData` isolates the corresponding hard chain-rule leaves for `-u₋`.

# 3. Testing by `z₊`

Insert `φ=z₊` in `(W)`. Using `(S2)`–`(S4)` gives

```text
⟨z_t,z₊⟩
  + ν ∫₀¹ |∂ₓz₊|²
  - χ₀ ∫₀¹ z₊ g ∂ₓz₊
  - ∫₀¹ c z₊²
≤ 0.
```

Equivalently,

```text
⟨z_t,z₊⟩
≤ -ν D(t)
   + χ₀ ∫₀¹ z₊ g ∂ₓz₊
   + ∫₀¹ c z₊².                              (E0)
```

Combining with `(T)`,

```text
(1/2)E'(t)
≤ -νD(t)
   + χ₀∫₀¹ z₊g∂ₓz₊
   + ∫₀¹ c z₊².                              (E1)
```

This is the exact weak energy identity/inequality to formalize.

# 4. Drift Young estimate with the exact constant

Assume

```text
|g(t,x)| ≤ G
```

almost everywhere, with `G≥0`. Then

```text
χ₀ ∫ z₊ g ∂ₓz₊
≤ |χ₀| G ∫ z₊ |∂ₓz₊|.
```

There are two equally formalizable ways to finish.

## Pointwise Young, then integrate

For every `a,b∈ℝ`,

```text
|χ₀| G |a||b|
≤ (ν/2)b² + (χ₀²G²/(2ν))a².                 (Y)
```

Indeed, with

```text
A := sqrt(ν)|b|,
B := |χ₀|G/sqrt(ν) |a|,
```

`2AB≤A²+B²`. Since `ν>0`, this gives `(Y)`.

Integrating `(Y)` with `a=z₊`, `b=∂ₓz₊` yields

```text
|χ₀|G ∫ z₊|∂ₓz₊|
≤ (ν/2)D(t)
  + (χ₀²G²/(2ν))E(t).                        (D)
```

## Cauchy–Schwarz, then scalar Young

Alternatively,

```text
∫ z₊|∂ₓz₊|
≤ ‖z₊‖₂ ‖∂ₓz₊‖₂,
```

followed by

```text
|χ₀|G ‖z₊‖₂ ‖∂ₓz₊‖₂
≤ (ν/2)‖∂ₓz₊‖₂²
  + (χ₀²G²/(2ν))‖z₊‖₂².
```

The pointwise route is usually easier in the repository's interval-integral style because it avoids introducing square roots of integrals.

# 5. Zero-order term

If

```text
|c(t,x)| ≤ Kc
```

almost everywhere, then

```text
∫₀¹ c z₊²
≤ ∫₀¹ |c|z₊²
≤ Kc E(t).                                   (R)
```

The sharper hypothesis is merely

```text
c(t,x) ≤ Kc,
```

because `z₊²≥0`. Thus one may use `Kc=essSup c₊`. The requested safe choice is


# 6. Absorption and differential inequality

Insert `(D)` and `(R)` into `(E1)`:

```text
(1/2)E'(t)
≤ -νD(t)
   + (ν/2)D(t)
   + (χ₀²G²/(2ν))E(t)
   + Kc E(t).
```

Hence

```text
(1/2)E'(t) + (ν/2)D(t)
≤ C E(t),                                    (E2)
```

where

```text
C := χ₀²G²/(2ν) + Kc.                        (C)
```

Since `D(t)≥0`,

```text
(1/2)E'(t) ≤ C E(t),
E'(t) ≤ 2C E(t).                             (E3)
```

With `G=‖g‖∞` and `Kc=‖c‖∞`, this is exactly

```text
C = χ₀² ‖g‖∞²/(2ν) + ‖c‖∞.
```

The sign `χ₀<0` is irrelevant to the comparison closure after taking absolute values; only `χ₀²` remains.

## Initial ordering

Suppose the restarted barrier is

```text
w(t,x)=exp(-M(t-s)) (S_ν(t-s)f(x))²
```

and the seed satisfies

```text
f(x)² ≤ u(s,x).
```

Then

```text
z(s,x)=w(s,x)-u(s,x)=f(x)²-u(s,x)≤0,
```

so

```text
z₊(s,x)=0,
E(s)=0.
```

Equation `(G)` gives

```text
0≤E(t)≤0,
```

hence `E(t)=0`.

Because the integrand `z₊²` is nonnegative,

```text
E(t)=0
⇒ z₊(t,x)=0 a.e.
⇒ w(t,x)≤u(t,x) a.e.
```

If the time slices of `u` and `w` are continuous on `[0,1]`, an a.e. inequality upgrades to every point: if `w(t,x₀)>u(t,x₀)`, continuity would make the same strict inequality hold on a nonempty interval of positive measure, contradicting the a.e. conclusion.

Finally, if `f≥0` is not identically zero, positivity improving of the Neumann heat semigroup gives

```text
S_ν(t-s)f(x)>0
```

for `t>s` and all `x∈[0,1]`; therefore

```text
w(t,x)>0,
u(t,x)≥w(t,x)>0.
```

At the restart time itself, the barrier need not be strictly positive everywhere; strict positivity is concluded for every later time.

# 8. Why the same divergence form is essential

The homogeneous closure depends on

```text
w g - u g = (w-u)g = zg.
```

If the barrier uses a different drift `ĝ`, then

```text
w ĝ - u g = zg + w(ĝ-g),
```

and the energy inequality contains an external forcing term

```text
χ₀∫ w(ĝ-g)∂ₓz₊.
```

This does not vanish when `E(s)=0` and does not yield the zero solution by homogeneous Gronwall without an additional favorable sign or a separately vanishing error.

Similarly, if the reaction coefficients differ,

```text
c_w w - c_u u = c_u z + (c_w-c_u)w,
```

again producing an inhomogeneous forcing term.

Thus the barrier and solution must use exactly the same divergence coefficient and reaction coefficient, or the coefficient mismatch must be proved to have the correct one-sided sign.

If one instead expands the divergence term,

```text
-χ₀∂ₓ(ug)
 = (-χ₀g)u_x + (-χ₀g_x)u,
```

then the exact nondivergence coefficients are

```text
B = -χ₀ g,
C = c - χ₀ g_x.
```

A barrier theorem using only `B=-χ₀g` and reaction `c` omits `-χ₀g_x` and is not the same operator. Staying in divergence form avoids this bookkeeping in the comparison estimate and requires no bound on `g_x` there.

# 9. Minimal regularity and absence of circularity

A standard sufficient package is

```text
z=w-u ∈ L²(s,T;H¹(0,1)),
z_t ∈ L²(s,T;H¹(0,1)*),
z ∈ C([s,T];L²(0,1)),
g,c ∈ L∞((s,T)×(0,1)),
```

together with the weak subsolution inequality `(W)` and the initial ordering.

Equivalently it is enough to carry the corresponding fields separately for `u` and `w`.

No step uses

```text
u_xx pointwise,
w_xx pointwise,
u^γ∈H²,
or u>0.
```

The comparison proof uses only:

```text
u,w ∈ L²_tH¹_x,
weak time derivative in H^{-1},
bounded coefficients,
Stampacchia truncation,
energy continuity,
Gronwall.
```

For the chemotaxis coefficients, nonnegativity is enough. If `u∈H¹∩L∞`, `u≥0`, and `γ≥1`, then

```text
u^γ∈H¹,
∂ₓ(u^γ)=γu^{γ-1}u_x a.e.,
```

because `r↦r^γ` has bounded first derivative on `[0,M]`. No positive floor is needed at this first derivative order. The floor is needed only for higher spatial derivatives when factors such as `u^{γ-2}` occur.

In fact, if comparison is retained in divergence form, the energy estimate itself does not need `u^γ∈H¹`; it only needs enough resolver regularity to know `g∈L∞` and enough source regularity to establish the weak equation.

# 11. Lean-level elementary inequalities

The drift bound can be proved without an abstract normed-space Young theorem. Establish the pointwise inequality by `nlinarith` from a square:

```text
0 ≤ (sqrt ν * |b| - |χ₀|*G/sqrt ν * |a|)².
```

Or avoid square roots by multiplying through by `2ν>0`:

```text
2ν |χ₀|G |a||b|
≤ ν² b² + χ₀²G² a²,
```

which follows from

```text
0 ≤ (ν|b| - |χ₀|G|a|)².
```

After division by `2ν`, this is exactly `(Y)`. This second form is usually more stable in Lean because it uses only field division at the final step.

Useful ordinary Mathlib ingredients include:

```text

abs_mul
abs_nonneg

intervalIntegral.integral_mono_on
```
