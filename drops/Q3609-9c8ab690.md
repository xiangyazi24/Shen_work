# Q3609 audit: proving the endpoint `BoundaryChemDiv*LimitBound` residuals

## Bottom line

The full proof of

```lean
MinPersistenceAtoms.BoundaryChemDivLeftLimitBound p
MinPersistenceAtoms.BoundaryChemDivRightLimitBound p
```

is mathematically valid with the current classical-regularity package, but it is not the next smallest Lean patch.  The compile-realistic route should first split the endpoint chem-div limit into two smaller layers:

1. a **general interior product-rule formula** for `intervalDomainChemotaxisDiv` without the critical-point hypothesis;
2. an **endpoint flux-coefficient limit/bound** residual (or theorem) saying the coefficient
   `-β(1+v)^(-β-1)v_x^2 + (1+v)^(-β)v_xx` has a one-sided endpoint limit bounded by `fluxCoeffConst`.

Then prove `BoundaryChemDivLeftLimitBound` / `Right` from that coefficient-limit residual plus the existing Neumann limits for `u_x` and `v_x`.

The important correction to the proposed direct formula is: `v_slice_coeff_bounds` gives **pointwise interior bounds**, not directly an endpoint limit.  It is still very useful: after proving the coefficient has a one-sided limit, the pointwise bound plus closedness of `{z | |z| ≤ K}` gives `|gchem| ≤ K`.

## Why the full proof is plausible

For `y → 0+`, inside `Set.Ioo 0 1`, write:

```text
U = intervalDomainLift (u s)
V = intervalDomainLift (v s)
P(y) = V_x(y) / (1 + V(y))^p.β
C(y) = -p.β * (1 + V(y))^(-p.β - 1) * V_x(y)^2
       + (1 + V(y))^(-p.β) * V_xx(y)
```

Then on the interior:

```text
chemDiv(y) = U_x(y) * P(y) + U(y) * C(y).
```

The endpoint limits available or derivable are:

```text
U_x(y) → 0          by the u-Neumann endpoint limit from hsol.regularity
V_x(y) → 0          by the v-Neumann endpoint limit from hsol.regularity
U(y) → U(0)         by closed-domain continuity of the u slice
V(y) → V(0)         by closed-domain continuity of the v slice
V_xx(y) → μ V(0) - ν U(0)^γ
                  by the interior elliptic PDE_v plus continuity of U,V
```

Therefore:

```text
P(y) → 0
C(y) → (1 + V(0))^(-p.β) * (p.μ * V(0) - p.ν * U(0)^p.γ)
chemDiv(y) → U(0) * gchem
```

where:

```text
gchem = (1 + V(0))^(-p.β) * (p.μ * V(0) - p.ν * U(0)^p.γ)
```

or equivalently with the vanished `-β...V_x^2` term included.  The right endpoint is identical with `0+` replaced by `1-`.

## Hidden endpoint issues

There are two Lean-relevant endpoint issues, but neither invalidates the route.

First, `intervalDomainChemotaxisDiv` is an ordinary derivative of the zero-extension at a subtype coordinate.  At endpoints, ordinary derivatives of zero-extensions are the wrong object.  The current residual wisely avoids this by using:

```lean
boundaryChemDivReal p u v
```

under the interior filters:

```lean
nhdsWithin (0 : ℝ) (Set.Ioo (0 : ℝ) 1)
nhdsWithin (1 : ℝ) (Set.Ioo (0 : ℝ) 1)
```

On these filters, the `if y ∈ Icc 0 1` branch is always the genuine interval chemotaxis divergence:

```lean
filter_upwards [self_mem_nhdsWithin] with y hy
-- hy : y ∈ Set.Ioo 0 1
have hyIcc : y ∈ Set.Icc (0 : ℝ) 1 := Set.Ioo_subset_Icc_self hy
simp [boundaryChemDivReal, hyIcc]
```

Second, closed `ContDiffOn ℝ 2 ... (Set.Icc 0 1)` does not automatically give a named endpoint value of `v_xx` in the zero-extension sense.  Do not ask for one.  Prove the one-sided interior limit of `v_xx` using `pde_v` on `Ioo 0 1`:

```lean
V_xx(y) = p.μ * V(y) - p.ν * U(y)^p.γ
```

and then pass to the endpoint by continuity of `U,V` on `Icc 0 1`.

## What `v_slice_coeff_bounds` does and does not give

`MinPersistenceAtoms.v_slice_coeff_bounds` gives, on `Set.Ioo 0 1`,

```lean
|deriv (intervalDomainLift (v s)) y| ≤ 2 * (p.ν * M' ^ p.γ)
|deriv (deriv (intervalDomainLift (v s))) y| ≤ 2 * (p.ν * M' ^ p.γ)
```

This is enough to prove the pointwise interior flux-coefficient bound by calling:

```lean
MinPersistenceAtoms.flux_coeff_bound
```

for each interior `y`.  It does **not** by itself produce `gchem` or a `Tendsto` theorem.  The coefficient `Tendsto` must come from the Neumann/PDE_v/continuity limit argument above.  Once you have

```lean
Tendsto C (nhdsWithin 0 (Set.Ioo 0 1)) (nhds gchem)
```

and an eventual bound `∀ᶠ y, |C y| ≤ K`, use a closed-set/tendsto lemma to conclude:

```lean
|gchem| ≤ K
```

A useful helper is:

```lean
/-- A bound closed under limits. -/
theorem abs_le_of_tendsto_of_eventually_abs_le
    {l : Filter ℝ} [NeBot l] {f : ℝ → ℝ} {a K : ℝ}
    (hf : Tendsto f l (nhds a))
    (hK : ∀ᶠ x in l, |f x| ≤ K) :
    |a| ≤ K := by
  have hclosed : IsClosed {z : ℝ | |z| ≤ K} :=
    isClosed_le continuous_abs continuous_const
  exact hclosed.mem_of_tendsto hf hK
```

If `mem_of_tendsto` has a slightly different namespace in this Mathlib version, this is still the right proof shape.

## Step 1: product-rule formula for interior chemDiv

Place this either in the new file or, better long-term, in `IntervalDomainChemDivCritical.lean` next to `chemDiv_at_critical`.

```lean
/-- General interior chemotaxis-divergence product rule, without assuming
`u_x = 0`.  The critical-point theorem `chemDiv_at_critical` is the special case
`ux = 0`. -/
theorem chemDiv_at_interior_product
    {p : CM2Params} {u v : intervalDomainPoint → ℝ} {x : intervalDomainPoint}
    {ux vx vxx : ℝ}
    (hux : HasDerivAt (intervalDomainLift u) ux x.1)
    (hv : HasDerivAt (intervalDomainLift v) vx x.1)
    (hvxx : HasDerivAt (deriv (intervalDomainLift v)) vxx x.1)
    (hvnn : ∀ y, 0 ≤ intervalDomainLift v y) :
    intervalDomainChemotaxisDiv p u v x =
      ux * (vx / (1 + intervalDomainLift v x.1) ^ p.β) +
        intervalDomainLift u x.1 *
          (-p.β * (1 + intervalDomainLift v x.1) ^ (-p.β - 1) * vx ^ 2 +
            (1 + intervalDomainLift v x.1) ^ (-p.β) * vxx)
```

Proof skeleton:

* copy the setup from `chemDiv_at_critical`;
* use `flux_integrand_hasDerivAt hv hvxx (hpos x.1)` to get the derivative of `V_x*(1+V)^(-β)`;
* prove the division/rpow equality as in `chemDiv_at_critical`;
* use `hux.mul hP` instead of the critical-point product rule with `0`;
* unfold `intervalDomainChemotaxisDiv` and `ring`.

This is a good first compile-now patch: it is local calculus, no new residual, and it removes the main reason the existing `chemDiv_at_critical` cannot be used at endpoint filters.

## Step 2: define the coefficient representative

In the same new file:

```lean
/-- The flux-divergence coefficient
`∂x(v_x/(1+v)^β)` written in the rpow form used by `flux_coeff_bound`. -/
def boundaryFluxCoeffReal
    (p : CM2Params) (v : intervalDomainPoint → ℝ) (y : ℝ) : ℝ :=
  -p.β * (1 + intervalDomainLift v y) ^ (-p.β - 1) *
      (deriv (intervalDomainLift v) y) ^ 2 +
    (1 + intervalDomainLift v y) ^ (-p.β) *
      deriv (deriv (intervalDomainLift v)) y
```

Then prove the interior equality to `boundaryChemDivReal`:

```lean
/-- On the open interval, `boundaryChemDivReal` has the product-rule form. -/
theorem boundaryChemDivReal_eq_product_interior
    {p : CM2Params} {u v : intervalDomainPoint → ℝ} {y ux vx vxx : ℝ}
    (hy : y ∈ Set.Ioo (0 : ℝ) 1)
    (hux : HasDerivAt (intervalDomainLift u) ux y)
    (hv : HasDerivAt (intervalDomainLift v) vx y)
    (hvxx : HasDerivAt (deriv (intervalDomainLift v)) vxx y)
    (hvnn : ∀ z, 0 ≤ intervalDomainLift v z) :
    boundaryChemDivReal p u v y =
      ux * (vx / (1 + intervalDomainLift v y) ^ p.β) +
        intervalDomainLift u y * boundaryFluxCoeffReal p v y
```

Proof skeleton:

* set `x : intervalDomainPoint := ⟨y, Set.Ioo_subset_Icc_self hy⟩`;
* unfold `boundaryChemDivReal` and take the `dif_pos` branch;
* apply `chemDiv_at_interior_product` at `x`;

## Step 3: split coefficient-limit residual

If the full coefficient-limit proof is still too large, introduce this narrower residual before attempting to prove it.

```lean
/-- Left endpoint one-sided limit and bound for the flux-divergence coefficient. -/
def BoundaryFluxCoeffLeftLimitBound (p : CM2Params) : Prop :=
  ∀ {T s M' : ℝ} {u v : ℝ → intervalDomainPoint → ℝ},
    IsPaper2ClassicalSolution intervalDomain p T u v →
    0 < s → s < T → 0 ≤ M' →
    (∀ x : intervalDomainPoint, u s x ≤ M') →
    ∃ gchem : ℝ,
      |gchem| ≤ fluxCoeffConst p.β (p.ν * M' ^ p.γ) ∧
      Tendsto (boundaryFluxCoeffReal p (v s))
        (nhdsWithin (0 : ℝ) (Set.Ioo (0 : ℝ) 1))
        (nhds gchem)

/-- Right endpoint one-sided limit and bound for the flux-divergence coefficient. -/
def BoundaryFluxCoeffRightLimitBound (p : CM2Params) : Prop :=
  ∀ {T s M' : ℝ} {u v : ℝ → intervalDomainPoint → ℝ},
    IsPaper2ClassicalSolution intervalDomain p T u v →
    0 < s → s < T → 0 ≤ M' →
    (∀ x : intervalDomainPoint, u s x ≤ M') →
    ∃ gchem : ℝ,
      |gchem| ≤ fluxCoeffConst p.β (p.ν * M' ^ p.γ) ∧
      Tendsto (boundaryFluxCoeffReal p (v s))
        (nhdsWithin (1 : ℝ) (Set.Ioo (0 : ℝ) 1))
        (nhds gchem)

structure BoundaryFluxCoeffEndpointLimitBounds (p : CM2Params) : Prop where
  left : BoundaryFluxCoeffLeftLimitBound p
  right : BoundaryFluxCoeffRightLimitBound p
```

This is narrower than `BoundaryChemDivLeftLimitBound`: it removes the `U_x * P` vanishing and `U → U(endpoint)` multiplication from the residual, both of which are already available from classical regularity and Neumann data.

After the reducer is in place, the true analytic proof is:

```lean

    (p : CM2Params) :

```

and similarly for right, if all necessary Tendsto/rpow helpers cooperate.

Proof sketch for the left:

1. Fix `hsol hs0 hsT hM' hu_le`; set `U := intervalDomainLift (u s)`, `V := intervalDomainLift (v s)`.
2. Extract from `hsol.regularity`:
   * closed slice C² / continuity for `U,V` from `h7`;
   * interior C² for `V` from `h3` or `h7.mono`;
   * v-Neumann left limit `(h6 s htmem).2.1`.

   ```lean

     (nhdsWithin 0 (Set.Ioo 0 1))
   ```

4. Combine with `V_x → 0` and `V → V 0` to prove:


## Answer to the concrete question about `v_slice_coeff_bounds`

`v_slice_coeff_bounds` does **not** already prove a stronger coefficient limit.  It proves the interior derivative bounds needed for the coefficient **bound**:

```lean
|v_x| ≤ 2B
|v_xx| ≤ 2B
```

Together with `flux_coeff_bound`, this gives:

```lean
∀ᶠ y in nhdsWithin 0 (Set.Ioo 0 1),
  |boundaryFluxCoeffReal p (v s) y| ≤ fluxCoeffConst p.β B
```

It does not supply the endpoint `Tendsto` or identify `gchem`.  The `Tendsto` part should be proved from `pde_v` and continuity, or temporarily exposed as `BoundaryFluxCoeffLeftLimitBound` / `Right`.

## Sign algebra downstream

No new sign algebra is needed after `BoundaryChemDivLeftLimitBound` / `Right` are produced.  `IntervalDomainBoundaryHboundChiNonpos.lean` already proves:

```lean
hbdry_left_chi_nonpos_of_chemDiv_limit
hbdry_right_chi_nonpos_of_chemDiv_limit
```

and the algebra is correct:

```text
PDE:        Gtime = uxx - χ₀*chemDiv + R
therefore:  uxx = Gtime + χ₀*chemDiv - R
limit:      0 ≤ Gtime + χ₀*(m*gchem) - R
so:         R - χ₀*(m*gchem) ≤ Gtime
```

With `χ₀ ≤ 0`, `-χ₀ = |χ₀|`, `m ≥ 0`, and `|gchem| ≤ K`,

```text
-χ₀ * m * gchem ≥ -|χ₀| * K * m.
```

The reaction algebra gives:

```text
R = m(a - b*m^α) ≥ -b*M'^α*m.
```

Hence:

```text
-(|χ₀|*K + b*M'^α)*m ≤ Gtime.
```

This is exactly the target in the windowed boundary residual.

## Pitfalls to avoid

* Do not evaluate ordinary endpoint derivatives of `intervalDomainChemotaxisDiv`; stay under `nhdsWithin endpoint (Set.Ioo 0 1)` and use `boundaryChemDivReal`.
* Do not use `chemDiv_at_critical` near the endpoint.  Interior points approaching the endpoint are not critical points of `u`.
* Do not ask closed `ContDiffOn` for a zero-extension endpoint `v_xx` value.  Use `pde_v` to transfer `v_xx` to the continuous RHS on the interior and take the one-sided limit.
* Keep `M'` and `M` separate.  Boundary chem-div residuals use `M'` directly; B-form window wrappers pass `M' := SupNormBridge.regimeBound p M`.
