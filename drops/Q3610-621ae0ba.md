## 1. x-endpoint cancellations are mathematically correct

The proposed statements

```lean
intervalNeumannConjugateKernel t 0 y = 0
intervalNeumannConjugateKernel t 1 y = 0
```

are correct for `0 < t` with the repo sign convention

```lean
K̃(t,x,y) = ∑' k, -G(t, x - y + 2k) + G(t, x + y + 2k)
```

where `G = heatKernel` is even.

Recommended theorem statements:

```lean
-- Target file: ShenWork/PDE/IntervalFullKernelSourceIBP.lean
-- or put x=0 in IntervalFullKernelInitialIBP.lean and x=1 in SourceIBP.

namespace ShenWork.IntervalNeumannFullKernel

/-- x-face cancellation: `K̃(t,0,y)=0`. -/
theorem conjugateKernel_x_zero {t : ℝ} (ht : 0 < t) (y : ℝ) :
    intervalNeumannConjugateKernel t 0 y = 0

/-- x-face cancellation: `K̃(t,1,y)=0`. -/
theorem conjugateKernel_x_one {t : ℝ} (ht : 0 < t) (y : ℝ) :
    intervalNeumannConjugateKernel t 1 y = 0

end ShenWork.IntervalNeumannFullKernel
```

Proof sketch for `x = 0`:

* Unfold `intervalNeumannConjugateKernel`.
* Split the `tsum` using summability from `latticeGaussianSummable ht (-y)` and `latticeGaussianSummable ht y`.
* It remains to show

  ```lean
  (∑' k : ℤ, heatKernel t (y + 2 * (k : ℝ))) =
    ∑' k : ℤ, heatKernel t (-y + 2 * (k : ℝ))
  ```

* Set `f k := heatKernel t (-y + 2 * (k : ℝ))` and reindex by `Equiv.neg`.  For each `k`, use `heatKernel_neg` and ring normalization:

  ```lean
  heatKernel t (y + 2 * (k : ℝ))
    = heatKernel t (-y + 2 * ((-k : ℤ) : ℝ))
  ```

Proof sketch for `x = 1`:

* Unfold and split the `tsum` exactly as above.
* The required equality is

  ```lean
  (∑' k : ℤ, heatKernel t (1 + y + 2 * (k : ℝ))) =
    ∑' k : ℤ, heatKernel t (1 - y + 2 * (k : ℝ))
  ```

* This is **not** the same shift as the existing `y = 1` theorem.  The correct map is

  ```lean
  k ↦ -k - 1
  ```

  because

  ```lean
  1 - y + 2 * ((-k - 1 : ℤ) : ℝ)
    = -(1 + y + 2 * (k : ℝ))
  ```

  and `heatKernel` is even.  In Lean, the equivalence can be built from `Equiv.neg` followed by `Equiv.addRight (-1 : ℤ)`.

Flag: both x-face statements should include `ht : 0 < t`; unlike the existing `y = 0` cancellation, these are not termwise zero and need summability/reindexing.

## 2. Smallest next theorem that materially advances the reducer

After x-endpoint cancellation lands, the exact face operator theorem is immediate and useful as a sanity check:

```lean

theorem initialLegConjugateEndpointOperator_zero_at_left
    {df : ℝ → ℝ} {t : ℝ} (ht : 0 < t) :
    -(∫ y in (0 : ℝ)..1,
        df y * intervalNeumannConjugateKernel t 0 y) = 0

/-- At the exact absorbing x-faces, the conjugate-kernel operator is zero. -/
theorem initialLegConjugateEndpointOperator_zero_at_right
    {df : ℝ → ℝ} {t : ℝ} (ht : 0 < t) :
    -(∫ y in (0 : ℝ)..1,
        df y * intervalNeumannConjugateKernel t 1 y) = 0

end ShenWork.IntervalSemigroupC1ApproxIdentity
```

But this is **not** enough for the current endpoint-layer reducer, because `InitialLegConjugateEndpointOperatorVanish` asks for `x ≤ η ∨ 1 - η ≤ x`, not just `x = 0 ∨ x = 1`.  Do not try to get the layer theorem merely from pointwise continuity in `x`: the continuity modulus of the heat kernel degenerates as `t → 0+`, so that route is not uniform in `t`.

The smallest genuinely material next theorem is the absolute first-moment bound for the conjugate kernel.  It is a narrow kernel lemma and uses only existing facts:

* `abs_conjugateKernel_le`
* `conjugateKernel_L1_bound`
* `continuousOn_conjugateKernel_snd`
* `IntervalSemigroupUniform.intervalNeumannFullKernel_abs_moment_le`

Recommended statement:

```lean
-- Target file: ShenWork/PDE/IntervalSemigroupC1ApproxIdentity.lean
-- No new imports needed: this file already imports IntervalSemigroupUniform
-- and IntervalFullKernelSourceIBP.

namespace ShenWork.IntervalSemigroupC1ApproxIdentity

/-- Absolute first moment of the conjugate kernel, dominated by the full Neumann
first moment.  This is the workhorse for oscillation control and endpoint-layer
operator vanishing. -/
theorem conjugateKernel_abs_moment_le
    {t : ℝ} (ht : 0 < t) {x : ℝ} (hx : x ∈ Set.Icc (0 : ℝ) 1) :
    (∫ y in (0 : ℝ)..1,
        |y - x| * |intervalNeumannConjugateKernel t x y|)
      ≤ 4 * t / Real.sqrt (4 * Real.pi * t)

end ShenWork.IntervalSemigroupC1ApproxIdentity
```

Proof sketch:

1. Prove interval-integrability of both moment integrands from continuity on `[0,1]`:

   ```lean
   (continuous_abs.comp (continuous_id.sub continuous_const)).continuousOn
   ```

   multiplied by either `continuousOn_conjugateKernel_snd ht x` or `continuousOn_intervalNeumannFullKernel_snd ht x`.

2. Apply `intervalIntegral.integral_mono_on (by norm_num : (0:ℝ) ≤ 1)` with pointwise bound

   ```lean
   |y - x| * |intervalNeumannConjugateKernel t x y|
     ≤ |y - x| * intervalNeumannFullKernel t x y
   ```

   using `abs_conjugateKernel_le ht x y` and `abs_nonneg`.

3. Finish with the existing first-moment estimate:

   ```lean
   ShenWork.IntervalSemigroupUniform.intervalNeumannFullKernel_abs_moment_le
     ht x hx
   ```

This theorem is non-circular: it does not mention `InitialLegConjugateDerivativeApprox`, any split residual, or any future Dirichlet approximate identity theorem.

### Immediate theorem unlocked by the moment bound: oscillation control

Once `conjugateKernel_abs_moment_le` exists, the oscillation split can be closed from continuity alone:

```lean
-- Target file: ShenWork/PDE/IntervalSemigroupC1ApproxIdentity.lean

namespace ShenWork.IntervalSemigroupC1ApproxIdentity

/-- Continuous derivative profiles have vanishing conjugate-kernel oscillation term. -/
theorem initialLegConjugateOscillationControl_of_continuousOn
    {df : ℝ → ℝ}
    (hdf_cont : ContinuousOn df (Set.Icc (0 : ℝ) 1)) :
    InitialLegConjugateOscillationControl df

end ShenWork.IntervalSemigroupC1ApproxIdentity
```

Proof sketch:

* Use compactness of `Set.Icc 0 1` to get a bound `|df y| ≤ M` and uniform continuity.
* As in `IntervalSemigroupUniform.intervalFullSemigroup_tendstoUniformlyOn`, build a linear modulus

  ```lean
  |df y - df x| ≤ ε / 2 + C * |y - x|
  ```

  for all `x,y ∈ [0,1]`.
* Estimate

  ```lean
  |-(∫ (df y - df x) * K̃)|
    ≤ ∫ |df y - df x| * |K̃|
    ≤ ε/2 * ∫ |K̃| + C * ∫ |y-x|*|K̃|.
  ```

* Use `conjugateKernel_L1_bound ht x` for the first term and `conjugateKernel_abs_moment_le ht hx` for the second.
* Choose `t` small using the same square-root estimate already used in `IntervalSemigroupUniform.lean`:

  ```lean
  4 * t / Real.sqrt (4 * Real.pi * t) ≤ 2 * Real.sqrt t
  ```

This fully discharges the oscillation residual without positivity of `-K̃`.

### Endpoint-layer operator vanish from current facts

The same moment bound also gives the endpoint operator layer, again without Dirichlet positivity:

```lean
-- Target file: ShenWork/PDE/IntervalSemigroupC1ApproxIdentity.lean

namespace ShenWork.IntervalSemigroupC1ApproxIdentity

/-- Bounded profiles that are small near the endpoints have vanishing
conjugate-kernel operator on endpoint x-layers. -/
theorem initialLegConjugateEndpointOperatorVanish_of_endpointSmall_bound
    {df : ℝ → ℝ} {M : ℝ}
    (hdf_cont : ContinuousOn df (Set.Icc (0 : ℝ) 1))
    (hM : 0 ≤ M)
    (hdf_bound : ∀ y ∈ Set.Icc (0 : ℝ) 1, |df y| ≤ M)
    (hsmall : InitialLegDerivativeEndpointSmall df) :
    InitialLegConjugateEndpointOperatorVanish df

end ShenWork.IntervalSemigroupC1ApproxIdentity
```

Proof sketch:

* Given `ε`, use `hsmall (ε/4)` to get an endpoint size `ηs` with

  ```lean
  |df y| < ε/4
  ```

  when `y ≤ ηs` or `1 - ηs ≤ y`.

* Take the operator endpoint layer `η := ηs / 2` (or a min with `1/4`).

* For `x ≤ η`, prove the pointwise linear endpoint bound

  ```lean
  |df y| ≤ ε/4 + (2 * M / ηs) * |y - x|
  ```

  for all `y ∈ [0,1]`: if `y ≤ ηs`, use endpoint smallness; if `ηs < y`, then `|y-x| ≥ ηs/2`, so the global bound `|df y| ≤ M` is absorbed by the linear term.

* For `1 - η ≤ x`, use the symmetric argument: if `y < 1 - ηs`, then `|y-x| ≥ ηs/2`; if `1 - ηs ≤ y`, use endpoint smallness.

* Then estimate the operator by

  ```lean
  |-(∫ df y * K̃)|
    ≤ ∫ |df y| * |K̃|
    ≤ ε/4 * ∫ |K̃| + (2*M/ηs) * ∫ |y-x|*|K̃|.
  ```

* Use `conjugateKernel_L1_bound` and `conjugateKernel_abs_moment_le`; choose `t` small for the moment term.

This theorem is stronger than exact endpoint operator zero and directly supplies the existing reducer:

```lean

```


## 3. Full positivity of `-intervalNeumannConjugateKernel` is not necessary next

A full theorem

```lean
0 ≤ -intervalNeumannConjugateKernel t x y
```

is mathematically true: `-K̃` is the Dirichlet heat kernel on `[0,1]`.  But proving it directly from the signed periodized lattice may be substantially heavier than what Task151 needs next.

It is **not** necessary for either of these near-term advances:

1. **Oscillation control**: use

   ```lean
   |K̃| ≤ K_full
   ```

   and full-kernel moment concentration.

2. **Endpoint operator vanish**: use endpoint smallness of `df`, the same `|K̃| ≤ K_full`, `∫ |K̃| ≤ 1`, and the conjugate absolute first moment.

For weighted mass defect, full positivity is also avoidable.  The better route is to introduce only the reflected half-kernel and a reflected-mass/tail lemma.

Recommended narrow definitions/theorems for the mass-defect path:

```lean
-- Target file: a new file is reasonable:
-- ShenWork/PDE/IntervalConjugateKernelMassDefect.lean
-- Imports:
-- import ShenWork.PDE.IntervalSemigroupC1ApproxIdentity

namespace ShenWork.IntervalNeumannFullKernel

/-- Reflected half of the full Neumann image kernel. -/
noncomputable def intervalNeumannReflectedKernelPart (t x y : ℝ) : ℝ :=
  ∑' k : ℤ, heatKernel t (x + y + 2 * (k : ℝ))

/-- Algebraic mass-defect identity: since `-K̃ = A - B` and `K_full = A + B`,
`∫(-K̃)-1 = -2∫B`. -/
theorem conjugateKernel_massDefect_eq_neg_two_reflectedMass
    {t : ℝ} (ht : 0 < t) (x : ℝ) :
    (-(∫ y in (0 : ℝ)..1, intervalNeumannConjugateKernel t x y) - 1)
      = -2 * (∫ y in (0 : ℝ)..1,
        intervalNeumannReflectedKernelPart t x y)

/-- Interior reflected mass tends to zero away from the absorbing endpoints. -/
theorem reflectedKernelPart_interior_mass_tendsto_zero
    {η : ℝ} (hη0 : 0 < η) (hηhalf : η < (1 / 2 : ℝ)) :
    TendstoUniformlyOn
      (fun t x : ℝ =>
        ∫ y in (0 : ℝ)..1,
          intervalNeumannReflectedKernelPart t x y)
      (fun _ : ℝ => 0) (𝓝[>] (0 : ℝ)) (Set.Icc η (1 - η))

end ShenWork.IntervalNeumannFullKernel
```

The reflected part is a sum of nonnegative heat kernels, so its positivity is cheap.  This is much narrower than full positivity of `A - B`.

## 4. Circularity and false endpoint assumptions to avoid

### Avoid confusing y-face and x-face cancellations

Existing facts:

```lean
conjugateKernel_at_zero  -- y = 0
conjugateKernel_at_one   -- y = 1
```

The proposed new facts are different:

```lean
conjugateKernel_x_zero   -- x = 0
conjugateKernel_x_one    -- x = 1
```

The proof of `conjugateKernel_x_one` uses `k ↦ -k - 1`, not the `k ↦ k + 1` shift from `conjugateKernel_at_one`.

### Do not assume closed-interval convergence without endpoint compatibility

The closed-`Icc` theorem for

```lean
-(∫ y in 0..1, df y * K̃ t x y) → df x
```

is false unless

```lean
df 0 = 0 ∧ df 1 = 0
```

At `x = 0` and `x = 1`, the Dirichlet/conjugate operator vanishes, so the limiting endpoint value must be zero.

### Do not use exact x-face cancellation as a uniform endpoint-layer proof

Exact cancellation gives the operator value at `x = 0` and `x = 1`; it does not by itself give the whole endpoint layer `x ≤ η ∨ 1 - η ≤ x`.  The uniform endpoint layer should be proved by moment/tail estimates, not by pointwise continuity in `x` at fixed `t`.

### Best next route

The recommended order is:

1. Add the x-face cancellations (`conjugateKernel_x_zero`, `conjugateKernel_x_one`).
2. Add `conjugateKernel_abs_moment_le` in `IntervalSemigroupC1ApproxIdentity.lean`.
3. Use it to close `InitialLegConjugateOscillationControl` from `ContinuousOn df [0,1]`.
4. Use it with endpoint smallness and a boundedness hypothesis to close `InitialLegConjugateEndpointOperatorVanish`.
5. For weighted mass defect/interior approximation, add the reflected-half kernel and prove reflected-mass decay away from endpoints.  This avoids the heavy full positivity theorem for `-K̃`.
