# Q4353: pointwise `C²` bookkeeping for the truncated chemotaxis flux

## Executive verdict

The formulas for `h`, `g`, and `Q` are correct, and the denominator part is indeed δ-free because `R ≥ 0` gives `1 + R ≥ 1`.

However, the claimed bound

```text
‖Q‖_{C²} ≤ C(M, β, ‖R‖_{C³})
```

is **not valid from only**

```text
0 ≤ u ≤ M,
R ∈ C³,
Q = u · R_x · (1 + R)^(-β).
```

The obstruction is the Leibniz formula:

```text
Q'' = u'' g + 2 u' g' + u g''.
```

So a genuine `C²` bound for `Q` needs bounds on `u'` and `u''`, or an equivalent way to express/control those derivatives. A correct envelope is

```text
‖Q''‖∞ ≤ U₂ · ‖g‖∞ + 2 U₁ · ‖g'‖∞ + M · ‖g''‖∞,
```

where

```text
U₁ := ‖u'‖∞,
U₂ := ‖u''‖∞.
```

With `U₁,U₂` available, the flux part is δ-free. Without them, the stated `C(M,β,‖R‖_{C³})` bound is under-hypothesized.

## Derivatives of the denominator factor

Let

```text
h := (1 + R)^(-β).
```

Then

```text
h' = -β (1 + R)^(-β - 1) R'
```

and

```text
h'' = β(β + 1) (1 + R)^(-β - 2) (R')²
      - β (1 + R)^(-β - 1) R''.
```

Since `R ≥ 0`,

```text
1 + R ≥ 1.
```

For `β ≥ 0`, all denominator powers satisfy

```text
|(1 + R)^(-β)|     ≤ 1,
|(1 + R)^(-β - 1)| ≤ 1,
|(1 + R)^(-β - 2)| ≤ 1.
```

Thus if

```text
K := max (‖R'‖∞) (max ‖R''‖∞ ‖R'''‖∞)
```

or simply `K := ‖R‖_{C³}`, then

```text
‖h‖∞  ≤ 1,
‖h'‖∞ ≤ β K,
‖h''‖∞ ≤ β(β + 1) K² + β K.
```

No lower floor for `u` is used here.

## Bounds for `g = R' h`

By product rule,

```text
g  = R' h,
g' = R'' h + R' h',
g'' = R''' h + 2 R'' h' + R' h''.
```

Using the previous estimates:

```text
‖g‖∞ ≤ K,
```

```text
‖g'‖∞ ≤ K + β K²,
```

and

```text
‖g''‖∞
  ≤ K + 2β K² + K · (β(β + 1)K² + βK)
  = K + 3β K² + β(β + 1)K³.
```

A clean bundled constant is therefore:

```text
G0(K,β) := K
G1(K,β) := K + β K²
G2(K,β) := K + 3β K² + β(β + 1)K³.
```

Then

```text
‖g‖∞  ≤ G0(K,β),
‖g'‖∞ ≤ G1(K,β),
‖g''‖∞ ≤ G2(K,β).
```

## Correct `Q = u g` bound

For

```text
Q = u g,
```

we have

```text
Q' = u' g + u g'
```

and

```text
Q'' = u'' g + 2 u' g' + u g''.
```

Assume

```text
0 ≤ u ≤ M,
‖u'‖∞ ≤ U1,
‖u''‖∞ ≤ U2,
‖R‖_{C³} ≤ K,
R ≥ 0,
β ≥ 0.
```

Then

```text
‖Q‖∞ ≤ M · G0(K,β),
```

```text
‖Q'‖∞ ≤ U1 · G0(K,β) + M · G1(K,β),
```

and

```text
‖Q''‖∞ ≤ U2 · G0(K,β)
          + 2 U1 · G1(K,β)
          + M · G2(K,β).
```

Expanded:

```text
‖Q''‖∞ ≤ U2 · K
          + 2 U1 · (K + βK²)
          + M · (K + 3βK² + β(β + 1)K³).
```

This is the formalizable constant structure I would use in Lean.

## Where the δ-floor is and is not needed

For the flux denominator part:

```text
(1 + R)^(-β)
```

there is no δ-floor requirement because `R ≥ 0` gives `1 + R ≥ 1`.

For the product

```text
Q = u · R' · (1 + R)^(-β),
```

the estimate is also δ-free once `u`, `u'`, and `u''` are bounded. The formula never divides by `u`.

The δ-floor enters only when differentiating real powers such as

```text
u^γ
```

at orders where negative powers appear. For example,

```text
(u^γ)'' = γ(γ - 1) u^(γ - 2) (u')² + γ u^(γ - 1) u''.
```

If `1 ≤ γ < 2`, then `γ - 2 < 0`, so the factor `u^(γ-2)` requires a positive lower bound `δ ≤ u`. That is the real δ-floor issue.

For logistic

```text
u(a - b u^α) = a u - b u^(α + 1),
```

the second derivative only contains powers `u^α` and `u^(α-1)`. Since `α ≥ 1`, these are nonnegative powers, so no δ-floor is needed at second order.

## Lean-facing recommendation

Do not state the flux lemma as depending only on `M`, `β`, and `‖R‖_{C³}` unless the local context also gives `u'` and `u''` bounds or an equation that converts those into controlled `R` derivatives.

The safe lemma shape is:

```lean
lemma chemFlux_C2_bound_delta_free
    (hM : ∀ x, 0 ≤ u x ∧ u x ≤ M)
    (hu1 : ∀ x, |u' x| ≤ U1)
    (hu2 : ∀ x, |u'' x| ≤ U2)
    (hR0 : ∀ x, 0 ≤ R x)
    (hR1 : ∀ x, |R' x| ≤ K)
    (hR2 : ∀ x, |R'' x| ≤ K)
    (hR3 : ∀ x, |R''' x| ≤ K)
    (hβ : 0 ≤ β) :
    ∀ x,
      |(u * (R' * (1 + R)^(-β)))'' x| ≤
        U2 * K
        + 2 * U1 * (K + β * K^2)
        + M * (K + 3 * β * K^2 + β * (β + 1) * K^3)
```

The exact Lean syntax will depend on whether derivatives are represented as `deriv`, named derivative functions, or local `HasDerivAt` witnesses, but mathematically this is the right bound.

## Bottom line

The denominator and `g` estimates are δ-free. The flux `Q` estimate is δ-free **provided** you have `u'` and `u''` sup bounds. But `‖Q‖_{C²} ≤ C(M,β,‖R‖_{C³})` alone is not valid, because `Q''` explicitly contains `u''` and `u'`. The δ-floor belongs to the `u^γ` Nemytskii/elliptic source step, not to the denominator-controlled flux product itself.
