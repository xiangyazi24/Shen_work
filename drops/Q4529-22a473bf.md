# 1. Exact sign-flip dictionary

Let

```lean
q t x := W t x - U t x
V t x := U t x - W t x
```

so `V = -q`. Then

```text
negativePart V = positivePart q,
(negativePart V)^2 = (positivePart q)^2,
negativePartEnergy V = ∫ (W-U)₊²,
negativePartTest V = -(W-U)₊.
```

The active-set identities are identical:

```text
V < 0  ↔  W-U > 0,
∂t V = -∂t(W-U),
∂x V = -∂x(W-U).
```

Consequently, on the active set,

```text
V_t (-V₋)
  = (-q_t)(-q₊)
  = q_t q₊
  = ½ ∂t(q₊²),

V_x ∂x(-V₋)
  = (-q_x)(-∂x q₊)
  = |∂x q₊|².
```

This is not merely analogous; it is the same multiplication after changing `q` to `V := -q`.

A one-time rewrite lemma is enough to expose the identity in Lean. Schematically:

```lean
lemma negativePart_sub_eq_positivePart_reverse (a b : ℝ) :
    negativePart (a - b) = positivePart (b - a) := by
  -- Exact simp normal form depends on the imported definitions.
  simp [negativePart, positivePart, sub_eq_add_neg, add_comm]
```

It is usually even cheaper not to rewrite until the final conclusion: keep all energy code in terms of `negativePart V`, obtain `0 ≤ V`, and finish with `sub_nonneg.mp`.

# 2. The committed derivative/IBP template

The basic objects in
`ShenWork/Paper2/IntervalBFormCron2NegativePartEnergy.lean` are

```lean

  fun x => negativePart (intervalDomainLift u x)

def negativePartEnergy
    (u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ :=
  ∫ x, (negativePartLift (u t) x) ^ 2 ∂ intervalMeasure 1

def negativePartDissipation
    (u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ :=

    (u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ → ℝ :=
  fun x => -negativePartLift (u t) x
```

The exact test is therefore **`-u₋`**, not `u₋`. This sign matches the repository's weak-form convention and makes both the time term and the diffusion term positive.

# 3. The newer mild-restart/Jensen template

For the proposed Jensen comparison, this is likely the more important reuse path.

`ShenWork/Paper2/IntervalNegativePartWeakEnergy.lean` already proves the generic convex/Markov facts that the 3,500-line V6 producer uses.

## 3.1 Pointwise Jensen and semigroup contraction

The pointwise convex Jensen theorem is

```lean
negativePart_sq_intervalFullSemigroupOperator_le
```

and its integrated consequence is

```lean
negativePartEnergy_semigroup_le.
```

The latter says schematically

```text
∫ (S_h f)₋² ≤ ∫ f₋².
```

For `V := U-W`, this is exactly the contraction

```text
∫ (S_h(W-U))₊² ≤ ∫ (W-U)₊²
```

after the sign flip. There is no reason to reprove convexity for `r ↦ (r₊)²`.

## 3.2 Supporting-line inequality

The scalar convexity atom is

```lean
negativePart_sq_sub_le_final
```

with

```text
(b₋)² - (a₋)² ≤ (-2 b₋)(b-a).
```

Combining this with heat-semigroup contraction gives the generic restart lemma

```lean
negativePartEnergy_sub_le_remainder_pairing.
```

Its essential statement is:

```text
u = S_h f + z
────────────────────────────────────────────────────────────
∫ u₋² - ∫ f₋² ≤ ∫ (-2u₋) z.
```

This theorem is completely PDE-independent. It already contains both pieces that a Jensen comparison needs:

1. the homogeneous heat leg cannot increase the bad-part energy; and
2. the inhomogeneous remainder is paired against the final bad-part test.

## 3.3 Apply it directly to `(W-U)₊`

Take

```lean
u := intervalDomainLift (V t)
f := intervalDomainLift (V a)
z := comparisonRestartRemainder a t
```

with

```text
V(t) = S_{t-a} V(a) + comparisonRestartRemainder(a,t).
```

Then

```text
u₋ = V(t)₋ = (W(t)-U(t))₊,
f₋ = V(a)₋ = (W(a)-U(a))₊,
```

and the generic theorem returns

```text
Ecmp(t) - Ecmp(a)
  ≤ ∫ -2 (W(t)-U(t))₊ · comparisonRestartRemainder(a,t).
```

That is the Jensen comparison energy increment. The comparison-specific proof should start from this theorem rather than rebuild the convexity, heat-kernel mass, Fubini, or semigroup-contraction layers.

If the restart relation is initially available only as an inequality, either:

```text
(a) isolate a favorable signed remainder and prove the needed pairing bound, or
(b) define z := V(t)-S_{t-a}V(a) and use the subsolution inequality only to estimate
    ∫(-2V(t)₋)z.
```

The generic energy lemma itself only needs the equality representation and a bound on that final pairing.

# 4. Exact structural correspondence

| Negative-part nonnegativity | Jensen comparison |
|---|---|
| trajectory `u` | difference `V := U-W` |
| bad part `u₋` | `V₋=(W-U)₊` |
| test `-u₋` | `-V₋=-(W-U)₊` |
| energy `∫u₋²` | energy `∫(W-U)₊²` |
| target `u≥0` | target `V≥0`, i.e. `W≤U` |
| heat leg contracts `u₋` | same heat leg contracts `(W-U)₊` after sign flip |
| PDE remainder paired with `-2u₋` | difference remainder paired with `-2(W-U)₊` |
| `E'≤KE` or backward quotient bound | same scalar inequality |
| initial trace gives `E→0` | restart ordering often gives `E(r₀)=0` exactly |
| zero energy → pointwise nonnegativity | zero energy → `W≤U` pointwise |

The only new analytic content is the **difference remainder estimate**. The energy machinery is already present.

# 6. What is not directly reusable

The following lemmas are tied to one truncated trajectory and should not be applied blindly to a difference:

```lean
NegativePartWeakTestIdentityAt
truncatedChemFluxLifted_mul_negDeriv_integral_eq_zero
truncatedLogisticLocal_mul_neg_negativePart_le
negativePart_half_energy_deriv_le_regular
```

Why:

* `NegativePartWeakTestIdentityAt` is hardwired to the original truncated PDE evaluated at one field.
* The exact chemotaxis cancellation uses the same field's positive and negative supports. In general,
  `Flux(W)-Flux(U)` does not vanish on `{W>U}`.
* The logistic lemma is a one-state negative-zone estimate. A comparison needs a one-sided difference estimate such as
  `(R(W)-R(U))(W-U)₊ ≤ L(W-U)₊²`.
* `negativePart_half_energy_deriv_le_regular` combines those single-state facts, so it is not a generic comparison theorem unless the local `...CoreDataFor` refactor has already abstracted the residual bound.

The comparison-specific replacement is one of the following equivalent interfaces.

Derivative form:

```lean
½ Ecmp'(t) + Dcmp(t) ≤ α Dcmp(t) + C Ecmp(t),
α ≤ 1.
```

After absorption:

```text
Ecmp'(t) ≤ 2C Ecmp(t).
```

Mild/Jensen form:

```text
Ecmp(t)-Ecmp(a)
  ≤ ∫ -2V(t)₋ · comparisonRestartRemainder(a,t),
```

followed by an endpoint/time-average estimate whose limit is at most `K Ecmp(t)`.

# Bottom line

The exact identity is

```text
new bad part = (W-U)₊ = (U-W)₋.
```

So the Jensen comparison is a new **difference-remainder estimate**, not a new energy theory.

* If the proof is classical enough to have `E'` and spatial chain rules, reuse the derivative core and `intervalDomainL2_gronwall_exp_of_diffIneq`.
* If the proof is already based on a Jensen/mild restart, reuse
  `negativePartEnergy_sub_le_remainder_pairing` and the backward-Gronwall lemmas; explicit IBP is unnecessary.
* If the local generalized theorem `nonneg_of_negativePartEnergyCoreDataFor` exists, instantiate it at `V := U-W` and translate `0 ≤ V` to `W ≤ U`.

That is the structural mirror Codex should target; another multi-thousand-line positive-part build would be duplication.
