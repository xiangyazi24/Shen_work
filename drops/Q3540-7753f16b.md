## Boundary / interior warning

Do not try to use pointwise equality on the closed interval `[0,1]`. Existing PDE/representative equalities are on:

```lean
Set.Ioo (0 : ℝ) T ×ˢ Set.Ioo (0 : ℝ) 1
```

or strict time slabs with spatial `Set.Ioo 0 1`. Interval integrals over `0..1` are over `Set.uIoc 0 1`, i.e. `Set.Ioc 0 1` when `0 ≤ 1`: the left endpoint `0` is not included and the right endpoint `1` is included but null. The standard bridge is:

```lean
intervalIntegral.integral_congr_ae
Set.uIoc_of_le zero_le_one
Real.volume_singleton
```

with an eventual condition `x ≠ 1` to turn `x ∈ Ioc 0 1` into `x ∈ Ioo 0 1`.

This endpoint-a.e. step is essential. A proof that asks for closed-spatial equality is stronger than current APIs and will hit the endpoint ordinary-derivative trap.
