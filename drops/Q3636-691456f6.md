# Q3636: next producer for `InitialLegUniformHolderAtZero`

## Recommendation

The best next **faithful** producer is **true Hölder preservation of the Neumann heat semigroup**, not the Dirichlet/C1 producer through `intervalFullCoupledDuhamel_grad_initial_bound`.

The already-added theorem

```lean
InitialLegUniformHolderAtZero_of_semigroup_deriv_bound
```

is a good thin MVT consumer and should be kept.  It proves the right abstract bridge:

```text
uniform bound on ∂x S(t)u0 for 0<t<=T  ==>  uniform C^theta bound of S(t)u0 down to t=0.
```

But it is not yet a real producer for the intended `PositiveInitialDatum`, because the missing premise is exactly the uniform derivative bound.  The tempting existing theorem

```lean
intervalFullCoupledDuhamel_grad_initial_bound
```

is easy to consume, but it has the side condition

```lean
hu₀_one : u₀ 1 = 0
```

and therefore proves a **Dirichlet/right-end zero** special case, not the positive Neumann initial data needed by Theorem 1.1.  For positive initial data, `u0 1 = 0` contradicts uniform positivity.  So a no-sorry C1-Dirichlet wrapper would compile, but it would not move the real theorem.

## Proof skeleton for the true Neumann Hölder producer

For `x,y ∈ [0,1]`, use the reflected Brownian/Neumann kernel representation.  The Neumann semigroup is an average over reflected points.  In one dimension, reflection is nonexpansive modulo the interval folding map.  Therefore, for every kernel point, the distance between the two reflected samples generated from starts `x` and `y` is at most `|x-y|`.

Mathematically:

```text
|S(t)u0(x) - S(t)u0(y)|
  = |∫ [u0(R_x(z)) - u0(R_y(z))] dγ_t(z)|
  <= ∫ H0 * |R_x(z)-R_y(z)|^θ dγ_t(z)
  <= H0 * |x-y|^θ * ∫ dγ_t
  = H0 * |x-y|^θ.
```

In the current kernel-series formalization, the same argument can be implemented as a reflected-kernel coupling rather than as a derivative estimate.  The important feature is that the constant does **not** blow up as `t -> 0`.

A more kernel-explicit route is:

1. Define the interval folding map `fold01 : ℝ → intervalDomainPoint` with period `2` and reflection across endpoints.
2. Prove the pure metric lemma:

```lean
-- theorem dist_fold01_add_le (x y z : ℝ)
--   (hx : x ∈ Set.Icc 0 1) (hy : y ∈ Set.Icc 0 1) :
--   |(fold01 (x + z)).1 - (fold01 (y + z)).1| ≤ |x - y|
```

3. Re-express `intervalFullSemigroupOperator t (intervalDomainLift u₀) x` as the Gaussian expectation of `u₀ (fold01 (x+z))`.
4. Apply `hu₀_holder` pointwise under the integral and use mass one.

This is the faithful theorem for continuous/Hölder positive Neumann initial data.

```lean

```

whereas `PositiveInitialDatum` gives some `η > 0` with `η ≤ u₀ x`, in particular `0 < u₀ ⟨1,_⟩`.

## Endpoint pitfalls

1. **Zero extension is not globally Hölder on `ℝ`.**  `ChemMildHolderBootstrap` correctly states spatial Hölder bounds on `intervalDomainPoint`; do not move the target to `intervalDomainLift u₀` on all real `x,y`.  A strictly positive endpoint jumps to zero outside `[0,1]`.

2. **The MVT route needs a uniform derivative bound.**  Generic heat smoothing gives `|∂x S(t)u0| <= C t^{-1/2} ||u0||∞`, which blows up at zero and does not produce `InitialLegUniformHolderAtZero`.

3. **The existing IBP gradient bound is right-end zero.**  It is excellent for a Dirichlet-like C1 datum, but it is not a Neumann positive-data theorem.

4. **True Hölder preservation avoids endpoint derivative compatibility.**  It should rely on positivity, mass one, and reflected coupling/nonexpansiveness, not on `u₀'` or `u₀(1)=0`.

## Bottom line

For the real Paper2 route, the next best target is:

```text
InitialDatumHolder u₀ θ H₀ -> InitialLegUniformHolderAtZero u₀ T θ H₀
```

proved by Neumann heat Hölder preservation.  The C1/Dirichlet wrapper is easy but should be marked as a side lemma only; it does not discharge the positive-initial-data frontier.  The existing MVT consumer is already the right thin abstraction, but by itself it only shifts the burden to a uniform derivative estimate, which is unavailable for merely continuous positive Neumann data.
