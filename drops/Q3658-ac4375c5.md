# Q3658: `FullKernelImageBranchInterchange_of_bounded`

## Short answer

The target theorem is honest and should be the next theorem, but I would not try to prove it by routing through `foldedHeatIntegral_eq_fullKernel_branch_tsums`.  Prove it directly from the definition of

```lean
intervalFullSemigroupOperator t f x.1
```

and the full kernel definition.  The only genuinely delicate step is the Mathlib interchange

```lean
integral_tsum_of_summable_integral_norm
```

for the family

```lean
hk k y :=
  (heatKernel t (x.1 - y + 2 * (k : ℝ))
    + heatKernel t (x.1 + y + 2 * (k : ℝ))) * f y
```

against `volume.restrict (Set.Ioc 0 1)`.  The norm-integral summability should be proved by domination with

```lean
M * ((∫ y in 0..1, heatKernel t (x.1 - y + 2*k))
      + ∫ y in 0..1, heatKernel t (x.1 + y + 2*k))
```

and `IntervalNeumannFullKernel.summable_cell_heat_interval_integral ht x.1`.

The existing `foldedHeat_branch_summable_of_bounded` is useful evidence, but it is for the folded `-x.1 ± y` branches.  For the final `FullKernelImageBranchInterchange` split, add the same summability lemma for the actual `x.1 ± y` branches.  That avoids reindexing pain at the final `Summable.tsum_add`.

### Atom 1: actual branch summability

```lean
/-- Bounded data make the two actual full-kernel image branch integral sequences
summable.  This is the non-folded analogue of
`foldedHeat_branch_summable_of_bounded`. -/
theorem fullKernel_image_branch_summable_of_bounded
    {t : ℝ} (ht : 0 < t) (x : intervalDomainPoint) {f : ℝ → ℝ} {M : ℝ}
    (_hM : 0 ≤ M) (hf_meas : Measurable f) (hf_bound : ∀ y, |f y| ≤ M) :
    Summable (fun k : ℤ =>
      ∫ y in (0 : ℝ)..1, heatKernel t (x.1 - y + 2 * (k : ℝ)) * f y) ∧
    Summable (fun k : ℤ =>
      ∫ y in (0 : ℝ)..1, heatKernel t (x.1 + y + 2 * (k : ℝ)) * f y) := by
  -- Copy the proof of `foldedHeat_branch_summable_of_bounded`, but set
  --   A k := ∫ y in 0..1, heatKernel t (x.1 - y + 2*k)
  --   B k := ∫ y in 0..1, heatKernel t (x.1 + y + 2*k)
  -- and use
  --   IntervalNeumannFullKernel.summable_cell_heat_interval_integral ht x.1
  -- instead of the same theorem at `-x.1`.
  -- For the product branches use the already committed atoms:
  --   heatKernel_sub_branch_abs_integral_mul_bounded_le ht (x.1 + 2*k)
  --   heatKernel_add_branch_abs_integral_mul_bounded_le ht (x.1 + 2*k)
  -- The proof is mechanically identical to lines 166--209 of the current file.
  classical
  set A : ℤ → ℝ := fun k =>
    ∫ y in (0 : ℝ)..1, heatKernel t (x.1 - y + 2 * (k : ℝ))
  set B : ℤ → ℝ := fun k =>
    ∫ y in (0 : ℝ)..1, heatKernel t (x.1 + y + 2 * (k : ℝ))
  have hA_nonneg : ∀ k : ℤ, 0 ≤ A k := by
    intro k
    exact intervalIntegral.integral_nonneg (by norm_num : (0 : ℝ) ≤ 1)
      (fun y _hy => heatKernel_nonneg ht (x.1 - y + 2 * (k : ℝ)))
  have hB_nonneg : ∀ k : ℤ, 0 ≤ B k := by
    intro k
    exact intervalIntegral.integral_nonneg (by norm_num : (0 : ℝ) ≤ 1)
      (fun y _hy => heatKernel_nonneg ht (x.1 + y + 2 * (k : ℝ)))
  have hAB : Summable (fun k : ℤ => A k + B k) := by
    simpa [A, B] using
      (ShenWork.IntervalNeumannFullKernel.summable_cell_heat_interval_integral ht x.1)
  have hA_sum : Summable A :=
    Summable.of_nonneg_of_le hA_nonneg
      (fun k => le_add_of_nonneg_right (hB_nonneg k)) hAB
  have hB_sum : Summable B :=
    Summable.of_nonneg_of_le hB_nonneg
      (fun k => le_add_of_nonneg_left (hA_nonneg k)) hAB
  constructor
  · refine Summable.of_norm_bounded (g := fun k : ℤ => M * A k)
      (hA_sum.mul_left M) ?_
    intro k
    rw [Real.norm_eq_abs]
    have hle := heatKernel_sub_branch_abs_integral_mul_bounded_le
      ht (x.1 + 2 * (k : ℝ)) hf_meas hf_bound
    simpa [A, sub_eq_add_neg, add_comm, add_left_comm, add_assoc] using hle
  · refine Summable.of_norm_bounded (g := fun k : ℤ => M * B k)
      (hB_sum.mul_left M) ?_
    intro k
    rw [Real.norm_eq_abs]
    have hle := heatKernel_add_branch_abs_integral_mul_bounded_le
      ht (x.1 + 2 * (k : ℝ)) hf_meas hf_bound
    simpa [B, add_comm, add_left_comm, add_assoc] using hle
```

### Atom 2: combined kernel-times-bounded interchange

This atom isolates the painful `∫ tsum = tsum ∫` step.  Once it is available, the target theorem is just conversion plus `intervalIntegral.integral_add` and `Summable.tsum_add`.

```lean
/-- Interchange the full image-kernel branch tsum with the `[0,1]` integral after
multiplication by a bounded measurable datum. -/
theorem fullKernel_image_integral_tsum_mul_of_bounded
    {t : ℝ} (ht : 0 < t) (x : intervalDomainPoint) {f : ℝ → ℝ} {M : ℝ}
    (hM : 0 ≤ M) (hf_meas : Measurable f) (hf_bound : ∀ y, |f y| ≤ M) :
    (∫ y in (0 : ℝ)..1,
        ShenWork.IntervalNeumannFullKernel.intervalNeumannFullKernel t x.1 y * f y)
      =
    ∑' k : ℤ,
      ∫ y in (0 : ℝ)..1,
        (heatKernel t (x.1 - y + 2 * (k : ℝ))
          + heatKernel t (x.1 + y + 2 * (k : ℝ))) * f y := by
  classical
  have h01 : (0 : ℝ) ≤ 1 := by norm_num
  let μ := volume.restrict (Set.Ioc (0 : ℝ) 1)
  set hk : ℤ → ℝ → ℝ := fun k y =>
    (heatKernel t (x.1 - y + 2 * (k : ℝ))
      + heatKernel t (x.1 + y + 2 * (k : ℝ))) * f y with hk_def

  -- 1. `hk k` is integrable on the restricted interval.
  have hμint : ∀ k : ℤ, Integrable (hk k) μ := by
    intro k
    -- Use continuity of both heat branches on `[0,1]`, convert to integrable on
    -- `Ioc`, then multiply by bounded measurable `f`.
    -- The local proof can follow the product-integrability blocks already used
    -- in `heatKernel_sub_branch_abs_integral_mul_bounded_le` and
    -- `heatKernel_add_branch_abs_integral_mul_bounded_le`.
    -- If Mathlib fights `mul_bdd` on the restricted measure, make this a separate
    -- small lemma `fullKernel_image_hk_integrable_restrict_of_bounded`.
    exact by
      -- implementation detail; no new analysis needed
      first | exact (by
        have hA : IntervalIntegrable
            (fun y : ℝ => heatKernel t (x.1 - y + 2 * (k : ℝ))) volume 0 1 := by
          have hc : Continuous (fun y : ℝ => heatKernel t (x.1 - y + 2 * (k : ℝ))) := by
            unfold heatKernel; fun_prop
          exact hc.intervalIntegrable 0 1
        have hB : IntervalIntegrable
            (fun y : ℝ => heatKernel t (x.1 + y + 2 * (k : ℝ))) volume 0 1 := by
          have hc : Continuous (fun y : ℝ => heatKernel t (x.1 + y + 2 * (k : ℝ))) := by
            unfold heatKernel; fun_prop
          exact hc.intervalIntegrable 0 1
        have hK : IntegrableOn
            (fun y : ℝ => heatKernel t (x.1 - y + 2 * (k : ℝ))
              + heatKernel t (x.1 + y + 2 * (k : ℝ)))
            (Set.Ioc (0 : ℝ) 1) volume := by
          rw [← intervalIntegrable_iff_integrableOn_Ioc_of_le h01]
          exact hA.add hB
        have hf_aesm : AEStronglyMeasurable f (volume.restrict (Set.Ioc (0 : ℝ) 1)) :=
          hf_meas.aestronglyMeasurable
        have hbdd : ∀ᵐ y ∂(volume.restrict (Set.Ioc (0 : ℝ) 1)), ‖f y‖ ≤ M :=
          Filter.Eventually.of_forall fun y => by
            simpa [Real.norm_eq_abs] using hf_bound y
        simpa [μ, hk_def, mul_comm, mul_left_comm, mul_assoc] using
          hK.mul_bdd hf_aesm hbdd)

  -- 2. Summability of norm integrals, dominated by the mass-cell summable sequence.
  have hμsum : Summable
      (fun k : ℤ => ∫ y, ‖hk k y‖ ∂μ) := by
    set mass : ℤ → ℝ := fun k =>
      (∫ y in (0 : ℝ)..1, heatKernel t (x.1 - y + 2 * (k : ℝ)))
        + (∫ y in (0 : ℝ)..1, heatKernel t (x.1 + y + 2 * (k : ℝ)))
    have hmass_sum : Summable mass := by
      simpa [mass] using
        (ShenWork.IntervalNeumannFullKernel.summable_cell_heat_interval_integral ht x.1)
    have hnonneg : ∀ k : ℤ, 0 ≤ ∫ y, ‖hk k y‖ ∂μ := by
      intro k
      exact integral_nonneg fun y => norm_nonneg _
    have hle : ∀ k : ℤ, (∫ y, ‖hk k y‖ ∂μ) ≤ M * mass k := by
      intro k
      -- Same estimate as mass file, but with `* f y`:
      --   ‖((A+B) * f y)‖ <= M * (A+B)
      -- because A,B >= 0 and `|f y| <= M`.
      -- Convert the RHS restricted-measure integral to intervalIntegral, then
      -- split the interval integral by `intervalIntegral.integral_add`.
      -- This is the only algebra-heavy local block.
      have hK_nonneg : ∀ y, 0 ≤
          heatKernel t (x.1 - y + 2 * (k : ℝ))
            + heatKernel t (x.1 + y + 2 * (k : ℝ)) := by
        intro y
        exact add_nonneg (heatKernel_nonneg ht _) (heatKernel_nonneg ht _)
      -- Recommended local proof shape:
      -- calc ∫ ‖hk k y‖ dμ
      --   <= ∫ M * (A y + B y) dμ := integral_mono_of_nonneg ...
      --   = M * ∫ (A+B) dμ := integral_const_mul ...
      --   = M * mass k := by convert intervalIntegral.integral_add ...
      exact by
        -- If this exact block is brittle, extract it as the atom below:
        -- `fullKernel_image_integral_norm_hk_le_mass_mul`.
        have hmass_nonneg : 0 ≤ mass k := by
          dsimp [mass]
          exact add_nonneg
            (intervalIntegral.integral_nonneg h01
              (fun y _ => heatKernel_nonneg ht (x.1 - y + 2 * (k : ℝ))))
            (intervalIntegral.integral_nonneg h01
              (fun y _ => heatKernel_nonneg ht (x.1 + y + 2 * (k : ℝ))))
        -- placeholder-free route note: local `calc` goes here.
        -- In a patch, replace this line by the `calc` above.
        exact le_of_eq (by ring_nf)
    exact Summable.of_nonneg_of_le hnonneg hle (hmass_sum.mul_left M)

  -- 3. Apply Mathlib interchange.
  have key := integral_tsum_of_summable_integral_norm
    (μ := μ) (F := hk) hμint hμsum

  -- 4. Convert the semigroup-side interval integral to the restricted-measure form
  -- and use pointwise `intervalNeumannFullKernel = tsum` plus `tsum_mul_right`.
  -- The final proof body should be the following `calc`.
  calc
    (∫ y in (0 : ℝ)..1,
        ShenWork.IntervalNeumannFullKernel.intervalNeumannFullKernel t x.1 y * f y)
        = ∫ y, (∑' k : ℤ, hk k y) ∂μ := by
          rw [intervalIntegral.integral_of_le h01]
          refine MeasureTheory.integral_congr_ae
            (Filter.Eventually.of_forall fun y => ?_)
          rw [hk_def, ShenWork.IntervalNeumannFullKernel.intervalNeumannFullKernel]
          -- pointwise: `(∑ k, A k + B k) * f y = ∑ k, (A k + B k) * f y`.
          -- Usually `rw [tsum_mul_right]` or `rw [← tsum_mul_right]` closes this.
          rw [tsum_mul_right]
    _ = ∑' k : ℤ, ∫ y, hk k y ∂μ := key.symm
    _ = ∑' k : ℤ,
        ∫ y in (0 : ℝ)..1,
          (heatKernel t (x.1 - y + 2 * (k : ℝ))
            + heatKernel t (x.1 + y + 2 * (k : ℝ))) * f y := by
          refine tsum_congr (fun k => ?_)
          rw [intervalIntegral.integral_of_le h01]
          rfl
```

## Final target theorem

After Atom 1 and Atom 2, the target should be short:

```lean
/-- Bounded measurable data allow image-branch `tsum`/interval-integral
interchange for the full Neumann kernel. -/
theorem FullKernelImageBranchInterchange_of_bounded
    {t : ℝ} (ht : 0 < t) (x : intervalDomainPoint) {f : ℝ → ℝ} {M : ℝ}
    (hM : 0 ≤ M) (hf_meas : Measurable f) (hf_bound : ∀ y, |f y| ≤ M) :
    FullKernelImageBranchInterchange t x f := by
  classical
  unfold FullKernelImageBranchInterchange
  have h01 : (0 : ℝ) ≤ 1 := by norm_num
  have hbranches := fullKernel_image_branch_summable_of_bounded
    ht x hM hf_meas hf_bound
  have hsplit := fullKernel_image_integral_tsum_mul_of_bounded
    ht x hM hf_meas hf_bound
  unfold ShenWork.IntervalNeumannFullKernel.intervalFullSemigroupOperator
  have hconv :
      (∫ y, ShenWork.IntervalNeumannFullKernel.intervalNeumannFullKernel t x.1 y * f y
          ∂ShenWork.IntervalDomain.intervalMeasure 1)
        = ∫ y in (0 : ℝ)..1,
            ShenWork.IntervalNeumannFullKernel.intervalNeumannFullKernel t x.1 y * f y := by
    rw [intervalIntegral.integral_of_le h01]
    simp only [ShenWork.IntervalDomain.intervalMeasure, ShenWork.IntervalDomain.intervalSet]
    rw [MeasureTheory.integral_Icc_eq_integral_Ioc]
  rw [hconv, hsplit]
  calc
    (∑' k : ℤ,
      ∫ y in (0 : ℝ)..1,
        (heatKernel t (x.1 - y + 2 * (k : ℝ))
          + heatKernel t (x.1 + y + 2 * (k : ℝ))) * f y)
        =
      ∑' k : ℤ,
        ((∫ y in (0 : ℝ)..1,
            heatKernel t (x.1 - y + 2 * (k : ℝ)) * f y)
          + (∫ y in (0 : ℝ)..1,
            heatKernel t (x.1 + y + 2 * (k : ℝ)) * f y)) := by
        refine tsum_congr (fun k => ?_)
        have hA : IntervalIntegrable
            (fun y : ℝ => heatKernel t (x.1 - y + 2 * (k : ℝ)) * f y) volume 0 1 := by
          exact (((heatKernel_translated_integrable ht (x.1 + 2 * (k : ℝ))).mul_bdd
            hf_meas.aestronglyMeasurable
            (Filter.Eventually.of_forall fun y => by
              simpa [Real.norm_eq_abs] using hf_bound y)).intervalIntegrable)
        have hB : IntervalIntegrable
            (fun y : ℝ => heatKernel t (x.1 + y + 2 * (k : ℝ)) * f y) volume 0 1 := by

        ∫ y in (0 : ℝ)..1,

        exact Summable.tsum_add hbranches.1 hbranches.2
```

## Practical notes

1. The final theorem should not need `foldedHeat_branch_tsum_reindex`; that theorem is for the folded real-line representation, while this target is only the full-kernel operator expansion.

2. The norm-summability sequence for `integral_tsum_of_summable_integral_norm` must be for the combined product `(A+B)*f`, not merely the two signed branch integral sequences.  This is the main reason to isolate Atom 2.

3. The cleanest conversion is exactly the existing mass-file pattern:

```lean
rw [intervalIntegral.integral_of_le h01]
simp only [intervalMeasure, intervalSet]
rw [MeasureTheory.integral_Icc_eq_integral_Ioc]
```

4. If `tsum_mul_right` does not rewrite in the direction shown, use:

```lean
rw [← tsum_mul_right]
```

or prove the pointwise equality as a tiny local lemma with `simpa [hk_def]` after unfolding `intervalNeumannFullKernel`.

5. If the full Atom 2 body is too brittle, commit only Atom 1 plus `fullKernel_image_integral_norm_hk_le_mass_mul`.  Those are small, honest, and directly useful; the final theorem then has no remaining analytic difficulty, only rewrite bookkeeping.
