## 2. Do existing Young estimates produce these fields?

```lean

  (1 / 2) * lapL2sq u τ + (1 / 2) * H1PhysicalTaxisPartSq p u v τ

‖H1PhysicalReactX p u τ‖ ≤
  (1 / 2) * lapL2sq u τ + (1 / 2) * H1PhysicalReactPartSq p u τ
```

Those estimates feed the zero-window majorant/integrability route via:

```lean

H1PhysicalRHSYoungScalarZeroMajorantsBefore_of_spatialYoungData
```

So Task103’s route and the sqrt-bound route are parallel: Task103 closes the zero-window majorant side, while `H1PhysicalRHSSqrtBoundsBefore` remains a separate pointwise physical-estimate frontier.

## 4. Exact missing pointwise inequalities

The honest missing analytic estimates are these three pointwise bounds, for every `τ ∈ Set.Ioo 0 T`:

### Taxis

```lean
‖H1PhysicalTaxisX p u v τ‖ ≤
  V₁ * (H1lapL2Norm u τ * H1gradL2Norm u τ)
```

A natural proof would use Cauchy-Schwarz on

```lean
H1PhysicalTaxisX p u v τ =
  -(∫ x in 0..1, liftDeriv2 u τ x *
    H1PhysicalChemTaxisPart p u v τ x)
```

plus an L² factor estimate such as:

```lean
(∫ x in 0..1, (H1PhysicalChemTaxisPart p u v τ x)^2)^(1/2)
  ≤ V₁ * H1gradL2Norm u τ
```

Analytically, this needs a bound like `|v_x / (1+v)^β| ≤ V₁` on the interval, or an equivalent L∞/resolver gradient estimate, with the right measurability/integrability support.

### Uvxx / denominator derivative

```lean
‖H1PhysicalUvxxX p u v τ‖ ≤
  M * (V₂ * H1lapL2Norm u τ)
```

A natural proof would use Cauchy-Schwarz plus an L² or sup bound for

```lean
H1PhysicalChemUvxxPart p u v τ x
```

of the shape

```lean
(∫ x in 0..1, (H1PhysicalChemUvxxPart p u v τ x)^2)^(1/2)
  ≤ M * V₂
```

or an equivalent pointwise/sup estimate. Analytically this likely needs an L∞ bound for `u` by `M` and resolver/reaction bounds for the `v_xx` representative and denominator terms bundled into `V₂`.

### Reaction

```lean
H1PhysicalReactX p u τ ≤ L * (H1gradL2Norm u τ)^2
```

This is not the same as the Young estimate

```lean
‖H1PhysicalReactX p u τ‖ ≤
  (1 / 2) * lapL2sq u τ + (1 / 2) * H1PhysicalReactPartSq p u τ
```

The intended bound is usually an integration-by-parts/reaction-derivative estimate:

```lean
H1PhysicalReactX p u τ
  = ∫ x in 0..1, f'(u τ x) * (u_x τ x)^2
  ≤ L * ∫ x in 0..1, (u_x τ x)^2
  = L * (H1gradL2Norm u τ)^2
```

Thus it needs the exact reaction IBP identity and a pointwise derivative bound for the logistic reaction on the relevant `u` range.
