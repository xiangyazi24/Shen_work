# Verdict

The entropy half is formalizable from the current interval classical-solution API, but it is **not yet a one-line reuse** of `LyapunovFunction.lean`.

1. The exact relative-entropy identity can be built with one new entropy-specific time-Leibniz lemma, one new weighted-PDE-under-the-integral lemma, and two already-landed spatial integration-by-parts engines. There is no hidden positive-time regularity gap: `IsPaper2ClassicalSolution intervalDomain p T u v` already carries the closed-space `C²` slices, genuine time derivatives, joint continuity of the relevant derivative fields, strict positivity of `u`, nonnegativity of `v`, and Neumann data needed at every `t ∈ (0,T)`.
2. The desired resolver estimate
   \[
   \int_0^1 |v_x|^2\le C\int_0^1 |u_x|^2
   \]
   is **not currently exported by a single repository theorem**. The resolver representation and gradient realization are present, and `static_v_grad_L2_le_Eu` is nearby, but that theorem controls a two-solution `v`-gradient difference by an `L²` difference of the two `u` fields. The missing leaf is a weighted cosine/sine Parseval multiplier estimate. Its sharp first-mode constant is straightforward.
3. Full abstract LaSalle machinery is unnecessary. The minimal closing argument is sequential: integrated entropy dissipation gives a sequence `tₙ → ∞` with `D(tₙ) → 0`; time-translate precompactness gives a convergent subsequence; lower semicontinuity/continuity of `D` and zero-dissipation rigidity identify its limit with the equilibrium; hence one snapshot enters the local-stability basin. However, **`D(tₙ) → 0` by itself does not imply basin entry**. One needs either a direct coercivity estimate from `D` to the basin norm, or compactness + dissipation closure + rigidity.

There is also a coded-interface issue: the current `IntervalDomainStabilityChain.lean` wrappers ask for a derivative inequality on `chemotaxisThetaDissipation` itself. That is stronger than the paper-style route. The natural replacement input is an integrated relative-entropy dissipation package, followed by the sequential basin-entry lemma below.

# 0. Two repository facts to keep straight

## 0.1 The relative entropy here is not the existing `chemotaxisEntropyDensity`

The requested density is

\[
h_{u_*}(r)=r\log(r/u_*)-(r-u_*),
\qquad
H_{u_*}(u(t))=\int_0^1 h_{u_*}(u(t,x))\,dx.
\]

`ShenWork/Paper3/LyapunovFunction.lean` currently develops the paper's scalar `chemotaxisEntropyDensity m uStar`, plus generic derivative/dissipation combinators. Do not identify the two definitions by `rfl`. Introduce a separate relative-entropy definition and use the existing file only for the scalar inequalities and downstream Lyapunov patterns.

## 0.2 The compiled interval resolver has denominator `μ + λₖ`

For the current repository's interval system,

```text
v̂ₖ = ν · (u^γ)̂ₖ / (μ + λₖ),   λₖ = (kπ)².
```

Here `p.ν` is the elliptic source coefficient. The `u`-equation's diffusion coefficient is normalized to `1`. Consequently all constants below use `p.μ + λ₁`, not `p.μ + p.ν * λ₁`.

## 1.9 From identity to a coercive dissipation inequality

For `v≥0` and `β≥0`, `(1+v)^(-β) ≤ 1`. With `0<u≤M`, apply your weighted Young inequality pointwise with weight `u`:

\[
|\chi_0|\frac{|u_xv_x|}{(1+v)^\beta}
\le \varepsilon\frac{u_x^2}{u}
 +\frac{\chi_0^2}{4\varepsilon}
   u\frac{v_x^2}{(1+v)^{2\beta}}.
\]

Hence

\[
|\chi_0|\,|\mathrm{Cross}|
\le \varepsilon I_u+
\frac{\chi_0^2 M}{4\varepsilon}\int v_x^2,
\qquad
I_u=\int u_x^2/u.
\]

If the resolver estimate below gives

\[
\int v_x^2\le C_R\int u_x^2
\le C_R M I_u,
\]

then

\[
H'(t)\le-
\left(1-\varepsilon-
\frac{\chi_0^2M^2C_R}{4\varepsilon}\right)I_u
-bR_\alpha(u).
\]

This is a clean formal target:

```lean
theorem intervalDomain_relativeEntropy_deriv_le_neg_dissipation
    ...
    (hM : ∀ x, u t x ≤ M)
    (hresolver : ∫ x in 0..1, (deriv (intervalDomainLift (v t)) x)^2
      ≤ CR * ∫ x in 0..1, (deriv (intervalDomainLift (u t)) x)^2)
```

This Young estimate is intentionally coarse. It may not reproduce the paper's sharp threshold for Theorems 2.4–2.5; sharper threshold algebra can be substituted after the identity is established.

# 2. Resolver modal-Poincaré estimate

## 2.1 Exact coefficient calculation

Let

```text
λₖ = (kπ)²,
f = u^γ,
v̂ₖ = p.ν * f̂ₖ / (p.μ + λₖ).
```

For every `k ≥ 1`, `λ₁ ≤ λₖ`, so

\[
\lambda_k|\hat v_k|^2
=\lambda_k\left(\frac{\nu}{\mu+\lambda_k}\right)^2|\hat f_k|^2
\le
\left(\frac{\nu}{\mu+\lambda_1}\right)^2
\lambda_k|\hat f_k|^2.
\]

The zero mode contributes nothing because `λ₀=0`. With the repository's normalized positive-mode cosine coefficients, both gradient Parseval identities carry the same factor `1/2`, so it cancels. Thus

\[
\|v_x\|_2^2
\le
\left(\frac{p.\nu}{p.\mu+\pi^2}\right)^2
\|(u^\gamma)_x\|_2^2.
\]

If `1 ≤ γ` and `0≤u≤M`,

\[
|(u^\gamma)_x|
=\gamma u^{\gamma-1}|u_x|
\le\gamma M^{\gamma-1}|u_x|,
\]

so the final constant is

\[
C_R=
\left(
\frac{p.\nu\,p.\gamma\,M^{p.\gamma-1}}
     {p.\mu+\pi^2}
\right)^2.
\]

For `0<γ<1`, an upper bound alone is insufficient because `u^{γ-1}` is singular at zero. On an eventual persistence slab `δ≤u≤M`, use

\[
C_R=
\left(
\frac{p.\nu\,p.\gamma\,\delta^{p.\gamma-1}}
     {p.\mu+\pi^2}
\right)^2.
\]

Therefore a **uniform** L7 constant in the sublinear-`γ` regime depends on the Paper 3 persistence floor. This is not strict-positivity circularity; it is the natural dependency `P3.2 persistence → P3.6/7 entropy coercivity`.

# 3. L8: direct sequential LaSalle / basin entry

## 3.1 Integrated entropy inequality

The best consumer interface is integrated, not another derivative structure. Export:

```lean
structure IntervalRelativeEntropyDissipationData
    (uStar : ℝ)
    (u : ℝ → intervalDomain.Point → ℝ) where
  H : ℝ → ℝ
  D : ℝ → ℝ
  H_eq : ∀ t, H t = intervalRelativeEntropy uStar u t
  H_nonneg : ∀ t, 0 ≤ H t
  D_nonneg : ∀ t, 0 ≤ D t
  rate : ℝ
  rate_pos : 0 < rate
  integrated :
    ∀ s t, 0 < s → s ≤ t →
      H t + rate * ∫ r in s..t, D r ≤ H s
```

Here `D` can be the Fisher term plus the reaction entropy dissipation, with whichever positive coefficients the Young/resolver step leaves.

This interface follows from the exact identity by the usual interval FTC. The repository already repeatedly uses:

```lean
intervalIntegral.integral_eq_sub_of_hasDeriv_right_of_le
```

for this purpose.

The immediate consequence is a uniform tail bound:

```lean
∀ T, s ≤ T → ∫ r in s..T, D r ≤ H s / rate.
```

## 3.2 Minimal scalar sequence lemma

No abstract LaSalle theorem is needed to obtain a zero-dissipation sequence. The exact reusable scalar target is:

```lean
theorem exists_dissipation_zero_sequence_of_tail_bound
    {D : ℝ → ℝ} {s C : ℝ}
    (hC : 0 ≤ C)
    (hDcont : ContinuousOn D (Set.Ici s))
    (hDnonneg : ∀ t, s ≤ t → 0 ≤ D t)
    (hbound : ∀ T, s ≤ T → ∫ r in s..T, D r ≤ C) :
    ∃ times : ℕ → ℝ,
      Tendsto times atTop atTop ∧
      Tendsto (fun n => D (times n)) atTop (𝓝 0) := by
  ...
```

A direct proof chooses, for each `n`, a sufficiently long interval beyond time `n`. If `D` were everywhere larger than `1/(n+1)` on that interval, the integral would exceed `C`, contradicting `hbound`. This gives `times n ≥ n` and `D(times n) ≤ 1/(n+1)`.

This is the precise replacement for the first half of LaSalle.

## 3.3 Why the sequence alone is not enough

The implication

```text
D (times n) → 0  ⇒  u (times n) → uStar in sup norm
```

is not formal without another hypothesis. One needs one of the following.

### Route A: direct coercivity

Prove a quantitative basin coercivity lemma:

```lean
theorem dissipation_small_implies_sup_close
    ... :
    ∀ eps > 0, ∃ eta > 0, ∀ t,
      D t < eta →
        intervalDomain.supNorm (fun x => u t x - uStar) < eps := by
  ...
```

Then the zero-dissipation sequence immediately enters the L5 basin.

### Route B: compactness + closure + rigidity

This is closer to the paper and to `P3.5`.

The repository has:

```lean
TimeTranslateCompactnessConclusion intervalDomain p K u v
```

which gives, from `times → ∞`, a subsequence and an entire limit under the abstract relation `K.locallyConverges`.

However, `CompactnessData.locallyConverges` is an uninterpreted relation. To use it in the entropy argument, add two compatibility fields:

```lean
def LocallyConvergesAtZeroInSup
    (K : CompactnessData intervalDomain) : Prop :=
  ∀ F f,
    K.locallyConverges F f →
      Tendsto
        (fun n => intervalDomain.supNorm
          (fun x => F n 0 x - f 0 x))
        atTop (𝓝 0)

def DissipationLowerSemicontinuousUnderLocallyConverges
    (K : CompactnessData intervalDomain)
    (Dslice : (intervalDomain.Point → ℝ) → ℝ) : Prop :=
  ∀ F f,
    K.locallyConverges F f →
      Dslice (f 0) ≤
```
