# Positive-time bounded source coefficients

Write the Picard iterates as

`u^0(t) = S(t) u0`,

`u^(n+1)(t) = S(t) u0 + integral_0^t S(t-r) B(u^n(r)) dr`,

where `S(t)` is the Neumann heat semigroup on `[0,1]` and `B` is the truncated B-form source.  The data `DT` gives a uniform ball bound `||u^n(t)||_infty <= M` on `[0,T]` and uniform convergence of `u^n` to the Picard limit `u`.

The theorem only needs a uniform-in-`k` bound at a fixed positive time `s`.  A classical `C^1` slice is more than enough, but the coefficient estimate already follows from the weaker statement `u(s) in W^{1,infty}`.  Uniform convergence of smooth iterates alone does not prove this; the missing ingredient is a uniform positive-time gradient bound.

## 1. Heat smoothing for the zero iterate

For the Neumann heat semigroup,

`||partial_x S(t) f||_infty <= C_heat t^(-1/2) ||f||_infty` for `t > 0`.

Thus `u^0(t)=S(t)u0` is smooth for every positive `t`, and for every `tau > 0`,

`sup_{t in [tau,T]} ||partial_x u^0(t)||_infty <= C_heat tau^(-1/2) ||u0||_infty`.

In the project, the role of `||u0||_infty` is supplied by the initial bound stored in `DT`.

## 2. One-step source control from a gradient bound

Assume a slice `v` satisfies `||v||_infty <= M` and `||v_x||_infty <= G`.  Let

`R = resolver(p,v)` and `F = positivePart(v) * R_x / (1+R)^beta`.

The elliptic resolver estimates give constants depending only on `p` and `M`:

`||R_x||_infty <= R1(M)`, `||R_xx||_infty <= R2(M)`, and denominator bounds for `(1+R)^(-beta)` and its derivative factor.

Because `positivePart` is 1-Lipschitz, `|(positivePart(v))'| <= |v'|` a.e.  Hence

`F' = positivePart(v)' * R_x * (1+R)^(-beta) + positivePart(v) * R_xx * (1+R)^(-beta) - beta * positivePart(v) * R_x^2 * (1+R)^(-beta-1)`

a.e., and therefore

`||F'||_1 <= A0(p,M) + A1(p,M) * G`.

The constants are explicit from the preceding line; schematically,

`A1` is the bound for `R_x * (1+R)^(-beta)`,

and `A0` contains the `R_xx` term and the derivative of the denominator.

The Neumann boundary condition for the resolver gives `R_x(0)=R_x(1)=0`, so `F(0)=F(1)=0`.  Integration by parts gives, for `k >= 1`,

`|k*pi * intervalSineInner(F,k)| <= 4|F(0)| + 4|F(1)| + 2 integral |F'| <= 2 ||F'||_1`.

For `k=0`, the chem-div coefficient is zero because of the leading factor `k*pi`.  Therefore

`|truncatedChemDivSourceCoeff p v k| <= C_chem0(p,M) + C_chem1(p,M) * G`

uniformly in `k`.

The logistic part is bounded directly from the ball bound and the truncation:

`|truncatedLogisticSourceCoeff p v k| <= 2 C_log(p,M)`.

Thus the B-form source coefficients satisfy the affine envelope

`|truncatedBFormSourceCoeff p v k| <= B0(p,M) + B1(p,M) * G`

uniformly in `k`.

This is the key estimate for the induction: gradient bound in, source envelope out.

## 3. Feeding the source envelope through the coefficient ladder

The ladder lemmas (`WindowCoefficientEnvelope`, `WindowSourceEnvelope`, and `eigenvalue_weighted_summable_of_pass4`) should be used as the abstract estimate

`||partial_x integral_a^t S(t-r) f(r) dr||_infty <= integral_a^t q(t-r) E(r) dr`,

where `E(r)` is the source envelope and `q` is a locally integrable heat/laddder kernel.  In the ordinary heat-kernel estimate, `q(h)=C_q h^(-1/2)`.

Restart the linear equation at an arbitrary time `a`:

`u^(n+1)(t) = S(t-a) u^(n+1)(a) + integral_a^t S(t-r) B(u^n(r)) dr`.

Since `||u^(n+1)(a)||_infty <= M`, if `g_n(t)=||partial_x u^n(t)||_infty`, then

`g_(n+1)(t) <= C_heat M (t-a)^(-1/2) + integral_a^t q(t-r) * (B0 + B1 g_n(r)) dr`.

This is the Volterra inequality.

## 4. Uniform gradient bound and the small-time condition

Use the weighted local norm on `[a,a+delta]`:

`H_n(a,delta) = sup_{a<t<=a+delta} (t-a)^(1/2) g_n(t)`.

Define

`Theta_q(delta) = sup_{0<h<=delta} h^(1/2) integral_0^h q(h-r) r^(-1/2) dr`,

`Q_q(delta) = sup_{0<h<=delta} h^(1/2) integral_0^h q(r) dr`.

Multiplying the Volterra inequality by `(t-a)^(1/2)` gives

`H_(n+1)(a,delta) <= C_heat M + B0 Q_q(delta) + B1 Theta_q(delta) H_n(a,delta)`.

Therefore the small-time condition for a uniform gradient bound is

`B1 * Theta_q(delta) < 1`.

Under this condition,

`sup_n H_n(a,delta) <= max(H_0(a,delta), (C_heat M + B0 Q_q(delta))/(1 - B1 Theta_q(delta)))`,

with a bound independent of `n` and of the restart time `a`.

For the model heat kernel `q(h)=C_q h^(-1/2)`,

`Theta_q(delta) = C_q * pi * sqrt(delta)` and `Q_q(delta) = 2 C_q delta`.

So the concrete condition is

`C_q * B1 * pi * sqrt(delta) < 1`.

If one already has a uniform gradient bound at the left endpoint and works in an unweighted gradient norm, the corresponding condition is

`B1 * integral_0^delta q(r) dr < 1`,

which becomes `2 C_q B1 sqrt(delta) < 1` for `q(h)=C_q h^(-1/2)`.  The weighted condition is usually the cleaner one because it only needs the `L^infty` ball bound at the restart time.

For a compact positive-time window `[tau,T]`, choose `h0 = min(tau/2, delta/2)`.  For each `t in [tau,T]`, restart at `a=t-h0`.  Then the weighted estimate gives

`g_n(t) <= h0^(-1/2) sup_m H_m(a,delta)`.

Hence

`sup_n sup_{t in [tau,T]} ||partial_x u^n(t)||_infty <= G_{tau,T} < infinity`.

Equivalently, a global Volterra-resolvent proof is possible: iterating `g <= h + B1 q*g` gives a convergent convolution series.  The small-time proof is usually easier to formalize.

## 5. Passing the gradient bound to the limit

Since `u^n -> u` uniformly and the slices `u^n(t)` are uniformly Lipschitz on `[tau,T]`, for fixed `t in [tau,T]` and all `x,y`,

`|u(t,x)-u(t,y)| <= G_{tau,T} |x-y|`.

Thus `u(t,.) in W^{1,infty}` and `||u_x(t)||_infty <= G_{tau,T}`.

If the implementation needs a classical `C^1` slice, apply the same short-time estimate to Picard differences.  If the source map satisfies

`E(B(v)-B(w)) <= L0 ||v-w||_infty + L1 ||v_x-w_x||_infty`,

then a sufficient derivative-level contraction condition is

`L1 Theta_q(delta) + L0 Q_q(delta) < 1`.

For `q(h)=C_q h^(-1/2)`, this is

`C_q * (L1*pi*sqrt(delta) + 2*L0*delta) < 1`.

Together with the existing `C^0` contraction data in `DT`, this gives convergence in a positive-time weighted `C^1` norm, hence uniform convergence of derivatives away from the left endpoint.  But for the coefficient bound, the inherited `W^{1,infty}` regularity is enough.

## 6. Final coefficient bound at time s

Take `tau=s/2`.  The limit satisfies

`||u(s)||_infty <= M` and `||u_x(s)||_infty <= G_{s/2,T}`.

For

`F = positivePart(u(s)) * resolverGrad(p,u(s)) / (1+R(p,u(s)))^beta`,

the estimate from Step 2 gives

`F in W^{1,1}` and `||F'||_1 <= A0(p,M) + A1(p,M) G_{s/2,T}`.

The resolver Neumann condition gives `F(0)=F(1)=0`, so for every `k`,

`|truncatedChemDivSourceCoeff p u s k| <= 2 ||F'||_1 <= C_chem(p,M,G_{s/2,T})`.

The logistic coefficient satisfies

`|truncatedLogisticSourceCoeff p u s k| <= 2 C_log(p,M)`.

Therefore

`|truncatedBFormSourceCoeff p u s k| <= 2 C_log(p,M) + |p.chi0| C_chem(p,M,G_{s/2,T})`.

Choose

`C = 2 C_log(p,M) + |p.chi0| C_chem(p,M,G_{s/2,T})`.

Then `0 <= C` and the bound is uniform in `k`, which is exactly the desired positive-time boundedness of the source coefficients.
