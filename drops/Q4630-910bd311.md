# 3. Common lower-order producer: uniform mass

For every classical solution and every finite or global horizon, integrate the first PDE.  Neumann boundary conditions remove diffusion and flux:

```text
M₁'(t) = a M₁(t) - b ∫ u^(1+α).
```

Because the interval has volume one and `u ≥ 0`, Jensen gives

```text
∫ u^(1+α) ≥ M₁(t)^(1+α).
```

Hence

```text
M₁'(t) ≤ a M₁(t) - b M₁(t)^(1+α).
```

With `a,b,α>0`, a horizon-independent bound is

```text
M₁(t) ≤ max M₁(0) ((a/b)^(1/α)).
```

Implementation target:

```lean
theorem intervalDomainM_mass_uniform
    {p : CM2Params} {T : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomainM p T u v)
    (hb : 0 < p.b)
    (htrace : InitialTrace intervalDomainM u₀ u) :
    ∃ M1 ≥ 0, ∀ t, 0 < t → t < T →
      intervalDomainM.integral (u t) ≤ M1
```

The returned `M1` must depend only on `p` and `u₀`, never on `T`.

This mass estimate is not cosmetic.  On the interval it can replace the paper's critical seed whenever `α < 2`, because the fixed-target GN condition is only `α < 2*p0`, and one may take `p0=1`.

# 4. Fixed-target GN absorption: the small replacement for Corollary 2.1

Suppose a seed exponent `p0>0` and constant `M0` satisfy

```text
∫ u(t)^p0 ≤ M0
```

uniformly.  Fix the single endpoint exponent `P`.  Put

```text
w = u^(P/2),
r = 2(P+α)/P,
s = 2p0/P.
```

Then

```text
||w||_r^r = ∫ u^(P+α),
||w||_s^s = ∫ u^p0.
```

The one-dimensional GN exponent is

```text
a*r = 2(P + α - p0) / (P + p0).
```

Therefore

```text
a*r < 2  ↔  α < 2*p0.
```

Under that strict inequality, Young gives, for every `A>0` and `δ>0`,

```text
A ∫ u^(P+α)
  ≤ δ ∫ |∂x(u^(P/2))|² + K(A,δ,P,p0,α,M0)
```

with `K` independent of time and horizon.  Since

```text
∫ |∂x(u^(P/2))|²
  = (P²/4) ∫ u^(P-2)|u_x|²,
```

this is exactly the absorption needed by the `P`-energy identity.

Implementation-ready theorem:

```lean
theorem intervalDomainM_target_highMoment_absorb_of_seed
    {p : CM2Params} {T P p0 M0 A δ : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomainM p T u v)
    (hP : 2 < P)
    (hp0 : 0 < p0)
    (hsubcritical : p.α < 2 * p0)
    (hseed : ∀ t, 0 < t → t < T →
      intervalDomainM.integral (fun x => (u t x)^p0) ≤ M0)
    (hA : 0 ≤ A) (hδ : 0 < δ) :
    ∃ K ≥ 0, ∀ t, 0 < t → t < T →
      A * intervalDomainM.integral (fun x => (u t x)^(P+p.α)) ≤
        δ * intervalDomainM.integral
          (fun x => (u t x)^(P-2) *
            (intervalDomainM.gradNorm (u t) x)^2) + K
```

The theorem should be proved directly on `[0,1]`; do not require `1<p0`.  Requiring `p0>1` is useful in the general-domain Lemma 2.6 interface, but it discards the interval-specific mass shortcut.

Once this one theorem exists, critical cases only need a target-specific cross estimate at `P`; no generic all-`p` ladder is necessary.

Assume

With the actual interval elliptic constant, define

Proof split:

- if `α≤2`, then `q⋆=1`, `F₃(1)=0`, and continuity gives a nearby `p0>1`;

No existential “seed package” should remain after this theorem; this is a scalar real-analysis producer.

## 7.2 Seed differential inequality

At the selected `p0`, prove

```text
(1/p0) Y_p0'
 + (p0-1) G_p0
 + (b-F₃(p0)) Z_p0
 ≤ a Y_p0,
```

where

```text
G_q = ∫u^(q-2)|u_x|²,
Z_q = ∫u^(q+α).
```

Since `b-F₃(p0)>0`, scalar Young gives

```text
(a+1)Y_p0
 ≤ (b-F₃(p0))/2 * Z_p0 + K0,
```

and hence

```text
(1/p0)Y_p0' + Y_p0 ≤ D0.
```

This produces one horizon-independent seed:

```lean
LpPowerBoundedBefore intervalDomainM p0 T u.
```

## 7.3 Target `P⋆` estimate

At the final exponent `P`, the same elliptic multiplier identity gives a fixed coefficient

```text
χ₀(P-1) * SignedCross_P ≤ C₃(P) * Z_P.
```

No coefficient gap `b>C₃(P)` is required.  Apply


with `p0>α/2` to absorb `C₃(P)Z_P` into a small part of `G_P`.  Retain a positive part of the logistic sink, and absorb `(a+1)Y_P` into it.  The result is

```text

```

and therefore the common endpoint `L^P → L∞` applies.

This is exactly the content needed from Corollary 2.1, but only at one target exponent.

The paper's seed is supposed to be chosen immediately to the right of


But Proposition 2.2 is used at


and requires `s(q)>1`.  In case (iv),

```text

↔ q+2m+γ-2 > γ
```

Therefore the right-neighborhood argument at `q⋆` is available only if


This condition is absent from `StrongLogisticCondition`.


which, under the fourth equality, is exactly `q⋆>2-2m`.

```text

positivePart(α-2)=0,
```

The current fourth disjunct regards the `χ₀` smallness condition as vacuous, even for arbitrarily large positive `χ₀`.  But the Section 5.4 seed calculation is not valid until `q>3/2`, where its coefficient is proportional to `χ₀²`; the printed hypothesis supplies no reason for it to be below `b`.

## 8.2 Interval-specific partial repair

If

```text
α < 2,
```

then the uniform mass bound is already a sufficient fixed-target seed: take `p0=1`, and `α<2*p0`.  One can skip the Section 5.4 low-seed calculation entirely and use the target cross estimate at `P⋆` plus the fixed-target GN absorption.

This closes the whole `α<2` part of the vacuous positive-part branch on `[0,1]`.

Accordingly, the remaining uncovered wedge is

```text

max 1 (α/2) ≤ 2-2m.
```

Before claiming the full fourth alternative, the project needs one of the following genuine mathematical producers:

1. an extension/replacement of the weighted elliptic estimate valid in the required subunit exponent range;
2. a different interval-specific seed estimate for this wedge;

An assumption named `strongBootstrap` is not an acceptable substitute.

# 9. Global constants versus finite-horizon constants

Every energy producer should return a damping inequality with constants independent of `T`:

```lean
∃ D ≥ 0, ∀ t, 0<t → t<T →
  (1/P)Y_P'(t) + Y_P(t) ≤ D.
```

The ODE comparison then gives one explicit bound such as

```text
Y_P(t) ≤ max(Y_P(0+), P*D),
```

where the initial term is controlled by the continuous bounded datum and the initial trace.  The same bound works on every restriction of a global solution.

For critical cases, uniformity requires all of the following to be parameter/datum dependent only:

- mass constant `M1`;
- chosen scalar seed exponent `p0`;
- seed coefficient gap `b-F_i(p0)`;
- elliptic constants `Mstar` and `K`;
- GN/Young remainders at `p0` and `P⋆`;
- the endpoint restart window and resulting sup-norm constant.

The following shape is insufficient for the global branch:

```lean
∀ T>0, ∃ C_T, ∀ t<T, Y_P(t)≤C_T.
```

The needed shape is:

```lean
∃ C, ∀ t>0, Y_P(t)≤C.
```

The latest general-`m` restart file already has the correct consumer

```lean
boundedGlobal_of_lp_restarted_affine_general.
```

# 10. Local existence, uniqueness, and maximal continuation

## 10.1 Existing all-exponent work is a pattern, not yet the general-`m` close

`IntervalDomainLocalExistenceAllExponents.lean` proves positive-strip local existence for all positive powers, and `IntervalDomainTheorem12PositiveCriticalAllExponents.lean` proves overlap/gluing/floor-persistence machinery.  But the public classical-solution conclusions there are still typed on legacy `intervalDomain`, and the completed global branch is specialized to `m=1`.

For Theorem 1.3 the following genuine producers must be ported/proved on `intervalDomainM`:

```lean
intervalDomainM_localExistence_paperPositive_allExponents
intervalDomainM_thresholdLocalExistence_positiveStrip_allExponents
intervalDomainM_overlapUnique_on_positive_strips
intervalDomainM_maximalContinuation_exists
```

The scalar fixed-point estimates are available: on a strip `c≤u≤M`, every positive power is Lipschitz with

```text
L_q(c,M) = q * max(c^(q-1), M^(q-1)).
```

What must be checked is the actual general-`m` B-form PDE wiring, not merely reused by namespace coincidence.

## 10.2 Correct `m<1` conclusion

A finite `Paper2MaximalContinuation` branch carries

```lean
FiniteHorizonAlternative D Tmax u,
```

namely:

```text
upper blow-up  OR  loss of the positive floor.
```

When `m≥1`, it additionally carries

```lean
MGeOneFiniteHorizonAlternative,
```

so a finite maximal horizon forces upper blow-up.

The strong-logistic finite-`P` route gives

```lean
IsPaper2BoundedBefore intervalDomainM Tmax u,
```

which rules out upper blow-up.  Therefore:

- if `m≥1`, a finite maximal branch is impossible; the maximal solution is global;
- if `0<m<1`, boundedness only removes the upper-blow-up disjunct.  A finite maximal branch may remain through floor loss.

The exact finite-branch conclusion for `m<1` is

```text
∀ δ>0, ∃ t x,
  0<t ∧ t<Tmax ∧ x∈inside ∧ u(t,x)<δ.
```

Do not state that boundedness on `(0,Tmax)` gives a positive lower bound up to `Tmax`.  On each compact subhorizon positivity and compactness give a local floor, but the floor may tend to zero as `t↑Tmax`; when `m<1`, the derivative of `u^m` is singular there and the positive-strip restart time may collapse.

A faithful maximal theorem should have the shape:

```lean
theorem Theorem_1_3_intervalDomainM_maximal
    (p : CM2Params)
    (hN : p.N = 1)
    (ha : 0 < p.a) (hb : 0 < p.b)
    (hstrong : ActualStrongLogisticConditionInterval p) :
    ∀ u₀,
      PaperPositiveInitialDatum intervalDomainM u₀ →
      Nonempty (Paper2MaximalContinuation intervalDomainM p u₀) ∧
      ∀ branch : Paper2MaximalContinuation intervalDomainM p u₀,
        branch.IsBounded ∧
        (1 ≤ p.m → branch.IsGlobal) ∧
        (p.m < 1 →
          branch.IsGlobal ∨ branch.HasFiniteFloorLoss)
```

Here `ActualStrongLogisticConditionInterval` should be a direct definition using the realized `K`, not an assumption package.  `HasFiniteFloorLoss` should be a transparent predicate obtained by pattern matching on the finite constructor and spelling out the quantifiers above.

# 12. False shortcuts to reject

1. **“The unconditional `rho=2γ` estimate plus one strong-logistic seed handles all four cases.”** False.  The critical thresholds are calibrated to `rho=α`, and can supply only a seed just above `α/2`; they need not supply a seed above `γ`.
2. **“`intervalDomain_crossDiffusionBootstrapEstimate_of_classical` is a general-`m` theorem.”** False.  It controls the legacy affine cross term.
3. **“The affine restarted endpoint only needs `P>max(1,γ)`, so it handles general `m`.”** False.  General `m` needs `Q=P/m>1`, hence `P>m`.
4. **“All finite exponents are needed in cases (i),(ii).”** False.  One selected `P⋆` suffices, and the latest repo already implements this.
5. **“A critical seed alone closes boundedness.”** False.  One still needs either generic Corollary 2.1 or the smaller fixed-target GN producer.
6. **“Per-horizon constants imply a uniform global bound.”** False.
7. **“Boundedness implies globality for `m<1`.”** False; the floor-loss branch remains.
8. **“The fourth paper threshold automatically makes its elliptic exponent admissible.”** False for the current all-positive-parameter encoding.
9. **“An arbitrary nonnegative `C.K` is the paper constant.”** False.
10. **“The Theorem 1.2 `m=1` all-exponent global theorem can simply be reused.”** False; its patterns are reusable, but its operator and exponent specialization are not the general-`m` theorem.

# 13. Bottom line

The minimal faithful architecture is not a universal bootstrap package.  It is a case split with one common endpoint:

```text
(i)   direct damping at P⋆
      -> uniform L^P⋆
      -> general-m restarted endpoint

(ii)  direct weighted damping at P⋆
      -> uniform L^P⋆
      -> general-m restarted endpoint

(iii) threshold -> one low seed p0>max(1,α/2)
      -> one target-P⋆ GN absorption
      -> uniform L^P⋆
      -> general-m restarted endpoint

(iv)  α<2: mass seed p0=1
      -> one target-P⋆ GN absorption
      -> endpoint

      otherwise, where q⋆>2-2m:
      threshold -> valid low seed
      -> target-P⋆ GN
      -> endpoint

      remaining encoded wedge:
      NOT CLOSED by the paper proof or current repo.
```

For a finite maximal branch this gives boundedness on the entire open maximal interval.  It gives globality only when `m≥1`; for `m<1`, a finite horizon can survive only through loss of the positive floor.
