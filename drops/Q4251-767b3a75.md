# Q4251 — 1D gauge transform for bounded drift strict positivity

## Verdict

The simple 1D Liouville/gauge route is **not** a clean formalization shortcut for the chemotaxis drift unless you already have a bounded time derivative of the drift coefficient.

There are three important points:

1. The sign in the proposed transform is reversed for the equation

```text
u_t = ν u_xx + b u_x + c u.
```

To remove the drift, the useful gauge is

```text
w(t,x) = u(t,x) * exp((1 / (2ν)) * ∫_0^x b(t,ξ) dξ),
```

not the negative-exponent version.

2. Even with the correct sign, the transformed equation contains

```text
(1 / (2ν)) * ∫_0^x b_t(t,ξ) dξ.
```

So the gauge route needs a bound on `b_t`. For the chemotaxis drift

```text
b = -χ₀ * R_x / (1 + R)^β,
```

this means bounds on `R_t` and `R_xt`, hence on `u_t` through the elliptic resolver. That is likely circular if the goal is to obtain strict positivity before the classical bootstrap.

3. The gauge generally changes Neumann boundary conditions into Robin boundary conditions. In this chemotaxis case the boundary issue is probably harmless because `R_x = 0` at `x = 0,1`, hence `b = 0` at the boundary and Neumann is preserved. But this is an extra lemma, not automatic.

So: **the gauge transform works only under stronger hypotheses**: bounded `b`, bounded `b_x`, bounded `b_t` in the integrated sense, and `b(t,0)=b(t,1)=0`. Without `b_t`, it does not reduce the problem to the already-formalized driftless Neumann heat-kernel positivity.

The formalization-feasible recommendation is either:

- avoid strict positivity entirely if `u ≥ 0` plus `R[u] ≥ 0` suffices for the classical upgrade, or
- add a black-box/minimal positivity-improving theorem for 1D bounded-drift Neumann parabolic operators.

Do **not** expect the Liouville gauge to remove the need for a drifted maximum principle unless `b_t` is already available.

## Exact gauge calculation

Let

```text
B(t,x) := ∫_0^x b(t,ξ) dξ.
```

Write the gauge as

```text
w(t,x) = exp(Φ(t,x)) * u(t,x).
```

Then

```text
u = exp(-Φ) * w.
```

Compute:

```text
u_x  = exp(-Φ) * (w_x - Φ_x w),

u_xx = exp(-Φ) * (w_xx - 2 Φ_x w_x + (Φ_x^2 - Φ_xx) w),

u_t  = exp(-Φ) * (w_t - Φ_t w).
```

Substitute into

```text
u_t = ν u_xx + b u_x + c u.
```

After multiplying by `exp(Φ)`, one gets

```text
w_t - Φ_t w
  = ν w_xx + (-2ν Φ_x + b) w_x
      + (ν(Φ_x^2 - Φ_xx) - b Φ_x + c) w.
```

Therefore

```text
w_t = ν w_xx + (-2ν Φ_x + b) w_x
      + (Φ_t + ν(Φ_x^2 - Φ_xx) - b Φ_x + c) w.
```

To kill the drift, choose

```text
Φ_x = b / (2ν),
```

that is

```text
Φ(t,x) = (1 / (2ν)) * ∫_0^x b(t,ξ) dξ.
```

Thus the correct gauge is

```text
w(t,x) = u(t,x) * exp((1 / (2ν)) * ∫_0^x b(t,ξ) dξ).
```

Then

```text
Φ_x  = b / (2ν),
Φ_xx = b_x / (2ν),
Φ_t  = (1 / (2ν)) * ∫_0^x b_t(t,ξ) dξ.
```

The transformed equation is

```text
w_t = ν w_xx + c̃(t,x) w,
```

with

```text
c̃(t,x)
  = c(t,x)
    + (1 / (2ν)) * ∫_0^x b_t(t,ξ) dξ
    - (1 / 2) * b_x(t,x)
    - b(t,x)^2 / (4ν).
```

This is the load-bearing formula.

## What happens with the sign proposed in the question?

The proposed transform was

```text
w = u * exp(-(1 / (2ν)) * ∫_0^x b(t,ξ) dξ).
```

That corresponds to

```text
Φ_x = -b / (2ν).
```

Then the drift coefficient becomes

```text
-2ν Φ_x + b = 2b,
```

so the drift is not removed. The resulting equation is

```text
w_t = ν w_xx + 2b w_x
      + (c - (1 / (2ν)) * ∫_0^x b_t(t,ξ) dξ
           + b_x / 2 + 3 b^2 / (4ν)) w.
```

So the negative-exponent gauge is the wrong sign for the PDE convention

```text
u_t = ν u_xx + b u_x + c u.
```

If the PDE is written with the opposite drift sign, adjust accordingly, but for the displayed equation above the positive exponent is the drift-killing one.

## Boundary conditions

Assume `u_x(t,0)=u_x(t,1)=0`. Since

```text
w_x = exp(Φ) * (u_x + Φ_x u),
```

we get at the boundary

```text
w_x(t,i) = Φ_x(t,i) * w(t,i) = b(t,i) / (2ν) * w(t,i),   i = 0,1.
```

So the gauge normally changes Neumann into Robin:

```text
w_x = (b / (2ν)) w   at x = 0,1.
```

It preserves Neumann only if

```text
b(t,0) = b(t,1) = 0.
```

For the chemotaxis drift

```text
b = -χ₀ * R_x / (1 + R)^β,
```

and a Neumann resolver `R`, one expects

```text
R_x(t,0) = R_x(t,1) = 0,
```

hence

```text
b(t,0) = b(t,1) = 0.
```

So the boundary obstruction is probably removable in this model. In Lean, this should be a named lemma, for example:

```lean
lemma chemotaxis_drift_boundary_zero
    ... : b t 0 = 0 ∧ b t 1 = 0 := by
  -- unfold b, use resolver Neumann boundary derivative
```

Then the transformed `w` satisfies Neumann and the already-formalized driftless Neumann heat-kernel positivity can apply.

## The real obstruction: the `b_t` term

The transformed zero-order coefficient is

```text
c̃ = c + Φ_t - b_x / 2 - b^2 / (4ν),
Φ_t = (1 / (2ν)) * ∫_0^x b_t(t,ξ) dξ.
```

Thus to use the scalar-discount lower bound

```text
w(t) ≥ exp(-C(t-s)) * S_ν(t-s) w(s),
```

you need a lower bound on `c̃`, hence at least an `L∞` or integrable-in-space bound on

```text
∫_0^x b_t(t,ξ) dξ.
```

For chemotaxis,

```text
b = -χ₀ * R_x / (1 + R)^β.
```

Then schematically

```text
b_t
  = -χ₀ * ( R_xt / (1 + R)^β
      - β * R_x * R_t / (1 + R)^(β+1) ).
```

The resolver equation is

```text
(μ - ν ∂_xx) R = ν u^γ        -- after u ≥ 0 and truncation inactive
```

or `ν u_+^γ` before identifying truncation. Therefore `R_t` depends on the elliptic solve applied to something like

```text
γ u^(γ-1) u_t.
```

Even for `γ = 1`, this uses `u_t`. For noninteger real `γ ≥ 1`, the behavior at zeros also needs care unless strict positivity is already known or the positive-part formulation is retained.

So the gauge route requires time regularity of the solution and of the resolver **before** the strict-positivity argument. If strict positivity is being used to unlock the classical bootstrap that would provide `u_t`, this is circular.

## If `b_t` is available, the route closes

If you can prove the following hypotheses on a restart slab `[s,T] × [0,1]`:

```text
ν > 0,
|b| ≤ B,
|b_x| ≤ Bx,
|∫_0^x b_t(t,ξ)dξ| ≤ Bt,
|c| ≤ Cc,
b(t,0)=b(t,1)=0,
```

then the gauge route is clean.

Define

```text
Φ(t,x) = (1 / (2ν)) * ∫_0^x b(t,ξ) dξ,
w = exp(Φ) u.
```

Then:

```text
w_t = ν w_xx + c̃ w,
w_x(t,0)=w_x(t,1)=0,
```

with

```text
c̃ = c + Φ_t - b_x / 2 - b^2/(4ν).
```

Choose

```text
C ≥ sup |c̃|       -- or just c̃ ≥ -C.
```

Then

```text
w_t ≥ ν w_xx - C w.
```

Equivalently, for

```text
z(t) = exp(C(t-s)) w(t),
```

one has

```text
z_t ≥ ν z_xx.
```

With Neumann boundary conditions, comparison gives

```text
w(t) ≥ exp(-C(t-s)) * S_ν(t-s) w(s).
```

Since `exp(Φ)` is strictly positive and bounded above/below on `[0,1]`,

```text
u ≥ 0 and u not identically 0  ⇔  w ≥ 0 and w not identically 0.
```

Then positivity-improvement of the Neumann heat kernel gives

```text
S_ν(t-s) w(s)(x) > 0
```

for every `t > s` and every `x ∈ [0,1]`. Hence

```text
w(t,x) > 0,
u(t,x) > 0.
```

This is mathematically valid under the stronger `b_t` hypothesis.

## Is freezing coefficients enough?

Not by itself.

If one freezes

```text
b_s(x) := b(s,x)
```

and gauges against `b_s`, then the transformed equation has a residual first-order term

```text
(b(t,x) - b(s,x)) w_x.
```

This avoids `b_t`, but it does **not** reduce to a driftless scalar-discount comparison. The residual term has no sign and involves `w_x`; it cannot be absorbed into a zero-order scalar discount.

On a very short interval, this residual may be small if `b` is time-continuous, but a perturbative lower-bound proof is delicate: the positive heat lower bound from rough nonnegative data may be arbitrarily small, so subtracting a small gradient-error term is not obviously safe. This is not a good Lean route unless you build a full perturbative positivity theory.

So the frozen-coefficient idea does not give the desired shortcut. It merely trades the `b_t` obstruction for a residual-drift positivity argument, which is essentially the same missing maximum-principle/Harnack infrastructure.

## Minimal statement to add if strict positivity is truly needed

The right black-box theorem is not the gauge transform; it is a positivity-improving theorem for bounded-drift parabolic equations on the interval.

A useful minimal theorem would be:

```lean
/-- Positivity improvement for 1D Neumann parabolic equations with bounded drift. -/
theorem neumann_bounded_drift_pos_improving
    {ν : ℝ} (hν : 0 < ν)
    {b c : ℝ → ℝ → ℝ}
    (hb_bound : ∃ B, ∀ t x, |b t x| ≤ B)
    (hc_lower : ∃ C, ∀ t x, -C ≤ c t x)
    (hb_boundary : ∀ t, b t 0 = 0 ∧ b t 1 = 0) -- or the correct no-flux condition
    (u_nonneg : ∀ x, 0 ≤ u0 x)
    (u_nonzero : ∃ x, 0 < u0 x)
    (hu_sol : solves_neumann_bounded_drift_parabolic ν b c u0 u) :
    ∀ t > 0, ∀ x ∈ Set.Icc (0 : ℝ) 1, 0 < u t x := by
  ...
```

Analytically, this follows from any of:

1. the strong parabolic maximum principle plus boundary point lemma,
2. a positivity-improving evolution-family kernel,
3. Aronson-type Gaussian lower bounds for uniformly parabolic equations with bounded coefficients,
4. Feynman-Kac for reflected diffusion with bounded drift and potential.

For Lean, the smallest usable imported/axiom-free target is probably a bespoke 1D theorem with exactly the hypotheses needed by the chemotaxis bootstrap, rather than a full Aronson theory.

## Relation to Aronson/Harnack theory

For bounded measurable drift and bounded potential in a uniformly parabolic equation, the fundamental solution is strictly positive and satisfies Gaussian lower bounds of the schematic form

```text
Γ(t,x;s,y) ≥ C₁ (t-s)^(-1/2)
  * exp(-C₂ |x-y|² / (t-s) - C₃(t-s))
```

on compact time intervals, with constants depending on `ν`, the coefficient bounds, and the domain. On a bounded interval with reflecting/Neumann boundary, one gets a corresponding positive Neumann heat kernel/evolution kernel.

This would immediately imply strict positivity:

```text
u(t,x) = ∫ Γ(t,x;s,y) u(s,y) dy > 0
```

whenever `u(s,·) ≥ 0` and is not identically zero.

But formalizing Aronson/Nash/Moser from scratch is much heavier than the rest of the Picard bootstrap. It is not a quick replacement for the missing strict-positivity producer.

## Final recommendation

For the chemotaxis drift

```text
b = -χ₀ * R_x / (1 + R)^β,
```

the 1D gauge route is conditionally valid but not the clean path:

- The correct gauge is

```text
w = u * exp((1 / (2ν)) * ∫_0^x b).
```

- Boundary conditions remain Neumann only because `b = 0` at the endpoints, which follows from the Neumann resolver.

- The zero-order coefficient contains

```text
(1 / (2ν)) * ∫_0^x b_t.
```

- Bounding that term requires time regularity of the resolver and hence of `u_t`, which is likely unavailable at the point where strict positivity is supposed to enable the classical bootstrap.

Therefore:

```text
Gauge route: GO only with an extra bounded-b_t hypothesis.
Gauge route for the current chemotaxis bootstrap before classical regularity: NO-GO / circular.
```

If strict positivity is truly load-bearing, add a targeted bounded-drift Neumann positivity-improvement theorem. If the downstream bootstrap can be reorganized to use only `u ≥ 0` and `R[u] ≥ 0`, that is much more formalization-feasible and avoids the Harnack/Aronson infrastructure entirely.
