For **Paper 2 Proposition 2.5**, the correct endpoint is **not** an infinite Moser/Alikakos iteration.  The paper assumes one exponent

```text
p > max {N, m N, γ N}
```

The division of labor is therefore:

```text
weighted Lp energy / Corollary 2.1
    -> obtain a uniform finite-p bound, often for every finite p
    -> choose one p > max {N, m N, γ N}
```

# 1. What Proposition 2.5 actually does

The repository statement is exactly:

```lean
def Proposition_2_5 (D : BoundedDomainData) (p : CM2Params) : Prop :=
  ∀ u₀ : D.Point → ℝ, PositiveInitialDatum D u₀ →
    ∀ Tmax > 0, ∀ u v : ℝ → D.Point → ℝ,
      IsPaper2ClassicalSolution D p Tmax u v →
      InitialTrace D u₀ u →
        ∀ pExp,
          max (p.N : ℝ)
              (max (p.m * (p.N : ℝ)) (p.γ * (p.N : ℝ))) < pExp →
            LpPowerBoundedBefore D pExp Tmax u →
              IsPaper2BoundedBefore D Tmax u
```

where

```lean
def LpPowerBoundedBefore
    (D : BoundedDomainData) (pExp Tmax : ℝ)
    (u : ℝ → D.Point → ℝ) : Prop :=
  ∃ C, ∀ t, 0 < t → t < Tmax →
    D.integral (fun x => (u t x) ^ pExp) ≤ C


def IsPaper2BoundedBefore
    (D : BoundedDomainData) (Tmax : ℝ)
    (u : ℝ → D.Point → ℝ) : Prop :=
  ∃ M, ∀ t, 0 < t → t < Tmax → D.supNorm (u t) ≤ M
```

The paper's proof proceeds as follows.

## 1.1 Resolver endpoint

From

```text
pExp > γ N
```

choose an exponent `qv > N` such that

```text
γ qv < pExp.
```

The uniform `L^pExp` bound on `u` gives a uniform `L^qv` bound on `u^γ`.  Elliptic regularity for

```text
(μ - Δ) v = ν u^γ,
∂n v = 0
```

and the embedding `W^{2,qv} -> W^{1,∞}` yield

```text
sup_t ||v(t)||∞ + sup_t ||∇v(t)||∞ < ∞.
```

On the interval one may simply take

```text
qv := pExp / γ.
```

Then `qv > N` follows directly from `pExp > γ N`, and

```text
||(u(t))^γ||_{qv} = ||u(t)||_{pExp}^γ.
```

This avoids an existential exponent choice.

## 1.2 Mild formula

With

```text
A := -Δ_N + μ I,
```

the paper uses

```text
u(t)
  = exp(-A t) u₀
    - χ₀ ∫₀ᵗ exp(-A (t-s)) div
        (u(s)^m * ∇v(s) / (1+v(s))^β) ds
    + ∫₀ᵗ exp(-A (t-s))
        [u(s) * (μ + a - b u(s)^α)] ds.
```

The homogeneous term is bounded in `L∞` by the initial `L∞` norm.

## 1.3 Chemotaxis Duhamel term

From

```text
pExp > m N
```

choose `qf > N` with `m qf < pExp`.  Again, on the interval the clean choice is

```text
qf := pExp / m,
```

for which `qf > N` and

```text
||u(t)^m||_{qf} = ||u(t)||_{pExp}^m.
```

Since `v >= 0` and `β >= 0`,

```text
(1 + v)^(-β) <= 1.
```

Together with the resolver-gradient bound,

```text
||u(t)^m * ∇v(t) / (1+v(t))^β||_{qf}
  <= Mv * ||u(t)||_{pExp}^m.
```

The paper then chooses

```text
0 < σ < 1/2,
N / (2 qf) < σ.
```

A completely explicit choice is

```text
σ := 1/4 + N / (4 qf).
```

Indeed, `qf > N` gives

```text
0 < N / (2 qf) < σ < 1/2.
```

The fractional divergence estimate and Sobolev embedding give

```text
||∫₀ᵗ exp(-A(t-s)) div F(s) ds||∞
  <= C ∫₀ᵗ
      ((t-s)^(-σ) + (t-s)^(-1/2-σ))
      exp(-δ (t-s)) ||F(s)||_{qf} ds.
```

Both singularities are integrable at zero because

```text
σ < 1,
1/2 + σ < 1,
```

and the exponential factor makes the kernel integrable at infinity.  Hence this term is uniformly bounded for all `t < Tmax`, even when `Tmax = ∞`.

Equivalently, in one spatial dimension one may prove the sharper direct kernel estimate

```text
||∂x S_N(r) F||∞
  <= Cq * (1 + r^(-1/2 - 1/(2q))) * ||F||q,
```

where `q > 1`.  The exponent is strictly below `1`, so its damped time convolution is integrable.  This avoids fractional-power spaces entirely.

## 1.4 Reaction Duhamel term and the `b = 0` issue

The printed proof maximizes

```text
w ↦ w (μ + a - b w^α)
```

and bounds the source by a constant.  That calculation divides by `b`, so it directly applies when `b > 0`.

The repository's `CM2Params` stores only

```lean
hb : 0 ≤ b
```

and `Proposition_2_5` has no extra `0 < b` hypothesis.  The robust Lean proof should therefore avoid division by `b`.  Use instead

```text
u (μ + a - b u^α) <= (μ + a) u
```

and positivity of the heat semigroup.  The value-semigroup estimate gives

```text
||S_N(r) u(s)||∞
  <= Cp * (1 + r^(-N/(2 pExp))) ||u(s)||_{pExp}.
```

Since `pExp > N`, the exponent `N/(2 pExp)` is below `1`, and

```text
∫₀∞ (1 + r^(-N/(2 pExp))) exp(-μ r) dr < ∞.
```

Thus the reaction term is uniformly bounded without strengthening `hb`.

If the downstream theorem already assumes `0 < b`, the paper's constant-source bound can be used instead; it is slightly shorter but less general than the current Lean statement.

# 2. Why the proposed abstract `Lp_to_Linfty` premise is not sufficient

The following implication is false as a purely measure-theoretic statement:

```lean
(∀ p < ∞, ||f||p ≤ Cp) -> ||f||∞ < ∞.
```

The constants `Cp` may grow arbitrarily fast.  There are unbounded functions on a finite-measure interval that belong to every finite `L^p`.  For a time-dependent family, separate uniform-in-time constants for every finite `p` still do not give a uniform `L∞` bound unless their `p`-th roots are controlled.

The correct abstract endpoint would require a root envelope, for example

```text
p_n -> ∞,
||u(t)||_{p_n} <= C_n for every t,
sup_n C_n < ∞,
```

when `C_n` already denotes the norm, or

```text
∫ u(t)^(p_n) <= E_n,
sup_n E_n^(1/p_n) < ∞,
```

when `E_n` denotes the power integral.

That is exactly what a Moser/Alikakos root tower is designed to prove.  The current repository's

```lean
IntervalDomainMoserQuantitativeEndpoint
```

encodes this stronger information.  But Proposition 2.5 does not need it: the PDE and its mild formula turn **one sufficiently large finite exponent** into `L∞`.

Accordingly, the preferred theorem name should not be the unrestricted

```lean
Lp_to_Linfty : (∀ p, ...) -> ...
```

but something PDE-specific such as

```lean
intervalDomain_Proposition_2_5_via_one_large_Lp
```

or

```lean
intervalDomain_boundedBefore_of_largeLp_and_mild_smoothing.
```

```lean
      ∀ t, 0 < t → t < T →
        intervalDomain.supNorm
```

There are two honest implementations.

### Specialized fractional-power implementation

Use the already stated interfaces

```lean
Lemma_2_2 intervalDomain intervalDomainSemigroupEstimateData
Lemma_2_4 intervalDomain params intervalDomainSemigroupEstimateData
```

with

```text
σ = 1/4 + N/(4q).
```

`Lemma_2_4` gives

```text
fractionalNorm σ q (divergenceSemigroup r F)
  <= C r^(-σ) (1+r^(-1/2)) exp(-μr/2) ||F||q,
```

and the first branch of `Lemma_2_2`, with target `L∞`, gives the embedding because

```text
N/(2q) < σ.
```

Then integrate the two powers.

### Direct interval-kernel implementation

Prove the sharp one-dimensional estimate

```text
||intervalDomainHeatDivergenceSemigroup r F||∞
  <= Cq * (1 + r^(-1/2 - 1/(2q))) * ||F||q.
```

This follows from Hölder's inequality and a uniform `L^(q/(q-1))` bound on
`∂x K_N(r,x,·)`.  Since `q > 1`,

```text
1/2 + 1/(2q) < 1.
```

For the current repository, the **specialized integrated capstone** is preferable to formalizing a large general Banach fractional-power library solely for this theorem.  It can still be proved internally by the `Lemma_2_2`/`Lemma_2_4` route.

The currently committed nonsharp divergence endpoint is not enough.  The spectral `L¹ -> L∞` gradient bound in `HeatKernelGradientEstimates.lean` behaves like `t^(-2)`, and `IntervalDomainLemma21.lean` explicitly labels the downstream divergence route nonsharp.  A `t^(-2)` time singularity cannot be integrated at the Duhamel endpoint.  Codex must prove the sharp integrable exponent rather than wire the existing nonsharp theorem and hope Gronwall fixes it.

# 6. Mild formula target

The exact damped variation-of-constants identity should be exposed once, in a form reusable by Proposition 2.5:

```lean
def IntervalDomainProp25MildFormula (params : CM2Params) : Prop :=
  ∀ {u₀ : intervalDomain.Point → ℝ}
    {T : ℝ} {u v : ℝ → intervalDomain.Point → ℝ},
    IsPaper2ClassicalSolution intervalDomain params T u v →
    InitialTrace intervalDomain u₀ u →
      ∀ t, 0 < t → t < T → ∀ x,
        u t x =
          Real.exp (-params.μ * t) *
            intervalDomainHeatSemigroup t u₀ x
          - params.χ₀ *
            intervalDomainDampedDivergenceDuhamel params.μ
              (fun s => intervalDomainProp25Flux params u v s) t x

                u s y *
                  (params.μ + params.a -
                    params.b * (u s y) ^ params.α)) t x
```

The exact function names may need the repository's existing semigroup aliases, but this is the mathematical identity to formalize.  It should be derived from the classical PDE plus Neumann boundary conditions, or reused from an existing faithful Duhamel representation if one already matches the same source convention.

Do not use the known false value-at-zero assertion

# 7. Assembly skeleton

After the preceding leaves exist, the capstone is finite bookkeeping:

```lean

    (hMild : IntervalDomainProp25MildFormula params)

  intro u₀ hu₀ Tmax hT u v hsol htrace pExp hpExp hLp

    lt_of_le_of_lt (le_max_left _ _) hpExp
  have hpmN : params.m * (params.N : ℝ) < pExp :=
    lt_of_le_of_lt
      (le_trans (le_max_right (params.N : ℝ) _)
        (le_max_left _ _)) hpExp
  have hpγN : params.γ * (params.N : ℝ) < pExp :=
    lt_of_le_of_lt
      (le_trans (le_max_right (params.N : ℝ) _)
        (le_max_right _ _)) hpExp

  let qf : ℝ := pExp / params.m

    dsimp [qv]
    rw [lt_div_iff₀ params.hγ]
    simpa [mul_comm] using hpγN

    dsimp [qf]
    rw [lt_div_iff₀ params.hm]

  intro t ht0 htT
```

1. **Which bootstrap is Proposition 2.5?**  It is a direct one-large-`p` mild/semigroup jump.  It does not iterate `p_k -> κ p_k`.  On the bounded interval this is cleaner in Lean than a full Alikakos root tower, provided the sharp integrable divergence-semigroup estimate is proved.

   rather than a bare `(∀ p, ||u||p ≤ Cp) -> ||u||∞` implication.  The latter requires an additional uniform root-envelope hypothesis and is not Proposition 2.5.
