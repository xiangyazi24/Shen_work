The proof mirrors `IntervalGradDuhamelBound.gradDuhamel_sup_bound`:

1. Set `Cg := heatGradientLinftyLinftyConstant` and use `heatGradientLinftyLinftyConstant_nonneg`.
2. Prove the pointwise estimate on the open shifted interval:

   ```lean
   ∀ s ∈ Set.Ioo a t,
     |deriv (fun z => intervalFullSemigroupOperator (t - s) (q s) z) x|
       ≤ Cg * Cq * (t - s) ^ (-(1 / 2) : ℝ)
   ```

   This uses

   ```lean

   ```

   with `0 ≤ s` obtained from `ha : 0 ≤ a` and `a < s`.

3. Prove the shifted singular kernel is interval-integrable on `[a,t]` by composing the usual `u ↦ u^(-1/2)` integrability with `s ↦ t - s`:

   ```lean
   have h0 : IntervalIntegrable
       (fun u : ℝ => u ^ (-(1 / 2) : ℝ)) volume 0 (t - a) :=
     intervalIntegral.intervalIntegrable_rpow' (by norm_num)
   have h := (h0.comp_sub_left t).symm
   simpa using h
   ```

4. Use `intervalIntegral.integral_mono_on_of_le_Ioo` to dominate the absolute value of the gradient integrand by the singular majorant.

5. Evaluate the shifted singular integral:

   ```lean
   ∫ s in a..t, (t - s) ^ (-(1 / 2) : ℝ)
     = 2 * Real.sqrt (t - a)
   ```

   via `intervalIntegral.integral_comp_sub_left` and `intervalIntegral.integral_rpow`.

6. Finish with `Real.sqrt_le_sqrt` from `t - a ≤ T - a`.
