# 4. Why weighted `L²` plus a spatial modulus cannot produce the left tail

`WaveStabilityUpgrade.lean` correctly says that the exponential weight loses control at the left end. Here is the exact logical obstruction.

Fix `η>0` and a nonzero smooth compactly supported function `φ`. Define

\[
e_n(z)=\phi(z+n).
\]

The packet is centered near `z=-n`. Its sup norm and every fixed spatial modulus are independent of `n`, but

\[
\begin{aligned}
\int_{\mathbb R} e^{2\eta z}|e_n(z)|^2\,dz
&=e^{-2\eta n}
  \int_{\mathbb R}e^{2\eta y}|\phi(y)|^2\,dy\\
&\longrightarrow0.
\end{aligned}
\]

Thus all of the following can hold simultaneously:

```text
weighted L² norm → 0;
uniform spatial equicontinuity;
sup_z |e_n(z)| = constant > 0;
the bad points tend to -∞.
```

This is precisely the last case in Step 4. The weighted-energy/local-modulus argument rules out bad points that stay in a fixed half-line `z ≥ -R`; it has no mechanism to rule out a defect packet escaping to `-∞`. The separate left-tail theorem supplies exactly that missing localization.

Consequently option (b) is not merely absent from the current Lean proof. It is mathematically false at the level of the stated functional-analytic interface.

In §5.2, Step 4, the proof proceeds as follows.

## 5.1 Positive-time weighted `H¹` and Morrey

For a fixed `t₀>0`, the paper uses

```text
weighted error ∈ L²,
weighted spatial derivative ∈ L²
```

and one-dimensional Morrey/Sobolev control to obtain a pointwise weighted bound. Combined with the wave tail, this yields the two-sided comparison preparation (5.38).

## 5.2 A persistent far-left floor

Using the lower subsolution machinery from Remark 4.2, the paper obtains (5.39), equivalently the quantitative statement (5.40): there are

```text
d₀ > 0, R₀, T₀
```

such that

\[
u(t,z+c t)\ge d_0
\qquad(t\ge T_0,\ z\le -R_0).
\]

In repository language, this is naturally a predicate such as

```lean
def UniformCoMovingFarLeftFloor
    (c : ℝ) (u : ℝ → ℝ → ℝ) : Prop :=
  ∃ d R T : ℝ, 0 < d ∧
    ∀ t z, T ≤ t → z ≤ -R →
      d ≤ coMovingPath c u t z
```

This is much weaker than global `u(t,·)→1`, but it is the persistence input that survives in a far-left translate limit.

## 5.3 The bad sequence must escape left

Assume uniform moving-frame convergence fails. Choose

```text
tₙ → +∞,
zₙ ∈ ℝ,
|q(tₙ,zₙ)-U(zₙ)| ≥ ε₀.
```

The uniform spatial modulus makes the defect remain large on a fixed interval around `zₙ`. If `zₙ` is bounded below, that interval carries a fixed positive amount of the weighted energy, contradicting weighted-`L²` convergence. Hence

```text
zₙ → -∞.
```

This is exactly the split implemented abstractly by `WaveStabilityUpgrade.lean`: weighted energy handles the non-left region; the separate tail input handles this remaining case.

## 5.4 Time-space translates and an entire limit

Define

\[
q_n(s,y)=q(t_n+s,z_n+y),
\]

and translate the elliptic signal in the same way. Positive-time parabolic regularity and the bounded physical box give a locally convergent subsequence. The limit is a bounded entire classical solution of the same autonomous co-moving PDE.

Because `zₙ→-∞`, the persistent far-left floor passes to every fixed compact set and then to the limit:

\[
q_\infty(s,y)\ge d_0
\qquad\text{for all }(s,y)\in\mathbb R^2.
\]

The defect also passes at the origin. Since `U(zₙ)→1`,

\[
|q_\infty(0,0)-1|\ge\varepsilon_0.
\]

## 5.5 Entire-orbit rigidity

The paper then invokes Proposition 1.2 to conclude that the bounded, uniformly positive entire orbit is identically `1`, contradicting the preceding inequality.

For a Lean proof, that invocation should be made explicit and noncircular. One should not require arbitrary-solution BUC uniqueness merely to identify the entire limit with the canonical global solution.

# 6. The clean noncircular rigidity theorem

The repository already has the scalar slab barriers needed to prove entire-orbit rigidity directly:


in `WholeLineCauchyLongTimeFloor.lean`.

These are PDE theorems for an arbitrary bounded classical slab; they are not restricted to the canonical fixed point.

The appropriate new theorem is schematically:

```lean
theorem wholeLine_entire_eq_one_of_chi_nonpos_of_uniform_box
    (p : CMParams) (hχ : p.χ ≤ 0)
    {q r : ℝ → ℝ → ℝ}
    (hentire : IsEntireClassicalSolution p q r)
    {d M : ℝ} (hd : 0 < d)
    (hbox : ∀ t x, d ≤ q t x ∧ q t x ≤ M) :
    ∀ t x, q t x = 1
```

The exact `IsEntireClassicalSolution` name can be chosen to match the new translate-limit package; the proof itself is short.

Fix `t,x` and `N>0`. Restart the entire orbit at time `t-N` and view it on the slab `[0,N]`. Put

```text
d₁ := min d 1,
M₁ := max M 1.
```

The lower slab theorem gives

\[
1+(d_1-1)e^{-d_1N}\le q(t,x),
\]

and the upper slab theorem gives

\[
q(t,x)\le1+(M_1-1)e^{-N}.
\]

Both scalar barriers tend to `1` as `N→∞`; hence `q(t,x)=1`. Since `t,x` were arbitrary, the entire orbit is identically one.

This argument has three advantages:

1. no canonical-solution identification;
2. no imported Cauchy uniqueness theorem;
3. no circular use of the final traveling-wave stability theorem.

It is the rigorous version of the paper's “apply Proposition 1.2 backward from arbitrarily remote times.” The uniformity is important: the attraction modulus depends only on the common order interval `[d,M]`, not on the particular slice `q(t-N,·)`.

The limit passages use only the explicit exponential formulas and Mathlib facts such as `Real.tendsto_exp_atBot`; the lower formula already has the committed theorem

