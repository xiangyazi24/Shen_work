# Q4264: uniform `WindowSourceEnvelope` for the truncated B-form source

## Executive answer

The `WindowSourceEnvelope` route is correct, but there is one important correction:

**`W^{1,∞}` is not enough to get a summable cosine-coefficient envelope.**

For a cosine coefficient

```text
c_k(f) = ∫₀¹ f(x) cos(kπx) dx,
```

`f ∈ W^{1,∞}` only gives `|c_k(f)| = O(1/k)` after one integration by parts, which is not summable. To get the requested ℓ¹ envelope, use a `k⁻²` envelope, and therefore require a uniform `W^{2,1}`-type bound for the logistic source and a uniform `W^{3,1}`-type bound for the flux `Q` when the chemotaxis source coefficient is written as `kπ · sineCoeff(Q,k)`.

So the route is:

```lean
E k := Elog k + Echem k
```

with

```text
Elog 0      := Clog0
Elog (k+1) := Clog2 / (((k+1)π)^2)

Echem 0      := 0        -- divergence coefficient has zero mean if Q(0)=Q(1)=0
Echem (k+1) := Cchem2 / (((k+1)π)^2)
```

Then `Summable E` follows from the p-series `∑ (k+1)⁻²`.

The uniformity over `r ∈ [c,T']` is obtained by choosing `Clog0`, `Clog2`, and `Cchem2` from the **uniform compact-window spatial regularity constants** on `[c,T']`, not from pointwise-in-time bounds.

## 1. Logistic branch

Let

```lean
L r x := u r x * (p.a - p.b * (u r x) ^ p.α)
```

after `u ≥ 0`, so the positive part is inactive. The zero mode is bounded by the `L∞` bound:

```text
|c_0(L r)| ≤ ∫₀¹ |L r x| dx ≤ Clog0.
```

For `k ≥ 1`, integrate twice:

```text
c_k(L r) = ∫₀¹ L r x cos(kπx) dx.
```

First integration by parts gives no boundary term because `sin(kπx)=0` at `0,1`:

```text
c_k(L r) = -(1/(kπ)) ∫₀¹ ∂x L r x · sin(kπx) dx.
```

Second integration by parts gives

```text
c_k(L r)
  = (∂x L(r,1)(-1)^k - ∂x L(r,0))/(kπ)^2
    - (1/(kπ)^2) ∫₀¹ ∂xx L r x · cos(kπx) dx.
```

Hence

```text
|c_k(L r)| ≤ (|∂x L(r,0)| + |∂x L(r,1)| + ∫₀¹ |∂xx L r x| dx)/(kπ)^2.
```

If the Neumann condition gives `u_x(r,0)=u_x(r,1)=0`, then usually `∂x L(r,0)=∂x L(r,1)=0`, but the envelope does **not** need to rely on that. It can simply include the endpoint derivative terms.

So define a uniform constant

```text
Clog2 := sup_{r∈[c,T']} (|L_x(r,0)| + |L_x(r,1)| + ‖L_xx(r)‖_{L¹(0,1)}).
```

Then

```lean
Elog 0      = Clog0
Elog (k+1) = Clog2 / (((k+1 : ℝ) * Real.pi)^2)
```

works.

### Lean leaf lemma for the logistic branch

Use a local lemma with this shape:

```lean
lemma cosineCoeff_abs_le_inv_sq_of_W2_1
    {f f' f'' : ℝ → ℝ} {C2 : ℝ}
    (hf_absCont : -- f is AC on [0,1] with derivative f')
    (hf'_absCont : -- f' is AC on [0,1] with derivative f'')
    (hC2 : |f' 0| + |f' 1| + ∫ x in (0:ℝ)..1, |f'' x| ≤ C2) :
    ∀ k : ℕ, k ≠ 0 →
      |unitIntervalNeumannCosineCoeff f k| ≤
        C2 / (((k : ℝ) * Real.pi)^2)
```

or specialize it to the repository's cosine-coefficient definition.

Do **not** state this from `W^{1,∞}` alone. The actual load-bearing hypothesis is `W^{2,1}` or an equivalent `C¹ + f' absolutely continuous + f'' ∈ L¹` package.

## 2. Chemotaxis-divergence branch

Let

```lean
Q r x := u r x * resolverGradReal p (u r) x /
           (1 + intervalDomainLift (intervalNeumannResolverR p (u r)) x) ^ p.β
```

With the `ρ₊` resolver fix and `u ≥ 0`, the resolver is nonnegative:

```text
R[u](r,x) ≥ 0,
```

so

```text
1 + R[u](r,x) ≥ 1.
```

Therefore the denominator factors satisfy

```text
|(1+R)^(-β)| ≤ 1,
|(1+R)^(-β-1)| ≤ 1.
```

This simplifies the flux derivative envelope, but for the **coefficient envelope** the relevant question is the spatial regularity of `Q`.

The divergence cosine coefficient is, up to the fixed sign/constant convention,

```text
s_k^chem(r) = -χ₀ * kπ * ∫₀¹ Q(r,x) sin(kπx) dx.
```

Since `R_x=0` at the Neumann endpoints, one has

```text
Q(r,0)=Q(r,1)=0.
```

Then

```text
kπ ∫₀¹ Q sin(kπx) dx = ∫₀¹ Q_x cos(kπx) dx.
```

If only `Q_x ∈ L¹`, this is merely `O(1)`. If `Q ∈ W^{2,1}`, then it is only `O(1/k)`, still not summable. To get an ℓ¹ envelope, use one more integration by parts, i.e. require `Q ∈ W^{3,1}` or an equivalent bound on `Q''` endpoints plus `Q''' ∈ L¹`:

```text
|s_k^chem(r)|
  ≤ |χ₀| * (|Q_xx(r,0)| + |Q_xx(r,1)| + ‖Q_xxx(r)‖_{L¹}) / (kπ)^2.
```

Thus define

```text
Cchem2 := |χ₀| * sup_{r∈[c,T']}
  (|Q_xx(r,0)| + |Q_xx(r,1)| + ‖Q_xxx(r)‖_{L¹(0,1)}).
```

Then

```lean
Echem 0      = 0
Echem (k+1) = Cchem2 / (((k+1 : ℝ) * Real.pi)^2)
```

is summable and uniformly bounds the chemotaxis-divergence coefficients.

### Important regularity warning

The commonly used product-rule envelope

```text
|Q'| ≤ M H + β M Γ² + Γ G
```

is enough to show the chemotaxis-divergence coefficients are **bounded** in `k`; it is not enough for an ℓ¹ `WindowSourceEnvelope` over all modes.

For the ℓ¹ ladder, either:

1. use a stronger spatial regularity input giving `Q ∈ W^{3,1}` uniformly on `[c,T']`; or
2. strengthen the coefficient ladder so it consumes a weaker weighted envelope, if that is what the spectral reconstruction actually needs.

For the stated `WindowSourceEnvelope` as a single summable sequence `E_k`, the cleanest route is option 1.

## 4. Uniformity over `r ∈ [c,T']`

Uniformity is the key point and it is valid if the compact positive-time bootstrap supplies uniform spatial regularity constants. The constants should be chosen once for the whole compact window:

```text
M       := sup |u|
G1      := sup |u_x|
G2      := sup |u_xx|
G3      := sup |u_xxx|        -- needed for chemDiv k⁻² through Q∈W³,¹
Rj      := uniform resolver derivative bounds derived from M,G1,G2,G3
Clog0   := function of M
Clog2   := function of M,G1,G2 and endpoint derivative bounds
Cchem2  := function of M,G1,G2,G3 and resolver bounds
```

Then for every `r ∈ [c,T']`, the same constants apply. This is exactly what the `WindowSourceEnvelope` needs.

## 5. Bottom line

The HSpectral port is feasible, but the honest leaf is not merely “gradient bound implies ℓ¹ source envelope.” The honest leaf is:

```text
uniform W^{2,1} bound for the logistic source
+ uniform W^{3,1} bound for the chemotaxis flux Q
⇒ summable source-coefficient envelope.
```

If the current compact-window regularity is only `u ∈ W^{1,∞}` plus bounded `R,R_x,R_xx`, the proposed ℓ¹ envelope is under-hypothesized. If the Hölder-Schauder stage gives enough spatial derivatives uniformly on `[c,T']`, then define `E_k` with a `k⁻²` tail as above and the `WindowSourceEnvelope` construction is straightforward.
