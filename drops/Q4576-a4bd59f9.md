# Executive verdict

The modal Duhamel plan is the cleanest Lean route, and most of its **linear scalar/coefficient infrastructure is already present**. The right phase space for the fixed-point proof is a weighted-coordinate copy of `ℓ²(ℕ, ℂ)`, not the current subtype-only `FractionalPowerSpace` as-is.

However, a direct proof of the currently declared

```lean
IntervalDomainSpectralSemigroupOrbitBoundRaw p N
```

from L1–L5 is **not possible without repairing the interface**. Three cross-cutting issues are load-bearing:

1. The raw frontier compares the nonlinear orbit to the operator norm of the **pure Neumann heat** zero-mode complement. The committed wrapper then hard-codes the decay rate `Real.pi ^ 2`. Near the critical sensitivity, the true linearized spectral gap can be arbitrarily smaller than `π²`, so a bound of the form
   \[
   e^{-\delta t}\le C e^{-\pi^2 t}
   \]
   cannot hold for all `t` when `0 < δ < π²`.
2. `LinearlyStable` controls only modes `n ≠ 0`. In the minimal branch the zero mode is neutral and must be removed by the mass constraint; the current orbit frontier carries no zero-mode or mass hypothesis.
3. `ExponentialC1ConvergenceWith` asks for a uniform `C¹` estimate already at `t = 0`, while its neighborhood hypothesis is only an `X^s`/sup-type smallness assumption with `1/2 < s < 1`. Parabolic smoothing gives `C¹` for positive time, but its norm may blow up as `t ↓ 0`; without a `C¹` initial datum hypothesis, the all-`t ≥ 0` bound is not available.

After replacing the pure-heat comparison by an existential linearized rate, adding the correct zero-mode branch, and either starting the `C¹` estimate at a fixed positive time or strengthening the initial norm, the L1–L5 chain is straightforward and acyclic.

```lean

open ShenWork.PDE.SectorialOperator

  unitIntervalNeumannSpectrum.eigenvalue n
```


## Uniform gap is a separate lemma

`LinearlyStable` gives strict negativity mode by mode; L1 needs one common `delta`.

Use the following target:

```lean
def LinearizedModeGap
    (p : CM2Params) (uStar vStar delta : ℝ) : Prop :=
  0 < delta ∧
    ∀ n : ℕ,
      intervalLinearGrowth p uStar vStar n ≤ -delta


def LinearizedNonzeroModeGap
    (p : CM2Params) (uStar vStar delta : ℝ) : Prop :=
  0 < delta ∧
    ∀ n : ℕ, n ≠ 0 →
      intervalLinearGrowth p uStar vStar n ≤ -delta
```

For the positive equilibrium, mode zero satisfies

```text
sigma(..., 0) = -a*alpha < 0,
```

so the full gap follows from the nonzero gap and `0 < a`.

For the minimal branch, retain `LinearizedNonzeroModeGap` and work in `ModalHsZero`.

A clean extraction theorem is:

```lean
theorem exists_linearizedModeGap_of_linearlyStable

      LinearlyStable unitIntervalNeumannSpectrum p uStar vStar)
```

The proof does not redo the dichotomy. Use the explicit formula for `sigma`: the `-lambda_n` term forces the tail to `-infinity`, so only finitely many low modes remain; take the minimum of their positive decay rates. For the minimal branch use the analogous nonzero-mode theorem.

# 4. L2 smoothing

A uniform gap alone proves decay, but smoothing also needs comparison of the modal decay rate with the Laplacian frequency. Package that separately:

```lean
def LinearizedCoerciveGap
    (p : CM2Params) (uStar vStar delta c : ℝ) : Prop :=
  0 < delta ∧ 0 < c ∧
    ∀ n : ℕ,
      delta + c * (1 + unitIntervalNeumannSpectrum.eigenvalue n) ≤
        -intervalLinearGrowth p uStar vStar n
```

For the minimal branch quantify only over `n ≠ 0`.

This is true because the chemotaxis multiplier is bounded while the principal term is `-lambda_n`. For finitely many low modes, decrease `c` and `delta`; the already-proved strict negativity supplies positive room.

The target smoothing theorem is:

```lean
theorem intervalLinearSemigroup_smoothing
    (hcoercive : LinearizedCoerciveGap p uStar vStar delta c)
    {s theta : ℝ} (htheta : 0 ≤ theta) :
    ∃ Ctheta > 0,
      ∀ {t : ℝ}, 0 < t → ∀ q : ModalHs s,
        ‖intervalLinearSemigroupGain
            p uStar vStar s theta t q‖ ≤
          Ctheta * t ^ (-theta / 2) *
            Real.exp (-(delta / 2) * t) * ‖q‖
```

Here `intervalLinearSemigroupGain ...` has modal multiplier

```text
(1 + lambda_n)^(theta/2) * exp(t*sigma_n).
```

The proof factors half of the exponential for the spectral gap and applies

```lean
F1ProbeFractionalMultiplier.rpow_mul_exp_neg_mul_le
```

to the remaining half. L5 needs only `theta = 1`:

```text
X^(s-1) -> X^s,
‖S(t)‖ <= M * t^(-1/2) * exp(-omega*t).
```

# 5. Exact L4 nonlinear target

The fixed-point proof needs the Lipschitz-quadratic form, not merely `‖N(phi)‖ <= C‖phi‖²`.

Use this record:

```lean
structure IntervalQuadraticRemainderData
    (p : CM2Params) (uStar vStar s : ℝ) where
  N : ModalHs s → ModalHs (s - 1)
  radius : ℝ
  radius_pos : 0 < radius
  K : ℝ
  K_nonneg : 0 ≤ K
  map_zero : N 0 = 0
  lipschitz_quadratic :
    ∀ phi psi : ModalHs s,
      ‖phi‖ ≤ radius → ‖psi‖ ≤ radius →
      ‖N phi - N psi‖ ≤
        K * (‖phi‖ + ‖psi‖) * ‖phi - psi‖
```

Derive immediately:

```lean
theorem IntervalQuadraticRemainderData.norm_le_sq
    (H : IntervalQuadraticRemainderData p uStar vStar s)
    {phi : ModalHs s} (hphi : ‖phi‖ ≤ H.radius) :
    ‖H.N phi‖ ≤ H.K * ‖phi‖^2
```

by applying the difference estimate with `psi = 0`.

## Concrete analytic content of L4

Let `u = uStar + phi` and eliminate `v` by

```text
v - vStar = (mu - d_xx)^(-1)
  [nu * ((uStar + phi)^gamma - uStar^gamma)].
```

Subtract the linear term whose coefficient is the repository's `sigma`. The remainder consists of:

1. the quadratic Taylor remainder of the logistic term;
2. the quadratic Taylor remainder of `u^gamma` inside the resolver;
3. products of `phi`, the resolved perturbation, and its derivative in the chemotaxis flux;
4. denominator Taylor remainders for `(1+v)^(-beta)`.

For `s > 1/2` in one dimension:

```text
X^s embeds in C^0,
X^s is a multiplication algebra,
resolver: X^(s-1) -> X^(s+1),
d_x of the flux loses one derivative.
```

Choose the ball radius so the embedding gives

```text
uStar + phi >= uStar/2 > 0,
1 + vStar + resolverPerturbation >= (1+vStar)/2 > 0.
```

Then real powers are smooth on the relevant compact positive ranges. This proves the record above.

This is the genuinely new PDE lemma. The repository has coefficient embeddings and elliptic multiplier pieces, but no turnkey weighted-`ℓ²` Sobolev algebra/Nemytskii package that proves this whole estimate.

# 6. L5: weighted Duhamel fixed point

## Banach carrier

Use bounded continuous functions with the weight absorbed into the represented trajectory:

```lean
abbrev WeightedOrbit (s : ℝ) :=
  (Set.Ici (0 : ℝ)) →ᵇ ModalHs s
```

The notation `alpha →ᵇ beta` is Mathlib's `BoundedContinuousFunction alpha beta`. It is a complete normed space when the codomain is complete.

Interpret `Z : WeightedOrbit s` as the physical perturbation

```lean
def WeightedOrbit.physical
    (eta : ℝ) (Z : WeightedOrbit s)
    (t : ℝ) (ht : 0 ≤ t) : ModalHs s :=
  Real.exp (-eta * t) • Z ⟨t, ht⟩
```

Then `‖Z‖` is the weighted sup norm

```text
sup_{t>=0} exp(eta*t) * ‖phi(t)‖_s.
```

Construct each Duhamel-map output with

```lean
BoundedContinuousFunction.ofNormedAddCommGroup
```

after proving continuity and the uniform bound.

## Kernel constant

For the one-derivative loss, define

```text
I(omega, eta) = integral_0^infinity
  r^(-1/2) * exp(-(omega-eta)*r) dr
              = sqrt(pi/(omega-eta)).
```

It is enough to prove a local project lemma with a finite upper bound; the exact closed form is optional:

```lean
theorem invSqrt_exp_kernel_integrable
    (heta : eta < omega) :
    IntegrableOn
      (fun r : ℝ => r ^ (-(1 : ℝ) / 2) *
        Real.exp (-(omega - eta) * r))
      (Set.Ioi 0)
```

and then choose an explicit positive `I` dominating its integral.

## Duhamel estimates

For

```text
Phi(phi)(t) = S(t) phi0 + integral_0^t S(t-r) N(phi(r)) dr,
```

L1, L2 with `theta = 1`, and L4 give on the radius-`r` weighted ball:

```text
‖Phi phi‖_Y
  <= M0*‖phi0‖_s + Ms*K*I*‖phi‖_Y^2,

‖Phi phi - Phi psi‖_Y
  <= 2*Ms*K*r*I*‖phi-psi‖_Y.
```

Use the following target theorem:

```lean
theorem weightedDuhamel_fixedPoint
    {s eta omega M0 Ms K I r eps : ℝ}
    (heta : 0 < eta ∧ eta < omega)
    (hr : 0 < r)
    (hlin : LinearDecayBound ...)
    (hsmooth : OneDerivativeSmoothingBound ...)
    (hN : IntervalQuadraticRemainderData ...)
    (hkernel : DuhamelKernelBound omega eta I)
    (hmap : M0 * eps + Ms * K * I * r^2 ≤ r)
    (hcontract : 2 * Ms * K * r * I < 1)
    {phi0 : ModalHs s} (hphi0 : ‖phi0‖ ≤ eps) :
    ∃ phi : WeightedOrbit s,
      Function.IsFixedPt (weightedDuhamelMap ...) phi ∧
      ‖phi‖ ≤ r
```

For the final line, let

```lean
ball := Metric.closedBall (0 : WeightedOrbit s) r
q : ℝ≥0 := ⟨2 * Ms * K * r * I, by positivity⟩
```

prove `MapsTo Phi ball ball`, and prove

```lean
ContractingWith q (MapsTo.restrict Phi ball ball hmaps)
```

Then apply

```lean
ContractingWith.exists_fixedPoint'
```

with `Metric.isClosed_closedBall.isComplete`. This is cleaner than creating another bespoke Banach theorem.

```lean
def ExponentialC1ConvergenceWithAfter
    (t0 : ℝ) ... : Prop :=
  ∀ t, t0 ≤ t →
    N.c1Distance (u t) (fun _ => uStar) +
      N.c1Distance (v t) (fun _ => vStar) ≤
```

Take `t0 = 1` or any fixed positive number. To retain the existing all-`t ≥ 0` target, strengthen the input with a finite/small initial `C¹` norm and use local regularity on `[0,1]` to enlarge `C`.

# 9. From modal decay to physical `C¹`

The fixed point gives

```text
‖phi(t)‖_Xs <= r * exp(-eta*t).
```

To obtain `C¹` for large time, use a fixed positive lag, for example `1/2`. For `t >= 1`, restart the mild identity at `t - 1/2`. L2 gains enough derivatives to reach some `rSpace > 3/2`; the existing derivative reciprocal-trace theorem then gives the `C¹` cosine-series embedding. Because the lag is fixed, no singular factor depends on `t`, and the same exponential rate survives.

For `v`, use the elliptic resolver:

```text
v(t)-vStar = R[nu*((uStar+phi(t))^gamma-uStar^gamma)].
```

The resolver gains two spatial derivatives, so its `C¹` estimate is easier than the one for `u` once the Nemytskii Lipschitz bound is available.
