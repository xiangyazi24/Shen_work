## Direct route details

For a point `t ∈ Set.Ioc 0 δ`, use the strict closed window

```lean
Set.Icc (t / 2) δ
```

because:

- `0 < t / 2`, from `t ∈ Ioc 0 δ`;
- `t / 2 ≤ δ`, since `t / 2 ≤ t ≤ δ`;
- `δ < T`, from `hδ_before`;
- `t ∈ Set.Icc (t / 2) δ`;
- near `t`, the set `Set.Ioc 0 δ` is contained in `Set.Icc (t / 2) δ`.

The last point is the only local-filter step. Use these Mathlib APIs:

- `Ioi_mem_nhds`
- `mem_nhdsWithin_of_mem_nhds`
- `nhdsWithin_inter_of_mem`
- `nhdsWithin_mono`
- `ContinuousWithinAt.mono_left` / equivalently `Tendsto.mono_left`

The key local-filter proof shape is:

```lean
have hnear : Set.Ioi (t / 2) ∈ 𝓝[Set.Ioc (0 : ℝ) δ] t :=
  mem_nhdsWithin_of_mem_nhds
    (Ioi_mem_nhds (by linarith : t / 2 < t))

have hle : 𝓝[Set.Ioc (0 : ℝ) δ] t ≤
    𝓝[Set.Icc (t / 2) δ] t := by
  rw [← nhdsWithin_inter_of_mem hnear]
  exact nhdsWithin_mono t (by
    intro y hy
    -- hy : y ∈ Set.Ioc 0 δ ∩ Set.Ioi (t / 2)
    exact ⟨le_of_lt hy.2, hy.1.2⟩)
```

If the local Mathlib orientation of `nhdsWithin_inter_of_mem` differs, the same proof works by rewriting in the opposite direction. The mathematical filter fact needed is exactly
`𝓝[Ioc 0 δ] t ≤ 𝓝[Icc (t/2) δ] t`.

Endpoint `t = δ` is not special: the chosen strict window is `Icc (δ/2) δ`, and `ContinuousWithinAt` at the right endpoint of the closed interval gives the needed left-sided continuity on `Ioc 0 δ`.

### 1. Generic strict-window-to-`Ioc` continuity bridge

```lean
/-- Strict positive-time closed-window continuity gives continuity on a zero-start
Ioc window. -/
theorem continuousOn_Ioc_of_strictWindow_continuousOn
    {T δ : ℝ} {f : ℝ → ℝ}
    (hδ_pos : 0 < δ) (hδ_before : δ < T)
    (hcont : ∀ {a b : ℝ}, 0 < a → a ≤ b → b < T →
      ContinuousOn f (Set.Icc a b)) :
    ContinuousOn f (Set.Ioc (0 : ℝ) δ) := by
  intro t ht
  have ht0 : 0 < t := ht.1
  have htδ : t ≤ δ := ht.2
  have ha_pos : 0 < t / 2 := half_pos ht0
  have hat : t / 2 ≤ t := by linarith
  have haδ : t / 2 ≤ δ := le_trans hat htδ
  have hclosed : ContinuousOn f (Set.Icc (t / 2) δ) :=
    hcont (a := t / 2) (b := δ) ha_pos haδ hδ_before
  have htIcc : t ∈ Set.Icc (t / 2) δ := ⟨hat, htδ⟩
  have hcw : ContinuousWithinAt f (Set.Icc (t / 2) δ) t :=
    hclosed t htIcc
  have hnear : Set.Ioi (t / 2) ∈ 𝓝[Set.Ioc (0 : ℝ) δ] t :=
    mem_nhdsWithin_of_mem_nhds
      (Ioi_mem_nhds (by linarith : t / 2 < t))
  have hle : 𝓝[Set.Ioc (0 : ℝ) δ] t ≤
      𝓝[Set.Icc (t / 2) δ] t := by
    rw [← nhdsWithin_inter_of_mem hnear]
    exact nhdsWithin_mono t (by
      intro y hy
      exact ⟨le_of_lt hy.2, hy.1.2⟩)
  exact hcw.mono_left hle
```

This is the central lemma. Keep it generic over `f`; do not specialize it to the physical scalar names.

## Why not countable cover?

A countable-cover route is unnecessary and more fragile. It would require covering `Set.Ioc 0 δ` by compact windows such as `Set.Icc (δ / (n + 2)) δ`, then transporting `AEStronglyMeasurable` across a countable union/restriction. That creates avoidable work around real-index coverage and restricted measures. The local `ContinuousOn Ioc` proof above is simpler and aligns with the existing strict-before API.
