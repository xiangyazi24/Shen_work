# Definitive verdict

The cos↔sin divergence bookkeeping is clean and already largely present in the repository:

```text
cosineCoeffs (∂ₓQ) k = (kπ) · sineCoeffs Q k
```

for every positive mode, provided the flux vanishes at both endpoints. The zeroth cosine mode is also exactly zero. Thus taking one spatial divergence costs exactly one power of `k` in coefficient decay.

There is one important correction to the proposed general-order justification:

> `Rₓ(0)=Rₓ(1)=0` by itself proves only `Q(0)=Q(1)=0`. It does **not** by itself prove `Q''(0)=Q''(1)=0`, `Q⁽⁴⁾(0)=Q⁽⁴⁾(1)=0`, and so on.

The higher even-derivative vanishing of `Q` follows from the **full cosine/sine parity** of the global representatives: `u` and `R` are doubly even, `Rₓ` is doubly odd, `(1+R)^{-β}` is doubly even, hence

```text
Q = u · Rₓ · (1+R)^{-β}
```

is doubly odd. Every even derivative of a sufficiently smooth doubly-odd function vanishes at `0` and `1`. That parity statement, rather than the first-order Neumann condition alone, is the correct general-order boundary mechanism.

# 1. Exact divergence identity

Use the repository normalization from

```text
ShenWork/Paper2/IntervalDivergenceModeIdentity.lean
```

For `k > 0`,

```text
sineCoeffs Q k
  = 2 * ∫ x in 0..1, sin(kπx) * Q x,

cosineCoeffs f k
  = 2 * ∫ x in 0..1, cos(kπx) * f x.
```

The file already proves

```lean
rawCosCoeff_deriv_eq_kpi_rawSinCoeff
```

and its normalized form

```lean
cosineCoeffs_deriv_eq_sqrtLambda_sineCoeff
```

with

```lean
sqrt_lam_eq_kpi : Real.sqrt (lam k) = (k : ℝ) * Real.pi.
```

Concretely, if `Q'` is the spatial derivative of `Q`, `Q'` is continuous, and

```text
Q 0 = 0,
Q 1 = 0,
```

then

```text
cosineCoeffs Q' k = sqrt(λ_k) * sineCoeffs Q k
                  = (kπ) * sineCoeffs Q k.
```

The integration by parts is

```text
∫₀¹ Q'(x) cos(kπx) dx
  = [Q(x) cos(kπx)]₀¹
      + kπ ∫₀¹ Q(x) sin(kπx) dx.
```

For the chemotaxis flux

```text
Q(x) = u(x) Rₓ(x) / (1+R(x))^β,
```

Neumann for the resolver gives

```text
Rₓ(0)=Rₓ(1)=0,
```

and hence, with a nonzero denominator,

```text
Q(0)=Q(1)=0.
```

Therefore the boundary term vanishes exactly. This is the no-flux boundary condition in spectral form.

The mode `k=0` is also exact:

```text
cosineCoeffs Q' 0 = ∫₀¹ Q' = Q(1)-Q(0)=0,
sineCoeffs Q 0 = 0.
```

Thus the normalized identity holds for every `k : ℕ`, including zero.

The other repository spelling


has the same definition as `IntervalDivergenceModeIdentity.sineCoeffs`: zero at mode `0`, and `2∫ sin(kπx)Q(x)` at positive modes. A small bridge theorem between the two names should be definitional after unfolding.

# 2. Exact repeated sine-IBP bookkeeping

Fix `k ≥ 1` and abbreviate

```text
a := (k : ℝ) * π > 0.
```

Let

```text
D₀ = Q,
D₁ = Q',
D₂ = Q'',
...
Dᵣ = Q⁽ʳ⁾.
```

The two elementary IBP transitions are:

```text
∫₀¹ D_{2j}(x) sin(ax) dx
  = (1/a) ∫₀¹ D_{2j+1}(x) cos(ax) dx,
```

provided

```text
D_{2j}(0)=D_{2j}(1)=0,
```

and

```text
∫₀¹ D_{2j+1}(x) cos(ax) dx
  = -(1/a) ∫₀¹ D_{2j+2}(x) sin(ax) dx.
```

The second transition needs no boundary hypothesis, because `sin(0)=sin(kπ)=0`.

Consequently, to integrate the sine coefficient `r` times, the exact boundary assumptions are

```text
Q⁽²ʲ⁾(0)=Q⁽²ʲ⁾(1)=0
for every j with 2j < r.
```

Equivalently:

```text
r = 1 : Q vanishes;
r = 2 : Q vanishes;
r = 3 : Q and Q'' vanish;
r = 4 : Q and Q'' vanish;
r = 5 : Q, Q'', Q⁽⁴⁾ vanish;
...
```

The exact formulas are as follows.

For `r = 2m`,

```text
sineCoeffs Q k
  = 2 * (-1)^m / a^(2m)
      * ∫₀¹ sin(ax) Q⁽²ᵐ⁾(x) dx.
```

For `r = 2m+1`,

```text
sineCoeffs Q k
  = 2 * (-1)^m / a^(2m+1)
      * ∫₀¹ cos(ax) Q⁽²ᵐ⁺¹⁾(x) dx.
```

These formulas can also be written uniformly with the phase-shifted factor

```text
sin(ax + rπ/2),
```

but the even/odd split is easier to formalize because it reuses two simple IBP lemmas and avoids phase arithmetic.

If

```text
∫₀¹ |Q⁽ʳ⁾(x)| dx ≤ B,
```

then `|sin|,|cos| ≤ 1` gives the exact normalized estimate

```text
|sineCoeffs Q k| ≤ 2B / (kπ)^r.
```

If instead

```text
|Q⁽ʳ⁾(x)| ≤ M    on [0,1],
```

then the interval has length `1`, so one may take `B=M` and obtain

```text
|sineCoeffs Q k| ≤ 2M / (kπ)^r.
```

Combining this with the divergence identity gives

```text
|cosineCoeffs Q' k|
  = (kπ) |sineCoeffs Q k|
  ≤ 2M / (kπ)^(r-1).
```

In the ladder's `C/k^(r-1)` spelling, choose

```text
C = 2M / π^(r-1).
```

Therefore the divergence is exactly a one-power loss:

```text
sine flux decay k^{-r}
  ⟹ cosine divergence decay k^{-(r-1)}.
```

If the PDE source is `-χ₀ ∂ₓQ`, multiply the final constant by `|χ₀|`.

# 3. The higher boundary conditions: what is true and what is not

## First-order Neumann is sufficient for the first identity

The condition

```text
Rₓ(0)=Rₓ(1)=0
```

is enough to prove

```text
Q(0)=Q(1)=0,
```

and therefore enough for

```text
cosineCoeffs (∂ₓQ) k = kπ sineCoeffs Q k.
```

## First-order Neumann is not sufficient for the whole tower

It does not imply `Q''=0` at the endpoints. For example, take

```text
u = 1,
β = 0,
R'(x) = x²(1-x)².
```

Then `R'(0)=R'(1)=0`, so `R` satisfies the first Neumann condition and `Q=R'` vanishes at both endpoints, but

```text
Q''(0)=2,
Q''(1)=2.
```

Thus the statement “all even derivatives of `Q` vanish because `Rₓ` vanishes” is false without additional compatibility.

## The correct general-order mechanism is double parity

The repository already has the relevant even-parity framework in

```text
ShenWork/Paper2/IntervalSourceRepresentative.lean
```

with

```lean
EvenAboutZero
EvenAboutOne
DoublyEven
```

and proves that all odd derivatives of a doubly-even function vanish at both endpoints.

For the flux side, add the mirror definitions

```lean
def OddAboutZero (f : ℝ → ℝ) : Prop :=
  ∀ x, f (-x) = -f x

def OddAboutOne (f : ℝ → ℝ) : Prop :=
  ∀ x, f (2 - x) = -f x

structure DoublyOdd (f : ℝ → ℝ) : Prop where
  about0 : OddAboutZero f
  about1 : OddAboutOne f
```

Then prove the mirror parity facts:

```text
DoublyEven R          ⟹ DoublyOdd Rₓ,
DoublyEven R          ⟹ DoublyEven ((1+R)^(-β)),
DoublyEven u
  ∧ DoublyOdd Rₓ
  ∧ DoublyEven ((1+R)^(-β))
                      ⟹ DoublyOdd Q.
```

The first fact can be proved either by `iteratedDeriv_comp_neg` / `iteratedDeriv_comp_const_sub`, or directly from the resolver-gradient sine series. Each mode

```text
-kπ sin(kπx)
```

is odd about `0` and odd about `1`, so the whole resolver-gradient series is doubly odd termwise.

For a sufficiently smooth doubly-odd `Q`, every even iterated derivative remains doubly odd and therefore vanishes at the two fixed endpoints:

```text
Even n ⟹ iteratedDeriv n Q 0 = 0,
Even n ⟹ iteratedDeriv n Q 1 = 0.
```

This gives exactly

```text
Q⁽²ʲ⁾(0)=Q⁽²ʲ⁾(1)=0
```

for every even order required by the repeated sine IBP.

So the correct implication is

```text
cosine parity of u and R
  + sufficient spatial regularity
  ⟹ doubly-odd parity of Q
  ⟹ all even flux derivatives vanish at the endpoints.
```

It is not merely

```text
Rₓ(0)=Rₓ(1)=0 ⟹ all higher flux boundary conditions.
```

## 4.1 One new raw sine-step lemma

The repository already has the cosine-step lemma


which states, schematically,

```text
∫ cos(kπx) f(x) dx
  = -(1/(kπ)) ∫ sin(kπx) f'(x) dx.
```

Add the complementary sine-step lemma:

```lean
theorem rawSinCoeff_eq_inv_kpi_rawCosCoeff_deriv
    {f f' : ℝ → ℝ} {k : ℕ} (hk : 1 ≤ k)
    (hf : ∀ x ∈ Set.uIcc (0 : ℝ) 1, HasDerivAt f (f' x) x)
    (hf'int : IntervalIntegrable f' volume 0 1)
    (hf0 : f 0 = 0) (hf1 : f 1 = 0) :
    (∫ x in (0 : ℝ)..1,
        Real.sin ((k : ℝ) * Real.pi * x) * f x)
      = (1 / ((k : ℝ) * Real.pi)) *
          ∫ x in (0 : ℝ)..1,
            Real.cos ((k : ℝ) * Real.pi * x) * f' x
```

This need not be reproved from scratch: commute the products and solve the existing theorem


for the sine integral, using `kπ ≠ 0`.

## 4.2 Use a derivative-chain structure for the induction core

The most robust generic IBP theorem should not initially depend on the subtleties of `iteratedDeriv` at endpoints. Package explicit derivative representatives:

```lean
structure SineIBPChain (r : ℕ) (D : ℕ → ℝ → ℝ) : Prop where
  deriv : ∀ j, j < r → ∀ x ∈ Set.uIcc (0 : ℝ) 1,
    HasDerivAt (D j) (D (j+1) x) x
  integrable : ∀ j, j ≤ r → IntervalIntegrable (D j) volume 0 1
  even_boundary : ∀ j, 2*j < r →
    D (2*j) 0 = 0 ∧ D (2*j) 1 = 0
```

Then prove, by induction on `m`, the two exact formulas for `r=2m` and `r=2m+1`. Each induction step is just:

```text
raw sine step with the even boundary condition,
then existing raw cosine step with automatic sine boundary cancellation.
```

After that, add a convenience wrapper with

```text
D j = iteratedDeriv j Q
```

under a global `ContDiff ℝ r Q` hypothesis and the even-boundary tower.

## 4.3 The normalized decay theorem

A consumer-shaped theorem should be:

```lean
theorem sineCoeffs_iterated_decay_of_even_boundary
    {Q : ℝ → ℝ} {r k : ℕ} {M : ℝ}
    (hr : 1 ≤ r) (hk : 1 ≤ k)
    (hQ : ContDiff ℝ r Q)
    (hbc : ∀ j, 2*j < r →
      iteratedDeriv (2*j) Q 0 = 0 ∧
      iteratedDeriv (2*j) Q 1 = 0)
    (hM0 : 0 ≤ M)
    (hM : ∀ x ∈ Set.Icc (0 : ℝ) 1,
      |iteratedDeriv r Q x| ≤ M) :
    |IntervalDivergenceModeIdentity.sineCoeffs Q k|
      ≤ 2 * M / (((k : ℝ) * Real.pi) ^ r)
```

An `L¹`-top-derivative version is even more primitive and often easier:

```text
∫₀¹ |Q⁽ʳ⁾| ≤ B
  ⟹ |sineCoeffs Q k| ≤ 2B/(kπ)^r.
```

The sup-norm theorem then follows because the interval has measure `1`.

# 5. Important implementation detail: use a global parity representative

Do not formulate the higher derivative tower directly on the zero-extended

```text
intervalDomainLift
```

unless all endpoint matching has already been proved. The zero extension can create endpoint discontinuities, and Mathlib's `deriv` then returns junk value `0`; this can make a false boundary argument appear to close.

Instead:

1. construct the global cosine representative for `u` and `R`;
2. construct the global sine/odd representative for `Rₓ` and hence for `Q`;
3. prove `DoublyOdd Qrep`;
4. prove the repeated IBP and decay for `Qrep`;
5. transfer the interval integrals using `EqOn` on `[0,1]`.

This matches the repository's existing `DoublyEven` strategy and makes every endpoint statement genuine.

# Final bookkeeping

The definitive chain is

```text
u,R are smooth Neumann cosine representatives
  ⟹ u,R,(1+R)^(-β) are doubly even
  ⟹ Rₓ is doubly odd
  ⟹ Q = u·Rₓ·(1+R)^(-β) is doubly odd
  ⟹ Q⁽²ʲ⁾(0)=Q⁽²ʲ⁾(1)=0 for all needed j
  ⟹ |sineCoeffs Q k| ≤ 2M/(kπ)^r
  ⟹ cosineCoeffs(∂ₓQ,k) = kπ·sineCoeffs(Q,k)
  ⟹ |cosineCoeffs(∂ₓQ,k)| ≤ 2M/(kπ)^(r-1).
```

So yes: the chemotaxis divergence contributes exactly the `−1` step in the coefficient ladder. The analytic work is an elementary alternating sine/cosine IBP induction plus a parity lemma for the flux. No abstract PDE regularity theorem is involved.
