# Whole-line Cauchy uniqueness: the smallest faithful class

## Executive verdict

The current whole-line predicate is too weak for a sound uniqueness theorem, but the smallest faithful correction is **not** to impose an arbitrary Gaussian-growth condition.

For the intended chemotaxis problem, the standard phase space is:

```text
u ∈ C([0,T]; BUC(ℝ))  (or at least locally-in-time L∞ in x),
v(t) = (β₀ - μ∂ₓₓ)⁻¹(u(t)^γ) via the canonical Bessel kernel,
```

with either a mild Duhamel identity or enough genuine `C^{1,2}`/FTC regularity to derive it. Here `β₀, μ > 0`; the normalized Paper 1 equation has `β₀ = μ = 1`.

A symmetric exponentially decaying weighted norm

```text
∫ℝ |u₁-u₂|² e^{-κ|x|} dx
```

is an excellent **energy used to prove uniqueness**, but it is not by itself an adequate nonlinear solution class. The same-weight space is not closed under `u ↦ u^γ` or the product `u vₓ`; an unweighted local `L∞` bound, or comparably strong coefficient bounds, is still needed.

The shortest sound Lean architecture is therefore:

1. add local-in-time spatial boundedness to the Cauchy solution class;
2. require the signal to be the canonical whole-line resolvent, not merely an arbitrary solution of the elliptic ODE;
3. preferably formulate the solution by a BUC mild identity and prove uniqueness by a short singular-kernel contraction;
4. retain the symmetric weighted-`L²` energy theorem as a useful secondary uniqueness result.

This correction is non-vacuous for the traveling-wave stability branch. Its initial data are already bounded uniformly continuous, the wave profiles are bounded, and the intended Cauchy solutions are globally bounded. Every bounded slice automatically belongs to every symmetric exponentially weighted `L²` space.

and:


This stores no spatial growth condition and no local-in-time spatial bound. It also uses Mathlib's total functions `deriv` and `iteratedDeriv`; the type does not itself provide the continuity, second differentiability, weighted integrability, cutoff convergence, or FTC facts needed by an energy proof.

In `ShenWork/Paper1/Statements.lean`:

The uniform initial trace is valuable, but it does not replace a growth class on every positive-time slab. In the linear heat equation, a delayed Tychonoff solution can be identically zero up to a positive time and then branch: the usual construction is flat to every order at its starting time, so shifting it and extending by zero preserves the initial trace at `t=0`.

There is a second, independent omission. The condition

```text
β₀ v - μ vₓₓ = u^γ
```

on `ℝ` does not by itself select a unique signal. The homogeneous equation has the modes

```text
exp( sqrt(β₀/μ) x),   exp(-sqrt(β₀/μ) x).
```

The chemotaxis model means the **canonical Bessel resolvent**. The formal class should state that equality explicitly, or impose a bounded/decaying elliptic class from which it follows. Merely storing the pointwise elliptic differential equation is not faithful enough.

Finally, the existing stability weight is different from the one useful for uniqueness:

```lean
def WeightedL2InitialCloseness (η : ℝ) (u₀ U : ℝ → ℝ) : Prop :=
  Integrable (fun x => Real.exp (2 * η * x) * |u₀ x - U x| ^ 2)
```

This one-sided increasing weight is designed for a moving front. It is not an integrable whole-line cutoff and should not be reused as the generic uniqueness weight.

# 2. Tychonoff: what it does and does not show

## 2.1 Linear heat equation

For the heat equation on `ℝ`, pointwise classical solutions with zero initial data are not unique without a spatial-growth restriction. Tychonoff's construction gives nonzero smooth solutions that are flat at the initial time and grow extremely rapidly in space.

The standard classical uniqueness theorem is the **at-most Gaussian class**: on every finite strip, a bound of the form

```text
|u(t,x)| ≤ C_T exp(a_T |x|²)
```

restores uniqueness. Nontrivial zero-data Tychonoff solutions necessarily violate every fixed uniform Gaussian bound on the strip. Thus the phrase “the counterexample is approximately `exp(a x²)`” must be handled carefully: at-most Gaussian growth is on the uniqueness side of the theorem, not the counterexample side.

A convenient primary reference is Qian Lin, *The Tychonoff uniqueness theorem for the G-heat equation*, which recalls the classical theorem and its no-growth counterexamples:

- <https://arxiv.org/abs/1006.5300>

## 2.2 Relation to the nonlinear chemotaxis system

This does **not** mean that an arbitrary Tychonoff heat solution automatically solves the full nonlinear chemotaxis-logistic equation. It does not. An exact nonlinear nonuniqueness counterexample would require a separate construction.

What Tychonoff proves for the formalization is the following narrower but decisive point:

> A whole-line pointwise classical PDE predicate cannot obtain uniqueness merely from the diffusion equation and an initial trace; the solution class must enter the theorem.

For this chemotaxis problem the at-most Gaussian class is not the right class anyway. The Bessel kernel decays only exponentially, so convolution with a genuinely Gaussian-growing source `exp(a y²)` diverges. The canonical elliptic signal may not even be defined in that class.

## 2.3 Comparison of the proposed classes

| Class | Heat uniqueness | Compatible with canonical elliptic signal | Natural for the intended nonlinear orbit | Recommendation |
|---|---:|---:|---:|---|
| Locally-in-time spatial `L∞` / `BUC` | Yes | Yes | Yes | **Use this** |
| At-most Gaussian `exp(a|x|²)` | Classical Tychonoff class | Generally no: Bessel convolution can diverge | No | Linear-theory benchmark only |
| Symmetric exponentially weighted `L²` | Yes with an energy/FTC framework | Yes for differences under coefficient bounds | Useful metric, but nonlinear class incomplete | Use together with `L∞` |
| Polynomial growth | Yes | Yes | Stronger/weirder than needed for bounded fronts | Sound but not minimal |

There is no useful single “sharp” growth class for the full nonlinear nonlocal problem. The sharp classical statement belongs to the linear heat equation. For the chemotaxis system, bounded BUC mild solutions are the standard and shortest faithful class.

# 3. A complete weighted-energy uniqueness estimate

The following calculation answers exactly what hypotheses are needed if the formalization chooses the energy route.

## 3.1 Signal kernel estimates

Write the elliptic equation as

```text
(β₀ - μ∂ₓₓ)v = u^γ,       β₀>0, μ>0.
```

Set

```text
λ = sqrt(β₀/μ).
```

The canonical kernel is

```text
K(x) = 1/(2 sqrt(β₀ μ)) · exp(-λ|x|),
v = K * u^γ.
```

Its elementary norms are

```text
‖K‖₁  = 1/β₀,
‖K'‖₁ = 1/sqrt(β₀ μ).
```

Hence, if `0 ≤ u ≤ M`,

```text
‖v‖∞  ≤ M^γ/β₀,
‖vₓ‖∞ ≤ M^γ/sqrt(β₀ μ).
```

Choose a symmetric positive weight `ψκ` satisfying

```text
∫ℝ ψκ < ∞,
|ψκ'| ≤ κ ψκ,
ψκ(x)/ψκ(y) ≤ exp(κ|x-y|).
```

The literal `exp(-κ|x|)` is adequate in weak calculations. For Lean, a smooth comparable choice is

```text
ψκ(x) = exp(-κ sqrt(1+x²)).
```

The map `x ↦ sqrt(1+x²)` is `1`-Lipschitz, giving the ratio estimate, and `|ψκ'|≤κψκ`.

For `0 < κ < 2λ`, weighted Young convolution gives

```text
‖K' * h‖_{L²(ψκ)} ≤ Cκ ‖h‖_{L²(ψκ)},
```

where one may take

```text
Cκ = ∫ℝ |K'(s)| exp(κ|s|/2) ds
   = 1 / ( μ (λ - κ/2) ).
```

Proof: multiply the convolution by `sqrt(ψκ(x))`, use

```text
sqrt(ψκ(x)/ψκ(y)) ≤ exp(κ|x-y|/2),
```

and apply ordinary `L¹ * L² → L²` Young to

```text
|K'(s)| exp(κ|s|/2).
```

For shortest reuse of the repository's existing `ExponentialWeight` kernel-transfer lemmas, the stricter but harmless condition `κ < λ` can be imposed.

## 3.2 Difference of two bounded solutions

Fix a finite horizon `T`. Let `(u₁,v₁)` and `(u₂,v₂)` have the same initial datum and suppose

```text
0 ≤ uᵢ(t,x) ≤ M_T      for 0≤t≤T, x∈ℝ,
vᵢ(t) = K * (uᵢ(t)^γ).
```

Enlarge `M_T` so that `M_T ≥ 1`. Put

```text
w = u₁-u₂,
z = v₁-v₂.
```

On `[0,M_T]`, the power map is Lipschitz. One may take

```text
Lγ(T) = γ M_T^(γ-1).
```

Therefore

```text
‖zₓ‖_{L²(ψκ)}
  ≤ Cκ Lγ(T) ‖w‖_{L²(ψκ)}.
```

For the `m=1` flux,

```text
u₁ v₁ₓ - u₂ v₂ₓ = w v₁ₓ + u₂ zₓ.
```

Consequently

```text
‖u₁ v₁ₓ-u₂ v₂ₓ‖_{L²(ψκ)}
  ≤ A_T ‖w‖_{L²(ψκ)},
```

with

```text
A_T = M_T^γ/sqrt(β₀ μ) + M_T Cκ Lγ(T).
```

For general `m≥1`, replace the first term by

```text
m M_T^(m-1) · M_T^γ/sqrt(β₀ μ)
```

and the second by

```text
M_T^m Cκ Lγ(T).
```

## 3.3 Energy identity and exact inequality

Define

```text
Y(t) = ∫ℝ w(t,x)² ψκ(x) dx,
X(t) = ∫ℝ wₓ(t,x)² ψκ(x) dx.
```

The difference equation is

```text
w_t = wₓₓ
      - χ ∂ₓ(w v₁ₓ + u₂ zₓ)
      + r(u₁)-r(u₂).
```

Testing by `w ψκ` gives the following estimates.

### Diffusion

```text
∫ wₓₓ w ψκ
  = -X - ∫ w wₓ ψκ'
  ≤ -3/4 X + κ² Y.
```

Here

```text
κ sqrt(XY) ≤ 1/4 X + κ²Y.
```

### Chemotaxis transport

Let `F = w v₁ₓ+u₂zₓ`. Then

```text
|χ ∫ F (wₓ ψκ+w ψκ')|
  ≤ |χ| A_T sqrt(Y) (sqrt(X)+κ sqrt(Y))
  ≤ 1/4 X + (χ² A_T²+|χ|κA_T)Y.
```

### Reaction

For the Paper 1 logistic term

```text
r(s)=s(1-s^α),        s≥0,
```

one has the sharper one-sided estimate

```text
(r(u₁)-r(u₂))w
  = w² - (u₁^(α+1)-u₂^(α+1))w
  ≤ w²,
```

because `s ↦ s^(α+1)` is monotone. For a general reaction, replace `1` below by a Lipschitz constant of `r` on `[0,M_T]`.

Combining the three terms gives

```text
1/2 Y'(t) + 1/2 X(t)
  ≤ C_T Y(t),
```

where

```text
C_T = κ² + χ² A_T² + |χ|κA_T + 1.
```

Dropping `X` and applying Gronwall yields

```text
Y(t) ≤ Y(0) exp(2 C_T t).
```

The two solutions have the same trace, so `Y(0)=0`; hence `Y≡0`, `u₁=u₂`, and canonical resolver equality gives `v₁=v₂`.

## 3.4 Exact regularity needed to justify the calculation

Local boundedness alone does not justify the displayed differentiation and integration by parts. A rigorous energy theorem needs, on every finite horizon, one of the following equivalent styles of input.

### Weak-energy style

```text
w ∈ C([0,T]; L²(ψκ)),
w ∈ L²(0,T; H¹(ψκ)),
w_t ∈ L²(0,T; H⁻¹(ψκ)),
```

plus the weighted weak equation. Then `Y` is absolutely continuous and the energy identity is legitimate.

### Classical cutoff style

Use smooth cutoffs `χ_R`, test by `w ψκ χ_R²`, prove the finite-interval identity, and let `R→∞`. One needs enough genuine joint regularity and domination to pass to the limit. Pointwise `DifferentiableAt` plus total `deriv` values is not enough.

### Positive-time restart style

For BUC mild solutions, parabolic smoothing gives the weighted `H¹` facts on every `[τ,T]`, `τ>0`. Prove the inequality from `τ`, then let `τ↓0`. The existing `HasUniformInitialTrace` gives

```text
sup_x |u₁(τ,x)-u₂(τ,x)| → 0,
```

and, since `ψκ` is integrable,

```text
Y(τ) ≤ ‖u₁(τ)-u₂(τ)‖∞² ∫ψκ → 0.
```

This is the most faithful way to avoid placing unnecessary derivative assumptions at `t=0`.

# 4. Why weighted `L²+H¹` alone is not the right class

Suppose only

```text
∫ (u²+uₓ²)e^{-κ|x|} < ∞.
```

This does not imply an unweighted `L∞` bound. After smoothing near the origin, the function

```text
u(x)=exp(κ|x|/4)
```

belongs to that weighted `H¹` space but is unbounded.

The nonlinear defects are immediate:

- for `γ>1`, `u^γ` need not lie in the same weighted space;
- multiplication by `u` is not a bounded operator on `L²(e^{-κ|x|})`;
- in the example above, `u⁴ e^{-κ|x|}` is nonintegrable, so even simple quadratic products leave the space;
- the estimate of `u₂ zₓ` cannot be closed by the same weighted norm.

Therefore the statement

```text
weighted L² + weighted derivative control alone implies uniqueness
```

is not a sound general theorem for this chemotaxis nonlinearity. It can be repaired by adding a hierarchy of stronger weights and exponent-dependent integrability, but that is a larger and less paper-faithful Lean development than a local `L∞` bound.

The correct division of labor is:

```text
local L∞ / BUC       = nonlinear admissibility and coefficient control;
symmetric weighted L² = uniqueness energy and whole-line cutoff control.
```

One may replace `u∈L∞` by the exact coefficient assumptions used above, namely boundedness of `v₁ₓ`, boundedness of the multiplier `u₂`, and a weighted Lipschitz estimate for `u↦∂ₓR(u^γ)`. For the intended nonnegative power-law model, local `L∞` is by far the simplest way to produce all three.

# 5. The shorter canonical route: uniqueness of bounded mild solutions

For Lean, the weighted energy proof is mathematically clean but has a substantial cutoff/FTC surface. If a mild identity is available, uniqueness is shorter.

Let `S(t)=e^{t∂ₓₓ}`. A bounded mild solution satisfies

```text
u(t) = S(t)u₀
  - χ ∫₀ᵗ ∂ₓS(t-s)
      [u(s)^m ∂ₓR(u(s)^γ)] ds
  + ∫₀ᵗ S(t-s) r(u(s)) ds.
```

On an `L∞` ball of radius `M_T`, the map

```text
u ↦ u^m ∂ₓR(u^γ)
```

is Lipschitz in `L∞`, by the same Bessel-kernel estimates above. If

```text
d(t)=‖u₁(t)-u₂(t)‖∞,
```

then the heat-kernel bounds give

```text
d(t) ≤ C_T ∫₀ᵗ ((t-s)^(-1/2)+1)d(s) ds.
```

Let `D(t)=sup_{0≤s≤t}d(s)`. Then

```text
D(t) ≤ C_T(2sqrt(t)+t)D(t).
```

Choose `t_* > 0` with

```text
C_T(2sqrt(t_*)+t_*) < 1.
```

It follows that `D(t_*)=0`. Restart the same argument finitely many times to cover `[0,T]`.

This route uses no weighted differentiation, no spatial boundary terms, and no generalized singular Gronwall theorem. The repository already has the relevant heat-kernel foundation in:

```text
ShenWork/PDE/HeatSemigroup.lean
  heatKernel_integral_eq_one
  heatKernel_deriv_integrable
  heatKernel_deriv_abs_integrable
```

and whole-line resolvent-weight infrastructure in:

```text
ShenWork/PDE/ResolventEstimate.lean
  wholeLineResolventKernel
  weightedResolventKernelEnvelope_integrable
  weightedResolventKernelEnvelope_integral_eq

ShenWork/Paper1/Lemma25Helpers.lean
  kernel_weight_integral_le_psi
```

The mild route is the recommended public uniqueness theorem. The weighted energy route can remain an internal theorem supporting stability calculations.

# 6. Minimal Lean API

## 6.1 Spatial boundedness

```lean
def LocallyUniformlyBoundedInSpaceTime
    (u : ℝ → ℝ → ℝ) : Prop :=
  ∀ T > 0, ∃ M ≥ 0,
    ∀ t ∈ Set.Icc (0 : ℝ) T, ∀ x : ℝ, |u t x| ≤ M
```

A one-sided nonnegative version `0 ≤ u t x ∧ u t x ≤ M` is even more convenient for real powers.

## 6.2 Canonical signal

Use the existing whole-line frozen elliptic resolver rather than only `pde_v`:

```lean
def IsCanonicalEllipticSignal
    (p : CMParams) (u v : ℝ → ℝ → ℝ) : Prop :=
  ∀ t > 0, v t = frozenElliptic p (u t)
```

If the exact repository normalization of `frozenElliptic` differs, define the equality through the existing `Psi` Bessel-kernel operator.

## 6.3 Preferred admissible Cauchy solution

```lean
def IsAdmissibleGlobalCauchySolutionFrom
    (p : CMParams) (u₀ : ℝ → ℝ)
    (u v : ℝ → ℝ → ℝ) : Prop :=
  IsGlobalCauchySolutionFrom p u₀ u v ∧
  LocallyUniformlyBoundedInSpaceTime u ∧
  IsCanonicalEllipticSignal p u v ∧
  WholeLineMildDuhamelIdentity p u₀ u
```

The last field should be a genuine equation, not a uniqueness assumption. If a theorem derives the mild identity from a strengthened `C^{1,2}` classical predicate, it need not be stored twice.

The public result should have the shape:

```lean
theorem wholeLine_globalCauchy_unique
    (h₁ : IsAdmissibleGlobalCauchySolutionFrom p u₀ u₁ v₁)
    (h₂ : IsAdmissibleGlobalCauchySolutionFrom p u₀ u₂ v₂) :
    u₁ = u₂ ∧ v₁ = v₂
```

# 7. Non-vacuity for the traveling-wave-limit branch

The correction does not exclude the intended solution.

1. `NonnegativeInitialDatum` already contains `IsCUnifBdd`, so the Paper 1 initial datum is bounded and continuous.
2. The traveling-wave profiles have finite endpoint limits and the repository's explicit `ShenUpperBoundNegative`/`ShenUpperBoundPositive` bounds, hence are bounded.
3. The intended whole-line existence theorem is precisely a globally bounded Cauchy theory from bounded uniformly continuous data; this is standard in the chemotaxis literature. For example, Salako–Shen's whole-space parabolic–elliptic theory states existence of a unique globally bounded classical solution for bounded uniformly continuous nonnegative data: <https://arxiv.org/abs/1609.05387>.
4. A bounded slice automatically lies in every symmetric exponential weighted space:

   ```text
   ∫ℝ |u(t,x)|² e^{-κ|x|} dx
     ≤ ‖u(t)‖∞² · 2/κ.
   ```

5. Positive-time weighted derivative integrability follows from the parabolic smoothing that a genuine mild construction supplies. It should be proved on `[τ,T]`, not inserted at `t=0` for a merely BUC datum.

There is one formalization obligation: the producer for the mainline Cauchy solution must return the added bounded/resolver/mild fields. One must not argue circularly that eventual convergence implies boundedness if uniqueness is already needed to identify the constructed solution. The construction itself should carry its a-priori upper bound.

# 8. False shortcuts to avoid

- **Do not say a Tychonoff counterexample is bounded by a fixed `exp(a x²)` on a finite strip.** Such a bound is a uniqueness hypothesis.
- **Do not claim that a heat-equation Tychonoff function automatically solves the nonlinear chemotaxis equation.** It is a warning about the unrestricted principal-part class, not an instant nonlinear counterexample.
- **Do not identify an arbitrary elliptic ODE solution with the Bessel resolvent.** Whole-line homogeneous modes must be excluded.
- **Do not use the front weight `exp(2ηx)` as the symmetric uniqueness cutoff.** It is a different, one-sided stability topology.
- **Do not expect same-weight `H¹` to control nonlinear products.** The decaying weight permits unbounded functions.
- **Do not derive weighted FTC/IBP from the current `DifferentiableAt` fields.** Strengthen regularity or use the mild equation.
- **Do not add global-in-time one-constant boundedness unless the theorem needs it.** Local-in-time spatial boundedness on every finite horizon is enough for uniqueness and is the minimal standard condition.

# Bottom line

The smallest faithful correction is:

```text
bounded/BUC on each finite time slab
+ canonical elliptic resolver
+ mild Duhamel identity (preferred),
```

with a symmetric exponential weighted `L²` calculation available as a derived energy theorem. Weighted `L²+H¹` alone is not a closed nonlinear class. This correction is standard, non-vacuous for the traveling-wave branch, and substantially shorter to formalize than a “sharp” Tychonoff-Gaussian theory.
