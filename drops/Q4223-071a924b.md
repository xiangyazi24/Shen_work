# Q4223 — truncated chemotaxis nonnegativity by negative-part energy

## Verdict

Yes: with the **truncated flux using `u₊`** and the resolver driven by **`u₊^γ`**, the standard negative-part energy proof is clean and non-circular. The chemotaxis term does **not** require a favorable sign of `χ₀`; it vanishes in the `u₋` energy because the flux contains the factor `u₊`, while the weak gradient of `u₋` is supported on `{u < 0}`. The only growth contribution is the linear logistic part on the negative set, giving

```text
d/dt (1/2 ∫ u₋^2) ≤ a ∫ u₋^2.
```

Then Gronwall and `u₋(0)=0` imply `u₋(t)=0` for all times, hence `u ≥ 0` a.e. and, with continuous slices, pointwise on `[0,1]`.

The proof is formalizable, but the load-bearing facts are the Sobolev/weak chain rules for positive/negative parts and the Hilbert-triple chain rule for the energy. If the current Lean development only has classical pointwise differentiability, it may be easier to prove this first for the smooth Picard iterates and then pass to the limit, or to introduce the needed negative-part weak chain lemmas explicitly.

## Notation

Let

```text
u₊ = max(u, 0)
u₋ = max(-u, 0)
```

so

```text
u = u₊ - u₋,
u₊ u₋ = 0,
u₊ = 0 on {u ≤ 0},
u₋ = 0 on {u ≥ 0}.
```

The truncated equation is

```text
u_t = ν u_xx - χ₀ ∂x F[u] + u (a - b u₊^α),
```

where

```text
F[u] = u₊ R_x[u₊] / (1 + R[u₊])^β,
(μ - ν ∂xx) R[u₊] = ν u₊^γ,
∂x R = 0 at x = 0,1.
```

Since the resolver source is `ν u₊^γ ≥ 0`, the positive resolvent gives

```text
R[u₊] ≥ 0,
1 + R[u₊] ≥ 1,
```

so the denominator is harmless and the flux is well-defined.

## Weak formulation and test function

Use the weak form: for each test function `φ ∈ H¹(0,1)`,

```text
⟨u_t, φ⟩
  = -ν ∫ u_x φ_x
    + χ₀ ∫ F[u] φ_x
    + ∫ u (a - b u₊^α) φ.
```

This sign convention comes from integrating

```text
ν u_xx - χ₀ ∂x F
```

against `φ` and using the Neumann/no-flux boundary conditions.

Take

```text
φ = -u₋.
```

The Hilbert-triple negative-part chain rule gives

```text
⟨u_t, -u₋⟩ = d/dt (1/2 ∫ u₋^2)
```

for a.e. time. Therefore, for a.e. `t`,

```text
d/dt (1/2 ∫ u₋^2)
  = -ν ∫ u_x (-u₋)_x
    + χ₀ ∫ F[u] (-u₋)_x
    + ∫ u (a - b u₊^α) (-u₋).
```

Now handle the three terms.

## 1. Diffusion term

The weak chain rule for the negative part gives

```text
∂x u₋ = -1_{u<0} ∂x u   a.e.
```

Equivalently,

```text
u_x = -∂x u₋   a.e. on {u < 0},
∂x u₋ = 0      a.e. on {u ≥ 0}.
```

Thus

```text
-ν ∫ u_x (-u₋)_x
  = ν ∫ u_x (u₋)_x
  = -ν ∫ |(u₋)_x|^2
  ≤ 0.
```

This is the coercive dissipation.

## 2. Chemotaxis term vanishes

The chemotaxis contribution is

```text
χ₀ ∫ F[u] (-u₋)_x
  = -χ₀ ∫ F[u] (u₋)_x.
```

Since

```text
F[u] = u₊ · R_x[u₊] · (1 + R[u₊])^{-β},
```

it is enough to show

```text
u₊ · (u₋)_x = 0   a.e.
```

This follows from the Sobolev chain rule:

```text
(u₋)_x = -1_{u<0} u_x   a.e.
```

On `{u < 0}`, `u₊ = 0`. On `{u ≥ 0}`, `(u₋)_x = 0` a.e. Hence

```text
F[u] (u₋)_x =
  u₊ · R_x[u₊] · (1 + R[u₊])^{-β} · (u₋)_x
  = 0
```

a.e., assuming the multiplier

```text
R_x[u₊] · (1 + R[u₊])^{-β}
```

is measurable and finite. The resolver positivity `1+R ≥ 1` supplies finiteness of the denominator.

Therefore

```text
χ₀ ∫ F[u] (-u₋)_x = 0.
```

The sign `χ₀ < 0` is not used for this cancellation; the truncation factor `u₊` is what matters.

## 3. Reaction term

The reaction contribution is

```text
∫ u (a - b u₊^α) (-u₋).
```

Use the decomposition by sign.

On `{u ≥ 0}`, `u₋ = 0`, so the integrand is zero.

On `{u < 0}`,

```text
u = -u₋,
u₊ = 0,
u₊^α = 0      since α > 0,
```

so

```text
u (a - b u₊^α) (-u₋)
  = (-u₋) · a · (-u₋)
  = a u₋^2.
```

Equivalently, a.e. on all of `(0,1)`,

```text
u (a - b u₊^α) (-u₋) = a u₋^2.
```

If the model parameter satisfies `a ≥ 0`, then

```text
∫ u (a - b u₊^α) (-u₋) = a ∫ u₋^2.
```

If one wants a version allowing arbitrary real `a`, replace the right-hand side by `max a 0 * ∫ u₋^2`; for the logistic model, `a ≥ 0` is the usual hypothesis.

## Energy inequality

Putting the terms together gives, for a.e. `t`,

```text
d/dt (1/2 ∫ u₋(t,x)^2 dx)
  = -ν ∫ |∂x u₋|^2 dx
    + 0
    + a ∫ u₋^2 dx
  ≤ a ∫ u₋^2 dx.
```

Let

```text
E(t) = 1/2 ∫ u₋(t,x)^2 dx.
```

Then

```text
E'(t) ≤ 2a E(t)
```

for a.e. `t`.

Since `u₀ ≥ 0`,

```text
E(0) = 0.
```

By Gronwall,

```text
E(t) ≤ E(0) exp(2a t) = 0.
```

Since `E(t) ≥ 0`,

```text
E(t) = 0
```

for all `t`, hence

```text
u₋(t, ·) = 0 in L²(0,1).
```

Therefore

```text
u(t,x) ≥ 0
```

a.e. in space. If the slice `x ↦ u(t,x)` is continuous, this improves to pointwise nonnegativity on `[0,1]`: if `u(t,x₀) < 0`, continuity gives an interval on which `u₋ > c > 0`, contradicting `∫u₋² = 0`.

## Minimal regularity needed

For the weak energy proof, the natural sufficient assumptions are:

```text
u ∈ L²(0,T; H¹(0,1)),
u_t ∈ L²(0,T; H^{-1}(0,1)),
u ∈ C([0,T]; L²(0,1)),
```

plus enough measurability/integrability for the flux and reaction terms:

```text
F[u] ∈ L²(0,T; L²(0,1))       or enough that F[u] ∂x u₋ is integrable,
u(a - b u₊^α) ∈ L²(0,T; H^{-1}) or L¹ spacetime suffices for the tested identity.
```

In 1D, bounded continuous slices plus resolver estimates usually give the flux integrability needed once `R_x` and `(1+R)^{-β}` are bounded.

The load-bearing abstract facts are:

1. **Negative-part Sobolev chain rule**

   ```text
   u ∈ H¹ ⇒ u₋ ∈ H¹,
   ∂x u₋ = -1_{u<0} ∂x u a.e.
   ```

2. **Positive/negative disjointness**

   ```text
   u₊ u₋ = 0,
   u₊ ∂x u₋ = 0 a.e.,
   u₊^α u₋ = 0       for α > 0.
   ```

3. **Hilbert-triple chain rule**

   For `u ∈ L² H¹`, `u_t ∈ L² H^{-1}`,

   ```text
   t ↦ 1/2 ||u₋(t)||²₂ is absolutely continuous,
   d/dt (1/2 ||u₋||²₂) = ⟨u_t, -u₋⟩ a.e.
   ```

4. **Weak formulation allows the test** `φ = -u₋(t)`.

5. **Gronwall for absolutely continuous nonnegative functions**:

   ```text
   E ≥ 0,
   E(0)=0,
   E' ≤ c E a.e. ⇒ E=0.
   ```

6. **Zero L² norm plus continuous slice implies pointwise nonnegativity**.

## Boundary and denominator subtleties

### Boundary terms

If the PDE is already stated in weak form, no boundary term appears. If deriving the weak form from a classical PDE, one needs:

```text
u_x = 0 at x = 0,1
```

for diffusion, and no-flux for the chemotaxis flux. Since

```text
F[u] = u₊ R_x[u₊] / (1+R[u₊])^β
```

and the resolver has Neumann boundary condition

```text
R_x[u₊] = 0 at x = 0,1,
```

one has

```text
F[u](0)=F[u](1)=0
```

provided the denominator is finite. Thus the boundary flux contribution vanishes.

### Denominator

The denominator is safe only because the resolver uses the **nonnegative source** `u₊^γ`:

```text
u₊^γ ≥ 0,
R[u₊] ≥ 0,
1+R[u₊] ≥ 1.
```

This avoids any smallness assumption of the form `||R||∞ < 1`. It also avoids signed-real-power pathologies. The assumptions `γ > 0` and `β ≥ 0` are enough for the displayed denominator bounds in the usual setting.

## Lean formalization roadmap

The proof should be factored into small lemmas. Do not try to prove the full energy estimate monolithically.

### Sign and support lemmas

For real-valued functions:

```lean
u_pos_mul_u_neg_eq_zero :
  positivePart r * negativePart r = 0

positivePart_eq_zero_of_nonpos : r ≤ 0 → positivePart r = 0
negativePart_eq_zero_of_nonneg : 0 ≤ r → negativePart r = 0
negativePart_eq_neg_of_neg : r < 0 → negativePart r = -r
```

For Sobolev/weak derivatives, the essential lemma is:

```lean
weakDeriv_negativePart :
  weakDeriv (fun x => negativePart (u x))
    =ᵐ[μ] fun x => - Set.indicator {x | u x < 0} (weakDeriv u) x
```

Then derive:

```lean
positivePart_mul_weakDeriv_negativePart_eq_zero_ae :
  (fun x => positivePart (u x) * weakDeriv (negativePart ∘ u) x) =ᵐ[μ] 0
```

For the chemotaxis term, package the cancellation in the exact shape used by the weak form:

```lean
chemFlux_mul_grad_negativePart_eq_zero_ae :
  (fun x => chemFlux u x * weakDeriv (negativePart ∘ u) x) =ᵐ[μ] 0
```

where `chemFlux u x` unfolds as `positivePart (u x) * A x`.

### Reaction lemma

Prove pointwise/a.e.:

```lean
reaction_test_negativePart_eq :
  u x * (a - b * positivePart (u x) ^ α) * (-(negativePart (u x)))
    = a * (negativePart (u x))^2
```

under `0 < α` and the usual convention `0^α = 0`.

If using a sign convention where the weak test is written directly as `φ = -u₋`, keep this exact sign. A common source of mistakes is accidentally testing with `u₋`, which flips the time derivative sign.

### Diffusion lemma

Using the weak derivative chain rule:

```lean
∫ x, weakDeriv u x * weakDeriv (negativePart ∘ u) x
  = - ∫ x, (weakDeriv (negativePart ∘ u) x)^2
```

Then after the test `φ=-u₋`, the diffusion contribution becomes

```lean
-ν * ∫ |weakDeriv uNeg|^2
```

or directly a `≤ 0` lemma.

### Energy chain rule

Package the time-chain rule as a reusable lemma:

```lean
negativePart_energy_hasDeriv_or_ac :
  HasDerivAt (fun t => (1/2) * ∫ x, (negativePart (u t x))^2) 
    (dualityPairing (u_t t) (fun x => - negativePart (u t x))) t
```

In the weak setting, the more realistic statement is an absolutely-continuous a.e. derivative theorem:

```lean
energy_ac_and_deriv_ae :
  AbsolutelyContinuousOn E (Set.Icc 0 T) ∧
  ∀ᵐ t, deriv E t = ⟪u_t t, -uNeg t⟫
```

### Gronwall

Use or prove:

```lean
ac_gronwall_zero_of_deriv_le_const_mul
  (hE_nonneg : ∀ t, 0 ≤ E t)
  (hE0 : E 0 = 0)
  (hderiv : ∀ᵐ t, deriv E t ≤ c * E t) :
  ∀ t, E t = 0
```

Here `c = 2*a` if `E = 1/2 ∫u₋²`.

## Smooth-iterate alternative

If the current repository lacks Sobolev chain-rule infrastructure, a practical route is:

1. Prove the energy identity for sufficiently smooth Picard iterates or smooth approximants, where the chain rule is classical except at the kink of `negativePart`.
2. Avoid the kink by using a smooth convex approximation of `negativePart²`, then let the approximation parameter go to zero.
3. Pass to the limit using the Picard convergence and dominated convergence.

This is longer analytically, but may be easier in Lean if the repository already has classical derivative and dominated-convergence tools but not Sobolev chain rules.

## Bottom line

The nonnegativity proof is sound and should not require smallness. The key cancellation is

```text
u₊ ∂x u₋ = 0 a.e.
```

so the chemotaxis term vanishes in the negative-part energy. The only positive contribution is the linear growth term `a∫u₋²`, which Gronwall handles from zero initial negative part.

The exact load-bearing formal facts are the negative-part Sobolev chain rule, the energy chain rule for `⟨u_t,-u₋⟩`, the flux support cancellation, and AC Gronwall. Once those are in place, the proof is short and robust.
