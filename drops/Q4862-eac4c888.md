# Problem 2.5: the correct WZ target, a no-go for the requested exponential error kernel, and a finite certificate search

## Executive conclusion

There is a structural incompatibility in the four requested properties.

Let

\[
H_n=(-16)^n(2)_n^2(3)_n^2\left(\frac52\right)_n\left(\frac72\right)_n^2,
\qquad
\lambda_+=17+12\sqrt2,
\qquad
\rho=\lambda_+^{-1}=17-12\sqrt2.
\]

Write

\[
\widehat q_n=\frac{q_n}{H_n},
\qquad
\widehat p_n=\frac{p_n}{H_n},
\qquad
\widehat e_n=G\widehat q_n-\widehat p_n.
\]

The Birkhoff basis of the normalized degree-13 recurrence has the three modes

\[
f_+(n)=\lambda_+^n(1+O(n^{-1})),
\qquad
f_0(n)=n^{-3}(1+O(n^{-1})),
\qquad
f_-(n)=\rho^n(1+O(n^{-1})).
\]

The Catalan identity cancels the dominant coefficient, so, for the actual challenge rows,

\[
\boxed{
\widehat e_n
=C_0n^{-3}(1+O(n^{-1}))+C_-\rho^n(1+O(n^{-1})).
}
\tag{0.1}
\]

The observed gain of about \(1.5311\) decimal digits per step means \(C_0\ne0\). Consequently an independently defined integral equal to the normalized error and satisfying the same linear recurrence must obey

\[
\widehat e_n=\Theta(n^{-3}),
\]

not

\[
O(\rho^n n^\alpha).
\]

Indeed, since

\[
\widehat q_n=A_+\lambda_+^n(1+O(n^{-1})),
\]

we have

\[
\boxed{
\frac{p_n}{q_n}-G
=-\frac{\widehat e_n}{\widehat q_n}
=O\!\left(n^{-3}\rho^n\right).
}
\tag{0.2}
\]

Thus the silver-ratio decay belongs to the **quotient error**, which does not itself satisfy the homogeneous order-3 recurrence. If \(\widehat e_n\) were \(O(\rho^n)\), the quotient error would instead be \(O(\rho^{2n})\), giving about \(3.06\) digits per step, contrary to the verified behavior.

The corrected proof target is therefore:

\[
\boxed{
\widehat e_n
=\int_0^1\frac{-\log t}{1+t^2}K_n(t)\,dt,
\qquad
|\widehat e_n|\le Cn^{-3},
}
\tag{0.3}
\]

together with

\[
\boxed{
|\widehat q_n|\ge c\lambda_+^n.
}
\tag{0.4}
\]

Equations (0.3)-(0.4) imply (0.2).

No explicit, independently defined \(K_n\) satisfying the printed degree-13 recurrence is presently certified by the supplied data. The known Legendre-square kernel proves a different Catalan approximation, and the exact CMF moment formula is circular. What can be given rigorously is:

1. a proof that the requested exponential same-recurrence kernel cannot be the CMF error;
2. the exact smallest scalar WZ ansatz that would produce a valid kernel;
3. the more general mixed-cycle, matrix-valued Almkvist-Zeilberger identity that is expected to contain the answer;
4. exact initial and boundary conditions that make any output a complete proof rather than another spectral identification.

# 1. The exact error functional for a polynomial kernel

For the published initial rows

\[
\mathbf p_0=(30921,-32972,8240),
\qquad
\mathbf q_0=(33750,-36000,9000),
\]

define the degree-one polynomial row

\[
\boxed{
\mathbf R_0(X)=\mathbf q_0-(1+X)\mathbf p_0
=(2829-30921X,\,-3028+32972X,\,760-8240X).
}
\tag{1.1}
\]

If

\[
\mathbf R_n(X)=\mathbf q_n-(1+X)\mathbf p_n,
\]

then, because the step matrix is independent of \(X\),

\[
\mathbf R_{n+1}(X)=\mathbf R_n(X)M(n).
\]

After division by \(H_n\), the same statement holds with \(M_H(n)=M(n)/(H_{n+1}/H_n)\).

The Catalan functional is

\[
\mathcal C[P]
:=\int_0^1\frac{-\log t}{1+t^2}P(t^2)\,dt.
\tag{1.2}
\]

For \(P(X)=A+BX\),

\[
\int_0^1\frac{-\log t}{1+t^2}\,dt=G,
\]

and

\[
\int_0^1\frac{t^2(-\log t)}{1+t^2}\,dt
=\int_0^1(-\log t)\,dt-G
=1-G.
\]

Hence

\[
\boxed{
\mathcal C[A+BX]=G(A-B)+B.
}
\tag{1.3}
\]

Substituting \(A=q_n-p_n\) and \(B=-p_n\) gives

\[
\boxed{
\mathcal C[R_n]=Gq_n-p_n.
}
\tag{1.4}
\]

Therefore an **independent** formula for the polynomial \(R_n(X)\) immediately gives an independent Catalan error integral. The problem is reduced to constructing \(R_n(X)\) without using the CMF product.

# 2. Why the elementary Legendre-square kernel is not enough

The exact Delannoy-square summand is

\[
\boxed{
F(n,k)=2^k\binom{2k}{k}\binom nk\binom{n+k}{k},
\qquad
D_n^2=\sum_{k=0}^nF(n,k).
}
\tag{2.1}
\]

It has the shift ratios

\[
\boxed{
\frac{F(n+1,k)}{F(n,k)}
=\frac{n+k+1}{n-k+1},
}
\tag{2.2}
\]

and

\[
\boxed{
\frac{F(n,k+1)}{F(n,k)}
=\frac{4(2k+1)(n-k)(n+k+1)}{(k+1)^3}.
}
\tag{2.3}
\]

The corresponding polynomial kernel is

\[
\sum_{k=0}^n(-1)^kF(n,k)t^{2k}
=P_n^{\rm Leg}\!\left(\sqrt{1-8t^2}\right)^2.
\tag{2.4}
\]

Using this in the Catalan functional gives the valid approximation model

\[
\int_0^1\frac{t^2(-\log t)}{1+t^2}
P_n^{\rm Leg}\!\left(\sqrt{1-8t^2}\right)^2dt,
\]

but its projective rate is

\[
\frac{15+4\sqrt{14}}{17+12\sqrt2}=0.882135\ldots,
\]

not \(17-12\sqrt2\). It proves a different Catalan approximation.

The reason is geometric. A \(P_n(3)^2\) period is supported on the dominant/dominant saddle and has scale

\[
\lambda_+^n/n.
\]

The normalized CMF error is the **neutral** mode. In the symmetric-square model the neutral carrier is the mixed product

\[
P_n(3)Q_n(3),
\]

where \(Q_n\) is the Legendre function of the second kind. The standard asymptotics give

\[
\boxed{
P_n(3)Q_n(3)
\sim\frac{1}{4\sqrt2\,n}.
}
\tag{2.5}
\]

The extra formal shift from \(n^{-1}\) to \(n^{-3}\) must be supplied by a rational shear of order \(n^{-2}\). This explains both empirical facts:

- \(\widehat q_n/D_n^2\asymp n\): the dominant \(P^2\) mode is shifted upward by one power;
- \(\widehat e_n\asymp n^{-3}\): the mixed \(PQ\) mode is shifted downward by two powers.

There is an explicit one-dimensional mixed-period integral,

\[
\boxed{
Q_n(3)=\frac12\int_{-1}^1\frac{P_n(x)}{3-x}\,dx,
}
\tag{2.6}
\]

so

\[
\boxed{
P_n(3)Q_n(3)
=\frac12\int_{-1}^1
\frac{P_n(3)P_n(x)}{3-x}\,dx.
}
\tag{2.7}
\]

Any simple Legendre integral for the CMF error should therefore be sought as a rational contiguous combination of mixed products, not as a positive \(P_n^2\) integral.

The smallest such ansatz is

\[
\boxed{
J_n=\sum_{0\le i,j\le2}r_{ij}(n)
P_{n+i}(3)Q_{n+j}(3),
\qquad r_{ij}(n)\in\mathbb Q(n).
}
\tag{2.8}
\]

Testing whether \(J_n\) satisfies the normalized CMF operator is exactly a scalar Ore-Hom computation. If such \(r_{ij}\) are found and three initial values match, (2.7) gives an independent integral. No such rational coefficients have yet been certified.

# 3. The smallest scalar WZ search that could succeed

Let

\[
L_{25}^{(H)}=\ell_0(n)+\ell_1(n)S+\ell_2(n)S^2+\ell_3(n)S^3
\tag{3.1}
\]

be the primitive \(H_n\)-normalized scalar operator. Its four coefficients have degree \(13\).

Seek two rational functions \(A(n,k),B(n,k)\) and define

\[
\boxed{
R_n(X)=\sum_{k=0}^nF(n,k)\bigl(A(n,k)+XB(n,k)\bigr).
}
\tag{3.2}
\]

The required WZ certificate is a rational function

\[
C(n,k;X)=C_0(n,k)+XC_1(n,k)
\]

such that

\[
\boxed{
\sum_{j=0}^3\ell_j(n)
F(n+j,k)
\bigl(A(n+j,k)+XB(n+j,k)\bigr)
=\Delta_k\!\left(F(n,k)C(n,k;X)\right).
}
\tag{3.3}
\]

Use the reciprocal-gamma convention, so \(F(n,k)=0\) outside \(0\le k\le n\). If the certificate has no uncancelled boundary poles, summing (3.3) over all \(k\in\mathbb Z\) gives

\[
L_{25}^{(H)}R_n(X)=0.
\]

If, in addition,

\[
R_n(X)=\widehat q_n-(1+X)\widehat p_n
\quad(n=0,1,2),
\tag{3.4}
\]

then equality holds for every \(n\). Applying (1.2) yields

\[
\boxed{
\mathcal C[R_n]
=G\widehat q_n-\widehat p_n.
}
\tag{3.5}
\]

This would be the desired independent WZ integral.

## 3.1 Rational identity to solve

Divide (3.3) by \(F(n,k)\). Put

\[
N_j(n,k):=\frac{F(n+j,k)}{F(n,k)}
=\prod_{r=1}^j\frac{n+k+r}{n-k+r},
\qquad N_0=1,
\tag{3.6}
\]

and

\[
K(n,k):=\frac{F(n,k+1)}{F(n,k)}
=\frac{4(2k+1)(n-k)(n+k+1)}{(k+1)^3}.
\tag{3.7}
\]

Then (3.3) is equivalent to the pair of rational identities

\[
\boxed{
\sum_{j=0}^3\ell_j(n)N_j(n,k)A(n+j,k)
=K(n,k)C_0(n,k+1)-C_0(n,k),
}
\tag{3.8}
\]

and

\[
\boxed{
\sum_{j=0}^3\ell_j(n)N_j(n,k)B(n+j,k)
=K(n,k)C_1(n,k+1)-C_1(n,k).
}
\tag{3.9}
\]

After denominator clearing, these are linear equations for the coefficients of fixed numerator ansätze.

## 3.2 Recommended degree ladder

The empirical relation

\[
\widehat q_n/D_n^2\asymp n
\]

makes total degree one in \(n,k\) the first meaningful weight. Use the ladder

```text
A,B total degree: 0, 1, 2, 3, 4
certificate degree: weight degree + 3, +4, +5
```

A natural first certificate denominator is the Delannoy denominator already present in the short certificate,

\[
(n-k+1)(n-k+2)(n+k),
\]

augmented by the shifted factors required by \(N_3\), namely \(n-k+3\), and by powers of \(k+1\) forced by (3.7).

A nonzero solution must be checked for:

1. cancellation of every apparent pole at the summation endpoints;
2. vanishing of \(F(n,k)C(n,k;X)\) at the lower and upper boundaries;
3. exact initial matching (3.4);
4. exact recurrence identity over \(\mathbb Q(n,k,X)\), not only at sample points.

A nullspace-zero result for this ansatz would rule out only the single-sum Delannoy class. It would not rule out a double sum, a parameter derivative, or a matrix-valued period.

# 4. A Sage skeleton for the scalar WZ search

The following script is an exact linear-algebra setup. Insert the four exact normalized recurrence coefficients in `ELL`. It does not use the CMF product in the kernel ansatz; the product is used only to compute the three target initial polynomials for verification.

```python
# SageMath
from sage.all import *

# Base variables.
R.<n,k,X> = PolynomialRing(QQ)
K = FractionField(R)

# Paste the primitive degree-13 coefficients here.
# ELL = [ell0(n), ell1(n), ell2(n), ell3(n)]
ELL = load('L25_H_coefficients.sobj')


def n_shift_ratio(j):
    ans = K(1)
    for r in range(1, j + 1):
        ans *= (n + k + r) / (n - k + r)
    return ans


NR = [n_shift_ratio(j) for j in range(4)]
KR = 4*(2*k + 1)*(n - k)*(n + k + 1)/(k + 1)^3


def monomials_total_degree(d):
    return [n^a*k^b for a in range(d + 1)
                     for b in range(d + 1 - a)]


def shifted(expr, jn=0, jk=0):
    return K(expr(n=n + jn, k=k + jk))


def build_ansatz(prefix, basis, coeff_ring):
    coeffs = [coeff_ring.gen(prefix + i) for i in range(len(basis))]
    return sum(c*m for c, m in zip(coeffs, basis)), coeffs


# In practice it is easier to construct the coefficient matrix by linearity:
# for each basis monomial, compute its contribution to the cleared numerator,
# extract coefficients in n,k, and use these as one matrix column.

def cleared_numerator(expr):
    return R(K(expr).numerator())


def contribution_A(weight_basis_element, cert_basis_element=None):
    lhs = K(0)
    for j in range(4):
        lhs += K(ELL[j]) * NR[j] * shifted(weight_basis_element, jn=j)
    rhs = K(0)
    if cert_basis_element is not None:
        rhs = KR*shifted(cert_basis_element, jk=1) - cert_basis_element
    return cleared_numerator(lhs - rhs)


# Suggested outer loop:
#   for d_weight in range(0, 5):
#       choose bases for A and C0;
#       build the coefficient matrix of (3.8);
#       compute its nullspace over several finite fields first;
#       repeat for B and C1;
#       reconstruct over QQ only if nullity is stable.
#
# Once A,B are recovered, verify:
#   sum_j ell_j(n) * S_{n+j}(k;X) == Delta_k certificate
# exactly in K, and then verify n=0,1,2 initial sums.
```

For serious searches, use modular linear algebra first. Stable positive nullity must be observed at several primes and several specializations before interpolation and rational reconstruction.

# 5. The more general kernel: a mixed-cycle matrix AZ identity

The scalar ansatz (3.2) may be too small because the CMF appears to be a nontrivial integrated-period extension rather than the homogeneous Delannoy square. The canonical hyperexponential phase is nevertheless explicit.

Put

\[
\phi(u)=3+\sqrt2(u+u^{-1}),
\qquad
\Phi(u,v)=\phi(u)\phi(v),
\tag{5.1}
\]

so that

\[
D_n^2=\operatorname{CT}_{u,v}\Phi(u,v)^n.
\]

The critical values are

\[
17+12\sqrt2,
\qquad1,
\qquad17-12\sqrt2.
\]

Introduce the Catalan parameter family

\[
\boxed{
h_n(u,v,t;\varepsilon)
=\Phi(u,v)^n\frac{t^\varepsilon}{uv(1+t^2)}.
}
\tag{5.2}
\]

Then

\[
-\left.\partial_\varepsilon
\int_0^1\frac{t^\varepsilon}{1+t^2}\,dt
\right|_{\varepsilon=0}
=G.
\tag{5.3}
\]

Let

\[
P(u)=\sqrt2u^2+3u+\sqrt2.
\]

The shift and logarithmic derivatives are

\[
\frac{h_{n+1}}{h_n}=\Phi(u,v),
\tag{5.4}
\]

\[
L_u:=\partial_u\log h_n
=\frac{n\sqrt2(u^2-1)-P(u)}{uP(u)},
\tag{5.5}
\]

\[
L_v:=\partial_v\log h_n
=\frac{n\sqrt2(v^2-1)-P(v)}{vP(v)},
\tag{5.6}
\]

and

\[
L_t:=\partial_t\log h_n
=\frac{\varepsilon+(\varepsilon-2)t^2}{t(1+t^2)}.
\tag{5.7}
\]

Seek a rational \(1\times3\) row \(\mathcal P_n\) and certificate rows \(\mathcal U_n,\mathcal V_n,\mathcal T_n\) satisfying

\[
\boxed{
\Phi\,\mathcal P_{n+1}
-\mathcal P_nM_H(n)
=(\partial_u+L_u)\mathcal U_n
 +(\partial_v+L_v)\mathcal V_n
 +(\partial_t+L_t)\mathcal T_n.
}
\tag{5.8}
\]

Multiplying by \(h_n\) turns the right side into a sum of total derivatives. Integrating over two closed cycles and \(0\le t\le1\) gives the normalized CMF row recurrence, up to the explicit \(t\)-boundary

\[
[h_n\mathcal T_n]_{t=0}^{t=1}.
\]

This boundary should **not** automatically be forced to zero. The integrated elliptic module is a nontrivial extension, and the \(t=1\) boundary is the natural carrier of the neutral direction and of the Catalan connection coefficient. Killing it can destroy exactly the part of the module one is trying to prove.

## 5.1 Why the cycle must be mixed

The standard constant-term torus contains the dominant/dominant saddle and produces \(\lambda_+^n\). The error cycle must instead project to the mixed saddle

\[
(+,-)\quad\text{or}\quad(-,+),
\]

whose critical value is

\[
(3+2\sqrt2)(3-2\sqrt2)=1.
\]

This is the period-theoretic explanation of the neutral exponential base. The rational numerator/gauge must then supply the additional \(n^{-2}\) shear needed to turn the natural mixed \(n^{-1}\) behavior into \(n^{-3}\).

Thus a positive real \(P_n^2\) integral is the wrong cycle for the CMF error even though it has the right singular polynomial.

## 5.2 A smaller polynomial-valued row search

The external Catalan functional allows the \(t\) variable to be removed from the bulk search. Seek

\[
\mathcal P_n(u,v;X)=\mathcal P_n^{(0)}(u,v)
+X\mathcal P_n^{(1)}(u,v)
\]

and solve

\[
\boxed{
\Phi\mathcal P_{n+1}
-\mathcal P_nM_H(n)
=(\partial_u+L_u)\mathcal U_n
 +(\partial_v+L_v)\mathcal V_n.
}
\tag{5.9}
\]

After taking constant terms,

\[
\mathbf R_n(X)=\operatorname{CT}_{u,v}
\bigl(\Phi^n\mathcal P_n(u,v;X)\bigr)
\]

satisfies the normalized matrix recurrence. Impose the exact initial condition

\[
\boxed{
\mathbf R_0(X)=
(2829-30921X,-3028+32972X,760-8240X).
}
\tag{5.10}
\]

Then applying \(\mathcal C\) componentwise gives the CMF error row.

This is smaller than a full \(3\times3\) AZ search. With total geometric degree two, the \(u\leftrightarrow v\) symmetry, and \(n\)-degree two, the problem separates into two \(1\times3\) row blocks corresponding to the coefficients of \(1\) and \(X\). It is a finite linear system over \(\mathbb Q(n)\).

# 6. What would constitute a complete decay proof

A WZ identity proves only the recurrence. It does not automatically prove the required bound. A complete certificate needs the following four components.

## 6.1 Exact kernel identity

Produce explicit rational \(A,B,C\) in (3.3), or explicit rational rows in (5.8)/(5.9), and verify the identity after denominator clearing.

## 6.2 Initial matching

Verify the three initial polynomial values

\[
R_n(X)=\widehat q_n-(1+X)\widehat p_n,
\qquad n=0,1,2.
\]

This identifies the independently defined period with the CMF error by recurrence uniqueness.

## 6.3 Neutral-cycle bound

Prove that the chosen contour represents the mixed thimble and that no dominant/dominant component remains. Then use uniform steepest descent, or the Legendre \(P_nQ_n\) representation, to obtain

\[
|I_n|\le Cn^{-3}.
\]

It is not enough to show that a polynomial numerator vanishes to finite order at the dominant saddle; that only changes powers of \(n\) while retaining \(\lambda_+^n\). The dominant thimble coefficient must vanish **exactly**.

## 6.4 Dominant denominator lower bound

Use a validated Birkhoff/adjoint computation, or a positive dominant-cycle integral, to prove

\[
|\widehat q_n|\ge c\lambda_+^n
\]

for all sufficiently large \(n\). Combining this with the neutral bound gives

\[
\left|\frac{p_n}{q_n}-G\right|
\le\frac Cc n^{-3}\rho^n.
\]

# 7. Assessment of the four suggested approaches

## (a) Legendre polynomial integral

A \(P_n^2\) integral is explicit but has the wrong cycle and wrong rate. A viable Legendre ansatz must use the mixed \(P_nQ_n\) carrier, followed by a rational contiguous shear. Equation (2.8) is the exact finite search space.

## (b) Rodrigues-type formula

Applying finitely many derivatives or contiguous operators to \(D_n^2\) is equivalent to finding a rational Ore intertwiner. It can work, but it does not avoid the missing algebraic certificate. The asymptotic powers indicate that the transformation must raise the two outer modes by one power and lower the neutral mode by two powers.

## (c) Hypergeometric sum

This is the most concrete first search. Equations (3.8)-(3.9) reduce it to exact linear algebra. A solution gives a literal WZ certificate and the desired Catalan integral via (1.2).

## (d) Period integral at \(z=\rho\)

The Brafman/Abel period proves

\[
Y(k(\rho))=Y(1)=G.
\]

It does not identify the CMF cyclic vector. One still needs either the differential/Ore gauge or the AZ kernel (5.8). The value \(G\) is the boundary evaluation; the missing step is transporting the printed initial rows to that boundary section.

# 8. Relation to known Catalan creative-telescoping constructions

Zudilin's classical construction uses a very-well-poised hypergeometric family and obtains a different second-order recurrence by Zeilberger telescoping. See:

- W. Zudilin, *An Apéry-like difference equation for Catalan's constant*, https://arxiv.org/abs/math/0201024

It is a genuine Catalan WZ proof, but it does not match the degree-13 normalized Problem 2.5 operator.

The general algorithmic framework needed for (5.8) is standard creative telescoping in Ore algebras; the difficulty is the size and the extension boundary, not a missing theorem. Useful references include:

- F. Chyzak, M. Kauers, B. Salvy, *A Non-Holonomic Systems Approach to Special Function Identities*, https://arxiv.org/abs/0904.2761
- C. Koutschan, E. Wong, *Creative Telescoping on Multiple Sums*, https://arxiv.org/abs/2010.08889

No published formula presently identifies the particular printed CMF with Zudilin's recurrence or with the elementary Delannoy Catalan extension.

# Final answer

The desired independent kernel has **not** yet been found in explicit closed form. More importantly, the requested decay condition must be corrected:

\[
\boxed{
\text{same-recurrence normalized error: }O(n^{-3}),
\qquad
\text{quotient error: }O(n^{-3}\rho^n).
}
\]

The shortest credible route to an explicit certificate is:

1. solve the scalar WZ system (3.8)-(3.9) for a degree-one polynomial-valued Delannoy weight;
2. if that nullspace is zero, solve the two-row polynomial-valued matrix AZ system (5.9);
3. match the three initial polynomials (5.10);
4. identify the resulting cycle as the mixed Legendre \(PQ\) cycle and prove the \(n^{-3}\) bound;
5. combine with a dominant lower bound for \(\widehat q_n\).

Every step is finite and checkable. The genuinely nontrivial computation is the rational certificate in (3.3) or (5.9); spectral matching alone cannot replace it.
