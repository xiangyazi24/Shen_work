## Why not Option A as stated?

The existing provider chain has two flavors:

1. `IntervalBFormSpectralProvider.lean` is hardwired to
   `conjugatePicardLimit p u‚āÄ D.T` and `ConjugateMildExistenceData`.
2. `IntervalBFormSpectralPdeAgreementStandardFacts.lean` has a constructor
   `hasBFormSpectralPdeAgreement_of_standardFacts`, but it requires
   `BFormBoundedClassicalRegularity p T u` as input.  That is exactly what the
   bootstrap is supposed to prove, so using it here is circular.

The reusable parts of Option A are the **low-level non-circular lemmas**, not the top-level constructor.  In particular, keep reusing:

```lean

intervalConjugateMildSolution_pde_u_of_spectral

restartCosineSeries_hasDerivAt_time_of_on
bForm_restart_of_global_cosine            -- only when a global cosine formula is available
```

For arbitrary `S`, the missing theorem should not go through classical regularity.  It should produce the local restart data from `S.hmild`.

## The Option C theorem: local restart data from `S.hmild`

The heart of the route is this theorem.  It proves the restart representation by applying the full mild equation at `s` and at `ŌĄ = t‚āÄ/2`, subtracting/splitting the Duhamel integrals, and expanding the result in cosine modes.

The source ladder should provide a **soft-clamped global `DuhamelSourceTimeC1` witness per `t‚āÄ`**, equal to the canonical B-form source coefficients on the integration range.  This matches the design already used by `IntervalPicardLimitTimeNhdLocalized.lean`.

```lean
/-- Source-coefficient ladder data needed locally around a restart time.
The source witness `a` is global `DuhamelSourceTimeC1`, but it only has to agree
with the canonical B-form coefficients on the local restart integration range. -/
structure BFormMildLocalSourceLadder

  exists_source : ‚ąÄ t‚āÄ, 0 < t‚āÄ ‚Üí t‚āÄ < T ‚Üí
    ‚ąÉ (a : ‚ĄĚ ‚Üí ‚Ąē ‚Üí ‚ĄĚ) (_src : DuhamelSourceTimeC1 a) (offset : ‚ĄĚ),
      offset = t‚āÄ / 2 ‚ąß
      0 < t‚āÄ - offset ‚ąß
      (‚ąÄŠ∂† s in ūĚďĚ t‚āÄ, ‚ąÄ r ‚ąą Set.Icc (0 : ‚ĄĚ) (s - offset), ‚ąÄ n,
        a r n =
          coupledLogisticSourceCoeffs p u (offset + r) n
            - p.Ōá‚āÄ * coupledChemDivSourceCoeffs p u (offset + r) n)
      ‚ąß
      LogisticCosineFourierData p u t‚āÄ
      ‚ąß
```

## The true core theorem: restart formula from the mild equation

This is the theorem implementing Option C.  Its proof is not classical regularity; it is semigroup/Duhamel algebra plus coefficient expansion.

```lean
/-- Local restart cosine representation for an arbitrary B-form mild solution.
Proof outline: use `S.hmild` at `s` and at `offset`, split the Duhamel integrals,
use semigroup composition for the heat/conjugate legs, and identify the resulting
coefficients with `localRestartCoeff`. -/
theorem intervalConjugateMildSolution_localRestart_cosine
    {p : CM2Params} {u‚āÄ : intervalDomainPoint ‚Üí ‚ĄĚ}
    (S : ConjugateMildSolutionData p u‚āÄ)
    {a : ‚ĄĚ ‚Üí ‚Ąē ‚Üí ‚ĄĚ} {offset t‚āÄ : ‚ĄĚ}
    (hoff_eq : offset = t‚āÄ / 2)
    (hoff_pos : 0 < t‚āÄ - offset)
    (hsource_nhd :
      ‚ąÄŠ∂† s in ūĚďĚ t‚āÄ, ‚ąÄ r ‚ąą Set.Icc (0 : ‚ĄĚ) (s - offset), ‚ąÄ n,
        a r n =
          coupledLogisticSourceCoeffs p S.u (offset + r) n
            - p.Ōá‚āÄ * coupledChemDivSourceCoeffs p S.u (offset + r) n)
    (src : DuhamelSourceTimeC1 a) :
    ‚ąÄŠ∂† s in ūĚďĚ t‚āÄ, ‚ąÄ y : intervalDomainPoint,
      S.u s y = ‚ąĎ' n,
        localRestartCoeff
          (cosineCoeffs (intervalDomainLift (S.u offset)))
          a (s - offset) n * cosineMode n y.1 := by
  -- Work on the neighborhood `s ‚ąą Ioo offset S.T` and with `hsource_nhd`.
  -- From `S.hmild s` and `S.hmild offset`, derive
  --   u(s) = S_N(s-offset)(u(offset)) + ‚ąę‚āÄ^{s-offset} S_N/B_N(s-offset-r) source(offset+r) dr.

  -- Use `hsource_nhd` to replace canonical coefficients by the clamped witness `a`.
```

## Short answer

The best route is **Option C**, with selective reuse of the existing non-circular spectral machinery.

Do not call `hasBFormSpectralPdeAgreement_of_standardFacts`; it requires classical regularity and is circular.  Instead, prove local restart cosine data directly from `S.hmild`, package it into a stronger point-independent `BFormMildLocalRestartSpectralData`, and project that into both:

Then `BFormMildSpectralBootstrapData p S` is just a record constructor, because resolver fields 3 and 4 are already done.
