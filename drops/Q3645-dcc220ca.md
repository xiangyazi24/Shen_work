## Recommended constants and record

For best compile prospects, use an explicit zero-at-`k=0` majorant first. This avoids brittle `tsum` reindexing in the main theorem and handles the missing `1 ≤ k` hypothesis cleanly.

Add these near the Task175 theorem in `IntervalChemFluxHolderSourceDecay.lean`.

```lean
/- Add inside `namespace ShenWork.Paper2`, after current imports. -/

/-- Uniform quadratic decay of elliptic source coefficients on a time window. -/
structure UniformSourceCoeffQuadraticDecayOn
    (p : CM2Params) (u : ℝ → intervalDomainPoint → ℝ)
    (T Csrc : ℝ) : Prop where
  Csrc_nonneg : 0 ≤ Csrc
  decay : ∀ s, 0 < s → s ≤ T → ∀ k : ℕ, 1 ≤ k →
    |(ShenWork.PDE.intervalNeumannResolverSourceCoeff p (u s) k).re| ≤
      Csrc / ((k : ℝ) * Real.pi) ^ 2

/-- The per-mode uniform majorant for `resolverGrad2Real`.

The `k = 0` mode is zero because the factor `(kπ)^2` is zero, while the decay
record only applies for `1 ≤ k`. -/
noncomputable def resolverGrad2UniformMajorant (Csrc : ℝ) (k : ℕ) : ℝ :=
  if k = 0 then 0 else Csrc / ((k : ℝ) * Real.pi) ^ 2

/-- A concrete uniform bound for the resolver second derivative.  This is equal to
`Csrc / π² * ∑ n, 1/(n+1)^2`, but the zero-mode form is easier to use in the
first producer theorem. -/
noncomputable def resolverGrad2UniformBound (Csrc : ℝ) : ℝ :=
  ∑' k : ℕ, resolverGrad2UniformMajorant Csrc k
```

This constant is the most implementation-friendly choice. A later cosmetic theorem can show

```lean
resolverGrad2UniformBound Csrc =
  (Csrc / Real.pi ^ 2) * (∑' n : ℕ, 1 / ((n : ℝ) + 1) ^ 2)
```

but do **not** make that equality a dependency of the main proof unless you need the prettier printed constant.

### Case `k = 0`

The factor `((0 : ℝ) * Real.pi)^2` is zero, so:

```lean
term 0 = 0
resolverGrad2UniformMajorant Csrc 0 = 0
```

This is the reason to use the `if k = 0 then 0 else ...` majorant. No source-decay hypothesis is needed at `k = 0`, matching the `1 ≤ k` field of the record.

Lean proof shape:

```lean
by_cases hk0 : k = 0
· subst k
  simp [term, resolverGrad2UniformMajorant]
```

### Case `k ≠ 0`

First obtain `1 ≤ k`:

```lean
have hk1 : 1 ≤ k := Nat.succ_le_of_lt (Nat.pos_of_ne_zero hk0)
```

Then define the standard positivity/lower-denominator facts:

```lean
have hkpos : (0 : ℝ) < (k : ℝ) := by exact_mod_cast Nat.pos_of_ne_zero hk0
have hkpipos : 0 < (k : ℝ) * Real.pi := mul_pos hkpos Real.pi_pos
have hkp2pos : 0 < ((k : ℝ) * Real.pi) ^ 2 := by positivity
have hden_pos : 0 < p.μ + ShenWork.Paper3.unitIntervalNeumannSpectrum.eigenvalue k :=
  ShenWork.PDE.intervalNeumannResolver_denom_pos p k
have hlam : ShenWork.Paper3.unitIntervalNeumannSpectrum.eigenvalue k =
    (k : ℝ) ^ 2 * Real.pi ^ 2 := rfl
have hdenlow : ((k : ℝ) * Real.pi) ^ 2 ≤
    p.μ + ShenWork.Paper3.unitIntervalNeumannSpectrum.eigenvalue k := by
  rw [hlam]
  nlinarith [p.hμ.le, sq_nonneg ((k : ℝ) * Real.pi)]
```

The cosine factor bound is:

```lean
have hcos : |Real.cos ((k : ℝ) * Real.pi * z)| ≤ 1 := Real.abs_cos_le_one _

have hmode :
    |(-(((k : ℝ) * Real.pi) ^ 2) *
        Real.cos ((k : ℝ) * Real.pi * z))| ≤
      ((k : ℝ) * Real.pi) ^ 2 := by
  rw [abs_mul, abs_neg,
    abs_of_nonneg (by positivity : (0 : ℝ) ≤ ((k : ℝ) * Real.pi) ^ 2)]
  calc ((k : ℝ) * Real.pi) ^ 2 * |Real.cos ((k : ℝ) * Real.pi * z)|
      ≤ ((k : ℝ) * Real.pi) ^ 2 * 1 :=
        mul_le_mul_of_nonneg_left hcos (by positivity)
```

Use `resolverCoeff_re_eq`:


```lean
have hprod : ‖term k‖ ≤
    |(ShenWork.PDE.intervalNeumannResolverCoeff p (u s) k).re| *
```

and cancel:

```lean
have hbig :
    |(ShenWork.PDE.intervalNeumannResolverCoeff p (u s) k).re| *
      ((k : ℝ) * Real.pi) ^ 2 ≤
    Csrc / ((k : ℝ) * Real.pi) ^ 2 := by
  have hbase := mul_le_mul_of_nonneg_right hres_abs
    (by positivity : (0 : ℝ) ≤ ((k : ℝ) * Real.pi) ^ 2)

      Csrc / ((k : ℝ) * Real.pi) ^ 2 /
        ((k : ℝ) * Real.pi) ^ 2 * ((k : ℝ) * Real.pi) ^ 2 =
      Csrc / ((k : ℝ) * Real.pi) ^ 2 := by
    field_simp [hpne]
  rw [hrw] at hbase
  exact hbase
```

```lean
  _ ≤ Csrc / ((k : ℝ) * Real.pi) ^ 2 := hbig
  _ = resolverGrad2UniformMajorant Csrc k := by
      simp [resolverGrad2UniformMajorant, hk0]
```

## From termwise majorant to bound on `|resolverGrad2Real|`

Let `term` be the series term above.

1. Prove absolute summability:

```lean
have hmaj_sum := resolverGrad2UniformMajorant_summable H.Csrc_nonneg
have hterm_norm_sum : Summable fun k : ℕ => ‖term k‖ :=
  Summable.of_nonneg_of_le (fun k => norm_nonneg _) hterm_bound hmaj_sum
```

2. Convert to summability of `term`:

```lean
have hterm_sum : Summable term := Summable.of_norm hterm_norm_sum
```

3. Use the standard norm-of-tsum bound:

```lean
have hnorm_tsum : ‖∑' k : ℕ, term k‖ ≤ ∑' k : ℕ, ‖term k‖ :=
  norm_tsum_le_tsum_norm hterm_norm_sum
```

If the exact lemma name in this repo resolves differently, the fallback is the usual `Summable.norm_tsum_le` form; the mathematical input is `hterm_norm_sum`.

4. Bound the tsum of norms by the majorant tsum:

```lean
have hnorm_le_majorant :
    (∑' k : ℕ, ‖term k‖) ≤
      ∑' k : ℕ, resolverGrad2UniformMajorant Csrc k :=
  hterm_norm_sum.tsum_le_tsum hterm_bound hmaj_sum
```

5. Rewrite `resolverGrad2Real` to `tsum term` by `rfl`/`change`:

```lean
change |∑' k : ℕ, term k| ≤ resolverGrad2UniformBound Csrc
rw [Real.norm_eq_abs] at hnorm_tsum
exact hnorm_tsum.trans hnorm_le_majorant
```

Because `resolverGrad2UniformBound Csrc` is defined as that majorant `tsum`, there is no shifted-series algebra in the final theorem.
