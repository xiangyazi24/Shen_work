# Q3539 audit: remaining physical H¹ sqrt-bound frontier after Task105

## Verdict

I do **not** see an existing non-circular producer for

```lean
H1PhysicalRHSSqrtBoundsBefore p u v T V₁ V₂ M L
```

for the concrete physical scalars. The existing route has the right target interface, but not the source-side scalar estimates needed to fill it.

What exists:

* `H1PhysicalRHSSqrtBoundsBefore` is only a wrapper around `H1SqrtTermBoundsBefore` for the concrete scalar triple.
* `H1SqrtTermBoundsBefore` asks for fixed constants `V₁ V₂ M L` and three **pointwise-in-time** scalar estimates on every `τ ∈ Ioo 0 T`.
* `resolverGradReal_bounded`, `resolverGrad2Real_bounded`, and `lift_u_bounded` exist, but they return **per-time existential bounds**. They do not provide one fixed `V₁`, `V₂`, or `M` valid for all `τ < T`.
* Task103/104 zero-window Young/spatial data prove local integrability/majorants. They do not imply the global pointwise sqrt estimates and should not be used as a substitute.

So the answer is: **no existing complete route** unless the “explicit positive/boundedness/resolver-sup assumptions” are supplied as fixed-before-`T` uniform bounds. With those source assumptions, the producer should be a small new theorem in a new physical sqrt-bound file, not in the identity Route-C files.

## Why the existing resolver-bound names are not enough

Existing names that are useful but insufficient by themselves:

```lean

resolverGrad2Real_bounded
lift_u_bounded
solution_lift_v_deriv_eq_resolverGrad_Icc
solution_lift_v_nonneg_Icc
```

They are useful in a later producer for `H1PhysicalRHSFactorBoundsBefore`, but:

1. `resolverGradReal_bounded` gives `∃ G` for one fixed `τ`, not a uniform `V₁` for all `τ ∈ (0,T)`.
2. `resolverGrad2Real_bounded` similarly gives a per-time bound, not a uniform `V₂`.
3. `lift_u_bounded` gives a per-time `M`; the sqrt interface needs one fixed `M` before `T`.
4. The denominator control from `solution_lift_v_nonneg_Icc` is pointwise and good, but it only helps after a fixed uniform numerator bound exists.
5. No existing theorem I found packages the physical scalar estimates for `H1PhysicalTaxisX`, `H1PhysicalUvxxX`, and `H1PhysicalReactX` into `H1PhysicalRHSSqrtBoundsBefore`.

This is exactly the difference between local/classical compactness at each time and a before-`T` route constant.

### Taxis estimate

For `τ ∈ Ioo 0 T`, prove first:

```lean
H1PhysicalTaxisX p u v τ ≤
  V₁ * (H1lapL2Norm u τ * H1gradL2Norm u τ)
```

Then multiply by `-p.χ₀ ≥ 0`.

Proof skeleton:

1. Unfold `H1PhysicalTaxisX`.
2. Use `x ↦ liftDeriv2 u τ x` and `x ↦ deriv (intervalDomainLift (u τ)) x`.
3. Bound
   ```lean
   -∫ uxx * taxisPart ≤ |∫ uxx * taxisPart| ≤ ∫ |uxx * taxisPart|.
   ```
4. Use `hfac.taxis_part_le` to get
   ```lean
   |uxx * taxisPart| ≤ V₁ * |uxx * ux|.
   ```
5. Use `integral_abs_mul_le_sqrt` and the identities
   ```lean
   ∫ uxx^2 = lapL2sq u τ = (H1lapL2Norm u τ)^2
   ∫ ux^2 = 2 * H1energy u τ = (H1gradL2Norm u τ)^2
   ```
6. Normalize with `Real.sqrt_sq_eq_abs` or the existing nonneg lemmas.

The pointwise taxis factor bound can later be produced from a uniform resolver-gradient assumption:

```lean
∀ τ ∈ Ioo 0 T, ∀ x ∈ Icc 0 1,
  |resolverGradReal p (u τ) x| ≤ V₁
```

using `solution_lift_v_deriv_eq_resolverGrad_Icc` and `solution_lift_v_nonneg_Icc`.

### Uvxx estimate

For `τ ∈ Ioo 0 T`, prove:

```lean
H1PhysicalUvxxX p u v τ ≤ M * (V₂ * H1lapL2Norm u τ)
```

Then multiply by `-p.χ₀ ≥ 0`.

Proof skeleton:

1. Unfold `H1PhysicalUvxxX`.
2. Bound by `∫ |uxx| * |uvxxPart|`.
3. Use `hfac.uvxx_part_le` to get `≤ M * V₂ * ∫ |uxx|`.
4. Use Cauchy with the constant function `1` on `[0,1]`:
   ```lean
   ∫ |uxx * 1| ≤ sqrt (∫ uxx^2) * sqrt (∫ 1^2) = H1lapL2Norm u τ
   ```
5. Rearrange `M * V₂ * H1lapL2Norm` to `M * (V₂ * H1lapL2Norm)`.

A later resolver-sup producer may prove `hfac.uvxx_part_le` from fixed assumptions such as:

```lean
∀ τ x, |intervalDomainLift (u τ) x| ≤ M
∀ τ x, |(p.μ * intervalDomainLift (v τ) x -
          p.ν * (intervalDomainLift (u τ) x) ^ p.γ) /
          (1 + intervalDomainLift (v τ) x) ^ p.β
        - p.β * (deriv (intervalDomainLift (v τ)) x)^2 /
          (1 + intervalDomainLift (v τ) x) ^ (p.β + 1)| ≤ V₂
```

or an equivalent resolver-based bound. The existing per-time `resolverGrad2Real_bounded` is only an ingredient for that future uniform producer.

### Reaction estimate

Use `hfac.react_bound` initially. This is the least deceptive residual because the desired estimate is **not** a Cauchy estimate: Cauchy would give an unwanted `H1lapL2Norm` factor. The standard proof needs integration by parts:

```lean
H1PhysicalReactX p u τ
  = -∫ u_xx * (u * (a - b*u^α))
  = ∫ (a - b*(α+1)*u^α) * u_x^2
  ≤ L * ∫ u_x^2
  = L * (H1gradL2Norm u τ)^2
```

I do not see a landed theorem with this exact statement for `H1PhysicalReactX`. A good later task is:

```lean
theorem H1PhysicalReactX_le_gradSq_of_logistic_slope_bound
    {p : CM2Params} {T τ L : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hτ : τ ∈ Set.Ioo (0 : ℝ) T)
    (hL : p.a ≤ L) :
    H1PhysicalReactX p u τ ≤ L * (H1gradL2Norm u τ) ^ 2
```

This should be source-side and non-circular. It uses `p.hb`, `p.hα`, positivity of `u`, Neumann boundary for `u_x`, and interval integration by parts. It should not use the H¹ identity route, RHS integrability, or bounded-before.

## Red flags

Reject any proof that:

* treats `H1PhysicalRHSYoungScalarZeroMajorantsBefore` or zero-window spatial Young data as sqrt bounds;
* uses `H1PhysicalRHSStrictInitialRouteBefore`, `H1SupBoundSqrtRHSIntegrableBefore`, `H1IdentityRHSIntegrableBefore`, or bounded-before data to fill `H1PhysicalRHSSqrtBoundsBefore`;
* obtains fixed `V₁`, `V₂`, or `M` from per-time existential bounds without an explicit uniform-before-`T` assumption;
* proves the reaction estimate by Cauchy, producing an `H1lapL2Norm` factor and then silently absorbing it;
* hides the reaction integration-by-parts lemma inside the sqrt-bound producer.

The honest next step is a new `IntervalChiNegH1PhysicalSqrtBounds.lean` with the factor-bound residual and the Cauchy/IBP reducers above.
