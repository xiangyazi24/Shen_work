# Q4344 — Mathlib one-sided fencing / MeanValue lemma

The exact Mathlib theorem you want is:

```lean
image_le_of_deriv_right_le_deriv_boundary
```

It is in:

```lean
import Mathlib.Analysis.Calculus.MeanValue
```

## Shape

The relevant statement is the non-strict right-derivative comparison theorem:

```lean
theorem image_le_of_deriv_right_le_deriv_boundary
    {f f' : ℝ → ℝ} {a b : ℝ}
    (hf : ContinuousOn f (Set.Icc a b))
    (hf' : ∀ x ∈ Set.Ico a b,
      HasDerivWithinAt f (f' x) (Set.Ici x) x)
    {B B' : ℝ → ℝ}
    (ha : f a ≤ B a)
    (hB : ContinuousOn B (Set.Icc a b))
    (hB' : ∀ x ∈ Set.Ico a b,
      HasDerivWithinAt B (B' x) (Set.Ici x) x)
    (bound : ∀ x ∈ Set.Ico a b, f' x ≤ B' x)
    ⦃x : ℝ⦄ :
    x ∈ Set.Icc a b → f x ≤ B x
```

This is exactly the one-sided fencing lemma:

```text
f(a) ≤ B(a),
f'_right ≤ B'_right on [a,b),
f and B continuous on [a,b]
--------------------------------
f ≤ B on [a,b].
```

Use this when you have actual right derivatives, expressed as:

```lean
HasDerivWithinAt f (f' x) (Set.Ici x) x
```

for `x ∈ Set.Ico a b`.

## Dini / liminf-slope variant

If the energy argument gives only a right Dini/liminf slope bound rather than an actual right derivative, use:

```lean
image_le_of_liminf_slope_right_le_deriv_boundary
```

Its shape is:

```lean
theorem image_le_of_liminf_slope_right_le_deriv_boundary
    {f : ℝ → ℝ} {a b : ℝ}
    (hf : ContinuousOn f (Set.Icc a b))
    {B B' : ℝ → ℝ}
    (ha : f a ≤ B a)
    (hB : ContinuousOn B (Set.Icc a b))
    (hB' : ∀ x ∈ Set.Ico a b,
      HasDerivWithinAt B (B' x) (Set.Ici x) x)
    (bound : ∀ x ∈ Set.Ico a b, ∀ r, B' x < r →
      ∃ᶠ z in nhdsWithin x (Set.Ioi x), slope f x z < r)
    ⦃x : ℝ⦄ :
    x ∈ Set.Icc a b → f x ≤ B x
```

This is the right tool if your hypothesis is phrased as a liminf/right-slope upper bound rather than a proved `HasDerivWithinAt` for `f`.

## Strict/contact variant

If the available bound is strict only on the contact set `f x = B x`, use:

```lean
image_le_of_deriv_right_lt_deriv_boundary'
```

or the non-primed variant:

```lean
image_le_of_deriv_right_lt_deriv_boundary
```

The primed version uses right derivatives for both `f` and `B` on `[a,b)`. The non-primed variant assumes `B` has a full derivative everywhere.

## Recommended use for the energy majorant

For an integral Gronwall majorant `B`, usually use:

```lean
apply image_le_of_deriv_right_le_deriv_boundary
```

with:

```lean
f := E
B := fun t => C * ∫ s in a..t, E s
```

Then provide:

```lean
hf   : ContinuousOn E (Set.Icc a b)
hf'  : ∀ x ∈ Set.Ico a b, HasDerivWithinAt E (E' x) (Set.Ici x) x
ha   : E a ≤ B a
hB   : ContinuousOn B (Set.Icc a b)
hB'  : ∀ x ∈ Set.Ico a b, HasDerivWithinAt B (C * E x) (Set.Ici x) x
bound : ∀ x ∈ Set.Ico a b, E' x ≤ C * E x
```

For a pure Dini-slope energy inequality, replace the theorem with:

```lean
image_le_of_liminf_slope_right_le_deriv_boundary
```

## Bottom line

Use:

```lean
image_le_of_deriv_right_le_deriv_boundary
```

for the clean right-derivative comparison, and use:

```lean
image_le_of_liminf_slope_right_le_deriv_boundary
```

if the proof only has a right Dini / liminf slope bound.
