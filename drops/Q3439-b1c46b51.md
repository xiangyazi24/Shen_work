# Q3439 route audit: zero-time lap/H2 trace atom

## Verdict

The honest next producer target is **B**: produce the zero-start slab representative from construction-level or mild/Picard data, via a continuous physical RHS representative and interior PDE equality.

The target should not be “lap continuity at zero from classical solution” and should not be “lap continuity at zero from H1 energy convergence.” Those are too strong unless the relevant H2/second-derivative trace is explicitly present in the hypotheses.

The current atom is correctly isolated:

- `H1LapComponentZeroRightContinuous u := ContinuousWithinAt (fun τ => lapL2sq u τ) (Ici 0) 0`;
- `H1LiftDeriv2ZeroSlabRepresentative u F b`, with a continuous zero-start slab representative and interior `EqOn` to `liftDeriv2 u`.

The next producer should build the second package. It should not assume the first package, since that is what the bridge is meant to derive.

## Candidate audit

### A. Directly from classical solution plus initial H1 endpoint data

As stated, this is not mathematically honest.

`IsPaper2ClassicalSolution intervalDomain p T u v` has already produced the positive-time/physical-representative regularity needed for `H1UxxL1ContBefore`, but the remaining failure is precisely zero-time lap continuity. If the classical-solution record does not already contain a continuous zero-start representative for `liftDeriv2 u`, or an H2 trace at `t = 0`, then this route silently assumes the missing fact.

Initial H1 endpoint data is also too weak. H1 control gives information about `u` and `u_x` in L2, not continuity of `u_xx` in L2 as `t -> 0+`. Parabolic smoothing for `t > 0` can even coexist with blow-up of higher norms as `t -> 0+` unless the initial datum has matching higher regularity.

A direct theorem from classical data is acceptable only if its hypotheses explicitly include one of the following:

- a zero-start continuous representative for `liftDeriv2 u`;
- an H2/lap trace for the initial datum plus compatibility with the solution trace;
- construction data proving a continuous physical RHS up to `t = 0`.

Without one of those, A is an overclaim.

### B. From mild/Picard construction data and a zero-start physical RHS representative

This is the best producer target.

The intended PDE identity is not “literal lifted chemotaxisDiv agrees at the endpoint.” The robust identity is an interior physical-representative identity of the form:

`liftDeriv2 u = u_t + chi * chemRep - reaction`

on `(0,b] × (0,1)` or `Ioo 0 b × Ioo 0 1`, with the right-hand side represented by a continuous function on the closed slab `Icc 0 b × Icc 0 1`.

The first target should be a bridge lemma with this prose signature:

**`H1LiftDeriv2ZeroSlabRepresentative_of_zeroStartPhysicalRHSRep`**

Inputs:

- `0 < b`;
- a function `F : ℝ × ℝ -> ℝ`, intended as the physical RHS representative;
- `ContinuousOn (Function.uncurry F) (Set.Icc 0 b ×ˢ Set.Icc 0 1)`;
- an interior `EqOn` saying `liftDeriv2 u` agrees with `F` on the physical interior, e.g. `Set.Icc 0 b ×ˢ Set.Ioo 0 1` with the time endpoint handled as appropriate, or on `Set.Ioo 0 b ×ˢ Set.Ioo 0 1` if the existing bridge only needs positive times for equality;
- the exact endpoint/spatial-domain side conditions already used by the Task74 bridge.

Conclusion:

- `H1LiftDeriv2ZeroSlabRepresentative u F b`.

Then the actual construction-level producer should be a second theorem:

**`H1LiftDeriv2ZeroSlabRepresentative_of_mildPicard_zeroStartPhysicalRHS`**

Inputs should be construction data, not final estimates:

- `IsPaper2ClassicalSolution intervalDomain p T u v`, only for the interior PDE identity and compatibility with the existing formal API;
- `0 < b`, preferably with `b < T` if the construction data is only before `T`;
- zero-start continuous representatives on `Icc 0 b × Icc 0 1` for the pieces of the physical RHS: `u_t`, the chemotaxis divergence representative, and the reaction term;
- interior EqOn/PDE equality identifying the physical RHS with `liftDeriv2 u` on the open spatial interior;
- parameter/solution regularity needed to form the continuous products in the physical RHS.

Conclusion:

- `∃ F, H1LiftDeriv2ZeroSlabRepresentative u F b`, or directly `H1LiftDeriv2ZeroSlabRepresentative u F b` if `F` is fixed by the construction record.

This is honest because a continuous representative on a compact zero-start slab gives exactly the missing endpoint control. It also follows the successful pattern from `H1UxxL1ContBefore_of_classical_liftChemotaxisDivPhysicalRep`: use a physical representative and interior equality, rather than demanding endpoint equality of a literal lifted expression.

What must not be assumed in B:

- `H1LapComponentZeroRightContinuous u`;
- `H1IdentityRHSComponentsContinuousBefore`;
- `ContinuousOn (fun τ => lapL2sq u τ) (Icc 0 b)`;
- `IsPaper2BoundedBefore intervalDomain T u`;
- the final H1 bound being proved by the current route;
- endpoint equality of the literal lifted chemotaxis divergence if only an interior physical representative is available;
- an H2 trace hidden inside a “classical” hypothesis without naming it.

The main circularity check is whether the mild/Picard construction data was itself obtained using the H1 route or bounded-before theorem. If yes, this is circular. If it is local well-posedness/construction regularity independent of the global H1 closure, it is the right producer.

### C. Avoid zero-time lap continuity by using open intervals and a limiting argument

This is a legitimate alternate architecture, but it is not a producer for the current atom.

It may be mathematically better if the paper-level initial data is only H1 and no H2/lap trace at zero is available. In that case, requiring `H1IdentityRHSComponentsContinuousBefore` on closed windows starting at zero may indeed be too strong.

A faithful C-route would refactor the scalar DI layer to require only positive-time slabs:

- H1 identity/RHS component continuity on `Icc a b` for `0 < a <= b < T`;
- derivative inequality for every `r` with `0 < r < T`;
- enough local H1 seed near zero to avoid needing the derivative formula at `t = 0`;
- an improper/limit argument as `a -> 0+` only where the window starts at zero.

This avoids proving H2 trace at zero, but it is not free. It requires proving that all constants in the positive-time estimates are independent of the lower cutoff, and that the scalar average/window argument survives the `a -> 0+` limit. In particular, the `τ = 1` window `[0,1]` must be handled by a limit from `[ε,1]` plus local seed/integrability control; it cannot be handled by pretending closed-window lap continuity exists.

So C is viable as a fallback if B’s construction-level zero-slab representative is unavailable or analytically false under the paper hypotheses. But it is a larger API refactor, not the next smallest producer for Task74.

### D. Prove `H1InitialTraceEnergyTendsto` first and derive lap continuity from scalar H1 regularity

This is the clearest dead end if used to prove lap zero-right continuity.

`H1InitialTraceEnergyTendsto` would control `H1energy u τ` as `τ -> 0+`. That is useful for the local H1 seed and for scalar Gronwall/initial energy bookkeeping. It does not imply

`ContinuousWithinAt (fun τ => lapL2sq u τ) (Ici 0) 0`.

H1 convergence does not imply H2 convergence. Even a scalar H1 differential inequality bounds first-derivative energy, not the L2 norm of the second derivative at the endpoint. Deriving lap continuity from H1 energy convergence would smuggle in elliptic/parabolic regularity at exactly the missing strength.

D may be a worthwhile independent lemma for the local seed or initial-trace branch, but it should not be used as the zero-time lap/H2 trace producer.

## Recommended next Lean targets

### Target 1: a physical-RHS-to-zero-slab-representative bridge

Prose theorem:

**`H1LiftDeriv2ZeroSlabRepresentative_of_zeroStartPhysicalRHSRep`**

Given `0 < b`, a continuous zero-start slab representative `F` on `Icc 0 b × Icc 0 1`, and an interior `EqOn` identifying `F` with `liftDeriv2 u`, prove:

`H1LiftDeriv2ZeroSlabRepresentative u F b`.

This theorem should be small and non-PDE-heavy. It packages the exact atom that Task74 wants and keeps the physical representative strategy explicit.

### Target 2: a construction-data producer for the representative

Prose theorem:

**`exists_H1LiftDeriv2ZeroSlabRepresentative_of_mildPicard_zeroStartPhysicalRHS`**

Given local construction data near `t = 0` that provides continuous zero-start representatives for `u_t`, chemotaxis divergence, and reaction on a closed slab, plus the interior PDE equality, prove:

`∃ b > 0, ∃ F, H1LiftDeriv2ZeroSlabRepresentative u F b`.

If the horizon is already fixed, use:

`∀ b, 0 < b -> b < T -> ∃ F, H1LiftDeriv2ZeroSlabRepresentative u F b`.

The existential version is often more honest for local construction data; the universal-before-`T` version is stronger and should be used only if the construction record really supplies continuous zero-start representatives on every closed slab `[0,b]` with `b < T`.

### Target 3: the existing bridge to lap continuity

After Target 1/2, continue using the already introduced Task74 bridge:

`H1LiftDeriv2ZeroSlabRepresentative -> H1LapComponentZeroRightContinuous`

and then combine with positive-time strict-slab regularity to obtain the closed-window endpoint lap continuity needed by `H1IdentityRHSComponentsContinuousBefore`.

## Route ranking

1. **B is the recommended route.** It targets the missing H2/lap endpoint atom at the right strength and matches the physical-representative method that already worked for `H1UxxL1ContBefore`.

2. **C is the best fallback if the paper hypotheses do not justify zero-time H2/lap continuity.** It is honest but larger: it refactors the scalar DI/RHS continuity API away from closed zero-start windows.

3. **A is unsafe as stated.** It is only valid if the “classical solution plus endpoint data” hypotheses explicitly contain H2/lap trace or zero-start second-derivative representative data.

4. **D is a dead end for this atom.** H1 initial trace convergence does not imply lapL2 zero-right continuity.

## Bottom line

Do not try to manufacture zero-time lap continuity from H1 endpoint control. The next honest producer is a zero-start physical RHS representative coming from mild/Picard construction data, with interior PDE equality to `liftDeriv2 u`. If that construction data is not available under the original paper hypotheses, the mathematically honest alternative is to weaken the scalar DI/RHS-continuity route to positive-time slabs plus a cutoff limit, rather than asserting a false H2 trace at time zero.
