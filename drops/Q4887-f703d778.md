# Problem 2.7: there is no untwisted polynomial Ore transformation \(W_n\mapsto q_n\)

Let

\[
W_n=\frac1{256^n}\sum_{k=0}^{2n}\binom{2n}{k}(-2)^{2n-k}T_k,
\]

where \(T_k\) is Cooper's level-\(11\) sequence.  Use the Problem 2.7 recurrence in the sign convention compatible with the quoted Poincaré cubic,

\[
A_nu_{n+3}-B_nu_{n+2}+C_nu_{n+1}-D_nu_n=0.
\tag{0.1}
\]

The main conclusion is exact:

\[
\boxed{
\text{There are no }r\ge0\text{ and }P_0,\ldots,P_r\in\mathbf Q[n]
\text{ for which }
q_n=\sum_{j=0}^rP_j(n)W_{n+j}
\text{ for all sufficiently large }n.}
\tag{0.2}
\]

The same statement remains true with \(P_j(n)\in\mathbf Q(n)\).  Thus the requested minimal order \(r\) and polynomial coefficients do not exist.  The inference

\[
q_n/W_n\text{ is nonconstant}
\quad\Longrightarrow\quad
q_n=\sum_jP_j(n)W_{n+j}
\]

is false.

The obstruction is the half-integral formal exponent.  Every Cooper mode, and hence every centered-binomial/even mode contributing to \(W_n\), has formal power \(-3/2\pmod{\mathbf Z}\).  Every Problem 2.7 mode has formal power \(0\pmod{\mathbf Z}\).  A rational Ore operator changes a formal power only by an integer, so it cannot connect these two modules.

The correct possible form is instead

\[
\boxed{
q_n=h_n\sum_{j=0}^r u_j(n)W_{n+j},
\qquad
u_j(n)\in\mathbf Q(n),
\qquad
\frac{h_{n+1}}{h_n}
=1+\frac{3}{2n}+O(n^{-2}).}
\tag{0.3}
\]

A natural first Kummer/gamma twist is

\[
h_n=\frac{(5/2)_n}{n!},
\qquad
\frac{h_{n+1}}{h_n}=\frac{2n+5}{2n+2},
\tag{0.4}
\]

but (0.4) is only the simplest twist with the necessary exponent.  Its exact compatibility must be tested against the full transformed operator; the Poincaré polynomial does not determine it.

# 1. Exact formal exponent of Cooper's sequence

Cooper's recurrence is

\[
\begin{aligned}
(n+1)^3T_{n+1}={}&2(2n+1)(5n^2+5n+2)T_n\\
&-8n(7n^2+1)T_{n-1}
+22n(2n-1)(n-1)T_{n-2}.
\end{aligned}
\tag{1.1}
\]

Insert the formal ansatz

\[
T_n=t^n n^\sigma\left(1+O(n^{-1})\right).
\tag{1.2}
\]

The leading equation is

\[
t-20+\frac{56}{t}-\frac{44}{t^2}=0,
\]

or

\[
H_{11}(t)=t^3-20t^2+56t-44=0.
\tag{1.3}
\]

The coefficient of \(n^{-1}\), after division by the leading \(n^3t^n\), is

\[
\sigma\left(t-\frac{56}{t}+\frac{88}{t^2}\right)
+3t-30+\frac{66}{t^2}=0.
\tag{1.4}
\]

Putting \(\sigma=-3/2\) turns the left side into

\[
\frac{3}{2}t-30+\frac{84}{t}-\frac{66}{t^2}
=\frac{3}{2t^2}H_{11}(t)=0.
\]

The roots of \(H_{11}\) are simple, so the coefficient multiplying \(\sigma\) in (1.4) does not vanish simultaneously with the characteristic equation.  Hence every formal Cooper mode has

\[
\boxed{\sigma_T=-\frac32.}
\tag{1.5}
\]

# 2. The centered-binomial/even transform preserves the exponent class

Let

\[
Z(X)=\sum_{n\ge0}T_nX^n.
\]

The ordinary shifted binomial transform has generating function

\[
B(r)=\frac1{1+2r}
Z\!\left(\frac{r}{1+2r}\right).
\tag{2.1}
\]

The required even section is

\[
\mathcal W(z)=\sum_{n\ge0}W_nz^n
=\frac12\left[
B\!\left(\frac{\sqrt z}{16}\right)
+B\!\left(-\frac{\sqrt z}{16}\right)
\right].
\tag{2.2}
\]

For a Cooper singularity \(X=1/t\), the rational substitution in (2.1) gives

\[
r=\frac{X}{1-2X}=\frac1{t-2},
\qquad
z=256r^2=\frac{256}{(t-2)^2}.
\]

The corresponding coefficient ratio is therefore

\[
\lambda=\frac{(t-2)^2}{256},
\]

which is the established spectral map.

More importantly, the substitution \(X=r/(1+2r)\) is unramified at every finite singularity involved here, and the prefactor \((1+2r)^{-1}\) is regular and nonzero there.  It preserves the local singular exponent.  Passing to the even trace can cancel leading local coefficients, but such a cancellation raises a local exponent only by an integer.  Therefore every formal sequence exponent occurring in \(W_n\) lies in

\[
\boxed{-\frac32+\mathbf Z.}
\tag{2.3}
\]

For the dominant component there is no complete cancellation—\(W_n\) has the observed dominant root—so its leading power is the expected \(-3/2\).

This also explains the observed behavior

\[
\frac{q_n}{W_n}\asymp n^{3/2}.
\]

It is not evidence for a polynomial shift combination; it is evidence for a rank-one Kummer twist.

# 3. Exact formal exponent of the Problem 2.7 recurrence

Write the four coefficients as

\[
\begin{aligned}
A_n&=a\,n^{12}\left(1+\frac\beta n+O(n^{-2})\right),\\
B_n&=b\,n^{12}\left(1+\frac\beta n+O(n^{-2})\right),\\
C_n&=c\,n^{12}\left(1+\frac\beta n+O(n^{-2})\right),\\
D_n&=d\,n^{12}\left(1+\frac\beta n+O(n^{-2})\right).
\end{aligned}
\tag{3.1}
\]

Direct expansion of the displayed factors gives

\[
\begin{aligned}
a&=991952896,\\
b&=852459520,\\
c&=484352,\\
d&=946,
\end{aligned}
\qquad
\boxed{\beta=\frac{38571}{946}}
\tag{3.2}
\]

for **all four coefficients**.  For example,

\[
\frac{[n^{11}]A_n}{[n^{12}]A_n}
=4\frac52+3\frac72+3\frac92+\frac{6407}{946}
=\frac{38571}{946},
\]

and the analogous computations for \(B,C,D\) give the same value.

Insert

\[
u_n=\lambda^n n^\tau\left(1+O(n^{-1})\right)
\]

into (0.1).  The leading polynomial is

\[
P(\lambda)=a\lambda^3-b\lambda^2+c\lambda-d.
\tag{3.3}
\]

At order \(n^{-1}\), the common coefficient \(eta\) contributes \(eta P(\lambda)\), while the shifts contribute \(	au\lambda P'(\lambda)\).  Thus

\[
\beta P(\lambda)+\tau\lambda P'(\lambda)=0.
\tag{3.4}
\]

For a characteristic root, \(P(\lambda)=0\); the cubic is separable, so \(\lambda P'(\lambda)\ne0\).  Consequently

\[
\boxed{\tau_{27}=0}
\tag{3.5}
\]

for all three Problem 2.7 formal modes.

In the scaled variable \(\mu=64\lambda\), equation (3.3) is a nonzero scalar multiple of

\[
4\mu^3-220\mu^2+8\mu-1.
\]

# 4. Rational Ore operators preserve formal powers modulo integers

Let

\[
U=\sum_{j=0}^r u_j(n)S^j,
\qquad u_j(n)\in\mathbf Q(n),
\qquad (Sf)_n=f_{n+1}.
\tag{4.1}
\]

At infinity every nonzero rational coefficient has a Laurent expansion

\[
u_j(n)=c_jn^{m_j}\left(1+O(n^{-1})\right),
\qquad m_j\in\mathbf Z.
\]

Applying \(U\) to a formal mode gives

\[
U\bigl(\lambda^nn^\sigma(1+O(n^{-1}))\bigr)
=\lambda^nn^{\sigma+m}
\left(C_0+\frac{C_1}{n}+\cdots\right)
\tag{4.2}
\]

for some integer \(m\).  If \(C_0=0\), further cancellations lower the power by another integer.  Therefore a nonzero rational Ore operator preserves the exponent class

\[
\boxed{\sigma\pmod{\mathbf Z}.}
\tag{4.3}
\]

The dominant \(W\)-mode has class

\[
-\frac32+\mathbf Z=\frac12+\mathbf Z,
\]

whereas the dominant \(q\)-mode has class \(0+\mathbf Z\).  They cannot be related by (4.1).  Since they have the same dominant exponential root, the escape in which \(U\) kills the dominant root and leaves a different exponential mode is also impossible.

This proves (0.2) for every order and every polynomial degree; the 103-term linear-algebra search is unnecessary for the no-go theorem.

# 5. Consequence for factorization of the transformed operator

The statement

> “find the minimal annihilator of \(W\), then factor it by the Problem 2.7 operator”

is **not equivalent** to the polynomial-projector question.

The exact centered-binomial/even construction gives an annihilator of order at most six.  Let

\[
\mathcal L_{11}
=\theta_X^3
-2X(2\theta_X+1)(5\theta_X^2+5\theta_X+2)
+8X^2(\theta_X+1)(7(\theta_X+1)^2+1)
-22X^3(\theta_X+1)(\theta_X+2)(2\theta_X+3).
\tag{5.1}
\]

Put

\[
X_+(r)=\frac r{1+2r},
\qquad g_+(r)=\frac1{1+2r}.
\]

Because

\[
B(r)=g_+(r)Z(X_+(r)),
\qquad
\theta_X=(1+2r)\theta_r,
\]

an exact third-order annihilator of \(B(r)\) is the Ore conjugate

\[
\boxed{
\mathcal B_+(r)
=g_+(r)\,
\mathcal L_{11}\!\left(
X=\frac r{1+2r},
\theta_X=(1+2r)\theta_r
\right)g_+(r)^{-1}.}
\tag{5.2}
\]

Let

\[
\mathcal B_-(r)=\mathcal B_+(r)\big|_{r\mapsto-r,\;\theta_r\mapsto\theta_r}.
\tag{5.3}
\]

Then

\[
\mathcal N_r=\operatorname{LCLM}(\mathcal B_+,\mathcal B_-)
\tag{5.4}
\]

annihilates \(B(r)+B(-r)\) and has order at most six.  Substitute

\[
r^2=\frac z{256},
\qquad \theta_r=2\theta_z,
\]

and descend the invariant coefficients to \(\mathbf Q(z)\langle\theta_z\rangle\).  The resulting operator is an exact annihilator of \(\mathcal W(z)\).  Its coefficient recurrence is an exact annihilator of \(W_n\).

The minimal operator \(L_W\) is obtained by removing any right factor that does not annihilate the initial series.  Failure to find an order-three recurrence through degree \(19\) is evidence that the generic order is six, but it is not itself a proof of minimality.

The formal-exponent calculation gives a stronger factor statement:

\[
\boxed{
L_{27}\text{ is not an untwisted rational right factor of }L_W.}
\tag{5.5}
\]

Indeed, a right factor's formal exponent classes must occur among those of the full operator.  \(L_{27}\) has class \(0\pmod{\mathbf Z}\), whereas the centered-Cooper pushforward has class \(1/2\pmod{\mathbf Z}\).

Thus any genuine factorization must first apply a half-integral rank-one twist.

# 6. The corrected Ore problem

Let

\[
h_{n+1}=\rho(n)h_n,
\qquad
\rho(n)\in\mathbf Q(n),
\qquad
\rho(n)=1+\frac{3}{2n}+O(n^{-2}).
\tag{6.1}
\]

If

\[
L_W=\sum_{j=0}^s b_j(n)S^j,
\]

then \(v_n=h_nW_n\) is annihilated by

\[
\boxed{
L_W^{[\rho]}
=\sum_{j=0}^s
\frac{b_j(n)}{H_j(n)}S^j,
\qquad
H_0=1,
\quad
H_j(n)=\prod_{k=0}^{j-1}\rho(n+k).}
\tag{6.2}
\]

Only after this twist should one seek an intertwiner

\[
\boxed{
L_{27}U=V L_W^{[\rho]},
\qquad
U=\sum_{j=0}^r u_j(n)S^j,
\quad u_j(n)\in\mathbf Q(n).}
\tag{6.3}
\]

The distinguished denominator is then checked by

\[
q_n=U(hW)_n
\]

at three initial indices, after which the common order-three recurrence proves equality for every \(n\).

The candidate (0.4) has the required formal exponent, because

\[
\frac{(5/2)_n}{n!}
\sim\frac{n^{3/2}}{\Gamma(5/2)}.
\]

But formal data determine only the exponent \(3/2\), not the exact rational function \(ho(n)\).  The exact \(ho\) must satisfy the operator identities (6.3), or equivalently the companion-matrix equation

\[
C_{27}(n)R(n)
=\rho(n)R(n+1)C_W(n).
\tag{6.4}
\]

# 7. Exact Sage code for the finite computation

The following program does four things:

1. generates exact \(T_n,W_n,q_n\);
2. confirms that no bounded polynomial projector survives independent holdout terms;
3. guesses a minimal polynomial recurrence for \(W_n\);
4. repeats the projector search after the necessary twist \(h_n=(5/2)_n/n!\).

The formal-exponent theorem above proves the polynomial-only no-go for **all** bounds.  The finite search is for discovering the corrected twisted certificate.

```python
from sage.all import *

# ------------------------------------------------------------
# Exact sequences
# ------------------------------------------------------------

def A27(n):
    return 1024*(2*n+5)^4*(2*n+7)^3*(2*n+9)^3*(946*n^2+6407*n+10860)

def B27(n):
    return 128*(2*n+7)^3*(2*n+9)^3*(
        104060*n^6+1745370*n^5+12145238*n^4+44886481*n^3
        +92943995*n^2+102256019*n+46709052)

def C27(n):
    return 16*(n+3)^4*(2*n+9)^3*(
        3784*n^5+57792*n^4+351019*n^3+1059230*n^2
        +1587211*n+944620)

def D27(n):
    return (n+3)^4*(n+4)^6*(946*n^2+4515*n+5399)

q_init = [
    QQ(-215040420000),
    QQ(-167282265043404)/905,
    QQ(-964185327658080)/6071,
]

def q_terms(N):
    # Convention A_n q_{n+3}-B_n q_{n+2}+C_n q_{n+1}-D_n q_n=0.
    q = list(q_init)
    for n in range(N-3):
        q.append((B27(n)*q[n+2]-C27(n)*q[n+1]+D27(n)*q[n])/A27(n))
    return q[:N]

def cooper_terms(N):
    T = [ZZ(1), ZZ(4), ZZ(28)]
    for n in range(2, N-1):
        num = (
            2*(2*n+1)*(5*n^2+5*n+2)*T[n]
            -8*n*(7*n^2+1)*T[n-1]
            +22*n*(2*n-1)*(n-1)*T[n-2]
        )
        assert num % (n+1)^3 == 0
        T.append(num//(n+1)^3)
    return T[:N]

def W_terms(N):
    T = cooper_terms(2*N)
    return [
        QQ(sum(binomial(2*n,k)*(-2)^(2*n-k)*T[k]
               for k in range(2*n+1)), 256^n)
        for n in range(N)
    ]

def h_terms(N):
    h = [QQ(1)]
    for n in range(N-1):
        h.append(h[-1]*QQ(2*n+5,2*n+2))
    return h

# ------------------------------------------------------------
# Polynomial-recurrence guesser with independent holdout
# ------------------------------------------------------------

def relation_matrix(seqs, shifts, degree, rows):
    # seqs is a list of sequences.  A column is n^m seq[n+j].
    cols = []
    labels = []
    for name, seq in seqs:
        for j in shifts[name]:
            for m in range(degree+1):
                labels.append((name,j,m))
                cols.append([QQ(n)^m*seq[n+j] for n in rows])
    return matrix(QQ, len(rows), len(cols),
                  lambda i,k: cols[k][i]), labels

def eval_relation(v, labels, seqdict, n):
    return sum(v[k]*QQ(n)^m*seqdict[name][n+j]
               for k,(name,j,m) in enumerate(labels))

def guess_annihilator(W, max_order=8, max_degree=30, holdout=20):
    for r in range(1,max_order+1):
        for d in range(max_degree+1):
            unknowns = (r+1)*(d+1)
            if unknowns+holdout > len(W)-r:
                continue
            train = list(range(unknowns+2))
            M,labels = relation_matrix(
                [('W',W)], {'W':range(r+1)}, d, train)
            K = M.right_kernel()
            if K.dimension() != 1:
                continue
            v = K.basis()[0]
            if all(eval_relation(v,labels,{'W':W},n)==0
                   for n in range(unknowns+2,len(W)-r)):
                return r,d,v,labels
    return None

def guess_projector(W,target,max_order=6,max_degree=24,holdout=20):
    # Homogeneous relation sum P_j(n)W[n+j]-E(n)target[n]=0.
    # A nonzero E gives a rational Ore projector.
    for r in range(max_order+1):
        for d in range(max_degree+1):
            unknowns = (r+2)*(d+1)
            if unknowns+holdout > len(W)-r:
                continue
            train = list(range(unknowns+2))
            M,labels = relation_matrix(
                [('W',W),('Q',target)],
                {'W':range(r+1),'Q':[0]}, d, train)
            K = M.right_kernel()
            good = []
            for v in K.basis():
                # Require the target polynomial E to be nonzero.
                if all(v[k]==0 for k,l in enumerate(labels) if l[0]=='Q'):
                    continue
                if all(eval_relation(v,labels,{'W':W,'Q':target},n)==0
                       for n in range(unknowns+2,len(W)-r)):
                    good.append(v)
            if len(good)==1:
                return r,d,good[0],labels
    return None

N = 140
W = W_terms(N)
q = q_terms(N)
h = h_terms(N)
q_twisted_down = [q[n]/h[n] for n in range(N)]

print('Untwisted polynomial/rational search (expected to fail):')
print(guess_projector(W,q))

print('Twisted search for q_n/h_n as a rational Ore image of W_n:')
print(guess_projector(W,q_twisted_down))

print('Minimal recurrence search for W:')
rec = guess_annihilator(W)
print(rec[:2] if rec else None)
```

A candidate returned by this script is still only a discovery object until it is:

- verified on more exact terms than were used in the solve;
- converted to an Ore identity such as (6.3);
- and checked symbolically over \(\mathbf Q(n)\).

# 8. Bottom line

The exact answer to the question as posed is

\[
\boxed{\text{no polynomial-coefficient transformation }W\mapsto q\text{ exists}.}
\]

Therefore there is no minimal \(r\) and no list of polynomial \(P_j(n)\).  The minimal-annihilator problem for \(W\) is a separate order-at-most-six pushforward computation, and the Problem 2.7 operator is not an untwisted rational factor because their formal exponent classes differ by \(3/2\).

The next correct finite computation is:

1. construct the exact order-\(\le6\) annihilator \(L_W\) from (5.2)--(5.4), or guess and certify it from the exact terms;
2. determine a rational ratio \(ho(n)\) with exponent \(3/2\);
3. twist \(L_W\) by (6.2);
4. solve the rational Ore equation (6.3);
5. match the three initial values of \(q_n\).

The spectral identity determines the exponential roots.  The half-integral formal exponent shows why it cannot, by itself, produce the requested untwisted polynomial projector.
