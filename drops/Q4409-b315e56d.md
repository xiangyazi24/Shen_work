# Q4409: weighted / finite-`p` chemotaxis energy step for the Paper 2 `L^p` mountain

## Executive answer

The target lemma should be a **finite-exponent differential inequality** of the form

```text
y_p'(t) ≤ C_p * y_p(t) + C_p'
```

where

```text
y_p(t) = ∫_0^1 u(t,x)^p dx,
```

for every fixed `p > 1`.  The core identity is obtained by testing the `u` equation with `u^(p-1)`.  The bad term is the chemotaxis cross term

```text
χ₀ * (p - 1) ∫ u^(p-1) u_x * v_x / (1 + v)^β.
```

For the `0 < m ≤ 1` branch with nonlinear diffusion, the diffusion controls

```text
∫ u^(p+m-3) u_x^2
```

or equivalently

```text
∫ |∂ₓ(u^((p+m-1)/2))|^2.
```

Young absorbs half of the chemotaxis cross term into this diffusion.  The remaining piece has exponent

```text
u^(p+1-m)
```

up to the elliptic/sensitivity factor.  Since `0 < m ≤ 1`, the excess exponent is `1 - m ∈ [0,1)`.  Thus, under the usual logistic condition `α ≥ 1` — more sharply, `α ≥ 1 - m` — the negative logistic term

```text
-b ∫ u^(p+α)
```

absorbs the residual chemotaxis growth, leaving only `C_p * ∫u^p + C_p'`.

In the current interval-domain `Lean` vocabulary, this is exactly the analytic content behind `CrossDiffusionBootstrapEstimate` plus the final logistic absorption step in `IntervalDomainEnergyStep`.

## Clean PDE model and notation

Use the one-dimensional Neumann model in the form

```text
u_t = ∂ₓ(u^(m-1) u_x)
      - χ₀ ∂ₓ( u * v_x / (1 + v)^β )
      + u * (a - b u^α),
```

with

```text
0 < m ≤ 1,
β ≥ 1,
u ≥ 0,
v ≥ 0,
v_x = 0 at x = 0,1,
u_x = 0 at x = 0,1.
```

If the local API is the linear-diffusion interval-domain API, set `m = 1` in the formulas below.  Then the diffusion coefficient becomes the usual linear one.

Let

```text
A := v_x / (1 + v)^β.
```

Then the chemotaxis flux is `u*A`.

## Exact finite-`p` identity

For `p > 1`, multiply the `u` equation by `u^(p-1)` and integrate over `[0,1]`.  Neumann boundary terms vanish.  Formally,

```text
(1/p) d/dt ∫ u^p
  = -m(p-1) ∫ u^(p+m-3) u_x^2
    + χ₀(p-1) ∫ u^(p-1) u_x A
    + a ∫ u^p
    - b ∫ u^(p+α).
```

Equivalently, using

```text
∂ₓ(u^((p+m-1)/2))
  = ((p+m-1)/2) u^((p+m-3)/2) u_x,
```

the diffusion term is

```text
m(p-1) ∫ u^(p+m-3) u_x^2
  = 4m(p-1)/(p+m-1)^2
      ∫ |∂ₓ(u^((p+m-1)/2))|^2.
```

So the identity can be written as

```text
(1/p) y_p'(t)
  + D_{p,m} * G_{p,m}(t)
  = χ₀(p-1) ∫ u^(p-1) u_x A
    + a y_p(t)
    - b ∫ u^(p+α),
```

where

```text
D_{p,m} := 4m(p-1)/(p+m-1)^2,
G_{p,m}(t) := ∫ |∂ₓ(u^((p+m-1)/2))|^2.
```

For the linear-diffusion specialization `m = 1`, this becomes

```text
(1/p) y_p'(t)
  + 4(p-1)/p^2 * ∫ |∂ₓ(u^(p/2))|^2
  = χ₀(p-1) ∫ u^(p-1) u_x A
    + a y_p(t)
    - b ∫ u^(p+α).
```

## Chemotaxis cross-term absorption

The bad term is controlled by Young.  First rewrite

```text
u^(p-1) |u_x| |A|
  = (2/(p+m-1))
      |∂ₓ(u^((p+m-1)/2))|
      * u^((p-m+1)/2) |A|.
```

Hence

```text
|χ₀|(p-1) ∫ u^(p-1)|u_x||A|
  ≤ (D_{p,m}/2) * G_{p,m}(t)
    + (χ₀^2(p-1)/(2m)) ∫ u^(p+1-m) A^2.
```

This coefficient is convenient because the constants simplify exactly:

```text
K = 2|χ₀|(p-1)/(p+m-1),
D = 4m(p-1)/(p+m-1)^2,
K^2/(2D) = χ₀^2(p-1)/(2m).
```

Thus the energy identity gives

```text
(1/p) y_p'(t)
  + (D_{p,m}/2) * G_{p,m}(t)
  + b ∫ u^(p+α)
  ≤ a y_p(t)
    + Cχ(p,m) ∫ u^(p+1-m) A^2.
```

where

```text
Cχ(p,m) = χ₀^2(p-1)/(2m).
```

## Elliptic/sensitivity input

The branch-specific elliptic estimate should be packaged as either a pointwise sensitivity bound

```text
A^2 = |v_x|^2 / (1+v)^(2β) ≤ K_v^2,
```

or the more flexible already-repo-shaped bootstrap estimate

```text
∫ u^(p+1-m) A^2
  ≤ eps * G_{p,m}(t) + Ceps * ∫ u^(p+ρ)
```

for the appropriate `ρ`.  In the simplest bounded-gradient reading, `ρ = 1 - m`:

```text
∫ u^(p+1-m) A^2 ≤ K_v^2 ∫ u^(p+1-m).
```

This is the place where `β ≥ 1` is used.  Since `v ≥ 0`,

```text
(1+v)^(-β) ≤ 1,
```

and the signal-dependent sensitivity does not amplify the gradient.  In the paper branch, `β ≥ 1` is exactly what lets the elliptic estimate for `v` / `∇v` feed a finite constant into the cross-diffusion term.

In the repo, the abstract version of this step is the `CrossDiffusionBootstrapEstimate` consumed in `IntervalDomainEnergyStep`.  It is the correct target API for Lean: do not force the final proof to normalize the exact elliptic constant unless needed.

## Logistic absorption

After the cross estimate, the only remaining superlinear positive term has exponent

```text
p + 1 - m.
```

Because

```text
0 < m ≤ 1,
```

we have

```text
0 ≤ 1 - m < 1.
```

The logistic damping has exponent

```text
p + α.
```

So the sharp absorption condition is

```text
1 - m ≤ α.
```

The usual Paper 2 branch assumption `α ≥ 1` implies this automatically.  For every `η > 0` there is a constant `Cη` such that, pointwise for `z ≥ 0`,

```text
z^(p+1-m) ≤ η * z^(p+α) + Cη * (1 + z^p).
```

Therefore choose `η` small enough so that

```text
Cχ * K_v^2 * η ≤ b/2.
```

Then

```text
Cχ * K_v^2 ∫ u^(p+1-m)
  - b ∫ u^(p+α)
  ≤ - (b/2) ∫ u^(p+α)
    + C_p * (1 + ∫ u^p).
```

Thus

```text
(1/p) y_p'(t)
  + (D_{p,m}/2) * G_{p,m}(t)
  + (b/2) ∫ u^(p+α)
  ≤ C_p * y_p(t) + C_p'.
```

Dropping the nonnegative terms gives the finite-`p` Gronwall inequality

```text
y_p'(t) ≤ p*C_p*y_p(t) + p*C_p'.
```

This is the exact finite-exponent lemma the `L^p` producer needs.

For the CRS `0 < m ≤ 1` branch, set

```text
rho = 1 - m.
```

Then `rho ≤ α` follows from `α ≥ 1`.

For the current linear interval-domain statement, set

```text
m = 1,
rho = 0.
```

Then the cross-diffusion remainder is already at the same exponent as `y_p`.

## Where the branch hypotheses enter

The roles are:

```text
β ≥ 1
```

is used in the elliptic/sensitivity estimate for

```text
A = v_x / (1+v)^β.
```

It prevents the signal-dependent denominator from worsening the gradient term because `v ≥ 0` gives `(1+v)^(-β) ≤ 1`.

```text
0 < m ≤ 1
```

sets the cross-diffusion remainder exponent to

```text
p + 1 - m,
```

with `1 - m ∈ [0,1)`.  This makes the remainder lower than or equal to the logistic damping scale under `α ≥ 1`.

```text
α ≥ 1
```

is sufficient for the logistic absorption

```text
u^(p+1-m) ≲ η u^(p+α) + Cη(1+u^p).
```

```text
b > 0
```

is needed to have a genuinely negative high-power logistic term.

## Final target inequality

The final finite-`p` target should be stated as:

```text
For every p > 1, there exist C_p, C_p' ≥ 0 such that
  d/dt ∫_0^1 u(t,x)^p dx ≤ C_p ∫_0^1 u(t,x)^p dx + C_p'
for all interior times t.
```

Then the standard finite-time Gronwall step gives

```text
sup_{0 ≤ t ≤ T} ∫ u(t)^p < ∞
```

for that fixed `p`.  The Moser iteration / `L^p` mountain then iterates this finite-p estimate.

## Caution for Lean

Do not bake the exact elliptic constant into the top-level theorem.  Keep the source of the chemotaxis estimate abstract as either

```lean
CrossDiffusionBootstrapEstimate intervalDomain params T rho u v
```

or a named concrete lemma with the same shape.  The final `L^p` step only needs:

```text
cross ≤ eps * gradient + Ceps * ∫u^(p+rho)
```

followed by logistic absorption.  That is the smallest robust interface for `IntervalDomainEnergyStep`.
