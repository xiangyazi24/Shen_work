# Task150 audit: conjugate-kernel approximate identity for the homogeneous C¹ leg

## Verdict

The desired statement is mathematically the **Dirichlet heat-kernel approximate identity** in disguise:

```lean
|-(∫ y in (0 : ℝ)..1,
    g y * intervalNeumannConjugateKernel t x y) - g x| < ε
```

uniformly for `x ∈ Set.Icc 0 1` as `t → 0+`.

The endpoint compatibility needed for the **closed** interval is exactly

```lean
g 0 = 0 ∧ g 1 = 0
```

For the intended C¹ initial-leg use, where `g` is the initial derivative profile, this is the closed Neumann compatibility

```lean
du₀ 0 = 0 ∧ du₀ 1 = 0
```

No weaker endpoint condition can give the closed-`Icc` uniform theorem: at the Dirichlet faces `x = 0` and `x = 1`, the conjugate/Dirichlet operator should be identically zero for all `t > 0`, so its limit can equal `g x` only if the endpoint values vanish.

On current `main`, I do **not** see a compile-now proof of this from mere continuity plus `g 0 = 0`, `g 1 = 0`.  Task149 correctly introduced

```lean
InitialLegConjugateDerivativeApprox
```

in `ShenWork/PDE/IntervalSemigroupC1ApproxIdentity.lean`, and the source-IBP reducer consumes it.  But the repository still lacks the analytic Dirichlet/conjugate approximate-identity infrastructure that would produce this hypothesis from `ContinuousOn g (Set.Icc 0 1)` and zero endpoint values.

## Why existing Neumann approximate identity is insufficient

`IntervalSemigroupUniform.lean` proves the full Neumann kernel uniform approximate identity:

```lean
theorem intervalFullSemigroup_tendstoUniformlyOn
    (f : ℝ → ℝ) (hf : Continuous f) :
    TendstoUniformlyOn (fun t x => intervalFullSemigroupOperator t f x) f
      (𝓝[>] (0 : ℝ)) (Set.Icc 0 1)
```

That theorem is for the positive full Neumann kernel

```lean
K_N = A + B
```

where, schematically,

```lean
A t x y = ∑' k : ℤ, heatKernel t (x - y + 2 * (k : ℝ))
B t x y = ∑' k : ℤ, heatKernel t (x + y + 2 * (k : ℝ))
```

The conjugate expression is the Dirichlet kernel:

```lean
-intervalNeumannConjugateKernel = A - B
```

Thus the desired theorem is not a direct corollary of the Neumann approximate identity.  One must additionally show that the reflected leg

```lean
∫ y in (0 : ℝ)..1, g y * B t x y
```

tends to `0` uniformly when `g 0 = 0` and `g 1 = 0`.  Current `main` has no named `B` kernel, no reflected-leg vanishing theorem, and no Dirichlet-kernel approximate identity theorem.

The bound

```lean
|intervalNeumannConjugateKernel t x y| ≤ intervalNeumannFullKernel t x y
```

only gives an `L¹` bound.  It loses the sign/cancellation and does not supply the missing mass/concentration behavior needed to prove convergence to `g x`.

## Smallest compile-now safe reducer

The following theorem is compile-now and non-circular: it only converts the natural filter-form analytic target into the Task149 epsilon-delta hypothesis `InitialLegConjugateDerivativeApprox`.

Add it to:

```text
ShenWork/PDE/IntervalSemigroupC1ApproxIdentity.lean
```

No new imports are needed there.  Standalone check file:

```lean
import ShenWork.PDE.IntervalSemigroupC1ApproxIdentity

open MeasureTheory Filter Topology
open scoped Topology

open ShenWork.IntervalDomain
open ShenWork.IntervalNeumannFullKernel

namespace ShenWork.IntervalSemigroupC1ApproxIdentity

noncomputable section

/-- A filter-form uniform conjugate/Dirichlet approximate identity immediately
supplies the epsilon-delta hypothesis consumed by the C¹ initial-leg reducer. -/
theorem initialLegConjugateDerivativeApprox_of_tendstoUniformlyOn
    {g : ℝ → ℝ}
    (hlim : TendstoUniformlyOn
      (fun t x : ℝ =>
        -(∫ y in (0 : ℝ)..1,
          g y * intervalNeumannConjugateKernel t x y))
      g (𝓝[>] (0 : ℝ)) (Set.Icc (0 : ℝ) 1)) :
    InitialLegConjugateDerivativeApprox g := by
  intro ε hε
  rw [Metric.tendstoUniformlyOn_iff] at hlim
  have hev := hlim ε hε
  rw [Filter.eventually_iff, mem_nhdsGT_iff_exists_Ioo_subset] at hev
  rcases hev with ⟨δ, hδpos, hδsub⟩
  refine ⟨δ, hδpos, ?_⟩
  intro t ht htδ x hx
  have hdist := hδsub ⟨ht, htδ⟩ x hx
  simpa [Real.dist_eq] using hdist

end

end ShenWork.IntervalSemigroupC1ApproxIdentity
```

## Recommendation

Do not try to prove `InitialLegConjugateDerivativeApprox` from `Continuous g` alone.  The correct closed-interval theorem requires zero boundary values:

```lean
g 0 = 0 ∧ g 1 = 0
```

The next safe source-side move is to add the compile-now reducer above, then implement the reflected-leg analytic theorem.  The reflected-leg theorem is the smallest non-circular missing analytic target because it lets the already-proved full Neumann uniform approximate identity carry the main `A+B` concentration, while the new work only proves that the boundary-reflected `B` contribution dies for zero-boundary data.
