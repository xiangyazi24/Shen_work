# Problem 2.5: exact creative-telescoping test for the Jacobi kernel

## Executive conclusion

There are two corrections to make before searching for a certificate.

First, the logarithmic Catalan weight is **not hyperexponential**. Therefore a single rational function \(C_n(t)\) in

\[
\sum_{j=0}^3 c_j(n)R_{n+j}^{J}(t^2)
=rac{d}{dt}\!\left[t(1-t^2)C_n(t)rac{-\log t}{1+t^2}ight]
\]

is generally the wrong ansatz: after division by the weight, differentiating the right-hand side produces a \(1/\log t\) term. The correct continuous certificate has two rational components, or equivalently is obtained by introducing an auxiliary exponent and differentiating with respect to it.

Second, before doing any large creative-telescoping search, one should run an exact and much cheaper test:

\[
L_{m CMF}igl(I_n^Jigr)\stackrel{?}=0,\qquad
I_n^J=\int_0^1rac{-\log t}{1+t^2}R_n^J(t^2)\,dt.
\]

Because every polynomial Catalan moment is uniquely of the form

\[
I_n^J=q_n^J\,G-p_n^J,\qquad q_n^J,p_n^J\in\mathbf Q,
\]

the test is exact: both the \(G\)-coefficient and rational coefficient of every recurrence residual must vanish. A single nonzero residual proves that **no certificate for the raw family \(R_n^J\) can exist**. One must then search for a short contiguous combination of Jacobi kernels.

The code below performs all of the following without using floating-point arithmetic:

1. reconstructs the normalized order-three scalar recurrence directly from the printed \(3\times3\) CMF;
2. constructs \(R_n^J(X)\) from its terminating Jacobi formula;
3. computes its Catalan moment as the exact pair \((q_n^J,p_n^J)\);
4. tests the CMF recurrence exactly;
5. constructs a finite discrete WZ certificate whenever the residual vanishes;
6. sets up the correct two-component continuous certificate;
7. searches a finite contiguous Jacobi/Christoffel–Darboux band if the raw candidate fails.

I do **not** assert an explicit nonzero certificate without running this exact residual calculation. The formulas below reduce the question to a finite rational linear-algebra computation and clearly distinguish a positive certificate from a no-go result.

def primitive_polynomial_vector(fracs):
    """Clear rational-function denominators and primitive-normalize."""
    den = lcm_polynomials([f.denominator() for f in fracs])
    ps = []
    for f in fracs:
        g = Kn(f * den)
        assert g.denominator() == 1
        ps.append(Rn(g.numerator()))

    common = ps[0]
    for p in ps[1:]:
        common = common.gcd(p)
    if common != 0 and common != 1:
        ps = [p.quo_rem(common)[0] for p in ps]

    coeff_den = ZZ(1)
    for p in ps:
        for a in p.list():
            coeff_den = lcm(coeff_den, a.denominator())
    ps = [Rn(coeff_den*p) for p in ps]

    ints = [abs(ZZ(a)) for p in ps for a in p.list() if a != 0]
    if ints:
        content = reduce(gcd, ints)
        if content > 1:
            ps = [Rn(p/content) for p in ps]

    # Fix one global sign.
    for p in reversed(ps):
        if p != 0:
            if p.leading_coefficient() < 0:
                ps = [-q for q in ps]
            break
    return ps

def J_and_dJ(N):
    """
    Return J_N(X)=P_N^(-1/2,0)(1-2X) and its alpha/epsilon derivative.
    Uses the terminating coefficient formulas, not symbolic special functions.
    """
    pref = rf(QQ(1,2), N) / factorial(N)
    hN = sum(QQ(1)/(r + QQ(1,2)) for r in range(N))

    J = RX.zero()
    dJ = RX.zero()
    for k in range(N+1):
        c = ((-1)**k * binomial(N,k) * pref
             * rf(N + QQ(1,2), k) / rf(QQ(1,2), k))
        d = hN
        d += sum(QQ(1)/(N+r+QQ(1,2)) for r in range(k))
        d -= sum(QQ(1)/(r+QQ(1,2)) for r in range(k))
        J += c*X**k
        dJ += c*d*X**k
    return J, dJ


def R_jacobi(N):
    J, dJ = J_and_dJ(N)
    B = J(-1)
    dB = dJ(-1)
    numerator = B*dJ - dB*J
    C, rem = numerator.quo_rem(X+1)
    assert rem == 0

    kappa = QQ(4*N+1, 2)
    return RX(kappa * (
        B*J - QQ(1,2)*(X+1)*(2*X*C.derivative() + C)
    ))


def K_cd_jacobi(N):
    """Jacobi Christoffel--Darboux discovery kernel."""
    ans = RX.zero()
    for k in range(N+1):
        J, _ = J_and_dJ(k)
        B = J(-1)
        ans += QQ(4*k+1, 2)*B*J
    return ans

# Exact Catalan moment pair q*G-p

def moment_pair(P):
    q = QQ(0)
    p = QQ(0)
    partial = QQ(0)
    for k in range(max(P.degree()+1, 0)):
        a = P[k]
        sign = (-1)**k
        q += a*sign
        p += a*sign*partial
        partial += QQ(sign, (2*k+1)**2)
    return q, p

def discrete_certificate(P):
    if P == 0:
        return [QQ(0)], [QQ(0)]

    K = P.degree()
    coeff = [P[k] for k in range(K+1)]

    A = [QQ(0)]*(K+2)
    A[K+1] = QQ(0)
    for k in range(K, -1, -1):
        A[k] = -coeff[k]-A[k+1]

    B = [QQ(0)]*(K+2)
    B[0] = QQ(0)
    for k in range(K+1):
        B[k+1] = B[k]-A[k+1]/QQ((2*k+1)**2)

    q, p = moment_pair(P)
    assert A[0] == -q
    assert B[K+1] == -p

    # Verify the local WZ equations.
    for k in range(K+1):
        assert coeff[k] == -A[k+1]-A[k]
        assert B[k+1]-B[k] == -A[k+1]/QQ((2*k+1)**2)
    return A, B

# Restricted two-component continuous certificate at one N
# ------------------------------------------------------------------
def solve_continuous_certificate(P, degA, degB, powA=2, powB=2):
    """
    Search
      U = X*a(X)/(1+X)^powA,
      V = X*(1-X)*b(X)/(1+X)^powB
    with deg a <= degA, deg b <= degB, satisfying (4.2).

    Failure only rules out this denominator/degree support.
    """
    x = FX(X)
    h = -FX(1)/(2*x) - FX(1)/(1+x)

    def D0(f):
        return f.derivative() + h*f

    columns = []
    U_basis = []
    V_basis = []

    for i in range(degA+1):
        U = x*x**i/(1+x)**powA
        U_basis.append(U)
        V_basis.append(FX(0))
        columns.append((D0(U), U/x))

    for i in range(degB+1):
        V = x*(1-x)*x**i/(1+x)**powB
        U_basis.append(FX(0))
        V_basis.append(V)
        columns.append((FX(0), D0(V)))

    target = (FX(P), FX(0))
    den = RX.one()
    for f in list(target) + [g for col in columns for g in col]:
        den = den.lcm(RX(f.denominator()))

    cleared_cols = []
    for col in columns:
        cleared_cols.append(tuple(RX(den*f) for f in col))
    cleared_target = tuple(RX(den*f) for f in target)

    rows = []
    rhs = []
    for comp in range(2):
        maxdeg = max(
            [cleared_target[comp].degree()] +
            [col[comp].degree() for col in cleared_cols]
        )
        for k in range(maxdeg+1):
            rows.append([col[comp][k] for col in cleared_cols])
            rhs.append(cleared_target[comp][k])

    A_mat = matrix(QQ, rows)
    b_vec = vector(QQ, rhs)
    aug = A_mat.augment(matrix(QQ, len(rhs), 1, list(b_vec)))
    if A_mat.rank() != aug.rank():
        return None

    sol = A_mat.solve_right(b_vec)
    U = sum(sol[i]*U_basis[i] for i in range(len(sol)))
    V = sum(sol[i]*V_basis[i] for i in range(len(sol)))

    assert D0(U) == FX(P)
    assert D0(V) + U/x == 0
    return U, V

    def pair_S(N):

        p = sum(sol[i]*feature(N, labels[i], 1) for i in range(len(labels)))
        return q, p

    # Held-out exact recurrence checks.
    for N in range(nfit, nfit+nhold):
        for which in [0, 1]:

                QQ(ell[j](N))*pair_S(N+j)[which]
                for j in range(4)
            )
            if residual != 0:
                print('held-out failure at', N, 'part', which, residual)
                return None

    print('all held-out exact checks passed')
    for coeff, label in zip(sol, labels):
        if coeff:
            print(label, '->', coeff)
    return sol, labels, ell

# 6. How to interpret the output

## Case A: the first exact residual is nonzero

Suppose the program prints

```text
first raw-Jacobi failure:
  N = N0
  G coefficient = nonzero rational
```

or a nonzero rational coefficient. Then the conclusion is definitive:

\[
oxed{
I_n^J	ext{ does not satisfy the CMF recurrence, so no CT certificate for the raw }R_n^J	ext{ exists.}
}
\]

Increasing the degree of \(C_n(t)\) cannot repair this. One must change the kernel, most naturally by a finite contiguous combination such as

\[
R_n(X)=\sum_{r=0}^w a_r(n)R_{n+r}^J(X)
+\sum_{r=0}^w b_r(n)\mathcal K_{n+r}^J(X).
	ag{6.1}
\]

The final routine searches precisely this family.

## Case B: all raw residuals vanish

Then compute

\[
P_n(X)=\sum_{j=0}^3c_j(n)R_{n+j}^J(X)
\]

and run `discrete_certificate(P_n)`. If its two endpoint entries vanish, the arrays returned by (3.1)–(3.2) are already a complete finite WZ certificate for that \(n\).

For a uniform proof in \(n\), retain the formulas

\[
A_{n,k}=-\sum_{r=k}^{K_n}(-1)^{r-k}p_{n,r},
	ag{6.2}
\]

\[
B_{n,k}=-\sum_{r=0}^{k-1}rac{A_{n,r+1}}{(2r+1)^2},
	ag{6.3}
\]

insert the terminating formula for the coefficients of \(R_n^J\), and simplify the two boundary values. The required symbolic identities are exactly

\[
A_{n,0}=0,\qquad B_{n,K_n+1}=0.
	ag{6.4}
\]

They can be proved by Gosper/Zeilberger reduction of the terminating hypergeometric-plus-harmonic sums. No integral evaluation remains.

## Case C: a contiguous-band solution is found

The returned coefficients define an explicit independent kernel (6.1). Three initial moment pairs are already imposed, so once a symbolic CT verification proves its recurrence, uniqueness of an order-three recurrence gives

\[
\int_0^1rac{-\log t}{1+t^2}R_n(t^2)\,dt
=G\widehat Q_n-\widehat P_n
\]

for every \(n\).

Held-out exact values are not the final proof. The final artifact must be one of:

1. the closed discrete certificate (3.1)–(3.4), with its two boundary identities simplified symbolically;
2. rational functions \(U_n,V_n\) satisfying (4.2) and the boundary conditions (4.4);
3. an Ore identity showing that the candidate moment pair is annihilated by the primitive CMF operator.

# 7. Operator-factor direction

If the Jacobi construction has a minimal annihilator \(L_J\), then an order-three CMF operator \(L_{m CMF}\) annihilates it precisely when \(L_{m CMF}\) is a **left multiple** of \(L_J\) in the Ore algebra. If both primitive operators have order three, they must agree up to multiplication by a nonzero rational function.

It is not enough that the Jacobi annihilator merely contains the CMF operator as an arbitrary left or right factor. The exact residual test in Section 5 avoids this ambiguity.

# 8. Minimal complete proof

For a selected CMF column, the following data close the proof:

1. **Kernel:** an explicit formula for \(R_n(X)\), independent of the CMF product.
2. **CT identity:** either (3.3) with zero endpoint values or (4.2)–(4.4).
3. **Initial values:** the three identities
   
   \[
   \mathcal C[R_j]=G\widehat Q_j-\widehat P_j,\qquad j=0,1,2.
   \]
4. **Growth:** an independent estimate \( \mathcal C[R_n]=O(n^{-3}) \), or any bound \(o((17+12\sqrt2)^n)\) that removes the dominant mode.

Items 2 and 3 imply equality for all \(n\). Item 4 implies convergence. The formal Birkhoff index \(-3\) alone does not remove the dominant solution; the integral or certificate must supply that global cancellation.

The main computational recommendation is therefore:

```text
run the exact raw-Jacobi residual test
        ↓
if nonzero: abandon the raw kernel and fit a width-1/2 contiguous band
        ↓
construct P_n(X)=Σ c_j(n)R_{n+j}(X)
        ↓
use the automatic discrete certificate A_{n,k},B_{n,k}
        ↓
prove only the two endpoint identities symbolically
```

This is substantially smaller than searching directly for a large rational \(C_n(t)\), and it gives a clean exact certificate or an equally clean no-go result.
