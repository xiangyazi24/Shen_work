# Problem 2.5: exact creative-telescoping test for the Jacobi kernel

## Executive conclusion

There are two corrections to make before searching for a certificate.

First, the logarithmic Catalan weight is not hyperexponential. A single rational function `C_n(t)` in

```text
sum(j=0..3) c_j(n) R^J_{n+j}(t^2)
  = d/dt [ t(1-t^2) C_n(t) (-log t)/(1+t^2) ]
```

is generally the wrong ansatz. After division by the weight, differentiating the right side produces a `1/log(t)` term. The correct continuous certificate has two rational components, or equivalently comes from differentiating an auxiliary exponent.

Second, before doing a large creative-telescoping search, run the cheaper exact test

```text
L_CMF(I^J)_n = 0 ?
I^J_n = integral_0^1 (-log t)/(1+t^2) R^J_n(t^2) dt.
```

Every polynomial Catalan moment has a unique formal decomposition

```text
I^J_n = q^J_n G - p^J_n,
q^J_n, p^J_n in Q.
```

Therefore both rational residuals must vanish. One nonzero exact residual proves that no certificate for the raw family `R^J_n` can exist; the kernel must be replaced by a contiguous combination.

The Sage program below does all of the following with exact rational arithmetic:

1. reconstructs the normalized order-three scalar recurrence directly from the printed 3×3 CMF;
2. constructs `R^J_n(X)` from a terminating Jacobi formula;
3. computes its Catalan moment as the exact pair `(q^J_n,p^J_n)`;
4. tests the CMF recurrence;
5. constructs a finite discrete WZ certificate whenever the residual vanishes;
6. sets up the correct two-component continuous certificate;
7. searches a finite Jacobi/Christoffel–Darboux contiguous band when the raw candidate fails.

No explicit nonzero certificate should be asserted until this residual test has been run. The calculation below gives either a checkable certificate or a checkable no-go result.

# 1. Correct Jacobi convention

For the measure

```text
X^(epsilon-1/2) dX,   0 < X < 1,
```

the shifted Jacobi family is

```text
J_n(X;epsilon) = P_n^(epsilon-1/2, 0)(1-2X).
```

Thus the first Jacobi parameter is `epsilon-1/2`. Using `P_n^(0,epsilon)` requires simultaneously interchanging the endpoints `X <-> 1-X` and swapping the Jacobi parameters.

The Rodrigues and terminating hypergeometric formulas are

```text
J_n(X;epsilon)
 = X^(1/2-epsilon)/n! * d^n/dX^n [ X^(n+epsilon-1/2) (1-X)^n ],
```

```text
J_n(X;epsilon)
 = (epsilon+1/2)_n/n!
   * 2F1(-n, n+epsilon+1/2; epsilon+1/2; X).
```

At `epsilon=0`, write

```text
J_n(X)    = J_n(X;0),
dotJ_n(X) = partial_epsilon J_n(X;epsilon) at epsilon=0,
B_n       = J_n(-1),
dotB_n    = dotJ_n(-1),
kappa_n   = (4n+1)/2.
```

Define

```text
C_n(X) = [B_n dotJ_n(X) - dotB_n J_n(X)]/(1+X).
```

The numerator vanishes at `X=-1`, so `C_n(X)` is a rational polynomial. The independent Jacobi kernel is

```text
R^J_n(X) = kappa_n * (
    B_n J_n(X)
    - (1+X)/2 * (2X d/dX + 1) C_n(X)
).
```

It satisfies

```text
R^J_n(-1) = Q^J_n = kappa_n B_n^2
```

and

```text
integral_0^1 (-log t)/(1+t^2) R^J_n(t^2) dt
  = G Q^J_n - P^J_n
```

for a rational number `P^J_n`. This definition is independent of the CMF.

# 2. Exact Catalan moment algebra

Define

```text
m_k = integral_0^1 t^(2k) (-log t)/(1+t^2) dt.
```

Then

```text
m_k = (-1)^k * ( G - sum(r=0..k-1) (-1)^r/(2r+1)^2 )
```

and the key first-order relation is

```text
m_{k+1} + m_k = 1/(2k+1)^2.
```

For a polynomial

```text
R(X) = sum(k=0..d) r_k X^k,
```

its Catalan moment is

```text
integral_0^1 (-log t)/(1+t^2) R(t^2) dt = q(R) G - p(R),
```

where

```text
q(R) = R(-1) = sum(k=0..d) (-1)^k r_k,
```

```text
p(R) = sum(k=0..d) (-1)^k r_k
       * sum(r=0..k-1) (-1)^r/(2r+1)^2.
```

Let

```text
P_n(X) = sum(j=0..3) c_j(n) R^J_{n+j}(X).
```

The desired CMF recurrence for the Jacobi moments is exactly equivalent to

```text
q(P_n) = 0,
p(P_n) = 0.
```

This is the fastest exact test of the raw candidate.

# 3. Explicit discrete WZ certificate

Write

```text
P_n(X) = sum(k=0..K_n) p_{n,k} X^k.
```

Define finite arrays `A_{n,k}` and `B_{n,k}` by

```text
A_{n,K_n+1} = 0,
A_{n,k} = -p_{n,k} - A_{n,k+1}     for k=K_n,...,0,
```

```text
B_{n,0} = 0,
B_{n,k+1} = B_{n,k} - A_{n,k+1}/(2k+1)^2.
```

Using `m_{k+1}=1/(2k+1)^2-m_k`, one obtains the exact local identity

```text
p_{n,k} m_k
 = Delta_k( A_{n,k} m_k + B_{n,k} ),
Delta_k F_k = F_{k+1}-F_k.
```

The two endpoint obstructions are

```text
A_{n,0}       = -q(P_n),
B_{n,K_n+1}   = -p(P_n).
```

Thus this is a zero-boundary WZ certificate if and only if the two exact residuals vanish. If the raw Jacobi family passes the CMF recurrence, the certificate is generated automatically; no separate guess for `C_n(t)` is required.

Closed forms for the certificate arrays are

```text
A_{n,k} = -sum(r=k..K_n) (-1)^(r-k) p_{n,r},
```

```text
B_{n,k} = -sum(r=0..k-1) A_{n,r+1}/(2r+1)^2.
```

A uniform symbolic proof therefore reduces to simplifying only the two endpoints.

# 4. Correct continuous certificate

After the change `X=t^2`, the Catalan moment is

```text
I_n = -1/4 * integral_0^1 w_0(X) log(X) R^J_n(X) dX,
w_0(X) = X^(-1/2)/(1+X).
```

Set

```text
h(X) = w_0'(X)/w_0(X) = -1/(2X) - 1/(1+X),
D_0  = d/dX + h(X).
```

For the recurrence polynomial `P_n(X)`, a continuous certificate consists of two rational functions `U_n(X),V_n(X)` satisfying

```text
P_n = D_0 U_n,
0   = D_0 V_n + U_n/X.
```

Then

```text
w_0(X) log(X) P_n(X)
 = d/dX [ w_0(X) ( U_n(X) log(X) + V_n(X) ) ].
```

Sufficient zero-boundary conditions are

```text
U_n(X) = O(X) and V_n(X) = O(X) as X -> 0,
V_n(1) = 0.
```

The equivalent auxiliary-exponent formulation is

```text
w_epsilon(X) = X^(epsilon-1/2)/(1+X),
D_epsilon = d/dX + (epsilon-1/2)/X - 1/(1+X),
```

```text
D_epsilon( U_n + epsilon V_n ) = P_n + O(epsilon^2).
```

This is the appropriate parameter-differentiated creative-telescoping problem.

def delta_of(x):
    # H_{n+1}/H_n for
    # H_n=(-16)^n (2)_n^2 (3)_n^2 (5/2)_n (7/2)_n^2.
    return -2*(x+2)**2*(x+3)**2*(2*x+5)*(2*x+7)**2


def MH_of(x):
    return M_of(x) / delta_of(x)


def lcm_polynomials(polys):
    ans = Rn.one()
    for p in polys:
        ans = ans.lcm(Rn(p))
    return ans


def primitive_polynomial_vector(fracs):
    den = lcm_polynomials([f.denominator() for f in fracs])
    ps = []
    for f in fracs:
        g = Kn(f*den)
        assert g.denominator() == 1
        ps.append(Rn(g.numerator()))

    common = ps[0]
    for p in ps[1:]:
        common = common.gcd(p)
    if common != 0 and common != 1:
        ps = [p.quo_rem(common)[0] for p in ps]

    coeff_den = ZZ(1)
    for p in ps:
        for a in p.list():
            coeff_den = lcm(coeff_den, a.denominator())
    ps = [Rn(coeff_den*p) for p in ps]

    ints = [abs(ZZ(a)) for p in ps for a in p.list() if a != 0]
    if ints:
        content = reduce(gcd, ints)
        if content > 1:
            ps = [Rn(p/content) for p in ps]

    for p in reversed(ps):
        if p != 0:
            if p.leading_coefficient() < 0:
                ps = [-q for q in ps]
            break
    return ps


def scalar_recurrence(column=0):
    # Return ell[0..3] with sum_j ell[j](N)y_{N+j}=0.
    e = vector(Kn, [1 if i == column else 0 for i in range(3)])
    M0 = MH_of(Kn(n))
    M1 = MH_of(Kn(n+1))
    M2 = MH_of(Kn(n+2))

    v0 = e
    v1 = M0*e
    v2 = M0*M1*e
    v3 = M0*M1*M2*e
    vs = [v0, v1, v2, v3]

    V = matrix(Kn, 3, 4, lambda i, j: vs[j][i])
    cof = []
    for j in range(4):
        cols = [k for k in range(4) if k != j]
        cof.append((-1)**j * V.matrix_from_columns(cols).det())

    ell = primitive_polynomial_vector(cof)
    zero = vector(Kn, [0,0,0])
    assert sum((Kn(ell[j])*vs[j] for j in range(4)), zero) == zero
    return ell

def J_and_dJ(N):
    pref = rf(QQ(1,2), N) / factorial(N)
    hN = sum(QQ(1)/(r+QQ(1,2)) for r in range(N))

    J = RX.zero()
    dJ = RX.zero()
    for k in range(N+1):
        c = ((-1)**k * binomial(N,k) * pref
             * rf(N+QQ(1,2), k) / rf(QQ(1,2), k))
        d = hN
        d += sum(QQ(1)/(N+r+QQ(1,2)) for r in range(k))
        d -= sum(QQ(1)/(r+QQ(1,2)) for r in range(k))
        J += c*X**k
        dJ += c*d*X**k
    return J, dJ


def R_jacobi(N):
    J, dJ = J_and_dJ(N)
    B = J(-1)
    dB = dJ(-1)
    numerator = B*dJ - dB*J
    C, rem = numerator.quo_rem(X+1)
    assert rem == 0

    kappa = QQ(4*N+1, 2)
    return RX(kappa * (
        B*J - QQ(1,2)*(X+1)*(2*X*C.derivative()+C)
    ))


def K_cd_jacobi(N):
    ans = RX.zero()
    for k in range(N+1):
        J, unused = J_and_dJ(k)
        B = J(-1)
        ans += QQ(4*k+1, 2)*B*J
    return ans

# Exact Catalan moment pair q*G-p

def moment_pair(P):
    if P == 0:
        return QQ(0), QQ(0)

    q = QQ(0)
    p = QQ(0)
    partial = QQ(0)
    for k in range(P.degree()+1):
        a = P[k]
        sign = (-1)**k
        q += a*sign
        p += a*sign*partial
        partial += QQ(sign, (2*k+1)**2)
    return q, p

def cmf_normalized_sequences(Nmax, column=0):

    ps, qs = [], []

        qs.append(qrow[column])
        if N < Nmax:
            step = MH_of(QQ(N))
            prow = prow*step
            qrow = qrow*step
    return qs, ps

def test_raw_jacobi(Ntest=12, column=0, verbose=True):
    ell = scalar_recurrence(column)

    failures = []

                print('rational coefficient =', rp)

        print('Retain a symbolic endpoint certificate for the proof.')
    return ell, failures

def discrete_certificate(P):
    if P == 0:
        return [QQ(0)], [QQ(0)]

    K = P.degree()
    coeff = [P[k] for k in range(K+1)]

    A = [QQ(0)]*(K+2)
    for k in range(K, -1, -1):
        A[k] = -coeff[k]-A[k+1]

    B = [QQ(0)]*(K+2)
    for k in range(K+1):
        B[k+1] = B[k]-A[k+1]/QQ((2*k+1)**2)

    q, p = moment_pair(P)
    assert A[0] == -q
    assert B[K+1] == -p

    for k in range(K+1):
        assert coeff[k] == -A[k+1]-A[k]
        assert B[k+1]-B[k] == -A[k+1]/QQ((2*k+1)**2)
    return A, B


def recurrence_polynomial(N, ell, family=R_jacobi):
    return RX(sum(QQ(ell[j](N))*family(N+j) for j in range(4)))


def print_discrete_certificate(N, ell, family=R_jacobi):
    P = recurrence_polynomial(N, ell, family)
    A, B = discrete_certificate(P)
    q, p = moment_pair(P)
    print('N =', N)
    print('P_N degree =', P.degree())
    print('q(P_N), p(P_N) =', q, p)
    print('left boundary A[0] =', A[0])
    print('right boundary B[-1] =', B[-1])
    if q == 0 and p == 0:
        print('zero-boundary discrete WZ certificate found')
    else:
        print('no zero-boundary certificate for this candidate at this N')
    return P, A, B

def solve_continuous_certificate(P, degA, degB, powA=2, powB=2):
    # Search U=X*a(X)/(1+X)^powA and
    # V=X*(1-X)*b(X)/(1+X)^powB.
    x = FX(X)
    h = -FX(1)/(2*x) - FX(1)/(1+x)

    def D0(f):
        return f.derivative() + h*f

    columns = []
    U_basis = []
    V_basis = []

    for i in range(degA+1):
        U = x*x**i/(1+x)**powA
        U_basis.append(U)
        V_basis.append(FX(0))
        columns.append((D0(U), U/x))

    for i in range(degB+1):
        V = x*(1-x)*x**i/(1+x)**powB
        U_basis.append(FX(0))
        V_basis.append(V)
        columns.append((FX(0), D0(V)))

    target = (FX(P), FX(0))
    den = RX.one()
    for f in list(target) + [g for col in columns for g in col]:
        den = den.lcm(RX(f.denominator()))

    cleared_cols = [tuple(RX(den*f) for f in col) for col in columns]
    cleared_target = tuple(RX(den*f) for f in target)

    rows = []
    rhs = []
    for comp in range(2):
        degrees = [cleared_target[comp].degree()]
        degrees += [col[comp].degree() for col in cleared_cols]
        maxdeg = max(degrees)
        for k in range(maxdeg+1):
            rows.append([col[comp][k] for col in cleared_cols])
            rhs.append(cleared_target[comp][k])

    A_mat = matrix(QQ, rows)
    b_vec = vector(QQ, rhs)
    aug = A_mat.augment(matrix(QQ, len(rhs), 1, list(b_vec)))
    if A_mat.rank() != aug.rank():
        return None

    sol = A_mat.solve_right(b_vec)
    U = sum((sol[i]*U_basis[i] for i in range(len(sol))), FX(0))
    V = sum((sol[i]*V_basis[i] for i in range(len(sol))), FX(0))

    assert D0(U) == FX(P)
    assert D0(V)+U/x == 0
    return U, V

        rhs.append(qs_cmf[N])
        rows.append([feature(N, lab, 1) for lab in labels])
        rhs.append(ps_cmf[N])

    # CMF recurrence for the G and rational parts.
    for N in range(nfit):

        print('No solution for width, degree =', width, degree)

    sol = A_mat.solve_right(b_vec)
    print('solution found; nullity =', len(labels)-A_mat.rank())

    def pair_S(N):

        p = sum(sol[i]*feature(N, labels[i], 1) for i in range(len(labels)))

                QQ(ell[j](N))*pair_S(N+j)[which]
                for j in range(4)

                return None

    print('all held-out exact checks passed')
    for coeff, label in zip(sol, labels):
        if coeff:
            print(label, '->', coeff)
    return sol, labels, ell

# 6. Interpreting the output

## Case A: an exact residual is nonzero

If the program prints a nonzero `G coefficient` or `rational coefficient`, then

```text
I^J_n does not satisfy the CMF recurrence.
```

This rules out every CT certificate for the raw `R^J_n`, regardless of the degree chosen for a proposed `C_n(t)`. The kernel must be changed, for example to

```text
R_n(X) = sum(r=0..w) a_r(n) R^J_{n+r}(X)
       + sum(r=0..w) b_r(n) K^J_{n+r}(X),
```

where `K^J_n` is the Jacobi Christoffel–Darboux kernel. The last routine searches this family.

## Case B: the raw residuals vanish

For each `n`, form `P_n(X)` and call `discrete_certificate(P_n)`. If its two boundary entries vanish, the returned arrays are the complete finite WZ certificate.

For a uniform proof in `n`, substitute the terminating coefficient formula into

```text
A_{n,0} = 0,
B_{n,K_n+1} = 0
```

and simplify the two terminating hypergeometric-plus-harmonic sums. No integral evaluation remains.

## Case C: a contiguous-band solution is found

The returned coefficients define an explicit independent kernel. The fitting equations already impose the three initial moment pairs. After a symbolic CT or Ore verification of the recurrence, uniqueness gives

```text
integral_0^1 (-log t)/(1+t^2) R_n(t^2) dt
  = G Qhat_n - Phat_n
```

for every `n`.

Held-out exact terms are discovery evidence, not the final proof. The final artifact must be one of:

1. the closed discrete certificate with both endpoint identities proved symbolically;
2. rational functions `U_n,V_n` satisfying the continuous equations and boundary conditions;
3. an Ore identity proving that the candidate moment pair is annihilated by the primitive CMF operator.

---

# 7. Ore factor direction

If the Jacobi moment has minimal annihilator `L_J`, then `L_CMF` annihilates it exactly when `L_CMF` is a left multiple of `L_J` in the Ore algebra. If both primitive operators have order three, they must agree up to multiplication by a nonzero rational function.

It is not enough that one operator merely appears as an arbitrary left or right factor of another. The direct residual calculation avoids this ambiguity.

---

# 8. Minimal complete proof

For one CMF column, the following data close the proof:

1. an explicit kernel `R_n(X)` independent of the CMF product;
2. a zero-boundary discrete or continuous CT identity;
3. equality of the three consecutive values at `n=0,1,2`;
4. an independent bound `I_n=O(n^-3)`, or any bound `o((17+12*sqrt(2))^n)` that excludes the dominant mode.

Items 2 and 3 prove equality for every `n`. Item 4 proves convergence. The formal Birkhoff index `-3` alone does not remove the dominant solution.

The recommended workflow is

```text
exact raw-Jacobi residual test
        -> if nonzero, fit a width-1 or width-2 contiguous band
        -> form P_n(X) = sum_j c_j(n) R_{n+j}(X)
        -> generate A_{n,k}, B_{n,k} automatically
        -> prove only the two endpoint identities symbolically.
```

This is substantially smaller than guessing one large rational `C_n(t)` and produces either a rigorous certificate or a rigorous no-go result.
