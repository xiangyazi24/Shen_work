# Q3945: eigenvalue-weighted summability after the `H²` pass

## Verdict

Yes: the three-step Sobolev/coefficient ladder is the right one.  The key correction to keep in mind is exactly the one you found: the previously obtained gradient bound, by itself, is not enough.  The decisive extra output of the `ℓ²`-source pass is not merely

```text
Σ |kπ| |c_k(t)| < ∞,
```

but also

```text
Σ λ_k² |c_k(t)|² < ∞,
```

for the Duhamel part, hence `u(t) ∈ H²_N` at positive time after adding the homogeneous heat part.  Once that `H²_N` information is available, the source coefficients become `ℓ¹`, and the final parabolic eigenvalue gain gives

```text
Σ λ_k |c_k(t)| < ∞.
```

Here `H²_N` means the Neumann Laplacian domain on `[0,1]`: in coefficient language, `Σ λ_k² |c_k|² < ∞` plus the harmless zero mode; in physical language, `H²` with the Neumann trace inherited from the cosine representation.

## The correct ladder

Let

```text
λ_k = (kπ)²,
D_k(t) = ∫₀ᵗ exp(-(t-s)λ_k) a_k(s) ds.
```

If `|a_k(s)| ≤ A_k` uniformly on the relevant time window, then the parabolic gain gives

```text
λ_k |D_k(t)| ≤ A_k.
```

For `k = 0` this is harmless because `λ_0 = 0`; all weighted estimates ignore the zero mode, and the zero coefficient is controlled separately by the ball bound / finite-time integral.

### Pass 1: ball bound gives `H¹` at positive time

A ball bound gives a crude source envelope `A_k = C`, so

```text
|D_k(t)| ≤ C / λ_k       (k ≥ 1).
```

Therefore

```text
Σ λ_k |D_k(t)|² ≤ C² Σ 1/λ_k < ∞,
```

because `λ_k ~ k²`.  The homogeneous heat part is smoother at every positive time, since powers of `λ_k` are beaten by `exp(-t λ_k)`.  Thus `u(t) ∈ H¹` for `t > 0`.

For a theorem uniform on a positive compact subwindow, use the usual split: the early head has an exponential gap `exp(-ε λ_k)`, and the tail is estimated using the positive-window source envelope.

### Pass 2: `H¹` gives an `ℓ²` source envelope, hence `H²`

From `u ∈ H¹` in one dimension, `u ∈ L∞`.  The elliptic resolver gives the corresponding regularity for `v`; in coefficient language the Neumann resolver multiplies by `(1+λ_k)^{-1}` and gains two derivatives.

For the logistic term, the Nemytskii map preserves `H¹` in 1D, so the logistic source coefficients are `ℓ²`.

For the chemotaxis divergence term, write the flux as `F = χ₀ u v_x` in the simplified model.  Then

```text
source_k^chem = kπ · sineCoeff(F,k) = cosCoeff(F',k),
```

using `F(0)=F(1)=0`, since `v_x` has Neumann-zero trace.  With `u ∈ H¹` and elliptic regularity for `v`, one has `F' ∈ L²`, so the cosine coefficients of `F'` are `ℓ²`.  Therefore the total source has an `ℓ²` envelope `A^{(2)}`.

Now the same eigenvalue gain gives

```text
λ_k |D_k(t)| ≤ A^{(2)}_k,
```

with `A^{(2)} ∈ ℓ²`.  Squaring gives

```text
Σ λ_k² |D_k(t)|² = Σ (λ_k |D_k(t)|)² ≤ Σ (A^{(2)}_k)² < ∞.
```

So the Duhamel part is in `H²_N`.  The homogeneous heat part is again analytic at positive time.  Thus the full solution satisfies

```text
u(t) ∈ H²_N
```

on positive windows.

This same estimate also implies the gradient `ℓ¹` bound by Cauchy-Schwarz:

```text
Σ |kπ| |D_k(t)|
  = Σ (λ_k |D_k(t)|) / |kπ|
  ≤ (Σ (A^{(2)}_k)²)^{1/2} (Σ 1/(kπ)²)^{1/2}.
```

So the gradient bound is a corollary of the `ℓ²`-source pass, but the hidden stronger conclusion is `H²_N`.

### Pass 3: `H²_N` gives an `ℓ¹` source envelope

This is the step needed for eigenvalue-weighted summability.

#### Logistic source

Let `g = f(u)` be the logistic source.  Since `u ∈ H²_N` and the logistic nonlinearity is smooth/polynomial on the bounded range of `u`,

```text
g ∈ H²,
g' = f'(u) u_x,
trace g'(0) = trace g'(1) = 0.
```

The zero trace of `g'` follows from the Neumann trace of `u_x`, not from an arbitrary `H²` statement.

For `k ≥ 1`, two integrations by parts give, up to the repository's harmless cosine-normalization constants,

```text
cosCoeff(g,k) = - cosCoeff(g'',k) / (kπ)².
```

Since `g'' ∈ L²`, its cosine coefficients are `ℓ²` and in particular bounded.  Hence

```text
|cosCoeff(g,k)| ≤ C / (kπ)²,
```

and this is summable over `k ≥ 1`.  More invariantly:

```text
Σ |cosCoeff(g,k)|
  ≤ (Σ |cosCoeff(g'',k)|²)^{1/2} (Σ 1/λ_k²)^{1/2} < ∞.
```

So the logistic coefficients are `ℓ¹`.

#### Chemotaxis divergence source

Let again `F = χ₀ u v_x`.  From `u ∈ H²_N` and `-v'' + v = u` with Neumann boundary conditions, elliptic regularity gives

```text
v ∈ H⁴_N.
```

Then

```text
F'' = χ₀ (u'' v_x + 2 u' v'' + u v''') ∈ L².
```

The products are legitimate in 1D because `H² ⊂ W^{1,∞}` and the resolver gives enough boundedness for `v_x`, `v''`, and `v'''`; in fact not all factors need to be bounded, only enough to put each product in `L²`.

The boundary condition needed for the first integration by parts is

```text
F(0) = F(1) = 0,
```

again because `v_x(0)=v_x(1)=0`.  For `k ≥ 1`,

```text
kπ · sineCoeff(F,k) = cosCoeff(F',k).
```

Now integrate by parts one more time in the weak sense:

```text
cosCoeff(F',k)
  = ∫₀¹ F'(x) cos(kπx) dx
  = - (1/(kπ)) ∫₀¹ F''(x) sin(kπx) dx.
```

There is no boundary contribution from `F'` here, because the primitive of `cos(kπx)` is `sin(kπx)/(kπ)`, and `sin(kπx)` vanishes at both endpoints.  Thus no condition `F'(0)=F'(1)=0` is required.

Let

```text
b_k = sineCoeff(F'',k).
```

Since `F'' ∈ L²`, we have `b ∈ ℓ²`.  Therefore

```text
Σ |kπ · sineCoeff(F,k)|
  = Σ |b_k| / |kπ|
  ≤ (Σ |b_k|²)^{1/2} (Σ 1/(kπ)²)^{1/2} < ∞.
```

So the chemotaxis divergence coefficients are `ℓ¹`.

This is the important distinction: the estimate is not merely pointwise `O(1/k)` from a bounded `F''` coefficient.  The summability comes from the stronger fact that the numerator sequence is `ℓ²`, paired with `1/k ∈ ℓ²`.

Combining the logistic and chemotaxis estimates gives a positive-window source envelope

```text
A^{(1)} ∈ ℓ¹,
|a_k(s)| ≤ A^{(1)}_k.
```

The final eigenvalue gain yields

```text
λ_k |D_k(t)| ≤ A^{(1)}_k,
Σ λ_k |D_k(t)| ≤ Σ A^{(1)}_k < ∞.
```

Again, add the homogeneous heat contribution separately; it is eigenvalue-weighted summable for every positive time by the exponential heat trace estimates.

## Answer to the specific checks

Your chain

```text
ball bound
  → source O(1)
  → Duhamel H¹
  → source ℓ²
  → Duhamel H²
  → source ℓ¹
  → eigenvalue-weighted summability
```

is correct, provided the source envelopes are interpreted on positive windows, or the Duhamel integral is split into a gapped head and a positive-window tail.

The `u ∈ H² → source ℓ¹` step is also correct:

* `logistic(u) ∈ H²_N`, hence its cosine coefficients are `O(1/k²)` / `ℓ¹` by two IBPs.
* For chemotaxis, `F = χ₀ u v_x ∈ H²`, `F(0)=F(1)=0`, and

  ```text
  kπ sineCoeff(F,k)
    = cosCoeff(F',k)
    = - sineCoeff(F'',k)/(kπ).
  ```

  Since `F'' ∈ L²`, the numerator is `ℓ²`, so division by `k` gives `ℓ¹` by Cauchy-Schwarz.

The IBP chain is sound.  The boundary term that looked dangerous, involving `F'(0)` and `F'(1)`, does not occur in the second IBP because one integrates `F'` against `cos`, whose antiderivative is `sin/(kπ)`.

## Bottom line

The theorem is not obtained from the gradient bound alone.  It is obtained because the same `ℓ²` source envelope that gave the gradient `ℓ¹` bound also gives the stronger hidden estimate

```text
Σ λ_k² |c_k(t)|² < ∞.
```

That `H²_N` regularity is exactly what makes the final source coefficients `ℓ¹`; then the already-existing eigenvalue gain gives the desired eigenvalue-weighted summability.
