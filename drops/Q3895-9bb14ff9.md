## Bottom theorem 1a: eigenvalue times Gaussian is summable

This is the true M-test atom.  Everything else should reduce to this.

```lean
/-- Bottom summability atom: `(kπ)^2 exp (-c (kπ)^2)` is summable for `c > 0`. -/

    Summable (fun k : ℕ =>
      λN k * Real.exp (-c * λN k)) := by

  -- use `Real.summable_pow_mul_exp_neg_of_pos` with power `2` and parameter `c*π^2 > 0`.
```

# 2. `hmassCoeff`

## Bottom theorem 2a: zeroth cosine coefficient is the interval integral

The repository normalization is unscaled at `n = 0`.

```lean
/-- Zeroth normalized Neumann coefficient is exactly the interval integral. -/
theorem cosineCoeffs_zero_eq_intervalIntegral (f : ℝ → ℝ) :
    cosineCoeffs f 0 = ∫ x in (0 : ℝ)..1, f x := by
  -- unfold `cosineCoeffs`, `unitIntervalNeumannCosineCoeff`, `unitIntervalCosineRawCoeff`.
  -- `cos(0*pi*x)=1`; rewrite the complex integral by `intervalIntegral.integral_ofReal`.
  -- take real part: `(∫ (f:ℂ)).re = ∫ f`.
  simp [ShenWork.IntervalNeumannFullKernel.cosineCoeffs,
    unitIntervalNeumannCosineCoeff, unitIntervalCosineRawCoeff]
```

If `simp` does not close the complex-real coercion, use this slightly expanded version:

```lean
/-- Same theorem with explicit real/complex integral conversion. -/
theorem cosineCoeffs_zero_eq_intervalIntegral_explicit (f : ℝ → ℝ) :
    cosineCoeffs f 0 = ∫ x in (0 : ℝ)..1, f x := by
  rw [ShenWork.IntervalNeumannFullKernel.cosineCoeffs,
    unitIntervalNeumannCosineCoeff, if_pos rfl, unitIntervalCosineRawCoeff]
  simp only [Nat.cast_zero, zero_mul, Real.cos_zero, Complex.ofReal_one, one_mul]
  rw [intervalIntegral.integral_ofReal]
  simp
```

## Bottom definition 3a: coefficient envelope on a time window

```lean
/-- Bottom-level window coefficient envelope for source/restart coefficient families. -/
structure WindowCoefficientEnvelope
    (a : ℝ → ℕ → ℝ) (lo hi : ℝ) : Prop where
  E : ℕ → ℝ
  E_nonneg : ∀ k, 0 ≤ E k
  E_summable : Summable E
  coeff_bound : ∀ s ∈ Set.Icc lo hi, ∀ k, |a s k| ≤ E k
  eigen_weighted_summable :
    Summable (fun k : ℕ => λN k * E k)
```

This is the minimum usable object for all later `Summable.of_norm_bounded` calls.
