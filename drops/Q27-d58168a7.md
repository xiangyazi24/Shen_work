## Why the static ceiling cannot close the gap

The existing static argument proves invariance of any admissible constant ceiling. In the critical positive branch, every `C ≥ MChi` satisfies the scalar first-contact margin, so it preserves the initial clamp. But invariance alone cannot lower the clamp: restarting with the same static theorem only reproduces the same bound.

Therefore the route must use a genuinely time-dependent barrier. This is already exactly how the χ≤0 branch reaches the asymptotic ceiling `1` in
`WholeLineCauchyLongTimeBound.lean`.

The positive critical case is not blocked by the nonlocal term. At an approximate spatial maximum, the repo already proves in
`WholeLineCauchyStableCeilingPDE.lean` that, for χ>0,

```text

```

because the elliptic resolver is nonnegative. The first-derivative drift is bounded by `K * |u_x|`, which vanishes in the approximate-maximum limit. Consequently the only missing issue is the scalar reaction comparison against the moving barrier.

A robust proof decomposition is:

1. Obtain the positive critical branch from
   `hstable.positive_branch_of_chi_nonneg hχ.le`.
2. Rewrite `p.m + p.γ = 1 + p.α` using the critical equality.
3. Rewrite the scalar term as
   `-(1-p.χ) * r * (r^α - MChi^α)` using
   `MChi_rpow_alpha_eq_one_div_one_sub_chi` or an equivalent multiplied identity.
4. Prove the elementary gap estimate

   ```lean
   r - MChi p ≤ r ^ p.α - (MChi p) ^ p.α
   ```

   from `1 ≤ MChi p`, `MChi p ≤ r`, and `1 ≤ p.α`.
5. Since `1 ≤ r`, strengthen it to

   ```lean
   r - MChi p ≤ r * (r ^ p.α - (MChi p) ^ p.α).
   ```

6. Multiply by the negative coefficient `-(1-p.χ)` with the inequality direction reversed.

The only potentially awkward Mathlib atom is step 4 for real powers. It is still elementary. A convenient standalone lemma is:

```lean

    {α x y : ℝ}
    (hα : 1 ≤ α) (hy : 1 ≤ y) (hyx : y ≤ x) :
    x - y ≤ x ^ α - y ^ α := by
  ...
```

Possible proof routes, in preferred order:

- apply the mean-value theorem to `x ↦ x^α`, using derivative
  `α*x^(α-1) ≥ 1` on `[y,x]`;
- normalize by `q=x/y≥1` and use `q≤q^α`, then multiply by `y^α`;
- use an existing convexity/secant-slope lemma for `Real.rpow` if the imports already expose it.

Do not hide this atom in a large PDE proof. It is the mathematical hinge and deserves a named, separately audited theorem.

## PDE comparison on one slab

The right template is
`wholeLineSlab_le_expCeiling_of_nonpositive_resolver_pde` from
`WholeLineCauchyLongTimeBound.lean`, not a fresh compactness argument.

Set

```lean
B t := wholeLineCauchyCriticalExpCeiling p C t
w t x := u t x - B t
```

At any point with `0 < w t x`, one has

```text
u(t,x) = B(t) + w(t,x) ≥ B(t) ≥ MChi.
```

The scalar lemma gives

```text
χ u^(m+γ) + u(1-u^α)
  ≤ -(1-χ)(u-MChi)
  = B'(t) -(1-χ)w.
```

After subtracting `B'(t)`, the PDE therefore has the approximate-maximum form

```text
w_t ≤ w_xx + K |w_x| - (1-χ) w
```

whenever `w>0`. At a positive slab supremum, the existing penalized approximate-max theorem makes `w_x` and the positive part of `w_xx` arbitrarily small, while `1-χ>0`; contradiction.

There are two reasonable Lean implementations.

### Preferred: adapt the existing all-points scalar envelope

Follow `wholeLineSlab_le_expCeiling_of_nonpositive_resolver_pde` and define a continuous `G` valid for all signs of `w`, for example

```lean

```

where `Kneg` is a bounded-slab Lipschitz constant for the scalar terms on the region `w≤0`. Then

```lean

```

is immediate, and the existing
`wholeLineSlabSup_le_of_scalar_pde` can be reused unchanged.

This approach is longer but mechanically closest to the proven χ≤0 file and minimizes foundational edits.

## Risks and audit checks

1. **Do not use repeated static ceilings.** They preserve a chosen clamp but never lower it.
2. **Do not solve the exact scalar ODE unless forced.** The exponential supersolution above is simpler and needs only real-power monotonicity.
3. **Keep the critical equality explicit.** The `StableWaveParameterRegime` positive branch is critical; do not accidentally claim this `MChi` barrier for the supercritical Cauchy regime, whose parameter ceiling is different.
4. **Preserve the cocycle identity.** Without it the segment induction will stall.
5. **Prove an all-sign PDE envelope or localize the maximum theorem honestly.** The coercive scalar estimate is only for `u≥MChi`; the proof must not silently apply it where `w≤0`.
6. **Use the paper-needed statement `MChi < M → eventually u≤M`.** Exact finite-time `u≤MChi` is false for a decaying barrier started above `MChi` and is unnecessary.
7. **Axiom audit each new capstone.** Expected output remains only `[propext, Classical.choice, Quot.sound]`.
8. **Cold-build after wiring.** The likely implementation is a new file importing `WholeLineCauchyLongTimeBound`, followed by one positive-χ Section 5 assembly file; avoid editing the already-stable χ≤0 chain unless a shared helper is genuinely beneficial.
