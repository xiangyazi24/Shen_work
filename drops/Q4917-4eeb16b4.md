# Problem 2.5: the exact Delannoy-basis transform, the coefficient-ratio theorem, and what it does—and does not—prove

## Executive answer

The decomposition

\[
\widehat Q_N=\sum_{k=0}^N f_k B(N,k),\qquad
\widehat P_N=\sum_{k=0}^N g_k B(N,k),
\]

with

\[
B(N,k)=2^k\binom{2k}{k}\binom Nk\binom{N+k}{k}
\]

is much more structured than an arbitrary triangular inversion. It is a **weighted Riordan transform**, and at the level of ordinary generating functions it is the algebraic pullback

\[
\boxed{
A(z)=\frac1{1-z}\,H\!\left(\frac{2z}{(1-z)^2}\right).
}
\tag{E1}
\]

Here

\[
A(z)=\sum_{N\ge0}a_Nz^N,\qquad
H(u)=\sum_{k\ge0}\binom{2k}{k}^2 f_k u^k,
\]

and \(a_N=\sum_kB(N,k)f_k\). The inverse is

\[
\boxed{
H(u)=\frac{2}{1+\sqrt{1+2u}}
A\!\left(
\frac{\sqrt{1+2u}-1}{\sqrt{1+2u}+1}
\right).
}
\tag{E2}
\]

This gives the following precise answers.

1. **Yes, the transform preserves holonomicity.** Since \(\widehat P_N\) and \(\widehat Q_N\) satisfy the same polynomial-coefficient recurrence, their generating functions are annihilated by a common differential operator. Formula (E2), followed by the hypergeometric diagonal gauge \(\binom{2k}{k}^{-2}\), proves that \(f_k\) and \(g_k\) are P-recursive and have a common polynomial-coefficient annihilator.

2. **The transform carries the dominant connection coefficient unchanged.** Put

\[
\lambda=17+12\sqrt2,\qquad
\rho=\lambda^{-1}=17-12\sqrt2.
\]

If

\[
\widehat Q_N\sim A_Q\lambda^N,\qquad
\widehat P_N\sim A_P\lambda^N,
\]

then

\[
\boxed{
 f_k\sim \pi(1+\rho)A_Q\,k,\qquad
 g_k\sim \pi(1+\rho)A_P\,k,\qquad
 \frac{g_k}{f_k}\longrightarrow\frac{A_P}{A_Q}.
}
\tag{E3}
\]

But \(A_P/A_Q\) is exactly the original CMF limit

\[
L=\lim_{N\to\infty}\frac{\widehat P_N}{\widehat Q_N}.
\]

Therefore the basis change proves

\[
\boxed{\displaystyle \lim_{k\to\infty}\frac{g_k}{f_k}=L,}
\tag{E4}
\]

not, by module structure alone, that \(L=G\). If an independent integrated-\(K\), regulator, Markov, or normal-function calculation proves \(A_P=GA_Q\), then (E4) immediately becomes \(g_k/f_k\to G\). Without that global connection calculation, the coefficient limit is the same unknown connection constant in a new coordinate system.

3. **The observed geometric factor is explained.** After cancellation of the pole at \(u=1/16\), the next singularity of the inverse transform is the square-root branch point \(u=-1/2\). Generically,

\[
\boxed{
\frac{g_k}{f_k}-L
=C\left(-\frac18\right)^k k^{-3/2}
\left(1+O(k^{-1})\right).
}
\tag{E5}
\]

Thus the true exponential factor is \(8\). The finite-index inverse factor is

\[
8\left(1+\frac{3}{2k}+O(k^{-2})\right),
\]

which naturally appears numerically as \(8.3\)–\(8.5\) around the range currently computed.

4. **Your weighted-average proof is rigorous once eventual positivity and (E4) with \(L=G\) are established.** In fact it is a direct application of the Silverman–Toeplitz regular-matrix theorem. Positivity for every \(k\) is stronger than necessary; eventual positivity suffices because every fixed finite prefix contributes only polynomial growth, while \(\widehat Q_N\) grows like \(\lambda^N\).

5. **The statement “the CMF is in the same Delannoy basis” is not an Ore equivalence.** Every scalar sequence has a unique expansion in this triangular basis. The Brafman case is the special vector \(f_k\equiv1\); replacing \(1\) by an arbitrary diagonal sequence \(f_k\) is not a rational gauge. A scalar rational/algebraic multiplicative gauge is in fact excluded by the different singularity types: the Brafman function has a logarithm at \(u=1/16\), whereas the CMF denominator transform has a simple pole there.

The rest of the answer proves these statements and gives exact code for constructing and certifying the common \(k\)-recurrence.

## 1. Exact factorization of the Delannoy kernel

Use

\[
\binom Nk\binom{N+k}{k}
=\binom{2k}{k}\binom{N+k}{2k}.
\]

Then

\[
\boxed{
B(N,k)=d_kR_{N,k},\qquad
 d_k=\binom{2k}{k}^2,\qquad
 R_{N,k}=2^k\binom{N+k}{2k}.
}
\tag{1.1}
\]

For fixed \(k\),

\[
\begin{aligned}
\sum_{N\ge k}R_{N,k}z^N
&=2^kz^k\sum_{m\ge0}\binom{m+2k}{2k}z^m\\
&=\frac{2^kz^k}{(1-z)^{2k+1}}.
\end{aligned}
\tag{1.2}
\]

Thus \(R=(R_{N,k})\) is the Riordan array

\[
\boxed{
R=\left(\frac1{1-z},\frac{2z}{(1-z)^2}\right),
}
\tag{1.3}
\]

followed by the column scaling \(d_k\).

Let

\[
a_N=\sum_{k=0}^NB(N,k)f_k,
\quad
A(z)=\sum_{N\ge0}a_Nz^N,
\quad
H(u)=\sum_{k\ge0}d_kf_ku^k.
\]

Using (1.2),

\[
\begin{aligned}
A(z)
&=\sum_{k\ge0}d_kf_k\frac{2^kz^k}{(1-z)^{2k+1}}\\
&=\frac1{1-z}
H\!\left(\frac{2z}{(1-z)^2}\right),
\end{aligned}
\]

which proves (E1).

### 1.1 Exact inverse

Put

\[
u=\frac{2z}{(1-z)^2},\qquad
s=\sqrt{1+2u}.
\]

The branch satisfying \(z(0)=0\) is

\[
\boxed{
z=\frac{s-1}{s+1},\qquad 1-z=\frac2{s+1}.}
\tag{1.4}
\]

Consequently

\[
H(u)=(1-z)A(z)
=\frac2{1+s}A\!\left(\frac{s-1}{s+1}\right),
\]

which proves (E2).

There is also a closed coefficient inverse. Write \(h_k=d_kf_k\). Then

\[
\boxed{
 h_k=rac1{2^k(2k+1)}
 \sum_{n=0}^k
 (-1)^{k-n}(2n+1)
 \binom{2k+1}{k-n}a_n,
}
\tag{1.5}
\]

and hence

\[
\boxed{
 f_k=rac1{2^k(2k+1)\binom{2k}{k}^2}
 \sum_{n=0}^k
 (-1)^{k-n}(2n+1)
 \binom{2k+1}{k-n}a_n.
}
\tag{1.6}
\]

For example,

\[
f_0=a_0,\qquad
f_1=\frac{a_1-a_0}{8},\qquad
f_2=\frac{a_2-3a_1+2a_0}{4\binom42^2}.
\]

Formula (1.6) also shows why positivity is not formal: the inverse has alternating signs.

## 2. What is preserved at the module level?

### 2.1 D-finiteness and a common annihilator

Let

\[
\mathcal A_Q(z)=\sum_{N\ge0}\widehat Q_Nz^N,\qquad
\mathcal A_P(z)=\sum_{N\ge0}\widehat P_Nz^N.
\]

The two sequences are solutions of the same order-three polynomial-coefficient recurrence. Therefore their generating functions are annihilated by a common operator in \(\mathbf Q(z)[D_z]\), after the harmless finite startup polynomial is homogenized.

Define

\[
\mathcal H_Q(u)=\sum_{k\ge0}d_kf_ku^k,\qquad
\mathcal H_P(u)=\sum_{k\ge0}d_kg_ku^k.
\]

Formula (E2) gives

\[
\boxed{
\begin{aligned}
\mathcal H_Q(u)
&=\frac2{1+s}\,
 \mathcal A_Q\!\left(\frac{s-1}{s+1}\right),\\
\mathcal H_P(u)
&=\frac2{1+s}\,
 \mathcal A_P\!\left(\frac{s-1}{s+1}\right),
\qquad s^2=1+2u.
\end{aligned}}
\tag{2.1}
\]

D-finite functions are closed under algebraic substitution and multiplication by algebraic functions. Hence \(\mathcal H_Q\) and \(\mathcal H_P\) have a common differential annihilator over \(\mathbf Q(u)\). Their coefficient sequences \(h_k=d_kf_k\) and \(j_k=d_kg_k\) therefore have a common polynomial-coefficient recurrence. Since \(d_k\) is hypergeometric, so do \(f_k\) and \(g_k\).

This is an exact theorem; no numerical guessing is needed for existence.

### 2.2 Exact construction of the \(k\)-operator

Suppose the recurrence-derived Euler operator is written

\[
\mathscr D_z=\sum_i a_i(z)\theta_z^i,
\qquad \theta_z=zD_z.
\]

Under (1.4),

\[
\boxed{\theta_z=s\theta_u.}
\tag{2.2}
\]

This is a noncommutative substitution: powers of \(s\theta_u\) are Ore products, not ordinary commutative powers.

Since

\[
\mathcal A(z)=\frac{s+1}{2}\mathcal H(u),
\]

the operator over \(\mathbf Q(u,s)\) annihilating \(\mathcal H\) is

\[
\boxed{
\mathscr D_H
=M_{2/(s+1)}
\left.\mathscr D_z\right|_{z=(s-1)/(s+1),\,\theta_z=s\theta_u}
M_{(s+1)/2},
}
\tag{2.3}
\]

where \(M_r\) denotes multiplication by \(r\). Taking the least common left multiple of (2.3) and its conjugate under \(s\mapsto-s\) produces an operator in \(\mathbf Q(u)[D_u]\).

If its Euler form is

\[
\mathscr D_H=\sum_{r=0}^R u^rQ_r(\theta_u),
\]

then the exact recurrence for \(h_k\) is

\[
\boxed{
\sum_{r=0}^R Q_r(k-r)h_{k-r}=0.
}
\tag{2.4}
\]

Finally substitute \(h_k=d_kf_k\). The needed gauge ratios are explicit:

\[
\boxed{
\frac{d_{k+j}}{d_k}
=4^j\prod_{\ell=0}^{j-1}
\frac{(2k+2\ell+1)^2}{(k+\ell+1)^2}.
}
\tag{2.5}
\]

Equations (2.3)–(2.5) are the exact \(k\)-recurrence construction. They are preferable to reporting an unverified recurrence guessed from 40 values. The expanded primitive operator can be large because the scalar OGF operator has higher differential order than the three-dimensional first-order matrix system.

### 2.3 What “preserving the module” means here

The transformation is invertible as a formal lower-triangular sequence transform and as an algebraic generating-function pullback. It is **not** a finite same-index Ore gauge in \(N\). In particular:

- it preserves D-finiteness and carries analytic connection data through the quadratic cover;
- over \(\mathbf Q(u,s)\) it is an algebraic pullback and rank-one gauge;
- descent from \(\mathbf Q(u,s)\) to \(\mathbf Q(u)\) can increase the order of a scalar operator;
- it does not contradict the previously computed absence of a rational same-index Ore intertwiner between the CMF recurrence and the Delannoy-square recurrence.

Thus the answer to part (a) is “yes” in the D-module/generating-function sense, but “not automatically” in the finite rational difference-module sense.

## 3. The coefficient ratio carries the same connection constant

Let

\[
\lambda=17+12\sqrt2,\qquad \rho=17-12\sqrt2=\lambda^{-1}.
\]

The normalized Birkhoff expansions have the form

\[
\widehat Q_N=A_Q\lambda^N(1+o(1))+	ext{smaller modes},
\]

\[
\widehat P_N=A_P\lambda^N(1+o(1))+	ext{smaller modes},
\]

with \(A_Q\ne0\). Therefore

\[
L=\lim_N\frac{\widehat P_N}{\widehat Q_N}
=\frac{A_P}{A_Q}.
\tag{3.1}
\]

Near \(z=\rho\),

\[
\mathcal A_Q(z)
=\frac{A_Q}{1-z/\rho}+O(1),
\qquad
\mathcal A_P(z)
=\frac{A_P}{1-z/\rho}+O(1).
\tag{3.2}
\]

The elementary identity

\[
\boxed{
1-16u
=\frac{z^2-34z+1}{(1-z)^2}
=\frac{(z-\rho)(z-\lambda)}{(1-z)^2}
}
\tag{3.3}
\]

shows that \(z=\rho\) maps to \(u=1/16\). Using (E1),

\[
\boxed{
\begin{aligned}
\mathcal H_Q(u)
&=\frac{(1+\rho)A_Q}{1-16u}+O(1),\\
\mathcal H_P(u)
&=\frac{(1+\rho)A_P}{1-16u}+O(1).
\end{aligned}}
\tag{3.4}
\]

Hence

\[
[u^k]\mathcal H_Q(u)
\sim(1+\rho)A_Q16^k,
\qquad
[u^k]\mathcal H_P(u)
\sim(1+\rho)A_P16^k.
\tag{3.5}
\]

Since

\[
\binom{2k}{k}^2
\sim\frac{16^k}{\pi k},
\tag{3.6}
\]

we obtain (E3), and in particular (E4).

### 3.1 The central logical point

This proves that the \(k\)-coefficient limit and the original CMF limit are the same number. It does not evaluate that number.

Equivalently, the dominant functional

\[
(\widehat P,\widehat Q)\longmapsto(A_P,A_Q)
\]

is transported to the residue functional at \(u=1/16\). The quotient \(A_P/A_Q\) remains a global connection coefficient. A local recurrence, its Poincaré polynomial, and the algebraic basis change do not force that quotient to equal Catalan's constant: replacing the numerator initial row by \(p+cq\) leaves all those structures unchanged and shifts the limit by \(c\).

Therefore:

> If the “integrated \(K\) module” identification includes an exact global normal-function or period calculation proving \(A_P=GA_Q\), then the Delannoy-basis route gives a clean proof that \(g_k/f_k\to G\). If only the spectrum and formal indices have been matched, the Catalan evaluation is still missing.

## 4. Why the observed convergence factor is approximately \(8.35\)

Set

\[
E_N=\widehat P_N-L\widehat Q_N,\qquad
\mathcal E(z)=\mathcal A_P(z)-L\mathcal A_Q(z).
\]

The pole at \(z=\rho\) cancels. Along the principal branch of (1.4), the next geometric obstruction is

\[
s=\sqrt{1+2u}=0,\qquad u=-\frac12,\qquad z=-1.
\]

Assume, as is verified by the desingularized recurrence-derived OGF operator, that \(\mathcal E\) has no singularity at \(z=-1\). Expanding (E2) at \(s=0\),

\[
\frac2{1+s}\mathcal E\!\left(\frac{s-1}{s+1}\right)
=2\mathcal E(-1)+\beta_Es+O(s^2),
\tag{4.1}
\]

where

\[
\boxed{\beta_E=4\mathcal E'(-1)-2\mathcal E(-1).}
\tag{4.2}
\]

If \(\beta_E\ne0\), singularity analysis gives

\[
[u^k]\mathcal H_E(u)
\sim-rac{\beta_E}{2\sqrt\pi}
(-2)^k k^{-3/2}.
\tag{4.3}
\]

Dividing by (3.5) yields

\[
\boxed{
\frac{g_k}{f_k}-L
\sim-rac{\beta_E}
{2\sqrt\pi(1+\rho)A_Q}
\left(-\frac18\right)^k k^{-3/2}.
}
\tag{4.4}
\]

If \(\beta_E=0\), the first nonzero odd power is normally \(s^3\), giving an additional factor \(k^{-1}\), but the exponential base remains \(1/8\).

Thus a measured inverse error ratio near \(8.35\) is not a new algebraic number. It is the finite-\(k\) value of

\[
8\left(\frac{k+1}{k}\right)^{3/2}
=8\left(1+\frac{3}{2k}+O(k^{-2})\right).
\tag{4.5}
\]

On the transformed three-mode subspace, the corresponding formal exponential roots are generically

\[
1,\qquad -\frac18,\qquad -\frac18,
\]

with powers

\[
1,\qquad -\frac12,\qquad -\frac32,
\]

so the formal characteristic polynomial is

\[
\boxed{
(r-1)\left(r+\frac18\right)^2,
\quad\text{or}\quad
64r^3-48r^2-15r-1.
}
\tag{4.6}
\]

This is the formal spectrum of the transformed CMF solution subspace. A nonminimal scalar recurrence obtained after rational descent may contain additional apparent factors.

## 5. The weighted-average proof

Suppose first that \(f_k>0\) for all \(k\), and define

\[
r_k=\frac{g_k}{f_k},\qquad
T_{N,k}=
\begin{cases}
\displaystyle\frac{f_kB(N,k)}{\widehat Q_N},&k\le N,\\
0,&k>N.
\end{cases}
\tag{5.1}
\]

Then

\[
T_{N,k}\ge0,\qquad
\sum_{k\ge0}T_{N,k}=1,
\tag{5.2}
\]

and

\[
\frac{\widehat P_N}{\widehat Q_N}
=\sum_{k\ge0}T_{N,k}r_k.
\tag{5.3}
\]

For fixed \(k\),

\[
B(N,k)
=\frac{2^k\binom{2k}{k}}{(k!)^2}
N^{2k}\left(1+O(N^{-1})\right),
\tag{5.4}
\]

whereas \(\widehat Q_N\sim A_Q\lambda^N\). Hence

\[
T_{N,k}\longrightarrow0
\qquad(N\to\infty)
\tag{5.5}
\]

for every fixed \(k\). The three Silverman–Toeplitz conditions are therefore

\[
\sum_kT_{N,k}=1,\qquad
T_{N,k}\to0\text{ for fixed }k,\qquad
\sup_N\sum_k|T_{N,k}|=1.
\]

Consequently, whenever \(r_k\to R\),

\[
\boxed{
\frac{\widehat P_N}{\widehat Q_N}\longrightarrow R.
}
\tag{5.6}
\]

This is exactly your split-at-\(K\) argument in the standard language of regular matrix summability.

### 5.1 Eventual positivity is enough

Assume only that \(f_k>0\) for \(k\ge K_0\). The finite prefix satisfies

\[
\sum_{k<K_0}|f_k|B(N,k)=O(N^{2K_0-2}),
\]

while \(\widehat Q_N\asymp\lambda^N\). Thus the prefix divided by \(\widehat Q_N\) tends to zero. The normalized tail is a positive probability distribution up to an error tending to zero, and the same proof applies.

Therefore proving positivity for every \(k\) is not necessary for the limit theorem.

## 6. Positivity of \(f_k\)

### 6.1 What the module structure gives

From (E3),

\[
f_k\sim\pi(1+\rho)A_Qk.
\]

Thus an exact sign certificate \(A_Q>0\) proves

\[
\boxed{f_k>0\quad\text{for all sufficiently large }k.}
\tag{6.1}
\]

The sign of \(A_Q\) can be expressed as the pairing of the initial denominator row with the normalized dominant adjoint solution at \(N=0\). It may be certified by an invariant cone or by interval/rational bounds in the backward adjoint iteration. This is much weaker than evaluating \(A_P/A_Q\) as \(G\).

### 6.2 A finite all-\(k\) certificate

Write

\[
\mathcal H_Q(u)
=\frac{C_Q}{1-16u}+R_Q(u),\qquad
C_Q=(1+\rho)A_Q>0.
\tag{6.2}
\]

Choose any \(R_0\) with

\[
\frac1{16}<R_0<\frac12.
\]

If analytic continuation of the pulled-back operator gives

\[
|R_Q(u)|\le M\qquad(|u|=R_0),
\]

then Cauchy's estimate yields

\[
\left|[u^k]R_Q(u)\right|\le MR_0^{-k}.
\]

Hence

\[
[u^k]\mathcal H_Q(u)
\ge C_Q16^k-MR_0^{-k}>0
\]

as soon as

\[
C_Q(16R_0)^k>M.
\tag{6.3}
\]

Since \(d_k>0\), this proves \(f_k>0\) beyond an explicit threshold. The finitely many remaining values are checked exactly using (1.6).

This “dominant positive pole + Cauchy bound + finite verification” is the most direct rigorous positivity strategy.

### 6.3 Recurrence induction is possible but not automatic

If the common \(k\)-recurrence can be rearranged into a forward form preserving a positive cone, then positivity follows by induction from finitely many initial values. But the repeated negative root \(-1/8\) means one should not expect every primitive scalar recurrence to have manifestly nonnegative coefficients. A matrix recurrence with a projectively invariant cone is more flexible.

The alternating inverse formula (1.6) shows that positivity does not follow simply from positivity of \(\widehat Q_N\) or from total positivity of the forward Riordan array.

## 7. Comparison with the Brafman–Delannoy model

For the central Delannoy square,

\[
D_N^2=\sum_{k=0}^NB(N,k),
\]

so \(f_k\equiv1\). Its transformed generating function is

\[
\boxed{
\mathcal H_D(u)
=\sum_{k\ge0}\binom{2k}{k}^2u^k
=\frac2\pi K(4\sqrt u).
}
\tag{7.1}
\]

Substituting \(u=2z/(1-z)^2\) recovers Brafman's identity

\[
\sum_{N\ge0}D_N^2z^N
=\frac2{\pi(1-z)}
K\!\left(\frac{4\sqrt{2z}}{1-z}\right).
\tag{7.2}
\]

### 7.1 Why \(f_k\ne1\) is not a scalar gauge

At \(u=1/16\),

\[
\mathcal H_D(u)
\sim \text{constant}\cdot
\log\frac1{1-16u},
\]

whereas (3.4) gives

\[
\mathcal H_Q(u)
\sim\frac{C_Q}{1-16u}.
\]

A rational or algebraic multiplicative factor cannot turn a logarithmic singularity into a pure simple pole. Therefore

\[
\boxed{
\mathcal H_Q(u)\ne R(u)\mathcal H_D(u)
\quad\text{for every nonzero }R(u)\in\overline{\mathbf Q(u)}.
}
\tag{7.3}
\]

A differential operator can differentiate a logarithm into a pole, so a finite differential/Ore intertwiner is a logically different possibility. It requires an explicit certificate

\[
L_QT=RL_D,
\]

and is not implied by the common basis or by the common silver-ratio singular locus. The prior low-degree rational and harmonic searches for \(f_k\), together with the different finite operators, rule out the naive scalar versions but not every possible higher-order transmutation.

The exact algebraic statement is simply

\[
\widehat Q=R_{\rm Riordan}\,\operatorname{diag}(d_k)f,
\qquad
D^2=R_{\rm Riordan}\,\operatorname{diag}(d_k)\mathbf1.
\]

An arbitrary diagonal vector \(f\) is not a gauge transformation in the Ore category.

## 8. Answer to the five questions

### (a) Can module structure prove \(g_k/f_k\to G\)?

It proves

\[
\frac{g_k}{f_k}\to L=rac{A_P}{A_Q},
\]

and it transports any independently established period identity \(A_P=GA_Q\). It does not derive that period identity from the spectrum or the basis transform alone. The Delannoy transform preserves the D-finite connection through an algebraic pullback, but the Catalan value remains a global connection coefficient.

### (b) Do \(f_k,g_k\) satisfy the same recurrence?

They have a common polynomial-coefficient annihilator. Equations (2.3)–(2.5) construct it exactly from the CMF scalar operator. The formal transformed roots are generically \(1,-1/8,-1/8\), and the coefficient ratio has the asymptotic (4.4). A minimal expanded recurrence should be obtained by exact Ore pullback/descent or by a modular guess followed by exact verification; reporting a guessed low-order recurrence without that verification would be unsafe.

### (c) Standard reference for the basis-change proof?

There are three standard ingredients:

1. weighted Riordan arrays for (E1)–(E2);
2. the Silverman–Toeplitz theorem for (5.6);
3. holonomic closure and singularity analysis for the common recurrence and the coefficient asymptotics.

References are listed below.

### (d) Is there a gauge from \(f_k\) to \(1\)?

Not a rational/algebraic scalar multiplicative gauge; (7.3) rules that out by singularity type. A higher-order differential or Ore transmutation is a separate finite problem. The common basis by itself is not such a certificate.

### (e) Can positivity be proved structurally?

The dominant positive residue gives eventual positivity once \(A_Q>0\) is certified. Complete positivity follows by an explicit analytic bound such as (6.3) plus a finite exact check, or by an invariant-cone certificate for the \(k\)-recurrence. Poincaré data alone do not imply positivity.

    m33 = (n+2)^2*(-16*n^5-408*n^4-2912*n^3-8884*n^2-12254*n-6240)
    return matrix(QQ, [[m11,m12,m13],
                       [m21,m22,m23],
                       [m31,m32,m33]])


def delta_H(n):
    n = QQ(n)
    return -2*(n+2)^2*(n+3)^2*(2*n+5)*(2*n+7)^2


def cmf_sequences(N):
    q = vector(QQ, [33750,-36000,9000])
    p = vector(QQ, [30921,-32972,8240])
    Q = []
    P = []
    for n in range(N):
        Q.append(q[0])
        P.append(p[0])
        MH = M(n)/delta_H(n)
        q = q*MH
        p = p*MH
    return Q,P


def B(N,k):
    return QQ(2)^k*binomial(2*k,k)*binomial(N,k)*binomial(N+k,k)


def delannoy_inverse(a):
    f = []
    for N in range(len(a)):
        value = a[N] - sum(B(N,k)*f[k] for k in range(N))
        f.append(value/B(N,N))
    return f


def closed_inverse(a,k):
    h = sum((-1)^(k-n)*(2*n+1)*binomial(2*k+1,k-n)*a[n]
            for n in range(k+1))
    h /= QQ(2)^k*(2*k+1)
    return h/binomial(2*k,k)^2

    assert g[k] == closed_inverse(Phat,k)

# Search for sum_{j=0}^r p_j(k) a_{k+j}=0,
# deg p_j <= d, simultaneously for f and g.
Rk.<k> = PolynomialRing(QQ)


def common_recurrence(seqs, r, d, train_end, verify_end):
    labels = [(j,m) for j in range(r+1) for m in range(d+1)]
    rows = []
    for a in seqs:
        for n in range(train_end):
            rows.append([QQ(n)^m*a[n+j] for j,m in labels])
    A = matrix(QQ, rows)
    ker = A.right_kernel()
    for v in ker.basis():
        pols = []
        pos = 0
        for j in range(r+1):
            pol = sum(v[pos+m]*k^m for m in range(d+1))
            pos += d+1
            pols.append(Rk(pol))
        # Exact independent hold-out verification.
        ok = True
        for a in seqs:
            for n in range(train_end, verify_end-r):
                if sum(pols[j](n)*a[n+j] for j in range(r+1)) != 0:
                    ok = False
                    break
            if not ok:
                break
        if ok:
            # Primitive integer normalization.
            den = lcm([c.denominator() for p in pols for c in p])
            ints = [Rk(den*p) for p in pols]
            content = gcd([ZZ(c) for p in ints for c in p if c])
            ints = [Rk(p/content) for p in ints]
            if ints[-1].leading_coefficient() < 0:
                ints = [-p for p in ints]
            return ints
    return None


FOUND = None

## Final assessment

The Delannoy-basis discovery is genuine and useful. It gives:

\[
\boxed{
\text{an exact algebraic pullback, a common }k\text{-recurrence,}
\text{ eventual positivity, and a regular weighted-average proof.}
}
\]

It also predicts the observed coefficient-ratio rate \(8^{-k}k^{-3/2}\).

The one point it does not supply by itself is the transcendental evaluation

\[
\frac{A_P}{A_Q}=G.
\]

That is exactly the original global connection-coefficient problem. The most promising way to close the proof using this basis is therefore:

1. construct the exact common \(k\)-operator via (2.3);
2. identify its dominant/recessive connection coefficient through a Markov, normal-function, or regulator representation in the \(u\)-plane;
3. certify \(A_P=GA_Q\);
4. use eventual positivity and Silverman–Toeplitz to return to the original CMF ratios.
