# Triple-collision determinants for Apéry values: exact formulas, the degree barrier, and the state-return equation

## Executive verdict

The determinant idea is mathematically real and can be made completely explicit, but in its naive form it does **not** yet prove

\[
E(p)=\sum_{a\in\mathbf F_p}N_p(a)^2\ll p.
\]

The exact outcome is:

1. The Wronskian is
   \[
   \boxed{W_n:=a_{n+1}b_n-a_nb_{n+1}=\frac{6}{(n+1)^3}.}
   \]
   Hence the fundamental matrix is invertible for every \(0\le n\le p-2\), and all transfer matrices can be written explicitly.

2. For a **general** solution \(x_r=\alpha a_r+\beta b_r\), the condition
   \[
   x_n=x_m=x_\ell
   \]
   is equivalent to one affine-collinearity determinant
   \[
   \det\!egin{pmatrix}
   a_n&b_n&-1\\
   a_m&b_m&-1\\
   a_\ell&b_\ell&-1
   \end{pmatrix}=0.
   \]
   For the actual Apéry solution \(b\), this determinant is only a **necessary** condition: it forgets which solution direction produced the equality.

3. With \(m=n+h\), \(\ell=n+k\), the determinant becomes an explicit continuant polynomial
   \[
   A_{h,k}(n)=N_h(n)L_k(n)-N_k(n)L_h(n),
   \]
   where \(N_h\) are the canonical gap continuants. Using the existing Bézout identity, one gets the nontrivial factorization
   \[
   \boxed{
   A_{h,k}(n)=Q_h(n)\,\widetilde A_{h,k}(n),
   }
   \]
   with
   \[
   \deg \widetilde A_{h,k}\le 3(k-1),
   \]
   independent of the smaller gap \(h\). This is a genuine simplification.

4. Nevertheless, the determinant has eliminated the solution direction. It is one equation in the three index variables \((n,h,k)\), so a generic count is of order \(p^2\), whereas the desired triple count for one fixed solution direction is of order \(p\). The missing factor of \(p\) is exactly the lost direction condition.

5. The state-return condition
   \[
   (b_n,b_{n+1})=(b_m,b_{m+1})
   \]
   implies that the transfer matrix has eigenvalue \(1\). This gives the explicit necessary equation
   \[
   \operatorname{tr}T_{n\to m}=1+\det T_{n\to m},
   \]
   and, for a fixed gap \(h=m-n\), an explicit polynomial \(\Pi_h(n)\) of degree at most \(3h\). But again this only says that **some** solution has a repeated state; it does not force the repeated state to belong to the particular solution \(b\).

The best way to use the determinant route is therefore not as a direct root bound, but as a **direction-labeled incidence problem**: every root of \(A_{h,k}\) determines one projective solution direction. Proving that these directions are equidistributed, or at least that no direction receives more than \(O(p)\) triples, would prove the desired third-moment bound.

# 1. Moment reduction

Write

\[
E_r(p):=\sum_{a\in\mathbf F_p}N_p(a)^r,
\qquad
\sum_a N_p(a)=p.
\]

Then Cauchy--Schwarz gives

\[
E_2(p)^2
=\left(\sum_a N_p(a)^2\right)^2
\le
\left(\sum_aN_p(a)\right)
\left(\sum_aN_p(a)^3\right)
=pE_3(p).
\]

Hence

\[
\boxed{E_3(p)\ll p\quad\Longrightarrow\quad E_2(p)\ll p.}
\]

It is often cleaner to use the factorial third moment

\[
T_3(p):=\sum_a (N_p(a))_{\underline3}.
\]

For every integer \(r\ge0\),

\[
(r)_{\underline2}\le (r)_{\underline3}+2r,
\]

so

\[
E_2(p)
=p+\sum_a(N_p(a))_{\underline2}
\le 3p+T_3(p).
\]

Thus

\[
\boxed{T_3(p)\ll p\quad\Longrightarrow\quad E_2(p)\ll p.}
\]

The numerical values \(E_3(p)/p\approx11\) are therefore exactly on the right scale.

# 2. Exact Wronskian

The two sequences satisfy, for \(n\ge1\),

\[
(n+1)^3u_{n+1}=P(n)u_n-n^3u_{n-1},
\]

with

\[
P(n)=(2n+1)(17n^2+17n+5).
\]

Define

\[
W_n:=a_{n+1}b_n-a_nb_{n+1}.
\]

Using the recurrence for both \(a\) and \(b\),

\[
\begin{aligned}
W_n
&=
\frac{P(n)a_n-n^3a_{n-1}}{(n+1)^3}b_n
-a_n\frac{P(n)b_n-n^3b_{n-1}}{(n+1)^3}\\
&=
\frac{n^3}{(n+1)^3}
\left(a_nb_{n-1}-a_{n-1}b_n\right)\\
&=
\frac{n^3}{(n+1)^3}W_{n-1}.
\end{aligned}
\]

Since

\[
W_0=a_1b_0-a_0b_1=6,
\]

telescoping gives

\[
\boxed{W_n=\frac{6}{(n+1)^3}.}
\tag{2.1}
\]

Equivalently,

\[
a_nb_{n-1}-a_{n-1}b_n=\frac6{n^3}.
\]

For \(0\le n\le p-2\), the denominator is invertible modulo \(p\), so \(W_n\ne0\). This gives:

- the two-dimensional solution space is genuinely nondegenerate;
- the fundamental matrix is invertible;
- no nonzero solution can have two consecutive zeros;
- the local initial state \((u_n,u_{n+1})\) determines the solution uniquely;
- every determinant elimination below is legitimate away from the singular boundary.

The Wronskian supplies the exact algebraic framework, but by itself it does not control how the distinguished solution \(b\) moves through that framework.

# 3. Fundamental matrix and transfer matrix

Use the state convention

\[
S_n(u):=\binom{u_{n+1}}{u_n}.
\]

Then

\[
S_n(u)=M_nS_{n-1}(u),
\qquad
M_n=
\begin{pmatrix}
\dfrac{P(n)}{(n+1)^3}&-\dfrac{n^3}{(n+1)^3}\\[2mm]
1&0
\end{pmatrix}
\quad(n\ge1).
\]

Define the fundamental matrix

\[
\Phi_n:=
\begin{pmatrix}
 a_{n+1}&b_{n+1}\\
 a_n&b_n
\end{pmatrix}.
\]

Its determinant is exactly

\[
\det\Phi_n=W_n=\frac6{(n+1)^3}.
\]

For a general solution

\[
u_r=\alpha a_r+\beta b_r,
\]

one has

\[
S_n(u)=\Phi_n\binom\alpha\beta.
\]

Therefore the transfer from index \(r\) to index \(s\) is

\[
\boxed{T_{r\to s}=\Phi_s\Phi_r^{-1}.}
\tag{3.1}
\]

Since

\[
\Phi_r^{-1}
=\frac1{W_r}
\begin{pmatrix}
 b_r&-b_{r+1}\\
 -a_r&a_{r+1}
\end{pmatrix},
\]

we obtain

\[
\boxed{
T_{r\to s}
=\frac1{W_r}
\begin{pmatrix}
 a_{s+1}b_r-b_{s+1}a_r&-a_{s+1}b_{r+1}+b_{s+1}a_{r+1}\\
 a_sb_r-b_sa_r&-a_sb_{r+1}+b_sa_{r+1}
\end{pmatrix}.
}
\tag{3.2}
\]

Its determinant is

\[
\boxed{
\det T_{r\to s}
=\frac{W_s}{W_r}
=\left(\frac{r+1}{s+1}\right)^3.
}
\tag{3.3}
\]

## 3.1 Transfer from the initial state

If one prefers the state

\[
\widehat S_n(u)=\binom{u_n}{u_{n+1}},
\]

then the transfer from \((u_0,u_1)\) is

\[
\boxed{
\widehat T_{0\to n}
=
\begin{pmatrix}
 b_n-\dfrac56a_n&\dfrac16a_n\\[2mm]
 b_{n+1}-\dfrac56a_{n+1}&\dfrac16a_{n+1}
\end{pmatrix}.
}
\tag{3.4}
\]

Indeed, any solution with initial data \((u_0,u_1)\) is

\[
u_n
=u_0b_n+\frac{u_1-5u_0}{6}a_n.
\]

Formula (3.4) is the requested explicit expression for \(T_{0\to n}\).

# 4. Triple equality for a general solution

Suppose

\[
x_r=\alpha a_r+\beta b_r.
\]

The condition that \(x_n,x_m,x_\ell\) all equal a common value \(c\) is

\[
\begin{pmatrix}
 a_n&b_n&-1\\
 a_m&b_m&-1\\
 a_\ell&b_\ell&-1
\end{pmatrix}
\begin{pmatrix}\alpha\\\beta\\c\end{pmatrix}=0.
\]

A nonzero solution \((\alpha,\beta,c)\) exists if and only if

\[
\boxed{
\Delta(n,m,\ell)
:=
\det\!\begin{pmatrix}
 a_n&b_n&-1\\
 a_m&b_m&-1\\
 a_\ell&b_\ell&-1
\end{pmatrix}
=0.
}
\tag{4.1}
\]

Geometrically, the three points

\[
Q_r=(a_r,b_r)\in\mathbf F_p^2
\]

are affinely collinear. The normal direction of the line is the solution direction \([\alpha:\beta]\).

For the actual Apéry sequence \(b\), a triple collision

\[
b_n=b_m=b_\ell
\]

corresponds to the special kernel vector

\[
(\alpha,\beta,c)=(0,1,c).
\]

Therefore

\[
b_n=b_m=b_\ell\quad\Longrightarrow\quad \Delta(n,m,\ell)=0,
\]

but the converse is false: \(\Delta=0\) only says that **some** solution direction has equal values at those three indices.

This is the first decisive obstruction. Eliminating the solution direction discards one condition.

# 5. Local continuants and the explicit triple determinant

Fix a base index \(x\). Define

\[
N_0(x)=0,
\qquad
N_1(x)=1,
\qquad
N_2(x)=P(x+1),
\]

and

\[
\boxed{
N_{h+1}(x)
=P(x+h)N_h(x)-(x+h)^6N_{h-1}(x).
}
\tag{5.1}
\]

Also define

\[
Q_h(x):=\prod_{r=2}^{h}(x+r)^3,
\qquad Q_0=Q_1=1.
\tag{5.2}
\]

Then every solution \(u\) satisfies

\[
\boxed{
 u_{x+h}
=
\frac{
N_h(x)u_{x+1}
-(x+1)^3N_{h-1}(x+1)u_x
}{Q_h(x)}.
}
\tag{5.3}
\]

This is proved by induction from the recurrence. It is the exact nonautonomous replacement for a constant-coefficient power formula.

Put

\[
L_h(x):=Q_h(x)+(x+1)^3N_{h-1}(x+1).
\tag{5.4}
\]

If

\[
u_x=u_{x+h}=c,
\qquad s:=u_{x+1},
\]

then (5.3) gives

\[
N_h(x)s-L_h(x)c=0.
\tag{5.5}
\]

Thus, for \(1\le h<k\), the existence of a solution satisfying

\[
u_x=u_{x+h}=u_{x+k}
\]

is equivalent to the existence of a nonzero pair \((s,c)\) in the kernel of

\[
\begin{pmatrix}
N_h(x)&-L_h(x)\\
N_k(x)&-L_k(x)
\end{pmatrix}.
\]

Hence the exact triple-continuant certificate is

\[
\boxed{
A_{h,k}(x)
:=N_h(x)L_k(x)-N_k(x)L_h(x)=0.
}
\tag{5.6}
\]

## 5.1 Exact relation to the affine determinant

A direct change of coordinates from \((\alpha,\beta,c)\) to

\[
(s,u_x,c)=(u_{x+1},u_x,c)
\]

has determinant \(W_x\). One obtains

\[
\boxed{
\det\!\begin{pmatrix}
 a_x&b_x&-1\\
 a_{x+h}&b_{x+h}&-1\\
 a_{x+k}&b_{x+k}&-1
\end{pmatrix}
=
\frac{W_x}{Q_h(x)Q_k(x)}A_{h,k}(x).
}
\tag{5.7}
\]

Thus the global affine-collinearity determinant and the local continuant determinant are exactly the same object.

# 6. Bézout reduction of the degree

Naively,

\[
\deg A_{h,k}\le 3(h+k-2).
\]

The existing Bézout identity gives a better factorization. Let

\[
S_h(x):=\prod_{r=2}^{h}(x+r),
\qquad Q_h(x)=S_h(x)^3.
\]

Then for \(k>h\),

\[
\boxed{
N_{k-1}(x+1)N_h(x)-N_{h-1}(x+1)N_k(x)
=Q_h(x)^2N_{k-h}(x+h).
}
\tag{6.1}
\]

Expanding (5.6),

\[
\begin{aligned}
A_{h,k}(x)
={}&N_h(x)Q_k(x)-N_k(x)Q_h(x)\\
&+(x+1)^3
\left[
N_h(x)N_{k-1}(x+1)
-N_k(x)N_{h-1}(x+1)
\right].
\end{aligned}
\]

Using (6.1),

\[
A_{h,k}(x)
=N_hQ_k-N_kQ_h+(x+1)^3Q_h^2N_{k-h}(x+h).
\]

Since \(Q_h\mid Q_k\), this becomes

\[
\boxed{
A_{h,k}(x)=Q_h(x)\,\widetilde A_{h,k}(x),
}
\tag{6.2}
\]

where

\[
\boxed{
\widetilde A_{h,k}(x)
=
N_h(x)\frac{Q_k(x)}{Q_h(x)}
-N_k(x)
+(x+1)^3Q_h(x)N_{k-h}(x+h).
}
\tag{6.3}
\]

In the interior range

\[
0\le x<x+k<p,
\]

the boundary factor \(Q_h(x)\) is nonzero modulo \(p\), so only \(\widetilde A_{h,k}\) matters.

Each term in (6.3) has degree at most \(3(k-1)\). Therefore

\[
\boxed{\deg\widetilde A_{h,k}\le 3(k-1).}
\tag{6.4}
\]

This is a real gain: the degree depends only on the larger gap, not on \(h+k\).

## 6.1 The first example

For \((h,k)=(1,2)\),

\[
N_1=1,
\qquad
L_1=1,
\]

and

\[
L_2=(x+2)^3+(x+1)^3.
\]

Hence

\[
\begin{aligned}
A_{1,2}(x)
&=(x+2)^3+(x+1)^3-P(x+1)\\
&=\boxed{-4(2x+3)^3}.
\end{aligned}
\tag{6.5}
\]

Thus, if **some** solution has three consecutive equal values at

\[
x,x+1,x+2,
\]

then necessarily

\[
2x+3\equiv0\pmod p.
\]

This is sharp as an existence statement for a solution direction. It need not hold for the actual \(b\)-direction. For example, at \(p=11\), the candidate is \(x=4\), but

\[
(b_4,b_5,b_6)\equiv(1,0,1)\pmod{11},
\]

so the determinant vanishes because another solution direction has the triple equality.

This example displays the lost-direction obstruction concretely.

# 7. Why the determinant alone cannot give \(E_3(p)=O(p)\)

There are two complementary ways to see the barrier.

## 7.1 Codimension count

The actual triple equality

\[
b_x=b_{x+h}=b_{x+k}
\]

imposes two scalar equations:

\[
b_{x+h}-b_x=0,
\qquad
b_{x+k}-b_x=0.
\]

In three index variables \((x,h,k)\), two independent equations should leave \(O(p)\) points.

After eliminating the solution direction, one retains only

\[
A_{h,k}(x)=0,
\]

which is one equation in three variables. A generic surface has \(O(p^2)\) points.

Thus the determinant elimination removes precisely the codimension needed to pass from \(p^2\) to \(p\).

## 7.2 Even ideal fixed-gap root bounds are insufficient

Suppose optimistically that for every fixed pair \((h,k)\), the polynomial \(\widetilde A_{h,k}\) had only \(O(1)\) roots in \(\mathbf F_p\). There are \(\asymp p^2\) gap pairs, so the total count of determinant roots would still be \(O(p^2)\), not \(O(p)\).

The determinant variety is expected to parameterize triples for **all** projective solution directions. Since there are \(p+1\) directions, a total of order \(p^2\) is exactly compatible with order \(p\) triples per direction.

Therefore the correct next theorem is not merely a root count for \(A_{h,k}\), but a **fiber bound for the associated solution-direction map**.

# 8. The direction map attached to a determinant root

Assume \(A_{h,k}(x)=0\) and that the coefficient pair is nondegenerate. The local initial-state direction is determined by

\[
N_h(x)s=L_h(x)c.
\]

Thus

\[
[s:c]=[L_h(x):N_h(x)]
=[L_k(x):N_k(x)].
\tag{8.1}
\]

Since \(\Phi_x\) is invertible, this determines a unique global solution direction

\[
[\alpha:\beta]
=
\Phi_x^{-1}[s:c].
\tag{8.2}
\]

Define

\[
\Theta(x,h,k)=[\alpha:\beta]\in\mathbf P^1(\mathbf F_p).
\]

Then the actual factorial third moment for the Apéry solution is, up to repeated-index bookkeeping,

\[
T_3(p)
=
\#\{(x,h,k):A_{h,k}(x)=0,\ \Theta(x,h,k)=[0:1]\}.
\tag{8.3}
\]

The determinant route would succeed if one proved a directional nonconcentration theorem such as

\[
\boxed{
\max_{\xi\in\mathbf P^1(\mathbf F_p)}
\#\Theta^{-1}(\xi)
\ll p.
}
\tag{8.4}
\]

A stronger random-scale prediction is

\[
\#\Theta^{-1}(\xi)
\sim \frac{\#\{A_{h,k}(x)=0\}}{p+1}.
\]

This is the precise place where an \(\mathrm{SL}_2\)-mixing or monodromy theorem could enter. The abstract statement “the differential Galois group is \(\mathrm{SL}_2\)” is not enough; one needs quantitative equidistribution of the **direction map** \(\Theta\) for the deterministic prefix-product cocycle.

# 9. Cross-Wronskians of shifted states

Define

\[
D(n,m):=b_nb_{m+1}-b_mb_{n+1}.
\tag{9.1}
\]

This is not the same as an ordinary value collision.

If \(b_n,b_m\ne0\), then

\[
D(n,m)=b_nb_m
\left(
\frac{b_{m+1}}{b_m}-\frac{b_{n+1}}{b_n}
\right).
\]

Thus

\[
D(n,m)=0
\]

means that the two states are **projectively proportional**, not equal. If \(b_n=b_m=c\), then

\[
D(n,m)=c(b_{m+1}-b_{n+1}).
\]

So a scalar collision does not imply \(D=0\), except when the next coordinates also agree or the common value is zero.

The matrix

\[
\begin{pmatrix}
 b_n&b_m&b_\ell\\
 b_{n+1}&b_{m+1}&b_{\ell+1}
\end{pmatrix}
\]

has rank one exactly when the three states are projectively equal. This is much stronger than

\[
b_n=b_m=b_\ell.
\]

## 9.1 Fundamental cross-kernel versus same-solution cross-Wronskian

There **is** a simple formula for the cross-kernel of the two independent fundamental solutions. Define

\[
\mathcal K(r,s):=a_sb_r-a_rb_s.
\]

Then

\[
\mathcal K(r,r+1)=W_r,
\]

and from (5.3),

\[
\boxed{
\mathcal K(x,x+h)
=W_x\frac{N_h(x)}{Q_h(x)}.
}
\tag{9.2}
\]

This is the genuine continuant product formula.

By contrast, \(D(n,m)\) uses one solution at two different indices. It is a quadratic form in the local state. If \(m=x+h\), put

\[
c=b_x,
\qquad s=b_{x+1}.
\]

Then

\[
\begin{aligned}
Q_{h+1}(x)D(x,x+h)
={}&-(x+h+1)^3N_h(x)s^2\\
&+\Bigl[
N_{h+1}(x)
+(x+1)^3(x+h+1)^3N_{h-1}(x+1)
\Bigr]cs\\
&-(x+1)^3N_h(x+1)c^2.
\end{aligned}
\tag{9.3}
\]

The continuant coefficients are explicit, but the actual state \((c,s)\) remains. There is no state-free product formula analogous to (9.2).

This is another manifestation of the same obstruction: eliminating the actual direction is easy; retaining it is the hard part.

# 10. Repeated two-dimensional states

Now consider the stronger condition

\[
(b_n,b_{n+1})=(b_m,b_{m+1}).
\tag{10.1}
\]

Let

\[
S_n=\binom{b_{n+1}}{b_n}.
\]

Then

\[
S_m=T_{n\to m}S_n.
\]

If \(S_m=S_n\), then \(S_n\) is a fixed vector of \(T_{n\to m}\). Therefore \(1\) is an eigenvalue, so

\[
\det(T_{n\to m}-I)=0.
\]

For a \(2\times2\) matrix,

\[
\det(T-I)=1-\operatorname{tr}T+\det T,
\]

hence

\[
\boxed{
\operatorname{tr}T_{n\to m}=1+\det T_{n\to m}.
}
\tag{10.2}
\]

Using (3.2),

\[
\boxed{
\operatorname{tr}T_{r\to s}
=
\frac{
 a_{s+1}b_r-b_{s+1}a_r
+a_{r+1}b_s-a_sb_{r+1}
}{W_r}.
}
\tag{10.3}
\]

Equivalently, in terms of \(\mathcal K\),

\[
\operatorname{tr}T_{r\to s}
=
\frac{\mathcal K(r,s+1)-\mathcal K(r+1,s)}{W_r}.
\tag{10.4}
\]

Since

\[
\det T_{r\to s}=\frac{W_s}{W_r},
\]

the fixed-vector condition becomes

\[
\boxed{
\mathcal K(r,s+1)-\mathcal K(r+1,s)
=W_r+W_s.
}
\tag{10.5}
\]

This is the clean cross-Wronskian trace equation requested in the question.

# 11. Fixed-gap trace polynomial

Set \(m=x+h\). The transfer from

\[
S_x=(u_{x+1},u_x)^T
\]

to

\[
S_{x+h}=(u_{x+h+1},u_{x+h})^T
\]

is, by (5.3),

\[
\boxed{
T_h(x)=
\begin{pmatrix}
\dfrac{N_{h+1}(x)}{Q_{h+1}(x)}
&
-\dfrac{(x+1)^3N_h(x+1)}{Q_{h+1}(x)}\\[3mm]
\dfrac{N_h(x)}{Q_h(x)}
&
-\dfrac{(x+1)^3N_{h-1}(x+1)}{Q_h(x)}
\end{pmatrix}.
}
\tag{11.1}
\]

Its determinant is

\[
\boxed{
\det T_h(x)
=\left(\frac{x+1}{x+h+1}\right)^3.
}
\tag{11.2}
\]

Its trace is

\[
\operatorname{tr}T_h(x)
=
\frac{N_{h+1}(x)}{Q_{h+1}(x)}
-
\frac{(x+1)^3N_{h-1}(x+1)}{Q_h(x)}.
\tag{11.3}
\]

Therefore the existence of some nonzero solution with repeated state across the gap \(h\) forces

\[
\boxed{
\Pi_h(x)=0,
}
\tag{11.4}
\]

where

\[
\boxed{
\begin{aligned}
\Pi_h(x)
={}&N_{h+1}(x)
-(x+1)^3(x+h+1)^3N_{h-1}(x+1)\\
&-Q_h(x)\Bigl((x+h+1)^3+(x+1)^3\Bigr).
\end{aligned}
}
\tag{11.5}
\]

One has

\[
\deg\Pi_h\le3h.
\]

For \(h=1\),

\[
\Pi_1(x)
=P(x+1)-(x+2)^3-(x+1)^3
=\boxed{4(2x+3)^3}.
\tag{11.6}
\]

So there is only one base index at which **some** solution can have a one-step repeated state.

## 11.1 Why this does not prove the Apéry state collision bound

The condition \(\Pi_h(x)=0\) is necessary and sufficient for the existence of some nonzero fixed vector of \(T_h(x)\). The actual Apéry state collision requires the stronger equation

\[
\boxed{
(T_h(x)-I)
\binom{b_{x+1}}{b_x}=0.
}
\tag{11.7}
\]

The trace equation has forgotten which eigenvector is fixed.

Equivalently,

\[
\det(\Phi_{x+h}-\Phi_x)=0
\]

means that some solution repeats its state, while the Apéry state repeats only when the **second column** of \(\Phi_{x+h}-\Phi_x\) is zero.

For each fixed \(h\), the degree bound gives at most \(3h\) candidates for some-solution state returns. Summing over \(h<p\) gives only

\[
O(p^2),
\]

not \(O(1)\) or \(O(\log p)\). To obtain the observed almost-injectivity of the particular state path, one needs a second condition selecting the \(b\)-eigenvector.

Thus the numerical state-injectivity phenomenon is a genuine new two-dimensional equidistribution statement; it does not follow from the trace condition alone.

---

# 12. Where the degree \(\sim p\) reappears

There are two presentations of the same obstruction.

## 12.1 Global interpolation

As functions of the index \(n\in\mathbf F_p\), the sequences \(a_n\) and \(b_n\) can be represented by interpolation polynomials of degree at most \(p-1\). Substituting those into (4.1) gives a determinant of degree \(\asymp p\) in each index variable.

Thus a direct Weil or Lang--Weil argument sees a high-complexity variety.

## 12.2 Local gap coordinates

For fixed gaps \(h,k\), the continuant presentation lowers the degree to \(O(k)\). But the gaps themselves range up to \(p\). The total family

\[
\{\widetilde A_{h,k}:1\le h<k<p\}
\]

still has growing complexity, and there are \(\asymp p^2\) members.

The extra equality conditions do not magically make the whole family bounded-degree. They only make each fixed-gap member explicit.

---

# 13. How the triple determinant can still become a proof

The route is not dead. It needs one additional ingredient.

## 13.1 Directional equidistribution

Prove that the solution-direction map \(\Theta\) from (8.2) has fibers of size \(O(p)\). Then the Apéry direction receives only \(O(p)\) triple collisions, giving

\[
T_3(p)\ll p
\]

and hence \(E(p)\ll p\).

A variance form would be enough. If

\[
R_\xi
:=\#\{(x,h,k):A_{h,k}(x)=0,\ \Theta(x,h,k)=\xi\},
\]

then a bound

\[
\sum_{\xi\in\mathbf P^1(\mathbf F_p)}
\left(R_\xi-\frac{\sum_\eta R_\eta}{p+1}\right)^2
\ll p^3
\]

would imply

\[
R_\xi=O(p^{3/2})
\]

for every direction; a sharper random-scale bound \(O(p^2)\) for the variance would give \(R_\xi=O(p)\).

## 13.2 Parallel-chord formulation

Let

\[
Q_n=(a_n,b_n)\in\mathbf F_p^2.
\]

For a projective functional \(\lambda\), the sequence

\[
\lambda(Q_n)
\]

is a solution. Triple equal values correspond to three points \(Q_i,Q_j,Q_k\) on one affine line with normal \(\lambda\).

The determinant \(\Delta(i,j,k)\) counts collinear triples over **all** directions. The desired theorem asks for the number in one fixed parallel class.

This suggests studying the direction distribution of secants or parallel chords, not only the total collinear-triple count.

## 13.3 Use of Dodgson and Plücker identities

The determinant identity

\[
\det(Q_i-Q_j,Q_k-Q_l)
\]

expands as an alternating sum of Casorati kernels. Those kernels are the continuants \(N_h/Q_h\). Thus the fourth-moment or direction-variance problem is exactly where the existing Dodgson, Bézout, and Plücker identities are structurally relevant.

The concrete computational program should be:

1. factor \(\widetilde A_{h,k}\) over \(\mathbf Q\) for small \((h,k)\);
2. record the associated direction map \(\Theta\);
3. count how roots distribute among directions modulo many primes;
4. test whether primitive factors for different \((h,k)\) share only the known boundary factors;
5. compute the parallel-chord fourth moment and compare with the random \(O(p^2)\) prediction.

This is more informative than merely counting roots of \(A_{h,k}\).

# 14. Direct answers to the six questions

## 1. Exact Wronskian

Yes:

\[
\boxed{a_{n+1}b_n-a_nb_{n+1}=\frac6{(n+1)^3}.}
\]

It gives invertibility and exact transfer determinants, but not by itself a collision bound.

## 2. Transfer matrix

With \(S_n=(u_{n+1},u_n)^T\),

\[
T_{r\to s}=\Phi_s\Phi_r^{-1}
\]

with \(\Phi_n\) as above, and formula (3.2) is the full explicit answer. In the state convention \((u_n,u_{n+1})\), formula (3.4) gives \(T_{0\to n}\).

## 3. Triple elimination

For a general solution, triple equality is exactly the affine determinant (4.1). In fixed-gap coordinates it is exactly the continuant determinant (5.6). The resulting object is low-degree in the **sequence coordinates**, but not fixed-degree in the index variables.

## 4. Degree

The naive degree is \(3(h+k-2)\). The Bézout identity improves the primitive degree to

\[
\boxed{\deg\widetilde A_{h,k}\le3(k-1).}
\]

This is useful, but still grows with the larger gap. More importantly, one determinant equation naturally has \(O(p^2)\) solutions across all triples.

## 5. Cross-Wronskian

The fundamental cross-kernel \(a_sb_r-a_rb_s\) has the clean continuant formula (9.2). The same-solution quantity

\[
D(n,m)=b_nb_{m+1}-b_mb_{n+1}
\]

has the explicit quadratic-state formula (9.3), but no state-free product formula. Its vanishing means projective state equality, not ordinary value equality.

## 6. Repeated states

A repeated state forces

\[
\operatorname{tr}T=1+\det T.
\]

The trace is given explicitly by (10.3), and for a fixed gap one obtains \(\Pi_h(x)=0\) with \(\deg\Pi_h\le3h\). This is only a necessary condition for the particular Apéry state, because it detects a fixed vector for **some** solution direction. No current argument from these formulas alone proves \(O(1)\) or \(O(\log p)\) actual state collisions.

---

# Final assessment

The triple-collision route has produced a meaningful new algebraic object:

\[
\widetilde A_{h,k}(x)
=
N_h(x)\frac{Q_k(x)}{Q_h(x)}
-N_k(x)
+(x+1)^3Q_h(x)N_{k-h}(x+h).
\]

It is the exact state-free certificate for the existence of a solution with three equal values, and its degree is only \(3(k-1)\). This is worth developing.

But the determinant alone cannot prove \(E_3(p)=O(p)\), because it counts triple equalities for all projective solution directions. The missing theorem is:

\[
\boxed{
\text{roots of the triple-continuant certificate are not concentrated in one solution direction.}
}
\]

That directional nonconcentration—or the equivalent parallel-chord variance bound—is the precise rank-two statement that would turn this program into a proof.
