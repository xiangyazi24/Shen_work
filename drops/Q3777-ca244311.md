```lean
/-- A classical slice supplies the endpoint-insensitive closed representative for
`coupledChemDivSourceLift`.  The representative is the physical product-rule
expression; only equality on `(0,1)` is claimed, so no endpoint continuity of the
zero extension is used. -/
theorem coupledChemDivSourceLift_continuousRepresentative_of_classical
    {p : CM2Params} {T : ℝ} {u : ℝ → intervalDomainPoint → ℝ}
    {s : ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u
      (coupledChemicalConcentration p u))
    (hs : s ∈ Set.Ioo (0 : ℝ) T) :
    ∃ Gdiv : ℝ → ℝ,
      ContinuousOn Gdiv (Set.Icc (0 : ℝ) 1) ∧
      Set.EqOn (coupledChemDivSourceLift p u s) Gdiv (Set.Ioo (0 : ℝ) 1) := by
  let Gdiv : ℝ → ℝ :=
    fun x =>
      liftChemotaxisDivPhysicalRep p u (coupledChemicalConcentration p u) s x
  refine ⟨Gdiv, ?_, ?_⟩
  · have hslab :
      ContinuousOn
        (Function.uncurry
          (liftChemotaxisDivPhysicalRep p u (coupledChemicalConcentration p u)))
        (Set.Icc s s ×ˢ Set.Icc (0 : ℝ) 1) :=
      liftChemotaxisDivPhysicalRep_continuousOn_strictSlab_of_classicalSolution
        (p := p) (u := u) (v := coupledChemicalConcentration p u)
        (T := T) hsol hs.1 le_rfl hs.2
    have hmap : ContinuousOn (fun x : ℝ => (s, x)) (Set.Icc (0 : ℝ) 1) :=
      (continuous_const.prodMk continuous_id).continuousOn
    have hslice := hslab.comp hmap (fun x hx => by
      exact ⟨Set.mem_Icc.mpr ⟨le_rfl, le_rfl⟩, hx⟩)
    simpa [Gdiv, Function.uncurry] using hslice
  · intro x hx
    have heq :=
      lift_chemotaxisDiv_eq_liftChemotaxisDivPhysicalRep_interior
        (p := p) (u := u) (v := coupledChemicalConcentration p u)
        (T := T) hsol hs hx
    simpa [Gdiv, coupledChemDivSourceLift, intervalDomain] using heq

end ShenWork.Paper2.IntervalGradientSourceBridgeOpen
```

The proof route is short because the hard work is already in `IntervalChiNegH1ChemDivRepresentative.lean`. The only nontrivial move is choosing the degenerate strict time slab `[s,s]`; this is valid because `hs : s ∈ Ioo 0 T` gives `0 < s` and `s < T`.

## Why this is the right theorem, not a stronger one

Do **not** try to prove:

```lean
ContinuousOn (coupledChemDivSourceLift p u s) (Set.Icc (0 : ℝ) 1)
```

That reintroduces the zero-extension endpoint trap. `coupledChemDivSourceLift` is an `intervalDomainLift`, so its values just outside `[0,1]` are zero. The physical derivative expression generally has endpoint traces determined by the interior product rule; these need not equal the exterior zero-extension trace. Task303 correctly made endpoint values irrelevant.

Do **not** try to prove continuity of `deriv (chemFluxLifted p (u s))` as a function on closed `[0,1]`. The zero-extended flux can have endpoint derivative convention issues. The correct object is the physical product-rule representative, which is continuous on the closed interval but only equal to the literal source on the open interval.

Do **not** chase `solution_chemotaxisFlux_hasDerivAt` alone. It gives pointwise differentiability and identifies the derivative value at interior subtype points, but it does not by itself package a continuous closed representative. The product-rule representative theorem already does that.
