### 1. Real-rpow derivative of the logistic part

This is the exact local derivative lemma that is not currently named for `H1PhysicalLogisticReactionPart`:

```lean
/-- Fixed-time derivative of the logistic H¹ reaction factor.
The real-rpow chain rule uses strict positivity of the classical solution. -/
theorem H1PhysicalLogisticReactionPart_hasDerivAt_of_classicalSolution
    {p : CM2Params} {T τ x : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hτ : τ ∈ Set.Ioo (0 : ℝ) T)
    (hx : x ∈ Set.Ioo (0 : ℝ) 1) :
    HasDerivAt
      (fun y => H1PhysicalLogisticReactionPart p u τ y)
      ((p.a - p.b * (p.α + 1) *
          (intervalDomainLift (u τ) x) ^ p.α) *
        deriv (intervalDomainLift (u τ)) x)
      x
```

Proof route:

* Closed C² slice: use
  ```lean
  hCu := (hsol.regularity.2.2.2.2.1 τ hτ).1.1
  ```
  to get differentiability of `intervalDomainLift (u τ)` on the spatial interior via the already-used `hasDerivAt_of_contDiffOn_two_interior` pattern.
* Positivity: use `hsol.u_pos' hτ.1 hτ.2` after converting `x ∈ Ioo 0 1` to `x ∈ Icc 0 1`; this is required for `Real.rpow` with real exponent `p.α`.
* Chain rule pattern: copy the already-landed proof style from
  ```lean
  intervalDomain_lp_diffusion_test_deriv_eq
  ```
  in `IntervalDomainLpEnergyFrontiers.lean`, which uses `HasDerivAt.rpow_const` under strict positivity.
* Algebra: `u * (a - b*u^α) = a*u - b*u^(α+1)`, so the derivative is
  `(a - b*(α+1)*u^α) * u_x`.

### 2. Fixed-time logistic IBP equality

This is the main missing theorem:

```lean
/-- Logistic reaction scalar after one spatial integration by parts. -/
theorem H1PhysicalReactX_eq_logisticSlope_grad_sq_of_classicalSolution
    {p : CM2Params} {T τ : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hτ : τ ∈ Set.Ioo (0 : ℝ) T) :
    H1PhysicalReactX p u τ =
      ∫ x in (0 : ℝ)..1,
        (p.a - p.b * (p.α + 1) *
            (intervalDomainLift (u τ) x) ^ p.α) *
          (deriv (intervalDomainLift (u τ)) x) ^ 2
```

Proof route:

* Unfold
  ```lean
  H1PhysicalReactX
  H1PhysicalLogisticReactionPart
  ```
* Apply one-dimensional integration by parts to
  `q(x) = H1PhysicalLogisticReactionPart p u τ x` and
  `g(x) = deriv (intervalDomainLift (u τ)) x`:
  ```text
  ∫ q * g' = [q*g]_0^1 - ∫ q' * g
  ```
  hence
  ```text
  -∫ u_xx * q = ∫ q' * u_x
  ```
  because the Neumann endpoint derivative `u_x` vanishes at `0` and `1`.
* The appropriate existing IBP engine is either the raw Mathlib interval theorem used throughout the repo,
  ```lean
  intervalIntegral.integral_mul_deriv_eq_deriv_mul_of_hasDerivAt
  ```
  or the domain wrapper
  ```lean
  intervalDomain_spatial_integrationByParts_identity
  ```
  from `IntervalDomainNeumannIBP.lean`.
* Be careful: `gradIBP` in `IntervalChiNegH1EnergyCore.lean` is too specialized for `∫ w*w'' = -∫(w')^2`; it does not by itself handle the distinct logistic test function `q`.

Existing APIs for the proof obligations:

* closed C² for the `u` slice:
  ```lean

  ```
* derivative of the lifted `u` on the interior:

* integrability of `u_xx`:
  ```lean

  ```
* integrability of `u_x` and of the logistic derivative factor:
  use `intervalIntegrable_deriv_of_contDiffOn_two` for `u_x`, and continuity/compactness or a `ContDiffOn ℝ 1/2` proof for the logistic factor.
* endpoint boundary terms:
  use the classical Neumann field through either
  ```lean
  hsol.neumann hτ.1 hτ.2 intervalDomain_leftEndpoint_mem_boundary
  hsol.neumann hτ.1 hτ.2 intervalDomain_rightEndpoint_mem_boundary
  ```

### 3. Inequality with `L = p.a`

```lean
/-- The logistic reaction scalar is bounded by `a * ||u_x||₂²`. -/
theorem H1PhysicalReactX_le_a_H1gradL2Norm_sq_of_classicalSolution
    {p : CM2Params} {T τ : ℝ}
    {u v : ℝ → intervalDomainPoint → ℝ}
    (hsol : IsPaper2ClassicalSolution intervalDomain p T u v)
    (hτ : τ ∈ Set.Ioo (0 : ℝ) T) :
    H1PhysicalReactX p u τ ≤ p.a * (H1gradL2Norm u τ) ^ 2
```

Proof route:

* Rewrite by `H1PhysicalReactX_eq_logisticSlope_grad_sq_of_classicalSolution`.
* Pointwise coefficient bound:
  ```text
  p.a - p.b * (p.α + 1) * u^p.α ≤ p.a
  ```
  because `0 ≤ p.b`, `0 < p.α`, hence `0 ≤ p.α + 1`, and `0 ≤ u^p.α` from positivity/nonnegativity.
* Multiply by `(u_x)^2 ≥ 0` and integrate with `intervalIntegral.integral_mono_on`.
* Convert the gradient integral using
  ```lean
  H1gradL2Norm_sq
  ```
  and the definition
  ```lean
  H1energy u τ = (1 / 2) * ∫ x in 0..1, (deriv (intervalDomainLift (u τ)) x)^2
  ```
  so `(H1gradL2Norm u τ)^2 = ∫ x in 0..1, u_x^2`.

## Pitfalls

1. **Do not prove this by Cauchy-Schwarz.** Cauchy gives an unwanted `H1lapL2Norm` factor. The desired bound is a gradient-square bound and needs the logistic integration by parts.

2. **Do not use the H¹ identity route.** The reaction estimate is a source-side scalar estimate for the sqrt package. It should not consume `H1PhysicalRHSIdentityBefore`, RHS integrability, the strict-initial route, or bounded-before data.

3. **Do not ignore the real-rpow domain condition.** Since `α : ℝ`, the derivative of `u^α` must use strict positivity of `u`. This is exactly `hsol.u_pos'` on the closed interval.

4. **Do not rely on `gradIBP` alone.** `gradIBP` is the same-function identity `∫ w*w'' = -∫(w')²`; the logistic term needs a two-function IBP with test `u(a-bu^α)` and function `u`.

5. **Keep taxis/uvxx separate.** Proving this reaction field only fills `react_bound`; it does not discharge `taxis_l2_bound`, `uvxx_l2_bound`, or any zero-start primitive production.
