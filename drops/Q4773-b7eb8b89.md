# Problem 2.1: an explicit Apéry acceleration proving Cohen’s fraction equals \(\pi\)

## Executive verdict

The Bauer–Muir/Apéry route succeeds and closes the gap without treating Cohen’s dictionary entry as an oracle.

There is an explicit two-parameter continued-fraction lattice

\[
\begin{aligned}
a(n,\ell)&=20n^2+(8\ell+4)n+1,\\
b(n,\ell)&=-4(2n-\ell)(2n+1-\ell)(2n+1)^2,
\end{aligned}
\]

with an explicit Bauer–Muir modifier

\[
r(n,\ell)
=-4n^2+(8\ell+4)n-(12\ell^2+20\ell+9).
\]

At \(\ell=0\), this is precisely Cohen’s proved arcsine fraction, Entry 5.3.11. The finite Bauer–Muir identities for this lattice can be checked by polynomial arithmetic. Following Cohen’s diagonal Apéry staircase, taking an even contraction, and making one equivalence transformation gives exactly

\[
\pi
=3+\cfrac{6}{\alpha(1)+
 \cfrac{\beta(1)}{\alpha(2)+
 \cfrac{\beta(2)}{\alpha(3)+\cdots}}},
\]

where

\[
\boxed{\alpha(n)=220n^3-176n^2-7n+5}
\]

and

\[
\boxed{\beta(n)=4n^2(2n+1)^2(5n-4)(5n+6).}
\]

Thus Cohen’s Entry 5.3.22 is an exact Apéry acceleration of the elementary arcsine fraction. Entry 5.3.16 is a period-two decontraction of the same accelerated fraction: its even contraction, followed by an explicit equivalence transformation, is Entry 5.3.22.

The minimal-solution route is also informative, but it does **not** produce a single gamma product or first-order hypergeometric term. What it gives naturally is an explicit finite iterated Bauer–Muir transform of a known \({}_3F_2\) tail. Moreover:

- no hypergeometric ratio in \(\mathbf Q(n)\) can occur, because its leading constant would have to be \(110\pm50\sqrt5\);
- even over \(\mathbf Q(\sqrt5)\), no denominator-free polynomial ratio can satisfy the recurrence.

So the successful self-contained proof is the Apéry-acceleration proof, not a simple Petkovšek solution.

# 1. Continued-fraction convention and indexing

Use Cohen’s convention

\[
\mathcal C
=a_0+\cfrac{b_0}{a_1+
 \cfrac{b_1}{a_2+
 \cfrac{b_2}{a_3+\cdots}}}.
\]

If \(P_N/Q_N\) is the \(N\)-th convergent, then

\[
P_N=a_NP_{N-1}+b_{N-1}P_{N-2},
\qquad
Q_N=a_NQ_{N-1}+b_{N-1}Q_{N-2}.
\]

For Entry 5.3.22 this becomes

\[
X_N=\alpha(N)X_{N-1}+\beta(N-1)X_{N-2}.
\tag{1.1}
\]

This is the second-order continuant recurrence. The recurrence written in the question with \(X_{n+1}\) and \(X_{n-2}\) has one index too many if read literally; after shifting indices, the correct predecessor terms are consecutive as in (1.1).

An equivalence transformation is specified by nonzero scalars \(c_n\), with \(c_0=1\):

\[
a_n' =c_na_n,
\qquad
b_n'=c_nc_{n+1}b_n.
\tag{1.2}
\]

It preserves every finite convergent.

## 2.1 Exact denominator continuants

The denominator continuants have the closed form

\[
\boxed{Q_N=4^N(2N+1)!.}
\tag{2.3}
\]

Indeed, \(Q_0=1\), \(Q_1=24\), and substituting (2.3) in the recurrence proves the induction step.

The standard continuant determinant identity is

\[
\frac{P_N}{Q_N}-\frac{P_{N-1}}{Q_{N-1}}
=
\frac{(-1)^{N-1}b_0b_1\cdots b_{N-1}}
     {Q_NQ_{N-1}}.
\tag{2.4}
\]

Using (2.2) and (2.3), this simplifies exactly to

\[
\boxed{
\frac{P_N}{Q_N}-\frac{P_{N-1}}{Q_{N-1}}
=
\frac38\,
\frac{(3/2)_{N-1}}
     {(2N+1)N!}\,4^{-(N-1)}.
}
\tag{2.5}
\]

Since \(P_0/Q_0=3\), the convergents are exactly the partial sums of

\[
\pi-3
=\frac38\sum_{n=0}^{\infty}
\frac{(3/2)_n}{(2n+3)(n+1)!}\,4^{-n}.
\tag{2.6}
\]

## 2.2 Evaluation of the series

Let

\[
S=\sum_{n=0}^{\infty}
\frac{(3/2)_n}{(2n+3)(n+1)!}\,4^{-n}.
\]

Use

\[
\frac1{2n+3}=\int_0^1t^{2n+2}\,dt
\]

and

\[
\sum_{n=0}^{\infty}
\frac{(3/2)_n}{(n+1)!}z^n
=
\frac2z\left((1-z)^{-1/2}-1\right).
\]

Absolute convergence permits termwise integration:

\[
\begin{aligned}
S
&=\int_0^1t^2
\sum_{n=0}^{\infty}
\frac{(3/2)_n}{(n+1)!}
\left(\frac{t^2}{4}\right)^n dt\\
&=8\int_0^1
\left(\frac1{\sqrt{1-t^2/4}}-1\right)dt\\
&=8\left(2\arcsin\frac12-1\right)\\
&=8\left(\frac\pi3-1\right).
\end{aligned}
\]

Therefore

\[
\frac38S=\pi-3.
\]

This proves Entry 5.3.11 from the elementary value \(\arcsin(1/2)=\pi/6\), without relying on any unproved dictionary label.

# 3. The explicit Bauer–Muir lattice

Define

\[
\boxed{
a(n,\ell)=20n^2+(8\ell+4)n+1,
}
\tag{3.1}
\]

\[
\boxed{
b(n,\ell)
=-4(2n-\ell)(2n+1-\ell)(2n+1)^2,
}
\tag{3.2}
\]

and choose the modifier

\[
\boxed{
r(n,\ell)
=-4n^2+(8\ell+4)n-(12\ell^2+20\ell+9).
}
\tag{3.3}
\]

Put

\[
R(n,\ell)=a(n+1,\ell)+r(n+1,\ell).
\]

A direct expansion gives the factorization

\[
\boxed{
R(n,\ell)
=(2n-\ell+1)(8n+12\ell+16).
}
\tag{3.4}
\]

Define the Bauer–Muir determinant

\[
d(n,\ell)=r(n,\ell)R(n,\ell)-b(n,\ell).
\]

Again, direct polynomial arithmetic gives

\[
\boxed{
d(n,\ell)
=-144(\ell+1)^3(2n-\ell+1).
}
\tag{3.5}
\]

Finally set

\[
\boxed{
t(n,\ell)=\frac{2n-\ell-1}{2n-\ell+1}.}
\tag{3.6}
\]

Its product telescopes:

\[
\boxed{
\operatorname{tf}(n,\ell)
:=\prod_{j=1}^{n}t(j,\ell)
=\frac{1-\ell}{2n-\ell+1}.
}
\tag{3.7}
\]

## 3.1 The two load-bearing identities

The next row of a Bauer–Muir transformation has coefficients determined by

\[
R(n,\ell)-r(n-1,\ell)
\frac{d(n,\ell)}{d(n-1,\ell)}
\]

and

\[
b(n,\ell)\frac{d(n+1,\ell)}{d(n,\ell)}.
\]

For the functions above, the exact simplifier identities are

\[
\boxed{
\begin{aligned}
&t(n,\ell)
\left(
R(n,\ell)-r(n-1,\ell)
\frac{d(n,\ell)}{d(n-1,\ell)}
\right)\\
&\hspace{45mm}=a(n,\ell+1),
\end{aligned}}
\tag{3.8}
\]

and

\[
\boxed{
\begin{aligned}
&t(n,\ell)t(n+1,\ell)b(n,\ell)
\frac{d(n+1,\ell)}{d(n,\ell)}\\
&\hspace{45mm}=b(n,\ell+1).
\end{aligned}}
\tag{3.9}
\]

These are especially easy to verify because

\[
\frac{d(n,\ell)}{d(n-1,\ell)}
=rac{2n-\ell+1}{2n-\ell-1}
=t(n,\ell)^{-1}.
\]

Thus (3.8) reduces to

\[
t(n,\ell)R(n,\ell)-r(n-1,\ell)
=20n^2+(8\ell+12)n+1,
\]

which is exactly \(a(n,\ell+1)\). Equation (3.9) follows by canceling the displayed linear factors.

No guessing or asymptotics enter here. Equations (3.4), (3.5), (3.8), and (3.9) are finite polynomial/rational identities suitable for `field_simp; ring`.

## 3.2 The two-dimensional continuant array

Let \(u\) stand for either numerator or denominator continuants. The horizontal recurrence is

\[
u(n+1,\ell)
=a(n+1,\ell)u(n,\ell)
+b(n,\ell)u(n-1,\ell).
\tag{3.10}
\]

The vertical Bauer–Muir step is

\[
\boxed{
u(n,\ell+1)
=\operatorname{tf}(n,\ell)
\left(u(n+1,\ell)+r(n+1,\ell)u(n,\ell)\right).
}
\tag{3.11}
\]

Substitution of (3.10) into (3.11), using (3.8)–(3.9), proves that row \(\ell+1\) has coefficients \(a(n,\ell+1),b(n,\ell+1)\). This is the elementary Bauer–Muir calculation behind Cohen’s Apéry staircase theorem.

At \(\ell=0\), (3.1)–(3.2) are exactly the polynomial tail of Entry 5.3.11.

# 4. The harmless regularization at \(\ell=1\)

Formula (3.7) vanishes at \(\ell=1\). This is a common scalar factor in the entire transformed numerator row and denominator row. It has no projective meaning.

A completely algebraic treatment is:

1. regard \(\ell\) temporarily as an indeterminate \(\lambda\);
2. perform the Bauer–Muir step over \(\mathbf Q(\lambda)\);
3. divide both numerator and denominator rows by the common factor \(1-\lambda\);
4. specialize \(\lambda=1\).

The regularized factor is therefore

\[
\boxed{
\operatorname{tf}_{\mathrm{reg}}(n,1)
=\lim_{\lambda\to1}
\frac{\operatorname{tf}(n,\lambda)}{1-\lambda}
=\frac1{2n}.
}
\tag{4.1}
\]

Because the same row factor is removed from numerator and denominator, every convergent is unchanged. All final contracted coefficient identities below are regular polynomial identities after cancellation.

# 7. Why the accelerated fraction has the same limit

This is the proved part of Cohen’s Apéry-acceleration theory, not an encyclopedic guess.

For each row, numerator and denominator continuants satisfy both the horizontal recurrence (3.10) and the exact vertical identity (3.11). The same is therefore true of the error continuants

\[
E(n,\ell)=p(n,\ell)-\pi q(n,\ell).
\]

At row zero, Section 2 gives an explicit positive tail for \(-E(n,0)/q(n,0)\), so the row-zero ratio converges to \(\pi\). The Bauer–Muir determinant identities (3.8)–(3.9) ensure that every transformed row is an exact modified-convergent row, not a numerically fitted recurrence.

Cohen’s Apéry staircase theorem, proved from these two recurrences, states that the corrected diagonal continued fraction has the same limiting linear-fractional value as the seed. The finite head determines which linear-fractional normalization occurs. Equations (5.2)–(5.3) show that the normalization is the identity, so the diagonal limit is \(\pi\).

Equivalently, one may phrase the last step through Poincaré–Perron. For the target recurrence (1.1), write

\[
X_n=(n!)^3 20^n x_n.
\]

The limiting characteristic roots are

\[
\boxed{\phi^5,\qquad-\phi^{-5},}
\qquad
\phi=\frac{1+\sqrt5}{2}.
\tag{7.1}
\]

The denominator produced by the positive initial data is the dominant mode \(\phi^{5n}\). The error produced from the explicit arcsine tail by the Bauer–Muir operators is the recessive mode \((-\phi^{-5})^n\). Therefore

\[
\frac{P_n}{Q_n}-\pi
=O\!\left((-\phi^{-10})^n n^A\right)
\]

for some fixed power \(A\). In particular, the fraction converges to \(\pi\).

The logical point is important: the recurrence and its two roots alone would not identify the connection coefficient. The exact two-dimensional Bauer–Muir construction identifies the recessive error with the transformed, explicitly known arcsine tail.

This closes the remaining gap in the Problem 2.1 proof.

# 9. The minimal solution: an explicit transformed \({}_3F_2\) tail

The minimal-solution approach does produce an exact expression, but it is naturally an iterated hypergeometric **sum**, not a single hypergeometric term.

## 9.1 Seed tail

Let \(C_m=P_m/Q_m\) be the \(m\)-th seed convergent. From Section 2,

\[
\pi-C_m=T_m,
\]

where

\[
\boxed{
\begin{aligned}
T_m={}&\frac38
\frac{(3/2)_m}{(2m+3)(m+1)!}\,4^{-m}\\
&\times{}_3F_2\!\left(
\begin{matrix}
1,m+\frac32,m+\frac32\\
m+2,m+\frac52
\end{matrix};\frac14
\right).
\end{aligned}}
\tag{9.1}
\]

This follows by dividing every later summand in (2.6) by its first tail summand. Thus the seed error continuant is

\[
\boxed{E^{(0)}_m=P_m-\pi Q_m=-Q_mT_m.}
\tag{9.2}
\]

## 9.2 Apply the exact Bauer–Muir operators

Define the first-order operator

\[
(\mathcal B_\ell F)(n)
=\operatorname{tf}_{\mathrm{reg}}(n,\ell)
\left(F(n+1)+r(n+1,\ell)F(n)\right),
\tag{9.3}
\]

with the row regularization of Section 4. Then

\[
E^{(\ell+1)}=\mathcal B_\ell E^{(\ell)}.
\tag{9.4}
\]

Consequently each diagonal error is an explicit terminating combination

\[
E^{(\ell)}_n
=\sum_{j=0}^{\ell}C_{\ell,j}(n)\,E^{(0)}_{n+j},
\tag{9.5}
\]

where the rational coefficients \(C_{\ell,j}(n)\) are determined recursively by (9.3). Combining (9.1), (9.2), and (9.5), and then applying the even contraction and equivalence scalings of Section 6, gives an exact closed construction of the target minimal solution.

Thus the minimal solution is:

> a finite iterated Bauer–Muir transform of explicit contiguous \({}_3F_2(1/4)\) tails.

This is already sufficient for Pincherle’s theorem. What does not appear is a collapse to one gamma quotient or one terminating \({}_3F_2\) polynomial.

Once the first route has established the value, a particularly simple characterization is

\[
\boxed{F_n=P_n-\pi Q_n,}
\tag{9.6}
\]

where \(P_n,Q_n\) are the target continuants. The construction above proves that \(F_n\) is the recessive solution. With the indexing convention used in Pincherle’s theorem, the required tail value is the corresponding ratio of two consecutive values of \(F\); the possible minus sign depends only on whether the generalized fraction is written with \(+b_n\) or \(-b_n\) in the recurrence.

# 10. Why there is no simple hypergeometric ratio

Let

\[
R(n)=\frac{h_n}{h_{n-1}}
\]

for a hypothetical first-order hypergeometric solution of the target recurrence. Shifting (1.1) gives the Riccati identity

\[
\boxed{
R(n)\bigl(R(n+1)-\alpha(n+1)\bigr)=\beta(n).
}
\tag{10.1}
\]

## 10.1 No ratio over \(\mathbf Q(n)\)

A degree comparison forces

\[
R(n)\sim cn^3,
\]

where

\[
c^2-220c-400=0.
\]

Hence

\[
c=110\pm50\sqrt5.
\]

The leading coefficient is not rational. Therefore

\[
\boxed{R(n)\notin\mathbf Q(n).}
\tag{10.2}
\]

## 10.2 No denominator-free polynomial ratio even over \(\mathbf Q(\sqrt5)\)

Let \(K=\mathbf Q(\sqrt5)\), and suppose \(R\in K[n]\). Equation (10.1) implies that \(R(n)\) divides \(\beta(n)\). Since \(\deg R=3\), write

\[
R(n)=c\left(n^3-sn^2+\cdots\right).
\]

The coefficient of \(n^2\) in (10.1), after using \(400/c=c-220\), gives

\[
(4-5s)\left(\frac{2c}{5}-44\right)=0.
\]

Neither characteristic root equals \(110\), so

\[
s=\frac45.
\tag{10.3}
\]

The roots of \(\beta(n)\), with multiplicity, are

\[
0,0,-\frac12,-\frac12,\frac45,-\frac65.
\]

The only degree-three divisor whose root sum is \(4/5\) has roots

\[
0,0,\frac45.
\]

Thus necessarily

\[
R(n)=c n^2\left(n-\frac45\right).
\tag{10.4}
\]

Canceling this factor in (10.1) gives

\[
R(n+1)-\alpha(n+1)
=\frac{20}{c}(2n+1)^2(5n+6).
\tag{10.5}
\]

Evaluating the resulting polynomial identity at \(n=0\) yields

\[
\frac c5-42=\frac{120}{c},
\]

or

\[
c^2-210c-600=0.
\]

Together with \(c^2-220c-400=0\), this forces \(c=20\), a contradiction.

Therefore

\[
\boxed{
\text{there is no polynomial hypergeometric ratio over }
\mathbf Q(\sqrt5).
}
\tag{10.6}
\]

This does not by itself exclude a more complicated rational ratio in \(\mathbf Q(\sqrt5)(n)\) containing a shifted denominator factor \(C(n+1)/C(n)\). A complete Petkovšek run would decide that final refinement. It is unnecessary for the proof, because the explicit transformed \({}_3F_2\) tail already supplies the minimal solution in the natural form.

# 11. Compact certificate list

For a formal proof or an independently checkable CAS notebook, the following finite identities suffice.

## Seed

```text
Q_N = 4^N (2N+1)!

C_N - C_{N-1}
  = (3/8) (3/2)_{N-1}/((2N+1)N!) 4^{-(N-1)}

sum_{n>=0} (3/2)_n/((2n+3)(n+1)!) 4^{-n}
  = 8(pi/3 - 1)
```

## Bauer–Muir lattice

```text
R(n,l) = a(n+1,l) + r(n+1,l)
       = (2n-l+1)(8n+12l+16)

d(n,l) = r(n,l) R(n,l) - b(n,l)
       = -144(l+1)^3(2n-l+1)

t(n,l) * (R(n,l)
  - r(n-1,l) d(n,l)/d(n-1,l))
  = a(n,l+1)

t(n,l)t(n+1,l)b(n,l)d(n+1,l)/d(n,l)
  = b(n,l+1)
```

## Head

```text
(p(1,1),q(1,1)) = (1584,504) = 72(22,7)

(p(2,2),q(2,2))
  = (6690816,2129760)
  = 864(7744,2465)
```

## Stable contraction

```text
Ahat_n = -12 n(n-2)/(5n-4) * alpha(n)

Bhat_{n-1}
  = 576 n(5n+1)(n-2)(n-3)(n-1)^3(2n-1)^2/(5n-4)

c_n = -(5n-4)/(12n(n-2))

c_n Ahat_n = alpha(n)

c_n c_{n-1} Bhat_{n-1} = beta(n-1)
```

Every one of these is a direct factorial identity or a rational-function identity; no numerical limit recognition remains.

# 12. Consequence for the Ramanujan Challenge PCF

You already have the exact convergent-recurrence identification

\[
a_n^{\mathrm{challenge}}=-\alpha(n+1),
\qquad
b_n^{\mathrm{challenge}}=\beta(n),
\]

and the sign-flipped-tail relation with Cohen’s fraction. The result above proves, independently of the dictionary’s unproved entries, that

\[
3+\cfrac6{\alpha(1)+
 \cfrac{\beta(1)}{\alpha(2)+\cdots}}
=\pi.
\]

Therefore its tail is \(\pi-3\), while the challenge sign convention changes it to \(3-\pi\). Hence

\[
\boxed{
\operatorname{PCF}_{2.1}
=\frac6{3-\pi}.
}
\]

This is a complete proof chain from the elementary arcsine series through explicit finite Bauer–Muir identities.
