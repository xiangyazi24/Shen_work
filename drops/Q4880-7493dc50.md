# Problem 2.5: is the Catalan CMF a symmetric square?

## Executive conclusion

The silver-ratio spectrum is exactly the spectrum of a Delannoy symmetric square, but it is not by itself an exact symmetric-square certificate.

The rigorous conclusions are:

1. The recurrence written in the question is not the central-Delannoy recurrence. The correct recurrence for
   
   \[
   D_n=P_n(3)=1,3,13,63,321,\ldots
   \]
   
   is
   
   \[
   \boxed{(n+1)D_{n+1}=3(2n+1)D_n-nD_{n-1}.}
   \tag{0.1}
   \]

2. The exact natural rank-two transfer matrix is
   
   \[
   \boxed{
   B_D(n)=
   \begin{pmatrix}
   0&-\dfrac{n+1}{n+2}\\[2mm]
   1&\dfrac{3(2n+3)}{n+2}
   \end{pmatrix}.}
   \tag{0.2}
   \]
   
   Its symmetric square has Poincaré polynomial
   
   \[
   (\xi-1)(\xi^2-34\xi+1).
   \]

3. The printed \(M(n)\) is **not literally** a symmetric square over \(\mathbf Q(n)\). Its determinant is not a cube, and its columns fail the Veronese rank-one identities.

4. After the Pochhammer twist
   
   \[
   H_n=(-16)^n(2)_n^2(3)_n^2
   \left(\frac52\right)_n
   \left(\frac72\right)_n^2,
   \]
   
   the determinant obstruction to an \(n\)-dependent rational gauge disappears. The balanced limiting matrix is explicitly conjugate over \(\mathbf Q\) to the symmetric square of the limiting Delannoy matrix.

5. The exact statement still to be proved is
   
   \[
   \boxed{
   M_H(n)U(n+1)=U(n)\operatorname{Sym}^2_{\rm row}B_D(n),
   \qquad
   M_H(n)=\frac{M(n)}{H_{n+1}/H_n}.}
   \tag{0.3}
   \]
   
   No invertible rational \(U(n)\) satisfying (0.3) is presently certified.

6. The formal exponents matter. After the \(H_n\) normalization, the CMF exponents are
   
   \[
   0,-3,0,
   \]
   
   while the natural Delannoy symmetric-square exponents are
   
   \[
   -1,-1,-1.
   \]
   
   Hence a possible gauge is necessarily **singular at infinity**: it shifts the two outer modes by \(n^{-1}\) and the neutral mode by \(n^2\). A search restricted to matrices tending to a constant is therefore invalid.

7. The decisive invariant is a rational moving Veronese conic. In the row convention below, a symmetric-square gauge implies a nonsingular rational symmetric matrix \(J(n)\) satisfying
   
   \[
   \boxed{
   \overline M(n)J(n+1)\overline M(n)^T
   =\left(\frac{n+1}{n+2}\right)^2J(n),}
   \tag{0.4}
   \]
   
   where \(\overline M\) is the balanced \(H\)-normalized matrix. This is a linear rational-difference problem in six unknown functions.

8. Even a positive symmetric-square result would **not automatically prove the Catalan limit**. It identifies the homogeneous carrier, not the global connection coefficient selected by the initial rows. The plain Legendre module naturally supplies \(\log 2\); \(G\) belongs to an integrated or parameter-derivative extension. Also, the irrationality of \(G\) is unknown, so no known “Delannoy irrationality measure for \(G\)” can be invoked.

The Sage program in Section 8 performs the corrected search. It allows the required \(n^2\) growth, verifies all identities exactly, and distinguishes a genuine invertible gauge from a singular morphism.

# 1. Exact Delannoy rank-two and rank-three systems

Set

\[
\alpha_n=\frac{n+1}{n+2},
\qquad
\beta_n=\frac{3(2n+3)}{n+2}.
\tag{1.1}
\]

For the row state \(r_n=(D_n,D_{n+1})\), equation (0.1) gives

\[
r_{n+1}=r_nB_D(n),
\qquad
B_D(n)=
\begin{pmatrix}0&-\alpha_n\\1&\beta_n\end{pmatrix}.
\tag{1.2}
\]

For a row action by

\[
B=\begin{pmatrix}a&b\\c&d\end{pmatrix},
\]

the symmetric square in the basis \((x^2,xy,y^2)\) is

\[
\boxed{
\operatorname{Sym}^2_{\rm row}(B)=
\begin{pmatrix}
a^2&ab&b^2\\
2ac&ad+bc&2bd\\
c^2&cd&d^2
\end{pmatrix}.}
\tag{1.3}
\]

Therefore

\[
\boxed{
S_D(n)=\operatorname{Sym}^2_{\rm row}B_D(n)
=
\begin{pmatrix}
0&0&\alpha_n^2\\
0&-\alpha_n&-2\alpha_n\beta_n\\
1&\beta_n&\beta_n^2
\end{pmatrix}.}
\tag{1.4}
\]

It sends

\[
(D_n^2,D_nD_{n+1},D_{n+1}^2)
\]

to

\[
(D_{n+1}^2,D_{n+1}D_{n+2},D_{n+2}^2).
\]

Its determinant is

\[
\boxed{
\det S_D(n)=\left(\frac{n+1}{n+2}\right)^3.}
\tag{1.5}
\]

Eliminating the mixed product gives

\[
\boxed{
\begin{aligned}
&(2n+3)(n+3)^2u_{n+3}\\
&\quad-(2n+5)(35n^2+140n+131)u_{n+2}\\
&\quad+(2n+3)(35n^2+140n+131)u_{n+1}\\
&\quad-(2n+5)(n+1)^2u_n=0.
\end{aligned}}
\tag{1.6}
\]

The limiting rank-two matrix is

\[
B_D(\infty)=
\begin{pmatrix}0&-1\\1&6\end{pmatrix},
\]

with roots \(3\pm2\sqrt2\). Its symmetric-square roots are

\[
(3+2\sqrt2)^2,\quad1,\quad(3-2\sqrt2)^2
=17+12\sqrt2,\quad1,\quad17-12\sqrt2.
\]

# 2. Literal symmetric square: exact no-go

For every \(2\times2\) matrix \(B\),

\[
\det\operatorname{Sym}^2(B)=(\det B)^3.
\tag{2.1}
\]

The printed determinant is

\[
\boxed{
\det M(n)=
-8(n+1)(n+2)^6(n+3)^5
(2n+3)^2(2n+5)^3(2n+7)^4.}
\tag{2.2}
\]

Its divisor has exponents not divisible by three, so

\[
\det M(n)\notin\mathbf Q(n)^{\times3}.
\]

Thus

\[
\boxed{M(n)\ne\operatorname{Sym}^2B(n)}
\]

for every \(B(n)\in\mathrm{GL}_2(\mathbf Q(n))\), even after a constant conjugation.

A literal row symmetric square also obeys

\[
s_{12}^2=s_{11}s_{13},
\qquad
s_{32}^2=s_{31}s_{33},
\]

\[
s_{21}^2=4s_{11}s_{31},
\qquad
s_{23}^2=4s_{13}s_{33}.
\tag{2.3}
\]

The printed columns fail these identities.

An \(n\)-dependent difference gauge is different: it changes the determinant by

\[
\det U(n+1)/\det U(n).
\]

Hence the literal no-go does not settle the difference-module question.

# 3. Pochhammer twist and balanced system

The one-step ratio of \(H_n\) is

\[
\boxed{
\delta_H(n)=\frac{H_{n+1}}{H_n}
=-2(n+2)^2(n+3)^2(2n+5)(2n+7)^2.}
\tag{3.1}
\]

Define

\[
M_H(n)=\frac{M(n)}{\delta_H(n)}.
\tag{3.2}
\]

Then

\[
\boxed{
\det M_H(n)
=\frac{(n+1)(2n+3)^2}{(n+3)(2n+7)^2}.}
\tag{3.3}
\]

Introduce the balancing matrix

\[
D(n)=\operatorname{diag}(1,n+1,(n+1)^2)
\]

and

\[
\boxed{
\overline M(n)=D(n)^{-1}M_H(n)D(n+1).}
\tag{3.4}
\]

Its limit is

\[
\boxed{
A_\infty=
\begin{pmatrix}
17&-24&0\\
-12&17&0\\
8&-12&1
\end{pmatrix}.}
\tag{3.5}
\]

The limiting Delannoy square is

\[
\boxed{
S_\infty=
\begin{pmatrix}
0&0&1\\
0&-1&-12\\
1&6&36
\end{pmatrix}.}
\tag{3.6}
\]

An explicit rational intertwiner at the level of limiting matrices is

\[
\boxed{
V_\infty=
\begin{pmatrix}
1&3&17\\
0&-2&-12\\
0&0&8
\end{pmatrix},
\qquad
A_\infty V_\infty=V_\infty S_\infty.}
\tag{3.7}
\]

This proves the symmetric-square statement at infinity.

# 4. The required non-scalar formal shear

The \(H_n\)-normalized CMF has formal powers

\[
\tau_+=0,
\qquad
\tau_0=-3,
\qquad
\tau_-=0.
\tag{4.1}
\]

The natural Delannoy symmetric-square system has power \(-1\) on all three products. Therefore an exact rational gauge, if it exists, has modal powers

\[
-1,\quad2,\quad-1.
\tag{4.2}
\]

Let \(P_0\) be the spectral projector of \(S_\infty\) for the eigenvalue \(1\). Since the other roots have sum \(34\) and product \(1\),

\[
\boxed{
P_0=\frac{34S_\infty-S_\infty^2-I}{32}
=
\begin{pmatrix}
-1/16&-3/16&-1/16\\
3/8&9/8&3/8\\
-1/16&-3/16&-1/16
\end{pmatrix}.}
\tag{4.3}
\]

A model formal shear is

\[
\boxed{
E(n)=\frac{1}{n+1}(I-P_0)+(n+1)^2P_0.}
\tag{4.4}
\]

It has determinant \(1\), because its modal eigenvalues are

\[
(n+1)^{-1},\quad(n+1)^2,\quad(n+1)^{-1}.
\]

Thus the expected leading form is

\[
\boxed{V(n)\sim V_\infty E(n),}
\tag{4.5}
\]

up to multiplication by a matrix commuting with \(S_\infty\) and lower formal corrections.

This is why a search of the form

\[
V(n)=V_\infty+O(n^{-1})
\]

is too restrictive. A correct ansatz must allow entries of growth \(n^2\).

# 5. Determinant of a possible gauge

The exact balanced equation is

\[
\boxed{
\overline M(n)V(n+1)=V(n)S_D(n).}
\tag{5.1}
\]

Taking determinants gives

\[
\frac{\det V(n+1)}{\det V(n)}
=\frac{\det S_D(n)}{\det\overline M(n)}.
\tag{5.2}
\]

Now

\[
\det\overline M(n)
=\frac{(n+2)^3(2n+3)^2}
{(n+1)^2(n+3)(2n+7)^2}.
\tag{5.3}
\]

Hence

\[
\boxed{
\frac{\det V(n+1)}{\det V(n)}
=
\frac{(n+1)^5(n+3)(2n+7)^2}
{(n+2)^6(2n+3)^2}.}
\tag{5.4}
\]

A rational primitive is

\[
\boxed{
\det V(n)
=C\frac{(n+2)(2n+3)^2(2n+5)^2}{(n+1)^5}.}
\tag{5.5}
\]

The formal normalization (3.7), (4.4) suggests \(C=-1\), for which the determinant tends to \(-16\). Any nonzero rational solution of (5.1) has determinant equal to (5.5) up to a constant.

For \(U(n)=D(n)V(n)\),

\[
\det U(n)
=C\frac{(n+2)(2n+3)^2(2n+5)^2}{(n+1)^2}.
\tag{5.6}
\]

Thus the determinant obstruction has been removed exactly, but existence remains unproved.

# 6. Moving Veronese conic

Use

\[
\boxed{
J_0=
\begin{pmatrix}
0&0&-1\\
0&2&0\\
-1&0&0
\end{pmatrix},}
\tag{6.1}
\]

which represents \(2(Y^2-XZ)\). For every \(2\times2\) matrix \(B\),

\[
\operatorname{Sym}^2_{\rm row}(B)
J_0
\operatorname{Sym}^2_{\rm row}(B)^T
=(\det B)^2J_0.
\tag{6.2}
\]

If \(V\) satisfies (5.1), then

\[
J(n)=V(n)J_0V(n)^T
\]

satisfies

\[
\boxed{
\overline M(n)J(n+1)\overline M(n)^T
=\left(\frac{n+1}{n+2}\right)^2J(n).}
\tag{6.3}
\]

This is linear in the six independent entries of \(J(n)\).

Because \(V\) has the shear (4.4), \(J(n)\) can grow like \(n^4\). A search restricted to a conic tending to a constant is again too small.

A nondegenerate rational solution of (6.3) is necessary. If the corresponding conic is split over \(\mathbf Q(n)\), it gives a projective rank-two lift. One must then straighten the conic and compare the resulting rank-two matrix with \(B_D(n)\).

---

# 7. Why this would not yet prove \(G\)

Suppose (5.1) is proved. Then the two CMF rows become rational quadratic combinations of two Delannoy solutions. The dominant limit is still

\[
L=
\frac{p_0U(0)w_+^{(2)}(0)}
     {q_0U(0)w_+^{(2)}(0)},
\tag{7.1}
\]

where \(w_+^{(2)}\) is the dominant adjoint functional of the symmetric square.

Nothing in the symmetric-square identity forces this ratio to equal \(G\). Indeed, replacing \(P_n\) by \(P_n+cQ_n\) preserves the same module but changes the limit by \(c\).

The ordinary second Legendre solution has

\[
\mathsf Q_0(3)=\frac12\log2,
\]

so the homogeneous Delannoy module naturally carries \(\log2\). Catalan's constant arises from

\[
G=\frac12\int_0^1K(k)\,dk,
\]

an integrated/parameter-derivative extension. A separate period or initial-connection identity is still required.

Finally, the irrationality of \(G\) is unknown. Therefore the proposed appeal to a known rank-two irrationality measure cannot be used.

def rational_to_poly(f):
    f = K(f)
    d = f.denominator()
    if d.degree() > 0:
        raise ValueError("denominator not cleared")
    return R(f.numerator()/QQ(d[0]))


def coeff(p,k):
    if p == 0 or k > p.degree():
        return QQ(0)
    return QQ(p[k])


def homogeneous_equation_matrix(Fbasis):
    """Coefficient matrix for sum c_i Fbasis[i](n)=0."""
    rows = []

    if not Fbasis:
        return matrix(QQ,0,0)

    nr,nc = Fbasis[0].nrows(),Fbasis[0].ncols()

    for i in range(nr):
        for j in range(nc):
            rats = [K(F[i,j]) for F in Fbasis]
            common = R(1)
            for f in rats:
                common = lcm(common,R(f.denominator()))

            polys = [rational_to_poly(K(common)*f) for f in rats]
            maxdeg = max([-1]+[p.degree() for p in polys if p != 0])

            for d in range(maxdeg+1):
                row = [coeff(p,d) for p in polys]
                if any(x != 0 for x in row):
                    rows.append(row)

    return matrix(QQ,rows)


def rational_matrix_basis(den,max_growth,symmetric=False):
    """
    Entries p(n)/den(n), with deg p <= deg(den)+max_growth.

    max_growth=2 permits n^2 growth.
    max_growth=4 permits n^4 growth.
    """
    den = R(den)
    maxdeg = den.degree()+max_growth
    if maxdeg < 0:
        return []

    basis = []
    positions = ([(i,j) for i in range(3) for j in range(3)]
                 if not symmetric else
                 [(0,0),(0,1),(0,2),(1,1),(1,2),(2,2)])

    for i,j in positions:
        for k in range(maxdeg+1):
            E = matrix(K,3,3,0)
            E[i,j] = n^k/den
            if symmetric:
                E[j,i] = n^k/den
            basis.append(E)
    return basis


def linear_combination(basis,v):
    A = matrix(K,3,3,0)
    for c,B in zip(v,basis):
        A += QQ(c)*B
    return A


def deterministic_vectors(kernel,max_pairs=30):
    """Small deterministic set of vectors in a nullspace."""
    out = list(kernel)
    m = len(kernel)
    count = 0
    for i in range(m):
        for j in range(i+1,m):
            for a,b in [(1,1),(1,-1),(2,1),(1,2)]:
                out.append(a*kernel[i]+b*kernel[j])
                count += 1
                if count >= max_pairs:
                    return out
    return out
