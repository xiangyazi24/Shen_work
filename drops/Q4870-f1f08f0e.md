# Problem 2.7: the exact error integral, and the missing non-circular kernel

Put

\[
L:=\zeta(2)+\zeta(3)
\]

and let \(p_n,q_n\) be the two rational solutions specified in the question. There are two distinct requests hidden in “find \(R_n\)”:

1. an **exact rational kernel** whose integral is \(q_nL-p_n\) and satisfies the recurrence;
2. an **independent Beukers/creative-telescoping kernel** of degree growing with \(n\), from which the recurrence and the decay follow without already using \(p_n,q_n\).

The first kernel can be written down completely. The second has not yet been identified from the supplied data. The spectral level-11 correspondence does not determine it.

## Executive answer

Define \(t=xy\) and

\[
\boxed{
R_n^{\rm aff}(x,y)
=q_n-\frac{p_n}{2}(1-xy)
=\left(q_n-\frac{p_n}{2}\right)+\frac{p_n}{2}xy.
}
\tag{1}
\]

Then

\[
\boxed{
\int_0^1\!\int_0^1
R_n^{\rm aff}(x,y)
\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy
=q_n\bigl(\zeta(2)+\zeta(3)\bigr)-p_n.
}
\tag{2}
\]

Because \(p_n\) and \(q_n\) satisfy the same scalar recurrence, \(R_n^{\rm aff}\) satisfies it **pointwise** in \(x,y\). Hence the creative-telescoping certificate is the exact zero certificate

\[
\mathcal L_nR_n^{\rm aff}=0
=\partial_x(0)+\partial_y(0).
\tag{3}
\]

This answers the literal integral question, but it is circular as a convergence proof: the kernel already contains the challenge solutions. It does not explain why the dominant mode cancels or why the remaining integral is small.

A genuine non-circular solution would require a polynomial \(Q_n(t)\), given independently by a Rodrigues formula, a terminating hypergeometric sum, or multiple orthogonality, such that

\[
R_n(x,y)=Q_n(xy),
\qquad Q_n(1)=q_n,
\tag{4}
\]

and the associated Hermite--Padé numerator equals \(p_n\). The most concrete candidate framework is the two-measure multiple-orthogonal system displayed in Section 5 below. Matching that system to the printed degree-12 recurrence is a finite exact computation, but it has not yet been certified.

# 1. A sign and indexing audit

The Poincaré polynomial quoted in the question fixes the alternating sign convention. The leading coefficients are

\[
\begin{aligned}
[A_n]_{n^{12}}&=991952896,\\
[B_n]_{n^{12}}&=852459520,\\
[C_n]_{n^{12}}&=484352,\\
[D_n]_{n^{12}}&=946.
\end{aligned}
\]

Thus the recurrence compatible with

\[
4\mu^3-220\mu^2+8\mu-1
\]

is

\[
\boxed{
A_nu_{n+3}-B_nu_{n+2}+C_nu_{n+1}-D_nu_n=0.
}
\tag{5}
\]

Indeed, with \(c=\mu/64\), its leading characteristic equation becomes

\[
991952896c^3-852459520c^2+484352c-946
=946\bigl(4\mu^3-220\mu^2+8\mu-1\bigr).
\]

The signs written in the question,

\[
A_nu_{n+3}+B_nu_{n+2}-C_nu_{n+1}+D_nu_n=0,
\]

instead give \(4\mu^3+220\mu^2-8\mu+1\). This convention must be fixed before checking any proposed certificate.

There is a second indexing convention in the challenge repository:

\[
u_{n+1}=\frac{B_n}{A_n}u_n
-\frac{C_{n-1}}{A_{n-1}}u_{n-1}
+\frac{D_{n-2}}{A_{n-2}}u_{n-2}.
\tag{6}
\]

Formula (1) is independent of this choice: let \(\mathcal L\) denote the **actual** common operator annihilating the supplied \(p_n,q_n\). Then every statement below remains valid with that \(\mathcal L\).

# 2. The mixed moment functional

For a polynomial \(R(t)\), define

\[
\mathcal W[R]
:=\int_0^1\!\int_0^1
R(xy)\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy.
\tag{7}
\]

For \(m\ge0\), expand \((1-xy)^{-1}\) geometrically. Since

\[
\int_0^1\!\int_0^1(xy)^a\,dx\,dy=\frac1{(a+1)^2}
\]

and

\[
\int_0^1\!\int_0^1(xy)^a\log(xy)\,dx\,dy
=-\frac2{(a+1)^3},
\]

one obtains

\[
\begin{aligned}
\mathcal W[t^m]
&=\sum_{r=0}^{\infty}
\left(\frac1{(m+r+1)^2}+\frac1{(m+r+1)^3}\right)\\
&=\boxed{
L-H_m^{(2)}-H_m^{(3)}.}
\end{aligned}
\tag{8}
\]

In particular,

\[
\boxed{\mathcal W[1]=L,}
\qquad
\boxed{\mathcal W[1-t]=2.}
\tag{9}
\]

Equation (2) now follows at once:

\[
\mathcal W\!\left[q_n-\frac{p_n}{2}(1-t)\right]
=q_nL-p_n.
\]

More generally, if

\[
R(t)=\sum_{m=0}^d r_mt^m,
\]

then

\[
\boxed{
\mathcal W[R]
=R(1)L-
\sum_{m=0}^d r_m\bigl(H_m^{(2)}+H_m^{(3)}\bigr).
}
\tag{10}
\]

This identity is useful for testing any proposed polynomial kernel entirely by rational arithmetic.

With the initial values in the question, formula (1) gives

For \(n=1\),

For \(n=2\),

\boxed{

}
\tag{13}

Define all later \(R_n(t)\) by the same scalar recurrence, coefficientwise in \(\mathbf Q[t]\). Uniqueness of the recurrence solution proves that the resulting polynomial is exactly (1) for every \(n\).

## Parameter form

\[
J_n(s)=\int_0^1\!\int_0^1
\frac{(xy)^sR_n^{\rm aff}(xy)}{1-xy}\,dx\,dy.
\tag{15}

Then

\[
\boxed{
I_n=J_n(0)-\frac12J_n'(0).
}
\tag{16}
\]

For the affine kernel,

\[
J_n(s)
=q_n\zeta_H(2,s+1)
-\frac{p_n}{2(s+1)^2},
\tag{17}
\]

so (16) is another direct verification of (2).

# 4. Why this is not yet a Beukers proof

The zero certificate (3) is mathematically valid, but it transports the recurrence through coefficients already known to satisfy it. It supplies no independent estimate for \(I_n\).

There is also no uniqueness from the instruction “degree about \(n\).” Put

\[
h_m=H_m^{(2)}+H_m^{(3)}.
\]

For every \(m\ge0\), the polynomial

\[
\boxed{
\begin{aligned}
\Psi_m(t)={}&(h_{m+1}-h_{m+2})t^m\\
&+(h_{m+2}-h_m)t^{m+1}\\
&+(h_m-h_{m+1})t^{m+2}
\end{aligned}}
\tag{18}
\]

satisfies

\[
\mathcal W[\Psi_m]=0.
\tag{19}
\]

Therefore infinitely many kernels of arbitrarily large degree represent exactly the same error:

\[
R_n^{\rm aff}(t)+\sum_m c_{n,m}\Psi_m(t).
\tag{20}
\]

To single out a useful \(R_n\), one must impose additional structure, for example:

- a proper-hypergeometric coefficient formula;
- a Rodrigues formula;
- multiple orthogonality;
- prescribed zeros or positivity;
- fixed boundary behavior;
- or explicit rational telescoping certificates.

The Poincaré roots do not provide those finite-\(n\) data.

# 5. The strongest concrete non-circular candidate: multiple orthogonality

The product-distribution identity

\[
\int_0^1\!\int_0^1 f(xy)\,dx\,dy
=\int_0^1(-\log t)f(t)\,dt
\tag{21}
\]

reduces the desired double integral to

\[
\boxed{
\mathcal W[R]
=\int_0^1
\frac{R(t)}{1-t}
\left(-\log t+\frac12\log^2t\right)dt.
}
\tag{22}
\]

Introduce the two measures

\[
d\mu_2(t)=(-\log t)dt,
\qquad
d\mu_3(t)=\frac12\log^2t\,dt.
\tag{23}
\]

Their moments are rational:

\[
\int_0^1t^k\,d\mu_2(t)=\frac1{(k+1)^2},
\qquad
\int_0^1t^k\,d\mu_3(t)=\frac1{(k+1)^3}.
\tag{24}
\]

Their Markov functions are

\[
F_j(z)=\int_0^1\frac{d\mu_j(t)}{z-t},
\]

and

\[
F_2(1)=\zeta(2),
\qquad F_3(1)=\zeta(3).
\tag{25}
\]

For any polynomial \(Q(t)\), define

\[
P_{j,Q}(z)
=\int_0^1\frac{Q(z)-Q(t)}{z-t}\,d\mu_j(t).
\tag{26}
\]

This is a polynomial in \(z\), and the exact Hermite--Padé remainder identity is

\[
\boxed{
Q(z)F_j(z)-P_{j,Q}(z)
=\int_0^1\frac{Q(t)}{z-t}\,d\mu_j(t).
}
\tag{27}
\]

At \(z=1\), summing \(j=2,3\) gives

\[
\boxed{
\begin{aligned}
&Q(1)\bigl(\zeta(2)+\zeta(3)\bigr)
-P_{2,Q}(1)-P_{3,Q}(1)\\
&\qquad=\int_0^1\frac{Q(t)}{1-t}
\left(-\log t+\frac12\log^2t\right)dt\\
&\qquad=\int_0^1\!\int_0^1
Q(xy)\frac{1-\frac12\log(xy)}{1-xy}\,dx\,dy.
\end{aligned}}
\tag{28}
\]

Consequently, a genuine solution of Problem 2.7 would follow from the following finite certificate.

> Find a step-line type-II multiple-orthogonal polynomial \(Q_n\) for the pair \((\mu_2,\mu_3)\), possibly after a rational change of variable and a hypergeometric scalar gauge, such that
> \[
> Q_n(1)=q_n,
> \qquad
> P_{2,Q_n}(1)+P_{3,Q_n}(1)=p_n.
> \]
> Then the desired non-circular kernel is
> \[
> \boxed{R_n(x,y)=Q_n(xy).}
> \]

## Explicit determinant candidate

For a multi-index \(\mathbf n=(n_2,n_3)\), let \(N=n_2+n_3\). The monic type-II polynomial is characterized by

\[
\int_0^1t^kQ_{\mathbf n}(t)d\mu_2(t)=0
\quad(0\le k<n_2),
\tag{29}
\]

and

\[
\int_0^1t^kQ_{\mathbf n}(t)d\mu_3(t)=0
\quad(0\le k<n_3).
\tag{30}
\]

Since all moments are given by (24), it has the fully explicit rational determinant

\[
\boxed{
Q_{\mathbf n}(t)=\frac1{\Delta_{\mathbf n}}
\det\begin{pmatrix}
\left(\frac1{(r+c+1)^2}\right)_{\substack{0\le r<n_2\\0\le c\le N}}\\[1mm]
\left(\frac1{(r+c+1)^3}\right)_{\substack{0\le r<n_3\\0\le c\le N}}\\[1mm]
1&t&t^2&\cdots&t^N
\end{pmatrix},
}
\tag{31}
\]

where \(\Delta_{\mathbf n}\) is the determinant obtained from the first \(N\) columns of the moment block. The natural step line is

\[
(0,0),(1,0),(1,1),(2,1),(2,2),\ldots.
\tag{32}
\]

A step-line system for two measures generically satisfies a **four-term recurrence**. This is the most compelling structural explanation for why a simultaneous approximation to \(1,\zeta(2),\zeta(3)\) should have order three.

What is not yet proved is that the recurrence of (31), after any required pullback and gauge, is the printed Problem 2.7 recurrence. That comparison should be done exactly from the first several recurrence coefficients and initial values. If it fails, the measures must be modified by a level-11 rational pullback or an algebraic modular weight.

# 6. What the genuine creative-telescoping certificate must look like

Let \(F_n(x,y;s)\) be an independently defined hyperexponential or proper-hypergeometric kernel. A proof requires rational functions \(U_n,V_n\) such that

\[
\boxed{
\begin{aligned}
&A_nF_{n+3}-B_nF_{n+2}+C_nF_{n+1}-D_nF_n\\
&\qquad=\partial_x\bigl(U_nF_n\bigr)
+\partial_y\bigl(V_nF_n\bigr).
\end{aligned}}
\tag{33}
\]

The identity should hold uniformly in the auxiliary parameter \(s\), with

\[
F_n(x,y;s)=\frac{(xy)^sP_n(x,y)}{1-xy}
\tag{34}
\]

or an equivalent Euler/Mellin--Barnes form. Boundary terms must vanish on \([0,1]^2\), and one must verify the three initial integrals. Differentiating at \(s=0\) then produces the logarithmic part automatically.

For the affine kernel, (33) holds with \(U_n=V_n=0\), but \(P_n\) already contains \(p_n,q_n\). For a proof, \(P_n\) must instead be supplied by an independent finite sum, determinant, Rodrigues operator, or modular period.

A practical search should proceed in this order:

1. compute the step-line polynomials (31) exactly;
2. derive their four-term recurrence by moment linear algebra;
3. compare it with the challenge operator under rational shifts, pullbacks, and scalar gauges;
4. if the level-11 pullback is needed, replace \(t\) by the modular coordinate and repeat the Hermite--Padé construction;
5. only after an operator match, run multivariate creative telescoping to obtain \(U_n,V_n\).

# 7. Why the level-11 spectral identity is not enough

Let

\[
F(\mu)=4\mu^3-220\mu^2+8\mu-1
\]

and

\[
C_{11}(t)=t^3-20t^2+56t-44.
\]

The exact identity

\[
\boxed{
16F\!\left(\frac{(t-2)^2}{4}\right)
=-C_{11}(t)C_{11}(4-t)
}
\tag{35}
\]

proves the quadratic relation between the asymptotic spectra. It is realized at the sequence level by a centered binomial transform followed by extraction of the even part.

It does **not** determine:

- lower-order recurrence coefficients;
- the extension class carrying \(\zeta(3)\);
- the numerator and denominator initial conditions;
- the connection matrix;
- or a polynomial kernel on \([0,1]^2\).

Indeed, the directly centered-binomial-transformed Cooper sequence has the correct Poincaré roots but fails the printed finite-\(n\) recurrence. Thus (35) is a necessary spectral fingerprint, not a creative-telescoping certificate.

The real root of \(F\) is

\[
\mu_0=54.96369509838273\ldots,
\]

and the complex pair has modulus

\[
|\mu_\pm|=\frac1{2\sqrt{\mu_0}}.
\]

For the actual recurrence roots \(c_j=\mu_j/64\),

\[
|c_\pm|=\frac1{128\sqrt{\mu_0}}
=0.001053785\ldots,
\tag{36}
\]

while the projective error base is

\[
\left|\frac{c_\pm}{c_0}\right|
=\frac1{2\mu_0^{3/2}}
=0.001227032\ldots.
\tag{37}
\]

A classical fixed positive Beukers phase

\[
\left(\frac{x(1-x)y(1-y)}{1-xy}\right)^n
\]

has the unique real saddle value

\[
\varphi^{-5}=0.0901699\ldots,
\]

and cannot produce the nonreal conjugate recessive pair merely by multiplication by a fixed polynomial. A successful level-11 kernel must therefore contain an \(n\)-dependent terminating hypergeometric factor, complex cycles, or a modular/Eichler normal-function extension.

# 8. Comparison with the known simultaneous \(\zeta(2),\zeta(3)\) construction

There is a useful precedent: Dauguet and Zudilin construct simultaneous rational approximations to \(\zeta(2)\) and \(\zeta(3)\) from two explicit very-well-poised hypergeometric/Mellin--Barnes kernels with a common denominator sequence. Whipple transformations identify the denominators, and Gosper--Zeilberger creative telescoping produces an order-three recurrence.

One of their specialized rational terms is

\[
R_n^{(2)}(t)=
\frac{(t+1)_{8n}(t+n+1)_{6n}(t+2n+1)_{8n}}
{(8n)!^2(t+9n+1)_{6n+1}},
\tag{38}
\]

and the weight-three companion is

\[
\widehat R_n^{(3)}(t)=
\frac{(2t+11n+2)_{5n}}{(5n)!}
\frac{(t+1)_{8n}}{(8n)!}
\frac{(7n)!}{(t+9n+1)_{7n+1}}
\frac{(6n)!}{(t+10n+1)_{6n+1}}.
\tag{39}
\]

Their sum yields a linear form in \(\zeta(2)+\zeta(3)\). This is exactly the kind of certificate sought here. However, no equality between their recurrence and the Problem 2.7 operator is known, and their asymptotic data do not match the challenge under a simple scalar gauge. These kernels are therefore a model for the missing construction, not the answer to Problem 2.7.

Reference: Simon Dauguet and Wadim Zudilin, [*On simultaneous Diophantine approximations to \(\zeta(2)\) and \(\zeta(3)\)*](https://arxiv.org/abs/1401.5322).

For the level-11 recurrence itself, see Shaun Cooper, [*Apéry-like sequences defined by four-term recurrence relations*](https://arxiv.org/abs/2302.00757).

# 9. Exact Sage check for the affine kernel and the MOP candidate

The following code uses exact rational arithmetic. It does not guess a certificate; it verifies (1)--(3) and constructs the determinant candidate (31).

```python
from sage.all import *

QQn = PolynomialRing(QQ, 'n')
n = QQn.gen()
t = PolynomialRing(QQ, 't').gen()

# Initial challenge solutions.
p = [
    QQ(-612218384750),
    QQ(-9525021973931919) / 18100,
    QQ(-29561828382772029) / 65380,
]
q = [
    QQ(-215040420000),
    QQ(-167282265043404) / 905,
    QQ(-964185327658080) / 6071,
]

R = [q[j] - p[j] * (1-t) / 2 for j in range(3)]
assert R[0] == 91068772375 - 306109192375*t

# W(t^m) = L - h[m], with h[m]=H_m^(2)+H_m^(3).
def h(m):
    return sum(QQ(1)/k^2 + QQ(1)/k^3 for k in range(1, m+1))

def mixed_moment(poly):
    # Return (coefficient of L, rational part).
    poly = poly.polynomial(QQ)
    coeff_L = poly(1)
    rat = -sum(poly[m] * h(m) for m in range(poly.degree()+1))
    return coeff_L, rat

for j in range(3):
    assert mixed_moment(R[j]) == (q[j], -p[j])

# Type-II multiple-orthogonal polynomial for multi-index (n2,n3).
def mop_Q(n2, n3):
    N = n2 + n3
    rows = []
    for r in range(n2):
        rows.append([QQ(1)/(r+c+1)^2 for c in range(N+1)])
    for r in range(n3):
        rows.append([QQ(1)/(r+c+1)^3 for c in range(N+1)])
    Delta = matrix(QQ, [row[:N] for row in rows]).det()
    if Delta == 0:
        raise ArithmeticError('singular moment determinant')
    M = matrix(PolynomialRing(QQ, 't'), rows + [[t^c for c in range(N+1)]])
    return (M.det()/Delta).monic()

step_line = [(0,0)]
for m in range(5):
    step_line += [(m+1,m), (m+1,m+1)]

for idx in step_line:
    Q = mop_Q(*idx)
    print(idx, Q, 'Q(1)=', Q(1), 'moment=', mixed_moment(Q))
```

The decisive comparison is to derive the four-term recurrence of the printed `step_line` polynomials and test whether a rational gauge/pullback sends it to the Problem 2.7 operator. A positive result would give the desired non-circular \(R_n\) through (28).

# Final conclusion

The exact answer presently supported by the data is

\[
\boxed{
R_n(x,y)=q_n-\frac{p_n}{2}(1-xy).
}
\]

It gives the required integral and the same recurrence with a pointwise zero telescoping certificate. It is not a Beukers proof because it is recurrence-defined.

The most concrete route to the genuinely desired degree-\(n\) kernel is the logarithmic two-measure Hermite--Padé system (23)--(31). If its step-line recurrence can be matched to the challenge operator—possibly after the level-11 quadratic pullback—then

\[
\boxed{R_n(x,y)=Q_n(xy)}
\]

will be the sought non-circular kernel, and equation (28) will give the error integral automatically. Until that finite operator-and-initial-data certificate is supplied, writing a more elaborate explicit degree-\(n\) polynomial would be an unsupported guess rather than a proof.
