# 2. The imaginary-norm term

From

```lean
hef : e * f = A * D * (A + D)
```

and

```lean
normImag A D = 2 * A * D * (A + D)
```

we obtain

\[
\mathrm{normImag}(A,D)=2ef.
\]

Consequently,

\[
2\,\mathrm{normImag}(A,D)^2
=2(2ef)^2
=8(ef)^2.
\]

A namespace-neutral Lean helper is:

```lean
private lemma normImag_eq_two_mul_ef
    {A D e f : ℤ}
    (hef : e * f = A * D * (A + D)) :
    normImag A D = 2 * (e * f) := by
  calc
    normImag A D = 2 * (A * D * (A + D)) := by
      simp only [normImag]
      ring
    _ = 2 * (e * f) := by
      rw [hef]
```

In the final line `rw [hef]` rewrites the occurrence of `e * f` on the right-hand side. Equivalently, one may write the same step using `rw [← hef]` on the left after arranging the products.

The requested squared identity is then literally:

```lean
private lemma eight_mul_ef_sq_eq_two_mul_normImag_sq
    {A D e f : ℤ}
    (hef : e * f = A * D * (A + D)) :
    8 * (e * f) ^ 2 = 2 * normImag A D ^ 2 := by
  have himag : normImag A D = 2 * (e * f) :=
    normImag_eq_two_mul_ef hef
  rw [himag]
  ring
```

So the coefficient `8` is exactly right.

# 3. Exact Lean proof of `C ^ 2 = F18Positive A D`

The following uses precisely the existing theorem

```lean
F18Positive_norm (A D) :
  F18Positive A D = normReal A D ^ 2 + 2 * normImag A D ^ 2
```

and no nonlinear arithmetic tactic. Everything substantive is `ring`.

## 3.1 Form (a)

```lean
lemma C_sq_eq_F18Positive_of_formA
    {A D C e f : ℤ}
    (hef : e * f = A * D * (A + D))
    (hreal : normReal A D = e ^ 2 - 2 * f ^ 2)
    (hCabs : |C| = e ^ 2 + 2 * f ^ 2) :
    C ^ 2 = F18Positive A D := by
  have himag : normImag A D = 2 * (e * f) :=
    normImag_eq_two_mul_ef hef
  calc
    C ^ 2 = |C| ^ 2 := (sq_abs C).symm
    _ = (e ^ 2 + 2 * f ^ 2) ^ 2 :=
      congrArg (fun z : ℤ => z ^ 2) hCabs
    _ = (e ^ 2 - 2 * f ^ 2) ^ 2 + 8 * (e * f) ^ 2 := by
      ring
    _ = normReal A D ^ 2 + 2 * normImag A D ^ 2 := by
      rw [hreal, himag]
      ring
    _ = F18Positive A D :=
      (F18Positive_norm A D).symm
```

# 5. Exact field-algebra lemma producing the rational point

It is useful to isolate the field algebra from all integer coercions. The following lemma should compile against the stated signature of `F18Hom_clear_denominators`:

```lean
lemma F18_quotient_point_equation
    {A B C : ℚ}
    (hB : B ≠ 0)
    (hC : C ^ 2 = F18Hom A B) :
    (C / B ^ 3) ^ 2 = hyperellipticF18 (A / B) := by
  have hclear :
      B ^ 6 * hyperellipticF18 (A / B) = F18Hom A B :=
    F18Hom_clear_denominators (A := A) (B := B) hB
  calc
    (C / B ^ 3) ^ 2 = C ^ 2 / B ^ 6 := by
      field_simp [hB] <;> ring
    _ = F18Hom A B / B ^ 6 := by
      rw [hC]
    _ = hyperellipticF18 (A / B) := by
      exact (div_eq_iff (pow_ne_zero 6 hB)).2 <| by
        calc
          F18Hom A B = B ^ 6 * hyperellipticF18 (A / B) :=
            hclear.symm
          _ = hyperellipticF18 (A / B) * B ^ 6 := by
            ring
```

Now set \(B=A+D\). Assuming

```lean
hAD : A + D ≠ 0
```

we have:

```lean
have hADq : (((A + D : ℤ) : ℚ)) ≠ 0 := by
  exact_mod_cast hAD

have hcurve :
    ((C : ℚ) / (((A + D : ℤ) : ℚ) ^ 3)) ^ 2 =
      hyperellipticF18
        ((A : ℚ) / ((A + D : ℤ) : ℚ)) := by
  exact F18_quotient_point_equation hADq hChomQ
```

This is precisely

```lean

```

for

```lean

```

No square-root choice is involved: `C` itself is the numerator of the rational \(Y\)-coordinate.

# 6. Proving `U ≠ 0` and `U ≠ 1`

These conclusions require the descent data to provide, or imply,

```lean
hA  : A ≠ 0
hD  : D ≠ 0
hAD : A + D ≠ 0
```

They do **not** follow from `hef` and the two quadratic-form identities alone. In the intended five-descent application they normally follow from the primitive/nondegenerate or positivity hypotheses.

## 6.1 `U ≠ 0`

```lean
lemma fiveDescent_U_ne_zero
    {A D : ℤ}
    (hA : A ≠ 0)
    (hAD : A + D ≠ 0) :
    (A : ℚ) / ((A + D : ℤ) : ℚ) ≠ 0 := by
  apply div_ne_zero
  · exact_mod_cast hA
  · exact_mod_cast hAD
```

## 6.2 `U ≠ 1`

A proof avoiding dependence on the exact name of a `div_eq_one` lemma is:

```lean
lemma fiveDescent_U_ne_one
    {A D : ℤ}
    (hD : D ≠ 0)
    (hAD : A + D ≠ 0) :
    (A : ℚ) / ((A + D : ℤ) : ℚ) ≠ 1 := by
  have hADq : (((A + D : ℤ) : ℚ)) ≠ 0 := by
    exact_mod_cast hAD
  have hsplit :
      (1 : ℚ) - (A : ℚ) / ((A + D : ℤ) : ℚ) =
        (D : ℚ) / ((A + D : ℤ) : ℚ) := by
    field_simp [hADq] <;> push_cast <;> ring
  intro hU
  have hzero :
      (D : ℚ) / ((A + D : ℤ) : ℚ) = 0 := by
    rw [← hsplit, hU]
    ring
  have hDq : (D : ℚ) ≠ 0 := by
    exact_mod_cast hD
  exact (div_ne_zero hDq hADq) hzero
```

# 7. Why this is noncuspidal

For the alternating-sign model

\[
y^2=U^6-4U^5+10U^4-10U^3+5U^2-2U+1,
\]

the six rational cusps are the four affine points above

\[
U=0,\qquad U=1,
\]

and the two points at infinity. The constructed point is affine because \(A+D\ne0\). Therefore:

- `U ≠ 0` excludes both affine cusps above `0`;
- `U ≠ 1` excludes both affine cusps above `1`;
- being constructed in the affine chart excludes the two points at infinity.

Thus the point is noncuspidal.

The final bridge can be packaged in the shape:

```lean
lemma fiveDescent_nonCusp_F18
    {A D C e f : ℤ}
    (hef : e * f = A * D * (A + D))
    (hforms :
      (normReal A D = e ^ 2 - 2 * f ^ 2 ∧
        |C| = e ^ 2 + 2 * f ^ 2) ∨
      (normReal A D = 2 * e ^ 2 - f ^ 2 ∧
        |C| = 2 * e ^ 2 + f ^ 2))
    (hA : A ≠ 0)
    (hD : D ≠ 0)
    (hAD : A + D ≠ 0) :
    ∃ U Y : ℚ,
      Y ^ 2 = hyperellipticF18 U ∧ U ≠ 0 ∧ U ≠ 1 := by
  have hCpos : C ^ 2 = F18Positive A D :=
    C_sq_eq_F18Positive_of_fiveDescent_forms hef hforms
  have hChomZ : C ^ 2 = F18Hom A (A + D) :=
    hCpos.trans (F18Positive_eq_F18Hom_add A D)
  have hChomQ :
      (C : ℚ) ^ 2 =
        F18Hom (A : ℚ) ((A + D : ℤ) : ℚ) := by
    calc
      (C : ℚ) ^ 2 = ((C ^ 2 : ℤ) : ℚ) := by norm_num
      _ = ((F18Hom A (A + D) : ℤ) : ℚ) := by
        exact congrArg (fun z : ℤ => (z : ℚ)) hChomZ
      _ = F18Hom (A : ℚ) ((A + D : ℤ) : ℚ) := by
        simpa using F18Hom_intCast A (A + D)
  let U : ℚ := (A : ℚ) / ((A + D : ℤ) : ℚ)
  let Y : ℚ := (C : ℚ) / ((A + D : ℤ) : ℚ) ^ 3
  refine ⟨U, Y, ?_, ?_, ?_⟩
  · dsimp [U, Y]
    apply F18_quotient_point_equation
    · exact_mod_cast hAD
    · exact hChomQ
  · dsimp [U]
    exact fiveDescent_U_ne_zero hA hAD
  · dsimp [U]
    exact fiveDescent_U_ne_one hD hAD
```

If the current rational-point type is a structure rather than a pair of rationals, replace the final existential by its affine constructor; the equation proof is unchanged.

# Bottom line

All three requested algebra checks are correct:

```text
form (a):  (e²+2f²)² = (e²−2f²)² + 8(ef)²;
form (b):  (2e²+f²)² = (2e²−f²)² + 8(ef)²;
normImag = 2ef  ⇒  2 normImag² = 8(ef)².
```

Therefore

\[
C^2=F18Positive(A,D).
\]

With \(A+D\ne0\), the correct point is

\[
\boxed{
(U,Y)=\left(\frac{A}{A+D},\frac{C}{(A+D)^3}\right).

With \(A\ne0\) and \(D\ne0\), it has \(U\ne0,1\), hence is noncuspidal. The handoff’s \(D/A\) must either be replaced by \(A/(A+D)\), or corrected to the alternative chart \(-D/A\).
