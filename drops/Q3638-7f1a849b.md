```lean
/-- Initial spatial Holder modulus on the interval-domain subtype. -/
def InitialDatumHolder
    (u₀ : intervalDomainPoint → ℝ) (θ H₀ : ℝ) : Prop :=
  ∀ x y : intervalDomainPoint,
    |u₀ x - u₀ y| ≤ H₀ * |x.1 - y.1| ^ θ
```

```lean
import ShenWork.Paper2.ChemMildHolderBootstrap

open MeasureTheory Filter
open ShenWork.IntervalDomain (intervalDomainPoint intervalDomainLift intervalMeasure)
open ShenWork.IntervalNeumannFullKernel (intervalFullSemigroupOperator)

namespace ShenWork.Paper2

noncomputable section

/-- A coupling-level certificate that the Neumann heat kernels based at `x` and
`y` can be coupled by pairs in `[0,1]²` whose distance is at most `|x-y|`, and
whose integral difference realizes the semigroup difference for the selected
observable `f`.

This is intentionally an interface: it avoids encoding the period-two fold map
inside the Holder bootstrap file. -/
structure NeumannHeatContractiveCouplingFor
    (t : ℝ) (x y : intervalDomainPoint) (f : ℝ → ℝ) : Prop where
  μ : Measure (ℝ × ℝ)
  prob : IsProbabilityMeasure μ
  support : ∀ᵐ z ∂μ, z.1 ∈ Set.Icc (0 : ℝ) 1 ∧ z.2 ∈ Set.Icc (0 : ℝ) 1
  dist_le : ∀ᵐ z ∂μ, |z.1 - z.2| ≤ |x.1 - y.1|
  diff_integrable : Integrable (fun z : ℝ × ℝ => f z.1 - f z.2) μ
  semigroup_diff_eq :
    intervalFullSemigroupOperator t f x.1 -
      intervalFullSemigroupOperator t f y.1 =
        ∫ z : ℝ × ℝ, f z.1 - f z.2 ∂μ
```

Then prove:

```lean
/-- Holder preservation from a contractive Neumann heat coupling.  This is the
clean consumer theorem.  The remaining hard work is to produce the coupling for
`f = intervalDomainLift u₀` from the full Neumann kernel. -/
theorem InitialLegUniformHolderAtZero_of_contracting_couplings
    {u₀ : intervalDomainPoint → ℝ} {T θ H₀ : ℝ}
    (hθ0 : 0 < θ) (hH₀ : 0 ≤ H₀)
    (hholder : InitialDatumHolder u₀ θ H₀)
    (hplan : ∀ t, 0 < t → t ≤ T → ∀ x y : intervalDomainPoint,
      NeumannHeatContractiveCouplingFor t x y (intervalDomainLift u₀)) :
    InitialLegUniformHolderAtZero u₀ T θ H₀ := by
  intro t ht htT x y
  rcases hplan t ht htT x y with ⟨μ, hprob, hsupp, hdist, hint, hdiff⟩
  rw [hdiff]
  have hpoint :
      ∀ᵐ z : ℝ × ℝ ∂μ,
        |intervalDomainLift u₀ z.1 - intervalDomainLift u₀ z.2| ≤
          H₀ * |x.1 - y.1| ^ θ := by
    filter_upwards [hsupp, hdist] with z hz hdistz
    have hz1 : z.1 ∈ Set.Icc (0 : ℝ) 1 := hz.1
    have hz2 : z.2 ∈ Set.Icc (0 : ℝ) 1 := hz.2
    have hH := hholder ⟨z.1, hz1⟩ ⟨z.2, hz2⟩
    have hpow : |z.1 - z.2| ^ θ ≤ |x.1 - y.1| ^ θ :=
      Real.rpow_le_rpow (abs_nonneg _) hdistz hθ0.le
    calc
      |intervalDomainLift u₀ z.1 - intervalDomainLift u₀ z.2|
          = |u₀ ⟨z.1, hz1⟩ - u₀ ⟨z.2, hz2⟩| := by
              simp [intervalDomainLift, hz1, hz2]
      _ ≤ H₀ * |z.1 - z.2| ^ θ := hH
      _ ≤ H₀ * |x.1 - y.1| ^ θ :=
          mul_le_mul_of_nonneg_left hpow hH₀
  calc
    |∫ z : ℝ × ℝ,
        intervalDomainLift u₀ z.1 - intervalDomainLift u₀ z.2 ∂μ|
        ≤ ∫ z : ℝ × ℝ,
            |intervalDomainLift u₀ z.1 - intervalDomainLift u₀ z.2| ∂μ := by
          simpa [Real.norm_eq_abs] using norm_integral_le_integral_norm
            (fun z : ℝ × ℝ => intervalDomainLift u₀ z.1 - intervalDomainLift u₀ z.2)
    _ ≤ ∫ _z : ℝ × ℝ, H₀ * |x.1 - y.1| ^ θ ∂μ := by
          exact integral_mono_ae hint.abs
            (integrable_const (H₀ * |x.1 - y.1| ^ θ)) hpoint
    _ = H₀ * |x.1 - y.1| ^ θ := by
          simp

end

end ShenWork.Paper2
```

Lean pitfalls in this proof:

- The last `simp` may need `[hprob.measure_univ]`, depending on what `IsProbabilityMeasure` exposes in this repo/mathlib. If it does not close, replace the last step by rewriting `MeasureTheory.integral_const` and `measure_univ` explicitly.
- `integral_mono_ae` may need the constant integrability argument written as `by infer_instance` if the instance search does not find the finite-measure instance from `IsProbabilityMeasure μ`.
- The `simp [intervalDomainLift, hz1, hz2]` step is the key reason the coupling support field is included.

## How to prove the hard coupling theorem later

Mathematically, use the synchronous reflected-Brownian coupling:

```text
S(t)u₀(x) = E[ u₀(fold01(x + sqrt(2) B_t)) ]
S(t)u₀(y) = E[ u₀(fold01(y + sqrt(2) B_t)) ]
```

and prove

```text
|fold01(x + z) - fold01(y + z)| ≤ |x - y|.
```

Then Holder follows under the same Gaussian variable.

In Lean, do not start by defining `fold01` with raw `Int.floor` and proving all periodic/reflection properties by case splits.  That is likely too hard and brittle.  The current repo already has method-of-images kernel algebra, tiling identities, and `AddCircle`/even-reflection Fourier scaffolding; use those to hide the fold complexity behind a single transport theorem.

A better eventual route is one of:

1. Define a period-two reflected projection using an `AddCircle 2` quotient metric and prove the synchronous contraction there, then identify its pushforward density with `intervalNeumannFullKernel`.
2. Prove the transport plan directly from the method-of-images kernel as a measure theorem, without exposing `fold01` in the Holder bootstrap layer.

## Why the tempting existing routes are not enough

- `InitialLegUniformHolderAtZero_of_semigroup_deriv_bound` is a C1/MVT route and requires a uniform derivative bound. It does not consume only Holder data.
- `neumannHeat_Linf_to_Ctheta` gives a Holder estimate with `t^(-theta/2)` blow-up, so it cannot prove the zero-time uniform initial-leg bound.
- `intervalFullSemigroup_tendstoUniformlyOn` proves small-time uniform convergence from continuity, but this does not give a bound proportional to `|x-y|^theta`.
- Kernel positivity and mass one alone give an `L∞` contraction, not a Holder-seminorm contraction.
- The spectral clean theorem gives a cosine-series representation for continuous data, but there is no current theorem that the cosine heat semigroup contracts the Holder seminorm.
