The challenge to the earlier route assessment is valid in one important sense:

* `capWeighted_genuineFluxDifference_spatialDifferenceQuotient_l2_bounded` is a genuine, noncircular producer.

> The simpler two-profile route is valid only at a fixed cap radius, or potentially at a lighter weight `eta < kappa c`. It cannot produce the exact-weight, cap-uniform H¹ estimate required by the corrected headline. For `p.χ ≠ 0`, a matched cancellation such as the four-profile flux estimate remains necessary; for `p.χ = 0`, the flux disappears, but the parallel reaction estimate still needs matched cancellation.

This conclusion is about the wrong exponent window, not a preference for sharper constants.

## 2. Conditional scalar closure: if the false wave bound were available, the `41dc6352` route would work

This section gives the exact constants, to separate the valid Volterra algebra from the invalid wave input.

Set

```lean
Cχ := capWeightedChemotaxisOperatorSquareConstant p M eta
L  := 1 + reactionLip p.α M
K0 := capMildKernelConstant p M eta c T
K1 := capMildKernelInvSqrtConstant p M eta c T
```

The last two are defined in
`ShenWork/Paper1/WholeLineWeightedRegularityMild.lean:682` and `:688`:

```lean
K0 = 2 * capMildGrowthBound eta c T * L +
     2 * capMildGrowthBound eta c T * eta * Real.sqrt Cχ

K1 = 2 * capMildGrowthBound eta c T *
       (2 / Real.sqrt (4 * Real.pi)) * Real.sqrt Cχ
```

For one history time, write

```text
x(s)^2 = ∫ capWeight eta R · |D_h(u-U)|²,
F_U^2  = ∫ capWeight eta R · |D_h U|².
```

Multiplying the flux theorem by `-p.χ` and taking square roots gives

```text
‖D_h[-χ(Flux(u)-Flux(U))]‖₂
  ≤ 2 sqrt(Cχ) x(s) + sqrt(6) sqrt(Cχ) F_U.
```

The reaction theorem gives

```text
‖D_h[ShiftedReaction(u)-ShiftedReaction(U)]‖₂
  ≤ 2 L x(s) + sqrt(6) L F_U.
```

The cap heat-gradient Schur estimate and the value Schur estimate then yield the exact common coefficients

```lean
def capDqKernelConstant
    (p : CMParams) (M eta c T : ℝ) : ℝ :=
  2 * capMildKernelConstant p M eta c T

def capDqKernelInvSqrtConstant
    (p : CMParams) (M eta c T : ℝ) : ℝ :=
  2 * capMildKernelInvSqrtConstant p M eta c T

def capDqWaveForcingFactor : ℝ := Real.sqrt 6 / 2
```

Indeed, the self-history coefficients are

```text
C0 = 2 K0,
C1 = 2 K1,
```

while both wave coefficients are the same fixed ratio

```text
sqrt(6)/2
```

times the corresponding self coefficient. Thus the delayed inequality has the form

```text
x(r) ≤ A0 r^(-1/2) + A1
     + (sqrt(6)/2 * F_U) * (C0*r + 2*C1*sqrt(r))
     + ∫₀ʳ [C0 + C1*(r-s)^(-1/2)] x(s) ds.
```

This is exactly the input shape of
`volterra_const_invSqrt_profile_bound` in
`ShenWork/Paper1/WholeLineWeightedRegularityH1Volterra.lean:183`.
The short-time condition is

```text
2*C0*h + pi*C1*sqrt(h) < 1,
```

i.e.

```text
4*K0*h + 2*pi*K1*sqrt(h) < 1.
```

There is no difficulty choosing a smaller restart window. Therefore the answer to the conditional question is:

> **Yes.** If `F_U` were finite uniformly in `R` and uniformly for the nonzero quotient steps used in the difference-quotient limit, `41dc6352` plus the reaction analogue would close H¹ with the constants above. No `hWx2` or flux-derivative premise would be needed.

The hidden failure is solely that this `F_U` is false in the corrected `eta` window.

For each fixed `h ≠ 0`, the committed wave-tail theorem implies

```text
sup_R ∫ capWeight eta R · |D_h U|² = +∞
```

when `eta > kappa c`. Hence the numerator of the Henry estimate grows with `R`, and the cap exhaustion cannot produce exact exponential `H¹` regularity.

Taking `h → 0` does not help. The tail coefficient

```text
(exp (-(kappa c)*h) - 1) / h
```

tends to `-kappa c`, so it stays bounded away from zero for all sufficiently small nonzero `h`.

## 4. Why the matched four-profile route avoids the obstruction

The matched route never estimates `D_h U` as a standalone exact-weight `L²` function.

For the power term, `ShenWork/Paper1/WholeLineWeightedRegularityFourProfilePower.lean:35` proves
`paper5_four_profile_secant_relative_increment_bound`, and line 202 proves
`paper5_four_profile_power_quotient_bound`. The crucial input is the relative wave increment

```text
|D_h U| ≤ B · (positive reference segment),
```

not `D_h U ∈ L²_eta`.

For `1 < m < 2`, the reference factor cancels the singular power secant, leaving a bound linear in

```text
|u(x)-U(x)| + |u(x+h)-U(x+h)|
```

and in `D_h(u-U)`. Those are the perturbation fields carrying the exact weight. No standalone wave energy remains.

The reaction analogue is `reaction_matched_fourPoint_quotient_abs_le` in
`ShenWork/Paper1/WholeLineWeightedRegularityFourProfileReaction.lean:180`.

This is the substantive difference between the routes:

* the `41dc6352` estimate takes a triangle inequality **after separating the wave quotient**, so it creates an isolated nonintegrable term;
* the matched estimate performs the four-profile algebra **before** taking absolute values and `L²` norms, preserving cancellation against the perturbation.

Recovering that cancellation later from an inhomogeneous forcing estimate is impossible without undoing the triangle inequality—in effect, reconstructing a matched argument.

2. **`p.χ = 0`:** the chemotactic flux term vanishes, so no four-profile *flux* estimate is needed. However, the parallel `4/6` reaction theorem still introduces `D_h U`; the matched reaction estimate is still needed for the same exponent reason.

3. **A lighter auxiliary weight `eta < kappa c`:** the isolated wave quotient is no longer ruled out by the tail exponent and a two-profile route may be viable. That is outside the corrected headline window, and current main contains no full theorem supplying the needed uniform wave `L²` bound there.

4. **Fixed `R`:** the two-profile route is valid and useful. It proves cap-local H¹ estimates whose constants depend on `R`. Such estimates cannot be exhausted to the full exponential weight.

5. **Circularity:** `41dc6352`, the wave-tail obstruction, the cap exhaustion theorem, and the scalar Henry closure are all axiom-clean and contain `#print axioms` audits. The failure is analytic, not logical or packaging-related.

Therefore the earlier claim should be refined as follows:

> A matched four-profile estimate is not algebraically necessary to obtain a fixed-cap quotient estimate. It **is** necessary, or must be replaced by an equivalent pre-norm cancellation, to obtain the cap-radius-uniform exact-weight H¹ estimate in the corrected Theorem 1.2/1.3 exponent window.
