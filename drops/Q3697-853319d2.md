It only contains `hrecon`, a cosine-series reconstruction of `u₀p`, and a separate floor for `W.u₀`.  Therefore the honest Lean obstruction for the monolithic record is either:

### B. Floor-robust fixed point

This is the only faithful path to an all-PPID strict-chi-negative local-existence headline.

The replacement analytic estimate is not another Wiener/floor lemma.  It must avoid the complex/Wiener negative-power constants entirely.  Work in an order interval or real-valued `L∞`/Hölder space and use real-power Lipschitz estimates on `[0,M]`, not on a positive floor interval `[fm,M]`.

The concrete estimate that should replace the current floor/Wiener route is:

```text
For q ≥ 1 and 0 ≤ r,s ≤ M,
  |r^q - s^q| ≤ C_q(M) |r-s|.
```

Then combine it with heat/resolver smoothing estimates of the schematic form:

```text
If 0 ≤ u,w ≤ M on [0,1], then
  ||u^γ - w^γ||∞ ≤ Cγ(M) ||u-w||∞,
  ||R(u^γ) - R(w^γ)||C¹ ≤ C_res,γ(M) ||u-w||∞,
  ||u R(u^γ)_x - w R(w^γ)_x||∞ ≤ C_flux(M) ||u-w||∞,
```

```text
∫₀ᵀ ||∂x S(T-s) F(s)||∞ ds ≤ C sqrt(T) sup_s ||F(s)||∞
∫₀ᵀ ||S(T-s) G(s)||∞ ds ≤ T sup_s ||G(s)||∞.
```

The self-map should preserve `0 ≤ u ≤ 2M` by order/maximum-principle arguments, not by staying inside a ball around a heat-evolved datum with radius `floor/2`.  This is the key conceptual change: the lifespan depends on `M`, not on a datum floor or Wiener norm.
