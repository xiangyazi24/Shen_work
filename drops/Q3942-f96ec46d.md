# Q3942 — positive-time gradient bound for the truncated/conjugate Picard limit

## Main correction

Your second computation is right: a **constant** coefficient bound for the source is not enough for a gradient bound.

If

```text
|source_k(s)| ≤ C
```

then the Duhamel gain gives

```text
|duh_k(t)| ≤ C / λ_k,
```

and hence

```text
kπ |duh_k(t)| ≤ C / (kπ),
```

which is harmonic and diverges.  So no proof of

```lean
Summable (fun k => |c_k(t)| * ((k : ℝ) * Real.pi))
```

can be obtained from `DuhamelSourceBddOn.M` alone.

The key point is also slightly different from what you propose:

> For the **gradient bound**, you do **not** need a source envelope in `ℓ¹`.  You only need a positive-window source envelope in `ℓ²`.

Indeed, if on a positive window `[t/2,t]` we have

```lean
|source_k(s)| ≤ E k,     Summable (fun k => (E k)^2),
```

then

```text
kπ |duh_k| ≤ E_k / (kπ),
```

and

```text
Σ E_k / k < ∞
```

by Cauchy-Schwarz since `E ∈ ℓ²` and `(1/k) ∈ ℓ²` for `k ≥ 1`.

So the right circularity-breaker is:

```text
bounded u
  → flux Q(u) ∈ L²
  → chemotaxis Duhamel gives u(t) ∈ H¹ for t>0
  → u ∈ H¹ on a positive window
  → flux Q(u) ∈ H¹ on that window
  → chemDiv source coefficients are ℓ² on that window
  → Duhamel tail has gradient-weighted ℓ¹
```

This is weaker and cheaper than building a full summable source envelope.  If later you need **eigenvalue-weighted** coefficient summability `Σ λ_k |c_k|`, then `ℓ²` source is not enough; for that stronger goal you need one more Sobolev/coefficient ladder step to get the source into `ℓ¹`.  But for the gradient bound itself, `ℓ²` is exactly the right target.

## Correct positive-time split

For fixed `0 < t ≤ T`, set

```lean
τ := t / 2
L := t - τ = t / 2
U := truncatedConjugatePicardLimit p u₀ DT.T
```

Use the restart formula at time `τ`:

```text
c_k(t)
  = exp(-L λ_k) * cosineCoeffs(U(τ), k)
    + duhamelSpectralCoeff (fun σ k => sourceCoeff(τ + σ,k)) L k.
```

Then:

1. **Homogeneous restart part.**  Since `U(τ)` is bounded, its cosine coefficients are bounded.  The factor `exp(-L λ_k)` gives
   ```text
   Σ kπ exp(-L λ_k) |cosineCoeffs(U(τ),k)| < ∞.
   ```

2. **Duhamel tail.**  Prove the shifted source has an `ℓ²` envelope on `[0,L]`:
   ```lean
   |sourceCoeff (τ + σ) k| ≤ E k,
   Summable (fun k => (E k)^2).
   ```
   Then use the `λ`-gain per mode and Cauchy-Schwarz:
   ```text
   kπ |duh_k| ≤ E_k / (kπ),     Σ E_k/k < ∞.
   ```

This avoids the singular behavior at `s=0` completely; it only asks for source regularity on the positive window `[τ,t]`.

---

## Why this is non-circular

The source `ℓ²` envelope on `[τ,t]` does not require the final gradient bound.  It requires `u ∈ H¹` on `[τ,t]`, and that can be proved from the mild equation using only the ball bound.

For the chemotaxis term, write the source as a divergence of a flux:

```text
source_chem,k(s) = kπ * sineInner(Q(s), k),
Q(s) = truncatedChemFluxLifted p (U s).
```

If `Q(s) ∈ L²`, then its sine coefficients are `ℓ²`.  The Duhamel coefficient of the divergence term is essentially

```text
duh_k ~ ∫ e^{-λ(t-s)} k q_k(s) ds,
```

so in `H¹`:

```text
λ_k |duh_k|² ~ k² |duh_k|² ≤ |q_k|².
```

Thus bounded/L² flux gives an `H¹` bound for `u` at positive times.  In 1D, `H¹` is an algebra and compositions with Lipschitz/smooth bounded functions preserve `H¹`; the elliptic resolver improves regularity, so

```text
u ∈ H¹  ⇒  Q(u) ∈ H¹  ⇒  kπ * sineInner(Q(u),k) ∈ ℓ².
```

That is enough for gradient-weighted `ℓ¹` of the next Duhamel tail.

This is the same spirit as the repo’s `TrajectoryHSigmaEnvelope` / `trajLadder_step` infrastructure: start from an `ℓ²`/`H⁰` flux envelope, gain one derivative through the parabolic divergence Duhamel, then use algebra/composition to improve the source.

## What about eigenvalue-weighted summability?

Do not conflate the two goals:

* **Gradient bound** needs
  ```lean
  Summable (fun k => |c_k| * k)
  ```
  and is obtained from a positive-window `ℓ²` source envelope.

* **Eigenvalue-weighted summability** needs
  ```lean
  Summable (fun k => λ_k * |c_k|)
  ```
  and requires a positive-window `ℓ¹` source envelope, because the Duhamel gain only gives
  ```text
  λ_k |duh_k| ≤ sourceEnvelope_k.
  ```

For the stronger eigenvalue-weighted theorem, continue the same Sobolev ladder one more step until the source coefficient envelope is `ℓ¹`.  In coefficient terms, it is enough to get source coefficients in `H^r` with `r > 1/2`, since Cauchy-Schwarz gives `ℓ¹`.  This is where the existing `TrajectoryHSigmaEnvelope` / pass-4 ladder belongs.

So the circularity is broken in two stages:

```text
bounded ball
  → H¹ positive-time solution
  → ℓ² source envelope
  → gradient bound
  → higher source regularity / HSigma ladder
  → ℓ¹ source envelope
  → eigenvalue-weighted summability
```

Trying to jump directly from a constant source bound to either gradient-l1 or eigenvalue-weighted-l1 is impossible.

---

## Final verdict

There is no direct gradient proof from `DuhamelSourceBddOn.M` alone.  The correct non-circular proof is to introduce a **positive-window `ℓ²` source envelope**.

The minimal theorem chain is:

```text
truncatedLimit_H1CoeffEnvelope_positiveWindow
  → truncatedSource_l2Envelope_of_H1CoeffEnvelope
  → summable_gradient_duhamelSpectralCoeff_of_l2Envelope
  → truncatedPicardCoeff_gradient_l1_positive_time
  → truncatedPicardLimit_gradient_bound_positive_time
```

For later `Σ λ_k |c_k|`, add one more HSigma/pass-4 ladder step to upgrade the positive-window source envelope from `ℓ²` to `ℓ¹`.
