# Problem 2.5: complete Sage setup for the matrix Almkvist–Zeilberger search

## Two corrections before running the linear algebra

The normalization is

\[
\delta(n)=-2(n+2)^2(n+3)^2(2n+5)(2n+7)^2,
\qquad
M_H(n)=\frac{M(n)}{\delta(n)}.
\]

If

\[
\widehat P_{n+1}=\widehat P_nM_H(n),
\]

then the literal left side in the question is

\[
\frac{uv}{Q}\widehat P_{n+1}-\widehat P_nM_H(n)
=\widehat P_n\left(\frac{uv}{Q}-1\right)M_H(n),
\qquad Q=1-u^2v^2.
\]

This exposes the most useful formulation.  Write the fluxes as

\[
X_u=\widehat P_n\,uQ\,A(n,u,v,t),\qquad
X_v=\widehat P_n\,vQ\,B(n,u,v,t),\qquad
X_t=\widehat P_n(1+t^2)C(n,u,v,t).
\]

It is then sufficient to solve the **universal \(3\times3\) identity**

\[
\boxed{
\left(\frac{uv}{Q}-1\right)M_H(n)
=uQ A_u+vQ B_v+(1+t^2)C_t
+\bigl(n+1+(n-1)u^2v^2\bigr)(A+B),
}
\tag{1}
\]

with

\[
B(n,u,v,t)=A(n,v,u,t).
\]

Left multiplication by either challenge row then gives the desired \(2\times3\) identity.  This is preferable to fitting \(2\times3\) flux coefficients directly to the accumulated trajectory: the entries of \(\widehat P_n\) are exponential P-recursive sequences, not low-degree polynomials in \(n\).

Consequently the universal search has \(9\), not \(6\), matrix entries.  With all seven symmetric \(C\)-monomials it has

\[
9(10+7)(d_n+1)=153(d_n+1)
\]

scalar unknowns.  Four of the seven \(C\)-monomials are independent of \(t\), hence invisible to \(C_t\); they are pure bulk-gauge columns.  Removing them gives the effective count

\[
\boxed{9(10+3)(d_n+1)=117(d_n+1).}
\]

The code below supports both modes:

- `TARGET_MODE = "universal"` solves (1), which is the recommended computation;
- `TARGET_MODE = "trajectory"` literally implements the proposed \(2\times3\) fit and is useful as a diagnostic.

There is also a small differentiation correction.  From \(X_u=uQA\),

\[
(\partial_u+\ell_u)X_u
=uQ A_u+\bigl(n+1+(n-1)u^2v^2\bigr)A.
\]

It is **not** \(uQ\,\partial_u(uQA)\).  The program uses the displayed equation (1).

# 1. Explicit normalized matrix

For brevity put

\[
\begin{aligned}
p_{11}&=136n^4+1424n^3+5548n^2+9551n+6141,\\
p_{12}&=384n^6+6384n^5+44168n^4+162698n^3+336377n^2+369933n+169011,\\
p_{13}&=480n^4+4980n^3+19210n^2+32690n+20730,\\
p_{21}&=48n^3+386n^2+1017n+879,\\
p_{22}&=272n^5+3848n^4+21732n^3+61184n^2+85761n+47808,\\
p_{23}&=320n^3+2540n^2+6610n+5640,\\
p_{31}&=32n^4+302n^3+1037n^2+1530n+813,\\
p_{32}&=192n^6+2984n^5+19116n^4+64452n^3+120256n^2+117279n+46476,\\
p_{33}&=16n^5+408n^4+2912n^3+8884n^2+12254n+6240.
\end{aligned}
\]

After cancelling common factors, the normalized matrix is

\[
\boxed{
M_H(n)=
\begin{pmatrix}
\dfrac{p_{11}}{2(n+2)^2(2n+7)^2}
&-\dfrac{p_{12}}{2(n+2)^2(n+3)^2(2n+5)(2n+7)^2}
&\dfrac{p_{13}}{2(n+2)^2(n+3)^2(2n+5)(2n+7)^2}
\\[4mm]
-\dfrac{p_{21}}{(2n+7)^2}
&\dfrac{p_{22}}{2(n+3)^2(2n+5)(2n+7)^2}
&-\dfrac{p_{23}}{2(n+3)^2(2n+5)(2n+7)^2}
\\[4mm]
\dfrac{p_{31}}{(2n+7)^2}
&-\dfrac{p_{32}}{2(n+3)^2(2n+5)(2n+7)^2}
&\dfrac{p_{33}}{2(n+3)^2(2n+5)(2n+7)^2}
\end{pmatrix}.}
\tag{2}
\]

Its determinant is

\[
\det M_H(n)=\frac{(n+1)(2n+3)^2}{(n+3)(2n+7)^2}.
\]

The least common denominator of (2) is

\[
D_M(n)=2(n+2)^2(n+3)^2(2n+5)(2n+7)^2.
\]

# 2. Complete Sage program

Save the following as `az_matrix_search.sage` and run

```bash
sage az_matrix_search.sage
```

The program performs exact coefficient matching in \(u,v,t\); it does **not** substitute geometric points.  It solves nine independent modular systems in universal mode, reports consistency, rank and affine nullity, verifies the candidate at additional \(n\)-values, prints the modular certificate, and attempts one-prime rational reconstruction followed by an exact verification over \(\mathbb Q(n,u,v,t)\).

```python
#!/usr/bin/env sage
from sage.all import *

# ================================================================
# Configuration
# ================================================================

P_MOD = ZZ(2)**61 - 1          # Mersenne prime 2305843009213693951
TARGET_MODE = "universal"     # "universal" (recommended) or "trajectory"
DEGREES_TO_TRY = [0, 1, 2]
DENOMINATOR_MODES = ["one", "squarefree"]
INCLUDE_T_INDEPENDENT_C_GAUGES = False
EXTRA_HOLDOUT_VALUES = 8
STOP_AT_FIRST_CANDIDATE = True

# Challenge initial rows; H_0=1, so these are also normalized at n=0.
INITIAL_ROWS_ZZ = matrix(ZZ, [
    [30921, -32972, 8240],
    [33750, -36000, 9000],
])


# ================================================================
# Polynomial data and the exact normalized matrix
# ================================================================

def p_data(z):
    p11 = 136*z**4 + 1424*z**3 + 5548*z**2 + 9551*z + 6141
    p12 = (384*z**6 + 6384*z**5 + 44168*z**4 + 162698*z**3
           + 336377*z**2 + 369933*z + 169011)
    p13 = 480*z**4 + 4980*z**3 + 19210*z**2 + 32690*z + 20730
    p21 = 48*z**3 + 386*z**2 + 1017*z + 879
    p22 = (272*z**5 + 3848*z**4 + 21732*z**3 + 61184*z**2
           + 85761*z + 47808)
    p23 = 320*z**3 + 2540*z**2 + 6610*z + 5640
    p31 = 32*z**4 + 302*z**3 + 1037*z**2 + 1530*z + 813
    p32 = (192*z**6 + 2984*z**5 + 19116*z**4 + 64452*z**3
           + 120256*z**2 + 117279*z + 46476)
    p33 = 16*z**5 + 408*z**4 + 2912*z**3 + 8884*z**2 + 12254*z + 6240
    return p11, p12, p13, p21, p22, p23, p31, p32, p33


def raw_matrix(z, K):
    p11, p12, p13, p21, p22, p23, p31, p32, p33 = p_data(z)
    return matrix(K, [
        [-(2*z+5)*(z+3)**2*p11,
         p12,
         -p13],
        [2*(2*z+5)*(z+2)**2*(z+3)**2*p21,
         -(z+2)**2*p22,
         (z+2)**2*p23],
        [-2*(2*z+5)*(z+2)**2*(z+3)**2*p31,
         (z+2)**2*p32,
         -(z+2)**2*p33],
    ])


def normalized_matrix_reduced(z, K):
    p11, p12, p13, p21, p22, p23, p31, p32, p33 = p_data(z)
    return matrix(K, [
        [p11 / (2*(z+2)**2*(2*z+7)**2),
         -p12 / (2*(z+2)**2*(z+3)**2*(2*z+5)*(2*z+7)**2),
         p13 / (2*(z+2)**2*(z+3)**2*(2*z+5)*(2*z+7)**2)],
        [-p21 / ((2*z+7)**2),
         p22 / (2*(z+3)**2*(2*z+5)*(2*z+7)**2),
         -p23 / (2*(z+3)**2*(2*z+5)*(2*z+7)**2)],
        [p31 / ((2*z+7)**2),
         -p32 / (2*(z+3)**2*(2*z+5)*(2*z+7)**2),
         p33 / (2*(z+3)**2*(2*z+5)*(2*z+7)**2)],
    ])


def exact_matrix_audit():
    Rn = PolynomialRing(QQ, 'N')
    N = Rn.gen()
    Kn = Rn.fraction_field()
    z = Kn(N)
    delta = -2*(z+2)**2*(z+3)**2*(2*z+5)*(2*z+7)**2
    M0 = raw_matrix(z, Kn)
    M1 = normalized_matrix_reduced(z, Kn)
    M2 = M0 * (1/delta)
    assert all(x == 0 for x in (M1-M2).list())
    det_expected = (z+1)*(2*z+3)**2 / ((z+3)*(2*z+7)**2)
    assert M1.det() == det_expected
    print("Exact normalization audit: PASS")


# ================================================================
# Finite-field geometry and monomial bases
# ================================================================

F = GF(P_MOD)
R = PolynomialRing(F, names=('u', 'v', 't'), order='degrevlex')
u, v, t = R.gens()
Qgeom = 1 - u**2*v**2

SYM_BASIS = [
    R(1),
    u + v,
    u*v,
    u**2 + v**2,
    t*(u+v),
    t**2,
    t,
]

ANTI_BASIS = [
    u-v,
    t*(u-v),
    (u-v)*(u+v),
]

A_BASIS = SYM_BASIS + ANTI_BASIS
# Swapping u and v fixes the symmetric basis and negates the antisymmetric basis.
A_SWAP_BASIS = SYM_BASIS + [-m for m in ANTI_BASIS]

if INCLUDE_T_INDEPENDENT_C_GAUGES:
    C_BASIS = list(SYM_BASIS)
else:
    # The first four symmetric monomials are independent of t and disappear
    # from (1+t^2) C_t.  Remove these pure bulk-gauge columns.
    C_BASIS = [t*(u+v), t**2, t]


def cert_den_value(nn, mode):
    z = F(nn)
    if mode == "one":
        return F(1)
    if mode == "squarefree":
        return ((z+1)*(z+2)*(z+3)*(2*z+3)*(2*z+5)*(2*z+7))
    raise ValueError(mode)


def cert_den_degree(mode):
    return 0 if mode == "one" else 6


def cert_den_exact(N, mode):
    if mode == "one":
        return N.parent()(1)
    if mode == "squarefree":
        return (N+1)*(N+2)*(N+3)*(2*N+3)*(2*N+5)*(2*N+7)
    raise ValueError(mode)


def M_H_mod(nn):
    z = F(nn)
    # Reject actual poles before dividing.
    den = 2*(z+2)**2*(z+3)**2*(2*z+5)*(2*z+7)**2
    if den == 0:
        raise ZeroDivisionError("bad n modulo p")
    return normalized_matrix_reduced(z, F)


def good_n(nn, denominator_mode):
    z = F(nn)
    dm = 2*(z+2)**2*(z+3)**2*(2*z+5)*(2*z+7)**2
    return dm != 0 and cert_den_value(nn, denominator_mode) != 0


def first_good_ns(count, denominator_mode, start=0):
    out = []
    k = ZZ(start)
    while len(out) < count:
        if good_n(k, denominator_mode):
            out.append(k)
        k += 1
    return out


# ================================================================
# Normalized two-row trajectory, used only in diagnostic mode
# ================================================================


def normalized_trajectory(max_n):
    rows = [matrix(F, INITIAL_ROWS_ZZ)]
    for nn in range(max_n):
        rows.append(rows[-1] * M_H_mod(nn))
    return rows


# ================================================================
# Linearized AZ operator
# ================================================================


def unknown_labels(d):
    labels = []
    for r in range(d+1):
        for j in range(len(A_BASIS)):
            labels.append(('A', j, r))
        for j in range(len(C_BASIS)):
            labels.append(('C', j, r))
    return labels


def contribution_polynomial(label, nn):
    """Q times the right side contribution of one numerator coefficient."""
    kind, j, r = label
    z = F(nn)
    scalar = z**r
    Kpoly = R(z+1) + R(z-1)*u**2*v**2

    if kind == 'A':
        m = A_BASIS[j]
        ms = A_SWAP_BASIS[j]
        inner = (
            u*Qgeom*m.derivative(u)
            + v*Qgeom*ms.derivative(v)
            + Kpoly*(m+ms)
        )
        return scalar * Qgeom * inner

    if kind == 'C':
        m = C_BASIS[j]
        inner = (1+t**2)*m.derivative(t)
        return scalar * Qgeom * inner

    raise ValueError(kind)


def target_scalar(nn, i, j, mode, trajectory=None):
    if mode == "universal":
        return M_H_mod(nn)[i, j]
    if mode == "trajectory":
        if trajectory is None:
            raise ValueError("trajectory table missing")
        return trajectory[nn+1][i, j]
    raise ValueError(mode)


def target_polynomial(nn, i, j, mode, denominator_mode, trajectory=None):
    E = cert_den_value(nn, denominator_mode)
    scalar = target_scalar(nn, i, j, mode, trajectory)

    if mode == "trajectory":
        # Audit the direct cleared LHS:
        # uv P_{n+1} - Q P_n M_H = (uv-Q)P_{n+1}.
        Pn = trajectory[nn]
        Pnext = trajectory[nn+1]
        direct = u*v*R(Pnext[i, j]) - Qgeom*R((Pn*M_H_mod(nn))[i, j])
        simplified = (u*v-Qgeom)*R(Pnext[i, j])
        assert direct == simplified

    return R(E) * (u*v-Qgeom) * R(scalar)


# ================================================================
# Assemble and solve one matrix-entry block
# ================================================================


def build_block(i, j, d, train_ns, mode, denominator_mode, trajectory=None):
    labels = unknown_labels(d)
    rows = []
    rhs = []

    for nn in train_ns:
        cols = [contribution_polynomial(label, nn) for label in labels]
        target = target_polynomial(
            nn, i, j, mode, denominator_mode, trajectory
        )

        support = set(target.dict().keys())
        for col in cols:
            support.update(col.dict().keys())

        for exponent in sorted(support):
            row = [col.dict().get(exponent, F(0)) for col in cols]
            b = target.dict().get(exponent, F(0))
            if any(c != 0 for c in row) or b != 0:
                rows.append(row)
                rhs.append(b)

    A = matrix(F, rows, sparse=True)
    b = vector(F, rhs)
    return A, b, labels


def solve_block(i, j, d, train_ns, mode, denominator_mode, trajectory=None):
    A, b, labels = build_block(
        i, j, d, train_ns, mode, denominator_mode, trajectory
    )
    rank = A.rank()
    bcol = matrix(F, A.nrows(), 1, list(b), sparse=True)
    aug_rank = A.augment(bcol).rank()
    consistent = (rank == aug_rank)
    affine_dimension = A.ncols() - rank if consistent else None

    info = {
        'equations': A.nrows(),
        'unknowns': A.ncols(),
        'rank': rank,
        'aug_rank': aug_rank,
        'consistent': consistent,
        'affine_dimension': affine_dimension,
    }

    if not consistent:
        return None, labels, info

    solution = A.solve_right(b)
    return vector(F, solution), labels, info


def verify_block(solution, labels, i, j, test_ns, mode,
                 denominator_mode, trajectory=None):
    for nn in test_ns:
        lhs = R(0)
        for c, label in zip(solution, labels):
            if c != 0:
                lhs += c * contribution_polynomial(label, nn)
        rhs = target_polynomial(
            nn, i, j, mode, denominator_mode, trajectory
        )
        if lhs != rhs:
            return False, nn, lhs-rhs
    return True, None, None


# ================================================================
# Degree search
# ================================================================


def matrix_shape(mode):
    return (3, 3) if mode == "universal" else (2, 3)


def conservative_n_degree_bound(d, denominator_mode):
    # After multiplication by the common denominator D_M (degree 7),
    # the numerator degree is bounded by max(deg(E)+7, d+8).
    return max(cert_den_degree(denominator_mode)+7, d+8)


def run_search(d, mode, denominator_mode):
    if mode == "universal":
        proof_count = conservative_n_degree_bound(d, denominator_mode) + 1
    else:
        # Trajectory mode has no polynomial-in-n proof bound; this is only
        # a train/holdout diagnostic.
        proof_count = max(12, 2*(d+1)+8)

    all_ns = first_good_ns(
        proof_count + EXTRA_HOLDOUT_VALUES,
        denominator_mode,
        start=0,
    )
    train_ns = all_ns[:proof_count]
    holdout_ns = all_ns[proof_count:]

    rows, cols = matrix_shape(mode)
    trajectory = None
    if mode == "trajectory":
        trajectory = normalized_trajectory(max(all_ns)+1)

    print("\n" + "="*78)
    print("mode =", mode,
          " denominator =", denominator_mode,
          " d =", d,
          " train n =", train_ns,
          " holdout n =", holdout_ns)

    solutions = {}
    labels0 = None
    for i in range(rows):
        for j in range(cols):
            sol, labels, info = solve_block(
                i, j, d, train_ns, mode, denominator_mode, trajectory
            )
            labels0 = labels
            print(
                "entry", (i, j),
                "eq", info['equations'],
                "unk", info['unknowns'],
                "rank", info['rank'],
                "aug", info['aug_rank'],
                "consistent", info['consistent'],
                "affine_dim", info['affine_dimension'],
            )
            if sol is None:
                return None

            ok, bad_n, residual = verify_block(
                sol, labels, i, j, holdout_ns, mode,
                denominator_mode, trajectory
            )
            print("  holdout:", "PASS" if ok else "FAIL")
            if not ok:
                print("  first failing n =", bad_n)
                print("  residual =", residual)
                return None
            solutions[(i, j)] = sol

    return {
        'd': d,
        'mode': mode,
        'denominator_mode': denominator_mode,
        'solutions': solutions,
        'labels': labels0,
        'train_ns': train_ns,
        'holdout_ns': holdout_ns,
    }


# ================================================================
# Pretty-print the modular candidate
# ================================================================


def candidate_matrices_over_F(candidate):
    d = candidate['d']
    mode = candidate['mode']
    solutions = candidate['solutions']
    labels = candidate['labels']
    nr, nc = matrix_shape(mode)

    SR = PolynomialRing(R, 'Ncoef')
    Ncoef = SR.gen()
    Aout = matrix(SR, nr, nc, 0)
    Bout = matrix(SR, nr, nc, 0)
    Cout = matrix(SR, nr, nc, 0)

    for (i, j), sol in solutions.items():
        a = SR(0)
        b = SR(0)
        cexpr = SR(0)
        for coeff, label in zip(sol, labels):
            if coeff == 0:
                continue
            kind, idx, r = label
            factor = SR(coeff) * Ncoef**r
            if kind == 'A':
                a += factor * SR(A_BASIS[idx])
                b += factor * SR(A_SWAP_BASIS[idx])
            else:
                cexpr += factor * SR(C_BASIS[idx])
        Aout[i, j] = a
        Bout[i, j] = b
        Cout[i, j] = cexpr

    return Aout, Bout, Cout


def print_modular_candidate(candidate):
    Aout, Bout, Cout = candidate_matrices_over_F(candidate)
    print("\nMODULAR CERTIFICATE over GF(%s)" % P_MOD)
    print("A_num =")
    print(Aout)
    print("B_num = A_num(v,u,t) =")
    print(Bout)
    print("C_num =")
    print(Cout)
    print("common certificate denominator =",
          candidate['denominator_mode'])


# ================================================================
# One-prime rational reconstruction and exact verification
# ================================================================


def lift_residue(c):
    a = ZZ(c)
    try:
        return QQ(a.rational_reconstruction(P_MOD))
    except (ValueError, ArithmeticError, ZeroDivisionError):
        if a > P_MOD//2:
            a -= P_MOD
        return QQ(a)


def attempt_exact_verification(candidate):
    if candidate['mode'] != "universal":
        print("Exact symbolic verifier is implemented only for universal mode.")
        return False

    d = candidate['d']
    den_mode = candidate['denominator_mode']
    solutions = candidate['solutions']
    labels = candidate['labels']

    S = PolynomialRing(QQ, names=('N', 'U', 'V', 'T'))
    N, U, V, T = S.gens()
    K = S.fraction_field()
    N, U, V, T = map(K, (N, U, V, T))
    Qg = 1-U**2*V**2
    Kpoly = N+1 + (N-1)*U**2*V**2

    sym = [
        K(1), U+V, U*V, U**2+V**2,
        T*(U+V), T**2, T,
    ]
    anti = [U-V, T*(U-V), (U-V)*(U+V)]
    Abasis = sym + anti
    Aswap = sym + [-m for m in anti]
    Cbasis = sym if INCLUDE_T_INDEPENDENT_C_GAUGES else [T*(U+V), T**2, T]

    A = matrix(K, 3, 3, 0)
    B = matrix(K, 3, 3, 0)
    C = matrix(K, 3, 3, 0)

    for (i, j), sol in solutions.items():
        for coeff_mod, label in zip(sol, labels):
            if coeff_mod == 0:
                continue
            coeff = K(lift_residue(coeff_mod))
            kind, idx, r = label
            factor = coeff*N**r
            if kind == 'A':
                A[i, j] += factor*Abasis[idx]
                B[i, j] += factor*Aswap[idx]
            else:
                C[i, j] += factor*Cbasis[idx]

    MH = normalized_matrix_reduced(N, K)
    E = K(cert_den_exact(N, den_mode))
    Au = A.apply_map(lambda x: x.derivative(U))
    Bv = B.apply_map(lambda x: x.derivative(V))
    Ct = C.apply_map(lambda x: x.derivative(T))

    inner = U*Qg*Au + V*Qg*Bv + (1+T**2)*Ct + Kpoly*(A+B)
    residual = E*(U*V-Qg)*MH - Qg*inner

    ok = True
    for i in range(3):
        for j in range(3):
            num = K(residual[i, j]).numerator()
            if num != 0:
                ok = False
                print("Exact residual nonzero at", (i, j))
                print(num)

    if ok:
        print("EXACT VERIFICATION OVER Q(N,U,V,T): PASS")
        print("A_num =", A)
        print("B_num =", B)
        print("C_num =", C)
        print("E(N) =", E)
    else:
        print("One-prime rational reconstruction did not verify exactly.")
        print("Use two or more primes, CRT the unique solution coordinates,")
        print("then run the same exact verifier.")
    return ok


# ================================================================
# Main driver
# ================================================================


def main():
    print("prime =", P_MOD, "is_prime =", is_prime(P_MOD))
    print("target mode =", TARGET_MODE)
    print("A monomials per entry =", len(A_BASIS))
    print("C monomials per entry =", len(C_BASIS))
    exact_matrix_audit()

    found = False
    for den_mode in DENOMINATOR_MODES:
        for d in DEGREES_TO_TRY:
            candidate = run_search(d, TARGET_MODE, den_mode)
            if candidate is None:
                continue
            found = True
            print_modular_candidate(candidate)
            if TARGET_MODE == "universal":
                attempt_exact_verification(candidate)
            if STOP_AT_FIRST_CANDIDATE:
                return

    if not found:
        print("\nNo candidate found in the requested search ladder.")
        print("Next actions:")
        print("  1. keep universal mode;")
        print("  2. try d=3,4;")
        print("  3. retain only the existing shifted factors in E(N);")
        print("  4. enlarge the geometric total degree to 3;")
        print("  5. do not interpret trajectory-mode failure as an AZ no-go theorem.")


if __name__ == '__main__':
```

# 3. How to read the output

For each matrix entry the program prints

```text
rank       = rank of the coefficient matrix
aug        = rank after adjoining the target column
consistent = whether a solution exists
 affine_dim = dimension of the affine solution space
```
