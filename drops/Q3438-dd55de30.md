frontier.  This is convenient for route-facing wrappers while keeping the core
record parametric and `Prop`-friendly. -/

    (u : ℝ → intervalDomainPoint → ℝ) : Prop :=
  ∃ (F : ℝ → ℝ → ℝ) (b : ℝ),
