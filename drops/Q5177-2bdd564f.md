### Audit of `∫ e² ≤ sup(e) · ∫ |e|`

The proposed BUC-to-L² step is mathematically valid **provided the error is L¹**, not merely bounded. If

```text
|e_t(x)| ≤ E(t),       E(t) → 0,
∫ |e_t| ≤ L            uniformly,
```

then

```text
∫ e_t² ≤ E(t) ∫ |e_t| ≤ E(t)L → 0.
```

A directly reusable Lean lemma is:

```lean
theorem integral_sq_le_const_mul_integral_abs
    {f : ℝ → ℝ} {C : ℝ}
    (hf : Integrable f) (hC : 0 ≤ C)
    (hbound : ∀ x, |f x| ≤ C) :
    Integrable (fun x => f x ^ 2) ∧
      (∫ x : ℝ, f x ^ 2) ≤ C * ∫ x : ℝ, |f x| := by
  have habs : Integrable (fun x => |f x|) := hf.abs
  have hmajor : Integrable (fun x => C * |f x|) := habs.const_mul C
  have hsq : Integrable (fun x => f x ^ 2) := by
    refine hmajor.mono' (hf.aestronglyMeasurable.pow 2) ?_
    filter_upwards with x
    rw [Real.norm_eq_abs, abs_of_nonneg (sq_nonneg _)]
    calc
      f x ^ 2 = |f x| * |f x| := by rw [← sq_abs, pow_two]
      _ ≤ C * |f x| :=
        mul_le_mul_of_nonneg_right (hbound x) (abs_nonneg _)
  refine ⟨hsq, ?_⟩
  calc
    (∫ x : ℝ, f x ^ 2) ≤ ∫ x : ℝ, C * |f x| := by
      apply integral_mono hsq hmajor
      intro x
      calc
        f x ^ 2 = |f x| * |f x| := by rw [← sq_abs, pow_two]
        _ ≤ C * |f x| :=
          mul_le_mul_of_nonneg_right (hbound x) (abs_nonneg _)
    _ = C * ∫ x : ℝ, |f x| := by rw [integral_const_mul]
```

For compactly supported `g`, one can additionally prove

```text
∫ |S(t)g| ≤ weightedMovingHeatGrowth eta c t · ∫ |g|,
```

by positivity, column mass one, and `MeasureTheory.integral_integral_swap`; hence

```text
∫ |S(t)g-g| ≤ (growth(t)+1)∫|g|.
```

So the direct route is sound. It is not the shortest current implementation, because it requires a separate L¹ preservation/Fubini theorem. The landed polarization proof avoids that theorem and only needs the pairing bound

```lean

```

Also, **BUC convergence alone is not an L² argument**. On the whole line a uniformly small bounded function can have infinite or drifting L² mass. Compact support or an independent L¹ bound is load-bearing.

## 4. One honest local Gaussian dominator

To pass from pointwise differentiation to L¹ differentiation, prove one local majorant for the derivative kernel. A convenient explicit envelope is as follows.

```lean
def weightedMovingHeatGeneratorLocalEnvelopeConst
    (eta c r : ℝ) : ℝ :=
  let lam := eta ^ 2 - c * eta
  let d := c - 2 * eta
  let shift :=
    Real.exp ((1 / (12 * r)) * (|d| * (r / 2)) ^ 2)
  shift *
    (Real.exp (|lam| * (3 * r / 2)) *
        (5 * ((1 / r) * (1 / Real.sqrt (2 * Real.pi * r)))) +

            ((1 / r) * (1 / Real.sqrt (2 * Real.pi * r)) *
              Real.sqrt (12 * r)) +

    (eta c r z : ℝ) : ℝ :=

    Real.exp (-(1 / (24 * r)) *
```

The scalar shift estimate needed to make the center independent of nearby time is fully elementary:

```lean
theorem exp_neg_sq_shift_le
    {a w v V : ℝ} (ha : 0 ≤ a) (hV : 0 ≤ V)
    (hv : |v| ≤ V) :
    Real.exp (-a * (w + v) ^ 2) ≤
      Real.exp (a * V ^ 2) *
        Real.exp (-(a / 2) * w ^ 2) := by
  have hv2 : v ^ 2 ≤ V ^ 2 := by
    have h := (sq_le_sq₀ (abs_nonneg v) hV).2 hv
    simpa [sq_abs] using h
  have hquad : w ^ 2 / 2 - v ^ 2 ≤ (w + v) ^ 2 := by
    nlinarith [sq_nonneg (w + 2 * v)]
  have h1 := mul_le_mul_of_nonneg_left hquad ha
  have h2 := mul_le_mul_of_nonneg_left hv2 ha
  have harg : -a * (w + v) ^ 2 ≤
      a * V ^ 2 - (a / 2) * w ^ 2 := by
    nlinarith
  calc
    Real.exp (-a * (w + v) ^ 2) ≤
        Real.exp (a * V ^ 2 - (a / 2) * w ^ 2) :=
      Real.exp_le_exp.mpr harg
    _ = Real.exp (a * V ^ 2) *
        Real.exp (-(a / 2) * w ^ 2) := by
      rw [← Real.exp_add]
      congr 1
      ring
```

## 7. Generator shift: differentiate the landed semigroup identity

Once `(ii)` is available, the shortest rigorous proof of `(iv)` does **not** need another Fubini argument. Differentiate the already-proved semigroup equality near the positive time `r` and use uniqueness of the derivative.

The apply-level proof is nearly self-contained:

```lean
theorem weightedMovingHeatL2Generator_semigroup_apply
    (eta c : ℝ) {r q : ℝ} (hr : 0 < r) (hq : 0 ≤ q)
    (Z : WholeLineRealL2) :
    weightedMovingHeatL2Generator eta c r
        (weightedMovingHeatL2Semigroup eta c q Z) =
      weightedMovingHeatL2Generator eta c (r + q) Z := by
  let S : ℝ → WholeLineRealL2 →L[ℝ] WholeLineRealL2 :=
    weightedMovingHeatL2Semigroup eta c
  let A : ℝ → WholeLineRealL2 →L[ℝ] WholeLineRealL2 :=
    weightedMovingHeatL2Generator eta c

  have hleft :
      HasDerivAt (fun s : ℝ => S s (S q Z)) (A r (S q Z)) r := by
    simpa [S, A] using
      weightedMovingHeatL2Semigroup_hasDerivAt eta c hr (S q Z)

  have hrq : 0 < r + q := add_pos_of_pos_of_nonneg hr hq
  have hinner : HasDerivAt (fun s : ℝ => s + q) 1 r := by
    simpa using (hasDerivAt_id r).add_const q
  have hright :
      HasDerivAt (fun s : ℝ => S (s + q) Z) (A (r + q) Z) r := by
    simpa [S, A] using
      (weightedMovingHeatL2Semigroup_hasDerivAt eta c hrq Z).comp r hinner

  have heq :
      (fun s : ℝ => S s (S q Z)) =ᶠ[𝓝 r]
        (fun s : ℝ => S (s + q) Z) := by
    filter_upwards [Ioi_mem_nhds hr] with s hs
    have h := congrArg
      (fun L : WholeLineRealL2 →L[ℝ] WholeLineRealL2 => L Z)
      (weightedMovingHeatL2Semigroup_add
        (eta := eta) (c := c) hs.le hq)
    simpa [S, ContinuousLinearMap.comp_apply] using h

  have hrightOnLeft :
      HasDerivAt (fun s : ℝ => S s (S q Z)) (A (r + q) Z) r :=
    heq.hasDerivAt_iff.mpr hright
  exact hleft.unique hrightOnLeft
```

If `HasDerivAt.unique` elaborates in the opposite orientation under the local Mathlib version, the final line is simply

```lean
exact (hrightOnLeft.unique hleft).symm
```

Then lift to CLM equality:

```lean
theorem weightedMovingHeatL2Generator_comp_semigroup
    (eta c : ℝ) {r q : ℝ} (hr : 0 < r) (hq : 0 ≤ q) :
    (weightedMovingHeatL2Generator eta c r).comp
        (weightedMovingHeatL2Semigroup eta c q) =
      weightedMovingHeatL2Generator eta c (r + q) := by
  ext Z
  exact weightedMovingHeatL2Generator_semigroup_apply eta c hr hq Z
```

This proof keeps all dependencies honest:

```text
concrete Gaussian Schur construction
    → strong continuity
    → L¹ kernel derivative
    → L² orbit derivative
    → semigroup derivative uniqueness
    → generator shift.
```
