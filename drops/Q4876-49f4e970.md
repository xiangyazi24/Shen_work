## Result

The finite computation gives a clear negative answer to the proposed same-index and constant-width-two identifications.

There is also a parameter correction to make before interpreting the result. The actual Q4873 kernel is built from

\[
J_n(X;\varepsilon)=P_n^{(\varepsilon-1/2,0)}(1-2X),
\]

not from \(P_n^{(0,\varepsilon)}(1-2X)\). The latter is a different Jacobi deformation: its parameter derivative corresponds to a different weight deformation. The program below therefore tests both:

1. the actual Q4873 polynomial \(R_n^{\rm J}\), including its mixed-Wronskian correction; and
2. the literal derivative polynomial stated in the question,
   \(\left.\partial_\varepsilon P_n^{(0,\varepsilon)}(1-2X)\right|_{\varepsilon=0}\).

All matrix products and Jacobi coefficients are computed exactly with `Fraction`. `mpmath` is used with 140 working digits. An independent numerical quadrature agrees with the exact coefficientwise Catalan-moment evaluation to relative error below

\[
4.42\times 10^{-127}.
\]

The conclusions are:

- The normalized CMF error \(\widehat e_n=G\widehat Q_n-\widehat P_n\) is **not proportional** to \(\mathcal C[R_n^{\rm J}]\). The ratio changes from
  \(-15.6363875\ldots\) at \(n=0\) to \(-0.0544288\ldots\) at \(n=10\).
- Fitting
  \[
  \widehat e_n=a\,\mathcal C[R_n^{\rm J}]
  +b\,\mathcal C[R_{n-1}^{\rm J}]

  at \(n=1,2\) gives
  \[
  \begin{aligned}

  b&= 1060.2623624892154610133678467554156419373593021608245933827518233863269779\ldots,

  \[
  40.2203081824712750901871566256504103200320346329831048195653\ldots.
  \]
  Thus the constant-coefficient width-two ansatz fails decisively.
- Treating the Catalan moments formally as elements of \(\mathbf QG+\mathbf Q\), the coefficientwise width-two test already has first discrepancy
  \[
  \left(n,\Delta_G,\Delta_{\mathbf Q}\right)
  =\left(1,0,\frac{18247321725}{56056}\right).

The numerical failure does **not** rule out the wider relation suggested in Q4873, involving index shifts, rational functions of \(n\), and the Christoffel-Darboux kernel. It rules out only literal equality, scalar proportionality, and the requested constant width-two band.

#!/usr/bin/env python3
"""Q4876: compare the normalized Problem 2.5 CMF error with Jacobi kernels.

The CMF matrix product and all Jacobi coefficients are exact Fractions.
mpmath is used with 140 working digits for Catalan's constant and for an
independent quadrature audit.
"""

from __future__ import annotations

from dataclasses import dataclass
from decimal import Decimal
from fractions import Fraction as Q
from typing import Sequence

from mpmath import mp

mp.dps = 140


def as_q(x: int | Q) -> Q:
    return x if isinstance(x, Q) else Q(x)


@dataclass(frozen=True)
class Dual:
    val: Q
    der: Q = Q(0)

    @staticmethod
    def coerce(x: int | Q | "Dual") -> "Dual":
        if isinstance(x, Dual):
            return x
        return Dual(as_q(x), Q(0))

    def __add__(self, other: int | Q | "Dual") -> "Dual":
        o = Dual.coerce(other)
        return Dual(self.val + o.val, self.der + o.der)

    __radd__ = __add__

    def __sub__(self, other: int | Q | "Dual") -> "Dual":
        o = Dual.coerce(other)
        return Dual(self.val - o.val, self.der - o.der)

    def __rsub__(self, other: int | Q | "Dual") -> "Dual":
        return Dual.coerce(other) - self

    def __mul__(self, other: int | Q | "Dual") -> "Dual":
        o = Dual.coerce(other)
        return Dual(self.val * o.val,
                    self.der * o.val + self.val * o.der)

    __rmul__ = __mul__

    def __truediv__(self, other: int | Q | "Dual") -> "Dual":
        o = Dual.coerce(other)
        if o.val == 0:
            raise ZeroDivisionError("dual division by zero")
        return Dual(
            self.val / o.val,
            (self.der * o.val - self.val * o.der) / o.val**2,
        )

    def __rtruediv__(self, other: int | Q | "Dual") -> "Dual":
        return Dual.coerce(other) / self

    def __neg__(self) -> "Dual":
        return Dual(-self.val, -self.der)


def trim_q(p: Sequence[Q]) -> list[Q]:
    out = list(p)
    while len(out) > 1 and out[-1] == 0:
        out.pop()
    return out


def p_add(a, b):
    n = max(len(a), len(b))
    is_dual = ((a and isinstance(a[0], Dual)) or
               (b and isinstance(b[0], Dual)))
    zero = Dual(Q(0)) if is_dual else Q(0)
    out = [zero for _ in range(n)]
    for i, c in enumerate(a):
        out[i] = out[i] + c
    for i, c in enumerate(b):
        out[i] = out[i] + c
    return out


def p_sub(a, b):
    n = max(len(a), len(b))
    is_dual = ((a and isinstance(a[0], Dual)) or
               (b and isinstance(b[0], Dual)))
    zero = Dual(Q(0)) if is_dual else Q(0)
    out = [zero for _ in range(n)]
    for i, c in enumerate(a):
        out[i] = out[i] + c
    for i, c in enumerate(b):
        out[i] = out[i] - c
    return out


def p_scale(a, s):
    return [s * c for c in a]


def p_xmul(a):
    zero = Dual(Q(0)) if a and isinstance(a[0], Dual) else Q(0)
    return [zero] + list(a)


def p_mul_q(a: Sequence[Q], b: Sequence[Q]) -> list[Q]:
    out = [Q(0)] * (len(a) + len(b) - 1)
    for i, x in enumerate(a):
        for j, y in enumerate(b):
            out[i + j] += x * y
    return trim_q(out)


def p_comp_linear_q(p: Sequence[Q], a: Q, b: Q) -> list[Q]:
    """Return p(a+bX), with ascending coefficients in X."""
    result = [Q(0)]
    power = [Q(1)]
    for c in p:
        result = p_add(result, p_scale(power, c))
        power = p_mul_q(power, [a, b])
    return trim_q(result)


def p_eval_q(p: Sequence[Q], x: Q) -> Q:
    acc = Q(0)
    for c in reversed(p):
        acc = acc * x + c
    return acc


def p_div_x_plus_1_q(p: Sequence[Q]) -> list[Q]:
    """Divide exactly by X+1."""
    p = trim_q(p)
    if len(p) == 1:
        if p[0] != 0:
            raise ArithmeticError("not divisible by X+1")
        return [Q(0)]
    degree = len(p) - 1
    quotient = [Q(0)] * degree
    quotient[-1] = p[-1]
    for k in range(degree - 1, 0, -1):
        quotient[k - 1] = p[k] - quotient[k]
    remainder = p[0] - quotient[0]
    if remainder != 0:
        raise ArithmeticError(f"nonzero remainder: {remainder}")
    return trim_q(quotient)


def jacobi_dual_polys(
    nmax: int,
    *,
    alpha0: Q,
    alpha_der: Q,
    beta0: Q,
    beta_der: Q,
) -> tuple[list[list[Q]], list[list[Q]]]:
    """Jacobi polynomials and their exact epsilon derivatives.

    alpha=alpha0+alpha_der*eps and beta=beta0+beta_der*eps.
    The standard Jacobi recurrence is differentiated by dual arithmetic.
    """
    a = Dual(alpha0, alpha_der)
    b = Dual(beta0, beta_der)

    polys: list[list[Dual]] = [[Dual(Q(1))]]
    if nmax >= 1:
        # P_1^(a,b)(x)=((a-b)+(a+b+2)x)/2.
        polys.append([(a - b) / 2, (a + b + 2) / 2])

    for n in range(1, nmax):
        # 2(n+1)(n+a+b+1)(2n+a+b)P_{n+1}
        # = (2n+a+b+1)[((2n+a+b)(2n+a+b+2)x+a^2-b^2)]P_n
        #   -2(n+a)(n+b)(2n+a+b+2)P_{n-1}.
        left = 2 * (n + 1) * (n + a + b + 1) * (2*n + a + b)
        c0 = (2*n + a + b + 1) * (a*a - b*b)
        c1 = ((2*n + a + b + 1) *
              (2*n + a + b) *
              (2*n + a + b + 2))
        back = 2 * (n + a) * (n + b) * (2*n + a + b + 2)

        rhs = p_add(p_scale(polys[n], c0),
                    p_scale(p_xmul(polys[n]), c1))
        rhs = p_sub(rhs, p_scale(polys[n - 1], back))
        polys.append(p_scale(rhs, 1 / left))

    return (
        [[c.val for c in p] for p in polys],
        [[c.der for c in p] for p in polys],
    )


def q4873_jacobi_kernels(nmax: int):
    """Construct Q4873 equations (3.8)-(3.9)."""
    px, dpx = jacobi_dual_polys(
        nmax,
        alpha0=Q(-1, 2), alpha_der=Q(1),
        beta0=Q(0), beta_der=Q(0),
    )

    kernels = []
    for n in range(nmax + 1):
        J = p_comp_linear_q(px[n], Q(1), Q(-2))
        dJ = p_comp_linear_q(dpx[n], Q(1), Q(-2))
        B = p_eval_q(J, Q(-1))
        dB = p_eval_q(dJ, Q(-1))

        numerator = p_sub(p_scale(dJ, B), p_scale(J, dB))
        C = p_div_x_plus_1_q(numerator)
        opC = [Q(2*k + 1) * c for k, c in enumerate(C)]
        correction = p_scale(p_mul_q([Q(1), Q(1)], opC), Q(1, 2))
        bracket = p_sub(p_scale(J, B), correction)
        kappa = Q(4*n + 1, 2)
        R = trim_q(p_scale(bracket, kappa))

        # Q4873 equation (3.13).
        assert p_eval_q(R, Q(-1)) == kappa * B * B
        kernels.append(R)

    return kernels


def literal_beta_derivative_polys(nmax: int) -> list[list[Q]]:
    """Literal derivative d/deps P_n^(0,eps)(1-2X)|_0."""
    _, dpx = jacobi_dual_polys(
        nmax,
        alpha0=Q(0), alpha_der=Q(0),
        beta0=Q(0), beta_der=Q(1),
    )
    return [p_comp_linear_q(p, Q(1), Q(-2)) for p in dpx]


def q_to_mp(x: Q):
    return mp.mpf(x.numerator) / mp.mpf(x.denominator)


def catalan_moment_pair(poly: Sequence[Q]) -> tuple[Q, Q]:
    """Return q,r such that C[poly]=q*G+r exactly."""
    q = Q(0)
    r = Q(0)
    partial = Q(0)
    for k, coeff in enumerate(poly):
        sign = Q(1 if k % 2 == 0 else -1)
        q += coeff * sign
        r -= coeff * sign * partial
        partial += sign / Q((2*k + 1)**2)
    assert q == p_eval_q(poly, Q(-1))
    return q, r


def p_eval_mp(poly: Sequence[Q], x):
    acc = mp.mpf("0")
    for c in reversed(poly):
        acc = acc * x + q_to_mp(c)
    return acc


def catalan_moment_quad(poly: Sequence[Q]):
    # t=exp(-u) removes the logarithmic endpoint singularity.
    def integrand(u):
        e = mp.e ** (-u)
        x = e * e
        return u * e * p_eval_mp(poly, x) / (1 + x)

    return mp.quad(integrand, [0, 1, 3, 8, 20, mp.inf])

    m22 = (n+2)**2*(-272*n**5-3848*n**4-21732*n**3-61184*n**2-85761*n-47808)
    m23 = (n+2)**2*(320*n**3+2540*n**2+6610*n+5640)

    return ((m11, m12, m13),

def row_times_matrix(row: Sequence[int], mat) -> tuple[int, int, int]:
    return tuple(sum(row[i] * mat[i][j] for i in range(3))
                 for j in range(3))


def delta_h(n: int) -> int:
    # H_{n+1}/H_n.
    return -2 * (n+2)**2 * (n+3)**2 * (2*n+5) * (2*n+7)**2


def cmf_normalized_data(nmax: int):
    p_row = (30921, -32972, 8240)
    q_row = (33750, -36000, 9000)
    H = Q(1)
    out = []

    for n in range(nmax + 1):
        phat = Q(p_row[0]) / H
        qhat = Q(q_row[0]) / H
        err = q_to_mp(qhat) * mp.catalan - q_to_mp(phat)
        out.append((phat, qhat, err))

        if n < nmax:
            M = matrix_M(n)
            p_row = row_times_matrix(p_row, M)
            q_row = row_times_matrix(q_row, M)
            H *= delta_h(n)

    return out


def fmt(x, digits=72):
    return mp.nstr(x, digits)


def rational_guess(x, max_denominator=10**9):

def proportional_fit(target, moments):
    a = target[0] / moments[0]
    return a, [target[n] - a*moments[n] for n in range(len(target))]


def width2_fit(target, moments):
    # Fit e_n=a*m_n+b*m_{n-1} at n=1,2.
    det = moments[1]**2 - moments[0]*moments[2]
    a = (target[1]*moments[1] - moments[0]*target[2]) / det
    b = (moments[1]*target[2] - moments[2]*target[1]) / det
    residuals = [mp.nan]
    for n in range(1, len(target)):
        residuals.append(target[n] - a*moments[n] - b*moments[n-1])
    return a, b, residuals


def exact_width2_formal_test(target_pairs, moment_pairs):
    """Coefficientwise test in the formal two-dimensional space Q*G+Q."""
    tq = [x[0] for x in target_pairs]
    tr = [x[1] for x in target_pairs]
    mq = [x[0] for x in moment_pairs]
    mr = [x[1] for x in moment_pairs]

    det = mq[1]**2 - mq[0]*mq[2]
    a = (tq[1]*mq[1] - mq[0]*tq[2]) / det
    b = (mq[1]*tq[2] - mq[2]*tq[1]) / det

    for n in range(1, len(target_pairs)):
        dq = tq[n] - a*mq[n] - b*mq[n-1]
        dr = tr[n] - a*mr[n] - b*mr[n-1]
        if dq != 0 or dr != 0:
            return a, b, (n, dq, dr)
    return a, b, None

## Bottom line

The canonical Q4873 Jacobi construction remains an independently defined Catalan linear-form family with the right silver-ratio geometry, but the printed CMF error is neither the same family nor a constant two-term contiguous combination of it. Any successful Jacobi identification must therefore use a genuinely nonconstant finite band—most plausibly rational functions of \(n\), index shifts, and/or the Christoffel-Darboux kernel proposed in Q4873.
