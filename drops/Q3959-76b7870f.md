# Q3959: passing a Lipschitz bound to a pointwise limit

The core theorem should be split into two reusable lemmas:

1. a scalar lemma: if `aₙ → a`, `bₙ → b`, and `|aₙ - bₙ| ≤ C` for every `n`, then `|a - b| ≤ C`;
2. a function-on-a-set lemma: apply the scalar lemma pointwise for each pair `x,y ∈ S`.

The repo already uses the needed Mathlib idiom in `IntervalMildPicardLimitRegularity.lean`:

```lean
exact le_of_tendsto htendsto
  (Eventually.of_forall (fun n => ...))
```

Here the limit is `|a - b|`, obtained from `(ha.sub hb).abs`.

## General utility lemmas

```lean
import Mathlib.Topology.Algebra.Order

open Filter
open scoped Topology

noncomputable section

namespace ShenWork.Paper2.LipschitzLimitTools

/-- If `aₙ → a`, `bₙ → b`, and `|aₙ - bₙ| ≤ C` for all `n`, then
`|a - b| ≤ C`.

This is the scalar limit-passage lemma behind Lipschitz-bound closure. -/
theorem abs_sub_le_of_tendsto_of_forall_le
    {a b : ℕ → ℝ} {A B C : ℝ}
    (ha : Tendsto a atTop (nhds A))
    (hb : Tendsto b atTop (nhds B))
    (hbound : ∀ n : ℕ, |a n - b n| ≤ C) :
    |A - B| ≤ C := by
  have habs : Tendsto (fun n : ℕ => |a n - b n|) atTop (nhds |A - B|) := by
    simpa using (ha.sub hb).abs
  exact le_of_tendsto habs (Eventually.of_forall hbound)

/-- Pointwise limits preserve a uniform Lipschitz bound on a real set.

If each `F n` is `G`-Lipschitz on `S`, and `F n x → f x` for every `x ∈ S`,
then `f` is also `G`-Lipschitz on `S`. -/
theorem lipschitz_bound_on_of_pointwise_tendsto
    {S : Set ℝ} {F : ℕ → ℝ → ℝ} {f : ℝ → ℝ} {G : ℝ}
    (hlim : ∀ x ∈ S, Tendsto (fun n : ℕ => F n x) atTop (nhds (f x)))
    (hLip : ∀ n : ℕ, ∀ x ∈ S, ∀ y ∈ S,
      |F n x - F n y| ≤ G * |x - y|) :
    ∀ x ∈ S, ∀ y ∈ S, |f x - f y| ≤ G * |x - y| := by
  intro x hx y hy
  exact abs_sub_le_of_tendsto_of_forall_le
    (hlim x hx) (hlim y hy) (fun n => hLip n x hx y hy)

end ShenWork.Paper2.LipschitzLimitTools
```

This is the most robust API for the general step. It does **not** require `0 ≤ G` because the assumption already gives the scalar upper bound `G * |x-y|` for every iterate. In normal Lipschitz use, of course, `0 ≤ G` will hold upstream.

## Applying it to `truncatedConjugatePicardLimit`

The file `ShenWork/Paper2/IntervalBFormCron2TruncatedPicard.lean` defines

```lean
def truncatedConjugatePicardIter ...
def truncatedConjugatePicardLimit ... :=
  if 0 < t ∧ t ≤ T then
    atTop.limUnder (fun n => truncatedConjugatePicardIter p u₀ n t x)
  else 0
```

and proves pointwise convergence as an existence theorem. The first small bridge is to turn the `∃ L, Tendsto ... (nhds L)` statement into convergence to the actual `limUnder`-defined limit.

```lean
import ShenWork.Paper2.IntervalBFormCron2TruncatedPicard
import Mathlib.Topology.Algebra.Order

open Filter Topology
open scoped Topology

noncomputable section

namespace ShenWork.Paper2.BFormPositiveDatumNegPart

open ShenWork.IntervalDomain (intervalDomainLift intervalDomainPoint)
open ShenWork.Paper2.LipschitzLimitTools

/-- Convert the existing pointwise-convergence existence theorem into convergence
of the Picard iterates to the actual `truncatedConjugatePicardLimit`, whose value
is defined with `atTop.limUnder` on `(0,T]`. -/
theorem truncatedConjugatePicardIter_tendsto_limit_of_pointwise_convergent
    (p : CM2Params) (u₀ : intervalDomainPoint → ℝ) {T t : ℝ}
    (ht : 0 < t) (htT : t ≤ T)
    (hconv : ∀ t, 0 < t → t ≤ T → ∀ x : intervalDomainPoint,
      ∃ L : ℝ, Tendsto
        (fun n : ℕ => truncatedConjugatePicardIter p u₀ n t x)
        atTop (nhds L)) :
    ∀ x : intervalDomainPoint,
      Tendsto (fun n : ℕ => truncatedConjugatePicardIter p u₀ n t x)
        atTop (nhds (truncatedConjugatePicardLimit p u₀ T t x)) := by
  intro x
  obtain ⟨L, hL⟩ := hconv t ht htT x
  have hlim_eq : truncatedConjugatePicardLimit p u₀ T t x = L := by
    unfold truncatedConjugatePicardLimit
    simp only [ht, htT, and_self, ite_true]
    exact hL.limUnder_eq
  rw [hlim_eq]
  exact hL

/-- If all truncated B-form Picard iterates have the same spatial Lipschitz bound
on `[0,1]` at a fixed time `t ∈ (0,T]`, then the `limUnder` Picard limit has the
same bound.

This is the direct form needed for the lifted limit function. -/
theorem truncatedConjugatePicardLimit_lipschitz_of_iter_lipschitz
    (p : CM2Params) (u₀ : intervalDomainPoint → ℝ) {T t G : ℝ}
    (ht : 0 < t) (htT : t ≤ T)
    (hconv : ∀ t, 0 < t → t ≤ T → ∀ x : intervalDomainPoint,
      ∃ L : ℝ, Tendsto
        (fun n : ℕ => truncatedConjugatePicardIter p u₀ n t x)
        atTop (nhds L))
    (hLip_iter : ∀ n : ℕ, ∀ x ∈ Set.Icc (0 : ℝ) 1, ∀ y ∈ Set.Icc (0 : ℝ) 1,
      |intervalDomainLift (truncatedConjugatePicardIter p u₀ n t) x
        - intervalDomainLift (truncatedConjugatePicardIter p u₀ n t) y|
        ≤ G * |x - y|) :
    ∀ x ∈ Set.Icc (0 : ℝ) 1, ∀ y ∈ Set.Icc (0 : ℝ) 1,
      |intervalDomainLift (truncatedConjugatePicardLimit p u₀ T t) x
        - intervalDomainLift (truncatedConjugatePicardLimit p u₀ T t) y|
        ≤ G * |x - y| := by
  refine lipschitz_bound_on_of_pointwise_tendsto
    (S := Set.Icc (0 : ℝ) 1)
    (F := fun n z => intervalDomainLift (truncatedConjugatePicardIter p u₀ n t) z)
    (f := fun z => intervalDomainLift (truncatedConjugatePicardLimit p u₀ T t) z)
    ?hlim ?hLip
  · intro z hz
    have hz_tendsto :
        Tendsto
          (fun n : ℕ => truncatedConjugatePicardIter p u₀ n t ⟨z, hz⟩)
          atTop
          (nhds (truncatedConjugatePicardLimit p u₀ T t ⟨z, hz⟩)) :=
      truncatedConjugatePicardIter_tendsto_limit_of_pointwise_convergent
        p u₀ ht htT hconv ⟨z, hz⟩
    simpa [intervalDomainLift, hz] using hz_tendsto
  · intro n x hx y hy
    exact hLip_iter n x hx y hy

end ShenWork.Paper2.BFormPositiveDatumNegPart
```

## If you want to use the existing geometric convergence theorem directly

If your context has the geometric step bound used by
`truncatedConjugatePicardIter_pointwise_convergent`, specialize the previous theorem like this:

```lean
import ShenWork.Paper2.IntervalBFormCron2TruncatedPicard
import Mathlib.Topology.Algebra.Order

open Filter Topology
open scoped Topology

noncomputable section

namespace ShenWork.Paper2.BFormPositiveDatumNegPart

open ShenWork.IntervalDomain (intervalDomainLift intervalDomainPoint)
open ShenWork.Paper2.LipschitzLimitTools

/-- Same Lipschitz pass-to-limit theorem, but with the convergence discharged from
the repo's geometric pointwise-convergence theorem. -/
theorem truncatedConjugatePicardLimit_lipschitz_of_iter_lipschitz_geometric
    (p : CM2Params) (u₀ : intervalDomainPoint → ℝ) {T K C₀ G t : ℝ}
    (hK : K < 1) (hK_nn : 0 ≤ K) (hC₀ : 0 ≤ C₀)
    (hbound : ∀ (n : ℕ) (t : ℝ), 0 < t → t ≤ T →
      ∀ x : intervalDomainPoint,
        |truncatedConjugatePicardIter p u₀ (n + 1) t x
          - truncatedConjugatePicardIter p u₀ n t x| ≤ K ^ n * C₀)
    (ht : 0 < t) (htT : t ≤ T)
    (hLip_iter : ∀ n : ℕ, ∀ x ∈ Set.Icc (0 : ℝ) 1, ∀ y ∈ Set.Icc (0 : ℝ) 1,
      |intervalDomainLift (truncatedConjugatePicardIter p u₀ n t) x
        - intervalDomainLift (truncatedConjugatePicardIter p u₀ n t) y|
        ≤ G * |x - y|) :
    ∀ x ∈ Set.Icc (0 : ℝ) 1, ∀ y ∈ Set.Icc (0 : ℝ) 1,
      |intervalDomainLift (truncatedConjugatePicardLimit p u₀ T t) x
        - intervalDomainLift (truncatedConjugatePicardLimit p u₀ T t) y|
        ≤ G * |x - y| := by
  apply truncatedConjugatePicardLimit_lipschitz_of_iter_lipschitz
    (p := p) (u₀ := u₀) (T := T) (t := t) (G := G) ht htT
  · exact truncatedConjugatePicardIter_pointwise_convergent
      p u₀ hK hK_nn hC₀ hbound
  · exact hLip_iter

end ShenWork.Paper2.BFormPositiveDatumNegPart
```

## Practical insertion point

A good place for the first two utility lemmas is a small file such as

```text
ShenWork/Paper2/LipschitzLimitTools.lean
```

Then the Picard-limit bridge can live next to the truncated Picard file or in the downstream file that needs the limit Lipschitz bound. If you do not want a new file, place the two utility lemmas in a namespace near the consumer and import only `Mathlib.Topology.Algebra.Order` in addition to the existing truncated Picard imports.
