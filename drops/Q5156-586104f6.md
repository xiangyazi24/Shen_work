## 4. False implications to reject

### False implication A

```text
W ∈ L² and the unweighted slice is C² with bounded derivatives
    ⇒ Wx ∈ L² or Wxx ∈ L².
```

False. A smooth weighted field can have disjoint increasingly narrow spikes whose squared amplitudes times widths are summable while squared slopes divided by widths are not summable. Multiplying that field by `e^(-ηx)` on a right-half-line produces a bounded smooth unweighted perturbation. Thus unweighted smoothness and weighted zeroth-order decay do not control weighted derivative tails.

### False implication B

```text
capWeight ≤ a constant on the right and f is bounded
    ⇒ capWeight * f² is integrable on the right half-line.
```

False. `capWeight η R` approaches the positive plateau `exp (2ηR)` as `x → +∞`; it is not integrable there. Fixed-cap integrability requires an ordinary right-tail `L²` estimate for the field being capped. For `Wx` and `Wxx`, that right-tail estimate is not currently proved.

### False implication C

```text
A uniform cap estimate for W²
    ⇒ a uniform cap estimate for Wx².
```

False. `fullWeightedL2_integrable_of_uniform_cap` preserves the field to which it is applied; it performs no differentiation.

### False implication D

```text
∫₀ᵀ ∫ Wx² < ∞
    ⇒ ∫ Wx(t, x)² dx < ∞ for every t > 0.
```

False. It gives an `L²` slice only for almost every time. To feed the current fixed-time theorem at every positive time, one needs positive-time smoothing or weighted-space continuity.

### False implication E

```text
sup_{s in a time ball} ∫ |Wt(s)|² ≤ C
    ⇒ there is an integrable pointwise dominator G(x) for all Wt(s,x).
```

False. Uniform boundedness in `L²` does not imply domination by one `L²` function. This is the main trap in `hhalf`.

### False implication F

```text
Use paper5WeightedEnergy_deriv_le_common_of_coreIntegrability
or its consequence to prove the cap estimate that is then used to produce
hWx2, hdiff_int, and hhalf.
```

Circular. The theorem already assumes those inputs.

### False implication G

The comment in `Theorem12SolutionTailDecay.lean` saying that one remaining pointwise tail-decay lemma would by itself close `paperWeightedCoreIntegrability` is too strong at the current repository state. Population tail decay can close a zeroth-order finiteness seam. It does not provide weighted derivative regularity, a time-ball material-time envelope, or the joint differentiability wrapper required by `paper5WeightedHalfEnergy_hasDerivAt_of_dominated`.

### Node 6: weighted `H²` smoothing

The raw Hessian estimate has norm singularity

```text
||∂xx Sη(r)||_{L²→L²} = O(r^(-1)).
```

Therefore

```text
∫₀ʰ ||∂xx Sη(r) F(t-r)||₂ dr
```

cannot be bounded from a uniform `L²` bound on `F`; `∫₀ʰ r^(-1) dr` diverges.

The correct cancellation is

```text
∫₀ʰ Aη Sη(r) F(t-r) dr
 = ∫₀ʰ Aη Sη(r) (F(t-r)-F(t)) dr
   + (Sη(h)-I) F(t),
```

where `Aη` is the conjugated moving heat generator. A positive weighted-L² Hölder exponent makes the first integrand `O(r^(-1+θ))`, which is integrable.

The required capstone is:

```lean

modulus of the complete forcing. -/
```

#### Chemotaxis-specific singularity

If one differentiates the original divergence Duhamel term twice without rewriting it, a third heat derivative appears:

```text
∂xxx Sη(r) G,
```

with raw norm `O(r^(-3/2))`. Moving one derivative to the source gives

```text
∂xx Sη(r) Gx,
```

which still has the nonintegrable `r^(-1)` singularity. Thus one needs either:

```text
weighted-L² time Hölder continuity of Gx,
```

so that the same cancellation applies, or a stronger spatial assumption such as weighted `Gxx ∈ L²`, permitting another integration by parts and leaving the integrable gradient singularity `r^(-1/2)`.

The unweighted theorem

```lean

```

uses spatial Hölder cancellation pointwise. It does not by itself supply either weighted alternative.

## 7. A valid time-uniform dominating function for `hhalf`

Let

```text
W_s(x) = paper5WeightedPopulation η (coMovingPath c u) U s x
T_s(x) = paper5WeightedPopulationT η
           (paper5CoMovingMaterialTime c u) s x.
```

Assume a time-ball envelope `GT` as above and the pointwise fundamental theorem of calculus on the ball. Then for `s ∈ Metric.ball t δ`, almost every `x` satisfies

```text
|W_s(x)|
  ≤ |W_t(x)| + |s-t| GT(x)
  ≤ |W_t(x)| + δ GT(x).
```

A valid dominating function is

```lean
fun x =>
  ((|W_t x| + δ * GT x) ^ 2 + (GT x) ^ 2) / 2
```

because

```text
|W_s(x) T_s(x)|
  ≤ (|W_s(x)|² + |T_s(x)|²)/2
  ≤ ((|W_t(x)| + δ GT(x))² + GT(x)²)/2.
```

It is integrable whenever `W_t²` and `GT²` are integrable, since

```text
((|W_t| + δ GT)² + GT²)/2
  ≤ W_t² + (δ² + 1/2) GT².
```

The corresponding reusable wrapper should be:

