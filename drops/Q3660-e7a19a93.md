## Why not the other remaining inputs first?

### `w_split` / cutoff representation

This is not the smallest. It is the actual differentiated mild representation with the cutoff source:

```lean
w x = initialValueLeg t (intervalDomainLift u₀) x
  - χ₀ * chemLitLeg t (chemFluxCthetaCutoffSource p Dsol.u Dsol.T) x
  + reactionValueLeg t L x
```

To prove it from `GradientMildSolutionData`, one needs the mild fixed-point identity plus a cutoff-agreement argument showing that the cutoff source equals the true chem flux a.e. on the integration window `0 < s ≤ t ≤ Dsol.T`. This is a representation theorem, not just regularity wiring. It should remain honest until the mild representation/cutoff bridge is audited.

### endpoint no-flux

Do not take endpoint no-flux next. Existing restart-cosine theorems do provide endpoint data for the concrete slice, e.g.

```lean
gradientMild_closedC2_endpointDerivs_of_restartCosineRepresentations
neumannBoundarySlice_of_restartCosineRepresentations
```

but those are not immediately the same as the current input

```lean
derivWithin w (Set.Icc (0 : ℝ) 1) 0 = 0 ∧
derivWithin w (Set.Icc (0 : ℝ) 1) 1 = 0
```

for the `DifferentiableOn` route. In particular, endpoint no-flux for the `derivWithin` representative should be proved from a boundary-condition/representation theorem, not from endpoint values of individual legs. The old assembly file explicitly warns that endpoint no-flux is a solution-level invariant, not a termwise consequence of `∂ₓₓS(σ)Q`.

### full init/react differentiability + Holder

This is partly easy, partly not.

Already easy:

- `initialValueLeg_hasDerivAt` and `initialValueLeg_deriv_eq` prove differentiability of the initial value leg.
- `reactionValueLeg_hasDerivAt` and `reactionValueLeg_deriv_eq` prove differentiability of the reaction value leg.
- `gradLeg_holder_global` proves the derivative Hölder bound for the initial heat leg.

Still nontrivial:

- `react_holder` for the reaction Duhamel derivative is not yet a single existing theorem. It should be proved later by a Duhamel-gradient Hölder estimate using `holder_of_duhamel_integral` and the integrable rate `(t-s)^(-(1+η)/2)`, or by a dedicated reaction source regularity/boundedness wrapper.

So Task196 should harvest the easy Phase-1 value-leg part and keep only `react_holder` explicit.

```lean

This discharges `init_diff`, `react_diff`, and the initial-leg derivative Holder

theorem chemMild_C1eta_slice_diffOn_of_gradientMild_initialHolder_smallTheta_cutoff_phase1ValueLegs
    {p : CM2Params} {u₀ : intervalDomainPoint → ℝ}
    (Dsol : GradientMildSolutionData p u₀)
    {χ₀ t θ η H₀ Cu₀ CL Areact : ℝ}
    {L : ℝ → ℝ → ℝ} {w : ℝ → ℝ}
    (hη0 : 0 < η) (hη1 : η < 1) (hθη : η < θ)
    (hθ0 : 0 < θ) (hθlt : θ < (1 / 2 : ℝ))
    (hH₀_nonneg : 0 ≤ H₀)
    (hholder : InitialDatumHolder u₀ θ H₀)
    (hplan : ∀ r, 0 < r → r ≤ Dsol.T → ∀ x y : intervalDomainPoint,
      NeumannHeatContractiveCouplingFor r x y (intervalDomainLift u₀))
    (ht : 0 < t) (htT : t ≤ Dsol.T)
    (hu₀_meas : AEStronglyMeasurable (intervalDomainLift u₀)
      (ShenWork.IntervalDomain.intervalMeasure 1))
    (hCu₀_nonneg : 0 ≤ Cu₀)
    (hu₀_bdd : ∀ y : ℝ, |intervalDomainLift u₀ y| ≤ Cu₀)
    (hL_meas : Measurable (Function.uncurry L))
    (hCL_nn : 0 ≤ CL)
    (hL_bdd : ∀ s y, |L s y| ≤ CL)
    (hAreact_nn : 0 ≤ Areact)
    (w_split : ∀ x : ℝ,
      w x = initialValueLeg t (intervalDomainLift u₀) x
        - χ₀ * chemLitLeg t (chemFluxCthetaCutoffSource p Dsol.u Dsol.T) x
        + reactionValueLeg t L x)
    (react_holder : ∀ x ∈ Set.Icc (0:ℝ) 1, ∀ y ∈ Set.Icc (0:ℝ) 1,
      |reactionDerivLeg t L x - reactionDerivLeg t L y| ≤
        Areact * |x - y| ^ η)
    (hNeumann : derivWithin w (Set.Icc (0 : ℝ) 1) 0 = 0 ∧
      derivWithin w (Set.Icc (0 : ℝ) 1) 1 = 0) :
    ∃ HQ : ℝ, 0 ≤ HQ ∧
      DifferentiableOn ℝ w (Set.Icc (0:ℝ) 1) ∧
        (∀ x ∈ Set.Icc (0:ℝ) 1, ∀ y ∈ Set.Icc (0:ℝ) 1,
          |derivWithin w (Set.Icc (0:ℝ) 1) x -
              derivWithin w (Set.Icc (0:ℝ) 1) y| ≤

                t ^ (-((1 + η) / 2) : ℝ) * Cu₀)
```

## Proof sketch

The proof is a wrapper around the current small-theta consumer.

1. Define local abbreviation `Ainit` as above.
2. Prove `hAinit_nn` from:
   - `hCu₀_nonneg`
   - `ht`
   - `secondDerivSmoothingConst_nonneg`
   - `gradSmoothingConst_nonneg`
   - positivity of `Real.rpow` on positive bases.
3. Produce initial differentiability:
   ```lean
   have init_diff : Differentiable ℝ (initialValueLeg t (intervalDomainLift u₀)) :=
     fun x => (initialValueLeg_hasDerivAt ht hu₀_meas hu₀_bdd x).differentiableAt
   ```
4. Produce reaction differentiability:
   ```lean
   have react_diff : Differentiable ℝ (reactionValueLeg t L) :=
     fun x => (reactionValueLeg_hasDerivAt ht hL_meas hCL_nn hL_bdd x).differentiableAt
   ```
5. Produce initial derivative Holder. Either unfold `initialValueLeg`, or rewrite with `initialValueLeg_deriv_eq`:
   ```lean
   have init_holder : ∀ x ∈ Set.Icc (0:ℝ) 1, ∀ y ∈ Set.Icc (0:ℝ) 1,
       |deriv (initialValueLeg t (intervalDomainLift u₀)) x -
         deriv (initialValueLeg t (intervalDomainLift u₀)) y| ≤
         Ainit * |x - y| ^ η := by
     intro x hx y hy
     simpa [Ainit, initialValueLeg] using
       gradLeg_holder_global (δ := t) (η := η) ht hη0 hη1
         (f := intervalDomainLift u₀) hu₀_meas (Cf := Cu₀) hu₀_bdd x y
   ```
6. Convert the supplied reaction Holder from `reactionDerivLeg` to `deriv (reactionValueLeg t L)` by rewriting with `reactionValueLeg_deriv_eq`:
   ```lean
   have react_holder_deriv : ∀ x ∈ Set.Icc (0:ℝ) 1, ∀ y ∈ Set.Icc (0:ℝ) 1,
       |deriv (reactionValueLeg t L) x - deriv (reactionValueLeg t L) y| ≤
         Areact * |x - y| ^ η := by
     intro x hx y hy
     rw [reactionValueLeg_deriv_eq ht hL_meas hCL_nn hL_bdd x,
         reactionValueLeg_deriv_eq ht hL_meas hCL_nn hL_bdd y]
     exact react_holder x hx y hy
   ```
7. Call the current theorem:
   ```lean
   chemMild_C1eta_slice_diffOn_of_gradientMild_initialHolder_smallTheta_cutoff_components
     Dsol hη0 hη1 hθη hθ0 hθlt hH₀_nonneg hholder hplan ht htT
     init_diff react_diff hAinit_nn hAreact_nn w_split init_holder
     react_holder_deriv hNeumann
   ```

This should be a small 0-sorry commit.

## Semantic traps

1. **Endpoint no-flux is not implied by the current C1/η bridge package.** `DifferentiatedMildSliceDiffOn` proves regularity/Hölder of `derivWithin`; it does not encode boundary conditions.

2. **Do not prove endpoint no-flux by summing endpoint values of the three differentiated legs.** The chemotaxis second-derivative leg does not vanish at endpoints in general. Endpoint no-flux is a solution-level invariant.

3. **The restart-cosine endpoint theorem gives `deriv (intervalDomainLift (D.u t)) 0 = 0`, not immediately the current required `derivWithin w (Icc 0 1) 0 = 0`.** Bridging global `deriv`/one-sided `derivWithin` for the zero-extended interval slice is a separate boundary/representation task.

4. **Cutoff agreement must be explicit.** A theorem using `chemFluxCthetaCutoffSource` must not silently replace it by `chemFluxLifted`; it needs an a.e. equality on `0 < s ≤ t ≤ Dsol.T`.

5. **Reaction Holder is not yet automatic.** `reactionValueLeg_hasDerivAt` discharges differentiability, but the `η`-Holder bound of `reactionDerivLeg` requires a separate Duhamel-gradient Holder estimate. Keep `react_holder` explicit in Task196.
