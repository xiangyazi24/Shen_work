# KubertBridgeN16: exact invariants, the full-2-torsion square condition, and the route to `E_N16`

## Executive verdict

The three canonical functions in the Tate-normal-form normalization are

\[
\boxed{\begin{aligned}
\Phi_{16}(b,c)={}&-bc^{12}
 +(b-c)(4b+c)c^{10}
 -10b(b-c)^2c^8\\
&+b(b-c)^3(b^2+3bc+6c^2)c^4\\
&-b(b-c)^4(4b^2-2bc+3c^2)c^2\\
&+b(b-c)^5(2b^2-2bc+c^2),
\end{aligned}}
\]

\[
\boxed{
\operatorname{tateM16}(b,c)
 =16b^2-8bc^2-20bc+b+c(c-1)^3,
}
\]

and

\[
\boxed{
\operatorname{discQ16bc}(b,c)
 =b\,\operatorname{tateM16}(b,c).
}
\]

For

\[
E_{b,c}: y^2+(1-c)xy-by=x^3-bx^2,
\]

the Weierstrass discriminant is

\[
\boxed{
\Delta(E_{b,c})=b^3\operatorname{tateM16}(b,c).
}
\]

Thus `discQ16bc` is exactly the square-class reduction of the curve discriminant: the omitted factor is the square `b^2`.

The analysis proposed in the question for sorry #1 is correct. The order-16 point supplies Tate parameters satisfying `Phi16 = 0`. The independent rational 2-torsion point forces the residual quadratic factor of the completed-square 2-division cubic to split. Its discriminant is therefore a rational square. The standard cubic-discriminant identity then converts that square into

\[
\eta^2=b\,\operatorname{tateM16}(b,c).
\]

There is one important correction for sorry #2: the full-level curve is not birational to the elliptic obstruction curve merely because the latter receives a rational point. Adding a choice of independent 2-torsion gives a finite cover of the genus-two curve `X_1(16)`; the map to

\[
E_{N16}: w^2=u^3-u^2-u
\]

is a **quotient map**, not generally a birational equivalence of the entire full-level curve. Once the quotient formulas are fixed, their verification is `field_simp`/`ring`. Deriving those formulas is an elimination/model-conversion computation.

The most robust Lean implementation is therefore:

```text
Tate extraction + residual-quadratic square
  -> (b,c,eta) cover datum
  -> fixed quotient / binary-quartic datum
  -> universal quartic-to-E_N16 map
  -> nondegenerate E_N16 point.
```

The exact universal last map is given below. I would not inline the entire quotient calculation in the theorem at line 361.

# 1. Exact formulas and identities

## 1.1 Tate normal form

The relevant Tate normal form is

\[
E_{b,c}:\quad y^2+(1-c)xy-by=x^3-bx^2,
\qquad P=(0,0).
\]

Its Weierstrass coefficients are

\[
a_1=1-c,\quad a_2=-b,\quad a_3=-b,\quad a_4=a_6=0.
\]

Hence

\[
\begin{aligned}
B_2&=(1-c)^2-4b,\\
B_4&=b(c-1),\\
B_6&=b^2,\\
B_8&=-b^3.
\end{aligned}
\]

A direct expansion of the standard discriminant formula yields

\[
\Delta
=-B_2^2B_8-8B_4^3-27B_6^2+9B_2B_4B_6
=b^3M_{16}(b,c),
\]

where

\[
M_{16}(b,c)=16b^2-8bc^2-20bc+b+c(c-1)^3.
\]

Lean-shaped definitions:

```lean
def tateM16 (b c : ℚ) : ℚ :=
  16 * b^2 - 8 * b * c^2 - 20 * b * c + b + c * (c - 1)^3

def discQ16bc (b c : ℚ) : ℚ :=
  b * tateM16 b c

lemma tate_discriminant_eq (b c : ℚ) :
    tateDiscriminant b c = b^3 * tateM16 b c := by
  simp [tateDiscriminant, tateM16]
  ring
```

The symbolic substitution

\[
b=rs(r-1),\qquad c=s(r-1)
\]

gives

\[
M_{16}
=s(r-1)^2 H_{16}(r,s),
\]

where

\[
\begin{aligned}
H_{16}(r,s)={}&r^2s^3-8r^2s^2-2rs^3+16r^2s+5rs^2+s^3\\
&-20rs+3s^2+3s+1,
\end{aligned}
\]

and therefore

\[
bM_{16}=rs^2(r-1)^3H_{16}(r,s).
\]

This is a useful independent normalization check.

## 1.2 The order-16 polynomial

Put

\[
r=\frac bc,
\qquad
s=\frac{c^2}{b-c}.
\]

Conversely,

\[
c=s(r-1),\qquad b=rs(r-1).
\]

The raw Kubert equation is

\[
\begin{aligned}
F_{16}(r,s)={}&r^3s^2-4r^3s+2r^3+3r^2s^2+2r^2s-2r^2\\
&-rs^5+4rs^4-10rs^3+6rs^2-3rs+r+s^4.
\end{aligned}
\]

Clearing the denominator `c^3 (b-c)^5` gives the polynomial used above:

\[
\Phi_{16}(b,c)=c^3(b-c)^5
F_{16}\!\left(\frac bc,\frac{c^2}{b-c}\right).
\]

Expanded:

```lean
def Phi16 (b c : ℚ) : ℚ :=
    -b * c^12
  + (b - c) * (4*b + c) * c^10
  - 10*b * (b - c)^2 * c^8
  + b * (b - c)^3 * (b^2 + 3*b*c + 6*c^2) * c^4
  - b * (b - c)^4 * (4*b^2 - 2*b*c + 3*c^2) * c^2
  + b * (b - c)^5 * (2*b^2 - 2*b*c + c^2)
```

The lower-level factors excluded in deriving this exact-order equation include

```text
s,
 r - 1,
 r*s - 2*r + 1.
```

The last one becomes, after clearing denominators,

\[
rs-2r+1
=
\frac{bc^2-2b^2+3bc-c^2}{c(b-c)}.
\]

Depending on how the local file packages denominator exclusions, a helper named similarly to `tateM16` might include this factor. The canonical `tateM16` occurring in the discriminant and in `discQ16bc`, however, is the polynomial `M16` displayed above.

# 2. Sorry #1: extracting `(b,c,eta)`

## 2.1 Group-theoretic input

Let

```text
ι : ZMod 2 × ZMod 16 ->+ E(ℚ)
```

be injective, and put

\[
P=\iota(0,1),\qquad T=\iota(1,0).
\]

Injectivity gives

\[
\operatorname{ord}(P)=16,
\qquad
\operatorname{ord}(T)=2,
\qquad
T\notin\langle P\rangle.
\]

In particular, `8P` is one nonzero rational 2-torsion point, while `T` and `T+8P` are the other two. They are pairwise distinct.

A generic Tate-normal-form theorem for a rational point of exact order at least four produces an isomorphism carrying `P` to `(0,0)` on `E_{b,c}`. Exact order 16 then supplies

```text
b ≠ 0,
c ≠ 0,
b - c ≠ 0,
Phi16 b c = 0,
```

plus the lower-order denominator exclusions. Nonsingularity gives

```text
tateM16 b c ≠ 0
```

because

```text
Delta(E_bc) = b^3 * tateM16 b c
```

and `b ≠ 0`.

This first part is not merely `ring`: it consumes the Tate-normal-form existence/isomorphism theorem and the exact-order-16 Kubert calculation. Once those two interfaces exist, the remaining adapter is elementary.

## 2.2 Completed-square 2-division cubic

Make the standard characteristic-not-two substitution

\[
Y=2y+a_1x+a_3=2y+(1-c)x-b.
\]

Then

\[
Y^2=f_2(x),
\]

with

\[
\boxed{
f_2(X)=4X^3+B_2X^2+2B_4X+B_6.
}
\]

The nonzero 2-torsion points are exactly the points `(x,0)` where `x` is a root of `f2`.

Let

\[
\alpha=x(8P),\quad
\beta=x(T),\quad
\gamma=x(T+8P).
\]

All three are rational and distinct. Thus

\[
f_2(X)=4(X-\alpha)(X-\beta)(X-\gamma).
\]

If one wants `alpha` purely in Kubert's raw parameters, it is

\[
\boxed{
\alpha=
\frac{r(r-1)(r-s)(r-s^2+s-1)}{(rs-2r+1)^2}.
}
\]

The residual quadratic after removing the known root `alpha` is

\[
\boxed{
Q_{16}(X)=
4X^2+(B_2+4\alpha)X+
(2B_4+\alpha B_2+4\alpha^2).
}
\]

Equivalently,

\[
Q_{16}(X)=4(X-\beta)(X-\gamma).
\]

Its discriminant is therefore the explicit square

\[
\operatorname{disc}(Q_{16})=16(\beta-\gamma)^2.
\]

Set

\[
\xi=4(\beta-\gamma).
\]

Then `xi^2 = disc(Q16)`.

## 2.3 Converting the quadratic discriminant to `discQ16bc`

Let

\[
q_\alpha=Q_{16}(\alpha)=f_2'(\alpha)
=12\alpha^2+2B_2\alpha+2B_4.
\]

For a cubic factored as `(X-alpha) Q16(X)`, the standard discriminant identity is

\[
\operatorname{disc}(f_2)
=q_\alpha^2\operatorname{disc}(Q_{16}).
\]

For the completed-square cubic of a Weierstrass equation,

\[
\operatorname{disc}(f_2)=16\Delta(E_{b,c}).
\]

Consequently

\[
q_\alpha^2\xi^2
=16b^3M_{16}(b,c).
\]

Define

\[
\boxed{
\eta=\frac{q_\alpha\xi}{4b}.
}
\]

Then

\[
\eta^2
=
\frac{16b^3M_{16}}{16b^2}
=bM_{16}
=\operatorname{discQ16bc}(b,c).
\]

This is the cleanest explicit derivation of the final square witness. It avoids expanding the residual discriminant as a very large rational expression in `b,c`.

Lean decomposition:

```lean
lemma completedSquare_twoDivision_cubic ... : ...

lemma residualQuadratic_factor
    (halpha : f2.eval alpha = 0) :
    f2 = (X - C alpha) * Q16 alpha := by
  ext n <;> simp [f2, Q16]
  ...

lemma residualQuadratic_disc_square
    (hbeta : f2.eval beta = 0)
    (hgamma : f2.eval gamma = 0)
    (hpairwise : ...) :
    ∃ xi : ℚ, xi^2 = discrimQ16 alpha := by
  refine ⟨4 * (beta - gamma), ?_⟩
  ...

lemma cubic_disc_eq_residual_disc ... :
    cubicDiscriminant f2 = (f2.derivative.eval alpha)^2 * discrimQ16 alpha := by
  ...

lemma eta_square_eq_discQ16bc ... :
    let eta := f2.derivative.eval alpha * xi / (4*b)
    eta^2 = discQ16bc b c := by
  dsimp
  field_simp [hb]
  nlinarith [hcubicDisc, hxi]
```

The final `kubert_C16_discriminant_data` should then be mostly orchestration.

# 3. Sorry #2: mapping the full-level cover to `E_N16`

## 3.1 Geometry: quotient, not global birationality

`X_1(16)` has genus two. Specifying an additional independent rational 2-torsion point gives a finite cover. A nonconstant map from that cover to a genus-one obstruction curve is natural, but it does not imply that the entire cover is birational to the elliptic curve.

Accordingly, the correct formal object is a rational quotient map on the nondegenerate affine chart:

```lean
(b,c,eta) |-> (u,w)
```

with

```lean
Phi16 b c = 0,
eta^2 = discQ16bc b c,
```

not an asserted two-sided birational equivalence unless one has separately checked the genera and degrees.

## 3.2 A fully explicit staged construction of the cover

The following coordinate chain is useful for generating and checking the quotient formulas.

First set

\[
r=\frac bc,
\qquad
s=\frac{c^2}{b-c}.
\]

On the nondegenerate chart put

\[
a=r-1+s-s^2,
\]

\[
\boxed{
x=\frac{(1-s)(r-s)}{a},
\qquad
y=(1-s)x-s.
}
\]

Then

\[
y^2+(x^3+x^2-x+1)y+x^2=0.
\]

Complete the square:

\[
z=2y+x^3+x^2-x+1.
\]

This gives

\[
z^2=(x^2-1)(x^2+1)(x^2+2x-1).
\]

Set

\[
\boxed{
U=\frac{1-x}{1+x},
\qquad
V=\frac{2z}{(1+x)^3}.
}
\]

Then

\[
\boxed{
V^2=U(U^2+1)(U^2+2U-1).
}
\]

Now regard

\[
D=bM_{16}(b,c)
\]

as an element of the quadratic function field `Q(U,V)`. Write

\[
D=A(U)+B(U)V
\]

and let conjugation send `V` to `-V`. A direct exact calculation gives

\[
D\bar D=S(U)^2,
\]

where

\[
\boxed{
S(U)=
\frac{U^3(U-1)^8(U^2-2U-1)(U^2+1)}
{(U+1)^7(U^2+2U-1)^3}.
}
\]

Since `eta^2 = D` and the hypotheses imply `eta ≠ 0`, define

\[
\bar\eta=\frac{S(U)}{\eta}.
\]

Then

\[
\bar\eta^2=\bar D.
\]

Therefore

\[
Z_+=\eta+\bar\eta,
\qquad
Z_-=\eta-\bar\eta
\]

satisfy

\[
\boxed{
Z_+^2=2(A+S),
\qquad
Z_-^2=2(A-S).
}
\]

Factoring either right-hand side and stripping rational squares exposes the genus-one quotient. This is a reliable CAS-to-Lean route: all identities above are exact rational identities, and every verification can be frozen as a `field_simp; ring` certificate.

## 3.3 The universal binary-quartic-to-`E_N16` map

The most reusable endpoint is the following. Suppose one has

\[
\boxed{
dZ^2=d^2X^4-dX^2B^2-B^4,
}
\]

with `d ≠ 0`, `B ≠ 0`. Define

\[
\boxed{
u=d\frac{X^2}{B^2},
\qquad
w=d\frac{XZ}{B^3}.
}
\]

Then

\[
\begin{aligned}
w^2
&=d^2\frac{X^2Z^2}{B^6}\\
&=d^3\frac{X^6}{B^6}
  -d^2\frac{X^4}{B^4}
  -d\frac{X^2}{B^2}\\
&=u^3-u^2-u.
\end{aligned}
\]

This is exactly the desired affine equation.

Lean-shaped theorem:

```lean
lemma EN16_point_of_binaryQuartic
    {d X B Z : ℚ}
    (hd : d ≠ 0) (hB : B ≠ 0)
    (hquartic : d * Z^2 = d^2 * X^4 - d * X^2 * B^2 - B^4) :
    let u := d * X^2 / B^2
    let w := d * X * Z / B^3
    E_N16_AffineEquation u w := by
  dsimp [E_N16_AffineEquation]
  field_simp [hB]
  nlinarith [hquartic]
```

If the equation proof is not accepted by `nlinarith` after denominator clearing, use

```lean
linear_combination d * X^2 * hquartic
```

followed by `ring_nf`.

The nondegeneracy proof is separate. If the local definition is simply

```lean
E_N16_DegenerateParameter u := u = 0
```

then `hd`, `hB`, and `hX : X ≠ 0` give it immediately. If the predicate excludes more values, factor each equation `u = u_bad` after substituting the quotient formula and discharge it using the stored nonzero denominator/cusp factors.

## 3.4 What is still needed for the exact theorem at line 361

To turn the preceding endpoint into the exact current theorem, one needs the fixed rational formulas

```text
(b,c,eta) -> (d,X,B,Z)
```

or the already-composed formulas

```text
(b,c,eta) -> (u,w).
```

Those formulas depend on which quotient branch and which normalization the local file chose. They cannot be recovered uniquely from the theorem statement alone: multiplying `eta` by a rational square factor, changing the model of `X_1(16)`, or composing with an automorphism of `E_N16` changes the displayed rational functions while preserving the theorem.

I therefore recommend adding the formulas as named definitions and proving two small certificate lemmas, rather than hiding them inside `EN16_point_of_Phi16_and_disc`:

```lean
def N16QuotientU (b c eta : ℚ) : ℚ := ...
def N16QuotientW (b c eta : ℚ) : ℚ := ...

lemma N16Quotient_denominators
    (hPhi : Phi16 b c = 0)
    (heta : eta^2 = discQ16bc b c)
    (hb : b ≠ 0) (hc : c ≠ 0) (hM : tateM16 b c ≠ 0) :
    quotientDenominator b c eta ≠ 0 := by
  ...

lemma N16Quotient_equation_numerator :
    quotientDenominator b c eta ^ 2 *
      (N16QuotientW b c eta ^ 2
       - N16QuotientU b c eta ^ 3
       + N16QuotientU b c eta ^ 2
       + N16QuotientU b c eta)
      = A b c eta * Phi16 b c
        + B b c eta * (eta^2 - discQ16bc b c) := by
  ring
```

The second identity is the ideal Lean certificate. It reduces the curve equation to the two hypotheses by `rw [hPhi, heta]; ring`. It also makes normalization mistakes immediately visible.

# 4. Are deeper number-theoretic results required?

## Sorry #1

The theorem is elementary/modular algebra, but not only polynomial normalization.

It requires:

1. extracting an exact-order-16 point and an independent order-2 point from the injection;
2. a Tate-normal-form isomorphism carrying the order-16 point to `(0,0)`;
3. the exact-order-16 Kubert polynomial calculation;
4. elementary 2-division factorization and discriminant algebra.

There is no Mordell-Weil, descent, modularity, or Diophantine finiteness theorem here. The nontrivial infrastructure is the group-law/Tate-normal-form bridge.

## Sorry #2

Finding a good quotient formula is an algebraic-geometry/elimination computation. Once the formula is fixed, proving that it satisfies the equation is purely algebraic:

```text
unfold definitions
field_simp [denominator_nonzero]
ring / ring_nf / linear_combination
```

The later theorem that `E_N16(ℚ)` has no nondegenerate rational points is the genuinely arithmetic part. It should remain downstream and separate from `KubertBridgeN16`.

```text

   - discriminant_tate16

   - P exact order 16

   - T, 8P, T+8P pairwise distinct

   - Q16 factorization

   - nondegenerate parameter
```

The hardest formal lemma is likely **Tate-normal-form extraction with a controlled isomorphism and exact-order data**, unless that is already banked. The quotient-equation check is longer algebraically but conceptually mechanical.

## Bottom line

- The proposed discriminant analysis for sorry #1 is correct.
- `tateM16` is the discriminant cofactor, and `discQ16bc = b*tateM16`.
- `Phi16` is the displayed denominator-free Kubert polynomial.
- The square witness can be written explicitly as
  \(
  \eta=f_2'(\alpha)\,4(\beta-\gamma)/(4b)
  \).
- The map to `E_N16` is a quotient calculation, not automatically a birational equivalence.
- Once reduced to the binary quartic, the explicit map is
  \(
  u=dX^2/B^2,
  w=dXZ/B^3
  \), and its verification is a short `field_simp`/`ring` proof.
- No deep number theory is required in either bridge theorem; the deep arithmetic belongs to the downstream rational-point exclusion on `E_N16`.
