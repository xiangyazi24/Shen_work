# Q3799 — minimum-cost chi-negative closure strategy

## Short answer

The strict-negative branch is blocked exactly where the current tree says it is blocked: local existence for the coupled flux system. The already-proved Lemma 3.1/max-principle machinery is enough for continuation **after** a local classical solution exists; it does not create the first local solution.

The minimum-cost route is **not** another headline alias and is **not** a reduction to `χ₀ = 0`. The best target is to close the local-existence residual directly:

```lean
ShenWork.Paper2.ChiNegResidual.CoupledFluxClassicalLocalExistenceResidual p
```

or, one level below it, the currently narrowest resolver-facing producer:

```lean
ShenWork.Paper2.ChiNegResidual.CoupledFluxResolverReducedCoreData p
```

The conjugate B-form route is still viable as a research path, but it is not the cheapest way to close the theorem unless we already have the missing weak-PID B-form local package. The old gradient-to-B-form bridge is explicitly obstructed by an endpoint theorem, so that subroute should be abandoned rather than repaired.

## (2) Can `χ₀ < 0` be reduced to `χ₀ = 0`?

No, not in a theorem-producing way.

The repulsive sign helps at the **a priori estimate** stage. At a spatial maximum of `u`, the chemotaxis term has the sign needed for Lemma 3.1, and that is exactly why the continuation/max-bound side is already done. But that argument starts with an existing classical solution.

It does not create the local solution. It also does not identify the `χ₀ < 0` solution with a `χ₀ = 0` logistic solution. The term

```lean
-p.χ₀ * intervalDomain.chemotaxisDiv p (u t) (v t) x
```

is a divergence of a nonlinear elliptic functional of `u`; it is not a pointwise nonnegative forcing and is not monotone-comparable to the logistic equation as an evolution operator.

A transformation such as the conjugate/B-form route can rewrite the drift structure, but it does not erase the local theory. It replaces it with source-regularity, weak-form, negative-part, and strict-positivity obligations. The current files confirm this: `PositiveDatumBFormLocalComponents` still asks for `Hpde`, `HmildWeak`, `Henergy`, `heat_lower_barrier`, `regularity`, `hpde_v`, and `neumann` (`IntervalBFormPositiveDatumLocalExistence.lean:32-69`).

So the safe statement is:

* `χ₀ = 0` can supply reusable heat/Picard/tower technology.
* `χ₀ < 0` cannot be reduced to `χ₀ = 0` by comparison or transformation.
* The only reduction already valid is the one in the repo: **local existence + Lemma 3.1 continuation ⇒ global theorem**.

## Viability verdict

### Conjugate B-form route

Viable only if kept as a **direct conjugate/B-form** construction. Do not use `hgradB`; that route is formally diagnosed as false at endpoints.

Cost profile:

* still needs weak-PID lower barrier or square-heat comparison;
* still needs mild-to-weak semigroup identity for the negative-part energy test;
* still needs positive-window source regularity for the conjugate limit;
* PPID squared-route results cannot be copied to weak PID because they use `paperPositiveFloor`.

So it is viable, but expensive.

### Cheaper route

Prove a sign-independent local classical existence theorem for the coupled system, then use the already-proved global continuation machinery.

This is mathematically the standard paper route: local classical existence by standard parabolic/elliptic theory; global extension by Lemma 3.1 and the logistic bound. In the Lean repo it means closing `CoupledFluxClassicalLocalExistenceResidual` or `CoupledFluxResolverReducedCoreData`, not making more B-form aliases.

### Reduction to `χ₀ = 0`

Not viable. The `χ₀ = 0` theorem solves a different PDE. The repulsive sign gives a maximum-principle inequality after a solution exists; it does not provide a solution, an operator comparison, or a transform from negative-chi dynamics to logistic dynamics.

## Final ranked recommendation

1. Main target: prove `CoupledFluxClassicalLocalExistenceResidual p` directly, preferably via a standard local parabolic theorem or via `CoupledFluxResolverReducedCoreData p` with a stronger regularity norm.
2. Do not try to prove divergence Lipschitz on the old sup-only ball; the source documents that as the wrong norm.
3. If continuing B-form, first prove weak-PID square/heat seed and then the weak lower comparison; this is the first real weak-PID floor.
4. Prove the semigroup weak Duhamel identity for the negative-part test class; do not bridge through arbitrary `φ : ℝ → ℝ` unless actually proved.
5. Replace global `lo = 0` source TimeC1On demands by positive-window source regularity consumed directly by `HasBFormSpectralPdeAgreement`.

No unconditional negative-chi headline is supported until one of these real local-existence floors is discharged.
