# 4. Recommended ODE construction in logarithmic coordinates

Construct the ODE in log variables first.  This makes positivity automatic and makes the Lyapunov function linear.

```text
a(t) := log Ū(t),
b(t) := log U̲(t).
```

The autonomous vector field on all of `ℝ × ℝ` is


## 4.1 Forward invariant region

Starting from `b(0) < 0 < a(0)`, prove

```text
b(t) ≤ 0 ≤ a(t),
```

hence

```text
0 < U̲(t) ≤ 1 ≤ Ū(t).
```

The boundary signs are exact:

```text
at Ū = 1:   Ū' = χ (1 - U̲^γ) ≥ 0,
at U̲ = 1:   U̲' = χ (1 - Ū^γ) ≤ 0.
```

In log coordinates these are the signs of `a'` at `a=0` and `b'` at `b=0`.  A first-exit argument gives the closed invariant quadrant `b≤0≤a`.  If strict inequalities are desired, uniqueness of the smooth autonomous ODE rules out reaching `(a,b)=(0,0)` at finite time unless the entire orbit is the equilibrium orbit.

## 4.2 Global continuation

Set `β=m+γ-1`.  On the invariant quadrant and under `β≤α`, the upper log equation satisfies

```text
a'
  ≤ χ Ū^β + 1 - Ū^α
  ≤ 1 - (1-χ) Ū^α.
```

Since `χ<1`, this prevents `a` from crossing above

```text
Aχ := (1/α) log(1/(1-χ))
```

once it is above that threshold.  Thus

```text
a(t) ≤ max(a(0), Aχ).
```

With `a` bounded above and `b≤0`,

```text
b' ≥ -χ exp(γ * max(a(0), Aχ)).
```

Consequently `b` is bounded below on every finite time interval.  The log orbit remains in a compact rectangle on every finite horizon, so local Picard--Lindelöf solutions continue globally forward.  This is the only new ODE-existence atom; it is standard finite-dimensional continuation, not a PDE estimate.

## 5.1 Critical-form calculation

Under `α=m+γ-1`, the merged equations give the equivalent exact formula

```text
W'
 = -(1-χ)(Ū^α-U̲^α)
   + χ(U̲^(m-1) Ū^γ - Ū^(m-1) U̲^γ).
```

Moreover

```text
(Ū^α-U̲^α)
  - (U̲^(m-1) Ū^γ - Ū^(m-1) U̲^γ)
= (Ū^(m-1)-U̲^(m-1)) (Ū^γ+U̲^γ)
≥ 0.
```

Therefore

```text
W' ≤ -(1-χ)(Ū^α-U̲^α) + χ(Ū^α-U̲^α)

```

This is the shortest proof in the equality branch.




# 7. Why both endpoints converge to `1`

The invariant rectangle and definition of `W` give the sharper sandwich

```text
exp(-W(t)) ≤ U̲(t) ≤ 1 ≤ Ū(t) ≤ exp(W(t)).
```

Indeed,

```text
Ū/U̲ = exp W,
Ū ≤ exp W          because U̲≤1,
U̲ ≥ exp(-W)        because Ū≥1.
```

Since

```text
W(t) → 0,
```

continuity of `exp` and two squeeze arguments give

```text
U̲(t) → 1,
Ū(t) → 1.
```

No separate monotonicity of `U̲` or `Ū` is required; in fact the coupled endpoints need not be individually monotone.  The straddling invariant `U̲≤1≤Ū` is what pins the vanishing-width rectangle to the unique positive diagonal equilibrium `(1,1)`.

One can also export an explicit uniform rate.  Since `0≤W(t)≤W(0)`,

```text
Ū(t)-1 ≤ exp(W(0)) W(t),
1-U̲(t) ≤ W(t),
```

so the eventual PDE error is bounded by a constant multiple of `exp(-δt)`.

## 8.1 Exact contact inequalities

Expand the PDE using `v_xx=v-u^γ`:

```text
u_t
 = u_xx
   - χ m u^(m-1) u_x v_x
   + χ u^m (u^γ-v)
   + u(1-u^α).
```

Before a first rectangle exit, resolver order gives

```text
U̲^γ ≤ v ≤ Ū^γ.
```

Define time-dependent scalar reactions

```text
gLower(t,r) := χ r^m (r^γ-Ū(t)^γ) + r(1-r^α),
gUpper(t,r) := χ r^m (r^γ-U̲(t)^γ) + r(1-r^α).
```

Then

```text
u_t ≥ u_xx - b(t,x) u_x + gLower(t,u),
u_t ≤ u_xx - b(t,x) u_x + gUpper(t,u),
```

where

```text
b(t,x) := χ m u(t,x)^(m-1) v_x(t,x).
```

The ODE equations are exactly

```text
U̲' = gLower(t,U̲),
Ū' = gUpper(t,Ū).
```

On every finite slab, `b` is bounded and both scalar reactions are uniformly locally Lipschitz on the common bounded range.  For the upper difference `w=u-Ū`, at positive points,

```text
w_t ≤ w_xx + Kb |w_x| + C w.
```

For the lower difference `w=U̲-u`, the same inequality holds.  Multiplying by `exp(-Ct)` removes `Cw`, and the existing theorem

# 9. Final squeeze to `UniformConvergesToConstant`

Once

```text
U̲(t) ≤ u(t,x) ≤ Ū(t)
```

and both endpoint limits are available, the final theorem is pure order wiring:

```lean
theorem uniformConvergesToConstant_one_of_positiveRectangle_squeeze
    {u : ℝ → ℝ → ℝ} {Ubar Ulow : ℝ → ℝ}
    (hsqueeze : ∀ t, 0 ≤ t → ∀ x, Ulow t ≤ u t x ∧ u t x ≤ Ubar t)
    (hupper : Tendsto Ubar atTop (𝓝 1))
    (hlower : Tendsto Ulow atTop (𝓝 1)) :
    UniformConvergesToConstant u 1 := by
  intro ε hε
  have hε2 : 0 < ε / 2 := by positivity
  have hU := (tendsto_order.1 hupper).2 (1 + ε) (by linarith)
  have hL := (tendsto_order.1 hlower).1 (1 - ε) (by linarith)
  rcases eventually_atTop.1 (hU.and hL) with ⟨T, hT⟩
  refine ⟨max T 0, ?_⟩
  intro t x ht
  have htT : T ≤ t := (le_max_left T 0).trans ht
  have ht0 : 0 ≤ t := (le_max_right T 0).trans ht
  have hs := hsqueeze t ht0 x
```
