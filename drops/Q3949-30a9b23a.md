# Q3949: Picard-level gradient bounds for the truncated 1D Keller--Segel source

## Executive answer

The logarithmic divergence is real for the endpoint estimate that differentiates the divergence forcing all the way onto the heat kernel. It is not fixed by the Neumann kernel. At high frequency the Neumann spectrum still has `lambda_k = (k*pi)^2`, so the mixed second derivative has the same endpoint singularity as on the line, up to harmless boundary/reflection constants.

The usual way out is not to prove

`uniform L^infty ball -> uniform O(1) chemDiv coefficients -> gradient ell^1`

in one step. That route is circular, and even the middle statement is too weak for the final `ell^1` gradient estimate. The correct Picard-level proof is a time-weighted bootstrapping/fixed-point estimate:

1. Put the Picard iterates in a ball that contains a positive-time Sobolev norm, for example
   `sup_{0 < t <= T} sqrt(t) ||u_n(t)||_{H^1} <= K`,
   in addition to the uniform `L^infty` bound `||u_n||_{L^infty} <= M`.
2. Use the previous iterate `u_n`, not the same iterate `u_{n+1}`, to estimate the source of the next iterate.
3. Prove
   `||source_n(t)||_{L^2} <= C(M) (1 + ||u_n(t)||_{H^1})`.
4. Use heat smoothing from `L^2` source to `H^1` for `u_{n+1}` with an integrable kernel `(t-s)^(-1/2)`.
5. Close the weighted `H^1` bound by a Volterra/Gronwall or small-time self-map argument.
6. Once the weighted `H^1` bound is uniform in `n`, the source coefficients are uniformly `ell^2` on positive time slabs. Then the spectral gradient bound follows from the one-dimensional smoothing estimate
   `ell^2 source -> sum_k k*pi |c_k(t)| < infinity`,
   with kernel `(t-s)^(-3/4)`, which is integrable.

So the answer to the circularity question is: the ladder does not give a source-independent bound. The source constants depend on the previous iterate's Sobolev norm. Uniformity comes from closing a Picard majorant with a one-step lag. On each short time interval the coefficient multiplying the previous-iterate norm is made small by the time weight. After that, continuation is standard because the truncation/logistic `L^infty` ball supplies the non-Sobolev bounds uniformly.

## 1. Why the naive second-kernel-derivative estimate fails

For the chemotaxis term in the mild formula,

`e^{(t-s)Delta} partial_x F_n(s)`,

taking one more spatial derivative of the mild solution gives an operator of the form

`partial_x e^{(t-s)Delta} partial_x F_n(s)`.

If one moves the derivative from `F_n` onto the kernel, then the relevant kernel has size like

`||partial_x partial_y K(t-s)|| ~ (t-s)^(-1)`

in endpoint norms. Therefore

`int_0^t (t-s)^(-1) ds`

diverges. This is the standard analytic-semigroup endpoint obstruction: the map

`F in L^infty -> partial_x e^{r Delta} partial_x F in L^infty`

has order `r^(-1)`, which is not time-integrable at `r = 0`.

The Neumann kernel does not remove this obstruction. Spectrally, the high-frequency modes are still governed by

`lambda_k = (k*pi)^2`,

so the same high-frequency singularity is present. Boundary terms are not the main issue here. In the Keller--Segel flux

`F_n = (u_n)^+ v_{n,x} / (1+R)^beta`,

one typically has `v_{n,x} = 0` at the endpoints from the Neumann condition for `v_n`, hence `F_n = 0` at the endpoints, so the boundary contribution in the integration-by-parts formula vanishes. The problem is the non-integrable endpoint singularity of the second derivative kernel.

Splitting the Duhamel integral at `t/2` helps only for the old part `[0,t/2]`, where high frequencies are exponentially damped by `exp(-lambda_k t/2)`. It does not by itself solve the near-endpoint interval `[t/2,t]`. The near-endpoint interval must be handled in a weaker source norm or with one derivative kept on the previous iterate's flux.

## 2. The important distinction: coefficientwise `O(1)` is not enough for gradient `ell^1`

Let

`A = -Delta_N`, `lambda_k = (k*pi)^2`, and `sqrt(lambda_k) = k*pi` for `k >= 1`.

For a Duhamel coefficient

`d_k(t) = int_0^t exp(-lambda_k (t-s)) g_k(s) ds`,

the gradient Fourier-algebra norm is

`sum_{k >= 1} sqrt(lambda_k) |d_k(t)|`.

If all one knows is the coefficientwise bound

`|g_k(s)| <= C`,

then the crude eigenvalue-gain estimate gives

`sqrt(lambda_k) |d_k(t)| <= sqrt(lambda_k) C / lambda_k = C / sqrt(lambda_k) ~ C/k`,

and

`sum_k 1/k`

diverges. Thus a uniform `O(1)` bound on the individual source coefficients is not enough to prove absolute convergence of the differentiated Fourier series.

The missing structure is an `ell^2` source bound. If `g(s) = (g_k(s))_{k >= 1}` is uniformly in `ell^2`, then one must sum before throwing away the heat factor:

`sum_k sqrt(lambda_k) |int_0^t exp(-lambda_k (t-s)) g_k(s) ds|`

`<= int_0^t (sum_k lambda_k exp(-2 lambda_k (t-s)))^(1/2) ||g(s)||_{ell^2} ds`.

In one dimension,

`(sum_{k >= 1} lambda_k exp(-2 lambda_k r))^(1/2) <= C r^(-3/4)`.

Since

`int_0^t r^(-3/4) dr < infinity`,

an `ell^2` source gives a finite gradient `ell^1` norm. This is the spectral version of the standard heat estimate

`||partial_x e^{r Delta} g||_{L^infty} <= C r^(-3/4) ||g||_{L^2}`

in one spatial dimension.

This is the key correction to the divergent computation: do not first replace `int exp(-lambda_k r) dr` by `1/lambda_k` mode-by-mode and then sum. That discards the high-frequency Gaussian cutoff near `r = 0`. The correct one-dimensional estimate keeps the heat factor until after the Cauchy--Schwarz summation.

## 3. Source estimates for the truncated Keller--Segel flux

Assume the Picard sequence is already in the uniform `L^infty` ball

`||u_n(t)||_{L^infty(0,1)} <= M`.

Let `v_n` solve

`-v_{n,xx} + v_n = u_n`,

with Neumann boundary conditions. Standard one-dimensional elliptic estimates give constants depending only on the interval such that

`||v_n||_{L^infty} + ||v_{n,x}||_{L^infty} + ||v_{n,xx}||_{L^infty} <= C_E M`.

Indeed, `v_{n,xx} = v_n - u_n`, and the Neumann resolvent has a positive bounded Green kernel on `[0,1]`.

For

`F_n = (u_n)^+ v_{n,x} / (1+R)^beta`,

where `(1+R)^(-beta)` is harmless as a fixed truncation factor, the derivative is, a.e.,

`partial_x F_n = (1+R)^(-beta) [1_{u_n > 0} u_{n,x} v_{n,x} + (u_n)^+ v_{n,xx}]`.

Thus, whenever `u_n(t) in H^1`,

`||partial_x F_n(t)||_{L^2}`

`<= C(M,R,beta) (||u_{n,x}(t)||_{L^2} ||v_{n,x}(t)||_{L^infty} + ||u_n(t)||_{L^infty} ||v_{n,xx}(t)||_{L^2})`

`<= C_F(M,R,beta) (1 + ||u_n(t)||_{H^1})`.

The positive-part map is Lipschitz and maps `H^1` to `H^1`; its weak derivative is `1_{u>0} u_x`. If the formal development uses a smooth truncation instead of the literal positive part, the same estimate is easier because the chain rule is classical.

For the full source

`g_n = f(u_n) - chi_0 partial_x F_n`,

the logistic part satisfies

`||f(u_n(t))||_{L^2} <= C_f(M)`,

so the source satisfies the crucial estimate

`||g_n(t)||_{L^2} <= C_S(M,R,beta,chi_0,f) (1 + ||u_n(t)||_{H^1})`.

In Fourier language this is

`||(g_{n,k}(t))_k||_{ell^2} <= C_S (1 + ||u_n(t)||_{H^1})`.

This is the estimate to formalize. A mere pointwise-in-`k` bound is not the right invariant for the gradient `ell^1` step.

## 4. The uniform Picard `H^1` majorant

Define the positive-time weighted Sobolev size

`P_n(T) = sup_{0 < t <= T} sqrt(t) ||u_n(t)||_{H^1}`.

For the homogeneous heat part, assuming only `u_0 in L^2`,

`||e^{t Delta} u_0||_{H^1} <= C t^(-1/2) ||u_0||_{L^2}`.

For the Duhamel part with `L^2` source,

`||int_0^t e^{(t-s)Delta} g_n(s) ds||_{H^1}`

`<= C int_0^t (t-s)^(-1/2) ||g_n(s)||_{L^2} ds`.

Using the source estimate from the previous section and the definition of `P_n(T)`,

`||g_n(s)||_{L^2} <= C_S (1 + P_n(T) s^(-1/2))`.

Therefore

`||u_{n+1}(t)||_{H^1}`

`<= C_0 t^(-1/2) + C_S int_0^t (t-s)^(-1/2) ds`

`   + C_S P_n(T) int_0^t (t-s)^(-1/2) s^(-1/2) ds`.

The two elementary integrals are

`int_0^t (t-s)^(-1/2) ds = 2 sqrt(t)`,

and

`int_0^t (t-s)^(-1/2) s^(-1/2) ds = pi`.

Multiplying by `sqrt(t)` and taking `sup_{0 < t <= T}` gives

`P_{n+1}(T) <= C_0 + C_1 T + C_2 sqrt(T) P_n(T)`.

This is the fixed-point/Volterra closure. Choose `T > 0` so small that

`C_2 sqrt(T) <= 1/2`,

and then choose `K` such that

`K >= 2 (C_0 + C_1 T)`.

If `P_n(T) <= K`, then `P_{n+1}(T) <= K`. Thus the Picard iterates have a uniform positive-time `H^1` bound on `[0,T]`, with the expected `t^(-1/2)` singularity at the initial time.

This is where the apparent circularity is broken. We do not prove the source bound for `u_n` using the gradient of `u_{n+1}`. We estimate the source of the next equation using the previous iterate's `H^1` norm, then close a one-step majorant.

If the initial Picard seed is not already regular, it is cleaner to choose

`u_0^{Picard}(t) = e^{t Delta} u_initial`

rather than the time-independent function `u_initial`. Then `P_0(T) < infinity` follows immediately from heat smoothing. Alternatively, assume the initial datum is already `H^1` for the strong Picard iteration and then later approximate rough data.

## 5. The uniform spectral gradient bound

Once `P_n(T) <= K` is uniform, the source has a uniform weighted `ell^2` bound:

`||g_n(s)||_{ell^2} <= C_S (1 + K s^(-1/2))`.

Define the gradient Fourier-algebra size

`G_n(t) = sum_{k >= 1} k*pi |c_k^n(t)|`.

The homogeneous part is finite for every `t > 0`. For example, from `u_0 in L^2`,

`sum_{k >= 1} sqrt(lambda_k) exp(-lambda_k t) |c_k^0|`

`<= (sum_{k >= 1} lambda_k exp(-2 lambda_k t))^(1/2) ||u_0||_{L^2}`

`<= C t^(-3/4) ||u_0||_{L^2}`.

For the Duhamel part, use the `ell^2 -> gradient ell^1` smoothing estimate:

`sum_{k >= 1} sqrt(lambda_k) |int_0^t exp(-lambda_k (t-s)) g_{n,k}(s) ds|`

`<= C int_0^t (t-s)^(-3/4) ||g_n(s)||_{ell^2} ds`.

Hence

`G_{n+1}(t)`

`<= C t^(-3/4) ||u_0||_{L^2}`

`   + C int_0^t (t-s)^(-3/4) [1 + K s^(-1/2)] ds`.

The two integrals are finite:

`int_0^t (t-s)^(-3/4) ds = 4 t^(1/4)`,

and

`int_0^t (t-s)^(-3/4) s^(-1/2) ds = B(1/2,1/4) t^(-1/4)`.

Thus, for every fixed `t > 0`,

`G_{n+1}(t) <= C(t,M,R,beta,chi_0,f,u_0,K)`,

uniformly in `n`. On a positive slab `[tau,T]`, this becomes

`sup_n sup_{tau <= t <= T} G_n(t) <= C(tau,T,M,R,beta,chi_0,f,u_0,K)`.

This is the spectral version of the usual one-dimensional parabolic regularization. The exponent `3/4` is integrable precisely because the spatial dimension is one. In dimension `d`, the corresponding `L^2 -> W^{1,infty}` heat singularity is `r^{-(1/2+d/4)}`, which is integrable near zero only when `d < 2`. This is one reason the 1D problem is much friendlier.

## 6. Where the `O(1)` chemDiv coefficient bound enters

After the uniform `H^1` estimate has been closed, one can also recover the coefficientwise `O(1)` chemDiv bound on every positive time slab.

For `s in [tau,T]`, the uniform weighted `H^1` estimate gives

`||u_n(s)||_{H^1} <= K / sqrt(tau)`.

Then

`||partial_x F_n(s)||_{L^1} <= ||partial_x F_n(s)||_{L^2} <= C(tau,M,R,beta,K)`.

Since `F_n(0)=F_n(1)=0` because `v_{n,x}=0` at the endpoints, integration by parts gives, for the chemotactic divergence coefficient,

`|chemDiv_{n,k}(s)|`

`= chi_0 k*pi |int_0^1 F_n(s,x) sin(k*pi x) dx|`

`= chi_0 |int_0^1 partial_x F_n(s,x) cos(k*pi x) dx|`

`<= chi_0 ||partial_x F_n(s)||_{L^1}`

`<= C(tau,M,R,beta,chi_0,K)`.

So the answer to question 1 is yes, but only after the positive-time Sobolev estimate has been obtained uniformly. The `L^infty` ball alone does not give this bound, because `partial_x F_n` contains `u_{n,x}`. The non-circular order is:

`weighted H^1 Picard ball -> source ell^2 -> next weighted H^1 -> uniform source O(1) coefficients on [tau,T]`.

The coefficientwise `O(1)` estimate is useful, but it is not the estimate that proves gradient `ell^1`. The gradient `ell^1` estimate uses the stronger `ell^2` source structure and the one-dimensional heat smoothing estimate described above.

## 7. The correct iterate-level ladder

A good Lean-oriented ladder is the following.

### Rung A: elliptic estimates for `v_n`

From `||u_n||_{L^infty} <= M`, prove uniform bounds for the Neumann resolvent:

`||v_n||_{L^infty} <= C M`,

`||v_{n,x}||_{L^infty} <= C M`,

`||v_{n,xx}||_{L^infty} <= C M`.

When `u_n in H^1`, also prove the corresponding Sobolev resolvent estimates needed to control products with `u_{n,x}`.

### Rung B: flux/source `L^2` estimate

Prove

`||partial_x F(u_n,v_n)||_{L^2} <= C(M,R,beta) (1 + ||u_n||_{H^1})`.

Then

`||f(u_n) - chi_0 partial_x F(u_n,v_n)||_{L^2}`

`<= C(M,R,beta,chi_0,f) (1 + ||u_n||_{H^1})`.

In Fourier terms, this is exactly the uniform `ell^2` source estimate.

### Rung C: weighted `H^1` self-map

Use the smoothing estimate

`||A^(1/2) int_0^t e^{-(t-s)A} g(s) ds||_{ell^2}`

`<= C int_0^t (t-s)^(-1/2) ||g(s)||_{ell^2} ds`.

Close

`P_{n+1}(T) <= C_0 + C_1 T + C_2 sqrt(T) P_n(T)`.

This is the main anti-circularity lemma.

### Rung D: source `ell^2` on positive slabs

From `P_n(T) <= K`, obtain for every `tau > 0`

`sup_n sup_{tau <= s <= T} ||source_n(s)||_{ell^2} <= C(tau,T,K,M,...)`.

### Rung E: spectral gradient `ell^1`

Use

`sum_k sqrt(lambda_k) |int_0^t exp(-lambda_k(t-s)) g_k(s) ds|`

`<= C int_0^t (t-s)^(-3/4) ||g(s)||_{ell^2} ds`.

This gives

`sup_n sup_{tau <= t <= T} sum_k k*pi |c_k^n(t)| <= C(tau,T,K,M,...)`.

This rung should not be formalized through the modewise bound

`|int exp(-lambda_k r) g_k| <= ||g_k|| / lambda_k`,

because that leads exactly to the fake harmonic divergence. Formalize the Cauchy--Schwarz heat-sum estimate instead.

## 8. Is there an algebraic fixed point `C_{n+1} <= f(C_n)`?

There can be, but the useful estimate is time-dependent rather than purely algebraic.

A crude global-in-time supremum may produce

`C_{n+1} <= a + b C_n`,

where `b` is not necessarily less than one. That is not the best way to close the argument.

The correct local estimate has a small time factor:

`P_{n+1}(T) <= C_0 + C_1 T + C_2 sqrt(T) P_n(T)`.

For `T` small, `C_2 sqrt(T) < 1`, so the Picard map sends the weighted Sobolev ball into itself. This is the fixed-point mechanism.

Equivalently, one can keep the Volterra form

`H_{n+1}(t) <= a(t) + C int_0^t (t-s)^(-1/2) H_n(s) ds`

and apply a singular Gronwall/Mittag--Leffler majorant. Both approaches express the same fact: the dependence on the previous iterate is through an integrable parabolic kernel, not through a same-time circular estimate.

For longer times, continue the argument interval by interval. On later intervals `[T_0,T_0+T]` with `T_0 > 0`, the initial data for the restarted problem already has `H^1` and gradient bounds, so the singular initial-time weight is no longer a difficulty. The truncation and logistic bounds keep the `L^infty` radius under control.

## 9. Passing the uniform bound to the Picard limit

Yes: if

`G_n(t) = sum_{k >= 1} k*pi |c_k^n(t)| <= C(t)`

uniformly in `n`, and if `u_n(t) -> u(t)` strongly in `L^2` for each fixed `t > 0`, then the limit inherits the same bound.

For each fixed Fourier mode, strong `L^2` convergence gives

`c_k^n(t) -> c_k(t)`.

Therefore, for every finite `N`,

`sum_{k=1}^N k*pi |c_k(t)|`

`= lim_{n -> infinity} sum_{k=1}^N k*pi |c_k^n(t)|`

`<= C(t)`.

Letting `N -> infinity` and using monotone convergence gives

`sum_{k >= 1} k*pi |c_k(t)| <= C(t)`.

Thus the limit has an absolutely convergent differentiated cosine series at every positive time for which the uniform bound is available. On a slab `[tau,T]`, the same argument gives

`sup_{tau <= t <= T} sum_{k >= 1} k*pi |c_k(t)| <= C(tau,T)`

provided the Picard convergence is strong enough to pass coefficients for each `t`. This is usually much weaker than the convergence already proved for the mild solution.

If one also wants convergence of the gradients, not just inheritance of the bound, then one needs an additional compactness or tail argument. But for the bound itself, lower semicontinuity of the weighted `ell^1` norm under coefficientwise convergence is enough.

## Direct answers to the three questions

### 1. Can the iterate source at positive time be `O(1)` with a uniform constant?

Yes on every positive time slab `[tau,T]`, but not from the `L^infty` ball alone. The chemotactic divergence coefficient bound

`|chemDiv_{n,k}(s)| <= C`

requires a uniform bound on `||partial_x F_n(s)||_{L^1}` or a comparable Sobolev norm. Since `partial_x F_n` contains `u_{n,x}`, this requires a uniform positive-time Sobolev estimate for `u_n`.

The non-circular proof obtains that Sobolev estimate first through the Picard one-step estimate

`previous H^1 -> source L^2 -> next H^1`,

closed in a time-weighted ball. After that, the `O(1)` coefficient bound is immediate.

### 2. Can the Sobolev ladder be run at the iterate level with uniform bounds?

Yes, if the ladder is run with the correct norms and order:

`uniform L^infty + previous weighted H^1`

`-> source_n in L^2 / ell^2`

`-> next weighted H^1 via (t-s)^(-1/2)`

`-> source_n uniformly ell^2 on positive slabs`

`-> next gradient ell^1 via (t-s)^(-3/4)`.

The constants are uniform in `n` after closing the time-weighted Picard ball. The ladder is not independent of source constants; rather, the source constants are controlled by the previous iterate's Sobolev majorant, and the majorant is closed by the integrability and small-time factors of the heat semigroup.

### 3. Does the Picard limit inherit a uniform gradient bound?

Yes. If `u_n -> u` strongly enough that each Fourier coefficient converges and

`sup_n sum_k k*pi |c_k^n(t)| <= C(t)`,

then

`sum_k k*pi |c_k(t)| <= C(t)`

by finite-part convergence followed by monotone convergence. This is the cleanest spectral passage-to-the-limit argument.

## Bottom line for the Lean formalization

Do not try to prove the gradient `ell^1` estimate from a coefficientwise `O(1)` source bound by modewise eigenvalue gain. That route necessarily produces the divergent harmonic series.

Instead, formalize these estimates:

1. `H^1` flux/source estimate:

   `||source(u)||_2 <= C(M) (1 + ||u||_{H^1})`.

2. Heat smoothing into `H^1`:

   `||A^(1/2) int e^{-(t-s)A} source(s) ds||_2 <= C int (t-s)^(-1/2) ||source(s)||_2 ds`.

3. Weighted Picard closure:

   `P_{n+1}(T) <= C_0 + C_1 T + C_2 sqrt(T) P_n(T)`.

4. One-dimensional spectral smoothing from `ell^2` source to gradient `ell^1`:

   `sum_k sqrt(lambda_k) |int e^{-lambda_k(t-s)} source_k(s) ds|`

   `<= C int (t-s)^(-3/4) ||source(s)||_{ell^2} ds`.

5. Lower semicontinuity/pass-to-limit lemma:

   coefficientwise convergence plus uniform weighted `ell^1` bound implies the same weighted `ell^1` bound for the limit.

This is the standard non-circular Picard strategy for the one-dimensional truncated chemotaxis problem in spectral form.
