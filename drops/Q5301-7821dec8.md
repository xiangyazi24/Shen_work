# What must remain global and what may vary locally

The proof must keep the following objects fixed over the entire tail:

* the solution and wave;
* the moving-frame speed `c`;
* the spatial exponential weight `η`;
* the common population bound `M`;
* the scalar coefficient `q_M(η)`;
* the physical energy
  `Ephys t := paper5WeightedEnergy η c u U t`.

The following objects may be reselected independently for each target time:

* `L`, `a`, `r`, `R`, and the finite local horizon;
* the canonical `L²` trajectories `X` and `F` used on that window;
* their representatives;
* local forcing bounds `K`, Hölder constants `H`, and gradient/modulus constants;
* the local full-generator value trajectory.

There is no compatibility problem if two overlapping windows choose different `L²` trajectory objects. At the target slice each object is proved almost everywhere equal to the same physical weighted gradient or forcing field. The derivative conclusion is about the intrinsic scalar function `paper5WeightedEnergy`, not about the auxiliary choice of representative.

A possible blow-up of a local Hölder constant as the window moves to large time is therefore harmless: the proof uses its finiteness to obtain a right derivative, but the constant is not inserted into `q_M(η)` and is never integrated over the tail.

# Tail Grönwall and the “all-time” formulation

There are two slightly different statements that should not be conflated.

## 1. Differential dissipation of the physical energy

Because the useful bound `u ≤ M` is only eventual, the negative-coefficient differential inequality is naturally a tail statement:

```text
∀ t > t₀,
  Ephys'(t) ≤ 2 q_M(η) Ephys(t),
q_M(η) < 0.
```

One should not claim that this same inequality holds before `t₀`; the solution can exceed `M` there.

Choose a finite-energy time `t₁ > t₀`. A shifted scalar argument applied to

```text
Eshift(s) := Ephys(t₁ + s)
```

gives

```text
Ephys(t) ≤ Ephys(t₁) * exp (2 q_M(η) * (t - t₁))
```

for `t ≥ t₁`. This is the exact exponential decay needed for convergence.

The existing theorem

```lean
scalarEnergy_eventual_exponential_bound_of_deriv_le
```

is organized from time zero, so the missing formal step is either:

* a translation lemma for `Eshift`; or
* a direct version on `Icc t₁ T`.

That is a scalar calculus adapter, not a new parabolic estimate.

## 2. An all-time exponential majorant for the headline interface

The formal headline bridge in `Theorem12Corrected.lean` does not require the auxiliary function `E` to be the physical energy. It requires only eventual control

```lean
∀ᶠ t in atTop, coMovingWeightedL2Energy ... t ≤ E t
```

while asking the auxiliary `E` itself to have a global scalar differential inequality.

After the tail estimate, define

```text
λ := -2 q_M(η) > 0,
A := Ephys(t₁) * exp (λ t₁),
Emajor(t) := A * exp (-λ t).
```

Then:

```text
Ephys(t) ≤ Emajor(t)            eventually,
Emajor'(t) = -λ Emajor(t)       for every t,
Emajor is smooth on all ℝ.
```

This `Emajor` directly satisfies the existing `hcontrol`, `hcont`, `hderiv`, and `hdiss` shape in

```lean
CoMovingWeightedL2Convergence.of_paper531_energy_inequality
```

without pretending that the physical energy had the negative-coefficient inequality before attraction occurred.

If the desired paper-facing statement literally asks for

```text
Ephys(t) ≤ C exp (-λt)  for every t ≥ 0,
```

rather than merely eventually, the compact initial interval can be absorbed into a larger constant `C`, provided the already-developed exact-weight `H0` propagation and local `L²` continuity are assembled on `[0,t₁]`. This is again prefix patching; it does not require extending one generator window to infinity.
