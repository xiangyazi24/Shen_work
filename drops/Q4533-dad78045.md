# Two exact endpoint checks

## Spatial endpoint

The theorem quantifies over

```lean
x : intervalDomainPoint
```

and `intervalDomainPoint` is the subtype of the closed interval `Icc 0 1`.
Thus the Codex theorem must cover `x=0` and `x=1`. If the comparison first gives
positivity only on `Ioo 0 1`, the spatial endpoint bridge must be applied before
calling the reduction theorem.

## Time endpoint

The reduction theorem requires

```lean
0 < t → t ≤ C.T
```

not merely `t < C.T`. Therefore:

- if Codex returns positivity on `(0,C.T]`, the call above is direct;
- if Codex returns only positivity on `(0,C.T)`, the field is not yet filled.

Mere continuity at `C.T` would only yield nonnegativity in the limit and does
not by itself preserve a strict positive inequality. The comparison/barrier
result must either include `t=C.T`, or provide a strictly positive terminal
barrier bound. This is a time-endpoint issue distinct from the closed spatial
boundary issue.
