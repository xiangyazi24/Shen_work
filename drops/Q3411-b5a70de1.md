# Task60 audit: physical RHS component transfer for `liftDeriv2`

## Summary verdict

The proposed Task60 theorem is good **if it is framed as a seam-narrowing transfer**, not as a proof of the physical PDE equality. It should consume:

1. strict-slab continuity of `liftTimeDeriv u`,
2. strict-slab continuity of the lifted chemotaxis-divergence component,
3. strict-slab continuity of the lifted logistic/reaction component, and
4. a strict-slab `EqOn` saying `liftDeriv2 u` agrees with
   `time + p.χ₀ * chem - reaction`.

Then it should feed Task59’s existing theorem:

```lean
H1LiftDeriv2JointContinuousBefore_of_strictSlab_eq_continuous
```

and derive both:

```lean
H1LiftDeriv2JointContinuousBefore u T
H1UxxL1ContBefore u T
```

This is a useful minimal no-sorry Task60 commit because it lowers the next physical proof obligation from “prove `H1LiftDeriv2JointContinuousBefore` somehow” to the exact PDE-RHS component continuity plus equality seam. It does **not** pretend to discharge the hard chemotaxis-divergence or endpoint equality proof.

## 1. Sign/order audit

The sign/order should be:

```lean
liftTimeDeriv u t x
  + p.χ₀ * chem
  - reaction
```

where

```lean
reaction = intervalDomainLift (u t) x *
  (p.a - p.b * (intervalDomainLift (u t) x) ^ p.α)
```

This matches the existing theorem in `ShenWork/PDE/P3MoserDxJointContinuity.lean`:

```lean
theorem intervalDomain_laplacian_u_eq_time_chem_logistic
    {params : CM2Params} {T t : ℝ}
    {u v : ℝ → intervalDomain.Point → ℝ} {x : intervalDomain.Point}
    (hsol : IsPaper2ClassicalSolution intervalDomain params T u v)
    (ht0 : 0 < t) (htT : t < T) (hx : x ∈ intervalDomain.inside) :
    intervalDomain.laplacian (u t) x =
      intervalDomain.timeDeriv u t x
        + params.χ₀ * intervalDomain.chemotaxisDiv params (u t) (v t) x
        - u t x * (params.a - params.b * (u t x) ^ params.α)
```

So the candidate’s sign is correct. In particular, do **not** replace `p.χ₀` by `-p.χ₀` in the Laplacian/RHS equality. The later H¹ energy algebra uses `(-p.χ₀)` in the cross-term bound because it rewrites the energy identity for the negative-sensitivity route, but the pointwise parabolic equation solved for `u_xx` has `+ p.χ₀ * chemotaxisDiv - reaction`.

## 2. Usefulness vs recarry

This theorem is not vacuous if it is named and documented as a transfer/composition theorem. Task59 already says a continuous strict-slab representative plus `EqOn` gives the H¹ joint-continuity package. Task60 can specialize the representative to the physical PDE RHS and prove the representative is continuous from separately supplied component continuities.

That is more useful than a raw recarry because it fixes the exact future obligations:

```text
continuous time term
continuous chemotaxis-divergence term
continuous reaction term
EqOn liftDeriv2 = physical RHS
```

The hard subgoals become independently auditable. The theorem is **not** a proof of the physical RHS theorem yet; it still carries the chem component and `EqOn`. That is the right minimal no-sorry step.

## 3. Endpoint and circularity audit

The theorem should stay strict-positive-time:

```lean
∀ {a b : ℝ}, 0 < a → a ≤ b → b < T → ... on Icc a b ×ˢ Icc 0 1
```

This does not claim continuity at `t = 0` or `t = T`.

The only endpoint-sensitive part is spatial: the `EqOn` is over `Set.Icc 0 1`, because `H1LiftDeriv2JointContinuousBefore` itself uses closed spatial slabs. That is not endpoint smuggling as long as `EqOn` remains an explicit hypothesis. Do **not** try to discharge this `EqOn` only from `intervalDomain_laplacian_u_eq_time_chem_logistic`, because that theorem requires `x ∈ intervalDomain.inside`, i.e. the interior. Endpoint equality needs an actual closed-C²/laplacian endpoint bridge or a later reformulation to an L¹/a.e. target.

No circularity is introduced by the transfer theorem itself. The circularity risk is in future proofs of the component continuities: do not prove the chemotaxis-divergence continuity using `IsPaper2BoundedBefore`, the final H¹ bound, or any route that depends on `H1UxxL1ContBefore` itself.
