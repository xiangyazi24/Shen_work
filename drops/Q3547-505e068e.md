## Existing producers and near-producers

### 1. Classical regularity gives strict-time continuity only

`ShenWork/PDE/IntervalDomain.lean`, definition `intervalDomainClassicalRegularity`, is the source of the classical solution regularity used throughout the interval route. Its relevant fields are explicitly on positive-time domains:

- solution-field continuity: `Ioo 0 T √óňĘ Icc 0 1`, not `Icc 0 b √óňĘ Icc 0 1`;
- closed-spatial `C¬≤` and endpoint Neumann values: for each `t ‚ąą Ioo 0 T`;
- time-derivative continuity: `Ioo 0 T √óňĘ Icc 0 1`.

Therefore `hsol.regularity` is a near-producer for `u_cont0`/`v_cont0` only after restriction to strict slabs `[a,b]` with `0 < a`; it does not include `t = 0`.

`ShenWork/Paper2/Statements.lean`, `IsPaper2ClassicalSolution`, also has positivity/nonnegativity only for `0 < t` and `t < T`. The accessors `IsPaper2ClassicalSolution.u_pos'`, `IsPaper2ClassicalSolution.u_pos`, and `IsPaper2ClassicalSolution.v_nonneg` do not state anything at `t = 0`.

### 2. Spatial derivative joint continuity is proved only on `(0,T) √ó [0,1]`

`ShenWork/PDE/P3MoserDxJointContinuity.lean` proves the strongest existing derivative-continuity near-producers:

- `intervalDomain_dx_u_jointlyContinuous`:
  `ContinuousOn (Function.uncurry (fun t x => deriv (intervalDomainLift (u t)) x))
  (Set.Ioo 0 T √óňĘ Set.Icc 0 1)`.
- `intervalDomain_dx_v_jointlyContinuous`:
  the analogous theorem for `v_x`.

Both are strict-time. Their proofs are built from positive-time identities and primitive-continuity transfer lemmas such as
`continuousOn_parametric_primitive_of_continuousOn_Ioo_Icc`,
`intervalDomain_dx_v_eq_reactionPrimitive`, and
`intervalDomain_dx_u_eq_time_flux_logisticPrimitive`; those pointwise equalities also require `ht : t ‚ąą Set.Ioo 0 T`.

Thus these theorems are excellent strict-slab near-producers for `ux_cont0` and `vx_cont0`, but they do not produce closed-time-at-zero continuity.

### 3. Chemotaxis representative files consume strict-time primitives; they do not close zero-start

`ShenWork/Paper2/IntervalChiNegH1ChemDivRepresentative.lean` proves:

- `liftChemotaxisDivPhysicalRep_continuousOn_strictSlab_of_classicalSolution`, requiring `0 < a`, `a ‚Č§ b`, `b < T`, and proving continuity on
  `Set.Icc a b √óňĘ Set.Icc 0 1`.
- `lift_chemotaxisDiv_eq_liftChemotaxisDivPhysicalRep_strictSlab_interior`, also requiring `0 < a` and only asserting interior spatial equality.

These are not zero-start producers. They explicitly reduce strict slabs into `Ioo 0 T √óňĘ Icc 0 1`, then use `intervalDomain_dx_u_jointlyContinuous`, `intervalDomain_dx_v_jointlyContinuous`, and positive-time `hsol.v_nonneg`.

### 4. Zero-start RHS bridge deliberately keeps zero-start continuity as input

`ShenWork/Paper2/IntervalChiNegH1ZeroSlabPhysicalRHS.lean` is a useful downstream bridge, but not a producer of subfrontier A. Its header says it packages the next honest producer target and does not produce zero-time trace data. In particular:

## Derivability decision

Not currently derivable in general from existing assumptions.

It is not sound to claim derivability from
`IsPaper2ClassicalSolution intervalDomain p T u v` alone. Source definitions support only positive-time regularity/sign facts:

- `intervalDomainClassicalRegularity` records `u`, `v`, `u_x`, `v_x` continuity on `Ioo 0 T √óňĘ Icc 0 1` or per-time `t ‚ąą Ioo 0 T` fields.
- `IsPaper2ClassicalSolution.u_pos'` and `.v_nonneg` require `0 < t`.
- Existing strict-slab theorems uniformly require `0 < a` or `t ‚ąą Ioo 0 T`.

The target record requires every slab `Set.Icc 0 b √óňĘ Set.Icc 0 1`, hence includes `t = 0`. It is derivable only from an explicit zero-start source package proving the six closed-slab fields, or from a construction-specific theorem that proves those fields directly.
