# Definitive verdict

The Paper 2 finite-`p` absorption **does close**, but the sound route is more specific than the one-step estimate suggested in the question.

There are two different branches.

1. For `0 < m < 1`, the proof is the finite exponent-descent in equations (4.1)–(4.5) of Chen–Ruau–Shen. A single estimate using only an `L∞` bound on `∂ₓv` is too crude. The exponent attached to `u` is lowered by `2(1-m)` at each elliptic/Young step until it reaches zero. Only then is the remaining weighted `v_x` term controlled by the weighted elliptic estimate. This produces a scalar inequality of the form

   ```plain text
   y' ≤ -c y + C,
   ```

   for one sufficiently large finite exponent `p`.

2. For `m = 1`, the exponent does not descend. A direct seed estimate closes only when

   ```plain text
   1 < p < (2β-1)/χ₀,
   ```

   hence one needs the paper's smallness condition

   ```plain text
   χ₀ < 2(2β-1) / max{2, γN}.
   ```

   This supplies one initial `L^{p₀}` bound; the later bootstrap uses the weighted elliptic estimate to obtain the abstract cross-diffusion estimate with `rho = γ`.

For the concrete unit interval, `N=1`, so there is no dimensional Sobolev obstruction. The required choices reduce to

```plain text
0 < m < 1: choose n with p=2n(1-m)>max{1,γ};
m=1: choose max{1,γ/2}<p₀<(2β-1)/χ₀.
```

However, the current repository does **not** yet implement this faithful closure. Its concrete chemotaxis operator and cross-diffusion energy term use `u`, not `u^p.m`, and its current cross theorem yields the cruder exponent `rho = 2γ`. Thus Roadmap step 2 still needs the `m`-faithful finite-descent/weighted-elliptic package.

# 0. First correct the cross term

The faithful PDE is

```plain text
u_t = u_xx
      - χ₀ ∂ₓ( u^m v_x / (1+v)^β )
      + a u - b u^(1+α),

-v_xx + μv = νu^γ,

u_x=v_x=0 at x=0,1.
```

Testing the first equation by `u^(p-1)` and integrating by parts gives

```plain text
(1/p) y_p'
  = -(p-1) D_p
    + (p-1)χ₀ X_p
    + a y_p
    - b ∫u^(p+α),
```

where

```plain text
y_p := ∫₀¹ u^p,
D_p := ∫₀¹ u^(p-2) |u_x|²,
X_p := ∫₀¹ u^(p+m-2) u_x v_x /(1+v)^β.
```

The boundary term vanishes because `v_x=0` at both Neumann endpoints, hence the chemotaxis flux itself vanishes there.

For the attractive case one estimates `χ₀X_p` from above by replacing it with its absolute-value integrand. The case `χ₀≤0` belongs to the already easier negative-sensitivity branch.

In terms of `w=u^(p/2)`,

```plain text
D_p = (4/p²) ∫|w_x|².
```

Thus absorbing a coefficient times `D_p` is exactly absorbing a portion of the diffusion of `u^(p/2)`.

# 1. The first Young split

For any integer `n≥1`, the paper uses

```plain text
(p-1)χ₀ ∫ u^(p+m-2)|u_x||v_x|/(1+v)^β
≤ (p-1)/(n+1) D_p
  + C₀ I₁,
```

with

```plain text
C₀ := (p-1)(n+1)χ₀²/4,

I₁ := ∫ u^(p+2(m-1)) |v_x|²/(1+v)^(2β).
```

This is the precise pointwise Young inequality

```plain text
A B ≤ A²/(n+1) + (n+1)B²/4
```

applied to

```plain text
A = sqrt(p-1) u^((p-2)/2)|u_x|,
B = sqrt(p-1) χ₀ u^(p/2+m-1)|v_x|/(1+v)^β.
```

Equivalently, with an arbitrary diffusion budget `eps>0`,

```plain text
(p-1)χ₀ X_p
≤ eps D_p
  + (p-1)²χ₀²/(4 eps)
      ∫u^(p+2m-2)|v_x|²/(1+v)^(2β).
```

So question (1) has a qualified **yes**: the first split is exactly a diffusion piece plus a weighted higher-power `v_x` piece. But for `m<1`, that second piece is not closed by a single crude resolver-gradient bound. It is the input to a finite descent.

# 2. Why the crude `||v_x||∞` route is not the Paper 2 closure

The currently committed interval theorem follows the estimate

```plain text
||v_x||∞
  ≤ C_R ||νu^γ||₂
```

and therefore obtains, in the `m=1` model,

```plain text
cross ≤ eps D_p + C(eps) ∫u^(p+2γ).
```

The relevant committed declarations are

```lean
ShenWork.Paper2.crossDiffusion_pointwise_young
ShenWork.Paper2.crossDiffusion_pointwise_young_div
ShenWork.Paper2.intervalDomain_crossDiffusionEnergyTerm_young_sourceL2
ShenWork.Paper2.intervalDomain_crossDiffusionEnergyTerm_bootstrap_bound
ShenWork.Paper2.intervalDomain_crossDiffusionBootstrapEstimate_of_classical
```

and the last theorem packages `rho = 2*p.γ`.

That is a valid coarse estimate for the operator currently encoded, but it does **not** establish Theorem 1.2's sharp weak-cross-diffusion route:

- it loses the stabilizing denominator power `β`;
- it gives `rho=2γ` rather than the paper's critical-branch `rho=γ`;
- it does not perform the `m<1` exponent descent;
- by itself it gives a higher moment, not a scalar linear Gronwall inequality.

The correct elliptic estimate is the weighted `L^{2q}` estimate from Proposition 2.2:

```plain text
∫ |v_x|^(2q)/(1+v)^((1+β)q)
  ≤ Θ_β^q M*(N,q,μ,ν) ∫u^(γq),
```

for every `q>1` and `β≥0`.

This is the norm that matches the proof. The condition `β≥1` allows powers such as `2βq` to dominate `(1+β)q`, since

```plain text
2β ≥ 1+β.
```

That denominator comparison is load-bearing. A proof that never uses `β≥1` cannot be the intended proof of Theorem 1.2(1).

# 3. The `0<m<1` finite descent

Define, for `k≥1`,

```plain text
I_k := ∫ u^(p+2k(m-1)) |v_x|²/(1+v)^(2β).
```

Because `m-1<0`, each step lowers the `u` exponent by the fixed amount

```plain text
2(1-m)>0.
```

Using the elliptic equation tested with the corresponding weighted power of `u`, the denominator comparison from `β≥1`, Lemma 2.5's bound on `v/(1+v)^(β-1)`, and Young's inequality, the paper proves the recursive estimate

```plain text
C_(k-1) I_k
≤ (p-1)/(n+1) D_p
  + C_k I_(k+1)
  + y_p
  + C'_k,
```

for `k=1,...,n-1`, where one may take

```plain text
C_k := (n+1)/(4(p-1))
       * ((p+2k(m-1))/β)²
       * C_(k-1)²,

C'_k := [2k(1-m)/p]
        * [C_(k-1) μ Θ_(β-1)/β]^(p/[2k(1-m)])
        * |Ω|.
```

At the first step this is the paper's equation (4.1); replacing the lower power moment by `y_p+C'_1` gives (4.2).

Iterating yields

```plain text
(p-1)χ₀ X_p
≤ n(p-1)/(n+1) D_p
  + C_(n-1) I_n
  + (n-1)y_p
  + C',
```

where

```plain text
C' := Σ_{k=1}^{n-1} C'_k.
```

Now choose

```plain text
p := 2n(1-m).
```

Then the last `u` exponent is exactly zero:

```plain text
p+2n(m-1)=0,
```

so

```plain text
I_n = ∫ |v_x|²/(1+v)^(2β).
```

This is the key termination mechanism. It is why the `m<1` route works without any logistic damping and without an assumption relating `γ` to `m`.

Choose `n` large enough that

```plain text
p=2n(1-m)>max{N,γN}.
```

On the unit interval, `N=1`, so this is simply

```plain text
p>max{1,γ}.
```

Such an integer always exists because `1-m>0`.

# 4. Bounding the terminal elliptic term

Choose `q>1` with

```plain text
γq<p.
```

This is possible because `p>γN`, and in the interval case it is immediate from `p>γ`.

Pointwise Young on the terminal weighted gradient gives

```plain text
I_n
≤ ∫ |v_x|^(2q)/(1+v)^(2βq) + C''.
```

Since `β≥1`,

```plain text
2βq ≥ (1+β)q,
```

and `v≥0`, so Proposition 2.2 gives

```plain text
I_n
≤ Θ_β^q M*(N,q,μ,ν) ∫u^(γq) + C''.
```

Substituting the complete cross estimate into the energy identity leaves

```plain text
(1/p)y_p'
≤ -(p-1)/(n+1) D_p
  + C_(n-1) I_n
  + (a+n-1)y_p
  + C'.
```

The diffusion coefficient is positive because the `n` allocated pieces consume only

```plain text
n/(n+1)
```

of the original diffusion.

# 5. Mass-gradient interpolation and the final scalar inequality

Lemma 4.1 is precisely the mass-gradient interpolation estimate

```plain text
∀ delta>0,
  y_p ≤ delta D_p + C_(delta,p) (∫u)^p.
```

For a fixed finite horizon, let

```plain text
M₁ := sup_t ∫u(t)<∞.
```

This is the first-moment bound supplied upstream; it need not be literal mass conservation when `a` or `b` is nonzero.

Apply Lemma 4.1 with

```plain text
delta := (p-1)/((n+1)(a+n)).
```

Then

```plain text
(a+n)y_p
≤ (p-1)/(n+1)D_p + C''',

C''' := (a+n) C_(delta,p) M₁^p.
```

Writing `(a+n-1)y_p=(a+n)y_p-y_p`, the gradient term cancels exactly. Set

```plain text
A := C_(n-1) Θ_β^q M*(N,q,μ,ν),
K₀ := C''' + C_(n-1)C'' + C'.
```

The paper's equation (4.5) is

```plain text
(1/p)y_p' ≤ A ∫u^(γq) - y_p + K₀.
```

Let

```plain text
theta := γq/p ∈ (0,1).
```

On `[0,1]`, Hölder gives

```plain text
∫u^(γq) ≤ y_p^theta.
```

For `z≥0`, the explicit scalar Young bound

```plain text
A z^theta ≤ (1/2)z + C_A
```

holds with

```plain text
C_A := (1-theta)
       * A^(1/(1-theta))
       * (2*theta)^(theta/(1-theta)).
```

Therefore

```plain text
(1/p)y_p' ≤ -(1/2)y_p + K₀+C_A,
```

or equivalently

```plain text
y_p' ≤ -(p/2)y_p + p(K₀+C_A).
```

Thus

```plain text
y_p(t)
≤ exp(-(p/2)t)y_p(0)
  + 2(K₀+C_A)(1-exp(-(p/2)t)),
```

and in particular

```plain text
sup_t y_p(t) ≤ max{y_p(0), 2(K₀+C_A)}.
```

This is the definitive finite-`p` closure for `0<m<1`.

It is stronger than a generic `y'≤Cy+C'`: it has a strictly negative linear coefficient.

# 6. The `m=1` branch

When `m=1`, the descent does not move:

```plain text
p+2k(m-1)=p.
```

So a separate small-sensitivity argument is necessary.

The first Young split is

```plain text
(p-1)χ₀ X_p
≤ (p-1)/2 D_p
  + (p-1)χ₀²/2
      ∫u^p |v_x|²/(1+v)^(2β).
```

Applying the elliptic identity with denominator exponent `2β-1` gives a second estimate for the same weighted `v_x` term. The coefficient margin is positive exactly when

```plain text
χ₀ < (2β-1)/p,
```

or

```plain text
p < (2β-1)/χ₀.
```

More explicitly, set

```plain text
Aχ := (p-1)χ₀²/2,
Bχ := (p-1)χ₀(2β-1)/p - Aχ.
```

The condition above is equivalent to `0<Aχ<Bχ`. Hence one may choose

```plain text
0 < eta < min 1 ((p-1)/2 * (1-Aχ/Bχ)),
```

which gives the paper's required coefficient inequality

```plain text
Aχ ≤ (1-2eta/(p-1)) Bχ.
```

After inserting the two weighted estimates, one obtains

```plain text
(1/p)y_p'
≤ -eta D_p
  + (1+a+(p-1)χ₀μ/p)y_p
  - y_p.
```

Apply Lemma 4.1 with a sufficiently small interpolation parameter to absorb the positive multiple of `y_p` into `eta D_p`. This gives

```plain text
(1/p)y_p' ≤ -y_p + C_(eta,p) M₁^p,
```

and hence

```plain text
y_p' ≤ -p y_p + p C_(eta,p)M₁^p.
```

So every exponent

```plain text
1<p<(2β-1)/χ₀
```

is uniformly bounded.

To obtain a seed exponent large enough for the subsequent bootstrap, one needs an interval

```plain text
max{1,γN/2}<p₀<(2β-1)/χ₀.
```

This interval is nonempty precisely under the theorem's sufficient condition

```plain text
χ₀ < 2(2β-1)/max{2,γN}.
```

For the unit interval `N=1`, this becomes

```plain text
χ₀ < 2(2β-1)/max{2,γ}.
```

After the seed is obtained, the paper's second step proves, for arbitrary `p>1`,

```plain text
χ₀ X_p
≤ eps D_p + C_(eps,p) ∫u^(p+γ).
```

The exponent is `rho=γ`, not `2γ`, because the proof retains the denominator and applies the weighted elliptic estimate with

```plain text
q=(p+γ)/γ
```

and parameter `2β-1`. This estimate feeds the existing finite-`p` bootstrap/Corollary 2.1.

# 7. Is there a one-dimensional exponent obstruction?

No obstruction remains for the **paper's actual construction**.

## Slow branch `0<m<1`

The exponent requirement is not a delicate one-dimensional GN inequality. It is the design choice

```plain text
p=2n(1-m)>max{N,γN}.
```

On `[0,1]`, choose any sufficiently large integer `n`. Lemma 4.1 is available for every `p>1`.

## Critical branch `m=1`

The requirement is the seed interval

```plain text
max{1,γ/2}<p₀<(2β-1)/χ₀
```

on the interval. The stated `χ₀` threshold guarantees this.

## What a direct GN-only route would require

For comparison, if one had only

```plain text
(1/p)y_p' + c||∂ₓu^(p/2)||₂²
≤ C ∫u^(p+rho)+C',
```

then one-dimensional mass-based GN can absorb the higher moment into the gradient in one step only when

```plain text
rho<2.
```

Indeed, for `w=u^(p/2)` the gradient exponent after GN is

```plain text
2(p+rho-1)/(p+1),
```

which is strictly below `2` exactly when `rho<2`.

The crude committed estimate has `rho=2γ`, so it would require `γ<1` for this one-step route. Theorem 1.2 allows arbitrary `γ>0`, confirming that the crude route cannot be the intended proof. The weighted elliptic estimate and finite descent are what remove this apparent obstruction.

# Bottom line

- **Yes**, the diffusion absorption closes on `[0,1]`.
- **No**, it does not close by the simple committed `||v_x||∞`/source-`L²` estimate for arbitrary `γ`.
- For `0<m<1`, the correct closure is the finite exponent descent ending at `p+2n(m-1)=0`, followed by the weighted elliptic estimate and a scalar Gronwall inequality.
- For `m=1`, closure requires the smallness threshold on `χ₀`; the seed estimate gives a direct negative-linear Gronwall inequality, and the remaining finite-`p` bootstrap uses `rho=γ`.
- In one dimension there is no remaining exponent obstruction once these paper-faithful steps are used.
- The single largest missing Lean block is the **`m`-faithful finite descent plus weighted resolver-gradient estimate**; the elementary first Young split is already the easy part.
