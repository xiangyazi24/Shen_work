# Executive verdict

There are **three different obligations**, and they should not be collapsed into one phrase such as “Rothe compactness”:

1. `hprodAll`: solve every frozen-profile implicit Rothe step and supply the lower-barrier comparison packet;
2. `hstep`: fixed finite-step dependence of the Rothe orbit on the frozen profile;
3. `htail`: convergence of the Rothe orbit to its `k → ∞` limit, uniformly over trapped frozen profiles on every compact spatial interval.

The outer Arzelà–Ascoli/Helly compact-range theorem and the cube/Schauder fixed-point closure are already committed. Thus the Q4382 “Green compactness + closed graph + tail” diagnosis applies primarily to `hstep` and `htail`; it does **not** replace the separate per-step existence theorem `hprodAll`.

The live final assembly is therefore schematically

```text
concrete Route-A step solver
      │
      ├── hprodAll
      ├── one-step closed graph ── finite induction ── hstep
      └── uniform compact Cauchy/rate ──────────────── htail
                                                    │
existing paperRotheContinuousDependence + cube-G1/Schauder
                                                    │
                                         traveling-wave fixed point
```

There is also one important type-level warning: the current Route-A param-floor quantifies its producer over **all** functions `u`, while `PerStepBoxParams` contains a proof that `u` belongs to the monotone wave trap. Unless a producer for arbitrary non-trapped functions is supplied, that total field is not inhabitable. The executable interface should be trap-indexed, exactly as the frozen branch already does with `rotheSeqFromTrap`.

# 2. Route-A param core and the zero-profile vacuity

## 2.1 Why `b1_chiNeg_existence_final` is the wrong terminal interface

The legacy divergence-form theorem contains the field


and similarly asks for


For `0 ≤ M`, the zero profile belongs to the bare trap:


so `hpos` would imply `0 < 0`. The repository formalizes the same defect through

```lean

```

using the constant-zero map. Therefore an axiom-clean proof of the legacy theorem can still be conditional on an empty hypothesis surface.

The current floor is:


But every `producer u` contains

```lean

```

through `PerStepBoxParams.hu`. Therefore `producer : ∀ u, ...` asks for trap membership of every function `u : ℝ → ℝ`. This is not the intended PDE statement and is not satisfiable.

This is a **different** issue from the zero-profile fixed-point vacuity:

- lower-pinning fixes the zero fixed-point problem;
- trap-indexing fixes the universal-`u` producer problem.

The preferred executable repair is to totalize only the *sequence*, not the analytic producer. Mirror the already-committed frozen construction `rotheSeqFromTrap`:


### Green-kernel lemma that should sit underneath it

The reusable analytic core should be a theorem of the following form:

```lean
theorem greenConv_locallyUniform_of_source_locallyUniform_uniformBound
    {c lam : ℝ} (hc : 0 < c) (hlam : 0 < lam)
    {Rseq : ℕ → ℝ → ℝ} {R : ℝ → ℝ}
    (hRseq_cont : ∀ n, Continuous (Rseq n))
    (hR_cont : Continuous R)
    (hconv : LocallyUniformConverges Rseq R)
    (hbound : ∃ B : ℝ, 0 ≤ B ∧
      (∀ n y, |Rseq n y| ≤ B) ∧
      (∀ y, |R y| ≤ B)) :
    LocallyUniformConverges
      (fun n x => greenConv c lam (Rseq n) x)
      (greenConv c lam R)
```

Proof method: on `x ∈ [-R₀,R₀]`, split the Green integral into `|y| ≤ A` and `|y| > A`. The compact part is controlled by local-uniform convergence of the sources; the tail is controlled uniformly in `n` and `x` by the exponential Green kernel and the common source bound. This is the exact whole-line substitute for a bounded-domain compact-operator theorem.

For the nonlinear implicit step, use it in a standard closed-graph argument:

1. uniform step bounds and equi-Lipschitz/C² estimates give a locally-uniformly convergent subsequence `W_{n_j} → W*`;
2. the source terms
   `paperStepSource p c lam (u_{n_j}) (Z_{n_j}) (W_{n_j})`
   converge locally uniformly to the limiting source;
3. the Green theorem passes the fixed-point equation to the limit;
4. existing uniqueness of the fixed-source step identifies `W*` with the canonical step for `(u,Z)`;
5. every convergent subsequence has the same limit, so the full sequence converges locally uniformly.

That is the precise “compactness + closed graph” route. No Minty argument is involved or useful.

## 3.3 Exact uniform Rothe-tail target: `htail`

The existing definition is:

```lean
def PaperRotheTailUniform
    (p : CMParams) (c lam M κ Λ : ℝ)
    (hprodAll : ∀ v,
      PaperRotheStepProducer p c lam M κ Λ v)
    (hκ : 0 ≤ κ) (hM : 0 ≤ M) : Prop :=
  ∀ R > 0, ∀ ε > 0,
    ∃ K : ℕ,
      ∀ v : ℝ → ℝ,
        InMonotoneWaveTrapSet κ M v →
        ∀ k : ℕ, K ≤ k →
        ∀ x ∈ Set.Icc (-R) R,

              (rotheSeqOfPaper p c lam M κ Λ v
                (hprodAll v) hκ hM) x| < ε
```

This is a `k → ∞` tail, uniform in both the trapped frozen profile and the compact spatial window. It is distinct from the **spatial** Green-integral tail `|y| → ∞` used in the preceding DCT argument.
