# Q4309 — base spatial `C¹` gain from bounded data

## Verdict

Yes. The first spatial regularity gain is available from bounded data alone, provided the mild formula is in a full-kernel source form

```lean
u t = S t u₀ + ∫ s in 0..t, S (t - s) (F s)
```

with a uniform source bound

```lean
∀ s y, |F s y| ≤ B.
```

No source coefficient ladder is needed for this base step. The only analytic input is the positive-time heat-gradient estimate

```lean
‖∂ₓ S τ f‖∞ ≤ Cg * τ^(-1/2) * ‖f‖∞,
```

plus dominated differentiation of the Duhamel integral in the spatial variable. The singularity `(t - s)^(-1/2)` is integrable, so the Duhamel leg is spatially differentiable and has a finite sup bound.

This proves the ladder base:

```text
bounded source envelope / WindowCoefficientEnvelope 0
  ⇒ positive-time spatial C¹ / gradient envelope / WindowCoefficientEnvelope 1.
```

## Exact estimate

Assume:

```lean
0 < t
∀ x, |u₀ x| ≤ M₀
∀ s y, |F s y| ≤ B
```

and let `Cg` be the existing heat `L∞ → W^{1,∞}` constant, called in the repo as

```lean
ShenWork.HeatKernelGradientEstimates.heatGradientLinftyLinftyConstant
```

Then for every `x`,

```text
|∂ₓ u(t,x)|
  ≤ Cg * t^(-1/2) * M₀
    + ∫₀ᵗ Cg * (t-s)^(-1/2) * B ds
  = Cg * t^(-1/2) * M₀ + 2 * Cg * B * sqrt t.
```

On a positive window `t ∈ [t₀,T]`, this gives the uniform window bound

```text
|∂ₓ u(t,x)|
  ≤ Cg / sqrt(t₀) * M₀ + 2 * Cg * B * sqrt T.
```

This is exactly the estimate wanted for the base rung.

## 1. Free part

For the homogeneous heat part,

```lean
fun x => intervalFullSemigroupOperator t (intervalDomainLift u₀) x
```

use the already-existing pointwise heat-gradient estimate. In the current files, the relevant family of names is the one used by the positive-time gradient atoms:

```lean
ShenWork.IntervalNeumannFullKernel.intervalFullSemigroupOperator_deriv_Linfty_pointwise_sqrt_t
```

with shape:

```lean
theorem intervalFullSemigroupOperator_deriv_Linfty_pointwise_sqrt_t
    {t M : ℝ} (ht : 0 < t)
    (hmeas : AEStronglyMeasurable f (intervalMeasure 1))
    (hbound : ∀ y, |f y| ≤ M)
    (x : ℝ) :
    |deriv (fun z => intervalFullSemigroupOperator t f z) x|
      ≤ heatGradientLinftyLinftyConstant * t ^ (-(1 / 2) : ℝ) * M
```

or an equivalent local theorem under the same imports.

So the free part gives:

```lean
have hfree :
    |deriv (fun x => intervalFullSemigroupOperator t (intervalDomainLift u₀) x) x|
      ≤ Cg * t ^ (-(1 / 2) : ℝ) * M₀ := by
  exact intervalFullSemigroupOperator_deriv_Linfty_pointwise_sqrt_t
    ht hu₀_meas hu₀_bound x
```

For `t ≥ t₀ > 0`, convert

```lean
t ^ (-(1 / 2) : ℝ) = 1 / Real.sqrt t ≤ 1 / Real.sqrt t₀
```

using `Real.rpow_neg`, `Real.rpow_div_two_eq_sqrt`, and monotonicity of `sqrt`/division by a positive number.

## 2. Duhamel part

For the Duhamel part

```lean
D t x := ∫ s in 0..t, intervalFullSemigroupOperator (t - s) (F s) x
```

prove spatial differentiability by differentiating under the `s`-integral:

```lean
deriv (fun x => D t x) x
  = ∫ s in 0..t,
      deriv (fun z => intervalFullSemigroupOperator (t - s) (F s) z) x
```

The domination is exactly

```lean
|deriv (fun z => intervalFullSemigroupOperator (t - s) (F s) z) x|
  ≤ Cg * (t - s)^(-1/2) * B
```

for `s ∈ [0,t)`, and

```lean
∫ s in 0..t, (t - s)^(-1/2) = 2 * sqrt t.
```

This gives:

```lean
have hduh :
    |∫ s in 0..t,
        deriv (fun z =>
          intervalFullSemigroupOperator (t - s) (F s) z) x|
      ≤ Cg * (2 * Real.sqrt t) * B := by
  exact ShenWork.IntervalGradDuhamelBound.gradDuhamel_sup_bound
    (t := t) (T := t) ht (le_rfl)
    (q := F)
    hF_int
    hB_nonneg
    hF_sup
    x
    hgrad_integrable
```

The repo also has a shifted form used in the restart-window layer:

```lean
ShenWork.Paper2.TruncatedGradientWindow.gradDuhamel_shifted_sup_bound
```

with shape:

```lean
|∫ s in a..t,
    deriv (fun z => intervalFullSemigroupOperator (t - s) (q s) z) x|
  ≤ Cg * (2 * Real.sqrt (T - a)) * Cq.
```

For the base window starting at `0`, use the unshifted theorem. For later restart windows, use the shifted theorem.

## 3. Spatial derivative existence

The bound alone is not the derivative-existence statement. The formal proof should split the Duhamel leg into two facts:

```lean
have hD_hasDeriv :
    HasDerivAt
      (fun z => ∫ s in 0..t,
        intervalFullSemigroupOperator (t - s) (F s) z)
      (∫ s in 0..t,
        deriv (fun z =>
          intervalFullSemigroupOperator (t - s) (F s) z) x)
      x := by
  -- dominated differentiation under the integral
```

and then the bound above.

The domination needed for this `HasDerivAt` proof is the same integrable majorant:

```lean
s ↦ Cg * (t - s)^(-1/2) * B.
```

The key Mathlib pieces are:

```lean
intervalIntegral.intervalIntegrable_iff_integrableOn_Ioc_of_le
intervalIntegral.integral_mono_on
intervalIntegral.integral_hasDerivAt_right      -- for auxiliary FTC steps, when needed
HasDerivAt.integral_of_dominated               -- if available locally
```

If the last lemma is not directly available under that name, keep the existing project-style residual theorem: prove a local wrapper by dominated convergence for difference quotients. The proof is standard because the heat-kernel derivative estimate gives a uniform integrable bound.

In the repo, this is analogous to the residual theorem already appearing in the positive-time gradient atoms:

```lean
shiftedFullDuhamel_hasDerivAt_residual
```

which has the intended shape:

```lean
∀ x,
  HasDerivAt
    (fun z => ∫ s in a..τ, intervalFullSemigroupOperator (τ - s) (q s) z)
    (∫ s in a..τ,
      deriv (fun z => intervalFullSemigroupOperator (τ - s) (q s) z) x)
    x
```

under measurability, integrability, and a uniform source bound.

For the base case, instantiate `a = 0`, `τ = t`.

## 4. Full base theorem shape

A useful Lean statement is:

```lean

    {M0 B t : ℝ}
    (ht : 0 < t)
    (hu0_meas : AEStronglyMeasurable u0 (intervalMeasure 1))
    (hu0_bound : ∀ y, |u0 y| ≤ M0)
    (hF_meas : Measurable (Function.uncurry F))
    (hF_int : ∀ s, Integrable (F s) (intervalMeasure 1))
    (hB_nonneg : 0 ≤ B)

          intervalFullSemigroupOperator t u0 z

                intervalFullSemigroupOperator (t - s) (F s) z) x|

          heatGradientLinftyLinftyConstant * t ^ (-(1 / 2) : ℝ) * M0
            + heatGradientLinftyLinftyConstant * (2 * Real.sqrt t) * B := by
  intro x
  -- free-part derivative and bound
  -- Duhamel derivative under integral and bound
  -- combine by derivative of sum and abs_add
```

For a window version:

```lean
theorem bounded_fullDuhamel_spatial_C1_on_window
    {t0 T M0 B : ℝ}
    (ht0 : 0 < t0) (hT : t0 ≤ T)
    ... :
    ∃ G, 0 ≤ G ∧
      ∀ t, t0 ≤ t → t ≤ T →
      ∀ x,
        DifferentiableAt ℝ (fun z => u t z) x ∧
```

Then use:

```lean
Real.sqrt_le_sqrt ht
```

and the identity `t^(-1/2) = 1 / sqrt t` to compare the free part to the `t0` denominator.

## 5. Why this breaks the ladder circularity

The circularity risk is:

```text
source coefficient decay needs source regularity;
source regularity needs u regularity;
u regularity seems to need source coefficient decay.
```

The base breaks it because the first derivative of the heat Duhamel term only requires the source to be bounded. It does not require source differentiability or coefficient decay.

The logic is:

```text
|u| ≤ M
  ⇒ |F| ≤ B                         -- WCE 0 / bounded source envelope
  ⇒ ∂ₓu bounded on positive windows  -- WCE 1 / spatial C¹ base
  ⇒ flux derivative bounded
  ⇒ source coefficient ℓ¹ by one IBP or quadratic decay if enough boundary data
  ⇒ higher coefficient ladder starts.
```

So the base theorem is unconditional relative to the standard local-existence ball bound and bounded-source estimate.

## 6. Important caveat: full-source versus B-form source

The statement above is literally for a full-kernel source

```lean
∫ S(t-s) F(s) ds
```

with bounded `F`.

If the actual truncated chemotaxis mild formula is still in B-form,

```lean
∫ K_conj(t-s) Q(s) ds
```

then do not pretend `F = ∂ₓ Q` is bounded before the first derivative is available. In that case the base must be stated for the split formula:

```text
free heat part
+ value Duhamel from bounded logistic source
+ conjugate/B-kernel Duhamel from bounded flux Q.
```

For the value source, the argument above applies directly. For the conjugate/B-kernel leg, use the corresponding conjugate-kernel gradient estimate, or first prove the value-level restart identity only when an admissible source `F` has already been constructed.

Thus the base is cleanest if the ladder's `WindowCoefficientEnvelope 0` is a bounded full-source envelope. If the current formula is B-form, the correct base lemma should be called something like:

```lean
bounded_bFormDuhamel_spatial_C1_base
```

and it should use the B-kernel analogue of the same integrable singular estimate.

Do not use this full-source base to smuggle in a derivative of the flux before it has been proved.

## Bottom line

The first ladder step is valid and Lean-feasible:

```text
bounded source + heat gradient L∞→L∞ estimate
  ⇒ positive-time spatial C¹ and gradient bound.
```

The load-bearing local lemmas are:

```lean
intervalFullSemigroupOperator_deriv_Linfty_pointwise_sqrt_t
gradDuhamel_sup_bound
-- or shifted:
gradDuhamel_shifted_sup_bound
-- plus:
HasDerivAt of the Duhamel integral by dominated differentiation
```

The resulting bound is:

```text
|∂ₓu(t,x)| ≤ Cg * t^(-1/2) * ‖u₀‖∞ + 2 * Cg * sqrt(t) * ‖F‖∞.
```

On every positive window `[t₀,T]`, this is a finite uniform gradient envelope. This is the correct base rung for the coefficient/source ladder and does not require any pre-existing source regularity.
