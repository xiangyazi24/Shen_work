# 1. Cheapest reusable comparison theorem

The clean generic theorem should be independent of chemotaxis. It compares a bounded classical scalar supersolution with a spatially constant scalar subsolution.

A Lean-facing statement with exactly the hypotheses needed by the current slab-sup engine is:

## Constant-floor corollary

The theorem actually needed for `liminf inf u > 0` is a short specialization:

```lean
/-- A positive constant scalar subsolution remains a global spatial floor. -/
theorem wholeLineSlab_ge_const_of_scalar_supersolution
    {T A K L d : ℝ}
    {u : ℝ → ℝ → ℝ} {R : ℝ → ℝ}
    (hT : 0 < T) (hA : 0 ≤ A) (hK : 0 ≤ K) (hL : 0 ≤ L)
    (hd : d ∈ Set.Icc (0 : ℝ) A)
    (hRd : 0 ≤ R d)
    (hcont : Continuous (fun q : ℝ × ℝ => u q.1 q.2))
    (hurange : ∀ t ∈ Set.Icc (0 : ℝ) T, ∀ x,
      u t x ∈ Set.Icc (0 : ℝ) A)
    (hinit : ∀ x, d ≤ u 0 x)
    (hRlip : ∀ a ∈ Set.Icc (0 : ℝ) A,
      ∀ b ∈ Set.Icc (0 : ℝ) A,
        |R a - R b| ≤ L * |a - b|)
    (htime : ...)
    (hspace1 : ...)
    (hspace2 : ...)
    (hpde : ∀ ⦃t x : ℝ⦄, t ∈ Set.Ioc (0 : ℝ) T →
      deriv (fun s : ℝ => u s x) t ≥
        deriv (fun y : ℝ => deriv (fun z : ℝ => u t z) y) x -
          K * |deriv (fun y : ℝ => u t y) x| + R (u t x)) :
    ∀ t ∈ Set.Icc (0 : ℝ) T, ∀ x, d ≤ u t x := by
  apply wholeLineSlab_ge_of_moving_scalar_subsolution
    (B := fun _ => d) (R := R) hT hA hK hL hcont
  · exact continuous_const
  · exact hurange
  · intro _ _
    exact hd
  · exact hinit
  · intro t _
    simpa using hasDerivAt_const t d
```

This is the preferred endpoint for the eventual positive floor: after restarting at a time `t₀` with a quantitative uniform floor `c₀`, choose `0 < d ≤ c₀` satisfying `R d ≥ 0`, and propagate that same `d` forever.

# 2. Core proof: reuse the slab-sup theorem verbatim

Set

```lean
let D : ℝ := L + 1
let E : ℝ → ℝ := fun t => Real.exp (-(D * t))
let q : ℝ → ℝ → ℝ := fun t x => E t * (B t - u t x)
let G : ℝ → ℝ := fun r =>
  (L + D) * max (-r) 0 - max r 0
```

The range assumptions imply a uniform slab ceiling for `q`:

```lean
q t x ≤ A.
```

The initial comparison gives

```lean
q 0 x ≤ 0.
```

The derivative identities are

```lean
q_t = E * (B' - u_t) - D * q,
q_x = -E * u_x,
q_xx = -E * u_xx.
```

From the scalar subsolution and the PDE supersolution,

```text
B' ≤ R(B),
u_t ≥ u_xx - K |u_x| + R(u),
```

one obtains

```text
q_t ≤ q_xx + K |q_x|
      + E (R(B) - R(u)) - D q.
```

The Lipschitz estimate gives

```text
E (R(B) - R(u)) ≤ L |q|.
```

With `D = L + 1`, split on the sign of `q`:

```text
q ≥ 0:  L |q| - D q = -q,
q < 0:  L |q| - D q = (L + D)(-q).
```

Therefore, at every point of the slab,

```lean
q_t ≤ q_xx + K * |q_x| + G q.
```

The function `G` is continuous, and at any positive slab supremum `S`,

```lean
G S = -S < 0.
```

Hence the existing theorem applies directly:

```lean
have hqslab : wholeLineSlabSup T q ≤ 0 :=
  wholeLineSlabSup_le_of_scalar_pde
    (T := T) (C := 0) (A := A) (K := K) (u := q) (G := G)
    hT hK hcontq hupperq hinitq hGcont hGstrict
    htimeq hspace1q hspace2q hpdeq
```

Finally, `q t x ≤ 0` and `E t > 0` imply `B t - u t x ≤ 0` by the same
`nonpos_of_mul_nonpos_left` step already used at the end of
`wholeLineSlab_ge_expFloor_of_nonpositive_resolver_pde`.

This proof is essentially the committed nonpositive-sensitivity floor proof with all chemotaxis-specific algebra removed. It should be shorter than that theorem by several hundred lines.

# 3. Does one need a new `wholeLineSlabInf`?

No. The moving-floor theorem above is best proved with the positive violation field `B-u` and the existing supremum machinery.

If a public infimum API is desirable for other consumers, define it through negation rather than rebuilding the `sInf` theory:

```lean
def wholeLineSlabInf (T : ℝ) (u : ℝ → ℝ → ℝ) : ℝ :=
  -wholeLineSlabSup T (fun t x => -u t x)
```

Then the exact dual of the scalar maximum theorem is:

```lean

        deriv (fun y : ℝ => deriv (fun z : ℝ => u t z) y) x -

  -- Apply the existing theorem to `w t x := -u t x`
  -- with scalar field `H r := -G (-r)`.

```

The proof is a negation wrapper around `wholeLineSlabSup_le_of_scalar_pde`:

```text
w = -u,
H(r) = -G(-r),
w_t ≤ w_xx + K |w_x| + H(w).
```

It requires no `csInf`, `le_csInf`, or `csInf_le` bookkeeping.

## What dualizes in the existing approximate-maximum proof

If one nevertheless writes a named approximate-minimum corollary, apply
`exists_wholeLineSlab_approx_max_deriv_data` to `-u`. The exact sign translation is:

```text
maximum statement for -u              minimum statement for u
----------------------------------------------------------------
sup(-u) - 2δ < -u(t,x)          <=>   u(t,x) < inf(u) + 2δ
0 ≤ (-u)_t                       <=>   u_t ≤ 0
|(-u)_x| < ρ                     <=>   |u_x| < ρ
(-u)_xx < ρ                      <=>   -ρ < u_xx
```

In the direct penalization picture:

```text
u - εx² at an approximate maximum
```

dualizes to

```text
u + εx² at an approximate minimum.
```

The time-face argument changes from a nonnegative derivative at a positive-time maximum to a nonpositive derivative at a positive-time minimum. The local second-derivative sign changes from `u_xx ≤ small` to `u_xx ≥ -small`.

But none of this needs to be reimplemented because the public approximate-max theorem already applies to the negated field, and the moving barrier is even more directly handled by `q = exp(-Dt)(B-u)`.

# 4. The positive-sensitivity scalar residual

Write the expanded physical PDE as

```text
u_t = u_xx
      - χ m u^(m-1) u_x v_x
      + χ u^m (u^γ - v)
      + reactionFun α u.
```

Assume on the restarted slab

```text
0 ≤ u(t,x) ≤ A,
v(t,·) = frozenElliptic p (u(t,·)).
```

The committed resolver bounds give

```text
0 ≤ v(t,x) ≤ A^γ,
|v_x(t,x)| ≤ A^γ.
```

The exact repository producers are:

```lean
frozenElliptic_le_of_rpow_le
frozenElliptic_deriv_abs_le
```

The pattern is already present in `wholeLineSlab_ge_expFloor_of_nonpositive_resolver_pde`:

```lean
have hvA : frozenElliptic p (u t) x ≤ A ^ p.γ := by
  apply frozenElliptic_le_of_rpow_le p
    (Real.rpow_nonneg hA0 p.γ) hsliceCont hnonneg
  intro y
  exact Real.rpow_le_rpow (hnonneg y) (hupper y)
    (zero_le_one.trans p.hγ)

have hvxA : |deriv (frozenElliptic p (u t)) x| ≤ A ^ p.γ :=
  (frozenElliptic_deriv_abs_le p hsliceC hnonneg x).trans hvA
```

The first-order term is bounded from below by

```text
-χ m u^(m-1) u_x v_x
  ≥ -K_A |u_x|,

K_A := χ m A^(m-1) A^γ.
```

For the zeroth-order term, the direction needed for a lower bound is

```text
u^γ - v ≥ u^γ - A^γ.
```

Since `χ u^m ≥ 0`,

```text
χ u^m (u^γ - v)
  ≥ χ u^m (u^γ - A^γ).
```

Thus define the cap-dependent scalar residual

```lean
def wholeLinePositiveFloorResidual
    (p : CMParams) (A r : ℝ) : ℝ :=
  reactionFun p.α r +
    p.χ * r ^ (p.m + p.γ) -
    p.χ * A ^ p.γ * r ^ p.m
```

or equivalently on `r ≥ 0`,

```text
reactionFun α r + χ r^m (r^γ - A^γ).
```

The sharpened local PDE estimate to add is:

```lean
/-- Positive-sensitivity lower scalar estimate under a population cap. -/
theorem wholeLineCauchy_physical_pde_positive_floor_lower_bound
    (p : CMParams) {A u ux vx v ut uxx : ℝ}
    (hχ : 0 ≤ p.χ) (hA : 0 ≤ A)
    (hu0 : 0 ≤ u) (huA : u ≤ A)
    (hvA : v ≤ A ^ p.γ)
    (hvxA : |vx| ≤ A ^ p.γ)
    (hpde : ut = uxx - p.χ *
      (p.m * u ^ (p.m - 1) * ux * vx +
        u ^ p.m * (v - u ^ p.γ)) +
      reactionFun p.α u) :
    uxx -
        (p.χ * p.m * A ^ (p.m - 1) * A ^ p.γ) * |ux| +
        wholeLinePositiveFloorResidual p A u ≤ ut := by
  ...
```

This is a finite pointwise inequality, analogous to the committed
`wholeLineCauchy_physical_pde_drift_lower_bound`, but it retains the favorable
`+χ u^(m+γ)` term and uses the sharper adverse cap `-χ A^γ u^m`.

## Critical exponent simplification

Under

```lean
hcritical : p.α = p.m + p.γ - 1,
```

we have `p.m + p.γ = p.α + 1`, and for `r ≥ 0`,

```text
wholeLinePositiveFloorResidual p A r
  = r - (1 - χ) r^(α+1) - χ A^γ r^m.
```

This is the exact scalar reaction-residual requested in the question:

```lean
B_low'(t) ≤
  B_low(t)
    - (1 - p.χ) * B_low(t) ^ (p.α + 1)
    - p.χ * A ^ p.γ * B_low(t) ^ p.m.
```

Equivalently, before using the critical identity,

```lean
B_low'(t) ≤
  reactionFun p.α (B_low t) +
    p.χ * (B_low t) ^ p.m *
      ((B_low t) ^ p.γ - A ^ p.γ).
```

The adverse nonlocal contribution is exactly

```text
-χ A^γ B_low^m.
```

It comes from an **upper** bound on `v`; a lower resolver bound is useless for this sign. The derivative resolver bound enters only in the drift constant `K_A`.

# 5. Lipschitz constant for the residual

For the generic comparison theorem, use the expanded residual

```text
R_A(r) = reactionFun α r
       + χ r^(m+γ)
       - χ A^γ r^m.
```

On `[0,A]`, a committed-lemma-compatible constant is

```lean
def wholeLinePositiveFloorResidualLip
    (p : CMParams) (A : ℝ) : ℝ :=
  reactionLip p.α A +
    |p.χ| * rpowLip (p.m + p.γ) A +
    |p.χ| * A ^ p.γ * rpowLip p.m A
```

The exact existing inputs are

```lean
reaction_lipschitz_on_Icc p.hα hA
rpow_m_lipschitz_on_Icc (by linarith [p.hm, p.hγ]) hA
rpow_m_lipschitz_on_Icc p.hm hA
```

followed by `.dist_le_mul`, conversion from `dist` to `abs`, and triangle inequalities. No positivity floor is used in this Lipschitz estimate because all exponents are at least one.

Hence the positive-resolver slab theorem can internally instantiate

```lean
K := p.χ * p.m * A ^ (p.m - 1) * A ^ p.γ
L := wholeLinePositiveFloorResidualLip p A
R := wholeLinePositiveFloorResidual p A
```

and then invoke `wholeLineSlab_ge_of_moving_scalar_subsolution`.

# 6. Constant `d` and the actual eventual-floor route

For a constant barrier `B_low(t) = d`, the scalar subsolution condition is simply

```lean
0 ≤ wholeLinePositiveFloorResidual p A d.
```

In the critical case this is

```text
0 ≤ d - (1-χ)d^(α+1) - χ A^γ d^m.
```

A convenient sufficient condition, using `0 < d ≤ 1`, `m ≥ 1`, and hence `d^m ≤ d`, is

```text
(1-χ) d^α ≤ 1 - χ A^γ.
```

Indeed,

```text
R_A(d)
  = d - (1-χ)d^(α+1) - χ A^γ d^m
 ≥ d * ((1 - χ A^γ) - (1-χ)d^α).
```

For the worst case `m = 1`, one genuinely needs

```text
χ A^γ < 1.
```

This is supplied noncircularly by waiting until the already-proved upper envelope is below a cap `A > MChi` chosen sufficiently close to `MChi`. In the critical branch,

```text
MChi^α = 1 / (1-χ),
α = m+γ-1 ≥ γ,
MChi ≥ 1,
```

so

```text
χ MChi^γ ≤ χ / (1-χ) < 1
```

exactly when `χ < 1/2`. Therefore one may choose

```text
MChi < A  and  χ A^γ < 1.
```

Then choose a small `d > 0` satisfying the residual inequality.

# 7. Noncircular assembly

The dependency order should be:

```text
(1) Existing positive upper ceiling
      -> choose t₀ and A with u(t,x) ≤ A for t ≥ t₀
      -> obtain v ≤ A^γ and |v_x| ≤ A^γ.

(2) Quantitative finite-horizon lower persistence from the uniformly
    positive datum
      -> obtain c₀ > 0 with c₀ ≤ u(t₀,x) for all x.

(3) Choose 0 < d ≤ c₀ with
      wholeLinePositiveFloorResidual p A d ≥ 0.

(4) Shift the solution:
      uShift s x := u (t₀ + s) x.

(5) Apply the constant-floor comparison on every finite horizon T:
      d ≤ uShift s x for 0 ≤ s ≤ T.

(6) Since T was arbitrary:
      ∀ t ≥ t₀, ∀ x, d ≤ u(t,x).

(7) Conclude UniformLiminfGe u d, hence liminf_t inf_x u(t,x) ≥ d > 0.
```

The upper resolver bound in step (1) depends only on the already-established upper cap. It does not use the desired lower floor, so the chemotaxis residual estimate is noncircular.

## One separate interface still matters

Pointwise strict positivity at time `t₀` is not enough on `ℝ`; initialization requires one constant `c₀ > 0` valid for every `x`.

The proof of

```lean
wholeLine_pos_of_uniform_initial_of_drift_supersolution
```

in `WholeLineCauchyStrictPositivity.lean` already derives the quantitative inequality

```text
δ ≤ exp(Kzero * t) * u(t,x),
```

before weakening its conclusion to `0 < u(t,x)`. The needed reusable export is therefore essentially

```lean
theorem wholeLine_ge_exp_neg_zeroRate_of_uniform_initial_of_drift_supersolution
    ... :
    ∀ t ∈ Set.Icc (0 : ℝ) T, ∀ x,
      δ * Real.exp (-(Kzero * t)) ≤ u t x
```

or the same inequality in the existing multiplied form. At the fixed restart time, take

```text
c₀ := δ * exp(-Kzero * t₀) > 0.
```

This quantitative restart-floor export is logically separate from the new scalar comparison lemma. Without it, the constant-barrier route cannot be initialized merely from pointwise positivity.

# Bottom line

- **Generic slab-inf theorem:** not currently committed.
- **Specialized lower comparison:** already committed for `χ ≤ 0` as `wholeLineSlab_ge_expFloor_of_nonpositive_resolver_pde`.
- **New deep PDE work:** none.
- **Correct implementation:** apply the existing slab-sup theorem to `exp(-(L+1)t)(B_low-u)`.
- **Positive-χ residual under `u ≤ A`:**
  ```text
  reactionFun α r + χ r^m(r^γ-A^γ),
  ```
  or in the critical case
  ```text
  r - (1-χ)r^(α+1) - χ A^γ r^m.
  ```
- **Adverse nonlocal term:** controlled by the independently proved upper bound `v ≤ A^γ`; `|v_x| ≤ A^γ` supplies the drift constant.
- **For mere persistence:** use a constant `d > 0`; no moving ODE solution is required.
- **Only additional prerequisite:** export the quantitative uniform finite-time floor needed to initialize the late restart.
