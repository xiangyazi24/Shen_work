Proof route:

```lean

  have h01 : (0 : ℝ) ≤ 1 := by norm_num
  have hdu₀_uIcc : ContinuousOn du₀ (Set.uIcc (0 : ℝ) 1) := by
    simpa [Set.uIcc_of_le h01] using hdu₀_cont
  have hdu₀_int : IntervalIntegrable du₀ MeasureTheory.volume 0 1 :=
    hdu₀_uIcc.intervalIntegrable
  obtain ⟨B, hB⟩ := isCompact_Icc.exists_bound_of_continuousOn hdu₀_cont
  let M : ℝ := max B 0
  have hM : 0 ≤ M := le_max_right B 0
  have hdu₀_bound : ∀ y ∈ Set.Icc (0 : ℝ) 1, |du₀ y| ≤ M := by
    intro y hy
    exact (Real.norm_eq_abs (du₀ y) ▸ hB y hy).trans (le_max_left B 0)
  exact intervalFullSemigroup_initialLegC1Approx_of_Icc_repr_continuousOn_zero_bound
    hu₀Q hQ_meas hQ_bound hQ_deriv hdu₀_int
```

Expected pain points:

* `exists_bound_of_continuousOn` returns a norm bound, so use `Real.norm_eq_abs` to convert to `|du₀ y|`.
* `IntervalIntegrable` wants the `Set.uIcc` form; rewrite by `Set.uIcc_of_le h01`.
* Use `M := max B 0`, not `B`, since the compactness witness `B` is not syntactically nonnegative.

3. **Logistic Duhamel derivative leg** — prove:


   using the existing full-kernel gradient `L∞` bounds and source boundedness.

4. **Fixed-point/lift derivative wiring** — use the mild equation `D.u t = intervalGradientDuhamelMap ...` and a differentiability statement for the RHS as a function of the real spatial coordinate.  This is where the current package stops.

This decomposition should be done in a clean PDE/Paper2 bridge file rather than directly inside the BForm zero-start file if another pane owns those dirty files.

## Circularity / faithfulness flags

### Do not use the C¹ package in `IntervalCoupledClassicalBallEstimates.lean` as-is

That file and its structural bridge sit on the older ball-estimate stack, whose semigroup terms are built from:

```lean
intervalSemigroupOperator 1 t f x
```

The new package proves facts about:

```lean
intervalFullSemigroupOperator t f x
```

These are not definitionally or theorematically interchangeable in that stack.  Applying the new theorem there would either fail by type mismatch or silently mix the helper kernel with the full Neumann kernel, which is not faithful.

### Do not claim it closes `PatchedChemicalDerivUniformApproachAtZero`

The v-side patched chemical derivative trace is about static resolver-gradient continuity at the initial slice, not about `∂ₓ S_full(t) u₀`.  It needs a resolver-gradient continuity theorem, not source-IBP for the full heat kernel.

### Do not claim it closes `PatchedSliceDerivUniformApproachAtZero` alone

The homogeneous leg is only one term of the differentiated gradient mild map.  Closing the full residual still requires derivative vanishing for the chemotaxis and logistic Duhamel terms and a derivative-of-fixed-point-equation bridge.

Add the compact-continuity wrapper in `IntervalSemigroupC1ApproxIdentity.lean` first:
