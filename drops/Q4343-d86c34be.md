# Q4343 — zero-initial integral Gronwall for a continuous nonnegative scalar

## Verdict

For the statement

```lean
E : ℝ → ℝ
ContinuousOn E (Set.Icc 0 b)
∀ t ∈ Set.Icc 0 b, 0 ≤ E t
∀ t ∈ Set.Icc 0 b, E t ≤ C * ∫ s in (0:ℝ)..t, E s
```

with `0 ≤ b`, `0 ≤ C`, and `E 0 = 0`, the cleanest Mathlib route is **not** a direct integral-Gronwall theorem. Use the ODE Gronwall zero lemma on the primitive

```lean
F t := ∫ s in (0:ℝ)..t, E s
```

The exact useful Mathlib lemma is:

```lean
Mathlib.Analysis.ODE.Gronwall.eq_zero_of_abs_deriv_le_mul_abs_self_of_eq_zero_right
```

Its shape is:

```lean
theorem eq_zero_of_abs_deriv_le_mul_abs_self_of_eq_zero_right
    {E : Type*} [NormedAddCommGroup E] [NormedSpace ℝ E]
    {f f' : ℝ → E} {K a b : ℝ}
    (hf : ContinuousOn f (Set.Icc a b))
    (hf' : ∀ x ∈ Set.Ico a b,
      HasDerivWithinAt f (f' x) (Set.Ici x) x)
    (ha : f a = 0)
    (bound : ∀ x ∈ Set.Ico a b,
      ‖f' x‖ ≤ K * ‖f x‖)
    (x : ℝ) :
    x ∈ Set.Icc a b → f x = 0
```

For this problem instantiate it with:

```lean
f  := fun t => ∫ s in (0:ℝ)..t, E s
f' := E
K  := C
a  := 0
```

Then `F = 0` on `[0,b]`, and the original inequality gives `E t ≤ C * F t = 0`; combined with `0 ≤ E t`, this yields `E t = 0`.

## Minimal Lean skeleton

```lean
import Mathlib.Analysis.ODE.Gronwall
import Mathlib.MeasureTheory.Integral.IntervalIntegral.Basic

open Set MeasureTheory
open scoped Topology

lemma zero_of_nonneg_le_const_mul_integral
    {E : ℝ → ℝ} {C b : ℝ}
    (hb : 0 ≤ b)
    (hC : 0 ≤ C)
    (hcont : ContinuousOn E (Set.Icc (0 : ℝ) b))
    (hE_nonneg : ∀ t ∈ Set.Icc (0 : ℝ) b, 0 ≤ E t)
    (hineq : ∀ t ∈ Set.Icc (0 : ℝ) b,
      E t ≤ C * (∫ s in (0 : ℝ)..t, E s)) :
    ∀ t ∈ Set.Icc (0 : ℝ) b, E t = 0 := by
  let F : ℝ → ℝ := fun t => ∫ s in (0 : ℝ)..t, E s

  have hF0 : F 0 = 0 := by
    simp [F]

  have hF_nonneg : ∀ t ∈ Set.Icc (0 : ℝ) b, 0 ≤ F t := by
    intro t ht
    dsimp [F]
    have ht0 : 0 ≤ t := ht.1
    exact intervalIntegral.integral_nonneg ht0 (by
      intro s hs
      exact hE_nonneg s ⟨hs.1, hs.2.trans ht.2⟩)

  have hF_cont : ContinuousOn F (Set.Icc (0 : ℝ) b) := by
    -- Standard FTC/continuity-of-parameter proof.
    -- One route: prove `HasDerivAt F (E t) t` on a neighborhood using
    -- `intervalIntegral.integral_hasDerivAt_right`, then take continuity.
    -- Alternatively use the interval-integral continuity lemma already imported
    -- in the local project, if available.
    sorry

  have hF_deriv : ∀ t ∈ Set.Ico (0 : ℝ) b,
      HasDerivWithinAt F (E t) (Set.Ici t) t := by
    intro t ht
    -- From continuity of `E` near `t`.
    -- Typical lemma:
    --   intervalIntegral.integral_hasDerivWithinAt_right
    -- or derive from:
    --   intervalIntegral.integral_hasDerivAt_right
    -- then `.hasDerivWithinAt`.
    sorry

  have hbound : ∀ t ∈ Set.Ico (0 : ℝ) b,
      ‖E t‖ ≤ C * ‖F t‖ := by
    intro t ht
    have htIcc : t ∈ Set.Icc (0 : ℝ) b := ⟨ht.1, le_of_lt ht.2⟩
    have hEt : 0 ≤ E t := hE_nonneg t htIcc
    have hFt : 0 ≤ F t := hF_nonneg t htIcc
    rw [Real.norm_eq_abs, abs_of_nonneg hEt,
        Real.norm_eq_abs, abs_of_nonneg hFt]
    exact hineq t htIcc

  have hF_zero : ∀ t, t ∈ Set.Icc (0 : ℝ) b → F t = 0 := by
    intro t ht
    exact eq_zero_of_abs_deriv_le_mul_abs_self_of_eq_zero_right
      (f := F) (f' := E) (K := C) (a := (0 : ℝ)) (b := b)
      hF_cont hF_deriv hF0 hbound t ht

  intro t ht
  have hEt_nonneg : 0 ≤ E t := hE_nonneg t ht
  have hEt_le_zero : E t ≤ 0 := by
    have h := hineq t ht
    rw [hF_zero t ht] at h
    simpa [F] using h
  exact le_antisymm hEt_le_zero hEt_nonneg
```

The two `sorry`s in the skeleton are not Gronwall gaps; they are ordinary FTC facts for the primitive `F`. In a file already using interval integrals, they should be discharged with:

```lean
intervalIntegral.integral_hasDerivAt_right
intervalIntegral.integral_hasDerivWithinAt_right
```

or a local wrapper around them.

## Why not use `norm_le_gronwallBound` directly?

`norm_le_gronwallBound_of_norm_deriv_right_le` is also in `Mathlib.Analysis.ODE.Gronwall`, but it gives an explicit bound rather than the zero conclusion. It works too:

```lean
F t ≤ gronwallBound 0 C 0 (t - 0) = 0
```

then use `gronwallBound_ε0_δ0`. But `eq_zero_of_abs_deriv_le_mul_abs_self_of_eq_zero_right` is the shortest path because it packages exactly the zero-initial, no-forcing case.
