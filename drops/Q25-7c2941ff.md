## 1. The mathematically minimal positive-time route

For the scalar Neumann equation

```text
u_t - ΈΫ u_xx = F,
βà²x u(t,0) = βà²x u(t,1) = 0,
```

the classical endpoint brick is the constant-cancellation identity

```text
βà²xx S_N(œÉ) h(x)
  = βàΪβ²Ä¬Ι βà²xx K_N(œÉ,x,y) (h(y) - h(x)) dy,
```

because the Neumann semigroup preserves constants:

```text
βàΪβ²Ä¬Ι βà²xx K_N(œÉ,x,y) dy = 0.
```

If `h βàà C^ΈΗ`, `0 < ΈΗ < 1`, then

```text
βÄ•βà²xx S_N(œÉ) hβÄ•βàû
  βâΛ CΈΗ œÉ^(-1 + ΈΗ/2) [h]_{C^ΈΗ},
```

and `œÉ^(-1+ΈΗ/2)` is integrable at zero. This is the true analytic content of a theorem named along the lines of

```lean
neumannDuhamel_positiveTime_C2_slice
```

or, at the lower level already used by the project,

```lean
neumannHeatSecondDeriv_Ctheta_to_Linfty
```

### Divergence-form caveat

The chemotaxis mild leg has the form

```text
-œ΅β²Ä βàΪβ²ÄαΒ½ βà²x S_N(t-s) Q(s) ds.
```

To prove one spatial derivative of `u`, one differentiates once more and estimates

```text
βàΪβ²ÄαΒ½ βà²xx S_N(t-s) Q(s) ds.
```

Thus `Q(s,¬Ζ) βàà C^ΈΗ` gives `u(t,¬Ζ) βàà C^{1+ΈΖ}` for every `0 < ΈΖ < ΈΗ`. This is the honest role of the `ChemMildC1eta*` stack.

For full `u βàà C¬≤`, one needs one additional level:

```text
Q βàà C^{1+ΈΗ}
```

so that `Q_x βàà C^ΈΗ`, or an equivalent full divergence-form Schauder theorem. Merely knowing `Q βàà C^ΈΗ` is enough for `C^{1+ΈΖ}`, but not for the full `C¬≤` endpoint.

This gives a non-circular two-stage analytic route:

```text
continuous bounded mild solution
  βÜ£ positive-time C^ΈΗ smoothing
  βÜ£ Q βàà C^ΈΗ
  βÜ£ u βàà C^{1+ΈΖ}
  βÜ£ resolver elliptic gain and nonlinear composition
  βÜ£ Q βàà C^{1+ΈΗ'}
  βÜ£ u βàà C¬≤
  βÜ£ u_t = ΈΫu_xx + F is continuous.
```

On paper this is a short Schauder bootstrap. In Lean, however, formalizing a general parabolic Schauder theorem with Neumann boundary compatibility would be much larger than reusing the repository's spectral machinery.

## 2. The cheapest route in the current repository: restart-cosine spectral bootstrap

Fix an interior time `tβ²Ä βàà (0,T)` and restart at

```text
œ³ = tβ²Ä / 2.
```

Let

```text
aβ²Ä(k) = cosineCoeff(u(œ³), k),
s(t,k) = B-form source coefficient at time t.
```

The restarted coefficient is

```text
c_k(r)
  = exp(-ΈΫ ΈΜ_k r) aβ²Ä(k)
    + βàΪβ²Ä ≥ exp(-ΈΫ ΈΜ_k (r-q)) s(œ³+q,k) dq,
```

where `ΈΜ_k = (kœÄ)¬≤`.

The decisive local-positive-time requirements are:

```text
(A) βàëβ²• ΈΜ_k |c_k(r)| < βàû,
(B) βàëβ²• |c'_k(r)| < βàû,
(C) both majorants are locally uniform in r,
(D) c'_k(r) = -ΈΫ ΈΜ_k c_k(r) + s(œ³+r,k),
(E) s(¬Ζ,k) is C¬Ι with summable coefficient envelopes.
```

Then the cosine series may be differentiated twice in `x` and once in `t`:

```text
u(t,x)   = βàëβ²• c_k(t-œ³) cos(kœÄx),
u_xx(t,x)= -βàëβ²• ΈΜ_k c_k(t-œ³) cos(kœÄx),
u_t(t,x) = βàëβ²• c'_k(t-œ³) cos(kœÄx).
```

Local uniform absolute convergence gives the joint continuity needed for `C^{2,1}` and immediately gives the Neumann boundary condition because every cosine mode has zero derivative at `0` and `1`.

This is exactly why the current consumer uses:

```lean
Summable (fun k => unitIntervalCosineEigenvalue k * |b k|)
```

rather than merely square summability.

### Sobolev interpretation

If the project uses

```lean
MemHSob s a := Summable (fun k => (1 + ΈΜ_k)^s * a k ^ 2),
```

then Cauchy-Schwarz gives

```text
MemHSob s a,  s > 5/2
  β΅£ βàëβ²• ΈΜ_k |a_k| < βàû.
```

Indeed `ΈΜ_k ~ k¬≤`, and the residual series behaves like

```text
βàë k^(4-2s),
```

which converges precisely for `s > 5/2`. Therefore an integer-valued ladder should target at least the `s = 3` level. Stopping at `H¬≤` is not enough for the absolute second-derivative series used by the existing classical consumer.

### Why restart is essential

The original datum is only continuous. One should never demand weighted summability at `t = 0`. Restart at `œ³ > 0` separates:

```text
rough initial trace at 0
```

from

```text
smooth positive-time slice at œ³.
```

All coefficient ODE, source regularity, and termwise differentiation statements should be local on compact windows inside `(0,T)`.

## 3. Non-circular finite ladder

The proof must be stratified by regularity level. Do not use the final `C¬≤` conclusion to prove the source regularity that is supposed to yield `C¬≤`.

A safe dependency order is:

### G0. Continuous mild data and bounds

Use only the fields already present in `ConjugateMildSolutionData`:

```text
joint continuity, positivity, uniform boundedness, mild identity.
```

The positive lower floor on a compact positive-time window follows from strict positivity plus compactness.

### G1. Resolver at fixed time

From a bounded positive `u(t,¬Ζ)`, obtain the elliptic resolver `v = R[u]`, its positivity, and the fixed-time spatial gain furnished by the 1D Neumann resolver. This step does not require `u_t`.

### G2. Lowest source envelope

Build the first honest spatial envelope for the logistic and chemotaxis flux sources. For the divergence term, boundedness alone is not enough: its cosine coefficient contains a spatial derivative and may grow like `k`. The base must use the first positive-time spatial smoothing of `u`/the flux, not a fictitious `Lβàû` coefficient decay.

### G3. Source-from-solution pass

Prove one reusable nonlinear pass:

```lean
SourceFromSolutionEnvelopePass S
```

morally

```text
solution envelope at order m
  βÜ£ source envelope at order m-1.
```

Its analytic pieces are:

```text
u βÜΠ u^Έ≥ on a positive floor,
R[u] gains two spatial derivatives,
(u, R, R_x) βÜΠ chemotaxis flux by product/composition rules,
one derivative is lost when taking div(flux),
IBP converts spatial derivative bounds into coefficient decay.
```

This is the only place where real-power Nemytskii estimates and flux algebra should live.

### G4. Heat gain

The restarted Duhamel formula gains two spatial orders from the source envelope. Iterate `G3 βÜ£ G4` finitely until the coefficient exponent crosses the `s > 5/2` threshold.

### G5. Per-mode time ODE

For each `k`, prove the scalar derivative formula by ordinary FTC/Leibniz differentiation of

```text
βàΪβ²Ä ≥ exp(-ΈΫ ΈΜ_k (r-q)) s(œ³+q,k) dq.
```

This has no heat-kernel `1/(t-s)` singularity. The singularity only appears when differentiating the full kernel before resolving into modes.

### G6. Uniform summable majorant

Use the spatial weighted envelope to dominate

```text
|c'_k| βâΛ ΈΫ ΈΜ_k |c_k| + |s_k|.
```

This allows termwise time differentiation and continuity of the derivative series on compact positive-time windows.

### G7. Package the repository interfaces

Construct:

```text
DuhamelSourceTimeC1 (bFormSourceCoeffs p S.u),
RestartRepresentation S,
LogisticCosineFourierData,
ChemDivCosineFourierData,
ResolverSourceWitnessFromMild p S.
```

Then use the already-landed chain

```lean
BFormMildSpectralLeaves
  βÜ£ bFormMildSpectralBootstrapData_of_leaves
  βÜ£ localClassicalSolution_of_conjugateMild_spectral.
```

This avoids rebuilding the classical endpoint theorem and prevents two incompatible regularity architectures from coexisting.

### Cosine normalization

The project convention is:

```text

a_n = 2 βàΪβ²Ä¬Ι f(x) cos(nœÄx) dx,  n βâΞ 1.
```

Even reflection to `[-1,1]` and `tsum_sq_fourierCoeffOn` give

```text

  = |aβ²Ä|¬≤ + (1/2) βàë_{nβâΞ1} |a_n|¬≤.
```

The target theorem should be something like:

```lean

    (βàΪ x in (0 : β³ù)..1, βÄ•f xβÄ• ^ 2)
      = βÄ•cosineCoeff f 0βÄ• ^ 2
        + (1 / 2 : β³ù) * βàë' n : β³ï, βÄ•cosineCoeff f (n+1)βÄ• ^ 2
```

with the exact existing coefficient name substituted.

### Sine-output Parseval

For a real `G` on `[0,1]`, define its odd reflection on `[-1,1]`:

```text
Godd(x) = G(x)       for x βâΞ 0,
Godd(x) = -G(-x)     for x < 0.
```

For `n βâΞ 1`, if

```text
b_n = 2 βàΪβ²Ä¬Ι G(x) sin(nœÄx) dx,
```

then

```text
fourierCoeffOn(-1,1,Godd,n) = -(i/2) b_n,
fourierCoeffOn(-1,1,Godd,-n) =  (i/2) b_n,
```

and the zero coefficient vanishes. Parseval gives

```text
βàΪβ²Ä¬Ι |G|¬≤ = (1/2) βàë_{nβâΞ1} |b_n|¬≤.
```

Lean steps:

```text
1. unfold `fourierCoeffOn` with `fourierCoeffOn_eq_integral`;
2. split `[-1,1]` at zero;
3. transform the negative half with `intervalIntegral.integral_comp_neg`;
4. use oddness and `exp(-iΈΗ)-exp(iΈΗ) = -2i sin ΈΗ`;
5. apply `tsum_sq_fourierCoeffOn`;
6. pair the `n` and `-n` terms.
```

For many heat-gradient bounds, full sine completeness is unnecessary. If only

```text
βàë |sineCoeff G n|¬≤ βâΛ C βÄ•GβÄ•¬≤β²²
```

is needed, orthonormality/Bessel is sufficient. Use full odd-reflection Parseval only where equality or the reverse inequality is genuinely consumed.

## 5. Schauder versus spectral: final choice

### Abstract Schauder/maximal regularity

On paper:

```text
F βàà C^{ΈΗ/2,ΈΗ}
  β΅£ u βàà C^{1+ΈΗ/2,2+ΈΗ}
```

is the shortest statement. In Lean it would require a substantial general stack:

```text
Neumann realization of the Laplacian,
sectoriality/analytic semigroup domains,
parabolic HΟΕlder spaces,
boundary compatibility,
localization and endpoint traces,
Schauder or maximal-regularity estimates.
```

That is not economical for the present 1D closure.

### Explicit kernel HΟΕlder route

This is the best low-level analytic route if starting from nothing. Its irreducible brick is the cancellation estimate

```text
C^ΈΗ βÜ£ œÉ^(-1+ΈΗ/2) second-derivative bound.
```

The repository already uses this successfully for the `C^{1+ΈΖ}`/Wiener feed.

### Current repository spectral route

This is now cheapest because:

1. the classical consumer is complete;
2. cosine reconstruction, endpoint Neumann behavior, time derivative continuity, resolver regularity, and PDE agreement are already packaged;
3. `IntervalTruncatedSpectralProducer` has reduced the remaining work to a small explicit leaf bundle;
4. the restart representation handles continuous initial data cleanly;
5. the proof can reuse existing coefficient and Duhamel engines instead of formalizing a new general Schauder API.

Therefore:

> **Use the restart-cosine spectral route for the current theorem. Keep the HΟΕlder kernel cancellation lemmas as supporting bricks and as the independent route to `C^{1+ΈΖ}`, but do not make a new monolithic `neumannDuhamel_positiveTime_C2_slice` the primary closure path.**

## Bottom line

- `C^{2,1}` is attainable from continuous initial data **only for positive time**.
- Bounded-source smoothing alone is insufficient; the endpoint needs HΟΕlder cancellation or its spectral equivalent.
- The spectral threshold consumed by the current code is local uniform `βàë ΈΜ_k |c_k| < βàû` plus summable continuous time derivatives.
- In the `MemHSob` scale, cross `s > 5/2`; an integer ladder should reach `s = 3`.
- Restart at `tβ²Ä/2` is the clean non-circular device.
- Finish cosine Parseval through the already-landed even-reflection `AddCircle 2` bridge; use odd reflection only for the sine equality genuinely needed.
- For this repository, close `BFormMildSpectralLeaves`; do not build a competing abstract Schauder framework or a second monolithic mild-to-classical theorem.
