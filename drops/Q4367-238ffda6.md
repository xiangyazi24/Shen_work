# Definitive verdict

The clean weak Stampacchia comparison is valid, but the endpoint claim needs one important correction.

1. The Neumann resolver does satisfy

```text
R_x(t,0) = R_x(t,1) = 0.
```

Consequently the actual drift coefficient

```text
g := R_x / (1+R)^β
```

also vanishes at both endpoints, and any boundary trace such as

```text
[g u z_+]_0^1
```

is zero.

2. **Endpoint vanishing kills only the boundary term. It does not kill an interior term**

```text
∫ g_x u z_+.
```

If this term is genuinely left over, the operators used for `w` and `u` have not been matched. Even assuming `u` and `g_x` bounded does not turn it into a homogeneous `C‖z_+‖₂²` term. It is only linear in `z_+`.

3. If `w` and `u` are sub/supersolutions of the same divergence-form operator

```text
P[v] := v_t - κ v_xx + χ₀ ∂_x(gv) - c v,
```

then subtracting gives

```text
P[w-u] ≤ 0
```

and the test with `z_+`, `z=w-u`, contains no `g_x u z_+` term at all. In this direct divergence formulation, the energy proof needs `g∈L∞`, but **does not need `g_x∈L∞`**.

4. If one insists on expanding to nondivergence form, then the complete operator is

```text
v_t = κ v_xx + B v_x + C v,
B = -χ₀ g,
C = c - χ₀ g_x.
```

In that formulation `g_x∈L∞` is needed to bound the full zeroth-order coefficient `C`, but after subtraction it multiplies `z`, hence contributes `g_x z_+²`, which does close by Gronwall. It must never remain as `g_x u z_+`.

Thus the answer to question (1) is:

> `R_x|∂=0` is true and removes the boundary trace, but endpoint vanishing alone does not control an interior `g_x u z_+` mismatch. Direct matched divergence form needs no `g_x` bound; matched expanded form needs `g_x∈L∞` as part of the full reaction coefficient.

# 2. Exact matched-divergence subtraction

Let `I=(0,1)`, let `κ>0` denote the parabolic diffusion coefficient, and define

```text
P[v] := v_t - κ v_xx + χ₀ ∂_x(gv) - c v.
```

Use `κ` rather than `ν` in the proposed Lean API, because in the current `CM2Params` code the field `p.ν` is an elliptic-source parameter, while the parabolic diffusion in the compiled B-form PDE is normalized to `1`.

Assume, on a restarted time strip `[s,T]`,

```text
P[w] ≤ 0,             -- w is a subsolution
P[u] ≥ 0.             -- u is a supersolution
```

For an exact weak solution, `P[u]=0`, so the supersolution condition is automatic. Set

```text
z := w-u.
```

Linearity of the frozen-coefficient operator gives

```text
P[z] = P[w]-P[u] ≤ 0.
```

This exact subtraction is the structural heart of the argument. In particular,

```text
gw-gu = g(w-u)=gz.
```

No `g_x u` term is generated when the divergence terms are kept intact.

The weak inequality for nonnegative `φ∈H¹(0,1)` is

```text
⟨z_t,φ⟩
  + κ ∫ z_x φ_x
  - χ₀ ∫ g z φ_x
  - ∫ c z φ
≤ 0.                                                (W)
```

The signs follow from

```text
∫ χ₀ ∂_x(gz) φ = -χ₀ ∫ gz φ_x
```

with the natural Neumann/no-flux boundary condition.

# 3. Exact Stampacchia energy identity

Define

```text
z_+ := max z 0,
E(t) := ∫_0^1 z_+(t,x)^2 dx,
D(t) := ∫_0^1 |∂_x z_+(t,x)|^2 dx.
```

For

```text
z ∈ L²(s,T;H¹(0,1)),
z_t ∈ L²(s,T;H⁻¹(0,1)),
z ∈ C([s,T];L²(0,1)),
```

the Stampacchia truncation theorem gives

```text
z_+ ∈ L²(s,T;H¹(0,1)),
∂_x z_+ = 1_{z>0} ∂_x z       a.e.,
⟨z_t,z_+⟩ = 1/2 E'(t)         a.e.
```

It also gives, almost everywhere,

```text
z z_+ = z_+²,
z_x (z_+)_x = |(z_+)_x|²,
z (z_+)_x = z_+ (z_+)_x.
```

Insert `φ=z_+` into `(W)`. The exact energy inequality is

```text
1/2 E'(t) + κ D(t)
≤ χ₀ ∫_0^1 g z_+ (z_+)_x dx
  + ∫_0^1 c z_+² dx.                              (E)
```

If the barrier has a residual

```text
P[w]=r_w,
r_w≤0,
```

then subtraction yields `P[z]≤r_w≤0`; testing contributes

```text
⟨r_w,z_+⟩≤0,
```

which is favorable and is simply discarded. Thus the already-proved squared-heat residual inequality fits this comparison exactly, provided it was proved for this same complete operator.

# 4. Young absorption and exact Gronwall constant

Assume

```text
|g(t,x)| ≤ G,
c(t,x) ≤ C_c
```

a.e. on the strip, with `G≥0` and `C_c≥0`. The sharp quantity for the reaction is the essential supremum of `c_+`; using `‖c‖∞` is a safe larger choice.

By Cauchy-Schwarz and Young,

```text
|χ₀ ∫ g z_+ (z_+)_x|
≤ |χ₀| G ‖z_+‖₂ ‖(z_+)_x‖₂
≤ κ/2 D(t) + χ₀² G²/(2κ) E(t).
```

Also,

```text
∫ c z_+² ≤ C_c E(t).
```

Hence `(E)` gives

```text
1/2 E'(t) + κ/2 D(t)
≤ ell E(t),

ell := χ₀² G²/(2κ) + C_c.                         (H)
```

Dropping the nonnegative dissipation,

```text
1/2 E'(t) ≤ ell E(t),
```

or equivalently

```text
E'(t) ≤ 2 ell E(t).                               (G)
```

Therefore, for `s≤t≤T`,

```text
E(t) ≤ E(s) exp(2 ell (t-s)).
```

The sign assumption `χ₀<0` is not needed for this energy estimate; only `χ₀²` appears. The sign is relevant upstream when constructing the particular squared-heat barrier and its coefficient bounds.

From the restart comparison

```text
w(s,x)=f(x)^2 ≤ u(s,x),
```

one gets

```text
z_+(s,x)=0,
E(s)=0.
```

It is not necessary that `z(s)=0`; the inequality seed is enough. Gronwall then yields

```text
E(t)=0.
```

Thus `z_+(t)=0` a.e., hence `w(t)≤u(t)` a.e. Spatial continuity of the two slices upgrades this to every `x∈[0,1]`. Since the Neumann heat semigroup is positivity improving on a nonzero nonnegative seed,

```text
w(t,x)>0
```

for every strictly later time, and therefore

```text
u(t,x) ≥ w(t,x) > 0.
```

# 5. Why a surviving `g_x u z_+` term is fatal

Suppose one incorrectly compares the divergence-form solution

```text
u_t = κu_xx - χ₀ ∂_x(gu) + cu
```

with a barrier satisfying only the incomplete nondivergence equation

```text
w_t ≤ κw_xx - χ₀ g w_x + cw.
```

Expanding the equation for `u`,

```text
-χ₀ ∂_x(gu) = -χ₀ g u_x - χ₀ g_x u,
```

and subtracting gives an extra forcing term of the form

```text
χ₀ g_x u.
```

The positive-part test then contains

```text
χ₀ ∫ g_x u z_+.
```

Even if

```text
|u|≤M,
|g_x|≤H,
```

one only gets

```text
|χ₀ ∫ g_x u z_+|
≤ |χ₀| M H ‖z_+‖₁
≤ |χ₀| M H |I|^(1/2) E(t)^(1/2).
```

This is not bounded by `C E(t)` uniformly near `E=0`. Applying Young produces

```text
A E^(1/2) ≤ ε + A²/(4ε) E,
```

which introduces a positive forcing constant `ε`. Starting from `E(s)=0`, that only proves a positive upper bound, not `E≡0`.

Endpoint vanishing of `g` changes none of this interior estimate. It removes only

```text
[g u z_+]_0^1.
```

Therefore:

> If `∫g_x u z_+` appears, do not try to estimate it. Repair the operator so that it disappears.

There are exactly two sound repairs.

## Repair A: stay in divergence form

Use

```text
P[v]=v_t-κv_xx+χ₀∂_x(gv)-cv
```

for both `w` and `u`. Then subtraction gives `∂_x(gz)` and only `g∈L∞` is needed in the energy estimate.

## Repair B: expand the complete operator consistently

Write

```text
B := -χ₀ g,
C := c - χ₀ g_x.
```

Require both fields to satisfy

```text
v_t = κv_xx + Bv_x + Cv.
```

Then subtraction produces

```text
B z_x + C z,
```

and the `g_x` contribution is

```text
-χ₀ g_x z z_+ = -χ₀ g_x z_+²,
```

which is bounded by

```text
|χ₀| ‖g_x‖∞ E(t).
```

Here `g_x∈L∞` is sufficient because it multiplies `z`, not `u`.

This is precisely the coefficient-completeness issue noted in Q4350: a nondivergence reaction coefficient that omits `-χ₀g_x` is not the same operator as the original chemotaxis divergence equation.

# 6. Minimal analytic hypotheses

The proposed list

```text
u≥0,
u bounded,
u continuous,
u∈L²(H¹),
the mild identity,
w a divergence-form subsolution,
g endpoint-vanishing,
g_x∈L∞
```

is neither minimal nor quite sufficient as stated.

The true minimal comparison core is the following.

## Required by the abstract weak comparison

1. **Positive diffusion**

```text
κ>0.
```

2. **Energy-space regularity for both fields**, or directly for their difference

```text
u,w ∈ L²(s,T;H¹(0,1)) ∩ C([s,T];L²(0,1)),
u_t,w_t ∈ L²(s,T;H⁻¹(0,1)).
```

Equivalently, it is enough to assume these properties for `z=w-u`.

3. **Matched weak sub/supersolution inequalities** for the same `g` and `c`

```text
P[w]≤0,
P[u]≥0
```

in the variational sense for every nonnegative `H¹` test. Equality for `u` is sufficient.

4. **Coefficient bounds**

```text
g∈L∞((s,T)×(0,1)),
c_+∈L¹(s,T;L∞(0,1)).
```

For the constant-coefficient Gronwall used above, take the stronger convenient form

```text
|g|≤G,
c≤C_c
```

uniformly on the strip.

5. **Initial comparison in the `L²` trace sense**

```text
w(s)≤u(s)    a.e.
```

6. **Stampacchia chain rule** for `z_+`, including the time-chain identity. This follows from the energy-space hypotheses mathematically, but in the current repository style it should be an explicit producer field until the full Sobolev truncation theorem is available.

These hypotheses yield `w≤u` a.e.

## Needed only for the pointwise conclusion

7. Spatial continuity of `u(t,·)` and `w(t,·)` on `[0,1]`, so that an a.e. inequality upgrades to an everywhere inequality.

## Needed upstream to construct the PDE coefficients and barrier, but not by the comparison core

- `u≥0` is useful for the real powers, the resolver source, and the logistic coefficient.
- A uniform bound on `u` is useful to prove uniform bounds on `g`, `c`, and, in the expanded route, `g_x`.
- The mild identity is useful only after a **mild-to-weak bridge** converts it to the matched variational inequalities. A value-level mild identity by itself is not enough to justify testing with `z_+`.
- Resolver endpoint vanishing is useful when deriving the weak form from a strong form. It is not an additional requirement once the weak divergence form with natural boundary conditions has already been supplied.
- `g_x∈L∞` is not required by the direct divergence comparison. It is required only by the expanded nondivergence barrier route or by a separate proof that the full zeroth-order coefficient is bounded.

Thus the clean direct-divergence producer should **not** carry `g_x` at all.

The existing negative-part proof sets the test to

```text

```

Its chemotaxis term vanishes exactly because the faithful truncated flux has a leading `u_+` factor, while the derivative of `u_-` is supported where `u≤0`. This is implemented through

```lean
truncatedChemFluxLifted_mul_negDeriv_integral_eq_zero.
```

For the comparison proof, set

```text
z=w-u,
test=z_+.
```

There is no disjoint-support cancellation. The matched drift term survives and is controlled by Young's inequality. Therefore the new estimate data need a drift-bound field rather than the negative-part proof's exact chemotaxis-zero field.

The sign orientation is also essential:

```text
P[w]≤0   -- barrier subsolution
P[u]≥0   -- physical solution supersolution
```

implies

```text
P[w-u]≤0,
```

which is the direction compatible with testing `(w-u)_+`. Reversing either inequality gives the wrong sign. An exact solution `P[u]=0` is, of course, a supersolution.

So the answer to question (3) is:

> It is a mirror of the negative-part proof from the tested energy inequality onward, but it requires a subsolution/supersolution pair and Young absorption of the drift; it is not the same support-cancellation argument.

# 8. Recommended Lean API

The least risky implementation is two-layered.

## Layer 1: a repo-compatible positive-part energy core

Use definitions parallel to the existing negative-part file:

```lean
open Set MeasureTheory
open ShenWork.IntervalDomain
  (intervalDomainLift intervalDomainPoint intervalMeasure)

/-- Lifted positive comparison defect `(w-u)_+`. -/
def comparisonPositivePartLift
    (w u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ → ℝ :=
  fun x => positivePart
    (intervalDomainLift (w t) x - intervalDomainLift (u t) x)

/-- `L²` energy of the comparison defect. -/
def comparisonPositivePartEnergy
    (w u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ :=
  ∫ x, (comparisonPositivePartLift w u t x) ^ 2 ∂ intervalMeasure 1

/-- Spatial dissipation of the comparison defect. -/
def comparisonPositivePartDissipation
    (w u : ℝ → intervalDomainPoint → ℝ) (t : ℝ) : ℝ :=
  ∫ x, (deriv (comparisonPositivePartLift w u t) x) ^ 2
    ∂ intervalMeasure 1
```

For a genuinely weak implementation, the last definition should ultimately use the weak derivative supplied by the Sobolev representative. In the current repository style, carrying the required chain-rule identities as fields is consistent with `NegativePartEnergyEstimateData`.

The exact scalar core should be:


# 9. Exact Gronwall closure in Lean style

```lean

have hE_nonneg : ∀ t ∈ Set.Icc s T, 0 ≤ E t := by
  intro t ht
  have _hint := H.energy_integrable t ht
  exact integral_nonneg fun x => sq_nonneg (comparisonPositivePartLift w u t x)

    ∀ r ∈ Set.Ico s T, H.E' r ≤ (2 * H.ell) * E r := by

  exact ShenWork.Paper2.intervalDomainL2_gronwall_exp_of_diffIneq
    (E := E) (E' := H.E') (K := 2 * H.ell)
    ht.1
    (H.energy_cont.mono (Set.Icc_subset_Icc_right ht.2))
    (fun r hr => H.energy_has_deriv r ⟨hr.1, lt_of_lt_of_le hr.2 ht.2⟩)
    (fun r hr => hderiv_le r ⟨hr.1, lt_of_lt_of_le hr.2 ht.2⟩)

  simpa [H.initial_zero] using hgron
```

# Final coding recommendation

Build the direct-divergence producer first:

```text
same weak operator
+ positive-part Stampacchia chain
+ g∈L∞
+ upper bound on c
+ initial comparison
→ half-energy inequality
→ existing scalar Gronwall
→ w≤u.
```

Do **not** add `g_x` merely to estimate a leftover `g_x u z_+`; that leftover is evidence of a malformed subtraction. Use `g_x` only if deliberately choosing the fully expanded operator with the complete coefficient `c-χ₀g_x`.
