# 3. The exact `R ‚Üí L` derivative formula

For the restart series, each mode satisfies the scalar ODE

```text
c‚Çô'(œÑ) = src(œÑ,n) - Œª‚Çô c‚Çô(œÑ).
```

This requires only continuity of

```lean
fun œÑ => src œÑ n
```

for each fixed `n`; it does not require the source coefficients themselves to be differentiable in time.

At an absolute target time `t`, with restart base `t/2`, define

```lean
restartSliceCoeff S t n :=
  localRestartCoeff
    (cosineCoeffs (intervalDomainLift (S.u (t / 2))))
    (fun r n => bFormSourceCoeffs p S.u (t / 2 + r) n)
    (t - t / 2) n
```

and then define the candidate time derivative by

```lean
udot t x :=
  ‚àë' n,
    (bFormSourceCoeffs p S.u t n
      - unitIntervalCosineEigenvalue n * restartSliceCoeff S t n) *
      cosineMode n x.1.
```

There is no factor `p.ŒΩ` in front of `unitIntervalCosineEigenvalue` in this formula: the compiled `u` equation has unit diffusion. The parameter `p.ŒΩ` belongs to the elliptic resolver source, not to the parabolic heat eigenvalue in the `u` equation.

The summability estimate is

```text
|src(t,n) - Œª‚Çô c‚Çô(t)|
  ‚â§ |src(t,n)| + Œª‚Çô |c‚Çô(t)|.
```

Thus a summable source envelope plus `R` gives absolute summability of the derivative coefficients. Since

```text

```

the derivative series converges uniformly in `x`, and therefore `udot t` is continuous.

On a compact positive-time window, uniform source and eigenvalue envelopes give

```text

  ‚â§ ‚àë' n, sourceEnv n + eigEnv n,
```

uniformly in both `t` and `x`. This is exactly the third field required by `MildSolutionSliceHasDerivAtTime`.

```text

  ‚Üí WindowSourceEnvelope 2
```

The last arrow after `L` is not another smoothing bootstrap. It consists of:

- differentiating the logistic coefficient using `HasDerivAt (fun r => S.u r x)`;
- differentiating the resolver source and flux using elliptic linearity and the local chemotaxis chain rule;
- combining the logistic and chemotaxis coefficient packages into the full B-form source package.

So `R ‚Üí L` is the last **new regularity argument**. It is not the last literal line of the full HSpectral proof.
